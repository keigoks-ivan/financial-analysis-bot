為什麼這個排序

短期 12 個月內,**進場時機 + 動能狀態**是決定性因素。MSFT 在這層勝出的關鍵是「**估值極低位 + 動能已冷卻 + 結構性 thesis 不變**」三者齊全 — 這是極罕見的組合。Fwd PE 24.3x 對於一家 op margin 45%+ / FCF $80B+ / 護城河 9/10 的公司是 2023 以來最便宜進場機會。DD 4/18 寫稿時的「1 週 +13% 漂移」警示已被市場完全消化(過去 1 月對大盤 underperform),現在是合理進場區。

AMZN 在這層第二的問題是**動能極端透支**。雖然 thesis 強(Trainium $225B 訂單、AWS +28%),但 RSI 76.57 + 5/5 創 ATH 意味著歷史上 60% 機率 1 個月內回測 8-15%。DCA 報告本身也明確要求「條件式進場」— RSI 回到 50-60 或回測 $245 BB 中軌才可建首批。Q2 26 7/31 財報雖是 catalyst,但**現價任何進場都是追高**。

GOOGL 在這層最差不是基本面,而是**動能極端透支 + R:R 雙 < 1.0**。4 週漲 +31.5% 是三檔中最極端,DD 壓測 1/4 通過(僅 5Y 過關)。DD 明確警示「現價追高 R:R 高度不對稱」。等回測至 $340 RSI < 60 起首批 1.0%,$320 BB 中軌加至 2.5%,$286 W52 +10% 區再加至 4.0%,這才是合理進場節奏。

為什麼這個排序

2-3 年中期 view 下,「**拐點催化兌現 + Cloud margin 翻倍驗證**」雙重決定排序。GOOGL 在這層勝出的關鍵是 **Cloud op margin 從 17.8% → 32.9% 已是「真實已驗證」的 capex monetize 證據**,不是承諾。對比 MSFT 的「FY27 Cloud GM 從 64% 回升至 66%+」仍是承諾、AMZN 的「FY27 H1 FCF 修復至 $25B+」也是承諾。GOOGL 是三檔中**最近兌現的**,加上 Cloud +63% 是 hyperscaler 最快增速,2-3Y 拐點 alpha 明確。

但這個排序有個前提:**GOOGL 必須在 $320-340 進場**。現價 $385 已 priced in 大部分 2-3Y upside(DD 短中期 R:R 雙 < 1.0)。若無法等回測,GOOGL 中期 alpha 完全被吃掉。

MSFT 在這層第二是因為 **Pattern Match 自身 2018-22 經驗**:當時 Forward PE 22x 起 + Azure 從 25% YoY 升至 35% YoY + capex 從 $25B 升至 $50B,5 年股價 +200%。現在條件類似但 capex 強度高 2.4 倍 + AI 競爭激烈,所以 2-3Y +30-50% 是合理 attenuated 預估。Copilot seat economics 是純 software business(不依賴 capex ROI),這是 MSFT 獨有的 driver 結構。

AMZN 在這層第三的關鍵問題是**估值已 priced + 無 PE re-rate alpha**。Fwd PE 28x 對應 FY27 EPS $9.4 隱含「2028 EPS 達 $13」market expectation。即使 Trainium 兌現,PE 維持 26-30x = 沒有倍數擴張紅利,IRR 只能靠 EPS 複利 10%/yr 跑出。對比 MSFT 24.3x 起點仍有 +15% PE 修復空間 + GOOGL Cloud margin 翻倍故事更性感,AMZN 中期 alpha 確實最弱。

為什麼這個排序

5-10 年 view 下,**範式風險可控性 + 全棧深度**決定排序。GOOGL 在這層勝出的關鍵是「**5 條獨立成長線**」— Search + Cloud + YouTube + Android + Gemini/TPU,加上 Waymo 自駕(2027+ 商業化)和 Apple Gemini 整合是真正的第二曲線。10Y Bull case +130% 是三檔倍數最大(假設無 antitrust 拆分)。但 antitrust binary 是真實的 tail risk,**若觸發強制拆分,從第一掉到第三**。

MSFT 在 10 年 view 下第二,核心優勢是**「企業 IT 標準的事實壟斷」10 年難動搖**。Office + Active Directory + Identity + Copilot 四層 lock-in 是 mega cap 中最深的 switching cost 護城河,即使 OpenAI exclusivity 結束、Anthropic 部分走 AWS、capex ROI 質疑,核心軟體業務 GM 70%+ + 客戶 sticky 結構不變。範式風險分散(Anthropic ~20% 商業性 + antitrust ~10% 政策性,各自獨立)讓 Bear case -10% 下行保護最強。

AMZN 在 10 年 view 落到第三,核心問題是**「IRR 6-8%/yr 低於 S&P 10% 是結構性不利」**。Trainium ramp 即使成功,Anthropic + OpenAI 集中度高的結構性風險仍在;FCF 修復可能拖到 2029+,中途市場質疑會反覆觸發 drawdown 18-30 個月。FTC 反壟斷 Feb 2027 trial 雖然可能僅拆 Marketplace(對 AWS 不動),但**整體 thesis 的「三引擎飛輪 + Trainium 補位」的兌現速度明顯慢於 MSFT 的 Copilot ramp 或 GOOGL 的 Cloud margin 翻倍**。

推薦標的:MSFT(3-5 年 view)

3-5 年 view 下,MSFT 是**結構最對稱**的標的。四個關鍵理由:

第一,Pattern Match 是自身已驗證歷史對應,可信度遠高於跨公司對標。MSFT 2018-22 自身 Azure ramp 期 + capex 加碼期 + Nadella 同一 CEO 紀律,5 年股價 +200%(IRR 25%/yr)。當時 Forward PE 22x 起 + Azure 從 25% YoY 升至 35% YoY + capex 從 $25B → $50B,市場質疑 ROIC 但 Nadella 持續加碼 — 完整對應當前 setup。即使打 attenuation 折扣到 +57%/5Y(因 capex 強度高 2.4 倍 + AI 競爭激烈),仍是三檔中 Base IRR 最高的 9.4%/yr。

第二,IRR composition 三引擎全順風 + 變數獨立性最強。MSFT 是三檔中唯一「EPS +12-15% + PE re-rate +3% + buyback +2%」三引擎全順風的標的。EPS 複利強(Copilot seat ramp 是 pure software economics + Azure share gain 是雙 driver),PE re-rate 紅利穩定(起點 24.3x 是 5Y 極低位 10-15% 分位,修復至 26-28x 仍有 +15% 空間),buyback +2%/yr 提供結構性順風($25B+ 年回購)。Driver 抽象層校驗:Copilot 與 Azure 是兩條獨立因果鏈,即使 Anthropic 走 AWS(影響 Azure 部分),Copilot seat business 仍可獨立成長。

第三,Bear case -10% 是三檔中下行保護最強。即使最壞情境(Anthropic 走 AWS + OpenAI cap 提前 + Copilot 滲透 plateau)同時觸發,mega cap $80B+ FCF + 多元業務(Office + Azure + LinkedIn + GitHub)+ 軟體 lock-in 提供強保護。-10% 對比 AMZN -35% 和 GOOGL -25~-60%,**進場估值起點是 Bear case 跌幅最大的單一決定因素**。同樣的賭注,但下行不對稱地保守。

第四,現價即可進場,無需等回測。MSFT 是三檔中唯一動能已冷卻 + 估值 5Y 極低位的標的。Fwd PE 24.3x 是 2023 以來最便宜,4 週漂移 ~0%(已從 DD 4/18 寫稿時的 +13% 完全 mean reverted),RSI 中性 48-62。對比 AMZN RSI 76.57 嚴重超買 + GOOGL 4 週 +31.5% 極端過熱,**MSFT 是唯一不需等回測就能 capture alpha 的 hyperscaler**。
