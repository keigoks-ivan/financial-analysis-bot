本層排序

  1. 🥇 GLW — 三檔中唯一「進場條件距離可達（−12~−21%）+ 動能已收斂 + Pure MA 全多頭」三項齊全。Q2 2026 earnings（7/29 ± 1 週）是關鍵 binary，但即使 miss 也有 1.1% 股息 + 多元化 cushion；若 5/6 Investor Day 後 Solar 段持續虧損則減碼觸發但不清倉。
  2. 🥈 SNDK — 短期 thesis 動能最強（$42B backlog + GM 78% 史上最高 + Q3 FY26 +251% YoY），但 4w +132% 為「v12.2 框架以來最差時機」（DCA §7c 原文）。動能末段，若 5/27 之後類股輪動則 mean reversion 觸發 −20~−35%。「最強 thesis + 最差 timing」的結合 — 短期排第二而非第一是因為「能否進場」比「漲不漲」更重要。
  3. 🥉 MU — RSI 84.7 極度過熱 + DD 短期目標 $580 已被現價 $757 超越 + 4/13-5/10 4 週 +78% drift +「短期目標已破」三項負面齊全。DCA 明確結論：「進場窗口已關閉」+「現持有者建議減半」。短期內部人賣壓（CEO Mehrotra 5/1 賣 $21.5M、CBO Sadana 4/14 賣 $10.1M、37:1 sell:buy）為 confirmation bias。

為什麼這個排序

短期 <12M 視角下，「能否進場」與「動能可持續性」是決定性的兩個維度。三檔的 Pure MA 結構都 ✅ 多頭，但 GLW 是三檔中唯一「等回測能等到」的標的：

GLW 進場區 $155-175 距現價 $195 僅 −12~−21%，搭配 1.1% 股息 + 175 年材料平台基本盤，是「等回測能達到、即使等不到也不會 FOMO 受傷」的標的。短期 4w drift 雖然 1Y +200% 但已開始收斂、RSI 比 MU/SNDK 都低。

SNDK 在動能末段的問題不是 thesis 錯，而是 4w +132% 漂移超過 v1.3 框架以來所有先例。DCA §7c 原文「歷史頻率 RSI > 75 後 1 個月內回測 8-15% 機率約 60%，本標的 +132% drift 過熱程度遠超平均，回測幅度可能更大」— 短期排第二不是因為 thesis 弱（SNDK $42B backlog 是 unique alpha），而是因為 timing 極端。等回測 $700-820 才有合理 R:R。

MU 在短期最差的關鍵不是「估值貴」而是「DD 短期目標已破 + 現價更糟」。DD 4/18 在 $455 時短期目標 $580、Bear anchor $312.93；現價 $757 已超越短期目標 +30%。DCA §7c 明確標「⛔ 禁止一次建倉、禁止追高第一批」+「歷史 mean reversion：RSI > 80 後 1 個月內回測 12-18% 的機率 ~65-75%（memory cycle 股加倍）」。內部人 37:1 sell:buy + CEO/CBO 雙重高金額賣壓使「公司 sold out beyond 2026 narrative」與「管理層行動」明顯背離。

本層排序

  1. 🥇 GLW — 2-3Y 中期是 GLW 的「sweet spot」：Springboard $5.75B by 2028 增量計畫已被 NVDA $3.2B（5/2026）+ Meta $6B（1/2026）+ 2 家新 hyperscaler 多年合約「數學上覆蓋」（DCA §B.4 verdict 1 推導：合約合計 ~$12B > $5.75B 增量目標）。拐點催化是「執行兌現追蹤」而非「cycle binary」，路徑可控可監測。
  2. 🥈 SNDK — 中期 thesis 的 alpha 在「$42B take-or-pay 是否真實改變 NAND 商品本質」這個二元賭。DCA 自身判定機率：cycle 延長 8-10 季 30% / cycle 反轉 normalize 50% / cycle 急轉 20%。2-3Y cycle play 視角下 IRR 可能 +15-30%/yr（DCA §4 追加壓力測試原文），但時點極差（現價 $1,562 已比 DD 5/4 snapshot $1,187 又 +32%）。排第二是因為「正期望但需 timing 配合」。
  3. 🥉 MU — 中期最危險的不是「估值貴」而是「FY27 EPS $100 共識本身是 sellside peak chasing」。DCA §3 Munger Inversion 原文：「2018Q2 sellside FY19 EPS 共識在反轉前 6 個月還上修，實際 FY19 EPS 跌 −80%；2021Q4 共識上修 FY22 EPS，實際 FY22 −85%」。MU 2-3Y view 處在「FY26 $58 → FY27 $100 → FY28 跌回 mid-cycle $35-50」的 binary 之中，賣方共識上修的速度（90 天 +78%）本身就是 cycle 末期典型行為。

為什麼這個排序

2-3Y 中期視角下，拐點催化兌現的「可控性」是分水嶺。GLW 的拐點是「Springboard 增量達標率追蹤」— 季度可監測、可漸進兌現、4 條腿（Optical / Solar / Glass substrate / Display 重組）任一兌現都撐住基線。MU/SNDK 的拐點是 cycle binary — 無法分批兌現，反轉時刻往往是 1-2 季內完成。

SNDK 排第二的具體理由：DCA §4 追加壓力測試明確「只在 1-2 年 cycle play 視角下才有 alpha（短期 Q4/Q1 兌現 → 6-12 個月期望值可能 +20% ~ +35%），但必須以 1-2 年計時且嚴格進場時機」。中期 thesis 是「LTA 結構性改寫商品 + HBF 第二曲線」，但 §6b pre-mortem 顯示 LTA 在 cycle reversal 時實際執行率 60-70%、價格被砍 25-40%（歷史 DRAM LTA 2018-2019 教訓）。Backlog 名目 $42B 不等於現金折現 $42B。

MU 在 2-3Y 中期最差的關鍵在 §3 強制減法原文：「memory PE 應該結構性 re-rate 到 14-18x narrative 在每一個 cycle 頂端都會出現（2018 也有人說 server DRAM 是結構性），但 memory PE 從未 sustained 在 cyclical 倍數之上」。MU 中期賠率不僅依賴 cycle 不反轉（已 2 年上升期），還依賴 HBM4 Rubin 中段位置撐 mid-cycle EPS（DCA Bull case 機率僅 20%）。中期賠率三檔中最差。

  2. 🥈 MU — Base IRR +1.5% 弱正但仍正；對照 SNDK −14.5% 強負，MU 在 3-5Y 視角是「相對較好」但絕對水準仍弱。MU 的問題在於 +6%/yr re-rate 貢獻是「靠市場接受 mid-cycle PE 12x」這個假設（DCA §4 IRR Composition 原文「需市場相信 mid-cycle 估值，這個 wait 通常要 2-3 年，過程中股價路徑波動極大」）。質感是「估值收斂 + 股價平移」非自然複利好抱。
  3. 🥉 SNDK — 3-5Y view 是 SNDK 的「最劣勢時間軸」。DCA §4 原文「5Y 機率加權 IRR −13.4%/yr 落入強負期望」+「即使 HBF 兌現（10Y +50% 拉長視角），10Y IRR 仍為 −7.0%/yr 負（5Y NAND CAGR 僅 5.32% 不足以 outrun 起始估值壓力）」。FY30 normalized EPS $45 × PE 15x = $675 vs 現價 $1,562 = −57%。Y5 末實質營收（$10-11B）可能低於 FY26 cycle peak。

為什麼這個排序

3-5Y view 是這次對比的 alpha 集中地。三檔在這個時間軸的差異最大 — 不是「賠率高低」差異，而是「賠率為正 vs 為負」的根本差異。

GLW 的 IRR 質感是這次比較的核心洞察：三檔中只有 GLW 是「EPS 自跑型」，其他兩檔都是「靠 cycle / 靠市場配合型」。MU Base case 中 EPS 是反向複利（−5%/yr）、SNDK Base case 中 EPS 是衰退（−12%/yr）— 這兩檔的 5Y Base IRR 在數學上若想為正，必須依賴「估值 re-rate」這個我們無法控制的外部變數。GLW 即使估值從 51x 壓回 25x（這已經是相對嚴厲的假設）、EPS 仍能補回大部分缺口。

SNDK 在 3-5Y view 是「Pattern Match 對標歷史最不利」的時間軸 — Micron 2017Q4 cycle peak 後 5Y 平盤（+2~4%/yr）但中途兩次 −47% / −50% drawdown。本次 SNDK 的 setup 比 Micron 2017Q4 更極端：Forward PE 8.6x（cycle peak compression）與當時 Micron 4-6x 同樣便宜假象、GM 78% 史上最高與 Micron 2018Q2 GM 高峰類似、4w 漂移 +132% 比 Micron 2017Q4 ~+60% 更極端。歷史 5Y 終點報酬僅 +9-20% 累計（即 1.7~3.7%/yr）。

MU 排第二的關鍵不是「+1.5% IRR 不錯」而是「相對 SNDK 較好」。絕對水準上，MU 5Y IRR +1% 仍 dominated by S&P 500 +10% / NVDA +15% / SK Hynix +8-12%（這是 MU DCA §7a opportunity cost 原文）。3-5Y view 的「相對排名」與「絕對接受度」是兩件事，MU 在三檔中排第二但不代表「值得進場」。

為什麼這個排序

GLW 175 年材料累積的具體含義：歷史上 GLW 在光纖泡沫（1999-2002 −99%）後重生於 LCD 玻璃、LCD 飽和（2011-2014 −50%）後重生於 Gorilla Glass、Gorilla Glass 成熟（2018-2023 −25%）後現正重生於 AI optics + Solar + Glass substrate。每一輪 cycle reset 後都能找到下一條 S 曲線。這個 architecture optionality 是 MU/SNDK 沒有的（兩家在「DRAM/NAND 商品化」這個單一 architecture 內，cycle reversion 只能等下一輪 cycle）。

MU 排第二的核心邏輯：HBM 結構性擴張（$30B → $100-120B by 2030）的事實本身為正，但 MU 在 HBM4 旗艦戰役被擠到 Rubin CPX 中段的位置決定了「不會分到最高品質的 ASP mix」。10Y view 下，SK Hynix（HBM4 lead 60-70%）與 Samsung（4nm 自代潛在 binary upside）是 HBM 結構擴張的 prime 受益者，MU 是中段位置受益者。10Y IRR +2~4%/yr 並非「賠錢」而是「機會成本最高」— 同類選擇下 NVDA / SK Hynix 都更好。

SNDK 排第三的具體原因：DCA §A2 明確標 Runway 🔴 + Y5 末實質營收 $10-11B 低於 FY26E $24B。這意味著「即使 SNDK 完美執行所有 thesis（cycle 延長 + HBF 兌現 + LTA 結構持續）」，5 年後的營收絕對值仍會低於今天。NAND TAM CAGR 5.32% 已飽和，是這個結論的根本物理。
