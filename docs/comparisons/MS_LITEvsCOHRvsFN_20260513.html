業務地圖三個關鍵洞察(這節的 alpha):

① 業務 overlap 最深的是 LITE ↔ COHR(在 200G EML laser + 1.6T transceiver) — 但兩家定位完全不同:LITE 是 EML chip + Cloud Light 模組垂直整合,COHR 是 6-inch InP wafer 上游壟斷供應(LITE 也是 COHR wafer 客戶之一)。COHR 200G EML 2026 H2 對位 LITE 的時刻是 LITE thesis 最具體的 kill event。FN 業務本質是 EMS 純組裝,與 LITE/COHR 在不同 layer(LITE/COHR 是 chip/module vendor,FN 是 assembler),所以 FN 失效路徑是「被 LITE/COHR 自身垂直整合 + Broadcom in-house 三方擠壓組裝份額」,不是直接 socket 對打。

② COHR 業務最 differentiated — 三檔中唯一做 6-inch InP wafer 規模量產(獨家),這是上游晶圓供應商 layer,不在 LITE/FN 產品線裡。COHR 的成敗很大程度上獨立於 LITE 與 FN 的競爭 — 即使 LITE 失去 NVDA single source,COHR 仍可從 LITE wafer 採購 + 自身模組業務 + NVDA CPO laser source 三條路徑勝出。

③ 護城河 mechanism IP/IP 強度三分法(這是這個比較最關鍵的判斷):
  • 資本密集 + 規格壟斷(fab + 規模 + 政府 CFIUS 管制): COHR(6-inch InP 規模)+ LITE(InP epi + EML 專利 2000+) — 兩家都有 patentable IP 與 fab capital barrier 雙層保護
  • 純 process knowledge + 高 fixed cost cleanroom(無 IP): FN — 護城河本質是 organizational memory + 高 fixed cost 進入壁壘,但沒有專利,意味 vertical integrator(LITE / COHR 自家模組廠 + Broadcom / NVDA)有動機自建奪走份額時 FN 缺乏結構性防守
  • NVDA platform lock-in 三家皆有,但 FN 集中度最危險(27.6%),LITE / COHR 是 design-embedded 互相替代,FN 是純 assembly 可被 bypass

這個 IP 強度三分法直接解釋下方 §B.3 為何 COHR > FN > LITE 的 IRR 排序 — COHR 雙層 moat 最 durable、FN runway 中等但 cliff 在 5Y 後、LITE 短期最強但 NVDA single-source 集中 + COHR 2026 H2 對位即可大幅壓縮。

§B.3 3-5Y 中長期：護城河 mechanism 解剖 + TAM + IRR 質感(核心 alpha 層)

§B.3.a 護城河來源解剖 — 每檔 mechanism 級判斷

不是評分,是 anatomy。對每檔回答三個問題:(1) mechanism 具體是什麼? (2) 為什麼能撐 3-5 年? (3) 3-5 年內最可能讓 mechanism 失效的單一事件?

LITE — InP 國家管制 + EML 專利 + NVDA LTA single-source(短期極強,中期 NVDA dual-source 風險)

Mechanism:InP epitaxy 國家級製造難度(CFIUS 管制層級)+ EML 專利族 2000+ issued + NVDA $2B investment 含 preferred equity stake + NVDA LTA 至 2027 + co-design embedding 36 月 cycle。Components GM 47.9%(歷史新高)是這套 mechanism 的定價力具象化。

為什麼撐 3-5 年:(a) COHR 6-inch 2026 H2 達商業量產仍需 12-18 月 yield ramp 才能對位;(b) 1.6T CPO 替代在 Rubin Ultra (2027 Q4 後)才會 scale,3-5Y 內 pluggable 仍是主流;(c) NVDA LTA 至 2027 + preferred equity stake 結構性延長 switching penalty,co-design embedding 36 月 cycle 鎖死。
