Astera Labs (ALAB) vs Credo Technology (CRDO)

來源：DCA_ALAB_20260522.html（v1.4，DD v12.4 配套）｜ DCA_CRDO_20260602.html（v1.3，DD v12.4 配套）｜ 對比執行日 2026-06-03｜股價時效：ALAB 報告日 5/21 距今 13 天 → 重抓現價 $355.76；CRDO 報告日 6/2 距今 1 天 → 沿用 $226.10

§A 對比 Dashboard

推薦：CRDO（現價環境下的邊際資金首選）

適用時間框架：3-5 年為核心結論；<12M / 2-3Y 同向，5-10Y 為唯一接近的戰場。

主要理由（三條）：

  1. 同 thesis、估值天差地遠：CRDO Fwd PE 41x / PEG 0.85 vs ALAB（現價 $356）Fwd PE FY26 118.6x / PEG 3.45。ALAB 在 5/22 報告後又漲 +19.4%，估值由透支變成極端透支。
  2. IRR 質感方向相反：CRDO Base 5Y IRR +14.1%/yr，靠 EPS 自然複利（+22%/yr）撐起；ALAB 現價 Base 5Y IRR −4.3%/yr，EPS 8x 幾乎全被 multiple 從 118x 收斂至 26x 吞噬。機率加權 IRR：CRDO +13.3% vs ALAB −3.1%（差 ~16pp/yr）。
  3. 兩份 DCA 各自獨立指向同一答案：ALAB 報告 §9 明寫「邊際資金優先 CRDO」；CRDO 報告 §7a 明寫「邊際資金應優先 CRDO 而非加碼 ALAB」。互不引用卻同結論。

進場條件：兩者皆「觀望」非「現價追入」。CRDO 等回測 $185-205 / RSI 14d ≤60 / 4 週漂移 <+10% 分批；ALAB 需更深回測 $220-240（自現價 −33% 以上）才進入合理區間。關鍵 caveat：ALAB 是更好的「公司」，CRDO 是現價下更好的「投資」 — 決定變數是價格，不是品質（見 §E 時間軸敏感度）。

四層時間框架排序卡片

<12M 短期

🥇 CRDO

CRDO 在共識 PT 之下、剛交出 6/1 乾淨財報（Top1 解套）；ALAB 現價 +45% 超 PT、PE 118x、+60%/月 = mean reversion 機率最高。

2-3Y 中期

🥇 CRDO

CRDO optical 第二曲線 FY27 兌現窗、估值修復空間小；ALAB 中期被 118x→ 收斂的 multiple 拖累，催化兌現也難抵 re-rate。

3-5Y 中長期

🥇 CRDO

CRDO IRR 靠 EPS 自跑（+22%/yr）；ALAB 護城河更深但現價下 multiple 壓縮吃掉整段複利 → Base −4.3%/yr。排序對價格敏感。

5-10Y 長期

🥇 CRDO / ALAB 最接近

CRDO 10Y IRR ~18-20%/yr 數字更高；但 ALAB 護城河結構更硬（9 vs 7.5）、客戶更分散、範式風險機率較低 — 此層差距最小，若 ALAB 回測即可能翻盤。

一句話結論：四層皆 CRDO 第一，但這不是「CRDO 比 ALAB 好」的結論，而是「同一個 AI-connectivity 主題下，CRDO 41x / ALAB 118x，估值差距大到讓更好的公司（ALAB）變成更差的投資」。CRDO 是現價可分批佈局的核心候選；ALAB 是等深度回測的 watchlist。

§A.1 業務地圖 + 護城河來源對照

兩檔都是「AI 機櫃連接層」的 fabless 純玩家，但打的節點不同：ALAB 主場在 board 級 / PCIe-CXL retimer-switch（GPU↔GPU、GPU↔CPU 機櫃內 scale-up），CRDO 主場在 cable 級 AEC + optical DSP（機櫃內/跨機櫃 scale-out 連接）。這是理解「為何兩者部分重疊但非正面對打」的關鍵。

維度ALAB（Astera Labs）CRDO（Credo Technology）
營收 mix（latest）4 條產品線：Aries（PCIe/CXL retimer，~60%）+ Scorpio（PCIe/Ethernet switch fabric）+ Taurus（Ethernet SCM/AEC）+ Leo（CXL memory）。Aries 占比由 ~70% 降向 60%，多元化進行中。雙引擎：AEC（現金牛 ~50-60%+）+ optical（DSP / 矽光子 PIC / ZeroFlap，FY27 guide >$600M）+ 矽光子（DustPhotonics tuck-in）。FY26 Rev $1,335M（+206%）。

護城河來源（mechanism）① design-win lock-in：PCIe retimer 客戶切換成本極高（7-10 年 design-win lifecycle，重新 qualification 18-24 月）；② IP 投資門檻：PCIe Gen7 IP block ~$100M+ 排除小廠；③ 平台廣度：4 產品線 + NVLink Fusion / UALink 多 ecosystem 參與。① 系統級垂直整合：同時握 SerDes IP + Retimer IC + system design 三層，迭代快、成本優於「只做線材」或「只做晶片」對手；② 規模/認證壁壘；③ 條件性定價力：AI 連接緊缺 + 技術領先使「自研機會成本 > 外購」。
護城河趨勢來源→ 穩定 ← execution 加深（Scorpio X / Leo 部署）但 Marvell 競爭由「2027 量產」實質化為「已在 server vendor 量產」，pricing 端結構性升溫弱化加分。↑ widening ← NG GM 64.8%(FY25)→68.1%(FY26) +330bps；AEC 份額守 ~73%；Top1 集中度由 61%→34% = 議價風險下降。Execution 明確加深。

業務地圖洞察

  • 業務 overlap 最深處：ALAB Taurus（Ethernet AEC/SCM）vs CRDO AEC — 但兩者皆非以此為主力（ALAB 主力 PCIe/CXL retimer，CRDO 主力 AEC cable）。真正的「直接對打」其實各自指向 Marvell（ALAB 在 retimer/switch、CRDO 在 AEC），而非彼此。因此「ALAB vs CRDO」本質是同主題、不同節點的兩個收費站，可並存於組合（故兩報告皆設「合計 ≤8%」上限而非二選一）。
  • 誰最 differentiated：ALAB 的 PCIe Gen6/Gen7 retimer + CXL（Leo）是 CRDO 產品線裡沒有的獨家節點；CRDO 的 AEC（73% 份額）+ optical DSP 則是 ALAB 沒有重押的。兩者各有獨家 segment。
  • 護城河性質差異（關鍵）：ALAB 偏「結構性硬 lock-in」 — 7-10 年 design-win lifecycle + 18-24 月 requalification + $100M+ IP 門檻，抗擊穿能力強。CRDO 偏「執行領先 + 條件性定價力」 — 73% 份額靠系統級整合與良率/成本曲線，但 pricing 受 87% 客戶集中於 4 家 hyperscaler 結構性壓制，是「條件性」而非「結構性」。單看護城河品質，ALAB 的 mechanism 更硬；這是 §B.3 / §B.4 唯一可能讓 ALAB 翻盤的維度，也是為何「決定變數是價格」。

§B.3 3-5Y 中長期

§B.3.a 護城河來源解剖（mechanism 級）

ALAB — execution 9 / pricing 9，合併 9/10（A 級）
mechanism 具體是什麼：PCIe/CXL retimer 的 design-win lock-in。客戶在某世代加速器/交換機設計階段選定 ALAB 後，需 18-24 個月重新 qualification 才能換供應商，而單一 design-win lifecycle 達 7-10 年；疊加 PCIe Gen7 IP block ~$100M+ 的研發門檻排除小廠。為何能撐 3-5 年：design cycle 與 requalification 的時間尺度本身就 ≥3 年 — 即使對手 2026 開始追，歐美 Top hyperscaler 在 2029 前不會大規模切換已 lock 的世代。3-5 年內最可能讓 mechanism 失效的單一事件：NVIDIA Vera Rubin（FY28）自研 retimer ASIC（§D Single Thing，24M ~15%）或 Marvell 取得某 hyperscaler dual-source 完全切換 — 但兩者皆 binary、可觀測，非緩慢流失。

CRDO — execution 8 / pricing 7，合併 7.5/10（B 級）
mechanism 具體是什麼：AEC 賽道的系統級垂直整合（SerDes IP + Retimer IC + system 一條龍），使 CRDO 在量暴增 3x 下 GM 仍能自 64.8%→68.1%（規模攤薄 + 整合提升價值捕獲）。為何能撐 3-5 年：SerDes 是 design-in 元件，每世代（2-3 年產品週期）設計階段一旦選定即鎖定該世代；73% 份額 + 跨 4 家 hyperscaler 分散提供慣性。3-5 年內最可能失效的單一事件：任一前三大 hyperscaler 下一代加速器/交換機自研 SerDes 或轉 Synopsys/Cadence（§D Single Thing，12-24M ~20-25%）— 因 87% 營收集中於 4 家有自研能力的買方，單一客戶倒戈即量價雙殺並引發示範效應。

解剖結論：純看 mechanism，ALAB 的護城河結構上更硬 — 兩軸皆 9 分、lock-in 時間尺度更長（7-10 年 design-win vs 2-3 年 design-in cycle）、客戶更分散（Top1 25% vs 34%/87%）、失效事件機率更低（15% vs 20-25%）。CRDO 的 pricing power 是「條件性」（緊缺驅動，會隨 hyperscaler 規模放大而鬆動），這正是它 pricing 給 7 分、合併僅 7.5 的核心。唯一對 CRDO 有利的 mechanism 維度是趨勢方向：CRDO ↑ widening（GM 連升、客戶多元化）vs ALAB → 穩定（Marvell 競爭實質化）。

回應用戶問題：用 DCA 對比，該選哪一檔？

用戶要的是「用 DCA 嚴謹對比」。DCA 是 DD 之上的投資決策層，核心問題是「邊際資金放哪」。答案明確：現價環境下，CRDO 是 AI-connectivity 主題的邊際資金首選，四層時間框架皆第一。這不是「CRDO 公司比 ALAB 好」 — 恰恰相反，ALAB 護城河更深（9 vs 7.5）、GM 更高（76% vs 68%）、客戶更分散、產品線更多元，是更優質的公司。但投資報酬由「公司品質 × 進場價格」共同決定，而 ALAB 在 $356、PE 118x、PEG 3.45 的進場價下，把一個 A 級 thesis 變成了負 alpha 的投資。

為什麼不選 ALAB（具體理由）：

  1. 現價 5Y 機率加權 IRR −3.1%/yr（Base −4.3%/yr）— 顯著負 alpha，劣於 SPX ~8% 達 11pp/yr，也劣於 CRDO 達 ~16pp/yr。
  2. IRR 質感不可接受：EPS 能 4x 但 multiple 須從 118x 收斂至 26x（−78%），報酬完全人質於市場願否持續給超高倍數 — 賭市場配合，非 alpha。
  3. 時機極端透支：現價 +45% 高於分析師共識 PT $245、PE 118x、一個月 +60%；自身 DCA 對標 NVDA 2021 / SMCI 2024，短期 mean reversion 機率 60%+。
  4. 連 ALAB 自己的 DCA（§9）都說「邊際資金優先 CRDO」。

不選 ALAB 的唯一例外：若你的視角是純 10 年 + 願承受 12-18 個月 −30~40% drawdown，ALAB 的護城河深度使其在極長期仍可獲利（10Y IRR +5.8%/yr）；但即便如此，現價進場仍劣於「等回測 $220-240」。

⏱ 時間軸切換敏感度測試（本對比的真正 alpha）

若把「價格」當變數而非固定：排序會翻嗎？

  • 現價（ALAB $356 / CRDO $227）：四層皆 CRDO 第一。ALAB 估值極端透支，無懸念。
  • 若 ALAB 回測 $220-240（−33% to −38%）、CRDO 持平：ALAB 5Y IRR 回升至 +5~7%/yr，3-5Y 與 CRDO 差距大幅收窄；而 5-10Y 首選極可能翻為 ALAB — 因為一旦估值不再是致命傷，ALAB 更深的護城河（9 vs 7.5）、更長的 lock-in（7-10 年）、更低的範式風險機率（15% vs 25%）、更分散的客戶就成為決定性優勢。
  • 若 ALAB 回測至 $160-180（W104 區，極端 mean reversion）：ALAB 5Y IRR +10~13%/yr，3-5Y 與 5-10Y 兩層都可能翻為 ALAB（「更好的公司 + 合理的價格」）。

結論：本對比的排序對「ALAB 的進場價」高度敏感，對「CRDO 的進場價」次之。決定首選的核心變數是價格，不是商業品質。現價下 CRDO 全勝；但這是一個「估值差距」的勝利，不是「商業優劣」的勝利。理性 PM 的動作：現在分批佈局 CRDO（核心候選），同時把 ALAB 放 watchlist 等 $220-240 — 一旦 ALAB 回測到位，它在 3-5Y/5-10Y 反而是更優質的長期持有。
