🔭 形狀掃描 — 正在噴的多倍形狀 × 前瞻確認 × 覆蓋標籤（0 檔，—）

橫切研究狀態的雷達：用「形狀」過濾（不用覆蓋、也不用市值過濾）——掃 RS 領導（510 股日更）∪ 自訂全市場 screen（美股主板 × 市值 ≥ $0.3B × 營收 YoY ≥ +20%，系統性條件掃描、不是當日漲幅榜碰運氣），過爆發形狀閘（市值 ≥ $0.3B 無上限、季營收 YoY ≥ +20%、52 週位置 ≥ 0.6 或 RS ≥ 80、GM ≥ 25%、金融/REIT 剔除）。命中後三層標籤而非剔除：
• 前瞻：trailing 只證明過去，多倍股是「市場還沒 price in 的持續成長」——DD universe 內用 Koyfin FY1→FY3 EPS CAGR（與下方主表同源，附修正動能），universe 外用 yfinance 分析師 FY+1 營收預估。🟢 續強 ≥+15% ➖ 放緩 0~15% 🔴 反轉 <0（trailing 是基期效應/週期頂）⚪ 無預估（零分析師＝真沒被發現，是訊號不是剔除理由）。排序＝min(trailing, 前瞻) 保守取小——trailing +104% 但前瞻 −24% 的週期頂自動沉底。
• 加速：📈 預估下季 YoY 明顯高於已報告 YoY（營運槓桿點火中）／📉 明顯放緩／➖ 平穩。加速 × 🟢 前瞻 × ✅有DD 三者疊加＝本漏斗最強候選訊號。
• 覆蓋 + 市值級距：🔭 盲區零覆蓋 → 立 ID｜🟡 有 ID 無 DD → 形狀轉強＝加速訊號，建 DD｜✅ 有 DD → 懂它且剛突破＝最強訊號，回頭重看 DD（AR Live 只抓回檔、抓不到突破，這裡補上）。大型股照樣翻倍（NVDA $300B→$3T），小/中型看「沒被發現」、大型/mega 看週期/重定價。
• 週數：連續在榜週數（🆕＝本次新進榜）。形狀撐過 3 週比一日遊強得多；歷史同時累積成日後回測這條漏斗命中率的資料。
用途：告訴你注意力放哪——不是買入清單，估值與獲利品質刻意不在此把關（多倍股早期永遠看起來貴），那是管線②（DD 裁決層）的事。最終進場由個股 DD §14 裁決 + 時機決定。
