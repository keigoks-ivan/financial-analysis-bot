方法論

Final = 0.40·Quality + 0.30·Growth + 0.30·Entry
Quality (品質×護城河)：0.45·moat/10 + 0.25·pass/5 + 0.20·quality_score/10 + 0.10·max(execution,pricing)/10
Growth (成長持續性)：0.40·durability/10 + 0.30·clip(FY+1→FY+3 CAGR,0,40)/40 + 0.30·clip(ev5y,0,25)/25
Entry (進場品質)：0.30·(1−pct_5y/100) + 0.25·MA + 0.20·pullback(dist_52wH) + 0.15·trend(vs_200MA) + 0.10·val
Vetoes（任一觸發 → 不入榜）：ai_risk==🔴 · moat_grade∈(C,X) · trap==🔴
Tier 切分：Final ≥ 0.65 才入榜；Entry ≥ 0.65 → Tier A（Buy Zone Now），否則 Tier B（Wait）
投資原型（每檔標的旁的小徽章）
  • 成長型 Growth 子分 ≥ 0.55 — 三支柱都到位的標準複利成長股。報酬主要來自 EPS 持續成長 + 護城河擴張。讀法：Tier A 就是「現在能買的成長股」、Tier B 是「等切入點的成長股」。
  • 成熟型 Growth 子分 < 0.55 — 成長已放緩（年化個位數），但因護城河強 + 估值便宜仍入榜（KO/JNJ 路線）。報酬主要來自「估值修復 + 股利」而非「EPS 高成長」 — 不要套用成長股的「持有 5 年複利 20%+」期待；適合保守配置 / 防禦倉位。Tier A 的成熟型 Final 通常剛過 0.65 門檻、分數接近邊界，對輸入欄位變動較敏感。
讀法 caveats
  • 不是起漲點 screener：Entry 子分故意把「距 52w 高點 −10 ~ −15%」算作 sweet spot（不是 0 ~ −5%）。這與 Alpha Rank Angle 2「EV×起漲點」設計相反，刻意對應「不一定要起漲點」的目標。
  • 未做歷史勝率回測：「相對有勝率」claim 來自構造（cheap percentile + 趨勢未破 + 護城河）而非實證 win rate；如要 backtest 證據，需另外開回測 issue。
  • MA 標籤為近似：本頁 MA 燈號從 latest.json snapshot 推導（缺各 MA 個別斜率），與 DD §2 F 的完整六狀態機可能微差；以個股 DD 的 MA 段為準。
  • 缺資料中性處理：pct_5y / growth_durability / quality_score 缺值時用 fallback（如 moat_score 代理），不直接 0 分以免錯殺老 DD。
  • 跨期不可比：本頁分數僅在「同一份 latest.json snapshot 內」相對排序有意義；不同日期分數絕對值不直接比較。
