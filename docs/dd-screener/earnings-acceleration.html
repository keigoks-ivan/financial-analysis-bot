Earnings Acceleration — EPS 加速篩選
鎖定分析師上修中（EPS revision↑）+ 財報持續 beat的基本面加速標的。 三支柱複合分數：Revision Momentum（50%）/ Beat Track Record（30%）/ Quality Floor（20%）。 Tier A 是「雙引擎加速（上修 + beat 並行）」，Tier B 是「單引擎強但另一面尚未到位」。 與 Quality-Entry / Breakout 可交叉使用—— EA 篩基本面，timing screener 篩買點，兩個 Tier A 同時出現 = 高 conviction setup。

方法論

Final = 0.50·Revision + 0.30·Beat + 0.20·Floor
Revision Momentum（分析師上修）：0.30·fy0_rev_30d + 0.20·fy0_rev_90d + 0.30·fy1_rev_30d + 0.20·fy1_rev_90d — Λ 函數：−5%→0、0%→0.4、+2%→0.7、+5%→1.0（cap）。30d > 90d 權重（recency）。
Beat Track Record（財報 beat）：0.25·beat_count + 0.30·beat_accel + 0.25·latest_beat + 0.20·consistency — 過去 4 季 surprisePercent 驅動；accel = linreg slope。
Floor（品質地板）：0.40·moat/10 + 0.30·pass/5 + 0.15·quality/10 + 0.15·eps_rev_safety
Vetoes：ai_risk==🔴 · moat_grade∈(C,X) · trap==🔴 · moat_score<6 · EPS data缺失 · miss≥2/4q · FY0+FY1 30d均<−5%
Tier 切分：Final ≥ 0.75；A ≥ 0.6 AND B ≥ 0.55 → Tier A；否則 Tier B
💡 Factor Strategy — 使用前必讀
  • Timing-agnostic：EA 不告訴你「現在能不能買」，只告訴你「基本面在加速」。Tier A 標的可能在 ATH 也可能在 −40% 深跌位置。
  • Lagged signal：EPS 上修 + beat 都是 backward-looking。市場可能已 priced-in fundamentals strength。配合 valuation pillar（PEG / pct_5y）一起看。
  • Pair with timing screener：最佳用法：EA Tier A 篩出基本面加速，再 cross-reference 是不是 also 在 Quality-Entry / Breakout Tier A → confluence = 高 conviction setup。
  • 跟其他 screener 重疊正常：EA 跟 quality-entry / bottom-out / breakout 的 overlap 是 feature 不是 bug — EA 比 timing screener veto 更寬鬆，同檔可能同時上多個 screener。
