方法論

Final = 0.40·Position + 0.35·Setup + 0.25·Floor
Position（接近 ATH）：Λ(dist_52w_high_pct) — ATH 給 1.0，−2% 0.95，−5% 0.75，−7% 0.55（veto 邊界）；<−20% 給 0（用其他 screener）
Setup（動能確認）：0.40·rs_score_norm + 0.30·ma50_position_score + 0.20·ma250_slope_score + 0.10·drift_4w_score
Floor（品質地板）：0.40·moat/10 + 0.30·pass/5 + 0.15·quality/10 + 0.15·cagr_growth_score
Vetoes（任一觸發 → 不入榜）：ai_risk==🔴 · moat_grade∈(C,X) · trap==🔴 · moat_score<6 · dist_52w<−7% · rs<80 · MA250 slope<0 · vs MA50>+35% · eps_revision<−30%
Tier 切分：Final ≥ 0.65 才入榜；Setup ≥ 0.65 → Tier A（Active Breakout），否則 Tier B（Pre-Breakout Setup）
⚡ 動能區 — 使用前必讀
  • 不是 sure-thing pattern：Breakout 失敗率約 30-50%（Minervini 認知中屬合理）。Tier A 標的不保證繼續漲 — 請配 5-7% stop-loss 進場，讓市場告訴你對不對。
  • Overextended trap：Tier A 可能包含「爆衝最後一棒」標的。搭最新 IBD market state（見 markets.html）一起看；correction 中的 breakout 容易 fail，請在 confirmed uptrend 中操作。
  • 跟 Quality-Entry / Bottom-Out 互補不重疊：dist_52w > −7% → Breakout；−10% ~ −15% → Quality-Entry；< −25% → Bottom-Out。同檔股票不會同時在兩個 screener 的 Tier A。
