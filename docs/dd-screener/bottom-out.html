方法論

Final = 0.40·Drawdown + 0.35·Reversal + 0.25·Floor
Drawdown (深度)：Λ(dist_52w_high_pct) — 甜蜜點 −30% ~ −40%（1.0），兩側遞減；<−70% 給 0.1（可能是 broken thesis）
Reversal (觸底訊號)：0.35·drift_4w_score + 0.30·ma_slope_score + 0.20·price_vs_ma50_score + 0.15·rs_recovery_score
Floor (護城河地板)：0.40·moat/10 + 0.30·pass/5 + 0.15·quality/10 + 0.15·eps_rev_safety
Vetoes（任一觸發 → 不入榜）：ai_risk==🔴 · moat_grade∈(C,X) · trap==🔴 · dist_52w>−15% · eps_revision<−30% · moat_score<6
Tier 切分：Final ≥ 0.65 才入榜；Reversal ≥ 0.6 → Tier A（Reversal Confirmed），否則 Tier B（Deep Value Watch）
⚠ 高風險逆向區 — 使用前必讀
  • 不是價值陷阱 screener：通過 Floor pillar 不保證 thesis 還活著 — 務必先看個股 DD 確認護城河是否仍成立，再判斷是逆向機會還是接落刀。
  • 不是動量 screener：Tier A 代表「reversal 早期訊號」，不代表起漲點已確認或趨勢已反轉 — 可能還有再跌一段的空間。
  • 跟 Quality-Entry 互補不重複：dist_52w_high_pct > −15% 的標的被 veto 出 Bottom-Out，dist < −15% 的標的在 Quality-Entry 的 pullback 子分接近 0；兩個 screener 結果設計上不重疊。
