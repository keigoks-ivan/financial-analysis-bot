表格欄位說明

  • Ticker：股票代碼（正規格式）。
    來源：dd-meta JSON → scripts/dd_screener_dd_loader.py
    格式：美股 NVDA / 台股 2330.TW / 日股 6857.T / 歐股 RMS（帶交易所後綴的 EU 用內部 suffix map 轉換，DD 名稱保留無後綴形式）
  • Moat（護城河等級 + 分數 + 趨勢）：護城河評等，格式 等級 分數 趨勢，例如 S 9.8 ↑。
    來源：dd-meta JSON（moat_grade / moat_score）；趨勢箭頭以 DCA Phase A1 為 authoritative source（scripts/update_dd_index.py collect_dca_moat_trend_map()），無 DCA 時 fallback →（保守假設：穩定）
    等級：S（≥ 9.5）/ A（≥ 8）/ B（≥ 6）/ C（≥ 4）/ X（護城河存疑）
    趨勢：↑ 加深 / → 穩定 / ↓ 縮窄
  • FCF（FCF Margin）：TTM 自由現金流 / 營收，百分比。
    來源：優先 docs/qgm/latest.json 或 docs/qgm-tw/latest.json（hard_filter_details.fcf_margin.value）；不在 QGM 中的 ticker 走 scripts/dd_screener_quality.py → fundamentals_us.calc_fcf_margin()（yfinance）
    QGM MLB 門檻：≥ 10%（綠色通過 / 紅色不過 / 灰色無資料）
  • ROIC：NOPAT / Invested Capital，百分比。
    來源：同 FCF — QGM hard_filter_details.roic.value 優先；fallback fundamentals_us.calc_roic()（yfinance）
    QGM MLB 門檻：≥ 15%
  • EPS 2Y CAGR：前瞻 2 年 EPS 複合成長率，百分比。
    來源：QGM hard_filter_details.eps_cagr_2y_fwd.value 優先；fallback fundamentals_us._forward_2y_cagr()（yfinance earnings estimates）
    QGM MLB 門檻：≥ 15%
  • PEG：前瞻 PEG（Forward PE / EPS 2Y CAGR%）。
    來源：QGM hard_filter_details.peg.value 優先；fallback yfinance .info pegRatio；再 fallback 手算 forwardPE / eps_growth_pct（EPS CAGR > 300% 或 PEG > 10 時回傳 null，避免失真）
    QGM MLB 門檻：≤ 2.0x（上限；愈低愈好）
  • 2Y P/E：FY+2 Forward P/E（資訊欄，無通過門檻顏色）。
    來源：dd-meta JSON fpe_fy2 欄位，與 /research/ 表格 data-pe2y 同源
    規則：純數字顯示，不參與 pass/fail 計分，僅供估值參考
  • D/E（Debt/Equity）：Total Debt / Stockholders Equity。
    來源：QGM hard_filter_details.debt_to_equity.value 優先；QGM 有 D/E 缺口（如 ANET 零負債）時補抓 yfinance balance sheet；Negative equity（如 FICO / DELL / SBUX）回傳 null（負值無意義），顯示「—」
    QGM MLB 門檻：≤ 0.7x（上限）
  • 通過（Pass Count）：以上 5 條 QGM-MLB 門檻（FCF / ROIC / EPS / PEG / D/E）通過的數量。
    計算：client-side 即時計算，隨 Custom preset 的 slider 數值同步更新
    顏色：5/5 深綠 / 4/5 黃棕 / 3/5 淡黃 / < 3 紅
    Tab 分組：完全符合（5/5）/ 差一條（4/5）/ 差兩條（3/5）/ 全部
  • 5Y IRR：DCA §4 機率加權 5 年期預期價值 → 年化 IRR（%）。
    來源：scripts/update_dd_index.py collect_dca_ev_map() 解析最新 DCA HTML 的 §4 EV 數字，換算公式 IRR = (EV/100 + 1)^(1/5) - 1；無 DCA 或 §4 無法解析時顯示「—」
    顏色：≥ 12%/yr 深綠 / 8–12% 中綠 / 0–8% 琥珀 / < 0% 紅
  • MA（長期均線健康度）：股價相對 W52（1 年）與 W250（5 年）週均線的位置與斜率。
    來源：scripts/dd_screener_ma.py → yfinance 抓 5Y 週線資料（1wk interval）
    計算：w52 = 52 週 SMA / w250 = 250 週 SMA / slope_w250_pct = W250 現值 vs 4 週前 W250 的漲幅% / drift_4w_pct = 股價 4 週漲跌%
    燈號：🟢 healthy（W52 ↑ ∩ W250 ↑ ∩ slope > 0）/ 🟡 mixed（W52 ↑ ∩ W250 ↑，slope 平或負）/ 🔴 weak（跌破任一 MA）/ ⚪ n/a（歷史不足，如 ARM / NU / VIK / SNDK 上市 < 5 年，改依 W52 判斷並標示 W250 N/A）
  • 距 ATH（距 52 週高點）：當前股價離 52 週調整後收盤最高點的百分比距離。0% = 站在 52 週高；負值 = 低於高點。
    來源：美股 — docs/screener/latest.json dist_52w_high_pct（每日 06:00 TPE 更新，與 Flow ATH Hunter 同源）；非美股及不在 screener universe 的 DD ticker — build_dd_screener.py compute_yfinance_timing_fallback() 批量抓 300d 日線計算（timing_source: "yfinance_fallback"）；TW / JP / EU 若 fallback 也無法取得則顯示「—」
    顏色：🟢 sweet [-7%, 0%] 起漲點甜區 / 🟡 warm [-15%, -7%] base 築底 / ⚪ cool < -15% 深度回調 / 🟠 hot > +2% extended 已突破延伸
  • vs 50DMA（距 50 日均線）：當前股價相對 50 日 SMA 的百分比偏離。正值 = 股價在均線上方。
    來源：同「距 ATH」— screener ma50_pct 優先；fallback yfinance 批量抓 300d 日線，取末 50 根收盤均值計算
    顏色：🟢 sweet [0%, +5%] 剛收復均線、未過熱 / 🟡 warm [+5%, +15%] 延伸但可控 / 🟠 hot > +15% 過熱 / ⚪ cool < 0% 跌破 50 日均線（趨勢受壓）
  • RS（Relative Strength Score）：相對強度評分 0–100，相對 S&P500 全體成份股的百分位排名（多時間框架加權）。
    來源：美股在 screener universe — docs/screener/latest.json rs_score（EMA 平滑的 1w/4w/13w 加權複合分數，與 Flow ATH Hunter 同源）；不在 screener universe 的 DD ticker — fallback 批量抓 300d 日線，計算原始 1w/4w/13w 報酬率，對全 screener universe（~511 股）做百分位排名後加權（公式：r1w×0.2 + r4w×0.3 + r13w×0.5 + trend_bonus ±5）；TW / JP 本地上市股對同一美股 universe 排名（跨幣種，方法上有 FX 雜訊，以一致性優先）
    顏色：深綠 ≥ 80 強勢 / 中綠 60–79 OK / 灰 40–59 中性 / 淺灰 < 40 弱勢
    起漲點門檻：≥ 80
  • Links（DD / DCA / 🎯 Flow）：快速連結。
    DD：連到 docs/dd/DD_{TICKER}_{YYYYMMDD}.html（最新版，路徑來自 dd-meta）
    DCA：連到 docs/dca/DCA_{TICKER}_{YYYYMMDD}.html（最新版，路徑來自 filename scan）；無 DCA 時隱藏
    🎯 Flow：有 timing 資料（screener universe）的美股才顯示，連到 /flow/ath-hunter.html#{ticker}；TW / JP / EU 無此連結
  • 資料來源 badge（QGM-US / QGM-TW / YF / YF-EU）：顯示在 Ticker 旁，標示 5 條 QGM-MLB 質量欄位（FCF / ROIC / EPS / PEG / D/E）的實際資料來源。
    🔵 QGM-US：docs/qgm/latest.json（每週 GitHub Actions 跑 QGM 美股宇宙，~511 股）
    🟢 QGM-TW：docs/qgm-tw/latest.json（台股宇宙）
    ⚪ YF：yfinance 臨時抓取（不在 QGM 宇宙的美股、日股等）
    🟠 YF-EU：yfinance + EU 交易所後綴（如 RMS.PA / AENA.MC / BESI.AS）
