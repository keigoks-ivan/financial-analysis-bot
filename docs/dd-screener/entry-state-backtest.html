⚠️ 動手前必讀 — 這份回測有四個結構性侷限,數字不是無偏估計
  1. Survivorship bias(最重要):母體是今日通過品質閘的 96 檔 (Q ≥ 0.55 ∩ G ≥ 0.5 ∩ archetype=成長型),這些是「活下來且現在還夠強」的標的。用它們的歷史報酬回測, 天然偏高 — 當年同樣發出訊號但後來掉出母體 / 下市的標的完全沒被計入。 請當作「最好情況的上界」,不是期望值。
  2. 樣本稀疏:entry-state 頁自述 PULLBACK 命中率約 6%/檔、約 17 週才一次;訊號天生少, 任一格 n < 20 會標紅 ⚠,統計不顯著,勿過度解讀勝率。
  3. 無交易成本 / 滑價 / 稅:裸報酬,實盤會被點差、稅與沖銷成本侵蝕。
  4. 過去績效 ≠ 未來;且 RS 重建為近似:live 頁 RS 對 ~511 檔美股母體百分位排名, 歷史母體無法還原,本回測改在回測母體內部排名(同 blend 公式)。母體本身偏強 → RS≥80 是更嚴的門檻, BREAKOUT 數量可能低估 live 頁;故另列「移除 RS 閘」一行做敏感度對照。ma50_pct 亦以 10 週均線近似日線 50DMA。
