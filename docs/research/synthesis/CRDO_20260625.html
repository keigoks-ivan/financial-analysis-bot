Credo Technology (CRDO) — 趨勢 × 期望落差綜合研判

世界級 AI 連接收費站，但估值已 price 進牛市；−11% 回檔後逼近、未到再進場區。

§A 趨勢定位（選股漏斗上游）

就趨勢 mandate 而言，評估估值之前須先評估趨勢——標的所屬的長期趨勢、其 S 曲線位置、是否具收費站卡位、動能是否符合領導股特徵。此四項決定 B 層估值紀律的權重。

維度讀數判定
母趨勢AI 機架內銅纜連接（AEC，主，FY26 copper ~78%）＋ 光學第二曲線（DSP/SiPho/ZeroFlap，~11%）＋ retimer/IP（~10%）主趨勢＝AI 連接建置，真實且結構性
S 曲線位置兩段、不同期：① AEC（中段·高份額，1.6T/200G ASP uplift 推到 FY28、FY28 增速放緩）；② 光學第二曲線（極早期—FY26 ~$147M → FY27 guide >$600M，H2-loaded，ZeroFlap 新品未經 AEC 式驗證）AEC 中段 · 光學極早
收費站？AEC 晶片 ~73-88% 近壟斷（copper.json ⚑ 鎖喉，6-12 月換源成本）；但 retimer / 光學 DSP / CPO / transceiver 節點皆 contested（Marvell >60% 800G DSP）AEC 鎖喉，光學僅 challenger
動能 / 量化曾極度超買（+86% over W52、破布林上軌、財報後 melt-up +27%）；近 2 日 −11% 回落；Beta 3.2 高波動超買後回落中
A 層結論：CRDO 是純度最高的 AI 機架內銅連接 franchise，AEC 鎖喉真實、可追溯（copper.json 多源）。但 moat_trend → 持平（非擴大）：execution 擴大（73%→88%），然 88% 是天花板而非起跑點、光學為 challenger（非鎖喉）、Golden Cable 生態攻擊浮現、動能超買後回落。故趨勢鏡頭在此不強力 override 成「現在建倉」——持有核心但不追。

須明確區分的鏡頭衝突

  • 價值鏡頭：~44x FY27、逾共識均值目標 $256、5Y IRR ~7% → 偏貴、宜待回測；趨勢鏡頭：franchise 強、鎖喉真實，但 moat 持平 ＋ 動能回落。
  • 裁決不取兩者平均，而是依 mandate 取捨：即便趨勢 mandate，因 moat 持平 ＋ 動能超買回落 ＋ forward return 弱（re-rate drag），兩鏡頭同向偏謹慎——價值鏡頭在此升為主要進場閘（等回測），非僅風控。
  • 就趨勢股而言，核心紀律在於退出而非進場價，但本標的進場價亦關鍵（forward return 對進場價極敏感）——見 §退出觸發。

§1 市場 EPS 共識落點

探討「是否優於共識」前，須先界定共識。CRDO 的特殊之處在於賣方與站內 DD 普遍高於落後的市場共識，且四家賣方彼此在 FY28 分歧極大（光學 ramp 幅度）。下表為非 GAAP EPS，會計年度結束於 4 月。

關鍵點：四家賣方 ＋ DD 的 FY27 EPS（$5.65-6.27）皆高於落後的市場共識（~$5.08）+11~24%，FY28（$7.65-10.14）更高出 +15~53%——意味共識 EPS 偏低、上修順風。但現價 $269 仍逾共識均值目標 $256（+5%），只是低於近期上修的街頭高（Stifel $350、BofA $340、Roth $300）。賣方彼此 FY28 分歧（BofA $7.65 vs JPM/Barclays $10）＝光學 ramp 幅度，是本案最大的單一不確定。

修正方向：FY27 EPS 持續上修——Barclays CY27 EPS 一次由 $5.97 上修至 $9.80（+64%），主因 GM 棄守 63-65% 長期目標、改 guide ~68%；CRDO 季季 beat（FY26 每季 beat 營收 7-38%）。即近端上修動能強，但 FY28 光學幅度未定。

§2 EPS 可能優於共識嗎（上行向量）

有此可能，且證據強。理由有二：一為共識結構性偏低 ＋ 季季 beat 的紀錄；二為一系列具體上行向量未被基本情境充分計入。

  1. 共識 EPS 偏低 → 上修順風。賣方/DD FY27 $5.65-6.27 vs 共識 ~$5.08；FY28 $7.65-10.14 vs 共識 ~$6.63。CRDO 季季 beat（FY26 每季營收 beat 7-38%），共識長期落後實際。
  2. 毛利結構性 re-rate（最強多方點）。Q4 GM 68.3%（beat guide 64-66% ~300bps），管理層棄守 63-65% 長期目標→ 改 ~68%。每 +100bps GM @ $2.4B ≈ +$0.12 EPS——BofA/JPM 皆列為印中最重要揭露。
  3. 光學第二曲線＝大半未 model 的上行。FY27 光學 >$600M（3 線各 >$100M，ZeroFlap 三位數 ASP 最大）；FY28 光學 2-3x（$1.2-1.8B）幾乎不在任何 base PT（皆錨 CY27）。Jefferies 算 FY28 光學地板 $1.2B。
  4. DustPhotonics 垂直整合。SiPho PIC（雷射 8→2、−75%）、800G/1.6T/3.2T、FY27 accretive；自有 PIC 避開 merchant margin stack，改善 ZeroFlap 系統整合與遙測。
  5. 客戶基礎廣化。Q4 出現第 4 個 >10% 客戶（新）；管理層指 neocloud 未來 ~20%；AEC 已上 5-6 家 hyperscaler。
  6. AEC 本身仍貢獻 FY27 一半的絕對 $ 成長（管理層：「約一半光學、一半銅」）；且 1.6T／200G ASP uplift 推到 FY28（尚未計入近端）。

上述上行多數已反映於 ~44x FY27 的估值與逾均值目標的股價；兩面之收斂與方向見 §Δ 期望落差淨判讀。

§3 EPS 可能劣於共識嗎（下行向量）

同樣有可能——但對 CRDO，殺傷力主要在「倍數 ＋ 集中 ＋ 光學時點」而非「結構崩壞」。下列按可回復性與機率排序。

  1. 倍數 de-rate 是 forward return 的真殺手（非 EPS miss）。Base IRR 中，EPS +20%/yr 被 re-rate −14.5%/yr（49.5x→27x） 抵銷 → 淨 ~4.6%（@$302）／~7%（@$269）。即便 EPS 照漲，5Y 回報仍普通。
  2. Marvell Golden Cable ＝ 生態攻擊（critic 由 🟡 升 🔴）。非點產品，是「驗證參考設計 ＋ 軟體 ＋ 代工夥伴」的 commoditization 平台；Foxconn 2 個月即出貨。若 FY28 hyperscaler 二次源 20-30% → AEC 份額 88%→65-70%，PE 慢 de-rate 50x→30x。
  3. 客戶集中 air-pocket。top1 Amazon 34% / top2 MSFT 27% / top3 77%。FY24 殷鑑：單一客戶 pullback → 營收掉到 $193M。top-2 自建/二次源 AEC DSP 機率 ~12-18%/24mo。
  4. 光學 H2-FY27 back-loaded execution。~60-65% 光學在下半年；ZeroFlap 新品「非 AEC 式已證 ramp」、3nm 緊；一季 slip → FY27 miss，而 AEC（高基期）補不回。
  5. 雙觸發（Davis double-kill，critic Kill #1，25-30%/18mo）。Amazon/MSFT capex digestion × 光學 H2 miss → FY27 $2.4B→$1.6-1.8B、EPS $6.11→~$4、PE 50x→28-32x → $120-145（−52~−60%）。Beta 3.2 盤中可觸 $100-110。
  6. 估值／市場面。現價 $269 逾共識均值目標 $256；Max DD −55~−68%；9/9 F1Q27 法說 miss → 測 $200-220。
  7. CPO／光學 DSP lock-out（20-25%/3-5Y）。Marvell >60% 光學 DSP（1.6T Ara）、Broadcom switch-level CPO；OFC 2026 已展 6.4T CPO。若 CPO 早 1-2 年商化 → DustPhotonics $750M 變峰價 premature integration、減值風險。

§Δ 期望落差淨判讀（收斂 §2/§3）

§2 與 §3 不應各自停留於「可能更好、亦可能更差」。兩面收斂為一句：關鍵不在「CRDO 的 EPS 是否將不如預期」（共識偏低、季季 beat → 大機率優於共識），而在「樂觀情境已反映於現價、且 forward return 被倍數 de-rate 吃掉」。

時段 / 維度讀數方向
近端 FY27共識偏低（賣方高 +11~24%）、季季 beat、GM 68% step-up、光學 H2 接棒EPS 良好支撐、偏可超預期
遠端 FY28+光學 2-3x 未 model（上行）vs Golden Cable 份額侵蝕 ＋ CPO/光學-DSP lock-out（下行）；賣方 FY28 分歧大（$7.65 vs $10）雙向、幅度大、時點未定
對稱性 / 估值現價 $269 逾共識均值目標 $256、~44x FY27、5Y IRR ~7%、Max DD −55~−68%、Beta 3.2不對稱偏不利——upside 已 price，downside −52~−68%
操作含義：EPS 很可能優於共識（上修順風），但這已 price 進去；真正風險是倍數 de-rate（−14.5%/yr drag）＋ 客戶集中 ＋ 光學 H2 slip ＋ Golden Cable 份額侵蝕，而非 EPS miss。對此標的不預測 EPS level，而是追蹤 ① 光學 H2-FY27 出貨軌道（9/9 法說）② AEC 份額（Golden Cable 二次源比例）③ 上修是否續行 ④ 客戶集中度。回檔到 $240-265 才是 IRR ~9% 的進場點；$269 為上緣 cusp。

本頁最重要的一句：本標的 forward return 對進場價極敏感（同一條 EPS 路徑，$302 給 IRR 4%、$245 給 9%、$210 給 12-13%）。價值鏡頭在此不只是風控——它是主要的進場閘。即便趨勢 franchise 強，moat 持平 ＋ 動能回落使趨勢鏡頭不 override。

§4 估值再框架：44x 屬於哪一年的數字

「~44x」係以 FY27（NTM）EPS 計算。換到 out-year 倍數正常化得快，但 CRDO 的問題不在「貴」本身，而在 5Y re-rate 把回報吃掉。

賣方估值法：BofA 33x CY27、JPMorgan 26x CY27（明列「out-year 成長放緩」故低倍數）、Barclays 27x CY27、Jefferies 28x C28。PEG 0.77 看似便宜，但建立在 44% 超速成長；normalized 後 ~1.2。關鍵不是 EPS 漲不漲（會漲），是 5Y 倍數 de-rate −14.5%/yr 把 base IRR 壓到 ~7%（@$269）。

§7 管理層論述 vs 實際鎖定

以下取自 Q4 FY26 法說、BofA 科技會議 fireside 與 DustPhotonics M&A call 逐字稿 primary：

§退出觸發（趨勢部位核心紀律）
