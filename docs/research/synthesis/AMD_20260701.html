超微 (AMD) — 趨勢 × 期望落差綜合研判

EXPECTATIONS-GAP SYNTHESIS · 2026-07-01 · 最新收盤 $580.91（2026-06-30）
本文整合站內深度料（個股 DD、AI 加速器需求／AI 算力資本週期 ID、供應鏈節點圖、知識帳本）、外部賣方目標價與四份法說逐字稿（Q1 法說／Meta 特別電話會／BofA 會議／年度股東會），收斂為單一投資判斷。核心命題不是「AMD 好不好」，而是「市場對 AMD 未來 EPS 的預估是否過高或過低」——預期與現實的落差在哪一邊。結論：AI 運算趨勢仍在加速、AMD 是貨真價實的運算 franchise，但 6/30 一根 +7.7% 把股價推到 $580.91（≈歷史高、且高於分析師均值目標），新資金的紀律全押在「不追高、等回檔、守退出」。

§0 投資摘要

觀望 · 趨勢續抱 · 新資金緩進 · 退出紀律

franchise 是進場級、$581 的 ATH 價位不是——既有趨勢部位續抱守退出紀律，新資金等回檔至 $450-490 再分批。

分批配置：既有部位續抱（核心 4-6%）→ 新資金 starter ＠回檔 ~$490（NTM ~50x）→ 加碼 ＠$450-470（DD 進場區，FPE FY1→50x／FY2→30x）→ 尾部風險限額（Max DD −50~−65%）部位上限約 6%。

§A 趨勢定位（選股漏斗上游）

就趨勢 mandate 而言，評估估值之前須先評估趨勢——標的所屬的長期趨勢、其 S 曲線位置、是否具收費站卡位、動能是否符合領導股特徵。此四項決定 B 層估值紀律的權重。

維度讀數判定
母趨勢主趨勢＝AI 運算資本週期；次要選擇權＝代理式 AI（agentic AI）驅動的伺服器 CPU 升級與高核數 ASP；成熟基底＝嵌入式／FPGA、PC client。真實且多引擎

A 層結論：趨勢真實、階段中段、收費站坐落於 CPU（GPU 是高賠率選擇權而非已成形的鎖喉）、動能屬領導股但已極度延伸 → AMD 屬「核心持有」類型；但進場節奏受延伸動能與分位 ~90% 估值雙重約束。動能讀數取自 dd-screener MA／RS（QGM 未單列）。

須明確區分的鏡頭衝突

  • 價值鏡頭：現價 $580.91 高於分析師均值目標 $508、NTM ~66x、5Y 本益比分位 ~90%、2 年期合理價低於現價——昂貴；趨勢鏡頭：趨勢加速、收費站（CPU）擴張、EPS 仍在上修、供給已鎖到 2028——續抱。
  • 裁決不取兩者平均，而是依 mandate 取捨：趨勢 mandate → 趨勢鏡頭主導「續抱方向」，價值鏡頭降為「新資金的加碼地圖＋尾部限額」，並在價已顯著超前 EPS 時上升為「暫緩新資金」的風控訊號。
  • 對趨勢股，核心紀律在退出而非進場價：上修軌跡與份額一旦反轉才是賣點，單純「估值偏高」不是（見 §退出觸發）。

§1 市場 EPS 共識落點

AMD 的共識「窄在 EPS、寬在估值」：分析師對 EPS 路徑已大幅上修且彼此接近（FY26 約 $7.4、FY27 約 $13-14、FY28 約 $18-20，非 GAAP）；真正的分歧落在「該給幾倍本益比」與目標價（最低 $320 ↔ 最高 $700）。EPS 基準為非 GAAP 調整後 EPS。

關鍵點：現價 $580.91 已在共識均值目標 $508 之上（+14%），甚至高於多數近期上修目標（Wells Fargo $615 僅 +6%、UBS $560 為負）；而 EPS 共識本身已偏高——OpenAI／Meta 的需求上修多已納入。期望落差的關鍵因此不在「EPS 太低」，而在「股價已 price 到 EPS 兌現之前」。這是判讀對稱性的核心事實。

修正方向：近端 EPS 仍在上修（screener FY+2 EPS 月度修正 +1.6pp、FY3 修正 +2.1%；管理層稱 OpenAI／Meta 的 2027 預估「高於原計畫」）。但上修「速率」已開始落後股價——6/30 的跳升是賣方目標價追價、而非 EPS 再次跳升所致。

§2 EPS 可能優於共識嗎（上行向量）

會，但增量空間正在收斂——因為共識已把多數好消息上修進來。下列向量仍可能讓 EPS 再超預期：

  1. Beat-and-raise DNA：Q1 FY26 營收 $10.3B（+38% YoY）超財測高端、EPS $1.37（+43%）、毛利 55%（+170bps）；伺服器 CPU TAM 在半年內自 $60B（11 月分析師日）上修至 >$120B（5 月法說）——連續上修的執行紀錄。
  2. OpenAI／Meta 需求高於原計畫：管理層（BofA 6/2）明言兩大錨定客戶的 2027 預估「above our original plan」、且「持續增加供給」；這是落在共識之上的需求訊號。
  3. EPYC 份額 ＋ ASP 雙引擎：Q2 伺服器 CPU 指引 +70% YoY、價值份額 46%、其中 2/3 來自單位成長；代理式 AI 推升高核數零件，CFO 預期「CPU 單價持續上行」——量價齊揚且毛利友善。
  4. MI450／Helios 量產斜率：已 sample、H2-26 量產（Q3 起、Q4「顯著跳升」、續入 Q1-27），客戶已在自有資料中心跑滿 Helios 機架的正式工作負載——管理層稱屆時是「數十億美元規模」而非「幾套系統」。
  5. 尚未納入模型的選擇權：>$20 EPS 的策略目標「並未把 Meta deal 計入」（Su 明示），即為上行；其他「gigawatt 級客戶」管理層稱「正在進行」但未具名；MI500 在開發中。
  6. 管理層保守留底：DC AI 營收長期 CAGR >80%、TAM >$1T「more confident… exceed」；供給已向台積電鎖定（產能對話延伸至 2028）——供給端不再是 2026-27 的硬限制。

上述上行多數已反映於 6/30 的 +7.7% 與一連串賣方目標價上修——它們是否仍構成「優於共識」的增量，兩面之收斂與方向見 §Δ 期望落差淨判讀。

§3 EPS 可能劣於共識嗎（下行向量）

多數下行屬「倍數／時點」可回復路徑，但其中兩條（GPU 護城河深度、ASIC 內製）若惡化會是結構性。按可回復性排序：

  1. 倍數 de-rate（最可能的近端路徑）：NTM ~66x／FY26 79x、5Y 分位 ~90%、現價高於均值目標；6/30 是目標價驅動的情緒跳升、動能極度延伸。情緒退潮時 EPS 不變、股價亦可顯著回落——這是「期望落差」最直接的兌現方式。
  2. MI450「複雜 ramp」執行 ＋ HBM4／CoWoS 供給瓶頸：Su 自承這是「complex ramp」；HBM4 良率與 CoWoS 配置是上游單點，AMD 為下游客戶。ramp 落後即 Instinct 連兩季不達財測。
  3. 毛利稀釋：CFO 明示 MI450 於 Q3→Q4 量產時毛利「低於公司均值」；Instinct 毛利「先衝營收、細節未給」。GPU 組合上升初期會壓毛利率。
  4. GPU 護城河淺、對手強：ROCm 遠落後 CUDA、對手次代產品約 2.5x 效能差；GPU 段護城河本身「→ 持平」、且高度集中於 OpenAI／Meta。對手以 Rubin／CUDA／NVLink 的生態攻擊，DD 評為 24 個月 ~45% 機率。
  5. Hyperscaler ASIC 內製（結構位移）：客製 ASIC 合計約 10-15% 份額、~44% CAGR；站內 ID 將 AMD 定位為訓練池「#2 挑戰者，資本消化來臨時通常最先被砍」（conviction 🟡 swing）。
  6. capex 消化風險：資本週期處 Phase II（建置期）、正觀察轉入 Phase III（消化期）；變現缺口（AI 應用營收 $55-70B vs 上游晶片營收 $450-500B＝7-9x）使 ID 的 NC#2 判為 AT_RISK。Big4 capex YoY 首度跌破 +20% 即為拐點。
  7. 認股權證稀釋：Meta deal 附最多 1.6 億股 perf-based 認股權證、最後 tranche 於股價 $600 vest；分析師（Arya）直指其 ~$30B 價值約等於與 $30-35B 淨利「對換」，管理層未正面反駁該算式。
  8. 市場面／不可回復尾部：共識極度擁擠（51 人幾無賣出）、價在 ATH——一致預期本身降低正向意外空間。中國出口管制懸而未量化（Q1 Instinct QoQ 因「中國轉換」下滑，公司稱「不重大」但未給數字）。

§Δ 期望落差淨判讀（收斂 §2/§3）

收斂方向：近端 EPS 的期望落差「中性偏正、但增量有限」（共識已偏高、且已含上修）；估值與時點的期望落差「明確偏負」（股價已超前 EPS、2 年期 R:R 為負）。一句話——趨勢仍在、價已超買；在此價位，不對稱性偏負。

時段 / 維度讀數方向
近端 FY26/27共識已納入 OpenAI／Meta 上修；EPS 仍上修但「beat 既有高標」的空間收斂中性偏正
遠端 FY28+雙位數 GPU 份額、>$20 EPS 為願景非鎖定；ASIC 內製與 GPU 護城河深度為結構問號兩面（高賠率／高不確定）
對稱性 / 估值現價 +14% > 均值目標、NTM 66x、PEG 0.78（唯一便宜處在成長端）、2 年期合理價 $395-460 低於現價近端負向不對稱
操作含義：對 AI 運算趨勢領導股，重點不在預測 EPS 的絕對 level，而在追蹤上修「軌跡（derivative）」與份額消長。續抱者盯的是上修是否續、EPYC／AI GPU 份額是否擴；「兩面皆可能」於運作層以 §退出 的「上修反轉 ＋ 技術轉弱」處理，不靠預測點位。價已超前 EPS 的事實，只改變「新資金的進場節奏」，不改變「既有部位的續抱方向」。

本頁最重要的一句：在「找大趨勢」mandate 下，價值鏡頭不主導進出方向，而是（一）新資金的加碼節奏地圖、（二）尾部風險限額；唯有當價格如現在這般顯著超前 EPS 時，價值鏡頭才升格為「暫緩新資金」的風控訊號。

§4 估值再框架

現價 $580.91 的倍數是在 price 哪一年的 EPS？答案是「比 FY28 還遠」：以 FY26 計 79x、FY27 計 44x、FY28 計 32x。若 56.7% 的 EPS CAGR 全數兌現，倍數約需兩年才正常化到 40x 級；NTM ~66x 等於把多數兌現預先 price 進來。

§5 產業變動：淨判讀與查證路徑

競爭格局與結構位移的深度分析住在 DD §7／§9、ID 報告、供應鏈節點圖——本頁不複製，只給淨判讀 ＋ 下鑽連結 ＋ 可自行追蹤的指標，讓每個論點都能獨立查證、回溯到證據來源。

§7 管理層論述 vs 實際鎖定

以下取自 Q1 FY2026 法說（2026-05-05）、Meta 合作特別電話會（2026-02-24）、BofA 全球科技會議（2026-06-02）逐字稿 primary：

  • Meta 6GW／5 年是「框架」，僅首 1GW「committed」、H2-26 出貨：分析師（Lipacis）直接追問是否有 take-or-pay、客戶能否 opt-out，Lisa Su 迴避未答、僅重申「近端能見度很好」——願景成分大於契約鎖定。
  • 最強供給端實證：客戶「已在自有資料中心跑滿 Helios 機架的正式工作負載」、MI450 已 sample、H2-26 量產——此為實證可見而非願景。
  • 自承逆風／對沖：MI450 於 Q3→Q4 毛利「低於公司均值」（CFO 明示）、Su 形容 Helios 是「complex ramp」；2H26 PC／gaming 因記憶體與零件成本走弱（gaming 下半年較上半年 −20%+）；OpEx 連續超出財測。
  • 定價／毛利：代理式 AI 推升 CPU ASP 是「框架」順風；Instinct 毛利「先衝營收、細節未給」，毛利的兌現是「未來式」。
  • 地緣／關稅：Q1 Instinct 環比因「中國轉換」下滑、公司稱「不重大」但未量化；DRAM 成本通膨擬與客戶「共同吸收」（多數 DRAM 不過 AMD 損益表）。
  • >$20 EPS 目標「未把 Meta deal 計入」（Su）：屬上行選擇權、非已鎖定——管理層自己把它定位為「upside to」而非「baked in」。

§退出觸發（趨勢部位核心紀律）

趨勢股的超額報酬來自「在共識之前進、在飽和之前出」。進場價錯一點不致命，抱過頭才致命。下列任一觸發＝減碼／出場（不是「估值偏高即賣出」）：

  • ① 上修反轉：FY27 EPS 共識連續兩個月下修，或 OpenAI／Meta 的 forecast 回落至原計畫以下。目前 全綠（仍在上修）。
  • ② 技術轉弱：跌破 200 日均線、RS 跌出領導區、或 4 週 drift 轉負且不再創高。目前 需盯（極度延伸、距 ATH −0.7%、4 週 +16%，回吐風險高）。
  • ③ 收費站侵蝕：EPYC 價值份額自 ~46% 回落，或 AI GPU 份額卡在個位數無法進入雙位數。目前 全綠（46% 創高）。
  • ④ ASIC 防線破口（Single Thing）：OpenAI／Meta／Oracle／微軟級客戶公開把規劃中的 AMD Instinct 工作負載轉向自研 ASIC 或回到對手 GPU。目前 全綠（機率 ~15-20%，未發生）。
  • ⑤ 週期拐點：Big4 capex YoY 首度跌破 +20% 且 RPO 成長反轉（資本週期 Phase II→III）。目前 全綠（2026 +77%）。
  • ⑥ ramp 失速：MI400／MI450 連續兩季 QoQ 低於財測中值，或 HBM4／CoWoS 瓶頸造成出貨延遲。目前 待驗（2026-Q4 起為關鍵窗口）。
操作含義：六訊號目前「四綠、一需盯（技術延伸）、一待驗（ramp）」。在這些訊號轉負之前，「估值偏高」本身不構成賣出——但它確實構成「新資金暫緩、不追 ATH」。對既有部位而言，追蹤上修軌跡與份額消長，遠比盯短期股價關鍵。

§8 投資結論與分批進場

多個獨立錨點收斂為同一結論：知識帳本（6/27 觀望）、最新 DD（觀望、進場區 $450-470）、估值再評價（現價使 base 5Y IRR 自 ~9.5% 壓到 ~7%／yr、2 年期 R:R 為負）、賣方均值目標 $508 已低於現價、決策時 critic 冷讀（見下）。兩鏡頭方向相反——趨勢續抱 vs 價值暫緩；依「找大趨勢」mandate 取捨＝趨勢主導「方向」、價值決定「節奏」。

加碼至滿額 / 翻保守：FY27 EPS 不下修 ＋ MI400 量產達標 ＋ 價回 NTM ~50x → 加碼至滿額；Single Thing 觸發或週期拐點 → 翻保守，見 §退出。

總結：核心問題是「AI 運算趨勢是否仍加速」（是）與「AMD 是否仍是收費站」（CPU 是真收費站、GPU 是高賠率選擇權）。估值偏高只決定配置節奏、不否定這個 franchise——但此價位（ATH、且高於分析師均值目標）使新資金的紀律全押在「不追、等回檔、守退出」三件事上。
