期望落差綜合研判

把站內散落的分析（ID 產業深度 / 供應鏈節點圖 / DD / 知識帳本）＋ 外部賣方報告（CLSA、Nomura、HSBC、UBS、Barclays…）＋ 分析師會議與法說逐字稿，串成單一投資判斷。核心命題只有一個——市場對未來 EPS 的預估是否過高或過低。投資的勝負取決於預期與現實的落差，而非公司品質本身。
骨架：A 趨勢定位（選股漏斗上游）＋ B 期望落差（§2 可能更好 / §3 可能更差 / §Δ 收斂）＋ 退出觸發。雙鏡頭依 mandate 取捨；§5 連出 DD/ID/供應鏈可自行查證。
