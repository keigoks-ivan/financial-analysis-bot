附錄 0｜MCL 技術深度與營收影響模型

Micro-Channel Lid（MCL）是 3653 thesis 的核心技術賭注，下文拆解其量產經濟學：

MCL 製造複雜度

  • 微米級通道製造：通道寬度 50-200μm，深度 100-500μm，需要 high-precision wet etching + laser welding（vs 傳統 lid 僅需 stamping + machining）
  • 製造成本：估算為 Blackwell lid 5-7x（製程步驟從 8-12 步增至 25-35 步，cycle time 拉長 3-4x）
  • ASP premium：5-10x Blackwell lid（vendor 議價力強 + 客戶 thermal performance 需求剛性）
  • 毛利率影響：MCL 毛利率預估 45-55%（vs 現 lid 40-45%）— 因 ASP 增幅大於成本增幅

MCL 滲透情境模型

FYMCL 滲透率（lid 業務內）MCL ASP × 滲透率對 lid 業務 ASP 加權影響
FY265%（樣品 + 早期客戶）0.05 × 7x = 0.35lid ASP YoY +25-30%
FY2720-30%（量產初期）0.25 × 7x = 1.75lid ASP YoY +50-60%
FY2840-50%（成熟期）0.45 × 7x = 3.15lid ASP YoY +30-40%
FY29-3060-70%0.65 × 7x = 4.55lid ASP YoY +15-20%

假設：MCL ASP 為 Blackwell lid 7x（取 5-10x 中間值）；FY30 後 MCL 滲透接近飽和，但 ASP 仍維持 premium。

對 EPS 模型的影響

若 MCL 滲透率落 base case（FY27 25% / FY28 45%），FY27-28 EPS 路徑：

  • FY27 EPS = FY26 63.38 × (1 + 0.5 lid ASP × 0.7 散熱業務權重 + 0.15 其他成長) = 63.38 × 1.85 = 117.25 ≈ yfinance 共識 117.85 ✅
  • FY28 EPS = FY27 117.85 × (1 + 0.35 MCL 成熟 + 0.05 其他) = 117.85 × 1.40 = 165；本份 DCA 用保守 150（學習曲線折扣）

Bear case MCL 不達預期

  • 場景 1（MCL 延遲 1 年）：FY27 滲透率 仍 5%（vs base 25%），FY27 EPS 降至 80-90（vs 共識 117.85，下修 30%）
  • 場景 2（MCL ASP 僅 3x）：滲透率達標但 ASP 縮水，FY27 EPS 90-100
  • 場景 3（MCL 被 system supplier 取代）：thesis 結構性破壞，FY27 EPS 60-70（與 FY25 36 比僅 +70-95%）
