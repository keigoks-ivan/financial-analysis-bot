② AI infra 三大結構性 driver

Driver 1：Inference workload 遷移至 neocloud

同 NBIS — 未來 3 年內 neocloud 占比預計從 ~15% 上升至 25-30%。CRWV 規模優勢使其在 Microsoft / OpenAI / Meta 重要合約中是首選 partner（vs 2-tier neocloud）。

Driver 2：Power 成為新 bottleneck

CRWV vs NBIS 差距：CRWV 無自建 power 戰略，依賴 grid + co-location → 對未來 power 緊缺風險敞口更大。Vs NBIS Pennsylvania 1.2GW + Bloom Energy 戰略，CRWV 是 power 戰略落後方。

所有 hyperscaler 都在加速 in-house ASIC — 對 CRWV 比對 NBIS 影響更直接，因為：(1) Microsoft 對 CRWV 集中度 67% vs NBIS 40%；(2) OpenAI 自建 ASIC（Broadcom）2027-28 量產目標；(3) Meta MTIA 進展。CRWV 2028 後 thesis 完全依賴「dual-track 但仍需 NVIDIA bridge」假設成立。

⑧ Power 戰略劣勢量化

CRWV 對 NBIS 在 power 戰略上的劣勢可量化為「未來 5 年 power cost premium」：

  • NBIS Pennsylvania 1.2GW 自建 = blended $0.04-0.05/kWh（fuel cell + grid mix）
  • CRWV co-location grid 依賴 = blended $0.06-0.08/kWh（peak time premium）
  • 差距 $0.02-0.03/kWh × 4GW × 8760h × 0.85 utilization = ~$600-900M/year power cost premium for CRWV
  • 對應 ~3-5% EBITDA margin 持續性 headwind

這個 power cost 劣勢在 2027-30 期間累積到 ~$3-5B（vs NBIS），是 NBIS / CRWV thesis 對比中容易被忽略的長期 alpha 差距。

Phase B｜矛盾辨識與強制裁決

矛盾 1：Backlog $99.4B「鎖定」vs Microsoft 67% 客戶集中度「極端」

陳述：Phase A1 提到 $99.4B backlog 為主要 execution moat 強化證據；Phase A2 / A3 又指出 Microsoft 67% 集中度 + 3 大客戶 dual-track 是 thesis 最大結構性風險。如何同時存在？

裁決：Backlog $99.4B 主要是 Microsoft + OpenAI + Meta 三大合約的名義合約規模（multi-year commitments），實際執行隨 capacity 部署而 ramp。Microsoft 67% 集中度反映了 CRWV 在 2017-2022 期間「Microsoft 內部 outsourcing partner」的歷史定位。Backlog 是 CRWV 對 NBIS（$44B）的 2.3x 優勢，但其中 ~75-80% 是 Microsoft + OpenAI + Meta 三家 dual-track 客戶 → 實質意義：backlog 鎖定提供合約期內 cash flow，但合約期外（2031+）續約權力極弱。Backlog 是「5 年 cash flow 確定性」而非「永久 customer」。

矛盾 2：adj EBITDA margin 56%「hyperscaler-grade」vs Debt $25-29B + D/E 4.5x「業界極端槓桿」

陳述：CRWV adj EBITDA margin 56% 已達 hyperscaler-grade unit economics（vs MSFT/META ~40%），但 D/E 4.5x 是業界極端 leverage。如何同時存在？

裁決：EBITDA margin 56% 反映「current operating asset utilization 的 unit economics」強。但 Debt $25-29B 是「為了支撐 $99.4B backlog 履約而前置融資 capex」的結果 — debt 結構是「contracted backlog → debt → capex → cluster → execution」這個閉環的中間環節。核心問題：debt 利息 ~$1.5-2B/年 在 EBITDA margin 56% 下尚可服務（interest coverage 3.5-4x），但若 EBITDA 因客戶縮減 / commoditization 下滑 30%+ → interest coverage 跌至 2.5x 接近 covenant 違約閾值。投資結論：CRWV 是「對 FY27-30 Microsoft / OpenAI / Meta 合約完整履約 + EBITDA margin 維持 50%+ 的雙重 bet」。任一條件失敗 → debt 雪球觸發 → 估值倍數 + 槓桿雙重衝擊。

結語

  • 不適合 income / value 投資人：無 dividend、FCF 大幅為負、GAAP EPS 為負
  • 不適合與 NBIS 同時持有：thesis 高度相關，風險過度集中
