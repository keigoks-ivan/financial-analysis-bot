為什麼是這 1-2 個，而不是其他？

市場常討論「Goldman 連 4 次提目標」是 driver — 但其實 Goldman PT 是 driver 1 + driver 2 兩條的結果 reflection，不是獨立 driver。「股息率 4-5%」是 carry effect 不到 1%/yr IRR 貢獻不該歸 driver。「Vietnam 廠 / 三廠地緣」對 fab-less IC design 不適用。

Driver 抽象層 guardrail 校驗

driver 1（AI server 需求 + BMC content rise）與 driver 2（AST2700 技術領先 + 生態壁壘）是兩條獨立因果鏈：driver 2 可以在 driver 1 成立的情況下單獨被推翻（如：CPU 廠整合 BMC 進 SoC → AST2700 領先失效，但 AI server demand 仍在）。✓ 通過校驗。

BMC 業務的隱藏複利機制：每代 BMC 產品（從 AST2400 → AST2500 → AST2600 → AST2700）的 ASP 通常 +30-50%，而非 server cycle 的 +5-10%。這是「mix shift」而非「pricing」— 客戶為了新 server 平台必須升級 BMC，OEM cycle 是 BMC ASP 上升的 enforcer。AST2700 12nm + AI server content rise 是兩個 mix shift 疊加，造就 FY25 → FY26 EPS +82% 的非典型加速。但這個加速會在 mass adoption 後 normalize：FY27 +53% → FY28E +33% → FY29-30 step-down 至 +15-20%。長期 EPS CAGR 中期 ~25% / 後期 ~15%，給 IRR 8-13%/yr 的範圍。

客戶結構的隱藏分散度：表面上 ASPEED 客戶 ~80% 是 Top 10 server OEM，但這些 OEM 自身高度分散：Dell（hyperscaler + enterprise mix）+ HPE（enterprise + government）+ Supermicro（hyperscaler + scientific computing）+ Quanta + Wistron（hyperscaler ODM）+ Inspur（China + global）+ 其他 30-50 家 smaller OEM。即使單一 hyperscaler dual-source（如 AWS 找 Axiado），影響可能侷限於 OEM 內部 ~10-15% wallet，而非全 ASPEED 業務。這是 ASPEED 與 Accton（2345.TW）的關鍵差異 — Accton 直接面對 AWS dual-track，ASPEED 經 server OEM 間接 buffered。

資本配置紀律罕見的紀錄：22 年無重大 M&A — 在 IC design 業是極少數的（vs MediaTek 多次併購 / Marvell 大型整合）。Capex / Rev 3% 維持極低（fab-less + 設計成本主導）；Dividend payout 76.4% 高但合理（Net Cash 7.4B 充裕 + 業務不需大量再投資）。管理層基本上把所有 FCF 還給股東，過去 5 年 dividend per share 從 ~30 → ~52（FY24）→ FY26 預計 80+ — 這個 dividend 軌跡本身就是價值傳遞的證明。
