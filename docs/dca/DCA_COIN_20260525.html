§9 附錄｜Coinbase 業務 deep dive（DCA 補強）

9.1 USDC + Circle 共生與 dual-track 風險

USDC（Coin + Circle 共同創立 2018，Centre Consortium）— Coinbase 抓 USDC 全部 reserve income 的 50% 分潤，這是 thesis driver 1 的核心經濟。Q1 2026 USDC on-platform AUM 達 $19B 歷史新高（vs FY24 ~$10B = +90% in 16 個月），且 Coinbase 是「最大的 USDC 流動性池」— 約 25% of all USDC circulating outstanding + 50% of all USDC reserve economics。

但結構性風險：(1) Circle 於 2025 上市（CRCL），具備獨立融資與 distribution 能力，未來可能想直接 distribute USDC 透過 Visa / Apple Pay / Stripe 等通路繞過 Coinbase；(2) 50/50 分潤合約結構性綁定至 2030 年代，但 renegotiation 可能在 2027-2028 觸發；(3) 若新 stablecoin（如銀行發 onchain dollar / PYUSD by PayPal / RLUSD by Ripple）顯著吃 USDC 市佔 → AUM growth 失速。

Coinbase 反擊策略：(a) x402 payment protocol 整合 Amazon Bedrock Agentcore（2026-05-07，與 Q1 earnings 同日公告） — 把 USDC 設為 AI agent commerce 結算層 default，建構 distribution moat；(b) Coinbase One 訂閱用戶 USDC rewards yield — 構建 retail distribution exclusivity；(c) Base L2 + USDC 整合 — 控制底層基礎設施。

9.2 Deribit 整合進度評估

Deribit 收購於 2025-08-14 完成，總對價 $4.3B（$721.5M 現金 + $3.6B Class A 股票）。Q4 2025 為第一全季在 Coinbase 旗下運營，combined derivatives volume（spot + futures + options）達 $1.25T QoQ — Deribit 本身為全球最大 crypto options 平台（2025/7 月 vol $185B + open interest $60B）。

整合進度評估：(a) ✅ Q1 26 derivatives volume $4.2B +169% YoY 證明 cross-sell traction；(b) ✅ Deribit Asia base（新加坡 + Bermuda）為 Coinbase 海外擴張新樞紐；(c) ⚠️ 文化整合風險（Deribit 原以加密 native 文化為主，Coinbase 較合規導向）；(d) ⚠️ $3.6B 股票對價對 share count 影響 ~4.5% 稀釋；(e) ⚠️ 2026 全年 amortization 影響待觀察（goodwill / intangibles 約 $3B）。

對 thesis 影響：Deribit 是 driver 2「執行護城河」的重要加分，使 Coinbase 從「現貨」轉「衍生品全棧」競爭定位。Hyperliquid 在 perpetual swaps 已超 Coinbase 但 options 段 Deribit 仍為全球領先。

9.3 Base L2 與 sequencer 經濟

Base mainnet launch 於 2023 Q3，基於 Optimism Stack OP Stack。Q4 25 gross sequencer revenue 約 $19M（10月 $10M 為高峰受 crypto vol drive），FY25 全年估 $50-70M。Base 占 2025 全 L2 collective revenue 62%，是 dominant L2。

戰略意義：(a) Coinbase 控 Base sequencer = 控底層 transaction ordering = 抓 MEV 經濟 + sequencer fee；(b) Base 是 USDC native chain（USDC 在 Base 上 transaction 零 gas fee 透過 Coinbase 補貼），形成 USDC ↔ Base 雙向飛輪；(c) Coinbase + Anthropic + Stripe 等 partner 在 Base 上建構 AI agent commerce 結算層；(d) 長期 risk：sequencer 去中心化是 OP Stack 路線圖明定方向，2027-2028 可能執行 — 屆時 sequencer 收入模式重組。

9.4 競爭壓力深掘 — HOOD / DEX / Binance

9.5 估值倍數深度討論

COIN 的「正確估值錨點」是長期辯論。傳統 forward PE × EPS 在 cycle 股 unreliable（FY26 共識 90d -70% 修正即為例證），更穩健的框架：

  • EV/EBITDA normalized cycle：取 5-7 年 EBITDA 均（穿越 cycle 平均），對應 EV/EBITDA 20-25x 為合理。Coinbase 5Y average EBITDA ~$1.5B（FY21-25），對應合理 EV ~$30-37B vs 當前 EV $46B — 略貴 25-50%
  • EV/AUC (assets under custody)：對 Custody / Prime 業務適用。Coinbase AUC 約 $200-300B（業界估）對應 EV/AUC ~15-23bps，與 SCHW (10-15bps) 略貴
  • P/USDC reserve income capacity：USDC 50% 分潤 × $19B AUM × 5% rate = $475M/年穩定收入，給予 30x = $14B 隱含估值（佔總市值 30%），合理
  • Transaction revenue + adjusted operating margin：FY25 normalized transaction adj operating margin 約 50%，年 transaction revenue ~$3.5B × 50% × 25x = $43B 對應現市值 — 接近合理

四維加權結論：當前估值「略貴 10-25%」for crypto cycle base case，但「合理」for stablecoin 第二曲線兌現 case。這支撐 §4 base IRR ~9.5%/yr 與 §7 觀望（等更便宜進場區間 $170-200）的判斷。

9.8 為什麼 DCA 落 「觀望」而非 DD signal B 對應的 「進場」

DD §2 H 給 final_signal = B（衛星候選，QC-31 規則 thesis 完整時機不到）。為什麼 DCA 升一級到「觀望」（等於不馬上進場）？三個 PM 視角理由：

  1. Path-dependent Max DD 🔴 警示：§6c 估計 Max DD 範圍 -40~-60%（路徑高度痛苦）。即使 5Y base IRR 9.5%/yr 看似可接受，中途出場風險高 — 多數 PM 在 -50% drawdown 後會被被動洗出（即使 thesis 仍對）。為了避免被洗出，需要 (a) 嚴格部位控制 ≤ 8% portfolio + (b) 明確進場條件減少 unforced error + (c) 在等待期累積信心。「觀望 → 條件式試倉」是降低 path risk 的具體機制。
  2. Single Thing binary 接近：GENIUS Act Treasury Final Rule 預定 2026 Q3-Q4 公告（4-7 個月內），是 25-35% 機率的 thesis-killer event。在 binary event 之前進場 = 賭硬幣 + 大部位風險。理性 PM 策略是：等 Final Rule 公布後再決定 — 廣義即清倉、狹義即立即加碼（用更高 conviction 進場）。觀望期僅 4-7 個月，opportunity cost 有限。
  3. MA + 動能多面 confirmation 不足：Pure MA 🟠（破 W104，僅 W250 仍站）+ 2026-05-22 shooting star 衝高回落 -5.4% + 4 週均價漂移 -4.25% — 三個 timing 訊號都偏弱。儘管未到「迴避」級別，但歷史頻率上，在這類「破前高 -25% + 動能轉弱」配置下，6-12 個月內進一步下跌 10-20% 機率 ~55-60%（mean reversion + cycle 下半段典型行為）。等 MA 回穩可大幅改善進場價位（從 $185 → $170-180 區間，下行空間少 5-10%）。

觀望期動態管理：每月檢查三條件（MA 站回 W104 進度 / Q2 法說 take rate / USDC AUM YoY），任一條件實質性違背預期則重評觀望 vs 迴避邊界。觀望期最長 6 個月（至 2026-11-25），若仍未滿足三條件且 Q2 法說無正向 surprise，則升 watchlist 並 deactivate 此 DCA（產出更新版 DCA）。
