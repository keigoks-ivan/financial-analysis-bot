§2　One-Sentence Thesis（林區式檢驗）

「Cardinal Health = 美國藥品分銷 Big-3 #2 (24% 市佔)，業務從 commodity wholesale (Pharma 段 OI margin 1.4%) 向高毛利 Specialty/MSO 平台 (GI Alliance + Urology Alliance) 升級的 mid-cycle compounder — 估值 🟡 PEG 1.43 合理但需等動能 reset 後進場（W52 拉回 $185 或 BB lower $187）」

林區檢驗（一句話能說清楚嗎？）

  • ✅ 業務懂：藥品分銷 = 把 Pfizer / Merck / Lilly 等藥廠的藥品「送到」全美 80,000+ 零售藥房 + 6,000+ 醫院的物流公司；類似醫療版的 UPS
  • 🟡 為何便宜：估值並非極端便宜（PEG 1.43 vs MCK 1.18），fair value 而已；GAAP TPE 30.69 高是因 FY22-23 opioid GAAP loss 拉低；Fwd PE 16.75 在 5Y 60% 分位

A2.4 產業 thesis 獨立結論

產業 thesis：🟡 中性。藥品分銷是結構性寡占（Big-3 95%），無新進入威脅但內部 share 流動性高（OptumRx 案例證實）。Specialty / MSO 是未來 5-10 年的 differentiation 戰場，三家都在投入。CAH 處於 reorganize 階段（OptumRx 流失後），但 mgmt 在 specialty 轉型上動作積極（FY24-25 合計 ~$5.4B M&A），執行力獲市場認可。風險：① 大客戶議價結構不變 → pricing power 持續弱；② Amazon Pharmacy 5-8% 市佔（中期）會在零售端蠶食所有 Big-3；③ MSO 整合風險（physician 留存）。

矛盾 #1：成長率 vs PE 倍數錯位

3 年 Non-GAAP EPS CAGR 11.7%（mgmt target 12-14% mid）vs Fwd PE 16.75x → 隱含市場給 1.43 PEG，但同業 MCK 14-15% CAGR vs 17.16x PEG 1.18 — 暗示市場已認為 MCK 比 CAH 更具成長性（規模優勢 + share gain 來自 CAH）。
裁決：CAH 的 PEG 1.43 是「fair value with execution risk premium」 — 不是被低估，是被「合理定價且帶 risk discount」。買 CAH 需要 conviction Specialty 平台能拉回 share + margin。

矛盾 #2：FY25 FCF 大跌 vs FY26 guide 大幅恢復

FY25 FCF $1.85B（vs FY24 $3.25B，-43% YoY）— 表面 alarming。但 FY26 mgmt guide $3.3-3.7B（恢復至 FY24 水準）。
裁決：FY25 一次性壓縮（GI Alliance $2.8B + ADSG $1.1B + opioid pay-down ~$300M）— 不是結構性惡化。FY26 Q3 數據驗證 guide 可達成（FCF 已 normalize）。買方需驗證 FY27 是否回到 $4B+ run-rate（Specialty mix 升級 → cash conversion 改善）。

矛盾 #3：Big-3 中 CAH 規模最小（24% vs MCK 41%）vs ROIC 最高（16.8% vs MCK 23% 卻有負股權）

CAH ROIC 16.8% 看似低於 MCK 23%，但 MCK ROIC 高度受負股權扭曲（回購過度）。如比較 ROIIC（incremental invested capital），CAH 17%+ 與 MCK 接近。
裁決：CAH 規模劣勢被資本效率部分補償。但長期 share gain 需要規模優勢 — CAH 的 path 是透過 specialty 跳出 commodity 競爭，而不是 fight MCK on basic distribution。

核心矛盾總結

三個矛盾都指向同一個結論：CAH 是「合理定價的 mid-cycle compounder」，不是「便宜的轉機股」也不是「成長率被壓低的 hidden gem」。要買 CAH，需要相信：① Specialty / MSO 平台能拉回 share + margin（execution risk）；② mgmt 持續 capital allocation discipline（continued）；③ 大客戶結構不變（CVS / Walmart 不再流失）。

§4　機率框架（凱利公式）

4.1 凱利公式

輸入：

  • p（贏的機率）= 75%（Base + Bull 機率合計）
  • q（輸的機率）= 25%（Bear 機率）
  • b（賠率）= 平均贏的幅度 / 平均輸的幅度 = +58.7% / -23% = 2.55
  • 凱利百分比 = (p × b − q) / b = (0.75 × 2.55 − 0.25) / 2.55 = (1.91 − 0.25) / 2.55 = 65.1%

1/4 凱利（保守取）：65.1% / 4 = ~16%

解讀：機率框架建議目標倉位上限 ~16%（單一標的）。但 CAH 為 B 級衛星，整體組合中應 cap at 5-7%（衛星 satellite 上限）。最終建議倉位由 portfolio-manager skill 處理。

4.2 機率敏感度

情境p 變動新凱利%影響
