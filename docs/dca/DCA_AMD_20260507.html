§2 One-Sentence Thesis

AMD 是 AI 加速器市場唯一 viable 的 NVDA 第二供應商 — hyperscaler 從 single-source 走向 dual-source 的結構必經，吃 NVDA 不要的份額也夠飽。

Thesis 解讀

這不是「AMD 將打贏 NVDA」momentum framing — 那是不切實際。本 thesis 的核心是 「結構性次優選 + 三引擎現金流多元」雙腳支撐：AMD 不需要打贏 NVDA，只需要當「第二供應商」就贏 — hyperscaler $725B capex 中 5-15% 流向 AMD 即足以撐起當前市值。風險不在「能否打敗 NVDA」（不需要），而在「ROCm 是否成熟到讓客戶真的下單」 — OpenAI 6GW $90B 多年合約 + Oracle 27,000+ node MI355X cluster + Meta 6GW custom MI450 已給答案。

「次優選」三個字精準：① Q1 26 已驗證需求結構化 — Rev $10.3B (+38% YoY) record + Data Center $5.8B (+57% YoY) + Q2 26 guide $11.2B raised（管理層 Q1 26 法說 2026-05-05）；② 三引擎結構（EPYC server CPU +70% YoY guide / Instinct GPU / Embedded 復甦）讓 thesis 不依賴單一押注 — driver 1 失效但 driver 2 成立 = AMD 仍從 EPYC 故事獲利 +30-40%；③ MI355X MLPerf 6.0（2026-04-01 發布）達 NVDA B200 的 single-digit % 之內，ROCm 7.0 推論性能 +3.5x — 軟體 gap 在收斂中。但4 週漂移 +45%、12 週 +71%、AH +15% 後 RSI 將過熱至 75-85， thesis 對但時機極壞 — 這是 Charlie Munger「Show me the price」的直接應用。
