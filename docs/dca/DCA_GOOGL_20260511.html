§3 Key Drivers｜命脈是什麼

#Key Driver為什麼是命脈對應的可觀測指標
1 Cloud + TPU 自研鏈的規模化變現（供給側壟斷） GOOGL 是 hyperscaler 中唯一同時擁有自研晶片（TPU 7/8）+ 自研模型（Gemini 1.2T）+ 自研雲（GCP）的垂直整合者。Cloud +63% YoY 是 AI infra 需求的最直接變現通道；TPU 對 NVDA GPU 提供 supply diversification 是結構性護城河。若 Cloud + TPU 失效，整個 AI 規模化 thesis 崩塌。 (a) Cloud Rev YoY（目標 ≥ +35% CAGR）；(b) Cloud OI margin（目標 ≥ 28%）；(c) RPO 環比成長；(d) TPU 外賣客戶數（除 Anthropic 外）
2 Search 廣告引擎在 AI 時代的韌性（需求側持續性） Search 仍貢獻 $200B+ 年收入、60%+ 毛利率，是 Cloud 投資的現金牛。若 AI 搜尋（Perplexity/ChatGPT）真實質替代 Search query，整套 Capex / FCF 邏輯崩塌。Q1 +19% 是強反駁，但需持續驗證。Driver 1 + 2 是獨立因果鏈：driver 2「Search 韌性」可在不假設 driver 1「Cloud + TPU 兌現」成立的情況下單獨被推翻（例：Search Rev YoY 急降到 +5%，即使 Cloud 仍 +50% 也是 thesis 半崩）。 (a) Search Rev YoY（目標 ≥ +12%）；(b) CPC YoY（不下行）；(c) AI Overviews query 占比與整體 query 量；(d) Apple-Gemini deal 對 default search 結構的影響

為什麼是這 2 個而不是其他：YouTube +11%、Waymo $355M ARR、Other Bets 都是 nice-to-have，但 GOOGL 市值 $4.8T 中 Cloud + Search 合計占 ~75% Rev、~95% OI。任何 thesis 失敗都會從這兩條中的一條開始，不會從 YouTube/Waymo 開始。

Driver 抽象層 guardrail 校驗

Driver 2 是否為 Driver 1 sub-component？不是。Driver 2「Search 韌性」是需求側持續性（廣告 query 量 + CPC），Driver 1「Cloud + TPU」是供給側壟斷（算力 + 自研鏈）。兩條獨立因果鏈：

  • Search Rev 急降至 +5%，但 Cloud 仍 +50%、TPU 訂單仍滿 → thesis 半崩（driver 2 失敗但 driver 1 仍立）
  • Cloud OI margin 跌至 20%，但 Search 仍穩在 +15% → thesis 半崩（driver 1 失敗但 driver 2 仍立）

✅ Driver 2 可在不假設 Driver 1 成立的情況下被獨立推翻 → 兩條 driver 通過獨立性測試。
