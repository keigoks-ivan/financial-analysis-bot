Bear Case 戰術預演（補強 §6c）

路徑 1：dividend cut + estimating derate（最高機率 Bear，~30%）

2026 Q3 或 Q4 公告 dividend cut 30-50%（從 $1.26 → $0.70-0.90）→ 殖利率從 7.64% 跌至 4-5% → estimating compression 同時發生 → Fwd PE 從 8x 跌至 5-6x → 股價跌至 $10-12（-30% to -40%）。同步 FY27 EPS 共識下修 -10%。

戰術反應：(1) 若已建倉 → 出 50%（dividend cut = thesis 核心破裂）；(2) 觀察 90 天若 BM ramp 仍兌現 → 重新評估剩 50%；(3) 若進一步 cut → 清倉

路徑 2：中國地產 default + 高品位溢價收斂（次高 Bear，~25%）

2027 中國地產持續 default + 鋼鐵需求單季 -10% → 62% 普氏跌至 $70/t + Carajas 65% Fe premium 從 $10/t 收斂至 $3-5/t → Vale 鐵礦 EBITDA -30% → FY27 EPS $1.50 → 股價 $12 -27%。

戰術反應：(1) 若已建倉 → monitor，dividend 仍 carry；(2) 若 BM ramp 兌現 → 反向加碼至 5%；(3) 若兩個 thesis 同時受影響 → 減半

路徑 3：Lula 政府 royalty + 新 dam 事件（低機率，~15%）

Lula 政府 2026 H2 通過 mining royalty 加碼 5pp + 2027 H1 巴西另一 dam 出現安全事件 → 觸發 $3B+ 賠償 + 強制 capex $2B → FY28 EPS -25% → Fwd PE 壓至 4-5x → 股價 $7-8 -55%。

戰術反應：(1) 立即清倉（風險質變不可逆）；(2) 即使 Carajas 高品位仍在，市場無法 price-in 巴西結構性政治法務風險

路徑 4：複合最壞（極低，~5%）

三條 Bear 同時 + Vale Base Metals 也出意外 → 股價跌至 $5-6 (-65%)。
