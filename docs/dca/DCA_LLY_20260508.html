Driver 抽象層 guardrail check ✅

Driver 1 是「需求側」（TAM × 滲透）；Driver 2 是「供給側」（產品/製造優勢）。兩條獨立因果鏈：即使 driver 2 失敗（pipeline stumble），driver 1 仍可成立（LLY 仍可吃 TAM 60% share）；反之亦然。通過正交測試。

為什麼選這 1+2 而不是其他三個常被市場討論的：

  1. 「Alzheimer Kisunla 起飛」—不選為 driver，因為 Kisunla 2026 年化銷售估 $1.5-2B（vs LLY total $82-85B = < 2.5%），即便 5y +500% 對 EPS impact 也僅 ~3%。它是 optionality value，不是命脈。
  2. 「中國/印度新興市場擴張」—不選為 driver，因為國際擴張 80%+ volume 增速已被 driver 1 涵蓋（TAM 全球視角）；單獨拆出地理 driver 反而會與 driver 1 重複計算。
  3. 「PBM 議價權翻轉」—不選為 driver，因為這是 hygiene factor 而非 alpha source；MFN/Medicare 議價已是行業共同壓力，LLY 的相對優勢來自製造成本領先，那已被 driver 2 涵蓋。

Moat mechanism｜為什麼 5y 不會被打破

製造規模護城河的「時間 + 資本」雙閘門：GLP-1 是 peptide / small-molecule 雙線製劑，產能爬升非單純資本問題 — 從綠地建廠（greenfield）到 commercial-scale API 量產，FDA cGMP 認證 + 製程驗證 + 規模化 fill-finish 平均需 5-7 年。LLY 2020 啟動 Indianapolis 擴建、2024 Lebanon LEAP District 動工、2027 才有第一條全規模 API line 上線；對手 NVO 即便今天宣布 $50B、2030 才能對齊產能。期間 LLY 累積的 unit economics 學習曲線（每 doubling 產量 GM 推升 ~150bps）會持續擴大成本領先 — 2026 LLY GM 82.6% vs NVO ~78% 的 ~5pp 差距，預估 2028 擴大至 7-8pp。

口服小分子 vs peptide pill 的結構性差異：Foundayo 是 small-molecule oral GLP-1（分子量 ~600 Da），製造途徑是 chemical synthesis；NVO Rybelsus / 預計上市的 oral semaglutide 後續版仍是 peptide-based pill（需 SNAC absorption enhancer + 嚴格食物水限制）。Cost of goods 估計：Foundayo ~$8/月、NVO peptide pill ~$35/月（4-5x 差距）— 這個差距在 MFN $350/月、Medicare $50/月的價格 ceiling 環境下，等同 LLY 有 contribution margin 而 NVO 已逼近 break-even。

Pipeline width 的 optionality 護城河：除了當前三軸（Mounjaro/Zepbound 注射 + Foundayo 口服 + retatrutide 三重激動劑），LLY 還有第四軸 — GLP-1 + amylin 組合 LY3841136（Phase 2）+ 第五軸 oral retatrutide 探索性。NVO 對應只有 amycretin（GLP-1+amylin dual agonist）+ CagriSema。「LLY 的 N+2 比 NVO 的 N+1 多一條」 = 任一條失敗仍有 fallback、賣方無法用單一臨床事件結構性翻空。

共識的「2031-32 patent cliff -50% revenue」為什麼錯

市場用「Humira / Lipitor 模板」推 tirzepatide composition-of-matter patent expiry（2036 美國 / 2031-32 部分歐洲）會造成 -35% 至 -50% revenue 跳水。但 GLP-1 不是 small-molecule statin / antibody，而是複雜 peptide + delivery device + 製造製程組合。LOE defense 至少有 5 重：

  1. 製造製程專利（method-of-manufacture, MoM）：tirzepatide 的固相合成 + 純化製程涵蓋 60+ 項衍生 patent，到期日 2034-2038（vs composition 2036），仿製商即便 LOE 後仍受 MoM patent 阻擋
  2. 製劑 / device patent：Mounjaro/Zepbound 自動注射筆（KwikPen 平台）+ 預充懸浮液 stability formulation patent 至 2037-2040；biosimilar 需重做臨床等效（不像 small-molecule 直接 ANDA）
  3. FDA biosimilar pathway 高門檻：peptide GLP-1 雖技術上是 chemical entity 但 FDA 2023 起傾向視為「complex peptide」，需 PK/PD 等效 + 臨床相符性試驗（vs small-mol 只需 BE study）— 仿製商上市時間至少延後 18-24 個月、cost ~$200M vs $5M
  4. Pipeline replacement：2032 LOE 時 retatrutide / Foundayo / oral retatrutide / 第四代 GLP-1+amylin 已掌握 70%+ obesity 市場 — tirzepatide LOE 等同「替換為自家更新藥」而非真正侵蝕；類似 NVO 用 semaglutide 替換 liraglutide LOE 的 playbook
  5. 全球新興市場滲透：2032 美國 LOE 時，全球 50%+ rev 已來自 OUS（中國 / 印度 / 拉美），Trump MFN 把美國價拉到全球線後，各區 LLY 仍享高 contribution margin — 局部 LOE 對 global P&L wash 效果

結論：Patent cliff 真實 revenue erosion 估 -15-20%（vs 共識 -35-50%），且時間點 2031-32 太遠 — 對 5y DCF 影響 < 8%、對 10y DCF 影響 < 15%。
