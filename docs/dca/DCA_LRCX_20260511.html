D1 是供給側壟斷強度（LRCX 的設備有多不可替代），D2 是需求側持續性（客戶為什麼要持續花錢）— 兩條獨立的因果鏈。D1 的失效路徑是「對手追上 + 客戶切換」（TEL/AMAT/中國本土等），D2 的失效路徑是「AI capex 退潮 + HBM 投資 plateau」。兩者可獨立被推翻（測試：若 D1 失效但 D2 成立 = LRCX 變成普通 cyclical；若 D2 失效但 D1 成立 = LRCX 護城河仍在但 TAM 萎縮）— 通過 driver 抽象層 guardrail 測試。
