第一性原理：為何緯創估值「看起來這麼便宜」

緯創 Fwd PE 9.1x（FY27）、PEG 0.27 是 AI 成長股裡極端低的倍數——這需要第一性原理解釋，而非當「免費便宜」。市場本質是把每一塊 EPS 按其現金品質打折：緯創每一塊 EPS 背後，①毛利率僅 5%（安全邊際極薄）；②未轉化為自由現金流（FCF 深負，股東拿不到現金）；③靠舉債支撐（D/E 1.95 放大下行）。市場因此給「低品質 EPS」折扣倍數，與高 ROIC、強 FCF、低槓桿的「複利機器」享有的溢價倍數恰好相反。

所以投資緯創的本質賭注不是「撿便宜」，而是「賭折扣會收斂」——賭 FCF 由深負轉正、GM 止穩、槓桿下降，使市場把它從「低品質薄毛利代工」重估為「現金流改善的 AI ODM」。這個收斂若發生（§5 primary 兌現），9x → 12–13x re-rate + EPS 複利可帶來可觀報酬（共識中位 +27%、Bull +150%）；若不發生（FCF 續負），便宜倍數會「持續便宜」甚至更便宜（value trap，Bear −33%）。緯創比鴻海更極端——鴻海有正 FCF 緩衝，緯創的便宜完全建立在「FCF 會轉正」這個未驗證的賭注上。這就是裁決「觀望」而非「進場」、且倉位嚴控 ≤3% 的根本：賭注的賠率（PEG 0.27）很好，但賭注的不確定性（FCF binary）也很高，理性做法是等證據 + 等回檔，用小倉參與。

競爭格局與差異化深度

緯創在 AI 伺服器 ODM 的卡位有兩個獨特支柱，使其不只是「縮小版鴻海」：支柱一 AMD GPU 模組/基板獨家——在 AI 算力由 Nvidia 一家獨大下，hyperscaler 積極導入 AMD Instinct（MI300/MI350/MI400）作為第二供應、改善議價。緯創作為 AMD GPU 模組/基板的關鍵製造夥伴，是「去 Nvidia 集中化」的直接受益者，鴻海（Nvidia 一邊倒）與廣達都未取得同等地位。支柱二 緯穎（6669，majority-owned）——hyperscaler 自研 ASIC（Meta/微軟/AWS）機櫃龍頭，Q1'26 +62%，卡位「客戶繞過 Nvidia GPU」的路線。這兩支柱讓緯創同時押注「AMD GPU」+「自研 ASIC」兩條 Nvidia 替代路線。

五年財務軌跡與 FCF 機制

緯創營收 FY22 984.6B → FY23 867.1B → FY24 1,049.3B → FY25 2,186.5B（+108%！）；EPS 3.84→3.98→5.99→8.40。爆發成長由 AI 伺服器驅動。但代價是現金：FY25 OCF −105.8B（負！）、FCF −146.9B；TTM 更深 −195B~−264B。機制是營收 +144% 造成應收 + 存貨暴增（AI 機櫃單價高、備貨需求大），吞噬營運現金；公司用舉債（淨債由 ~202B 升至近期 ~354B、D/E 1.95）+ 動用現金支應 capex（60B）+ 高股息（payout 84.5%）。判別分水嶺：FY26–27 營收成長正常化後 WC 是否釋放、FCF 是否轉正——這是整份分析的 single judgment（§5 Single Thing）。

A3 sourced data points（9）：Q1'26 Rev 846.3B（+144.3%）、EPS 3.06、GM 5.21%、FY25 FCF −146.9B、FY25 OCF −105.8B、FY26 EPS 13.86（n=13）、FY27 17.46（n=14）、股息 3.80 payout 84.5%、淨債 ~354B / D/E 1.95（gurufocus/yfinance）。

A3 獨立結論：爆發量驅動的成長機器，但 FCF 深負 + 高槓桿 + 高 payout 是核心品質張力；EPS 複利可期但無現金背書，FCF 轉正與否決定機會 vs 陷阱。
