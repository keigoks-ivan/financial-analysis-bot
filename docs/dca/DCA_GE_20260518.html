補充：投資紀律與部位管理

為什麼 DCA 結論是「觀望」而非「進場 0.5%」

從決策矩陣優先序看，GE 落在第 8 條 baseline：「無 Hard Veto + DD ≥ B + 估值 = 🟠 → 觀望（等估值）」。即使 Pure MA 🟢 最佳進場給予正面 timing 訊號，估值 🟠 的 Soft Veto 仍鎖定上限為觀望。這不是基本面否定，是時機紀律 — 為了避免在 PE 37x 進場後即使 thesis 兌現，IRR 仍被估值 re-rate 拖累至大盤 IRR 之下。

歷史教訓對照：2014-2018 Rolls-Royce 在 Fwd PE 25x 進場、A350 backlog 10 年的相似 thesis 下，5Y 報酬僅 +12% / IRR 2.3%/yr，落後 SPX 約 7%/yr。RR 並非 thesis 全錯，而是「Trent 1000 Pkg-C 可靠性 + 公司執行紊亂」兩個 -10%~-15% 的小事件複利在 PE 25x 起點上造成大幅虧損。GE 起點 37x 更高、執行紀律雖然強很多但無「容錯空間」— 任何單一事件（Iran/油價、P&W 反攻、shop visit growth 不及預期）都會被 valuation multiple compression 放大。

本案的關鍵風險溝通（給 PM/委員會）

本案最大且不易察覺的風險不是「thesis 錯」，而是「thesis 對但估值 derate 完成需要時間」。即使 LEAP MRO 浪潮如期啟動，市場可能因為 Iran 衝擊 + 短期 EPS guidance 不修而把 Fwd PE 從 37x 慢慢壓回 28x — 這需要 12-24 個月。期間股價可能下跌 15-25%（達 $210-240 區），即使 thesis 完全正確。

FY28E EPS $10.40 推導路徑

從 FY27E $9.20 推導 FY28E EPS：

  1. 業務段營收推導：FY27 CES 預估 +20%（vs FY26 +18%）→ FY27 CES Rev $44B；FY28 CES 預估 +18%（LEAP MRO 進入加速期但 base 變大）→ FY28 CES Rev $52B
  2. OI margin 進一步擴張：CES margin FY26 ~31% → FY27 ~32% → FY28 ~33%（服務 mix 進一步加深）
  3. CES OI：$52B × 33% = $17.2B
  4. DPT 推估：FY27 DPT Rev $11B × 14.5% OI = $1.6B；FY28 DPT Rev $12B × 15% OI = $1.8B
  5. 合併 OI：$17.2B + $1.8B - corporate expenses ~$2B = $17B
  6. 稅率 18-20%（航發業常態）+ 利息支出 ~$0.5B → 淨利 ~$13.4B
  7. FY28 share count：FY26 末 1.05B - FY27 buybacks ~30M - FY28 buybacks ~25M = ~1.0B 股
  8. FY28E EPS = $13.4B / 1.0B 股 = $13.4 GAAP 或 ~$10.40 adj（含 spin-off legacy items 一次性差異收斂）

從 FY27 $9.20 → FY28E $10.40 隱含 +13% YoY；從 FY26 $7.54 → FY28E $10.40 = 17.4% CAGR (2Y)；從 FY25 $6.37 → FY28E $10.40 = 17.8% CAGR (3Y)。這是 §13.2 PEG 計算的 EPS CAGR 數字來源。

  • FY29: +13% YoY → $11.75
