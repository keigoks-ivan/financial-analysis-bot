核心問題：當前是「週期峰值」還是「結構新常態」？

這是 CIEN 整份 thesis 的樞紐，也是 §1 給「🟡 觀察期」而非「🟢/🔴」的原因。三個證據指向「結構 + 週期並存」：

支持「結構新常態」（多頭）：(1) cloud/AI DCI 是過去沒有的需求引擎——CIEN 歷史靠 telco capex 週期，現在 cloud 已 46% 且 +70% YoY，這是質變不是量變；(2) BofA 估 AI 光連接 39% CAGR 至 2030，1.6T→3.2T 世代升級是多年路線圖；(3) backlog $7.7B（>1 年營收）提供能見度，管理層稱 2027 能見度強。

支持「週期峰值」（空頭）：(1) FY26 +40% YoY 部分來自 FY24 +0% 的低基數（光學放大成長率）+ FY25 NG EPS $2.64 的極低基數使 FY26 +137% EPS 成長失真；(2) 「供給約束持續」暗示部分營收是產能受限下的遞延 + 可能訂單 pull-in（搶產能下單）；(3) hyperscaler AI capex 本身是週期（GPU 建置潮有節奏），歷史上光通訊每輪 capex 潮後都有消化期；(4) FY28 consensus EPS $13.19（較 FY25 $2.64 為 5x）是把當前甜蜜點線性外推。

裁決含義：真相大概率是「結構（cloud 新引擎，地板墊高）+ 週期（AI capex 潮 + 低基數，當前在峰值附近）」並存。這意味著：CIEN 的「常態化 EPS」可能高於歷史（cloud 結構墊高），但「當前 EPS」可能含週期峰值溢價。對估值的 implication：86x fwd PE 把「峰值 EPS」再給「峰值倍數」= 雙頂定價，沒有為「週期均值回歸」留任何安全邊際。這正是 §4 Base/Bear 機率偏空、§7 裁決觀望、且要求「等回檔 + 控小倉位」的根本邏輯——不是不信 AI 光互連，是不在「EPS 峰值 × 倍數峰值」雙頂進場。逐季驗證點：cloud 增速是否守 +30%+（結構）vs 跌破（週期見頂）。

業務結構與資本配置敘述（深度）

CIEN 的營收引擎正從「telco 週期」轉向「cloud/AI DCI」（cloud 已 46% +70% YoY）——這是結構性轉變，使本輪上行有「非純週期」成分。財務體質改善：adj GM 44.9%（+390bps）、FCF $665M（13.9%）、GAAP 已盈利、淨債/EBITDA ~0.9x。但兩個結構弱點：(1) FCF 週期性——FY22 FCF 為負 -$259M（後疫情囤貨 → FY24 消化），證明光通訊 FCF lumpy，當前高 FCF 含週期成分；(2) pricing 弱——cloud 46% 集中於 hyperscaler 買方 + pluggable 商品化威脅。資本配置紀律良好（有機 R&D 保 coherent 領先 + Capex/Rev 僅 3% 外包製造 + 無亂併購）。A3 獨立結論：Q2 爆發 + 體質改善（adj GM/FCF/GAAP 盈利），但週期性 FCF + pricing 弱 + 估值泡沫（fwd PE 86x）是核心風險，無財務生存風險。

盈餘品質 + 營運資金 watch（週期硬體特有）

相對 CRWD（GAAP 虧損 + SBC 22%），CIEN 盈餘品質較實——GAAP 已盈利（FY25 NI $123M）、SBC 中等（~6-8%）、GAAP/NG 差距主要來自攤銷 + 稅而非純 SBC。但週期硬體的隱藏風險在營運資金：「供給約束持續」下 CIEN 為急單備貨（存貨 + 應收膨脹），需求持續時是良性（backlog 兌現），但若 AI 急單退潮會出現「存貨跌價 + 應收回收放緩 + FCF 反轉」三連擊——這正是 FY22 FCF -$259M 的機制。逐季 watch：存貨成長 vs 營收成長（>1.5x = 為峰值備貨警訊）、應收天數、FCF 轉換率。這是「🟡 觀察期」定性的財務面依據，也是 §6c Max DD 🔴 的現金面佐證——光通訊的下行不只是股價，是 FCF 本身會隨週期反轉。
