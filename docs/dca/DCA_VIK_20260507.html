市場目前定價在什麼假設上共識 PT $77.50 + Fwd PE 25x 隱含市場已 price in：(a) Boomer demographic supercycle 結構真實；(b) Yield +mid-SD 持續至 2027；(c) Celebrity 進場吃 5-10pp 市占（已 priced）。市場其實過度討論的是 Celebrity 威脅（共識 PT 反映 -5% 估值折扣），但市場 underestimate 的是 Viking 50%+ repeat rate 的客戶 LTV。
我認為的不同假設Viking 的真正 alpha 是 50%+ repeat rate + 「intelligent / quiet luxury」品牌差異化 — 這個品牌資產比共識認可的更耐衝擊。Celebrity 是 RCL 旗下 mass cruise 品牌，luxury cruisers 不會把 Celebrity 視為同等選項。Celebrity 進場最多吃 first-time 客（Premium 試水），但 repeat customer 流失率 < 5% — 賣方把 first-time 流失等於 repeat 流失是錯誤類比。

校驗：三題都寫得出 → thesis 有獨立 alpha 來源（市場低估 Viking 客群粘性 + 過度看重 Celebrity 威脅）。但 alpha 觸發點是 Q2-Q3 26 兌現後的 multiple re-rating — 需要 6-12 個月 visibility，§7 倉位上完全反映（衛星 + 條件式進場）。
