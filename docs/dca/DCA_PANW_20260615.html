情境依據深掘：三情境的關鍵差異在「FY31 EPS」與「終端 PE」兩個變數。Bull（30%）假設 CyberArk synergy 完全兌現 + organic NGS ARR 維持 +28% → FY31 EPS 衝 ~$9.5，且市場因「AI 資安不可或缺」維持 48x 溢價 → $456（+65%）。Base（40%）假設成長照計畫但溢價自然收斂至 42x → FY31 EPS $8.28 × 42x = $348（+26%）。Bear（30%）是「業務略 miss + 板塊 de-rate」雙殺：FY31 EPS 僅 ~$6（CyberArk 拖累 + MSFT 侵蝕）× 28x（板塊溢價蒸發）= $168（−39%）。Bear 設 −39% 而非更深（如 −57% max DD），因 §4 是 5Y 終點報酬，而 max DD 是中途最深點——5 年後即使 de-rate，EPS 仍成長托底，故終點跌幅小於中途。這正是 §6c 強調的：抱不住的人死在中途 −57%，不是終點 −39%。

機率校準（時間視角）：Bear 30% > 20% 下限（合理，因 $25B 整合 + 高估值雙重不確定）；Base 40% < 50%（有壓測失敗路徑）；Bull/Bear 散布（+65% vs −39% = 104pp）足夠寬，反映 5Y 不確定性。對照 Pattern match（2021 MSFT/ADBE 高位）的歷史 de-rate 經驗，當前估計合理。

追加壓力測試：若拉長至 10 年，PANW 的 Runway 足以支撐（NGS ARR $20B FY30 後仍有 identity/AI agent 新曲線），10Y IRR 可能回到 ~8-10%/yr（時間攤平 entry 溢價）；但機會成本——同期可比資安/優質軟體在合理估值的 IRR 更高 → 仍指向「等更好價格」。

10 年視角 + 機會成本

把視角拉到 10 年，結論會反轉嗎？部分會。PANW 的 Runway（🟢 寬）足以支撐 10 年複利——NGS ARR FY30 $20B 目標後仍有 identity（AI agent machine identity）、AI-SOC 等新曲線。10 年期間，entry 倍數的拖累會被時間攤平：若以 $277 進場、持有 10 年至 FY36（假設 EPS 達 ~$18、PE 收斂至 32x → ~$576），10Y IRR ≈ 7.6%/yr。但這仍只是「接近大盤」的報酬，且要求你扛過中途 −57% 的 drawdown（§6c）。

機會成本的硬比較：同期，若等 PANW 回落至 $185（Fwd PE 45x）再進場，同樣 thesis 的 5Y IRR 可從 4.7% 跳升至 ~13-15%/yr（entry 倍數低 → re-rate 從 −10.8% 逆風變中性甚至順風）。等價格的紀律，本身就創造 ~8-10pp/yr 的 IRR 差距——這是「現在不買」的量化理由。對比同類資安標的（CRWD 90x 更貴、ZS 35x 但成長/平台廣度遜於 PANW），邊際資金的最優解是「等 PANW de-rate」，而非現在追任一檔。

兩個 driver 的交會點正是 CyberArk：它同時是「平台再擴一塊」(D1) 和「AI agent identity 新需求軸」(D2) 的載體——這也是為何 CyberArk 整合成敗（§5 Single Thing）是這檔的單一最關鍵 binary event。

競爭格局與護城河四源深度

① 轉換成本（最強一源）：PANW 的核心護城河機制是「整合債反向操作」——客戶過去買 30 個點狀資安工具產生整合債（工具間不互通、SOC 分析師在多個 console 間切換），PANW 把它們整成單一資料層 + 單一 AI 大腦（Precision AI）。一旦客戶部署 2+ 平台（佔 NGS ARR 80%），要 rip-and-replace 任一平台都需重建 SOC 流程、重訓團隊、重簽多年合約——切換成本隨平台數呈非線性上升。這是為何 PANW 敢持續以 platformization deals（給予前期免費/折扣換長約綁定）換取終身價值，且 churn 維持低檔。

② 數據/規模飛輪：跨數萬企業客戶的安全遙測數據，餵養 Precision AI 與 Cortex XSIAM 的威脅偵測模型——資料越多、模型越準、SOC 自動化價值越高、越易賣下一個平台。這是新進者（即使資本充足）無法快速複製的壁壘：你買得到工程師，買不到跨客戶的歷史攻擊數據。

③ 無形資產：#1 NGFW 品牌（網路安全基礎設施的預設選項）、FedRAMP 等政府認證（多年期、競爭者須重走流程）、CyberArk 的 PAM 市場領導地位（identity 監管級護城河）。

資本配置紀律（B 級）：PANW 是「併購建構平台 + 有機交叉銷售」的混合飛輪，自 2018 完成 21 筆併購，track record 良好（無重大減損史，多數成功整入 Prisma/Cortex）。但 $25B CyberArk 是數量級放大的最大賭注，使 ROIC 短期崩至 ~5%（GuruFocus），ROIIC 須 FY28 驗證 > WACC。SBC 14% + M&A 股權發行使淨稀釋 >1.5%/yr。計分卡 B 級（歷史佳但 SBC 高 + 最大賭注未驗證）。對標 Cisco-Splunk $28B（市場對大型資安併購普遍懷疑），PANW 須證明能避免「大併購 = 稀釋 + 拖累」劇本——synergy 提前 3-6 個月是早期正面差異化。

5Y IRR 分量基底推導：Base EPS CAGR ~17%/yr（FY26 $3.77 → FY31 $8.28，含 CyberArk 復甦 flattering effect；有機內生天花板僅 12%，故此 17% 不可長期外推）；5Y 末 forward PE 估 ~42x（從當前 67x 部分收斂，反映成長放緩 + 溢價自然收斂）→ re-rate 貢獻 −10.8%/yr；股息 0% + 淨買回 ~0%（SBC 抵銷回購）。三分量 17 − 10.8 + 0 ≈ +4.5-4.7%/yr（與 §4 Base IRR 一致）。
