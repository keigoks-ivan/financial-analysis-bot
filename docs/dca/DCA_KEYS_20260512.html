補充｜Patrick Lynch 風格 thesis 解構（用一句話講清楚為什麼買）

Lynch 框架：是「what business」、「why moat」、「what makes money」

What business：KEYS 賣的是「不犯錯保險」— 通訊晶片設計、6G 基地台 prototype、AI Ethernet 1.6T transceiver、衛星 modem，這些東西出貨前都必須通過「驗證」這一關，KEYS 是這道驗證的 anchor 供應商之一（與 R&S 雙寡頭）。

Why moat：因為「驗證錯誤」的代價極高 — AI Ethernet 1.6T 通道 design bug 可能讓整個 NVL72 機櫃損失千萬美元；6G base station 設計錯誤可能撤回全球部署。客戶不會為了省 10% 儀器成本去買一個「probably works」的替代品 — 這是 mission-critical 的 sticky 採購。

What makes money：(1) 儀器本身 GM 65%；(2) 配套 PathWave 軟體 license 是 recurring；(3) 校驗服務 / 維修 / 升級是 annuity；(4) 6G 標準制定參與讓 KEYS 在新規格上市前 12-18 月就有「準產品」競爭優勢。

Lynch 的「能講清楚」測試：✅ 通過 — 我能用 60 秒讓朋友理解為什麼 KEYS 有護城河，比起需要 5 分鐘解釋 BCD 製程的 MPWR，KEYS 的 thesis 更接近 Lynch 偏好的「無聊產業 + 隱形龍頭」配方。
