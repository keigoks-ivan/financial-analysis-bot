1AI server 訂單轉化率 + ISG 利潤率（供給側 moat 持續性）Dell 的差異化來自「能把 Nvidia GPU 包成企業可用 AI 系統 + 全球服務」。如果 Nvidia 直賣 L10 + hyperscaler 自研 ASIC 把這個 layer 拆掉，Dell 就只剩 chassis assembly。這條 driver 決定 5Y EPS 路徑與 PE 維持① AI server 訂單 backlog 季度數字；② ISG 營業利潤率；③ Dell 在 Nvidia Vera Rubin 平台 design-win 內含的價值區佔比
2Enterprise on-prem AI 採用率（需求側延長成長曲線）Hyperscaler capex 2027 peak 後，Dell 必須靠 enterprise + sovereign AI 接力才能維持成長。這條 driver 來自完全獨立的因果鏈（需求側 = 客戶從 hyperscaler 擴散到 enterprise），不是 driver 1 的子集① Dell AI Factory 商業合約數（公司年度揭露）；② sovereign AI 標案中標數；③ enterprise IT capex YoY（Gartner 季度報告）

為什麼是這兩個：① 決定 Dell 在 AI infra value chain 中位置不被擠壓；② 決定 Dell 在 hyperscaler capex digestion 後是否有第二曲線。其他常被討論的 driver（PC refresh、storage growth）對 Dell 5Y EPS path 邊際貢獻 <10%，不是命脈。

Driver 抽象層校驗：Driver 2 不依賴 Driver 1 成立 — 即使 Nvidia 拿走 hyperscaler 直銷，enterprise 端的 AI 採用仍可獨立發生（多數 enterprise 不會直接從 Nvidia 採購）。✅ 通過 guardrail 4b。
