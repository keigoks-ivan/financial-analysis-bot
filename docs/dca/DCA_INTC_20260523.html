§2 One-Sentence Thesis

INTC 是美國 AI 半導體自主供應鏈核心 + USG (9.9%) + NVDA + SoftBank 三重戰略股東救援；但 26 天 +45% / 距 W250 +275% / FPE FY+1 110x — 估值已遠超基本面修復速度。
— 50 字以內，敘事性 thesis（含 thesis floor 與 valuation ceiling 兩面）

Thesis 結構拆解（內部備忘）

本 thesis 是「政策 + 動能 + 資本結構」三維雙刃：

  1. 政策 floor：USG 9.9% 持股（$20.47 成本）+ CHIPS Act $5.7B 加速 + NVDA $5B + SoftBank $2B private placements = $20B+ 戰略資本注入 1 年內。在美國 AI 戰略供應鏈中，INTC 是唯一「國內 leading-edge fab」候選 — 這是 thesis 的結構性下檔。
  2. 動能 driver：Q1 FY26 Non-GAAP EPS $0.29 vs $0.01 cons (2800% surprise) + Q2 guide $14.3B 中位 +6% raise + Lip-Bu Tan 上任後第一份 print 全方位驗證執行力拐點。RSI 14d 78.20 sell signal 顯示動能末段。
  3. 估值 ceiling：FPE FY+1 110x / FY+2 77.9x / PEG 2Y 1.80 接近紅線 / 5Y PEG 3.1 遠超紅線 / P/S 9.4x 5Y 100% 分位新高。所有估值維度全部過熱，溢價來源是「動能 + 政府背書 + NVDA 入股」narrative — narrative 已過度 priced in。

三維結論：thesis 不是「便宜陷阱」（不是 buy on dip 的 value trap），而是「動能 + 敘事 + 政府背書」過熱型錯配的反向變體。entry 點極為敏感 — 現價 $120 entry 機率加權 5Y IRR 為負，等回測 < $60 進場才有正預期報酬。
