§2 One-Sentence Thesis

ONTO 是 advanced packaging 量測的 niche specialist：spectral ellipsometry 3D 建模 + Dragonfly G5/Atlas G6 雙平台，是 KLA 不重疊的 HBM4／2.5D 邏輯 metrology POR 位置。
— 50 字以內，敘事性 thesis（非財務指標主體）

Thesis 解讀

這不是「KLAC 第二」的說法 — 那是 cyclical commodity framing。本 thesis 的核心是 niche specialist with cross-platform reference：ONTO 在 advanced packaging metrology（HBM4 + CoWoS + 2.5D logic）這個 KLA 主場以外的 process control 子戰場，靠 Dragonfly G5（2D inspection + 3D bump metrology）與 Atlas G6（OCD GAA / DRAM）雙平台，握有 logic + memory 雙線 reference 位置。

「niche specialist」三個字精準：① 不是 KLA inspection 主場（53% 寡占）正面對決；② Dragonfly G5 已被 leading 2.5D logic 客戶 qualified（管理層 Q1 26 法說，2026-05-05）；③ 一個 leading HBM 製造商簽下 $240M+ VPA through 2027 — VPA 體量在 small-cap 設備商裡屬於 anchor-tenant 級別。但 niche 也意味著規模劣勢：KLA R&D ~$1.5B / year vs ONTO ~$200M，moat 寬度比 KLA 窄 1.5 級。
