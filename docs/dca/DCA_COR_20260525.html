mgmt Q2 FY26 上修 FY26 Adj EPS guide +$0.20 mid（$17.78 vs $17.60）；同時 5 月 Wells Fargo $429→$331、Citi $405→$355、Baird $420→$339、Evercore $360→$300。

核心矛盾總結

§4　機率框架（凱利公式）

4.1 凱利公式

輸入：

  • p（贏的機率）= 75%（Base + Bull 機率合計）
  • q（輸的機率）= 25%（Bear 機率）
  • b（賠率）= 平均贏的幅度 / 平均輸的幅度 = +68% / -23% = 2.96
  • 凱利百分比 = (p × b − q) / b = (0.75 × 2.96 − 0.25) / 2.96 = (2.22 − 0.25) / 2.96 = 66.6%

1/4 凱利（保守取）：66.6% / 4 = ~16.6%

解讀：機率框架建議目標倉位上限 ~17%（單一標的）。但 COR 為 B 級衛星，整體組合中應 cap at 5-7%（衛星 satellite 上限）。最終建議倉位由 portfolio-manager skill 處理。
