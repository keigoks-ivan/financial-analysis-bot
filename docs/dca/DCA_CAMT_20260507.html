Thesis 解讀

這不是「HBM 鏟子要賺」的 momentum framing — 那是 trader's bet。本 thesis 的核心是 「故事完整 9/10 vs 估值 priced 完美」的雙重結構：CAMT 在 HBM4 三大廠（SK Hynix / Samsung / Micron）都是 3D metrology reference tool（管理層 Q4 25 法說 2026/2 親自陳述），Hawk 平台 9th-gen 白光三角技術相對 Onto 雷射技術在 bump 量測覆蓋更高 — moat 是真的。但 forward PE 53x（5Y 88 分位）+ 距 MA104w +101%（追高區）+ DD 訊號燈 X，意味現價買入 = 「對的公司 + 錯的價格」。

Camtek 與 Onto 的 head-to-head 是 niche specialist 內部競爭：CAMT 在 HBM 純度更高（HPC 45-50% 收入 vs ONTO 多元）、ONTO 在 logic + memory 雙線；現價是 trader's bet 而非 investor's bet — 在 Onto Dragonfly G5 取得第二家 Tier-1 reference 之前，CAMT 短期領先；但這個資訊已 priced 進現價。等回測才是 R:R 健康的進場視窗。
