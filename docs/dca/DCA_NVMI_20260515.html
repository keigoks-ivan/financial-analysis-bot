#Key Driver為什麼是命脈對應可觀測指標
1製程節點 metrology intensity 結構性提升（需求側）每代節點所需 metrology 步驟翻倍（5nm → 3nm → 2nm → 1.4nm → CFET）；這是物理定律，非景氣循環Nova Rev YoY vs WFE YoY 差距（FY25 +31% vs WFE +5% = 26pp）；先進封裝佔產品營收（FY25 20% → FY26E 25%）
