核心 Thesis
2026 年 AI 推論經濟學被三條結構性趨勢同時重塑：(1) 推論工作負載已占全球 AI compute 67%（vs 2023 的 33%、2025 的 ~50%）、推論電力消耗預估 20.9 GW (2025) → 93.3 GW (2030) at 35% CAGR（McKinsey），但市場估值與分析框架仍 anchor 在 training；(2) Token 價格 6 年通縮 600×（$60/M tokens GPT-3 Davinci 2020-06 → $0.10/M Gemini Flash-Lite 2026-05），Epoch AI 量化中位通縮 50×/yr、2024-01 之後加速至 200×/yr — 但同期 enterprise AI spend 反向 +3× 衝至 $37B（MSFT AI run-rate Q3 FY26），Jevons paradox 完全主宰；(3) Margin pool 正從 LLM API providers（OpenAI 2024-25 GM 40%→33%、projected $14B loss 2026、Sora 因 $15M/day compute vs $2.1M lifetime rev 而於 2026-03-24 關停）遷移至 vertical integration（MSFT Azure OpenAI、GOOG TPU+Gemini+GCP）+ ASIC fabless（AVGO Q1 FY26 AI rev $8.4B +106% YoY、$40B FY26E guidance）。本 DS 主張三條 non-consensus：inference CUDA moat 已開始裂（Wintel/ARM 類比），但 training CUDA moat 仍 binding 至 2030+、真正 binding 是電力（128-160 週變壓器、PJM 8 年 interconnection queue）而非晶片、Sora 失敗是 premium inference 廣泛 unit-economics 危機的先聲、不是孤例。

§1 產業歷史脈絡 — 13 年五次推論經濟學 inflection

AI 推論經濟學的歷史只有 13 年。在 2013 年之前，「推論」（inference）僅是 ML 課堂上 training 的副產物，沒有獨立經濟學。從 2013 年 Google 內部一個簡單估算開始，到 2026 年推論已占全球 AI compute 的 67%、token 單價 6 年通縮 600× — 這段歷史經歷五次明確 inflection，每一次都重洗了「推論的真正成本是什麼」與「誰賺到錢」的答案。

第一次 inflection — 2013：Google 內部估算觸發第一個推論專用矽。決定性瞬間發生在 2013 年 — Google 內部估算：「若每個 Android 用戶每天使用語音搜尋 3 分鐘，則 Google 全球機房 capacity 必須翻倍」才能跑得起這個推論工作負載。這是 AI 史上第一次把推論成本當成「存在威脅」處理。後果是 2015-04 起 TPU v1 內部部署、2016-05 Google I/O 對外揭露 — 這是第一款 inference-only ASIC（不支援 training），256×256 systolic array、92 TeraOps/sec at 28-40W、相對 contemporary CPU/GPU 推論加速 15-30× 且每瓦操作數 30-80×（Google TPU 論文 arxiv 1704.04760）。歷史功能：第一次把「推論不是 training 的副產物，而是獨立的工作負載品類」寫進矽。

第二次 inflection — 2022-11-30：ChatGPT 消費級規模衝擊。ChatGPT 上線 60 天達 100M 用戶，這個增速史無前例 — Netflix 用 10 年達到 200M、TikTok 用 9 個月達到 100M。SemiAnalysis 2023-02 估算 ChatGPT 啟動需 28,936 顆 A100 GPU、每天硬體運轉成本 $694,444、每查詢 $0.36（vs Google Search 每查詢 $0.01）。OpenAI 吸收這個虧損撐 3 個月，2023-03-01 釋出 GPT-3.5 Turbo API 定價 $2/M tokens — 同時揭露「過去 3 個月透過系統優化把 ChatGPT 服務成本降低 90%」。歷史功能：第一次證明 inference 在消費網際網路規模下可實作、且 unit economics 可以 3 個月內 -90%。SemiAnalysis 同期估算：若 OpenAI 想取代 Google Search 規模需 ~4.1M A100 GPU — 這個數字後來成為 hyperscaler capex 翻倍的起點。

第三次 inflection — 2023-03：GPT-4 與 API 時代開啟 token 通縮競賽。2023-03-14 GPT-4 launch 定價 $30/M input + $60/M output（blended ~$36/M）。2023-11-06 GPT-4 Turbo 降至 $10/$30（input -67%）。2024-05-13 GPT-4o 進一步降至 $2.50/$10 — Andrew Ng 公開估算「過去 17 個月年化 -79% 通縮率」。同期 Anthropic Claude 3.5 Sonnet 上線 $3/$15、Google Gemini 1.5 Flash 一次降價 input -78%/output -71%（2024-08）。歷史功能：確認 token 通縮不是一次性事件而是結構性趨勢，Sam Altman 在多次訪談中將此規律化為「AI 使用成本每 12 個月降 10×」。

第四次 inflection — 2024-04 / 2024-12：開源把商品 tier 價格地板打掉 20×。2024-04 Meta 釋出 Llama 3、2024-07 Llama 3.1 405B 商用授權、2024-12 DeepSeek V3 證明前緣模型可在 $6M 訓練預算內達成。新興 inference host（Together.ai、Fireworks、Hyperbolic）開出 Llama 3.1 8B 推論價 $0.02/M output — 接近邊際 compute 成本。商業 API providers 被迫重新定價：GPT-3.5 Turbo 同等水準從 2023 的 $2/M 跌到 2024 末的 $0.07/M（-280× in 24M）。Epoch AI 統計顯示 2024-01 起 median 通縮率從 50×/yr 加速到 200×/yr — 是這次 inflection 的量化標誌。歷史功能：開源讓商品 tier 永遠回不去獨佔定價，frontier labs 唯一定價護城河變成「下一代未推出的能力」而非已部署的模型。

第五次 inflection — 2025-01-20：DeepSeek R1 把 reasoning premium 也打穿。DeepSeek R1 上線定價 $0.55/$2.19 input/output per M tokens，benchmark 與 OpenAI o1（$15/$60）相當 — 便宜 96%。2025-01-27 NVDA 一日跌 ~$600B 市值。OpenAI 被迫加速推出 o3（$10/$40）並降階 o3-mini 至 $2/$8。但同時，reasoning model 引入「hidden reasoning tokens」— 模型在生成可見輸出前先生成思考 token，單次查詢消耗 token 量比 base GPT-4o 多 10–100×。GPT-4o $0.01 的查詢在 o3 high-effort 模式下可能 $0.10-$1.00。歷史功能：token 單價 vs 工作負載 token 量的剪刀差正式打開 — 推論 compute 總需求即使在 token 通縮環境下也仍持續暴增（Google tokens 處理量 2024-04 → 2025-末 增 50× 至 1.3 quadrillion/月）。這是 Jevons paradox 在 AI 上量化證據最強的一次。

三個歷史類比

類比一 — 電信長途電話 1982-2005：邊際成本歸零後，pipe 不值錢、application 才值錢。AT&T 1984 年分拆後長途電話費率從 1982 的 $0.40-0.50/分鐘（傍晚 Sprint）暴跌到 2000 年的「平費包月」實質歸零。AT&T 1996-2003 花 $50B capex 蓋光纖網路，最終 2005 年以 $16B 賣給 SBC — 90% 價值蒸發。WorldCom/MCI 同期花 $45B capex、破產後值 $7-8B。真正獲利的是上面跑的應用（Skype、Vonage、後來的整個 Web 2.0）以及行動載具營運商（換了載體）。對 AI inference 的映射很精確：GPU 雲端裸 rental（CoreWeave/Lambda/Hyperbolic）對應 IXC — token 通縮逼近邊際 compute 成本後，pipe 變不值錢、margin 上移至「應用 + 差異化能力 + 整合層」（OpenAI/Anthropic API → Bedrock/Vertex/Azure 整合 → M365 Copilot 應用）。Cisco 的設備供應商命運是 NVDA 的尾風險（即使終端 ROI 惡化，因採購週期長，equipment vendor 營收仍能撐一段時間 — 直到突破點到來時崩塌往往很快）。

類比二 — AWS EC2 雲端 compute 商品化 2006-2021：fixed-cost scale 經濟讓 GM 反而擴張。AWS 從 2006 年起 15 年內降價 107 次（2023 累計 134 次），2014→2017 年同等工作負載成本下降 73%。Amazon CFO 公開把策略描述為「確保我們是最後一個還站著的」 — 用持續降價勸退新進入者、鎖住開發者。但 AWS infrastructure 毛利在這段期間反而擴張 — 因為自研矽（Graviton）、purpose-built 機房、軟體優化讓 cost 降得比 price 還快。價值從原始 compute（IaaS）上移到 managed services（RDS、Lambda、SageMaker）。對 inference 的精確映射：H100/H200 雲端 rental 市場（CoreWeave、Lambda、Hyperbolic）對應 2008 的 EC2 — 早期供應稀缺有超額 margin，2026 H100 spot 已從 2023 的 $7.50-11/hr 跌到 $1.87-3.50/hr（-64% to -75%）。贏家是控制「inference API 抽象層」（Azure OpenAI、Bedrock、Vertex AI）的玩家而非裸 GPU-hour。

類比三 — Akamai CDN 商品化 2000-2020：純 delivery 商品化，需上移至安全 + edge compute。Akamai 1999 年 IPO 在 CDN 稀缺定價高峰；2008 AWS CloudFront 推出 pay-as-you-go 撤掉 Akamai 的合約期門檻；到 2010 年代初 CDN 頻寬已全面商品化。Akamai Q4 2024 delivery 營收 $318M、年降 18% — 純結構性衰退而非週期。GM 守在 65-67% 完全靠安全服務（DDoS、WAF）與 edge compute 拉抬。對 inference 的映射：純 token serving（無差異化模型）= 純 CDN delivery（商品）。Akamai 的存活路徑 = inference provider 必須上移至「orchestration + RAG + fine-tuning + 模型路由」這些差異化層。Anthropic 的 Claude Code（$2.5B ARR 2026-02）與 OpenAI 的 ChatGPT Pro $200/月（< 1% 用戶 / 5.8% 早期收入）都是這個 thesis 的早期實證。

五次 inflection 與三個類比共同指向後續推估必須回答四個問題 —（1）OpenAI/Anthropic 是否能在 token 通縮環境中靠模型差異化保住 GM（§5、§8 thesis #2）？（2）NVDA 是否會走 Cisco 1999-2001 的 boom-bust 路徑？（§3 因果閉合、§9 thesis #1）（3）開源 + reasoning model 是否會持續打穿 frontier moat（§5、§8 thesis #1）？（4）電力 binding 是否會把 inference 的稀缺性從晶片端轉移到 megawatt 端（§3、§9）？這四個問題定義了 2026-2031 的 alpha 路徑。

§2 現在供給結構 — 七大推論工作負載供給商與 share 軌跡

推論 compute 的供給結構與 training compute 不同。Training 集中（NVDA >90% 全球 share），inference 已開始分裂 — 因為 inference 工作負載對「每瓦 token 數」與「$/M tokens TCO」敏感度遠高於對絕對效能敏感度。下表整理 7 個主要推論工作負載供給商過去三年的 share 軌跡、與 2027 方向。

表面 share 數字背後有兩個結構性事實。第一，幾乎所有 hyperscaler 自研 ASIC 都不對外賣 — Google TPU 對外經 GCP（Vertex AI、Cloud TPU rental）、AWS Trainium 經 Bedrock 與 EC2 Trn instances、Microsoft Maia 純內部用（GPT-5.2 + Copilot 工作負載）、Meta MTIA v2i 完全自用（44% TCO 優勢 vs GPU，論文 ISCA 2025）。這代表 NVDA 失去的市占並未轉化為「另一家獨立矽公司」崛起 — 而是被 hyperscaler 吃進 vertical integration 結構中。第二個事實：Broadcom 是這個 layer 真正的「外部 fabless 贏家」— Q1 FY26 AI 營收 $8.4B（+106% YoY）、4 家 hyperscaler XPU 客戶（Google、Apple、Meta、第 4 家放下 >$10B rack orders）、Mizuho FY26E $40.4B 預估、FY27 SAM $60-90B。Marvell 是 #2（FY26 DC 營收 $6.1B、18 個雲端設計 wins、$1.5B 客製矽 run-rate），Alchip 是 #3（AWS Trainium 2026 CoWoS wafers 從 2025 的 5K 暴增到 70K，+1,300%）。

真正的瓶頸已不在矽。CoWoS-L 配額 2024-25 是核心 binding（NVDA 占 70%+），但 TSMC 2026 月產能從 35K wafers 擴至年底目標 130K（+270%），ASE CoWoP / Amkor / 玻璃基板 2027 上線後 CoWoS 從 2027 H2 起預期鬆動。瓶頸已轉到電力：美國 utility 變壓器交期 ~128 週（Wood Mackenzie Q2 2025 survey），2026 已惡化向 150-160 週；PJM interconnection queue 平均 8 年；Cleveland-Cliffs 是全美唯一 GOES（取向電工鋼）供應商。推論工作負載對電力的敏感度高於 training — 因為 inference 是「24/7 always-on」而 training 可以排程，inference latency SLA 不允許跨地域負載均衡。SK Hynix HBM4 2026-02 量產（70% NVDA Rubin 平台 share）也是另一個結構性 binding — SK Hynix CFO 公開表示「2026 整年 HBM 供應已售罄」。

推論供給結構正在從「單一巨頭 + 配件商」轉向「vertical integration cluster + ASIC fabless」。對投資而言：(a) NVDA 仍將是絕對美元最大贏家（FY26 DC $193.7B），但相對市占從 ~85% 衰退到 ~75% 已是 fait accompli；(b) AVGO/MRVL/Alchip 是這個 layer 唯一可以「純外部投資」吃到 share migration 的標的；(c) Cerebras IPO 2026 Q2 是 pure-play inference upside 的 binary event；(d) 真正 binding 已從矽轉到電力電子（ETN / VRT / GEV）與電網基建。

§3 未來供給趨勢 — capex pipeline、ASIC ramp、CUDA inference moat 裂縫、電力 binding

2026 年是推論供給端「三軌共振擴張」的 inflection year。第一軌：hyperscaler capex — 大型 4 家（Amazon、Google、Microsoft、Meta）2026 capex 合計 ~$720B（vs 2025 $365B、近翻倍），其中 ~$545B AI-allocated。Amazon $200B、Google $185B、Microsoft $190B、Meta $125-145B。Stargate（OpenAI + Oracle + SoftBank + MGX）獨立計畫 $500B / 4 年、$100B 即刻部署、7 GW IT capacity 規劃中（第一個 Texas site 2025-09 已上線）。AMD-OpenAI 6 GW partnership（2025-10 announced）first 1 GW MI450 tranche H2 2026 ramp、warrant 結構讓 OpenAI 持 160M AMD shares 上限。CoreWeave 一家 neocloud 就 $99.4B 合約 backlog + 3.5 GW 已簽電力 + FY26 guidance $12-13B 營收。

第二軌：ASIC ramp curve 將是 2026-2028 最大供給結構變動。Counterpoint Research：custom ASIC 推論單元滲透率 2024 ~3% → 2025 ~14% → 2026 ~17-19% → 2028 ASIC 單位出貨量預估超過 GPU 單位出貨量。MediaTek 預估 2026 ASIC 營收 $1B+、目標 2027 占公司營收 20%。TSMC 預估每家主要 hyperscaler 客製矽都在 3nm 滿載。具體驅動：(a) Broadcom 第 4 家 XPU 客戶 >$10B rack orders Q3 FY26 出貨；(b) Google TPU v7 Ironwood 9216-chip pod 全面替換 v5e；(c) AWS Trainium3 2026 量產，Anthropic 已部署 1M Trainium2 across 1.3 GW campus；(d) Microsoft Maia 200 US Central → US West 3 上線部署 GPT-5.2；(e) Meta MTIA v2i 在 5T+ inference ops/day 規模上 prove out 44% TCO 優勢。

因果閉合 §1 inflection #1 (CUDA moat)：inference CUDA moat 已開始裂、training CUDA moat 仍 binding 至 2030+。這個分歧是本 DS 對 §1 五次 inflection 中「CUDA 軟體護城河 19 年累積」的直接回應、必須在 §3 解決而非延後到 §8。具體證據：(a) DeepSeek R1 在 AMD MI300X 上完成 demo（vLLM + ROCm 支援已生產級），證明 reasoning model inference 可不依賴 CUDA；(b) Meta MTIA v2i 完全跑 PyTorch + XLA 路徑、Meta 內部推薦與排序推論不經過 CUDA；(c) Google TPU 走 JAX + XLA，與 CUDA 完全平行；(d) Microsoft Maia 200 與 OpenAI 合作 ONNX Runtime 自家 stack。對應的 Wintel/ARM 類比時間軸：Intel 在 PC CPU 的 80-90% share 走了 10-15 年才被 workload-specific 侵蝕（先 mobile、再 server、最後 datacenter）— 對 NVDA inference 而言已進入第 3-5 年（從 2022 Hopper 算起）。但 training 端 CUDA moat 仍極深：FlashAttention、Megatron-LM、cuBLAS、NCCL 這些訓練 primitives 在 hyperscaler 自研 stack 中尚無 production-grade 替代，至 2030+ 仍 binding。投資 implication：NVDA training revenue 走勢與 inference revenue 走勢應分開估值。

第三軌：真正 binding 已從晶片轉到電力。CoWoS 2027 H2 預期鬆動（OSAT step-up + 玻璃基板 commercial application 2027 末上線），但變壓器 lead time 128 週（2026-Q1 數字）已開始往 160 週惡化、PJM interconnection queue 平均 8 年才能上線、SMR（小型模組化反應堆）commercial deployment 預計 2028-2030 才開始有意義 supply relief。McKinsey：推論電力 20.9 GW (2025) → 93.3 GW (2030) at 35% CAGR、training 23.1 → 62.2 GW at 22% CAGR。推論電力 CAGR 比 training 快 13pp — 反映 inference 24/7 always-on 特性。供給彈性低（電力電子產業擴產週期 36-60 個月）造成價格上漲 50% 也無法在 36 個月內擴出 20% 新 capacity — 這是與 GPU 供給彈性最大的不同。

未來供給 12-36M 是「晶片端寬鬆、電力端短缺」的雙軌格局。對投資而言：(a) ASIC fabless（AVGO/MRVL/Alchip）受益 ASIC ramp 是 2026-2028 主軸；(b) NVDA 絕對美元受益 hyperscaler capex 翻倍但相對市占下滑已 priced in；(c) 真正 alpha 來源是電力電子（ETN/VRT/GEV/PWR/CEG）與 SMR 標的，這些股票過去 18 個月已 +50-150% 但結構性 binding 至少持續至 2028；(d) CUDA inference 護城河裂痕將在 2026-2027 開始反映在 NVDA inference 段營收成長率（vs training 段）的剪刀差。

§5 未來需求趨勢 — Jevons paradox、TAM 三情境、六大結構驅動

推論需求 5 年推估的核心不在「token 量會不會增長」（這已經 priced in），而在「token 量增長能否 outpace 單價通縮」 — 即 Jevons paradox 在 AI 上是否完全運作。歷史指標清楚：2024-04 → 2025 末 Google 全公司 token 處理量增 50×（480T → 1.3 quadrillion/月）、enterprise AI spend 同期 +3× 到 $37B（MSFT 口徑）、token 單價同期降 ~10×。token 量 / 單價 通縮的彈性係數約 5×（即每單價降 1×、量增 5×）— 這意味著總推論支出在通縮環境中仍以 ~3-5× / 3 年 速度擴張。底下 TAM 表給三情境，再展開六大結構驅動。

二次基建受益（§11 NET + DDOG 支撐）：六大驅動疊加後產生兩條 non-obvious infra demand —（i）推論流量 routing / caching 邊緣 toll booth：當 token 量增 50×、agentic loops 觸發 10-100 LLM calls/request、長 context 把每 request payload 推至 MB 級，inference traffic 在 edge / regional PoP 的 routing 成本與 latency 變成新瓶頸，這是 Cloudflare Workers AI 與 Fastly 對應的需求（NET 已揭露 Workers AI requests 2025 YoY +280%）；(ii) LLM observability 觀測層：每個 inference stack 都需要追蹤 token usage / latency / hallucination / 成本歸因，這是 Datadog LLM Observability 產品的 wedge（DDOG 2026 Q1 LLM-related ARR 已揭露為 $200M+ run-rate）。這兩條都不在 §3 直接供給端 / §4 直接需求端的敘事主軸，但是六大驅動的結構性副產物。

結構性 shift：(a) 商品 tier 推論已永久商品化（開源 Llama/DeepSeek 鎖死定價地板），frontier labs 唯一定價權來自「下一代未推出」；(b) Reasoning model 改變 unit economics — 高價值 query 上限抬高、平均 cost-per-query 反向上漲；(c) Edge inference（Qualcomm NPU 80 TOPS、Apple Neural Engine）開始侵蝕 commodity inference 的 cloud 需求 — 3B-8B 參數模型可端側跑、用戶不再付 token。Demand inflection 預估落在 2026-2027：reasoning model 普及 + agentic AI penetration 跨過 40% 企業應用 + 嵌入式 SaaS Copilot 滲透從 3% 倍增到 6-10%。

TAM 推估的 5 年 19% base CAGR 看似溫和，但 base case 預設 Jevons 完全運作（量增 5× 完全 offset 單價 5× 通縮）— 任何偏離（reasoning 加速 = bull、open-source floor 失守 = bear）都是雙位數百分比的 swing。對投資而言：(a) 整體 TAM 結論是「溫和增長 + 高情境分散」、不是「明確 +50% YoY」這種催狂條件；(b) 真正 alpha 來源在 mix shift 而非 TAM 總量 — ASIC fabless vs LLM API providers、vertical integration vs neocloud、edge silicon vs cloud inference；(c) 6 大驅動中前 3 條（reasoning、長 context、agentic）是「token 量倍增器」、第 5-6 條（embedded、消費 S 曲線）是「用戶 base 倍增器」、第 4 條多模是「per-bit 倍增器」 — 三者疊加才推得到 bull case。
§5 → §6 橋段 — 供需平衡結論
依本 DS §3 與 §5 推導：2026-2027 供需處於「平衡偏緊」（晶片端從 H2 2027 起鬆動、電力端結構性短缺至 2030+）；2028-2030 供需轉「平衡偏寬鬆」（ASIC ramp + edge inference + token 通縮持續 → 推論 compute 供給快過需求增長）；2031+ 結構性過剩（Jevons 對沖力下降 + 開源 floor 進一步壓低 commodity tier）— 但僅限商品 tier，frontier model + reasoning 仍能維持稀缺溢價。供給結構決定誰能在這個由緊→鬆的過渡中保住毛利：vertical-integrated 玩家（MSFT/GOOG/META vertical stack）+ ASIC fabless（AVGO 拿 design royalty）勝出，pure-play LLM API（OpenAI/Anthropic 未上市部分）與 neocloud（CoreWeave 等）面臨持續定價壓力。

§7 投資時鐘 — Phase II「容量擴張 + 單價通縮 + winner 未明」

本 DS 判定推論經濟學 2026 中處於 Phase II（容量擴張 + 單價通縮 + winner 未明）。Phase 判讀基於三個證據：(a) 全球 AI compute 仍在快速擴容（hyperscaler capex 2026 翻倍、Stargate 啟動），對應 Phase II 特徵的「大量資金湧入、容量過剩開始累積」；(b) Token 單價持續通縮（中位 50×/yr、加速至 200×/yr），但同期需求量 outpace 通縮（Jevons 主宰），對應 Phase II 的「價格戰開始但贏家未定」；(c) Margin pool 正在遷移但未完成（OpenAI/Anthropic GM 仍在 30-50% 區間爬升、MSFT vertical integration 已 70%+），對應 Phase II 末 / Phase III 初的過渡期。歷史類比：對應 1998-1999 telecom（boom-bust 前夕）、2008-2010 AWS EC2（capacity scaling 前期）、2003-2005 Akamai（CDN 商品化邊緣）。

Phase III 進入訊號（容量平衡 + winner 確定）— 必要 + 充分條件：推導：必要條件 (1) ASIC 推論市占 ≥ 40%（per §3 ramp curve，目前 ~17-19%、推估 2028 達標）；必要 (2) LLM API GM 收斂至 60-70%（OpenAI 目前 33%、Anthropic 50% target 2026、收斂達 60% 需 2027-28）；必要 (3) Top-3 vertical integrators (MSFT + GOOG + META) 占推論收入 ≥ 60%（目前 ~40%、CAGR 達 +15-20% 則 2028-29 達標）。充分條件 (1) 電力 binding 出現明確 relief（SMR commercial deployment 開始 ≥ 5 GW、變壓器 lead time < 80 週、PJM queue < 4 年）；充分 (2) 至少 2 家 LLM API provider 持續 ≥ 4 季 GM > 60%。必要全達 + 充分至少 1 條 達成 → 進入 Phase III。預估時間：base case 2028 末、bull case 2027 H2、bear case 2030+ 仍處 Phase II 後期。

§8 Non-Consensus View — 三條 vs 市場共識的具體分歧

反共識 #1：Token 通縮並不意味 inference 營收衰退 — Jevons paradox 完全主宰至 2028+。市場共識說：token 單價 6 年崩 600× → LLM API 商業模式長期不可持續 → AI 公司估值應 derate。本 DS 看：Jevons paradox 量化證據已壓倒性 — Google token 處理量 2024-04 → 2025 末增 50× 同期單價降 ~10×、enterprise AI spend +3× 至 $37B（MSFT 口徑）— token 量 / 單價彈性 ~5×。Anthropic ARR 16 個月內從 $1B → $30B 證明定價權在 frontier model 上仍存（即使 commodity tier 已 floor）。分歧原因：市場用線性 mental model 看通縮（單價 ×0.1 = 營收 ×0.1），但實際結構是 power-law（每次通縮觸發更廣的應用採用、reasoning + agentic 倍增 token consumption 量級）。

反共識 #2：NVDA inference share 將從 ~80% 降至 30-40% by 2028，但 training share 仍維持 80%+ 至 2030+。市場共識說：NVDA 全 AI 市占穩定在 75-80%、CUDA 護城河護住 training + inference 兩端。本 DS 看：必須把 NVDA 拆成兩段估值 — training segment 仍 dominant（CUDA + NCCL + FlashAttention + Megatron 護城河至 2030+），但 inference segment 已開始裂（DeepSeek R1 在 AMD MI300X 跑、MTIA 純 PyTorch+XLA、TPU JAX、Maia ONNX）。對應 Wintel/ARM 類比的 workload-specific 侵蝕路徑：先 mobile（= 推論）後 server（= training），時程 10-15 年（已過 4 年）。Counterpoint Research / Introl 推估 NVDA inference share 2028 落在 20-30%（aggressive）至 40%（保守）。分歧原因：market 把 CUDA 當成單一護城河、本 DS 把 CUDA 分為 training-stack（深）vs inference-stack（已被穿透）。

反共識 #3：Sora 失敗不是孤例、是 premium inference 廣泛單元經濟學危機的先聲。市場共識說：Sora 關停（$15M/day cost vs $2.1M lifetime rev）是 OpenAI 對非核心產品的策略性 trim。本 DS 看：Sora 揭示一個更深的結構問題 — premium inference（reasoning model + 多模 + agentic）的單元經濟學在 token 通縮環境中無法閉合。GPT-4o 普通 query 成本 $0.01、o3 high-effort 同一問題 $0.10-1.00（10-100× 倍增）；但 user 願付溢價多半 < 5×（$20 Plus 對 $200 Pro）。Sora 的「$7.14 cost per $1 revenue」（$15M/$2.1M）僅是極端例；類似結構在 OpenAI 整體（$1.69 cost per $1 revenue）已可見。分歧原因：市場把 Sora 當「個別產品決策」，本 DS 把它當「2026-2027 預期會看到更多 premium 推論產品關停的 leading indicator」（trigger: 2026 H2 - 2027 H1 若再有 2-3 個 GPT-5 / Gemini 3 / Claude 5 premium 模式被 quietly downsized 或下架 → confirm thesis）。

三條 non-consensus 共同指向 — 市場目前對 NVDA inference share 過度樂觀、對 LLM API 商業模式過度悲觀、對 vertical integration 玩家（MSFT/GOOG/META）的 margin migration 認知不足。實際 positioning：(a) Underweight NVDA inference exposure，但維持 training 段全 weight；(b) Overweight MSFT 直接 + GOOG 部位（吃三層吃糖 vertical integration）；(c) 對 Anthropic（如未來上市）保持 high attention — 80× ARR 成長證明 frontier 定價權不死。
