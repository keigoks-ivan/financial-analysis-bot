§1 產業定義與邊界

「Space Economy」是 2024 年 WEF/McKinsey 重新定位的詞彙，把過去散落於「衛星通信」「火箭發射」「國防航太」的子產業整合為「以太空基礎設施為輸入，以地表 + 太空雙層服務為輸出」的單一價值鏈。本 ID 範圍刻意限縮在 商業 + 國防的太空基礎設施與下游服務，不含太空旅遊（Virgin Galactic 等小規模 hobby 市場）也不含 NASA 純科研項目（Artemis 程式雖大但屬政府支出非商業 TAM）。

2024-2026 的 inflection 來自三個獨立但同步發酵的力量：① SpaceX Falcon 9 重複使用降本讓 LEO 衛星發射成本從 Shuttle 時代 $54,500/kg 降至 SpaceX 對外 $7,000/kg、自家成本 $629/kg [T2: NextBigFuture 2026-02]；② Starlink 達到商業驗證（2025 $11.4B revenue +50% YoY / EBITDA $7.2B / 10M subs）證明 LEO megaconstellation 可獲利 [T3-B: TradingKey Starlink IPO 分析 2026]；③ SDA Tranche 3 + Golden Dome FY26 $13.4B 撥款把太空從「政府 hobby 預算」升級為「national security 主要 capex 池」[T1: SatNews FY26 defense bill 2026-01]。

💡 §1 Insight
1. 本 ID 內部其實是 3 個不同節奏的市場：「launch + 既有 EO」（2026-2027 cash flow 兌現週期 — RKLB / PL / IRDM 已開始正 EBITDA）、「商業 D2C / megaconstellation」（2026-2030 binary execution — Starlink IPO + ASTS + Kuiper 三家對殺）、「軍事太空（SDA / Golden Dome）」（2026-2030 政府訂單 step-up — LMT / NOC / RKLB prime）。共識把這三個節奏打包為「太空股」單一暴險，是估值偏差來源。
2. 真正最容易被忽略的灰色地帶是 NVDA 的 Starcloud 投資 + AMZN 的 Globalstar 收購：兩者都把「空間」從電信 / 國防領域延伸到 AI 基建 + retail 雲端 — 隱含「太空成為 hyperscaler infrastructure 一部分」的範式轉移；如果這個 thesis 成立，本 ID 的 TAM 應上修 30-50%，因為新 demand pool（hyperscaler）尚未完全反映在 WEF/McKinsey 的 $1.8T forecast。
3. 投資意義：對 PM 而言，本 ID 內部需要至少三個倉位籃 — 「政府訂單籃」（LMT/NOC/LHX/RKLB/PL/BAH）+「商業現金流籃」（IRDM/Starlink IPO 後）+「Binary optionality 籃」（ASTS/AMZN Kuiper/NVDA Starcloud）— 不能同視為「太空」單一暴險。

歷史類比（QC-I14 強制 2-3 個具體前例 + QC-I28 反例）

正類比 #1 — 1999 Iridium 1.0 破產 + 2002 Globalstar 破產：Motorola/Boeing 主導的 Iridium 1.0 1998 部署 66 顆衛星，1999-08 申請破產 ($5B 投資 → $25M 收購)。失敗原因：① 衛星太重（每顆 700kg）→ 終端設備龐大不便攜帶；② 90 年代末 GSM 普及，普通 cell phone 已涵蓋大部分商業旅遊地點；③ 訂價過高 ~$3-7/min。對 ASTS 的對照：本輪 ASTS BlueBird 2,400 sq ft array 大 3-4 倍、direct-to-cell 不需特殊終端、單台支援 10 GHz / 120 Mbps — 但仍存在「45 顆衛星 by end-2026」的執行 risk + BB7 已失敗一顆。本輪 vs 前例量化差異：技術已驗證（不是衛星電話特殊終端 dilemma），但 execution 風險仍高。

正類比 #2 — 2003-2008 商業 launch 業者大整併：Boeing Delta 第二代、Lockheed Atlas、Sea Launch、Soyuz 等 5+ 家在 2000s 中期 commercial launch 嚴重競爭，最後 ULA 2006 由 Boeing + Lockheed 合資成立（壟斷美國政府 launch ~$20B 市場）— 直到 2010s SpaceX 起來才被打破。對本輪 D2C 的指引：商業領域 5+ 家（Starlink T-Mo / ASTS AT&T+VZ / Globalstar/AMZN-Apple / IRDM Stardust / Lynk Global）2026-2028 同樣會經歷整併 — 預期 2-3 家 winners + 2 家被收購（GSAT 已被 AMZN 收購）。

正類比 #3 — 2018-2024 Falcon 9 重複使用 ramp：SpaceX 2017 達成 booster reflight，2018-2024 從 21 launches/year 升至 165 launches/year（8x increase），同期國防 + 商業 launch 市場全面被 SpaceX 吃下（82% 市佔 2025 vs 2018 ~30%）。對 Neutron 的對照警示：RKLB Neutron 已從 2025 滑期至 Q4 2026 — Falcon 9 從 2010 首飛到 2017 booster reflight 用了 7 年，Neutron 設計到首飛預計 5-6 年（2020-2026），時間軸合理；但「Neutron 商業化 ramp」在 2027-2030 是關鍵窗口，如同 Falcon 9 2018-2024 期。

反類比（QC-I28 強制）— 2007-2010 衛星廣播 satellite radio 市場：XM + Sirius 1999-2007 期間多次「結構性護城河」narrative（FCC 2-license 寡占），但 2008 GFC + Pandora / Spotify 興起讓 satellite radio 成為「dying middle」— Sirius XM 雖未破產，估值 +30%/年 漲 4 年後 -65%。對 D2C 的對照警示：D2C narrative 同樣有「結構性護城河」訴求（FCC 授權 + spectrum 限制），但若 5G NTN（3GPP 標準）普及讓「衛星 + 地面 5G 無縫切換」成為公用標準，純 D2C pure-play（如 ASTS）會被擠到「edge case」服務（極遠航海 / 極地）— 估值 derate 50%+ 風險不可忽視。

💡 §2 Insight
1. 本輪復興與前兩輪 LEO 商業化的本質差異是 unit economics 已驗證：Iridium 1.0 / Globalstar 1.0 失敗原因是 unit economics 從未證實；Starlink 2025 EBITDA $7.2B / margin 63% 已證實 LEO megaconstellation 可暴利。投資邏輯：Starlink 已驗證的 model 套用到 ASTS / Kuiper 仍非 trivial — 直連手機 / 純廣播 / 純 government 各自 unit economics 各異。
2. SpaceX 估值（$1.75-2T）仍可能被低估：與 2010 $0 市值對比，市場長期低估 SpaceX 主導力。本 ID 用 30-40x EBITDA 給 Starlink 已是保守 — 若 xAI orbital compute + Starshield 整合兌現，$2T 估值對應 Starlink alone fair value $400-600B（占 30-40% 比重）並非泡沫。
3. 反類比警示：D2C 領域要警惕「satellite radio 2008-2010 失速」case — 5G NTN 標準若普及，純 D2C pure-play 會被擠到 edge case 服務；ASTS 必須在 2026-2027 達成 commercial scale，否則可能重演 satellite radio 命運。
