灰色地帶

  • Rear-door heat exchanger 算液冷還是風冷？ 本 ID 納入，因使用液體介質
  • Cooling fans（風扇）：雖傳統，但仍與液冷 co-exist（供 CPU 風道 + secondary cool）
  • Dielectric fluids 3M 業務：3M 已 exit PFAS 爭議，但新化學仍在 scope
💡 Insight §1
1. 液冷邊界快速擴大：從「只有 AI rack」到「HPC + edge + 雲端一般 rack」；TAM 持續被重估
2. Dielectric fluids 是「化學端」的隱性贏家：每 rack immersion 需數百升，化學品商值得追蹤
3. Rear-door heat exchanger 是 retrofit 中盤的主力：不是每個 DC 都能重建，retrofit 是 5 年主力需求

💡 Insight §2
1. 液冷轉換週期 4-5 年：從第一波 rear-door（2021）到 direct-to-chip 主流（2026）吻合歷史 pattern
2. Immersion 2017-2022 在加密礦機已驗證：技術成熟，只缺標準化和規模
3. PC 水冷 → DC 液冷是「技術民主化」：Asetek 從消費跨界到 DC OEM，類似 PC PSU 廠跨界到 DC power

★ 最低估風險

🟡 PFAS 相關 dielectric fluid 環保監管 信心：中
  • 事實 1：3M 已退出 PFAS 業務（2025）
  • 事實 2：歐盟 / 美國 PFAS 規範持續收緊
  • 事實 3：Two-phase dielectric 多為 PFPE，受影響
→ 若 PFAS 規範使 two-phase immersion 成本大漲，single-phase 受益

§9.5 Kill Scenario

🟡 反方 1：Vertiv 被 Trane / Johnson Controls 衝擊 信心：中
  • 事實 1：Trane 併 LiquidStack（2026-02）
  • 事實 2：Johnson Controls、Carrier 也在液冷佈局
  • 事實 3：傳統 HVAC 公司有規模 + 通路優勢
→ 2027-2028 競爭加劇，Vertiv 毛利可能壓縮 2-3pp
⚠ 證偽：2027 Trane / Johnson Controls 液冷 rev > $1B
🟡 反方 2：Immersion 因安全 / 法規失敗 信心：低
  • 事實 1：Two-phase dielectric PFAS 爭議
  • 事實 2：消防規範對 tank 限制
  • 事實 3：維護複雜度
→ 若 immersion 無法規模化，DTC 繼續主導
🟡 反方 3：AI GPU 熱效率突破 → 液冷不再必需 信心：低
  • 事實 1：Blackwell 比 H100 每 token 效率 +25x
  • 事實 2：若晶片熱設計功耗不再線性增長
→ 液冷強制性降低，風冷延續
