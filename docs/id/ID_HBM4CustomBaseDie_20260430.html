💡 §0 Insight：base die 是「HBM 第二定價權階段」的真正勝負點
事實基礎：HBM3E 的競爭軸是「1c DRAM 良率 + stacking 良率」（兩個都是 memory 廠 internal，外人插不上）；HBM4 的競爭軸新增「base die logic process + IP 整合能力」（這個對 SK Hynix 是強項，對 Samsung 是反攻機會，對 Micron 是 TSMC 依賴度）；HBM4E + C-HBM4E 進一步加入「per-customer 客製能力」（這個對 IP 設計實力強的廠商有結構性優勢）。
推論邏輯：當競爭軸從「yield 戰」（純製造能力）轉到「IP 設計戰」（設計 + 製造雙軸），HBM 三巨頭的相對順序可能重洗—— 不是因為誰 yield 提高 5pp，而是因為誰 IP 整合能力強。SK Hynix 是 IP 投資最大（HBM3 做兩代領先）+ TSMC 深度合作（base die）+ 16-Hi 首發 = 三軸都領先；Samsung 是 IP 弱於 SK 但 4nm 自代是 cost / efficiency 突破點 = 高風險高 reward 反攻；Micron 是 yield 最穩但 base die 完全外包 TSMC = 結構性落後 IP 設計但仍贏在製程紀律 + 美系唯一。
投資意義：這個 framing 下 HBM 三巨頭 base case：SK Hynix（市佔結構穩 50%，PE 12-15x）+ Micron（穩定第二 25-30%，PE 14-18x quasi-monopoly）+ Samsung（binary 30-40% reverse OR 25-30% 失敗，PE 8-12x → 上修 12-15x）。TSMC 同時受益（QC-I33 non-obvious）。

§1 子題定義與邊界

本子題聚焦「HBM4 base die 從 DRAM-process 跳級到邏輯製程」這個技術轉折，及其引發的競爭結構重構。Base die 在 HBM 堆疊最底層，承載 PHY、控制邏輯、ECC、I/O routing，是「memory stack 與 SoC 之間的 interface」。HBM3E 之前 base die 用同 stack 上層的 DRAM-process（成本低但邏輯複雜度有限）；HBM4 起，因為 IO 從 1024-bit 翻倍到 2048-bit + 0.8V 低壓設計需要 finer metal stack，base die 必須跳到邏輯製程（TSMC N12FFC+ 或 Samsung 4nm）。HBM4E / C-HBM4E（2027 量產目標）進一步升到 N3P，且 JEDEC 開放 per-customer 客製化 base die 接口。這個技術轉折表面上是「HBM 性能升級」，本質上是「HBM 從 commodity 變 specified silicon」—— 過去 HBM 訂單由「廠商 yield + capacity」決定，未來 HBM 訂單由「per-customer IP 整合能力 + foundry 關係」共同決定。

本子題與母題的責任分工：母題（ID_HBM_Supercycle）涵蓋 HBM 整體市場規模、TAM、三巨頭整體 share / capex / capacity；本子題聚焦 base die 製程演進、客製化 JEDEC 規格、Samsung 4nm vs SK/MU 外包 TSMC 的 binary 比較、TSMC 隱性贏家量化、per-customer 客製化對 NVDA / AMD / AVGO / MRVL 的供應鏈影響。本子題不重述母題的 HBM 整體 TAM 數字（2030 ~$80-100B 等），而是專注於「base die 這個 sub-component 怎麼改變 HBM 競爭邏輯」。

§2 技術演進 + 歷史類比

HBM base die 的演進可分四代：HBM1-3 (2014-2024) base die 用 DRAM-process（與上層堆疊 die 同晶圓），邏輯複雜度有限（~10K gates），純粹處理 read/write commands；HBM3E (2024-2025) 仍 DRAM-process 但 PHY 速度提升（9.6 Gbps），是「DRAM-process 極限」一代；HBM4 (2026 量產) 首次跳到邏輯製程（TSMC N12FFC+ 或 Samsung 4nm）—— 這是歷史性製程跨越，因為 IO 從 1024-bit 翻倍到 2048-bit + 0.8V 低壓 + 完整 memory controller 整合，DRAM-process 已不夠細；HBM4E / C-HBM4E (2027 量產) 升到 N3P + 開放 per-customer 客製，base die 變成「memory-near compute」（offload ECC、signal conditioning、部分 memory management），效能是 HBM4 的 2x。

第一個關鍵動態：「IO 從 1024 → 2048 是 HBM4 真正的 enabler」。市場誤把 HBM4 的「per GPU 22 TB/s 頻寬」看作 DRAM 速度提升（從 9.6 → 11+ Gbps），但實際真正貢獻是 IO 寬度翻倍（1024 → 2048-bit）—— 這需要 base die 上 2x routing density + ESD 保護，DRAM-process 的 metal stack 不夠細，必須升到邏輯製程。對封裝端的同步影響：interposer 上每顆 HBM 的 routing density 翻倍 → CoWoS-L 的 RDL 層數從 6-8 升到 10+ 層 → 直接對應 NVDA Rubin Ultra 從 4 die 退到 2 die 的 warpage 問題（母題 §2 已述）。HBM4 IO 翻倍 + CoWoS-L 物理上限是同一事的兩面。

第二個關鍵動態：「Samsung 4nm 自代 vs SK/MU 外包 TSMC 是策略分岔」。Samsung 用 Samsung Foundry 4nm 自代 base die，SK Hynix 與 Micron 外包 TSMC N12FFC+。表面看是「Samsung 垂直整合 vs SK/MU 專業分工」，本質是「3-layer 工廠（DRAM + base die foundry + assembly）vs 2-layer 工廠（DRAM + assembly，base die 外包）」的策略選擇。Samsung 4nm 比 N12FFC+ 更 advanced（節點更小、更省電），如果 yield 衝過 70%，Samsung HBM4 stack 整體 cost / power efficiency 可贏 SK Hynix；但 4nm yield 風險也比 N12FFC+ 高（節點越小、defect density 越敏感）。這個 binary 是 2026 H2 ~ 2027 H1 HBM 競爭的核心 single point。

第三個關鍵動態：「HBM4E / C-HBM4E 把 base die 從 commodity → IP-rich 元件」。HBM4 base die 在 N12FFC+ 是 thin-routing（基本 PHY + 控制），但 HBM4E / C-HBM4E base die 在 N3P 開始整合「memory-near compute」邏輯（部分 ECC、signal conditioning、甚至少量 cache management）—— 等於把主 GPU 的部分功能下放到 HBM stack 內。客戶（NVDA / AMD / AVGO / MRVL）可以指定要哪些 offload 邏輯放在 base die，這就是「客製化」真正的 mechanism。對 memory 廠商，這意味從「DRAM cell + assembly」業務升級到「IP integration + co-design with customer」業務—— 商業模式根本性變化。

歷史類比：兩個有用、一個誤導（QC-I14 + I21 + I28）

有用類比 1：1990s 主機板 chipset 從 commodity → custom（北橋南橋分裂）。1990 年代 PC 主機板的 chipset 從通用版（Intel 440BX 一種供所有 OEM）演進到客製版（Intel/AMD 各自為大客戶定製）—— 同樣是「commodity → specified」轉折。當時誰能在 chipset IP 設計（不只 manufacturing）超前的廠商，賺走了利潤池的 60-70%（Intel chipset 業務毛利從 30% → 50%+）。今天 HBM4 base die 客製化路徑類似 —— SK Hynix 領先 IP 設計能力的廠商，將從「賣 commodity HBM stack」變成「賣 specified memory subsystem」，毛利結構性升 5-10pp。對應到 SK Hynix Q1 2026 NP margin 76.7% annualized > ₩200T（非常規高位）—— 這個高位部分反映客製化溢價已開始。

有用類比 2：2010s 智慧型手機 SoC integration 從 application processor → SoC + 整合 modem + ISP。Qualcomm Snapdragon 從純 AP 演進到「SoC + modem + ISP + GPU + AI engine + DSP 整合」—— 同樣是「specified silicon」邏輯：客戶（手機 OEM）越來越要求「single chip, customized」。今天 HBM 也走相同路徑：從「純 memory stack」→「memory + base die 整合運算 + per-customer IP」。Qualcomm 在 SoC 整合戰中贏出，因為它最早把 modem / ISP / GPU 各 IP 都做扎實；HBM 戰中對應的是 SK Hynix（最早投資 base die IP + TSMC 深度合作）。這個類比的對應結論：HBM 三巨頭中 SK Hynix 將拉開差距，不是因為 yield，而是因為 IP 設計實力。

誤導類比：Samsung 2018-2020 7nm logic foundry 「超車 TSMC」失敗。多數空頭用 Samsung 2018-2020 7nm foundry 嘗試超車 TSMC、後來失敗的歷史，警告 Samsung HBM4 4nm base die 自代也會失敗。反向類比警示：（a）2018 Samsung 7nm 客戶是外部第三方（Qualcomm、Nvidia、AMD），失敗代價是「失去客戶」；2026 Samsung 4nm base die 客戶是自家 Samsung Memory，失敗代價只是「cost / efficiency 不贏 TSMC N12FFC+」（仍可繼續用、internal transfer pricing 補貼）；（b）2018 7nm 在 leading-edge 與 TSMC 直接競爭；2026 4nm 是 mature node（落後 leading-edge 兩代）, yield 風險明顯小於 leading-edge；（c）2018 Samsung 設計 IP 落後 ARM 生態 4-5 年；2026 Samsung Memory 已 HBM3E 12-layer 通過 NVDA 認證、設計 IP 與 SK Hynix 差距明顯縮小。四項條件都不對應，所以這個類比是誤導，但很多空頭用它預測 Samsung 反攻會失敗 —— 這個誤用本身是子題 §12 非共識的支撐。

💡 §2 Insight：HBM 第三定價權階段已啟動 —「IP integration」是新護城河
事實基礎：HBM 至 HBM3E 競爭軸 = yield；HBM4 起新增「邏輯製程 + IP 整合」軸；C-HBM4E 起再加「per-customer custom design」軸 — 三軸疊加。
推論邏輯：這不是漸進升級，是「商業模式跳級」——從「memory product」變「memory subsystem with customer-specific IP」。能跨越的廠商（SK Hynix 最強、Samsung 反攻、Micron 紀律）保住 high-margin；不能跨越的，會被擠到「DRAM die supplier」位置（Samsung 7nm foundry 失敗模式類比）。
投資意義：portfolio 內 HBM 三巨頭的 conviction 排序應從「絕對 yield」轉到「IP 整合能力」—— SK Hynix 領先 conviction 升、Samsung 反攻是 binary（高 conviction 但高風險）、Micron 穩定第二（PE 重估邏輯不變）。

§3 核心技術棧拆解

HBM4 base die 客製化的技術棧由四層構成。第一層是 foundry process（TSMC N12FFC+ / N3P，或 Samsung Foundry 4nm），決定 base die 的物理製造能力 —— TSMC 用 N12FFC+ 做 HBM4 base die，每片 12 吋 wafer 約 ~$2,500 cost、可切 ~250-300 個 base die（考慮 yield），單個 base die 成本 ~$8-12（這是 TSMC 隱性收入的核心數字）。Samsung 4nm 自代成本未公開，估計每 base die ~$10-15（節點更先進但 internal transfer pricing 模糊）。第二層是 IP block design（PHY、ECC controller、memory management），這層是 SK Hynix 領先的核心 —— SK Hynix 自有完整 IP 庫，Samsung 部分外購（Synopsys / Cadence），Micron 與 TSMC + GUC 共同設計（GUC = 創意電子，TSMC 子公司）。第三層是 customization interface（JEDEC HBM4 / C-HBM4E spec），這層 2026 開始開放，允許客戶（NVDA / AMD / AVGO / MRVL）spec 自己的 base die 變體。第四層是 stacking & packaging（hybrid bonding、TCB、underfill），這層的 yield 直接影響整個 HBM4 stack 的良率。

事實基礎：第 3 + 第 4 層每代都重洗（16-Hi 今年勝出、明年 N3P stacking 又重洗）；第 1 + 第 2 層是多年累積（與 TSMC 關係、IP 庫深度）。
推論邏輯：portfolio 操作不應追逐每代 yield 領先（短期 catalyst），而應 anchor 在第 1 + 第 2 層領先的廠商 —— SK Hynix 雙層都領先（IP + TSMC 深合作），Samsung 第 1 層自代（高風險高 reward）+ 第 2 層追趕中，Micron 第 1 層完全外包（穩定但無彈性）。

§4 市場規模 + CAGR（聚焦 base die + 客製化部分）

§5 Value Chain（聚焦 base die 製造流程）

HBM4 base die 客製化的 value chain 比傳統 HBM 更碎片化，因為新增了「邏輯製程 foundry」+「客製 IP design」兩個段位。本節聚焦這個延伸 value chain，量化議價權變化。

這張 value chain 的關鍵解讀有四：（1）TSMC + IP 廠商（SNPS/CDNS）的議價權結構性升 —— base die 跳級邏輯製程後，這層成為 HBM 製造的必經關卡，TSMC 寡占且 IP 廠商高毛利持續；（2）中游 memory 廠議價權分裂 —— SK Hynix（IP 領先）強勢、Samsung（binary）unclear、Micron（穩定但缺彈性）持平；（3）下游客戶（NVDA/AMD/AVGO/MRVL）議價權兩極分化 —— NVDA 70% 訂單佔比 + 客製化需求最深，議價權極強；中型客戶（含一些非主流 hyperscaler）議價權弱；（4）NVDA 2027 自研 base die + hyperscaler 自研是中游 memory 廠的潛在威脅 —— 若實現，memory 廠變成「DRAM die supplier + assembly」，毛利結構性下修。

§12 Non-Consensus View（共識差異 — 子題核心）

三條本子題與市場共識的具體分歧（QC-I12 + I30 強制三欄式）。每條附 Bear / Base / Bull 三檔（Gate 2.2）。

非共識 #1：「Samsung 4nm 反攻是 binary single point — 不是漸進式 share 提升」

市場共識
Samsung HBM4 是「漸進式追趕」，share 每年 +3-5pp（25% → 28% → 31%）逐年靠近 SK Hynix；4nm 自代是「低效決定」（成本不贏 TSMC N12FFC+，又承擔 yield 風險）—— 多數空頭預期 Samsung 反攻緩慢且最終失敗。
本 ID 觀點
Samsung 4nm 自代是「binary single point」 —— Q3 2026 yield 達 70% 觸發step-function 上升至 35-40%（不是 +5pp 漸進）；失敗則維持 25-30% 5+ 年。Driver：4nm 整體 power efficiency + IP 整合彈性 inhouse 兩個結構優勢，加上 NVDA 對 power 的偏好，在 yield 達標後一次性 share unlock。
差異
Bull case 差距 +10-15pp（35-40% vs 共識 28-31%）；2027 EPS 差 +25-50%（Samsung Memory segment）；PE 從 8-12x → 12-15x。差異程度：顯著非共識（涉及 Samsung 整體投資邏輯）。
Bear: 4nm yield 卡 50-60% → Samsung HBM share 25-28% / PE 8-10x
Base: 4nm yield 達 65-70% → Samsung HBM share 30-33% / PE 10-12x
Bull: 4nm yield 達 70%+ + 16-Hi 趕上 → Samsung HBM share 35-40% step-function / PE 13-15x

支撐事實：（a）Samsung HBM3E 12-layer 已通過 NVDA（2025-09）+ HBM4 12H 通過 NVDA 驗證（2026-Q1）+ NVDA tests 評為「strongest overall」；（b）Samsung 4nm 整體 power efficiency 比 N12FFC+ 略低 5-10%；（c）IP 整合彈性 inhouse 對 C-HBM4E 客製化路徑有結構優勢；（d）Samsung 50% HBM capacity surge 2026 預備 step-function ramp。

分歧原因：市場用「Samsung 7nm foundry 失敗」歷史類比 + 「漸進式 share 提升」線性思維估 Samsung；本 ID 認為 4nm 自代 + IP 整合彈性 + NVDA spec 偏好三個結構性條件對齊後，是step-function 反攻而非漸進。Q3 2026 法說是 falsification 關鍵點。

非共識 #2：「TSMC HBM-related 2028 增量 $5-15B（vs 共識 $1-2B 差 5-10x）」

市場共識
TSMC HBM 業務是「邊緣 / 附帶」業務，僅 advanced packaging（CoWoS）有量化貢獻，base die foundry 視為「small extra」（2028 增量 $1-2B 量級）；TSMC 主要 thesis 仍是 NVDA / AVGO / AMD GPU 代工。
本 ID 觀點
TSMC 是 HBM 革命的隱性贏家，2028 HBM-related 增量 $5-15B（base die N12FFC+ + N3P + CoWoS 9.5 reticle + C-HBM4E premium 四線疊加）；對 TSMC 整體 PE 是獨立 +5-8pp driver，與 GPU 客戶的 thesis 是兩條平行線。
差異
2028 量級差 5-10x（$5-15B vs $1-2B）；對 TSMC fwd PE 影響 +5-8pp（22 → 27-30x）。差異程度：顯著非共識（涉及 TSMC 多 thesis 定價）。
Bear: NVDA 2027 自研 base die 成功 → TSMC base die cap 在老世代 / 2028 增量 $2-3B
Base: TSMC HBM-related 2028 增量 $5-10B（4 線疊加 base case）
Bull: C-HBM4E 客製化超預期爆發 + 9.5 reticle CoWoS 全產線 ramp / 2028 增量 $12-15B

支撐事實：（a）SK + MU 100% HBM4 base die 外包 TSMC N12FFC+；（b）N3P HBM4E base die 升級 per-die 收入 3-5x（從 $8-15 → $25-50）；（c）9.5 reticle CoWoS 2027 量產 + 12-24 stack/package；（d）C-HBM4E 客製化 NRE + per-stack 溢價（NVDA / AMD / AVGO / MRVL 各自付費）。

非共識 #3：「客製化結構轉折 = HBM 從 commodity 變 specified silicon — IP 設計實力是新護城河」

市場共識
HBM 仍是「memory commodity」業務，三家 yield 戰決定 share；客製化（C-HBM4E）視為「附加溢價」(+5-10%) 但不改變競爭結構；HBM 廠仍按 cyclical memory 廠估（PE 8-12x）。
本 ID 觀點
HBM4 客製化是「商業模式跳級」 —— 從「memory product」變「memory subsystem with customer-specific IP」。能跨越 IP 設計門檻的廠商（SK Hynix）保住 high-margin 並升至 quasi-monopoly（PE 12-15x）；不能跨越的（Samsung 失敗 / Micron 落後）退到 DRAM die supplier 角色。
差異
HBM 廠 PE 結構從 cyclical 8-12x → quasi-monopoly 12-15x（升 4-7pp）；競爭軸從 yield 變 IP 設計實力。差異程度：顯著非共識（涉及整個 HBM sector valuation framework）。
Bear: 客製化 limited（5-10%）+ HBM 維持 commodity 認知 → 三家 PE 8-12x
Base: 客製化 20-30% + IP-rich 廠商 quasi-monopoly → SK 14-15x / MU 14-16x / Samsung 10-13x
Bull: 客製化爆發 50%+ + memory-near compute 主流 → IP-rich 廠商 industrial monopoly → SK 16-18x

支撐事實：（a）SK Hynix Q1 2026 NP margin 76.7% annualized 已反映客製化早期紅利；（b）JEDEC C-HBM4E spec finalization 2025-12 進行中；（c）Hot Chips 2025 + ISSCC 2026 客製 base die 學術揭示 + 廠商 demo（Marvell SRAM-rich / GUC custom IP）；（d）NVDA / AMD / AVGO / MRVL 各自 spec C-HBM4E 規格（per-customer 客製確認）。
