Unit Glossary：Premium cabin = airline 中 business + first + premium economy 的合計（DAL Q1 25 premium ticket revenue $5.4B 接近超 main cabin $41M 是 milestone）；Main cabin / Coach = 經濟艙 / 標準艙；Co-brand royalty / remuneration = airline 從信用卡發卡銀行（Amex / Chase / Citi / BofA）收取的 fee（DAL Amex $8.2B FY25 / +11% / 中期目標 $10B 是 sector 王）；Loyalty revenue = airline 從 mile 銷售（賣給銀行 / partner）+ 直接 mile 兌換的營收；RPMs = Revenue Passenger Miles（運輸 capacity-output）；ASMs = Available Seat Miles（capacity）；PRASM = Passenger Revenue per ASM；CASM = Cost per ASM；ULCC = Ultra Low Cost Carrier（Spirit / Frontier / Allegiant — 多數結構性失敗）；LCC = Low Cost Carrier（LUV / RYAAY — LUV 已轉型 hybrid，RYAAY 純 LCC dominance）；Hybrid network carrier = LCC 升級加 premium / loyalty / 國際的混合模式（LUV 2025-2026 轉型路線）；Wide-body / Narrow-body = 寬體機（A350 / 787 用於 long-haul）/ 窄體機（A320 / 737 用於 narrow-body）；Long-haul = 跨大西洋 / 跨太平洋 / 跨赤道航線。所有 financials = Q4 FY25 + Q1 FY26 actual（截至 2026-04 已公告）。

💡 §0 5 條核心 insight（預告）
1. DAL premium ticket revenue $5.4B Q1 25 vs main cabin 僅差 $41M = 1 年內超越 milestone：這是 airline brand transformation 完成訊號，但市場仍把 DAL 當 cycle 股估值；本 ID 認為 DAL 估值倍數從 9-10x → 12-15x 是 3 年內可達 secular re-rating。
2. Co-brand royalty 重議擴散是「跨 vertical 重定價」事件而非單 sector：MAR Bonvoy +35%（2026）→ DAL Amex（2024-2026 中目標 $10B）→ UAL Chase（已 +12% FY）→ AAL Citi（2026 獨家）→ LUV Chase（2025 amended）→ HLT/H 預期 2027 跟進；payment partner（AXP / JPM / Citi）同時受惠 → multi-vertical alpha。
3. SAVE Chapter 22（5 個月後再次破產）是 ULCC 商業模式失敗終局，不是 isolated event；LUV 2025-2026 全面轉型成 hybrid 是 ULCC-light 模式失敗的另一證據；FRO / JBLU 結構性壓力 likely 2027-2028 再見破產潮；中型 niche airline 結構性消失。
4. UAL 130M+ MileagePlus 會員 + Chase co-brand +12% FY 是 sector 中第二大 alpha，但 UAL 估值 fwd PE ~8-10x 仍接近 cycle 倍數，與 DAL 估值倍數 gap 應該收窄至 1-2x（從目前 2-3x）。
5. LUV $4 EPS guide 2026 vs $0.93 2025 = +330% YoY 是 sector 中最大 turnaround 故事：但這是「轉型 hybrid」結構性升級，不是 cycle 反彈；若達成，LUV 估值倍數應從目前 ~12x → 15-18x（轉型成功 alpha）；下檔有限上檔顯著。

§1 產業定義與邊界

美國航空業在本 ID 的範圍是美國本土 + 國際長途客運航空 + 對應 loyalty / co-brand 信用卡 ecosystem。重點不在 capacity 增減的 cycle，而是 (1) K 形分化（premium cabin vs main cabin）、(2) Loyalty / Co-brand 重定價、(3) ULCC / 中型 niche 結構性死亡 / 整合三大結構性主題。本 ID 與兄弟 ID（嬰兒潮 supercycle / 連鎖旅館 / OTA）互鎖 — 嬰兒潮 ID 是 demand 端 demographic 母題，本 ID 是 airline supply / fee 結構分析。

邊界判斷的關鍵是「是否上市 + 主要 service 美國市場 OR 跨大西洋 / 跨太平洋 long-haul」與「是否有 co-brand 信用卡曝險」兩條軸：DAL / UAL / AAL / LUV — 全部符合（4 大美國 airline）；ALK / JBLU / SAVE / FRO — 中型 / niche 玩家也包含；國際 long-haul（LHA / IAG / AF.PA / SQ / CPA）— in-scope 但不深入；歐洲 LCC（RYAAY / Wizz）— in-scope 比較；亞洲航空（CX / SIA / 中國 4 大）— 簡略；purely cargo airline（FDX / UPS）— out-of-scope；private aviation（Wheels Up）— out-of-scope；regional airline（SkyWest）— 簡略。

§2 美國航空業演進史 + Loyalty 重議曲線 + 歷史類比

美國航空業過去 50 年歷經 4 個結構性轉折：(1) 1978 Deregulation Act → 1995 cycle 1.0（多家航空 / 高破產率）；(2) 1995-2008 hub-and-spoke 整合（多家 chapter 11 / 大整合 — Northwest+Delta / Continental+United / America West+US Airways+American）；(3) 2008-2020 4 大 + ULCC 興起 + COVID 衝擊；(4) 2020-2030 K 形分化 + Loyalty 重定價 + ULCC 死亡。本次循環最特別的是 「premium cabin 永久取代 main cabin + Co-brand royalty 重定價 + ULCC 結構性失敗」三因素同時。

本次循環的關鍵驅動：(a) Boomer + Affluent Millennial 共振推動 premium cabin（DAL Q1 25 premium +14% vs main +1%）；(b) Loyalty / Co-brand 從「airline marketing 工具」轉成「payment platform fee 流」（DAL Amex $8.2B 中期目標 $10B / 占 EBITDA 35%+）；(c) ULCC 商業模式（unbundled / no loyalty / 單一 cabin）已失敗 — SAVE Chapter 22 兩次 / 從 214 → 94 架機 / -56%；LUV 完全轉型 hybrid 確認 ULCC-light 也失敗。本 ID 認為 sector EBITDA 在 2027-2029 進入 sweet spot — premium 完全超越 main + co-brand royalty 全面 step-up + ULCC supply 被消化。

★ 歷史類比（2 個正類比 + 1 個反類比）

正類比 1：2008-2014 美國航空大整合（4 大從 8 家整合而來）。當年 DAL+NWA / UAL+CO / US+AA 三大整合 + LUV+ATA 後，sector 從 fragmented → 4 大寡占；估值倍數從 cycle 4-6x → 8-10x（2014 後）。量化對照：2008-2014 sector EBITDA margin 從 ~5% → 12-14%；本次 K 形分化 + Loyalty 重定價預計 2024-2030 sector EBITDA margin 從 13% → 17-18%（增量 4-5pp）— 比上一個整合 cycle 增幅小，但 fee 流穩定性大幅提升，估值倍數應該繼續 step-up（從 8-10x → 12-15x）。

正類比 2：MAR Bonvoy royalty +35% 重議（2026）。MAR 從 Chase / Amex 重新議價 royalty rate +35% 永久 step-up — 詳見 ID_HotelChains。量化對照：MAR 共品牌 royalty 從 ~$800M-1B → $1.1-1.4B (+30-40%) 一年；DAL Amex 已實現類似動作（2024 → 2025 +11%）但中目標 +20%（$8.2B → $10B）；UAL Chase / AAL Citi / LUV Chase 都將跟進類似重議。規模對照：airline mile 比 hotel chain 總規模大 2-3x（4 大 airline co-brand $17-19B 2026E vs 4 大 hotel chain co-brand ~$5-7B），意味 airline royalty 重議的絕對規模更大。

反類比：2018-2020 ULCC 興起（SAVE / FRO / Allegiant 短期 alpha）。當年市場曾期望 ULCC（unbundled fare + no loyalty + single cabin）將永久搶走 LCC + 4 大 airline 的部分客源，2018 SAVE / FRO / Allegiant 估值 P/E 達 15-18x。實際結果是 2020 COVID 衝擊後永遠沒有真正 recover；2024-2025 SAVE Chapter 22 / FRO 高 leverage / Allegiant 邊緣化。反類比警示：「low cost = 永遠 winner」是錯誤觀念；當基本服務不夠好（無 loyalty / 無 premium / single cabin），客戶在景氣好時轉向 4 大 airline。多條件對照：當年 ULCC 假設 (a) 客戶永遠選最低價，(b) loyalty 不重要，(c) cycle 性復甦會持續。實際 (a) 客戶在景氣好時選 brand + premium（K 形分化），(b) loyalty 是 sticky moat，(c) ULCC 在景氣弱時受擠壓更嚴重。本次 LUV 完全轉型成 hybrid + SAVE Chapter 22 = ULCC 模式失敗確認。

💡 §2 Insight
1. 本 thesis 的時間 bound 是 2027-2029（sector EBITDA secular peak）+ 2030（demographic peak 開始降溫）：與兄弟 ID_BoomerTravelSupercycle 共用同一時間框架；2026-2028 是 buy-and-hold 加速期。
2. ULCC 反類比是教科書級警示 — 「low cost = 永遠 winner」是錯誤觀念：airline 投資人應該完全 avoid ULCC（SAVE / FRO / Allegiant），即使估值看似便宜；JBLU 中端 niche 同樣結構性風險，需要 turnaround 確認。

§3 4 大 + Loyalty + ULCC 三層分化

美國航空業現在分三層：(A) DAL 一強（領先 12-18 個月）+ (B) UAL/AAL/LUV 跟進（各自策略路線）+ (C) ULCC / 中型 niche 結構性死亡（SAVE/FRO/JBLU）。本章對照三層 winner/loser。

第 A 層：DAL 一強領先

DAL 是美國 airline sector 結構性領先者，三引擎：(1) Premium cabin 領先 — Q1 25 premium revenue +14% vs main +1% / premium ticket revenue $5.4B vs main 僅差 $41M / brand transformation 完成；(2) Amex co-brand 規模王 — $8.2B 2025（+11%）+ 1M 新卡 + 1/3 SkyMiles 持卡 + 中目標 $10B；(3) Loyalty / FCF 雙引擎 — FY25 loyalty revenue +14% / FCF $5B+ / 中期 $10B+ co-brand 目標。估值倍數：當前 ~9x fwd PE，本 ID 認為應該 12-15x（payment platform 重定價）。催化：2026 Amex co-brand 持續加速 + premium 全面超越 main cabin + DAL Bastian 「premium 是 brand」commit。

§9.5 Kill Scenario / Steel-man 反方

反方論證 #1：Co-brand 信用卡監管收緊（Durbin Amendment 擴大）

路徑：2026-2028 美國國會通過 Durbin Amendment 擴大 / Credit Card Competition Act → 信用卡 interchange fee cap → airline / hotel co-brand royalty 結構性壓縮 -20-30%。當前證據：(a) Durbin Amendment 已存在 debit card；(b) Senator Durbin 多次 push 擴大到信用卡；但 (c) 信用卡業遊說極強 + 共和黨控制 Congress 不利此立法。量化影響：DAL Amex co-brand $8.2B → $5.5-6.5B (-20-30%) / UAL / AAL 同樣 / 估值倍數 -2-3x。falsification 事件：Credit Card Competition Act 立法進度。最壞情境 drawdown：DAL -25-30% / AXP -15-20%。

反方論證 #2：嬰兒潮 demographic 提前降溫（2027 而非 2030 peak）

路徑：兄弟 ID_BoomerTravelSupercycle 反方論證 #2 同步發生 → premium / 國際 long-haul demand 提前降溫；DAL premium revenue growth 從 +14% → +5%；sector EBITDA peak 提前。當前證據：見 ID_BoomerTravelSupercycle §9.5。量化影響：4 大 airline EBITDA -10-15% vs base case / 估值倍數 -1-2x。falsification 事件：見兄弟 ID。最壞情境 drawdown：basket -15-20%。

反方論證 #3：油價 $120+/桶 持續 — fuel cost 大幅壓制 airline margin

路徑：2026-2027 中東地緣（伊朗 / 以色列 / 葉門）失控 / OPEC 配額大幅減產 → 油價 $120-150/桶 持續 6-12 個月 → 4 大 airline 燃料 cost +30-40% / EBITDA margin -3-5pp / ULCC（SAVE / FRO）加速死亡。當前證據：(a) 中東持續地緣動盪；(b) OPEC 配額紀律 — 但 (c) 美國 shale + Saudi 增產潛力。量化影響：4 大 airline FY27 EBITDA -15-20% / 估值倍數短期 -1-2x。falsification 事件：油價走勢 / 中東地緣事件。最壞情境 drawdown：basket -15-25%。

💡 §9 + §9.5 Insight
1. 3 條反方論證中，#3（油價）是 base rate 最高（30-40%）但短期 cycle；#1（Durbin）是 long-tail high-impact（10-15% 5y）；#2（demographic）依賴兄弟 ID。
2. 對沖建議：basket 中保留部分 fuel-hedged airline（DAL 領先 hedge）+ 完全 avoid ULCC（油價衝擊放大）+ AXP/JPM 對沖 Durbin 部分風險（cross-vertical）。
