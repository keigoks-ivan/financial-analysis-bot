航空 K 形分化 × Loyalty 重定價
利潤正從賣座位轉向賣里程

市場用「航空是 commodity cycle 股、fwd PE 8–10x 才合理」在估值整個產業，焦點錯了。本輪美國航空的轉折不是又一次的 capacity 週期，而是一場利潤結構的相變——利潤從「賣座位」（受油價、capacity、票價戰擺布的 commodity）轉向「賣里程」（信用卡 co-brand royalty 這種接近 payment rail 的穩定 fee 流）。Delta Q1'26 premium 票收 $5.36B 已逼平 main cabin $5.40B、Q4'25 更已首度超越，這不是循環反彈，是百年品牌轉型完成的訊號。

同一時間，產業底部正在崩解：曾被視為「永遠的低成本贏家」的 Spirit（SAVE）在 2026-05-02 全面停飛清算、~17,000 人失業，ULCC（單艙、無 loyalty、unbundled）商業模式被宣告死亡；Southwest（LUV）則用 FY26 EPS guide ≥$4（vs 2025 $0.93）證明只有轉型成 hybrid 才能活。產業內部的 K 形分化——上半截 premium + loyalty 起飛、下半截 ULCC 死亡——同時發生。

所以本報告不站隊「航空股便宜 / 貴 yes-no」。誰會贏其實很清楚（規模 + premium + co-brand 三位一體的 DAL，加轉型兌現的 LUV）；真正的分歧在宏觀——這份結構性利潤改寫，會不會被一次消費走弱（壓 premium leisure）+ 油價走高（夏季 ~$3.40/加侖、+31%）的 cycle 逆風蓋過。判斷重心放在「結構利潤改寫的確定性，對上 cycle 逆風的脆弱度」。

PART I · DECISION SUMMARY

決策摘要層

三句話看完利潤結構改寫的現狀、走向與行動，後接 PM 級監測點。這一層是報告的全部重點——下面各節只是把這一層的每個判斷補上證據。

現在 / NOW

K 形分化已完成相變：DAL Q1'26 premium 票收 $5.36B 與 main $5.40B 幾乎並列、Q4'25 premium 已首超 main；Amex remuneration 季 >$2B（+10%）、FY25 $8.4B、契約朝 $10B（2029）；UAL premium RASM +8.9%（高 main 4pt）、loyalty +13%；Spirit 2026-05-02 全面停飛＝ULCC 結構性死亡確認。

未來 / NEXT

未來 3–5 年航空利潤池由座位轉向 loyalty fee——4 大 co-brand royalty 從 ~$18B 朝 2030 ~$30B+（與 capacity 脫鉤）；DAL 估值應從 cycle 9–10x 朝 payment-platform 12–15x re-rate。最大裂縫是宏觀：消費走弱壓 premium leisure ＋ 夏季油價 ~$3.40（+31%）壓利潤 ＋ Chase/Citi 重議結果未定。

行動 / ACTION

結構受益最純的是 co-brand 規模王 DAL 與轉型兌現的 LUV；UAL 為第二大 loyalty alpha 但估值已修；AAL 債務 + premium 落後、ULCC（已死的 Spirit / 高 leverage 的 Frontier）非 long。發卡行 AXP / JPM / Citi 為跨 vertical 鏡像受益。

The Key Call · 本報告最重要的一個判斷

市場把美國航空當「commodity cycle 股」在估值，真正的非共識是——本輪是一場利潤結構的相變：賺錢的引擎正從「賣座位」（被油價 / capacity / 票價戰擺布）換成「賣里程」（信用卡 co-brand royalty 這種接近 V/MA payment rail 的穩定 fee 流）。Delta 一家 Amex co-brand 一年收 $8.4B、朝 $10B，premium 票收已逼平百年來最大的 main cabin——這個 fee 流的特性根本不是 airline cycle。分歧不在「誰會贏」（規模 + premium + co-brand 的 DAL 很清楚），而在「這份結構利潤改寫，能不能扛得住一次宏觀 + 油價的 cycle 逆風」——這把焦點從「航空便不便宜」移到「結構紅利對上週期脆弱度」。

三道裂縫具體是：①宏觀消費走弱——premium leisure 是本輪的成長引擎，但它也最敏感於 affluent 消費信心；若高端消費降溫，premium 溢價（DAL 每座位比同業貴約 20%）會先收斂；②油價——jet fuel 從 Q4'25 ~$2.60/加侖升到夏季 ~$3.40（+31%），雖然目前 disciplined capacity 讓票價能 pass-through，但這是「需求得撐得住」的前提，需求一軟、高油價立刻吃掉利潤（AAL 已因高油價砍 6 條夏季國內航線）；③co-brand 重議結果未定——DAL Amex 已重定價且朝 $10B，但 UAL（Chase）目前只能改「持卡差異化」、新約還在談，AAL（Citi）剛換約 remuneration 年增僅 ~10%——重定價的幅度與時點仍有變數。盯這三條，比盯任何單季 RASM beat 都更早看到拐點。

PART II · HISTORICAL ANALOGS

兩個歷史類比，框住 bear 的天花板

歷史不必重演，但兩個對照各給一個可操作的量化教訓：2018–20 ULCC 泡沫框住「low cost 不等於永遠贏」，MAR Bonvoy royalty +35% 給出「co-brand 重定價的傳導幅度與時滯」。

反類比：2018–2020 ULCC 興起（SAVE / FRO / Allegiant 短期 alpha）——「low cost = 永遠 winner」是錯誤觀念。當年市場曾期望 ULCC（unbundled fare + no loyalty + single cabin）永久搶走 LCC 與 4 大的客源，2018 年 Spirit / Frontier 估值 P/E 達 15–18x。實際結局是 2020 COVID 後永遠沒有真正 recover——Spirit 在 2024–25 兩次破產，最終於 2026-05-02 全面停飛清算、~17,000 人失業、拍賣 LGA slots 與 loyalty program。量化警示：當年 ULCC 假設 (a) 客戶永遠選最低價、(b) loyalty 不重要、(c) cycle 性復甦會持續；實際 (a) 客戶在景氣好時選 brand + premium（K 形分化）、(b) loyalty 是 sticky moat（DAL 一家 co-brand 一年 $8.4B 就是證據）、(c) ULCC 在景氣弱時受擠壓更嚴重。LUV 完全轉型成 hybrid + Spirit 死亡＝ULCC 模式失敗的雙重確認。這把 bear 的天花板框得很清楚：押「ULCC 觸底反彈」是押一個已被證偽的模式。

正類比：MAR Bonvoy co-brand royalty +35% 重定價（2026）——最可操作的傳導幅度錨。Marriott 在 2026 年從 Chase / Amex 重新議價 co-brand royalty rate +35% 永久 step-up（詳見 ID_HotelChains、ID_CobrandCard）。量化對照：MAR 共品牌 royalty 從 ~$800M–1B 一年跳到 $1.1–1.4B（+30–40%）。航空是「下一個 vertical 重定價」，而且絕對規模更大——4 大 airline co-brand 合計 ~$18B（2026E）約是 4 大 hotel chain（~$5–7B）的 3 倍。傳導已在發生：DAL Amex 已實質重定價（FY25 $8.4B、契約朝 $10B 2029）、AAL 2026-01 換 Citi 獨家發卡（收 Barclays portfolio、10 年延約）、UAL 正改持卡差異化並洽簽新 Chase 約。傳導時滯教訓：co-brand 重議從「換約 / 改 benefit」到「remuneration 數字完整顯影」約有 12–24 個月——DAL 領先 UAL/AAL 約這個身位，所以不能用單季 remuneration 數字論成敗，要看契約軌跡。

PART III · SUPPLY SIDE

供給側：座位是 commodity、里程是準入戰

反直覺但可操作的結論：航空有兩種供給——「座位」（有實體產能、受油價 / capacity / 機隊週期決定的 commodity）與「里程」（無實體產能、由高消費會員基數 × 信用卡夥伴談判力決定的準入戰）。利潤正從前者流向後者。

供給端的結構是「3 家規模 + loyalty 平台（DAL / UAL / AAL）＋ 1 家轉型成功（LUV）＋ 一票結構性出局的 ULCC / 中型 niche」。座位這一端是傳統 commodity——產能由機隊（BA / AIR.PA backlog）、油價、capacity discipline 決定，沒有護城河；真正稀缺、有準入門檻的是里程：要把 co-brand royalty 做到 DAL 那種 $8.4B 規模，需要 (a) 夠大的高消費常客基數、(b) 夠強的對信用卡發卡行的談判力、(c) premium 產品支撐里程的兌換價值——這三者單靠 capacity 砸不出來，是多年品牌 + 網路積累的結果。Spirit 死亡正是因為它只有座位、沒有里程準入門檻。

Inference · 供給
里程準入門檻把規模不足 / 無 premium 的航空結構性關在 fee 流外
前提：co-brand royalty 規模 = 高消費會員基數 × 持卡滲透 × 卡消費 × royalty rate；DAL $8.4B 來自百年品牌 + 多艙 premium + 對 Amex 的談判力 + 高端 cohort 集中——後進者無法靠加 capacity 複製

當利潤引擎從「座位」（人人能加 capacity 的 commodity）換成「里程」（需要高消費常客 + premium 產品 + 發卡行談判力的準入戰），規模王與 premium 領先者（DAL，其次 UAL）自動成為 royalty 重定價的最大受益者；ULCC（單艙、無 loyalty）連門都進不去。座位這端的價格戰由油價與 capacity 決定（無護城河），里程這端由準入門檻決定（有護城河）——利潤池因此持續往「有里程平台」的航空集中。

可證偽條件：若出現「純座位 + 開放里程聯盟」的組合（如某中立里程貨幣讓 ULCC 也能分潤高消費客）重新讓無 premium 航空搶回高價值客、或 DAL/UAL premium RASM 對 main RASM 的 spread 連兩季收斂，則「里程準入門檻」論失效、利潤改寫論降級。

PART IV · DEMAND SIDE

需求側：K 形需求 × co-brand 重定價的雙引擎

需求最不可替代的是「affluent 常客的旅行剛性 + 信用卡夥伴的分潤承諾」——這是本輪航空利潤改寫的兩個引擎，但兩個引擎的脆弱點不同：前者敏感於宏觀消費信心，後者敏感於重議談判。

利潤改寫的需求由兩個引擎共同推動。第一是 K 形需求：嬰兒潮 + affluent millennial 的 wealth transfer 推動 premium leisure（DAL premium +9–14%、UAL premium +13.6%），而 main cabin 的價格敏感客增長平緩——這是「需求型 × 週期」雙重屬性（見下方架橋）。第二是 co-brand 重定價：信用卡發卡行（Amex / Chase / Citi）願意付更高 royalty 換取高消費 cardholder 的卡消費——這是 hotel（MAR +35%）已驗證、正擴散到 airline 的跨 vertical 事件。兩個引擎方向一致，都把航空利潤推向「高端常客 + 里程 fee 流」。

§5 框架架橋 · 為何航空需求不能只用單一框架
航空需求 = 週期型（受宏觀 / 油價 / capacity 擺布）＋ 結構型（K 形 premium + co-brand fee 流）；loyalty 段的稀缺性 = 高消費常客基數 × 信用卡夥伴談判力，不隨 capacity 增減

航空客運整體是週期型需求（GDP-sensitive、與油價 / 票價戰共振），不能單用「結構成長」框架；但本 ID 的核心 loyalty 段是結構型 fee 流——co-brand royalty 的稀缺性來自「高消費常客 + 信用卡夥伴談判力」這個準入門檻，而非座位 capacity。所以本報告對「座位需求」用週期框架（情境含宏觀 / 油價 trigger）、對「里程需求」用結構框架（看 co-brand 重定價軌跡）。兩個框架疊在同一產業上，正是 K 形的本質：上半截（premium + loyalty）結構成長、下半截（main + ULCC）週期擺布甚至出局。

怎麼讀：co-brand royalty 加總（~$18B）約占客運市場的 ~7%，但增速（與 capacity 脫鉤、靠 royalty rate + 卡消費 +10–15%）遠快於座位營收（~中個位數）——這個增速差本身就是「利潤從座位流向里程」的直接量化證據。口徑警示：本 ID 嚴格區分「premium 票收」（座位口徑、含艙等溢價）與「co-brand remuneration」（里程 fee 口徑、與發卡行結算），兩者不可混為「快過」——premium 票收成長是座位需求 K 形，co-brand remuneration 成長是 fee 流重定價，是兩個獨立引擎。

多空為何都對

多頭測的是結構利潤改寫（K 形 premium 起飛、co-brand 重定價、ULCC 死亡騰出 supply），這在 Q1'26 數字裡幾乎沒有爭議、很可能持續；空頭測的是週期脆弱度（宏觀消費走弱壓 premium leisure、夏季油價 +31%、票價已 +20% 的高基期）。兩者可以同時為真：利潤結構確實在改寫（多頭對），但結構紅利不代表能無痛扛過一次宏觀 + 油價的 cycle 逆風（空頭對）。所以本報告把判斷重心放在「結構改寫的確定性，對上 cycle 逆風的脆弱度」，而非「航空會不會 K 形」。

PART V · VERDICT & SCENARIOS

PART VI · TAXONOMY

個股特徵與 conviction tier

本框架只輸出「特徵」與「分類」，買哪一檔由個股 DD 系統與 Pure MA 訊號決定。以下 tier 為產業推論的特徵歸類，提及公司僅為「該特徵的代表案例」，不附帶買賣建議。

CORE 🔴

DAL

利潤改寫論最純的代表：規模 + premium + co-brand 三位一體。Q1'26 premium 票收 $5.36B 逼平 main $5.40B（Q4'25 已首超）、Amex 季 >$2B（+10%）、FY25 $8.4B、契約朝 $10B（2029）；FCF generation 領先同業。特徵是「co-brand 規模王 + premium 領先 12–18M」——本 ID 主張其估值應從 cycle 9–10x 朝 payment-platform 12–15x re-rate，但 cycle 逆風（宏觀 / 油價）是 caveat。conviction：高（結構最純，估值 re-rate 看宏觀配合）
CORE 🔴

LUV

轉型兌現的 turnaround 代表。Q1'26 EPS $0.45（vs -$0.13）、營收 $7.249B（+12.8%）、op margin 4.6%（+8.1pt），所有轉型 initiatives（bag fees + 指定座位 + premium）已到位；FY26 adj EPS guide ≥$4（vs 2025 $0.93）。其 edge 不在 co-brand 規模、而在「從 ULCC-light 升級成 hybrid 的結構性盈利改善」——是 ULCC 模式失敗的反向確認。特徵是「轉型成功 alpha、低基期高彈性」。conviction：高（兌現中，盯 guide 達成）
EDGE 🟡

UAL / AAL / AXP / JPM / C

第二梯隊與跨 vertical 鏡像。UAL：第二大 loyalty alpha（premium RASM +8.9%、loyalty +13%、MileagePlus 130M+），但 Chase 新約待簽、估值已修；AAL：規模王但 premium 落後 + 債務高，Citi 換約 remuneration 年增僅 ~10%。AXP / JPM / C：發卡行端的跨 vertical 鏡像受益（airline + hotel 雙 vertical royalty 重定價），AXP 品質最高但估值偏滿。特徵是「跟進者 + 鏡像受益」。conviction：中 · 視重議兌現
EDGE 🟢

SAVE / FRO / JBLU

利潤改寫的出局端，非 long。SAVE：2026-05-02 全面停飛清算，ULCC 模式終局；FRO：Spirit 退場後最大潛在受益 ULCC，但同樣 single-cabin / 高 leverage 結構性風險未解，押「ULCC 觸底反彈」是押已被證偽的模式；JBLU：中端 niche 受 DAL/UAL 擠壓、失去 Spirit 併購後失去 strategic option。特徵是「無里程準入門檻、cycle 裸露」。conviction：低 · 結構承壓 / 已出局

估值面總結：本主題的核心張力在「結構利潤改寫很清楚、但週期脆弱度未排除」。DAL 是結構最純的受益者（co-brand $8.4B 朝 $10B、premium 逼平 main），本 ID 主張其應從 cycle 9–10x 朝 payment-platform 12–15x re-rate——但這個 re-rate 需要宏觀 / 油價配合，現處 Phase III 中段。LUV 是轉型兌現的高彈性 turnaround（FY26 EPS ≥$4 達成是關鍵）。轉空的條件 = premium-main RASM spread 連兩季收斂 或 夏季油價破 ~$3.50 同時需求走弱（利潤雙殺）。ULCC（Spirit 已死、Frontier 高 leverage）特徵歸類為「出局 / 承壓」，非平台 long——即使估值看似便宜，押的是已被證偽的模式。

The One Line

航空的真正利潤正從「賣座位」轉向賣里程——Delta 一家 co-brand 一年收 $8.4B 朝 $10B、premium 逼平百年 main cabin，而 Spirit 已死；問題不是「誰會贏」，而是「結構紅利能不能扛過一次宏觀 + 油價的 cycle 逆風」。

這份報告刻意不做的三件事

  1. 不告訴你哪一檔該買——CORE / EDGE 是「特徵的代表案例」，買賣由你的個股 DD 與 Pure MA 系統決定。DAL「結構最純」不等於「最佳進場價」（cycle 逆風未排除）。
  2. 不預測股價走勢——只給「進入哪個情境」的觸發訊號（premium-main RASM spread、co-brand remuneration 軌跡、油價 × 需求），不給點位。
  3. 不假裝知道宏觀 / 油價走向——premium leisure 會不會被一次消費走弱 + 油價走高蓋過，目前無定論，已誠實標為 bear case 的核心觸發。
