三句話結論：國防工業是 2026-2031 地緣政治主題最確定性的結構牛市——烏克蘭戰爭消耗、Iran 戰爭 + Venezuela 干預、中國對台威脅 + 北韓、俄羅斯長期對抗，構成 4-5 個同時擴軍的全球需求面；供給面呈現結構性瓶頸（彈藥廠、造船廠、發動機、稀土）難以短期擴張。結果是 2026-2031 國防工業進入 10 年超級週期 — Lockheed Martin、RTX、L3Harris、Northrop、General Dynamics、HII 等美國龍頭 + Rheinmetall、BAE、Leonardo、Saab、MHI / KHI 等歐日夥伴 全線受益。關鍵是「彈藥 / 飛彈 / 造船 / 無人機」四個最緊瓶頸的結構性漲價與產能擴張；投資分層：Tier 1 大型系統整合商（LMT / RTX / NOC）、Tier 2 關鍵部件（GE Aerospace / HII / HEICO）、Tier 3 彈藥 & 無人機新創（AeroVironment / Kratos / Anduril）。

§1 核心判斷

★ 非共識 Thesis 1：2026-2031 是國防工業「超級週期」，不是週期性反彈 市場對國防股的估值仍主要用「事件驅動週期」思維（烏戰後 12-18 個月反彈峰值、後續回落）。本篇觀察這次是結構性週期——(1) 全球同時四個戰線（俄烏 + 中東 + 中台 + 加勒比）；(2) 供給面瓶頸需 5-7 年解除（彈藥廠建設 + 工人培訓）；(3) 需求面是「補庫存 + 擴軍 + 新科技」三疊加（新造 vs 修復 vs 新武器）；(4) 歐洲 NATO 國防支出從 2014 年 GDP 1.5% → 2026 年 2.8%，且 Poland / Baltic / 北歐已到 4%；(5) 日本 + 韓國 + 台灣 + 澳洲 + 菲律賓 Indo-Pacific 擴軍同步。結論：LMT / RTX / LHX 等龍頭 2030 revenue 比 2023 年高 50-70%，EPS 高 2-3 倍（due to operating leverage）。當前 PE 20-25x 是週期初段，2030 目標 PE 25-28x 且 EPS growth 中位數 12-18%/年。
★ 非共識 Thesis 2：「彈藥 / 無人機」是唯一真正稀缺的產能 大型飛機 / 艦艇 / 坦克雖然引人注目，但它們有 5-10 年建造週期，Ukraine 戰爭 + Iran 戰爭消耗量並不實質改變全球庫存（因為本來就少量在戰爭中用）。真正稀缺的是「可以短期內大量消耗、必須快速補充」的產品——(1) 155mm 炮彈（Ukraine 戰爭日消耗 5,000-10,000 發，全 NATO 產能不足）；(2) 精確導引彈藥（Tomahawk、JASSM、LRASM、ATACMS）；(3) 防空飛彈（Patriot PAC-3 / SM-6 / NASAMS）；(4) 無人機（Shahed / Lancet / Anduril / AeroVironment）；(5) 反戰車武器（Javelin / NLAW）。這些「高消耗、低庫存」產品的獲利能力 2026-2031 呈現爆發式成長。General Dynamics (Ordnance)、RTX (Raytheon 飛彈)、Northrop (BAE 合資彈藥 + 無人機)、L3Harris (精確導引)、新創 Anduril / AeroVironment / Shield AI 都是直接受益。投資分層上，這類股比大型整合商（LMT / NOC 本身）EPS 增速高 50-100%。
★ 非共識 Thesis 3：歐洲國防「Rheinmetall 式重估」持續，但進入第二階段 2022-2025 歐洲國防股（Rheinmetall、BAE、Leonardo、Saab）已從烏戰前的估值 1-2 倍漲到 10-20 倍。第一階段是「戰爭啟動 + 補庫存」。2026-2031 進入第二階段：(1) 真正建立「歐洲戰略自主」產能（原本靠美國的 Patriot / HIMARS / F-35 希望部分自製）；(2) 歐盟 EDF（European Defence Fund）$80B 預算 2026-2030；(3) Eurofighter、FCAS（法德西）、GCAP（英日意）三個新一代戰機計劃同步推進；(4) Poland 成為「歐洲 Israel」，國防 GDP 4.7% 下軍工投資爆發；(5) 德國 €100B 特別基金 + 2% GDP 承諾實質執行。Rheinmetall 2026 PE 已 35-40x，但 2030 EPS 若達預期 15-20x 反而合理。這是「歐洲補 30 年國防赤字」的長週期主題。
💡 3 條最高價值 insight
1. 造船業是最被低估的瓶頸：美國海軍要 381 艦但產能只能維持 295 艦；Huntington Ingalls（HII）+ General Dynamics（BIW / Electric Boat）雙寡頭近滿載。日本 Mitsubishi Heavy + Kawasaki Heavy（MHI / KHI）對台海情境需求倍增。韓國 HD Hyundai + Hanwha Ocean 在美國 Navy 造艦合作「Jones Act 例外」可能性增加（2026 H1 討論中）
2. 航太發動機產能是「隱藏護城河」：F135（F-35 引擎，Pratt & Whitney / RTX 獨家）、F110（F-15/16，GE Aerospace 獨家）、Eurojet EJ200（歐洲 Eurofighter），這些發動機產能難以短期擴張（冶金工藝 + 精密加工人才）。GE Aerospace（GE 分拆）+ RTX 是最受益
3. Anduril 是下一個 Palantir：2024 完成 C 輪 $1.5B，2025 完成 $2.5B，估值 $14B → $28B → $40B+（2026 Q1）。做無人系統（水面艇、空中、地面）+ 軟體平台 Lattice。2027 可能 IPO，年營收 $1B+。已拿下美國陸軍 RCV（機器戰車）合約、美海軍 Modular Attack Submarine、台灣 NCSIST 合作。公開可投資是透過 CHIPS & Science Act 相關 ETF / 未上市創投基金

§3 核心分析：四大瓶頸與受益股

§3.1 彈藥產能（155mm / 精確導引）

美國現況

  • 155mm 炮彈：2024 年 40K 發/月（從 2022 年 14K 發/月增長 178%），2026 mid 目標 100K/月。GD Ordnance Systems + American Ordnance + BAE Holston + Scranton Army Ammunition Plant + 新增 UNION Technologies（Dallas）
  • Tomahawk：RTX 產能 130 枚/月，目標 2027 年 250 枚/月。Iran 戰爭 2 月消耗 350-450 枚
  • JASSM / LRASM：Lockheed 產能 500/年，提升至 1,000/年（JASSM-ER）
  • PAC-3 MSE（愛國者攔截彈）：Lockheed 產能 550/年 → 1,500/年（2027 年）
  • SM-6 / SM-3：RTX 產能擴充中，目標加倍（SM-6 2026 年 150 枚 / SM-3 60 枚）
  • Javelin 反戰車：Lockheed / RTX 合資 Javelin Joint Venture，產能 600/月 → 1,000/月
  • Stinger 肩射防空：RTX 重新開啟產線，2027 全面量產

歐洲現況

  • Rheinmetall 獨家建成 Unterlüß 彈藥廠（歐洲最大）
  • Nammo（挪威 + 芬蘭合資）擴產固體火箭推進劑
  • Expal（西班牙）、MBDA（法德英合資）、Nexter（法國）、KMW（德國）各自擴產
  • EDIRPA 歐盟投資計劃 2026-2030 $25B 用於歐洲彈藥產能

亞洲現況

  • 日本 Mitsubishi Heavy + Daikin Industries 開始量產彈藥（原本限制，2023 法律鬆綁）

§8 風險與結論

§8.1 三個最大風險

  1. Data 風險：國防預算行政延遲——美國國會長期延誤預算案（continuing resolution），實際執行可能比批准慢 3-6 個月
  2. 前提風險：2028 美國總統大選後政策續——若政府變動，國防擴軍速度可能調整（升高或降低均有可能）
  3. Tail risk：中國反制導致稀土 / 關鍵部件斷供——國防工業對中國稀土 + 鎵 + 鍺 + 鎢依賴高，若危機發生反制可能限制美國武器生產（詳見子 8）

§8.3 本篇核心結論（3 句話）

1. 2026-2031 是國防工業超級週期，非週期反彈：全球同時四戰線 + 結構性瓶頸 + 補庫存 + 擴軍 + 新科技，LMT / RTX / NOC / GD / HII 等龍頭 2030 revenue +50-70%、EPS +100-200%。

2. Tier 2-3（彈藥 / 無人機）比 Tier 1 EPS 增速高 50-100%：GD Ordnance、RTX Raytheon Missiles、AeroVironment、Anduril / Shield AI 是真正的稀缺資源；AVAV / KTOS / PLTR 等是中小型高 Beta 選項。

3. Iran + Venezuela 的淨效果是結構性加速國防工業牛市：短期彈藥消耗 + 中期 FY27 擴軍 + 長期無人系統驗證。建議 Base case 10-14% 倉位、Stress case 加至 15-20%。
