§1 產業定義與邊界

「GLP-1 治療藍圖」這個主題範圍是 — 全球 incretin-based therapeutics 的整個治療生態系，涵蓋 mono / dual / triple agonist、injectable / oral 雙劑型、T2D / obesity / CV / CKD / MASH / OSA / OA / AD 全 indication 矩陣，以及上游 API 製造、CDMO 填裝、device delivery 三大 picks-and-shovels 環節。本 ID 是 GLP-1 系列的 母題 ID，下游消費者影響另由兩個兄弟 ID 涵蓋（packaged food + restaurant impact），本 ID 不重複下游分析，專注於藥物本身的治療藍圖、玩家格局、定價演進、與第二陣營分化。

關鍵的 boundary call 是「incretin vs 其他 anti-obesity 機制」 — 本 ID 包含 GLP-1（單機制如 semaglutide, orforglipron）、GLP-1+GIP（dual 如 tirzepatide, VK2735, CT-388, MariTide）、GLP-1+GCG（如 mazdutide, survodutide, pemvidutide）、GLP-1+GIP+GCG（triple 如 retatrutide）、GLP-1+amylin（如 CagriSema, amycretin）；out-of-scope 是 (a) 純 amylin（cagrilintide mono）；(b) MC4R agonist；(c) leptin analogue；(d) 手術治療 (bariatric surgery)；(e) phenotype 藥（contrave, qsymia 等）。

★ 灰色地帶（5 個 boundary call）

  • amylin 機制是否屬 GLP-1 範圍？：amylin 是 pancreatic beta-cell 分泌的另一種荷爾蒙，CagriSema = cagrilintide (amylin) + semaglutide combo；本 ID 視 amylin combo 為 incretin-augmenting strategy 列入範圍，但 amylin mono（如 cagrilintide alone）out-of-scope。
  • MariTide 是 GLP-1 agonist 還是 GIPR antagonist？：MariTide 是「GLP-1 agonist + GIPR antagonist」雙向設計（vs tirzepatide 是「GLP-1 agonist + GIP agonist」），本 ID 列入但標註 mechanism 反向；2026-2027 MARITIME-1/2 readout 將驗證 GIPR antagonism 是否實際 differentiated。
  • oral semaglutide (Rybelsus) 算 oral 革命嗎？：Rybelsus 2019 FDA approved 但 dosing 限制嚴格（空腹 30 分鐘 + sodium caprate enhancer + bioavailability < 1%），本 ID 視為 「pre-Foundayo」過渡期產品；Foundayo 是真正第一個 small molecule 無 dosing 限制 oral GLP-1。
  • Hims/Ro/Henry Meds 等 telehealth 算治療生態系嗎？：2024-2025 compounded GLP-1 是 access 主力但 2026-03 Hims+NVO 和解後實質回到 brand 通路；本 ID 列 HIMS 為 🟢 邊緣（從 disruptor 變 distribution partner），telehealth pure-play 不深入。
  • 中國 Mazdutide / Innovent 算 global 玩家還是中國 only？：Mazdutide 在中國 NMPA 2025 obesity + T2D approved，Lilly 持中國 royalty；本 ID 列 Innovent 為 🟡 但標註「中國本土 only」 — 全球 launch 需至少 5+ yr，且 patent landscape 限制 ex-China expansion。
💡 §1 Insight
1. 5 個子領域同時演化是 GLP-1 治療藍圖獨特性：與 statin、PPI、PD-1 等過去 wave 不同，GLP-1 同時在 mechanism (mono → triple)、formulation (injectable → oral)、indication (T2D → 6+ disease) 三維平行創新；這意味 LLY/NVO 不是單一藥物 LOE 風險而是組合 portfolio defense，傳統 patent cliff 模型低估其韌性。
2. 「灰色地帶」#1 (amylin) 與 #2 (MariTide GIPR antagonism) 揭示 differentiation 競賽：第二陣營必須找出 mono/dual/triple 之外的 mechanism niche 才能 escape commodity pricing；CagriSema (amylin 加成) + MariTide (GIPR 反向) + ALT pemvidutide (1:1 GLP-1/GCG balanced) 是三個最 differentiated 路徑，其他 me-too dual agonist 將被 LLY/NVO blanket。

§2 GLP-1 技術演進 + S 曲線 + 歷史類比

GLP-1 治療演進不是 sudden event 而是 30 年累積過程，目前處於 S 曲線陡峭段中段：(1) Phase I 機制發現（1986-2005）— 1986 GLP-1 由 Joel Habener 發現，1996 exenatide 從 Gila monster saliva 分離 (Eli Lilly + Amylin)，2005 exenatide (Byetta) FDA approved 為 first-in-class；(2) Phase II 第一代商業化（2005-2019）— liraglutide (Victoza 2010) → semaglutide (Ozempic 2017)，每週注射 + T2D only；(3) Phase III 第二代爆發（2020-2025）— Wegovy 2021 first obesity-only / Mounjaro 2022 dual agonist / SELECT/FLOW outcome trial 證 CV+CKD 效益 / 2024 Wegovy CV 適應症 / 2025 ESSENCE MASH / 11M user Q2 2025；(4) Phase IV 第三代切換（2026-2030）— Foundayo oral 2026-04-01 / triple agonist retatrutide 2026-2027 / amycretin 2027-2028 / Medicare BALANCE 第一次 obesity coverage。

當前最關鍵的 inflection 是 三個並行切換：(a) mechanism 從 mono → dual → triple，每代 weight loss efficacy 階梯式提升 (semaglutide 13.7% → tirzepatide 20.2% → retatrutide 28.7%)；(b) formulation 從 injectable → oral small molecule，access barrier 大幅下降；(c) indication 從 T2D → obesity → 6+ comorbid disease (CV/CKD/MASH/OSA/OA)，每 indication 開新 reimbursement source。三切換同時發生意味 TAM 擴張不是單純 user growth (11M → 25M) 而是 user × indication × spending intensity 三維乘積。

歷史類比（2 正 + 2 反）

正類比 1：Statin 1987-2010 cycle（高度可比）。Mevacor 1987 first-in-class → Lipitor 1996 best-in-class → Crestor 2003 differentiated → 2010 Lipitor LOE → 2014 PCSK9 (Repatha/Praluent) 第二代。Lipitor 2006 peak revenue $13B，LOE 後從 $13B → $2B（5y -85%），但 Pfizer total cardiovascular franchise 仍維持 $30B+ via Lipitor Plus + Caduet + 替代產品。本次相同點：Ozempic 類比 Lipitor (best-in-class first generation)；tirzepatide 類比 Crestor (differentiated 2nd-gen)；retatrutide 類比 PCSK9 (3rd-gen 大幅差異化)。本次不同點：(a) Lipitor 是純 LDL biomarker，GLP-1 涵蓋 weight + glucose + CV + kidney + MASH 多 indication 形成 portfolio moat；(b) injection 形態 + cartridge 供應壁壘比 oral pill 大；(c) US patent cliff 不同步 (semaglutide 2031 vs tirzepatide 2036+) → LLY/NVO 不會同時 LOE。類比指向：LOE 衝擊比 Lipitor (-85%) 溫和（本 ID 估計 -20% 5y）。

正類比 2：HIV antiretroviral 1996-2015 cycle（mechanism 演進類比）。1996 protease inhibitor 三聯療法 (HAART) breakthrough → 2006 Atripla single-tablet → 2012 Stribild → 2018 Biktarvy 簡化 + 高效。Gilead 從 NRTI 單機制 → integrase inhibitor 三聯 → single-tablet regimen 持續創新，30 年維持 HIV market 60%+ share，never face true LOE crisis。本次相同點：mechanism evolution (mono → dual → triple) + formulation evolution (multi-pill → single tablet) 同時發生，類比 GLP-1 mono → dual → triple + injection → oral 雙演化。類比指向：LLY/NVO 透過持續 mechanism + formulation 創新可維持 dominant share 到 2040+。

反類比 1：Insulin 1922-2024 cycle（biosimilar 風險）。Insulin 100+ 年沒有真正 generic disruption（biosimilar 占率僅 15-20% 美國），但 2018-2024 Mark Cuban 等 transparent pricing platform + Senate hearing pressure 導致 NVO/LLY/Sanofi insulin 美國 net price 5y -55%。本次相同點：biosimilar 規模化困難（peptide 製造 barrier），但 political pressure 隨 user 規模擴大而加劇。本次不同點：GLP-1 user 25M (2030E) 比 insulin user 7M 大 3-4x，political optical pressure 將更強；但 GLP-1 portfolio 多藥物 + 多 indication 比 insulin commodity 化更難。類比指向：MFN $245 + TrumpRx $350 已開啟類似 insulin 軌跡，但 portfolio 複雜度將延緩衝擊節奏。

反類比 2：bariatric surgery 1995-2015 cycle（過度樂觀後失望）。1995-2010 gastric bypass 從 50K/yr → 220K/yr peak (US)，被預期成為「mass obesity solution」；但 long-term complication + body image issue + reversal 需求導致 2015-2024 從 220K/yr → 130K/yr 衰退。本次相同點：超高速 user growth + mainstream narrative pressure + 「solution to chronic disease」expectation → 過度承諾風險。本次不同點：GLP-1 是 reversible therapy（停藥即停效）而非 surgical irreversible，risk profile 完全不同。類比指向：long-term GLP-1 discontinuation rate (現報告 50%+ 1 yr) + muscle mass loss + 反彈 weight 是被低估風險，但 retatrutide / triple agonist + indication expansion 已部分 address。

💡 §2 Insight
1. S 曲線位置在中點意味 2026-2030 是 user 倍數加成期：當前 11M → 25M (2030E) 是 2.3x growth (CAGR 18%)，比 statin 同階段 (1995 中點 → 2005 peak, CAGR 11%) 快 60%；LLY/NVO 不僅 share 競爭，更是 absolute volume 爆發期 — 這是 valuation 必須以 user growth 為主要 driver 而非單純 share 模型的原因。
2. Statin 類比給 LOE 衝擊「上限」框架（-85%），但 portfolio + indication moat 顯示實際將更接近 -20%：tirzepatide 2036+ patent + retatrutide / orforglipron / amycretin 第二/三代 + CV/CKD/MASH 各別 biosimilar dossier 障礙 + cartridge 供應壁壘 = 5 重 LOE defense；共識預期 -35-50% revenue erosion 過度悲觀。
3. 反類比 #1 (insulin) + #2 (bariatric) 是 GLP-1 sentiment 主要風險源：insulin 軌跡已啟動 (MFN $245 = 類似 insulin 政治壓力)，bariatric 反彈是 long-term tolerability 未證明 — 兩個風險在 2027-2030 將被 stress test，是 §13 Falsification 的核心觀察 metric。

§3 核心技術棧拆解

GLP-1 治療藍圖的核心技術棧分 5 層，每層都有獨立的 IP / scale / yield 壁壘，決定 winner / loser 分布：(1) 分子設計（peptide 結構優化 vs small molecule 仿生）；(2) 受體 selectivity（mono / dual / triple agonist + biased agonism + GIPR antagonism）；(3) 製造 scale（solid-phase peptide synthesis 大規模 + biased agonist 化學）；(4) delivery formulation（注射穩定性 + oral bioavailability + 半衰期延長）；(5) device integration（pen + autoinjector + cartridge）。LLY/NVO 在 1-3 + 5 全層佔優，第二陣營必須在 1 個或多個 vector 找 differentiation。

當前最 active 的子技術競爭是 (a) oral small molecule GLP-1 — orforglipron (Lilly, 2018 Chugai 授權) 領先；VK2735 oral (Viking) Phase 3 啟動 Q3 26；CT-996 (Roche, biased agonist) Phase 1 完成；oral amycretin (Novo) Phase 3；(b) triple agonist — retatrutide (Lilly) 唯一 Phase 3，目前無第二玩家；(c) amylin combo — CagriSema NVO NDA filed + amycretin Phase 3 Q1 26；(d) GIPR antagonism vs agonism — MariTide (Amgen) 是反向設計，是否真有 differentiation 待 MARITIME-1 readout。

關鍵子技術 1：Oral GLP-1 small molecule vs peptide oral

GLP-1 的 oral 化是 30 年技術瓶頸 — 天然 GLP-1 peptide 在胃腸道被快速降解，bioavailability < 0.1%。NVO 2019 Rybelsus 用 sodium caprate (SNAC) absorption enhancer 突破至 ~1% bioavailability，但需空腹 30 分鐘 + 高劑量補償 (14mg vs Ozempic 1mg subc)。真正革命是 small molecule non-peptide — Lilly 2018 從 Chugai 授權 orforglipron (LY3502970) 是第一個 small molecule GLP-1 receptor agonist，bioavailability 高且無 dosing 限制。Foundayo 2026-04-01 FDA 50 天核准（CNPV 第 5 個批准 / since 2002 最快 NME）標誌 oral 革命真正開始。

當前 oral GLP-1 玩家進度：Foundayo (Lilly) 已 FDA approved 2026-04-01，ATTAIN-1 36mg dose 12.4% 體重下降 72 週；VK2735 oral (Viking) Phase 2 12.2% 13 週，Phase 3 啟動 Q3 26；CT-996 (Roche) oral biased agonist，Phase 1 完成 2024，Phase 2 進行；oral amycretin (Novo) Phase 3 Q1 26 啟動；oral semaglutide 高劑量 (Novo) 25mg + 50mg dose Phase 3 完成。Foundayo 領先 1.5-2 年，但 VK2735 oral 雙劑型 (subc + oral 同時 advance) 是 SMID-cap 唯一玩法。

關鍵子技術 2：Triple agonist (retatrutide) vs Dual (tirzepatide)

Retatrutide 是 GLP-1 + GIP + glucagon 三激動劑，由 Lilly internal R&D 開發。Glucagon 受體激動表面看似矛盾（glucagon 升血糖），但低劑量 glucagon 加成可：(a) 增加 energy expenditure (BMR 提升 5-8%)；(b) 加速肝糖分解避免脂肪堆積；(c) 與 GLP-1 食慾抑制互補形成 synergy。TRIUMPH-4 12mg 28.7% 68 週體重下降是迄今最大 weight loss data，但 dysesthesia (感覺異常) 在 12mg 達 20.9% (vs placebo 0.7%)，為 safety warning。

retatrutide vs tirzepatide 比較：weight loss 28.7% vs 20.2% (SURMOUNT-5 head-to-head 中 tirzepatide vs semaglutide)，差距是 +8.5pp 即 +42% relative；但 retatrutide tolerability 較差 (dysesthesia + GI 副作用)。Lilly 策略：retatrutide 將定位為「max-efficacy obesity 治療」(BMI > 35) + OA / OSA / cardiovascular 重症群組；tirzepatide 維持 mainstream T2D + obesity；orforglipron 走 mass-market oral；形成 3-tier portfolio (oral entry → injectable mainstream → triple max-efficacy)。NVO 同時用 amycretin + CagriSema 構建 mirror portfolio。

💡 §3 Insight
1. 5 層技術棧的 LLY 優勢集中在第 2 + 4 層 (mechanism + delivery)，是真正 differentiation 來源：retatrutide triple agonist 唯一 Phase 3 + Foundayo 唯一 small molecule oral approved，這 2 個技術領先 NVO 約 18-24 個月；NVO 的 反擊是 amycretin + CagriSema 但 mechanism 差異性弱於 triple，是 LLY 2026-2028 share 進一步擴大的根本原因。
2. Picks-and-shovels (第 3 + 5 層) 的 4 寡占結構被嚴重低估：borosilicate cartridge 全球僅 STVN / GXI / SCHOTT Pharma / Nipro 4 家有 GMP commercial 規模，新進入需 4-5 yr 認證 + GMP build-out + customer qualification；CTLT 已被 NVO Holdings 私有化（2024-12 $16.5B）後上市 picks 進一步縮減 → 4 寡占 → 3 listed (STVN/GXI/SCHOTT)；SCHOTT $371M Wilson NC 廠 2027 投產 GLP-1 cartridge tripling 是 ASML EUV 2017-2020 類比，capacity 是 binding constraint。

§7 Unit Economics + ASP 動態

GLP-1 unit economics 分三層：(a) list price — Wegovy $1,349/月 / Mounjaro $1,069/月 / Ozempic $968/月（2025 list before MFN）；(b) net price after rebate — Q3 25 commercial 平均 $700-900/月（list 約 70% rebate to PBM）；(c) 新通路 net price — TrumpRx $350 / Medicare $245 / commercial cash $349 (Wegovy) / $499 (down from)。2026-2028 ASP 演進是本 ID 核心 thesis — blended ASP 將從 2025 ~$700-800 → 2027 ~$450-550 → 2030 ~$370-420（取決於 Medicare 滲透速率與 commercial 維持能力）。

但 ASP 下降不等於 revenue 下降 — user growth 11M (2025) → 25M (2030E) 是 2.3x，可大幅補償 ASP -45% 衝擊：(a) 2025 baseline = 11M × $750 × 12 = $99B；(b) 2030 estimate = 25M × $400 × 12 = $120B (+21%)；(c) 加上 indication expansion (CV/CKD/MASH/OSA/OA) 額外 $15-30B = $135-150B 總 TAM。關鍵是 mix shift 不是 absolute decline：Medicare 25M user 中將有 5-7M (2030E) 採 $245 price → 不是 net price 砍而是新增 Medicare-only volume。
