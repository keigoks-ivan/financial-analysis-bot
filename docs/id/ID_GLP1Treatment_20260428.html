市場習慣用「LLY/NVO 雙寡占永恆」或「2031–32 patent cliff 將毀滅一切」兩種框架看 GLP-1，兩種都漏掉了眼前正在發生的事：這是一個同時在三個維度（機制 mono→triple、劑型注射→口服、適應症 T2D→6 種代謝病）平行爆發的治療領域，而 2026 上半年的密集讀出已把 thesis 從「會不會」推進到「分化成什麼樣」。

關鍵的事實基準已經不同於一季前：retatrutide 三激動劑的 TRIUMPH-1 已於 2026-05 讀出 −28.3%（80 週、obesity 樞紐試驗），是史上 Phase 3 obesity 最大減重；口服 Foundayo（orforglipron）已於 2026-04-01 FDA 核准，自費 $149/月把 access 天花板打破；LLY Q1 26 Mounjaro+Zepbound ~$12.8B、FY26 guide 升到 $82–85B、FY25 總營收 $65.2B 已成為全球營收第一藥廠。這些不是預測，是已落地的 catalyst。

所以本報告不站隊「GLP-1 泡沫 yes/no」。主題仍活、且在加速（多頭對方向）；但贏家的分化、blended ASP 5 年 −45% 是否被 user 倍增抵銷、retatrutide 定價會不會被政治壓力砍——這些才是分歧所在。判斷重心放在「哪些讀出已把賭注實現、哪些還是 binary、現價 price 了多少」。

PART I · DECISION SUMMARY

決策摘要層

三句話看完 GLP-1 治療藍圖的現狀、走向與行動，後接以讀出日曆為軸的 catalyst 監測點。這一層是報告的全部重點——下面各節只是把這層的每個判斷補上證據。

現在 / NOW

S 曲線陡峭段、雙寡占且 LLY 持續拉開——LLY Q1 26 Mounjaro+Zepbound ~$12.8B、FY26 guide 升至 $82–85B、tirzepatide FY25 $36.5B 已超越 sema ~$33B；retatrutide TRIUMPH-1 已讀出 −28.3%、Foundayo 口服已核准。NVO Q1 26 guide 升至 −4~−12% CER 但 priced。

未來 / NEXT

未來 3–5 年由次世代讀出 + pricing 重塑決定贏家分化：CagriSema FDA ~2026-10、amycretin/VANQUISH/MARITIME 2026-27 集中讀出；MFN $245 + Foundayo $149 自費把 oral access 打開。最大裂縫＝blended ASP 5y −45% 能否被 user 倍增抵銷、retatrutide 定價政治壓力。

行動 / ACTION

主題型曝險續抱第一陣營 LLY（最強 portfolio + 估值 priced 但成長扎實）；NVO 是落後者非翻空、靠 CagriSema/amycretin 反擊；次世代 VKTX/ALT/RHHBY 是 differentiated 但 binary。picks-and-shovels STVN 是稀缺替代曝險。買賣由個股 DD + 系統決定。

The Key Call · 本報告最重要的一個判斷

市場用「patent cliff 末日」或「雙寡占永恆」兩種錯框看 GLP-1，真正的非共識是——這個治療領域在三個維度同時平行創新（機制 / 劑型 / 適應症），所以傳統「單分子 LOE」模型嚴重低估其韌性，而 2026 上半的讀出已把分化的勝負手攤開：retatrutide 三激動劑 TRIUMPH-1 已讀出 −28.3%（史上最大），口服 Foundayo 已核准把 access 打開，LLY 對 NVO 的差距非但沒收斂、還在 Q1 26（Mounjaro +125%、Zepbound +79%）進一步拉開。分歧不在「GLP-1 還行不行」（很清楚仍在加速），而在「次世代的 binary 讀出落在誰、現價 price 了多少 peak-sales 假設」。

三道裂縫具體是：①blended ASP 5 年 −45%——MFN $245/月、TrumpRx ~$350、Foundayo 自費 $149，把 net price 結構性下壓，問題是 user 11M→25M（2030E）的量能否抵銷（本 ID 判可抵銷、但 bear 不能）；②retatrutide 定價政治壓力——MFN 只是 first step，2027–28 民主黨選舉週期可能把 GLP-1 pricing 當 affordability 主議題再砍；③次世代 binary 讀出——CagriSema FDA ~2026-10、VANQUISH-2 ~2026-Q3、amycretin Phase 3、MARITIME 2027，任一 miss 都是個股 ±20–40%、但多數不會推翻主題。盯這三條，比盯任何單季處方數據都更早看到拐點。

PART II · HISTORICAL ANALOGS

兩個歷史類比，框住賭注的上下界

歷史不必重演，但兩個對照各給一個量化錨：1987 起的他汀黃金期框住「巨大 user pool × 長坡道」的上界，2014 起的 PD-1 競賽框住「best-in-class 拉開差距、me-too 被邊緣化」的分化下界。

他汀黃金期（1987–2006）——巨大 user pool 撐起 20 年複利，但 LOE 後仍快速崩。1987 年 lovastatin（Mevacor）上市，到 Lipitor（atorvastatin，1996）成為史上最暢銷藥（peak ~$13B/年），整個他汀類在美國 user 從 < 100K 一路爬到 2000 年代中期的 ~30M——18 年 user 約 300 倍、CAGR ~21%。這個對照給 GLP-1 的上界錨：GLP-1 的 user pool 比他汀更大（他汀主限 LDL > 130 患者，GLP-1 涵蓋 BMI > 27 全人口 + T2D + 6 種代謝適應症），所以「user 11M→25M（2030E）、CAGR ~22%」並不激進、有歷史先例支撐。但他汀也給了 bear 的天花板：Lipitor 2011 LOE 後 5 年 net sales −85%——因為它只有「分子 IP + brand」兩重 defense。GLP-1 與他汀的關鍵差異是多重 defense：tirzepatide US 專利到 2036+、次世代（retatrutide / Foundayo / amycretin / CagriSema）梯次到位、適應症 outcome trial 各別 dossier 門檻、cartridge 4 寡占供應壁壘——這 5 重 defense 讓 2031–32 patent cliff 衝擊更可能是溫和的 ≤ 20%，而非他汀式的 −85%。

PD-1 競賽（2014 起跑）——best-in-class 拉開、me-too 被邊緣化的分化錨。2014 年 Keytruda（pembrolizumab）與 Opdivo（nivolumab）幾乎同時上市，市場一度以為 PD-1 會是「人人有獎」的大類。但時滯給了答案：到 2020 年代，Keytruda 靠 KEYNOTE 系列的適應症卡位（NSCLC 一線等）一路把 peak 推到 ~$25B/年、成為新一代藥王，而 Opdivo 落後、後進的數家 me-too PD-1（多數中國授權）只能在價格戰裡分殘羹。這個對照給 GLP-1 第二陣營的下界錨：differentiation 是分化的勝負手，不是「玩家越多越分散」——retatrutide（triple、−28.3%）對 tirzepatide（−20%）就是一次 efficacy 拉開；次世代裡真正 differentiated 的是 VKTX（唯一 SMID 雙劑型）、ALT（pemvidutide MASH niche、肝纖維化已達標）、RHHBY（CT-388 −22.5% + CT-996 oral biased agonist）；而純 me-too dual agonist 會被 LLY/NVO 的 blanket 覆蓋、走向 PD-1 後段班的命運。對投資的含義：第二陣營不能當「一籃子 beta」買，要當「differentiated binary」逐檔看讀出。

PART III · SUPPLY SIDE

供給側：稀缺不是產能，是臨床數據與 cartridge 寡占

反直覺但可操作的結論：GLP-1 不是一般「商品過剩 / 短缺」——美國急性短缺已在 2024–25 解除，真正的稀缺已從「能不能造出藥」轉成「誰握有 best-in-class 臨床數據 + 誰握有 cartridge 寡占產能」。這把競爭從產能戰變成數據護城河戰。

供給端的結構是「第一陣營雙寡占（LLY + NVO，US 處方近 100%）+ 第二陣營 differentiated 挑戰者 + picks-and-shovels device 4 寡占」。美國的 tirzepatide 短缺已於 2024-12-19 解除、semaglutide 於 2025-02-21 解除（FDA 移出 shortage list），隨後 compounding 的 503A/503B enforcement discretion 在 2025 年中陸續終止——換言之「藥造不出來」這個 2021–24 的 binding constraint 已經消失。新的 binding constraint 有兩個：(a) 臨床數據護城河——retatrutide −28.3%、tirzepatide −20%、semaglutide −15% 的階梯，以及 CV/CKD/MASH/OSA/OA 各自的 outcome trial dossier，是後進者要花 5–8 年才能追的稀缺資產；(b) borosilicate cartridge 寡占產能——STVN / GXI / SCHOTT / Nipro 四家對 user 倍增的注射劑型是真實的供應瓶頸（Catalent 已被 Novo Holdings $16.5B 私有化、退出上市曝險）。

Inference · 供給
稀缺＝臨床數據護城河 + 產能爬坡，不是商品過剩 / 短缺框架

GLP-1 是治療型而非商品型市場，所以傳統「供給過剩壓垮價格」不適用——藥能造（短缺解除），但「best-in-class 數據 + outcome dossier + cartridge 產能」這三樣稀缺把競爭門檻撐高。第一陣營靠數據護城河（已讀出的 efficacy + 適應症）鎖住處方近 100%，次世代靠 differentiated 機制（triple / MASH / 雙劑型）卡 niche，device 4 寡占靠 cartridge 稀缺賺 picks-and-shovels 利潤。利潤池因此往「有數據 + 有產能」的玩家集中。

可證偽條件：若 (a) 某 me-too dual agonist 以顯著更低價 + 相當 efficacy 重新搶回大型 payer 的 formulary 一線、或 (b) cartridge 產能在 2027 大幅 ramp（SCHOTT/STVN 擴產 + generic vial 路線）使 device 稀缺溢價消失、或 (c) 一個 best-in-class 非 incretin 機制（amylin mono 等）出現，則「數據 + 產能稀缺」論降級。

PART IV · DEMAND SIDE

需求側：user 倍增 vs ASP −45%的拉鋸

需求最不可替代的是「肥胖 + 代謝病的龐大未滿足醫療需求」——這是結構性、抗景氣的；最能放大需求的是 access 打開（口服核准 + Medicare 覆蓋 + D2C 降價），但同一批 access 工具也把 blended ASP 結構性下壓。量與價在 2026–2030 拉鋸，是本主題的核心張力。

需求由兩股力量共同推動。第一是龐大未滿足醫療需求：美國成人約 12% 在用 GLP-1（任何用途，KFF/Gallup），自報用於減重者約占成人 ~12.4%（≈ 30M+，含品牌 + compounded + 自報），其中兩大藥廠品牌 user 約 10M、潛在 pool > 100M。第二是 access 打開：Foundayo 口服 2026-04 核准（自費 $149/月、商保 $25、Medicare bridge $50）、Wegovy 口服 2026-01 上市、MFN $245/月、TrumpRx D2C ~$350（起始劑量 $199）、Medicare Part D 肥胖試點 ~2026-07——這些都把「想用但用不起」的人拉進來。但同一批工具把 net price 下壓：blended ASP 從 2025 ~$700/月 → 2030E ~$370–400/月（約 −45%）。

怎麼讀：top-down 的 $95B（GS，窄）到 $200B（JPM，寬）差距幾乎全由口徑解釋——GS 偏 obesity-only + 高 price erosion，JPM 含 T2D + incretin 全適應症。bottom-up 雙分子 $70B（FY25）對 GS $95B（2030）留 ~$25B 給 user 成長 + 次世代，量級一致、不需大修。採信 GS $95B 作 2030 obesity 錨、JPM $200B 作 incretin 全口徑上界；關鍵證據是 LLY Q1 26 分子營收 +56% 遠快於 ASP 下壓速度，量在抵銷價。

Inference · 需求
ASP −45% 不等於 revenue −45%——user 倍增 + 適應症加成抵銷，但 bear 不抵銷
前提：blended ASP 2025 ~$700 → 2030 ~$400（−45%）× user 11M→25M（+127%）× 適應症擴張（CV/CKD/MASH/OSA/OA 各 +$3–8B TAM）

治療型需求的量價拉鋸算術：2025 baseline ≈ 11M × $700 × 12 ≈ $92B；2030E ≈ 25M × $400 × 12 ≈ $120B（+30%），再加 indication expansion ~$15–30B → $135–150B 總 TAM。關鍵是 Medicare/MFN 帶來的是新增 volume（incremental）而非替代既有 net price——5–7M Medicare user × $245 是純新增。所以 base case 下 user 倍增 + 適應症加成足以抵銷 ASP −45%，雙巨頭 incretin 營收 2030 仍 +30%。

可證偽條件：① 2027 commercial net price 跌破 $300（vs base $400–500）→ ASP collapse 快過 user 成長，−45% 變 −65%；② 長期 discontinuation rate 升破 60%（現 ~50%）+ muscle loss 成主流 narrative → user 成長放緩至 18–22M（vs 25M base）；③ retatrutide 定價被政治再砍至 $150/月 → blended ASP 2030 跌到 ~$260。任一成立，量價拉鋸翻向 bear。

多空為何都對

多頭測的是需求方向與 user 倍增（龐大未滿足需求、access 打開、適應症擴張），這幾乎沒爭議、很可能持續；空頭測的是blended ASP 軌跡與政治再砍（MFN $245、TrumpRx $350、Foundayo $149 把 net price 下壓，2027–28 可能再砍）。兩者可同時為真：user 確實在倍增（多頭對），但 ASP −45% 是否被量抵銷取決於哪個跑得快（空頭對 pricing 風險）。所以本報告把判斷重心放在「量價拉鋸誰勝、現價 price 了哪個情境」，而非「GLP-1 會不會繼續長」。

估值面總結：本主題的核心張力在「主題很活、但賭注已分叉成 binary」。LLY（P/E ~30–40x）把「最強 portfolio + 高成長」price 進去，現價隱含 peak-sales 假設樂觀（FY26 guide $82–85B 已含 ~25% 成長），reverse 推導下需要 retatrutide + Foundayo 商業化都順——TRIUMPH-1 −28.3% 已讀出降低了其中一個 binary。NVO（估值已 price 落後）的 CagriSema FDA ~2026-10 是「便宜有便宜的理由 vs 轉折催化」的分水嶺。次世代 VKTX/ALT/RHHBY 是高 beta binary，現價多 price 了「讀出成功」、miss 風險未充分反映。現處 Phase III（口服切換 + 次世代讀出年），轉空條件 = 2027 pricing 二次砍 或 多個次世代 binary 同時 miss 或 discontinuation narrative 壓 user。picks-and-shovels（STVN/SCHOTT）是稀缺替代但增速已減速。

The One Line

GLP-1 不是「會不會」的問題了——TRIUMPH-1 已讀出、口服已核准，賭注已從「主題成不成立」移到「次世代的 binary 落在誰、ASP −45% 被量抵銷多少」。

這份報告刻意不做的三件事

  1. 不告訴你哪一檔該買——CORE / NEXT-GEN / PICKS / EDGE 是「特徵的代表案例」，買賣由你的個股 DD 與 Pure MA 系統決定。LLY「最強 portfolio」不等於「最佳進場價」（P/E ~30–40x 已 price 高成長）。
  2. 不預測 binary 讀出的結果——只給「哪個讀出在何時、若 hit/miss 進入哪個情境」的觸發訊號（CagriSema FDA ~2026-10、VANQUISH-2 ~2026-Q3、MARITIME 2027），不給股價點位。
  3. 不假裝知道 2031–32 patent cliff 會多深——5 重 defense 支持「衝擊溫和（≤ 20%）」，但 pricing 政治再砍是真實尾部，已誠實標為 bear 情境的觸發條件而非基準。
