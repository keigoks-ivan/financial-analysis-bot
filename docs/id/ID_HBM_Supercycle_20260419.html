★ 本 ID 核心非共識 Thesis 市場對 HBM 有兩極敘事：多頭「永久成長」vs 空頭「DRAM 週期必崩」。本 ID 認為 HBM 已脫離傳統 DRAM 週期邏輯，進入「CAPEX-constrained secular growth」：① HBM 消耗 3x DRAM wafer（per GB），擴產受限於 leading-edge 製程；② 2022-23 教訓使三家廠商「不願過度擴產」，結構性供給紀律；③ 2026-2027 需求端（NVDA Rubin + Hyperscaler ASIC）全面鎖單。**真正的非共識：Samsung HBM4 翻身比 Street 預期快，Micron 份額從 26% → 35% 的路徑被低估**。
💡 3 條核心 insight（預告）
1. Samsung HBM4 通過 NVIDIA 驗證（2026 Q1）是本 ID 最大 event — Samsung 從 Q3 2025 開始供 NVDA，結束了 SK hynix 的獨占
2. Micron Q2 FY26 actual revenue $23.86B，Q3 guide $33.5B / 81% GM；HBM4 36GB 12-high 已在 2026 Q1 開始 NVIDIA Vera Rubin 量產出貨。2026 HBM 實際 run-rate 可能達 $20B+（非原估 $13B），市場仍低估
3. HBM4 stack 毛利是 LPDDR5 的 5-8x：三家廠商淨利能否進入「結構性高毛利期」取決於 2022-23 紀律能否維持

灰色地帶（★ 邊界爭議）

  • HBM base die（3nm 邏輯 die）算 HBM 還是 foundry？ Yole / TrendForce 計入 HBM TAM，本 ID 遵循。但這意味 TSMC 也 partly 受益於 HBM 升級。
  • AI 伺服器上的 DDR5 算不算本 ID？ 不算 — 但 AI 伺服器擴張確實擴大整體 DRAM 需求，值得交叉參考。
  • Custom HBM（Broadcom / Marvell 協助 Hyperscaler 客製） 新興現象，2027+ 可能需獨立處理。
💡 Insight §1
1. HBM 邊界正從「記憶體產品」擴大到「foundry + logic + 封裝整合」 — HBM4 base die 在 3nm 代工，意味 TSMC 間接吃到 HBM 增長
2. HBM 消耗 3x DRAM wafer capacity per GB，意味 HBM 擴產會擠壓一般 DRAM 產能 → 帶動整體 DRAM 價格上行（傳導至 Micron 非 HBM 業務）
3. Custom HBM 是下一步未知數：若 Hyperscaler 開始客製 HBM，將改變本 ID 競爭格局（類比 ASIC 之於 GPU）

💡 Insight §2
1. HBM 每代週期約 2 年（HBM1→HBM3E 約 9 年、HBM3→HBM4 約 4 年），下一波拐點 HBM5（2029-2030）時程可預測
2. ⚠ 擴產紀律出現裂縫：Micron FY26 capex 升至 $20B（從 $18B）、SK hynix M15X ₩20T 投資、Samsung 產能 +47%。三家都在積極擴產，與 Kill Scenario 反方 1「2028 crash」風險上升。若 thesis ② 要成立，關鍵在需求端能否同步消化
3. HBM 類比 NAND 企業 SSD 崛起：從通用產品變「AI 獨家耗材」，PE 倍數應該 re-rate

3. NVIDIA 驗證時程 = 記憶體股股價 driver：Samsung 2025 Q3 通過驗證 → 股價從底部反彈 40%+

§9.5 Kill Scenario / Steel-man 反方

🟡 反方 1：2028 HBM 供過於求引發 DRAM-style 崩盤 信心：中
  • 事實 1：三家 2026 全面擴產（SK M15X、Samsung +47%、Micron sold out 2026）
  • 事實 2：若 AI capex 2027 減速 + 三家產能齊發 → 類似 2022-23 情景
  • 事實 3：HBM 歷史上第一次由「一個週期」跨到「兩個週期」的產業，no precedent
→ 2028 H1 若 HBM 庫存水位 > 6 週 + ASP 下修 10% → crash 起點
⚠ 證偽：若 2028 Q1 HBM ASP 維持 flat，HBM 庫存 < 6 週
🟡 反方 2：CXL memory pooling 跳過 HBM 信心：低
  • 事實 1：Intel、Samsung 推 CXL 2.0 / 3.0 規格
  • 事實 2：CXL pool + LPDDR5 可作為 HBM 替代（低成本、延遲較高）
  • 事實 3：2028+ AI 推論場景可能採 CXL pool + HBM hybrid
→ 若 CXL 成熟 → 部分推論工作負載遷移，HBM TAM 增速放緩
⚠ 證偽：2028 Q2 有 hyperscaler 公告 CXL 替代 HBM 部署
🟡 反方 3：Custom HBM（hyperscaler 獨家綁定）改寫份額 信心：中
  • 事實 1：Broadcom 與 Samsung / SK 討論 Google TPU 客製 HBM
  • 事實 2：若 Meta / AWS 與某一家獨家鎖，其他兩家損失
  • 事實 3：Custom HBM 利潤更高但研發成本大
→ Custom HBM 可能使市佔劇變；勝者取得結構性溢價
⚠ 證偽：2026 H2 首個 Custom HBM 正式發表

§12 Non-Consensus View

🟡 分歧 1：Samsung HBM4 份額 2027 達 25-30%（vs 共識 20-25%，略偏上緣；2026-04 patch：原預估 35% 已下修，因 Samsung HBM3E 2025 中才過 NVDA qualification，HBM4 追到 35% 需 dual-source 意願 + Samsung 良率同時達標，2027 前同時兌現機率低） 信心：中（2026-04 patch：原標「高」下修為「中」）
  • 共識（T3）：Counterpoint、Yole 預期 SK 保持 50%+ HBM4 領先，Samsung 在 HBM4 僅略提升至 25%
  • 本 ID 觀點（修正）：SK hynix + Micron HBM4 已率先量產（2025 Q4 / 2026 Q1），Samsung HBM4 仍在追趕。但 Samsung 仍有：① 2026 HBM 產能 +47%、② Google TPU HBM3E ~60% 主供、③ HBM4E（2027 H2）有機會翻身。預期 2027 Samsung HBM4/4E 份額仍能達 30-35%（非 40%）
  • 分歧事實：Samsung 2026 HBM 產能 +47%（170K → 250K WPM），遠超 SK M15X +50%
→ Samsung 股價 re-rate 空間顯著；當前 PE 12x（全 DS），若 HBM 業務獨立計應 18x+
⚠ 證偽：若 2027 Q1 Samsung HBM4 發生 yield 問題，被 NVDA 減單
🟡 分歧 2：HBM 已脫離 DRAM 週期，2028 不會 crash 信心：中
  • 共識（T3）：空頭認為 2027-2028 HBM 產能齊發 + AI capex 見頂 = 2022-23 crash 重演
  • 本 ID 觀點：① HBM 消耗 3x wafer / GB，擴產受限 leading-edge 製程；② 三家因 2022-23 教訓紀律擴產；③ 需求端 hyperscaler 1-1.5 年鎖單
  • 分歧事實：SK hynix M15X 計劃 2026 pilot、Samsung +47% 擴產但仍 sold out、Micron 2026 sold out — 供需差距持續
→ 2028 HBM crash 機率 < 30%（vs 共識 50%+），若未發生 → 三家 PE 倍數 re-rate
⚠ 證偽：2028 Q1 HBM 庫存水位 > 8 週 + ASP 下修 10%+
🟡 分歧 3：TSMC 透過 HBM4 base die 成「隱性 HBM 贏家」，市場未 price in 信心：高
  • 共識（T3）：TSMC 被視為 foundry / CoWoS 故事，HBM 不在 TSMC 故事中
  • 本 ID 觀點：HBM4 base die 用 3nm 邏輯製程 = TSMC 獨家代工，這是新收入來源
  • 分歧事實：HBM4 每個 stack 需要 base die，2027 TSMC 每月代工估 5-10K wafers（全新業務）
→ 2027-2028 TSMC 新增 HBM base die rev ~$3-5B；對 2330 thesis 加一條結構性成長動能
⚠ 證偽：若 Samsung Foundry 搶下 HBM base die 訂單 > 30%（當前 ~10%）
