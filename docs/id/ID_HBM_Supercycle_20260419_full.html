§1 產業白話定義 + 歷史脈絡

HBM（High Bandwidth Memory，高頻寬記憶體）是把多顆 DRAM 晶片像積木一樣垂直堆疊、再用上千條極細通道接到 GPU 旁邊的特殊記憶體。AI 加速器要算得快，瓶頸往往不是運算單元，而是「資料餵進去的速度」——HBM 就是解這個「記憶體牆」的零件。誰付錢？NVIDIA、AMD 與自研晶片的雲端巨頭，他們為每顆 GPU 配上百 GB 的 HBM。

白話開場：一顆 AI GPU 的成本裡，HBM 往往是僅次於主晶片的第二大塊。HBM 之所以貴，是因為它要把 8–16 層 DRAM 用 TSV（through-silicon via，穿透矽晶片的垂直導線）疊起來，每 GB 消耗的晶圓產能約是一般 DDR5 的 3 倍——換句話說，多做一顆 HBM，就少做三顆普通記憶體。這條物理定律是整個超級循環的根：HBM 需求一爆，不只 HBM 漲價，連帶把普通 DRAM 的產能也吃掉，推升整個記憶體市場。

邊界界定：本報告聚焦 HBM3E → HBM4 / HBM4E → HBM5 的供需循環與競爭格局，以及 2026 起新增的一條暗線——base die（堆疊最底層、負責對外溝通的邏輯晶片）的製程升級。一般 DDR5 / LPDDR5 / GDDR7、NAND Flash 不在本報告主軸（屬不同產品族），但「HBM 擠壓一般 DRAM 產能」這條傳導鏈會貫穿全文。封裝側（CoWoS / SoIC）另見先進封裝 ID。灰色地帶是 base die：Yole / TrendForce 把它計入 HBM TAM，本報告遵循——這也正是「HBM 邊界從記憶體擴大到 foundry」的起點。

第一次轉折：2022-2023 的 DRAM 大崩盤。pandemic 拉出的需求消化完，疊加三家同步擴產，DRAM 現貨價在約一年內崩掉 ~70%，三家全部由盈轉虧（Micron、SK 單季虧數十億美元，Samsung 一度減產約 50%）。這場崩盤是整個產業的集體創傷，也是「這次三家會克制擴產」這個多頭敘事的心理基礎。

第二次轉折：2023-2024 SK hynix 靠 HBM3E 獨家拿下 NVIDIA H100/H200 訂單，率先把 HBM 從週期性商品變成「AI 獨家耗材」，毛利由谷底翻成全公司最肥的業務。第三次轉折：2025 Q3 Samsung 通過 NVIDIA HBM3E 驗證、重新上船，打破 SK 獨占；到 2026-06-05，NVIDIA 一次認證三家供 Vera Rubin HBM4——HBM 正式進入「三家全員、仍供不應求」的賣方市場巔峰。

歷史類比（兩個有用、一個誤導）。① 2022-23 DRAM crash（有用，框 bear 天花板）：當年空頭對了——crash 真來；但多頭低估了 drawdown 深度（−70% 現貨價、股價腰斬）。本次量化差異：HBM 需求底是 AI 結構需求（非消費循環），且擴產被 foundry／封裝節流，但「三家同步擴產」這個崩盤觸發條件仍在重演風險中。② 2019-21 NAND 企業級 SSD（有用，給上行錨）：NAND 從消費 commodity 被資料中心 QLC 重定價、估值上移；HBM 走更陡的同一條路，且多了 base die 規格化這層 NAND 沒有的護城河。③ 2000 電信光纖（誤導，反向類比警示）：有人把 HBM 比作當年「鋪過頭的暗光纖」，但 HBM 即刻產生對應營收（NVDA DC 已實現）、利用率近滿載，不是「鋪了沒人用」——直接套用會錯判時點。

§2 技術成熟度 + S 曲線

為什麼是現在？因為兩件事同時到位：① AI 訓練／推論把「記憶體牆」推成 GPU 的首要瓶頸，HBM 從選配變必配；② HBM4 把對外通道（IO）從 1024-bit 翻倍到 2048-bit，這個頻寬翻倍的真正 enabler 不是 DRAM 變快，而是 base die 升級到 foundry 邏輯製程——一個過去三年才成熟的工藝跨界。

HBM 的世代節奏約每兩年一代：HBM3（2022，24GB，819 GB/s，H100）→ HBM3E（2024，36GB，~1.2 TB/s，Blackwell，2026 主流約占 66%）→ HBM4（2026，48GB，16-layer，IO 2048-bit，Rubin/MI400，H2 ramp 約占 34%）→ HBM4E（2027 H2，64GB，20-layer，Rubin Ultra）→ HBM5（2028，base die 升 N3P，單顆 CoWoS 容 20 顆）→ HBM5E（2029，24 顆）。TSMC 官方 2026 symposium 把這條路線圖明確化：單顆 CoWoS 內 HBM 總頻寬 2024→2029 五年 +34X，這是供給端第一次給出「能 deliver 多少」的 anchor。

為什麼是現在，技術上的因果鏈：HBM4 的頻寬翻倍被市場簡化成「速度提升」，忽略了真正的 enabler 是 IO 從 1024-bit 翻倍到 2048-bit。每顆 stack 的 base die 上要塞 2x 的 IO routing 與 ESD 保護，DRAM 製程的 metal stack 細不到這個密度——所以 base die 必須升到 foundry 邏輯製程（N12FFC+ 或 4nm）。這就是「HBM 工藝瓶頸從自家 1c DRAM 轉移到 base die yield」的拐點，也是本報告 KEY CALL 的技術根據：HBM4 起，記憶體廠的命運第一次部分綁定在 foundry 身上。

技術成熟度對 entry timing 的含義：HBM4 的供應鏈格局其實延續 HBM3E（同 microbump、同三寡占），真正的競爭重組要等 HBM5（2028 強制 hybrid bonding + base die 升 N3P）。所以 2026-2027 是「領先者收割期」（SK 最受益），2028 才是「格局可能重洗」的窗口——這與裁決的 Phase II→III 轉換時點一致。

§3 供給側：三寡占之上，多了一層 base die foundry tax

HBM 供給故事過去是「SK / Samsung / Micron 三寡占誰擴得快」；2026 起多了一條暗線——base die 跳到 foundry 邏輯製程，TSMC 與 Samsung Foundry 開始從 HBM 增量價值裡切走一塊，且這塊不回流記憶體廠。

估值現在 price 了什麼：price 了「Phase II 續行、2026-27 EPS 創高」，但還沒充分 price「2028 oversupply ~35–45% 機率」。哪個變數動會 de-rate：庫存週數回升 + capex 增速破 +20% 的雙確認——那一刻市場會把估值錨從「峰值 EPS × 高倍數」切換到「正常化 EPS × 低倍數」。三家 PE 多數仍低於 2021 peak，倍數面留有 mean-reversion 空間，但這空間在 Phase III 會反向變成下行空間。

  • 我們說 Y：base die 從 DRAM 製程跳 foundry 邏輯製程（Samsung 4nm 自代、SK/MU 走 TSMC N12FFC+→N3），把 HBM 變成「客製化、foundry 把關」的規格化矽——勝負軸從 DRAM bit 成本移到 base die / IP 設計實力。🔵 [X: base 很可能——Samsung logic die 已漲價 40–50%、SK 考慮 TSMC 3nm，證據已現]
  • Priced-in：市場給三家的估值仍以「記憶體週期股」框（P/B 區間），幾乎沒有 price「base die 價值捕獲」這條——TSMC 的 HBM base die 增量、Samsung 自代的價值留存，都是未被定價的 optionality。推導：若 base die 為 TSMC 貢獻 2027-28 營收 ~1–2%，以 TSMC 體量計算屬「small but real、未入 consensus model」。
→ 推論邏輯：規格化既保護 margin（擴產更難）又重分配價值（漏向 foundry），市場只看到前者沒看到後者。⚠ 證偽：若 HBM4E/HBM5 後 base die 回到記憶體廠自製、foundry 角色淡出，本分歧失效。
🟡分歧 2：2028 HBM oversupply 機率 ~35–45%，高於市場樂觀派的「紀律會守住」信心：中
  • 共識說 X：多頭認為三家因 2022-23 教訓會維持擴產紀律，HBM 已脫離 DRAM 週期、不會崩。🟢 [F: T3 — 「supply discipline」敘事普遍]
  • 我們說 Y：紀律是心理的不是物理的，三家 2026 capex 已同步放大（SK M15X、Samsung +50%、Micron ~$20B）；疊加 hyperscaler capex 增速減速（+77% → 2027 增速放緩），2028 oversupply 機率應上修至 ~35–45%（vs 市場樂觀派隱含的 <30%）。🔵 [X: bear 不太可能但機率不低——需三家齊擴 + capex 急煞同時發生]
  • Priced-in：當前估值（PE 多數低於 2021 peak、股價創高）反映的是「Phase II 續行」，沒有 price 2028 digestion 的尾部風險。推導：若市場開始 price 35–45% 的 2028 oversupply，de-rate 會先於 EPS 反轉發生（週期股戴維斯雙殺前半）——這個分歧「對且未 priced」，最可操作。
→ 推論邏輯：紀律裂縫 + capex 減速是雙重 headwind，市場低估了 tail。⚠ 證偽：若 2027 三家明確減速擴產（capex 不再同步放大）或 capex 增速重新加速，機率回落 <30%。
🟡分歧 3：TSMC 是 HBM「隱性受益者」但非獨家——市場兩端都錯信心：中
  • 共識說 X：一端認為 TSMC 與 HBM 無關（純 foundry/CoWoS 故事）；另一端（舊敘事）誤以為「TSMC 獨家代工所有 HBM base die」。🟢 [F: T3 — 兩種說法都流傳]
  • 我們說 Y：真相在中間——TSMC 為 SK hynix + Micron 代工 base die（N12FFC+ 主流、HBM4E 考慮 N3），是 small-but-real 隱性增量；但 Samsung 用自家 4nm 自代、且 custom HBM logic die 將移 Samsung Foundry 2nm，分流掉一塊。🔵 [X: base 很可能——2027-28 對 TSMC 營收貢獻 ~1–2%，非「獨家」也非「無關」]
  • Priced-in：TSMC 估值由先進製程 + CoWoS 主導，HBM base die 這條 optionality 基本未入 consensus；但因 Samsung 自代分流，這條也不該被過度放大。推導：HBM base die 增量對 2330 是「錦上添花」而非「換 thesis」。
→ 推論邏輯：避免兩個極端——既不忽略也不誇大。⚠ 證偽：若 Samsung Foundry HBM4 base die 全市場份額破 40%（見 §8 F-7），「TSMC 隱性受益」明顯弱化。

Steel-man 反方（最有力的 3 條）

  • 反方 1：2028 DRAM 式崩盤。路徑：三家 2026-27 同步擴產的產能在 2028 齊發 + hyperscaler capex 增速反轉 → 庫存堆積、ASP 崩。當前證據：三家 capex 已同步放大、capex 增速由 +77% 向下。證偽窗口：2028 Q1 HBM 庫存週數 > 8 週 + ASP/GB 下修 10%+。這是最強反方（也是本報告 NC#2 判 AT_RISK 的根據）。
  • 反方 2：CXL memory pooling 跳過 HBM。路徑：Intel/Samsung 推 CXL 2.0/3.0，CXL pool + LPDDR5 在推論場景部分替代 HBM。當前證據：CXL 規格成熟中，但延遲較高、2028+ 才可能規模化。證偽窗口：2028 有 hyperscaler 公告 CXL 替代 HBM 部署。信心低——HBM 在訓練端不可替代。
  • 反方 3：中國 CXMT 國產 HBM 2027-2028 突破。路徑：中國政府補助 $20B+，CXMT 已量產 DDR5、HBM3 開發中，華為 Ascend 已用國產 HBM（規格較低）。當前證據：規格仍落後三巨頭兩代以上。證偽窗口：2027 Q4 CXMT HBM3E 規格 + 良率接近三巨頭 → 內需（約全球 22%）部分自給。信心低-中。

市場最低估的風險：不是上述任一條單獨發生，而是「紀律裂縫 + capex 減速」在 2028 同時發生（反方 1）——因為這兩者高度相關（capex 減速本身會讓三家後悔擴產、但產能已下訂），市場傾向分開評估、低估了它們的聯合機率。

三條分歧裡，分歧 2（2028 機率被低估）最可操作——它「對且未 priced」，且有明確的領先指標（庫存週數、capex 增速）可以提前 2-3 季捕捉 de-rating window。分歧 1（base die 規格化）是最深的結構洞見但 optionality 性質、難 time；分歧 3（TSMC 隱性）是 incremental、非換 thesis。
