💡 Insight §2
1. 本次不像 1999 光纖泡沫：因 GEV backlog 已簽 5 年 / VRT 15B backlog 是客戶確定訂單，非預測
2. 液冷技術 S 曲線 2026 正好從 25% → 50% 採用率，歷史上這階段 stock return 最強
3. 電力供應鏈建設期（7 年）> AI 晶片迭代（2 年）：這個時間差使「賣電器的」比「賣晶片的」有更長 runway

§9.5 Kill Scenario

🟡 反方 1：AI bubble 破裂，hyperscaler 砍 capex 信心：中
  • 事實 1：Hyperscaler FCF 惡化
  • 事實 2：訓練 ROI 不明確
  • 事實 3：類比 1999 光纖
→ 若 2027 hyperscaler capex flat / 下修 → VRT、GEV、ETN 全崩
⚠ 證偽：2027 Q1 任一 hyperscaler 公告 capex 下修 > 15%
🟡 反方 2：SMR 許可延遲，核電 2030 後才 meaningful 信心：高
  • 事實 1：NRC 核准週期 3-5 年
  • 事實 2：美國首個 SMR 延至 2030+
  • 事實 3：NuScale 取消 UAMPS 訂單過（成本）
→ ⚠ 邏輯修正：SMR 延遲實際上「強化」而非「破壞」本 ID thesis 2（gas turbine 是 2030+ 主力）。真正的 kill scenario 應為「SMR 2028 突破性 ramp → gas turbine 需求提前飽和」，不是「SMR 延遲」
🟡 反方 3：效率改善 vs GPU / 演算法使電力需求增速放緩 信心：低
  • 事實 1：Blackwell 每 token 能效 +25x vs H100
  • 事實 2：MoE / quantization 降低運算
→ Jevons 效應可能抵銷，但若效率提升超快，電力需求實際 < 預測

§12 Non-Consensus View

🟡 分歧 1：VRT 是比 NVDA 更安全的 AI pure play 信心：高
  • 共識：市場視 VRT 為「NVDA 衍生」，風險完全 track NVDA
  • 本 ID 觀點：VRT backlog 已簽 12+ 月、cycle 5 年以上；若 AI capex 2027 暫停，VRT 仍有 backlog 持續交付
  • 分歧事實：VRT Q4-25 book-to-bill 2.9x（歷史前所未有的 AI infra 訊號）、Q4 organic orders +252% YoY、15B backlog、EPS +43%、2026-03-23 加入 S&P 500（被動資金流入）— 現金流比 NVDA 更穩（無 inventory risk）
→ VRT 應享 premium PE（PEG < NVDA）；但被低估
⚠ 證偽：2026 Q3 book-to-bill 跌破 1.0
🟡 分歧 2：GEV gas turbine 不是過渡技術，是 2030+ 主力 信心：高
  • 共識：gas turbine 視為「核電 SMR 上線前過渡」，2028-2030 後需求放緩
  • 本 ID 觀點：SMR 許可期 5 年 + 建設期 5 年 = 2032+ 才大規模；gas 需求延續到 2035+
  • 分歧事實：GEV backlog 到 2030 售罄、CEO 稱 2026 末 turbine 賣至 2030
→ GEV gas 業務 runway 比共識長 3-5 年；PE 合理應比共識高 20%+
