💡 3 條核心 insight（預告）
1. Tether 用 50 員工年賺 $10B 是金融史最不對稱事件：2025 net profit $10B+、$141B Treasuries 持倉（17th 大美債持有者）、explore $500B 估值募資 — 這個利潤率（每員工 $200M 利潤）會吸引銀行體系全力反撲，下一個五年最大政治風險不是「stablecoin 不被允許」而是「Tether 自己被監管狙擊」。

§1 產業定義與邊界

「穩定幣 + 支付基建」是加密金融與傳統金融交集處的新興產業，邊界不像半導體那樣清晰，因為它跨四個產業層：① 穩定幣發行（issuer，加密原生）；② 支付軌道（rail，含區塊鏈 + 卡網 + ACH/SEPA/PIX 等傳統軌）；③ 錢包與分發通路（wallet/distribution，含交易所、銀行、fintech）；④ 商戶接入與結算（merchant/acquirer）。本 ID 把「穩定幣作為支付媒介」相關的玩家都納入，但排除「投機交易工具」面向（如 BTC/ETH 純投機、AMM 流動性提供）。

產業核心定義為「以法幣（主要是 USD）1:1 錨定的數位代幣，作為跨地域、跨機構的價值轉移媒介」。2025-07-18 GENIUS Act 簽署後，美國首次有聯邦級法律框架定義「payment stablecoin」，把它從「加密資產」分類正式變成「支付工具」 — 這個分類重塑是本 ID 成立的法律基礎。

灰色地帶（★ 邊界爭議）

  • Tokenized money market funds（如 BlackRock BUIDL $2B+）算不算 stablecoin？ 它們本質是「基金份額 tokenized」而非 1:1 USD peg，但實務上被機構當「生息穩定幣」用；GENIUS Act 把它們列為「digital security」而非 payment stablecoin。本 ID 把 BLK 列入 §11 邊緣（🟢）但不深入。
  • JPM Coin / Goldman GS DAP 算不算？ 屬封閉式 wholesale settlement token，僅機構間流通，不是 payment stablecoin。但它們會擴大「銀行間 tokenized settlement」市場，與本 ID 互補。
  • Stripe Bridge 是 stablecoin orchestration 還是發行商？ 介於兩者之間 — Bridge 自身不發 stablecoin，但提供「issue + manage」基建讓 fintech 發 white-label stablecoin（如 Coinbase USDC、SoFi SoFiUSD 部分）。本 ID 把 Stripe（私）列為核心觀察。
  • USDT 在「美國市場」的定位：GENIUS Act 排除離岸發行商在美國「sale to retail」，但 institutional B2B + DeFi 流動性仍然主要靠 USDT — 法律邊界與商業現實有 gap。本 ID 假設 2026-2028 USDT 在美國繼續被「容忍而非合規」。
💡 Insight §1
1. 「穩定幣 vs 卡網」不是替代關係而是「軌道分工」：消費端日常支付（咖啡 / 訂閱）卡網仍贏（用戶體驗 + chargeback）；機構端 cross-border B2B + 大額 settlement 穩定幣贏（速度 + 成本）；中間段（remittance、freelance payout、merchant settlement）正在重組 — 投資結構應 long V/MA（消費端護城河仍在）+ long 純 play 發行商（B2B 切利潤）。
2. GENIUS Act 創造的「在岸 vs 離岸」分裂是結構性而非過渡性：USDT 不會「升級」變美國合規（離岸利潤 + 隱私偏好太大），CRCL 不會「下降」去搶新興市場流動性（在岸合規護城河是核心），雙頭並存的市場結構至少維持到 2030+。
3. Stripe Bridge 是 2026 年最被低估的 cross-cutting play：私募估值 ~$70-90B，但它同時受益於發行商分層（white-label）、跨境 B2B（Bridge volume 2025 quadrupled）、合規通道（OCC 2026-02 trust bank 條件性核准）；若 2027 IPO 估值將定義整個 fintech valuation 重估。

§2 技術演進 + S 曲線 + 歷史類比

穩定幣的演進有兩條主線：① 從 crypto-native 工具（2014-2022）轉為主流 USD 支付基建（2023-2027）；② 從「離岸無監管」結構（USDT 主導 2014-2024）轉為「在岸合規 + 離岸並存」雙軌結構（2025+ GENIUS Act 後）。2025-07-18 GENIUS Act 簽署是這兩條主線的交會點 — 之前穩定幣是「監管套利產品」，之後穩定幣是「美國金融體系第二棧」。

§3 核心技術棧拆解

穩定幣 + 支付基建技術棧有五個關鍵層：① Reserve management（後台儲備管理）；② On-chain settlement（區塊鏈底層）；③ Fiat on/off ramp（法幣進出）；④ Compliance + AML（合規與反洗錢）；⑤ Distribution + wallet（分發與消費者觸點）。每層都有明確 kingmaker 與卡點，2026 年最關鍵的是 compliance（GENIUS Act 法律邊界）+ fiat ramp（銀行接入）—這兩層決定誰能 scale。

§5 供應鏈 Value Chain

穩定幣 + 支付基建的 value chain 與傳統金融服務不同 — 它同時跨越「加密原生」與「傳統金融」兩個生態，並且利潤池高度集中在 issuer（reserve interest income）+ 銀行（custody + fiat ramp）兩端，中間段（區塊鏈、wallet）多為 infrastructure 角色，少數例外（Coinbase）能透過 50% 分潤協議分到大餅。本節把 value chain 拆為三段（上游 reserve + 銀行 → 中游 issuer + chain → 下游 wallet + merchant），呈現各段毛利與議價權。
