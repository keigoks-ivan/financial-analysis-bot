全球大行 ROE 結構（Global Bank ROE Structure）

發布日：2026-04-28 ｜ 涵蓋 ticker：16 檔（6🔴 / 6🟡 / 4🟢）｜ 🟡 占比：18% ｜ 投資時鐘：Phase III（整合期 + 區域 ROE 分歧確立 + capital markets cyclical recover） ｜ industry-analyst v1.7

Unit Glossary：ROE = Return on Equity（淨利 / 平均股東權益，含 goodwill 與 intangibles）；ROTCE / RoTE = Return on Tangible Common Equity（剔除商譽 + 無形資產，US 銀行業最常用對比指標）；NIM = Net Interest Margin（淨利息收入 / 平均生息資產）；NII = Net Interest Income（絕對值，FY total）；RoCET1 = Return on CET1 capital（UBS 慣用指標，更嚴格分母）；C/I ratio = Cost / Income ratio（成本收入比，DBK 64% target）；Basel III Endgame = 2023 提案要求美國大行 +20% capital，2026 重提為 capital-neutral；SLR = Supplementary Leverage Ratio；GSIB = Global Systemically Important Bank；RWA = Risk-Weighted Assets；TBV = Tangible Book Value；AWM / AUM = Asset & Wealth Management / Assets Under Management；LGFV = Local Government Financing Vehicles（中國地方政府融資平台）；BoJ = Bank of Japan；所有 financials = FY2025 actual（截至 2026-04 已公告，HSBC FY 截至 12/2025；MUFG/SMFG/Mizuho 截至 3/2025 + H1 FY26 中期）。

★ 本 ID 核心非共識 Thesis 市場用 4 種錯誤框架看全球大行：① 「美國 vs 歐洲 ROE 結構性差距會永久存在」；② 「Citi turnaround story 仍會回到 mid-teens ROTCE」；③ 「日本 BoJ 升息對 megabank 是 incremental 利好，不是 multi-decade 結構性 transformation」；④ 「Basel III Endgame 對美國大行是負面 catalyst」。本 ID 認為：① 美歐 ROE 差距已在 2025 收斂 — UBS RoCET1 underlying 13.7% / HSBC ex-notable 17.2% / DBK RoTE 10.3% 全部已超過 cost of capital，與 BAC 16% / WFC 15% 重疊區間，過去「歐洲銀行結構性低 ROE」是 2010-2020 殘留 narrative，2026-2030 估值倍數收斂可期；② Citi 2026 ROTCE 10-11% target 是「too big to fix in one cycle」結構性 cap，Jane Fraser 4 年 turnaround 仍未到 mid-teens 是真正結構訊號 — Citi 估值倍數結構性低於同業 30-40% 是合理而非 mispricing；③ 日本三大行 ROE 從 8% → 11.5% (MUFG) 是 BoJ 升息 multi-decade 結構性 transformation 不是 cyclical bounce，12% mid-term target 仰賴 BoJ 政策利率升至 1%（vs 當前 0.75%），2027-2028 BoJ 進一步升息至 1.0-1.25% 是 base case，日本 megabank 估值仍有 multi-year re-rating 空間；④ Trump capital-neutral Basel III 是美國大行隱性正面 catalyst — 原 +20% capital 變 modest 削減 + SLR + GSIB surcharge 同時下調，意味著 JPM/BAC/GS 等 ROTCE target 17-18% 是 conservative，2027 達 19-20% 機率上升；⑤ Capital One-Discover $35.3B 2025-05-18 closed 是 post-2008 bank M&A 結構轉折點，2026-2027 大型銀行整合潮機率 elevated（next: regional bank consolidation），incumbent 非「太大不能倒」而是「太大要做大」；⑥ HSBC ex-notable RoTE 17.2% 是亞洲 wealth 主導逆襲確認（vs UBS 13.7% / GS 15%），HSBC $1T+ Asia booked wealth + 2026-2028 17% target 是被市場低估的結構性 alpha；⑦ Citi 是結構性 short candidate vs JPM 是結構性 long candidate，相對價值差距持續擴大是合理收斂機會。
💡 §0 5 條核心 insight（預告）
1. 「美歐 ROE 結構性差距已在 2025 大幅收斂」是市場仍未充分認知的事實：UBS RoCET1 underlying 13.7% / HSBC 17.2% ex-notable / DBK 10.3% RoTE 已全部超過 cost of capital；2010-2020「歐洲銀行 ROE 永久低於 CoC」narrative 已不成立。下一個 5 年估值倍數收斂機會 = 歐洲大行對 US peer 從 0.5-0.7x P/B 反向 to 0.8-1.0x。
2. BoJ 0.75% 升息 + 預期 2027-2028 達 1.0-1.25% 對日本 megabank 是 multi-decade transformation：MUFG 12% mid-term ROE target 是基於 BoJ 1% 假設，每多 25bp 政策利率 add ¥35-40B annual NII（per MUFG sensitivity）。日本 megabank 估值仍以 P/B ~0.7-0.9x 交易（vs 美國 1.5-2.0x），2027-2030 multi-year re-rating 空間。
3. Citi ROTCE 5-7% FY25 vs 10-11% 2026 target 仍未 mid-teens 是結構性確認：Jane Fraser 4 年 turnaround 仍未達同業水平 — Citi 業務模型（trading-heavy + 全球 footprint + 消費銀行 weakness）是 multi-cycle 限制，不是單純 cyclical mispricing。「Citi turnaround 仍可期待」是過度樂觀共識。
4. 2025 IB 復甦是 cyclical recover + GS/MS 集中度上升雙因素：Global IB rev $150B + M&A $4.81T (+41%) 是 cyclical 強反彈，但 GS 在 M&A advisory 領先 #2 對手 $250B+（佔全市場 30%+ 集中），MS Q4 +47% 同樣集中度跳升 — 結構性集中是真正 alpha 而非單純 cyclical。
5. Capital One-Discover $35.3B 是 post-2008 bank M&A 結構轉折點：是 first 大規模銀行整合 post-Trump 的 catalyst；2026-2027 regional bank（PNC/USB/TFC/RF）整合潮機率上升 — 是被市場低估的 multi-year structural catalyst。

§1 產業定義與邊界

「全球大行 ROE 結構」這個主題範圍是 — 全球 G-SIB（Global Systemically Important Banks）+ 主要區域 incumbent 的 ROE 結構性差距與其驅動因素。本 ID 涵蓋美國 6 大（JPM / BAC / WFC / C / GS / MS）+ 歐洲 4 大（UBS / HSBC / Barclays / Deutsche Bank）+ 日本 3 大（MUFG / SMFG / Mizuho）+ 中國 4 大（ICBC / CCB / BOC / ABC），同時涵蓋 Capital One-Discover 整合 case 作為 US bank consolidation 範本。

本 ID 的核心問題是：「為何 JPM ROTCE 20% 而 Citi 5-7%、HSBC 17.2% 而 ICBC 結構壓縮、MUFG 11.5% 而 ABC 持續走低？」這個 ROE 結構分歧不是純個別管理問題，而是 (a) 各區域 monetary policy 不同（美 5%+ → 4-4.5%, 歐 2.0-2.5%, 日 0.50→0.75%, 中 LPR 持續下調）；(b) capital markets 集中度（US top-5 控制 60%+ IB rev）；(c) wealth management 規模優勢（MS $9.3T / UBS $7T+ / GS $1.9T）；(d) 監管框架（Basel III Endgame US-EU 不對稱）；(e) 區域信用週期（US fully recovered / EU mid-cycle / 中國房地產 + LGFV 持續壓力）。

★ 灰色地帶（5 個 boundary call）

  • UBS 算 wealth-pure 還是傳統大行？：UBS 整合 Credit Suisse 後重心向 wealth + IB 雙引擎，本 ID 列為 🟡 全業大行因為 RoCET1 13.7% 是傳統大行 metrics + invested assets $7T+ 是 wealth dominant。

  • HSBC 算英國銀行還是亞洲銀行？：HSBC 重心 60%+ 在亞洲（$1T+ wealth booked Asia），本 ID 視為「亞洲 wealth dominant + 英國保留 banking 二元架構」。
  • Capital One 算消費銀行還是 networks 公司？：post-Discover 後 COF 變「closed-loop network bank」hybrid，本 ID 簡略提及作為 US M&A 案例 + post-2008 整合 milestone，深度不及大型 G-SIB。
  • 中國銀行算商業銀行還是政策銀行？：4 大行雖私有化但仍承擔政策職能（LGFV 隱性 bailout、地方政府融資），本 ID 視為政策性商業銀行 hybrid，結構性 ROE 受限。
💡 §1 Insight
1. 全球大行不能單一框架看 — 必須拆 5 個區域 + 5 個業務型：美/歐/日/中四個區域 monetary policy 不同 = ROE 起點不同；商業銀行 / 投行 / wealth / 信用卡 / 政策銀行五個業務型 = 利潤來源不同。投資人用「全 banks ETF」框架是過度簡化，個別玩家 ROE 結構必須區分。
2. UBS / HSBC / Barclays / DBK 已從「結構性低 ROE」過渡到「ROE > CoC」：是 2010-2020 narrative 已過時的證據，但市場估值仍以舊框架 P/B 0.5-0.7x 給；2026-2030 收斂是 multi-year alpha thesis。

§2 全球銀行 ROE 演進史 + S 曲線 + 歷史類比

全球銀行 ROE 結構過去 30 年歷經 4 個結構性轉折：(1) 1995-2007 是 Anglo-American 銀行黃金期 — JPM / BAC / GS / Lehman / Morgan Stanley / RBS / Barclays / Deutsche Bank 全部 ROE 15-25%，槓桿 30:1+；(2) 2008-2018 是後危機重整期 — Basel III 推升 capital ratios 從 8% 至 13%+，US 大行率先恢復 mid-teens ROE，歐洲大行 ROE 仍 high-single 多年；(3) 2018-2024 是 ZIRP 末期 + 通膨衝擊 — 美 NIM 擴張、歐 NIM 仍受負利率拖累、日本完全脫離 deflation；(4) 2024-2026 是區域 ROE 收斂期 — 歐洲銀行 ROE 達 9-13%、日本 megabank 達 10-12%、美國繼續 17-20% leadership，全球 ROE dispersion 雖仍存但 narrowing。

本次循環的特殊性：BoJ 從 ZIRP 退出是 30 年來首次。日本三大行 ROE 從 8% → 11.5%（MUFG H1 FY25 vs FY23）是 1990 以來首次有意義的 ROE 結構性升級；BoJ 0.75% 是 1995 以來最高利率，這個 transformation 是 20 年週期事件，可能比 1990 後通縮陷阱還要深刻。對應的 hedge 是 — 日本 megabank 從「low-ROE laggard」進入「multi-decade re-rating」階段。

★ 歷史類比（2 個正類比 + 1 個反類比）

類比核心 pattern本次相似 / 差異
正類比 #1：1990-2000 日本銀行 deflation 陷阱（10 年 ROE 〈 0%）1990 日本資產泡沫破裂 → 銀行 NPL 浪 + 0% policy rate + 10 年 deflation；MUFG/SMFG/Mizuho 1995-2005 ROE 平均 -1 到 +3%（對沖 NPL writedown）反向類比：本次是「日本 ROE 從低位 transformation 上升」（vs 1990s 「ROE 從高位崩跌」）；條件相反但時間長度類似（10 年復甦 vs 10 年衰退）；MUFG 12% mid-term target 是「1995 黃金期 ROE 一半」reference point。
正類比 #2：2008-2014 美國大行 turnaround（Citi / BAC 從 ROE -10% 至 mid-teens）2008-2010 Citi / BAC ROE 跌至 -5 to -10%；2010-2014 Vikram Pandit + Brian Moynihan turnaround；Citi 2013 達 ROE 8% / BAC 2014 達 ROE 4-7%；2018-2025 才完成 mid-teens 復甦（10+ 年）結構相似：歐洲大行（DBK / Barclays）2024-2026 從 high-single 升至 11-13% 路徑類比 Citi/BAC 2010-2018。差異：歐洲沒有過 -10% ROE 的 hole，所以從 9% 升至 13% 的時間應較短（4-5 年 vs 美 10 年）。但 Citi 自己仍 5-7% — 是「too big to fix in one cycle」結構警示。
反類比：2000 dot-com 後 IB 集中（GS/MS/Lehman/Bear Stearns/JPM IB）2000-2003 dot-com bust 後 IB rev 萎縮 50%+；GS/MS/JPM IB 集中度從 dispersed → 寡占；2007 IB 重新爆發；2008 Lehman/Bear Stearns 倒，集中度進一步上升至今本次部分類比 + 部分反類比：2025 IB rev $150B + M&A $4.81T 是 cyclical 強反彈但不是 dot-com 級重組；GS/MS 集中度上升是 multi-cycle 結構（不會像 2007 突然崩）；反向警示是 2027-2028 IB cycle 若進入 trough，GS Q4 IB +47% 樣強增長 unsustainable。
💡 §2 Insight
1. 1990-2000 日本銀行 deflation 反向類比是 MUFG re-rating thesis 最強支撐：1990 ROE 從高位崩跌的 mirror image 是 2024+ ROE 從低位 transformation 上升 — 兩者都是 multi-decade 大事件。MUFG 12% mid-term target + BoJ 預期 1% rate = 2027-2030 是日本 megabank 黃金期。
2. 歐洲大行類比 2010-2018 美國大行 turnaround 但時間更短（4-5 年）：UBS / HSBC / DBK / Barclays 2024-2026 從 ROE 9-11% 升至 11-13% 的速度比 BAC / Citi 2008-2018 快得多 — 因為沒踩到 -10% hole。但 Citi 仍是「to big to fix」warning。
3. 2000 dot-com 反類比警示 IB cycle 終究是 cyclical：2025 IB cycle peak vs structural 集中度上升要分清；GS Q4 IB +47% / MS Q4 +47% 都是 cyclical recover，2027-2028 cycle trough 仍會出現 — 估值 modeling 必須 normalize cycle peak。

§3 ROE 驅動因素拆解（5 個 component）

銀行 ROE 拆解（DuPont expansion）：ROE = (淨利息收入 + 手續費 - 損失) / 平均資產 × 平均資產 / 平均權益 = (NIM × 資產質量) × 槓桿。本章拆解 5 個核心驅動，分析為何全球大行 ROE 出現 17%→5% 的巨大分歧。

(1) NIM × 利率環境 — 區域 monetary policy 主導

NIM 是 ROE 第一驅動，受區域 monetary policy 主導：

  • 美國：Fed 5%+ peak（2023） → 4.0-4.5%（2025-2026）；JPM 2026 NII guide $103-104.5B（FY），意味著 NIM 維持 ~2.6-2.7% level（vs 2024 2.5-2.6%）。
  • 歐洲：ECB 4%+ peak → 2.0-2.5% 區間（2025-2026）；UBS 因 wealth 主導 NIM less relevant，HSBC NIM ~1.5-1.7%（混合）。
  • 日本：BoJ 0.50% (Jan 2025) → 0.75% (Dec 2025)，預期 1.0-1.25%（2027-2028）；MUFG NIM 0.77% Dec 2024Q（vs 0.68% YoY，+9bp）；SMFG 0.87%（+12bp）；Mizuho 0.44%（+7bp）— 三大行 NIM 同時擴張是 1995 以來首見。
  • 中國：LPR 持續下調 + Big 5 NIM 全部 sliding；ICBC 房地產佔比 5.4% 較低但仍受壓；CCB 房地產 RMB 908B（+6.37% over 2023）顯示無 deleverage 跡象。

(2) Fee Income × 業務組合 — IB / Wealth 結構優勢

IB / Wealth fee income 是 ROE 第二驅動，集中在少數玩家：

  • GS IB fee 2025 $5.0B (+26%) — M&A advisory $1.36B Q4 (+41%)；GS 廣告 M&A 推 $1.6T+ 領先 #2 對手 $250B+，是全市場 30%+ 集中。
  • MS Q4 IB rev $2.41B (+47%) — debt underwriting Q4 $785M 接近 doubled；M&A advisory ~$1.1B (+45%)；Total rev $116B FY 2025 vs $61.8B 2024 = 88% 雙年增。
  • UBS wealth invested assets $7T+ (+15% YoY) — 整合 Credit Suisse 後成為「世界最大 private wealth」之一；MS wealth+investment combined $9.3T 是全球第一。
  • HSBC Asia wealth $1T+ booked — 整體 wealth $2.1T；HSBC 在亞洲 + 中東是領先 wealth manager。
  • JPM IB + AWM：JPM 2025 IB rev 結構穩定 + AWM $5T+ 全球第二大 — 雙引擎驅動 17% ROE / 20% ROTCE。

(3) Cost / Income Ratio — 結構效率

C/I ratio 反映成本管理效率，影響 ROE 第三驅動：

  • JPM 2025 C/I ~57%（efficiency 接近全業最佳）；2026 expense $105B vs $96B 2025（重投資 → tech, AI, expansion）。
  • GS 2025 C/I ~65%（IB 業務本身 C/I 較高）；2025 EPS $51.32 (+27%) 是 operating leverage 證據。
  • DBK 2025 C/I 64%（達標 below 65% target，從 2024 76% 大幅改善 -12pp）；2025 record FY profit。
  • UBS 2025 cumulative cost cuts $10.7B（Credit Suisse 整合 cost saving + 自身 efficiency）。
  • Citi 2025 C/I ~70%+（結構性高 cost — 全球 footprint + 多層管理）；turnaround 主要是 cost 削減而非 revenue gain。

(4) Capital Ratios × Leverage — Basel III 限制

資本比率限制槓桿，影響 ROE 第四驅動：

  • JPM CET1 ~15.0%（2025 Q4），高於最低要求 12.0% → 有 buyback / 投資空間。
  • UBS RoCET1 underlying 13.7% — UBS 對 CET1 capital 結構敏感，整合 Credit Suisse 後 capital ratio 達 15%+。
  • BAC CET1 ~11.6%（仍在重建中）；2026 NII +5-7% guide 為 capital recover 提供空間。
  • Basel III Endgame 從 +20% capital 變 capital-neutral：2026 finalize, 2027 implement；Trump capital-neutral version 對美國大行是「額外 ROE upside」（無新加 capital 要求）。

(5) Asset Quality × 信貸週期 — 損失準備

信貸質量是 ROE 第五驅動：

  • 美國 2025 信貸週期 mid-cycle：JPM 2026 Card NCO 3.4%（normal）、BAC NPL 0.5%、Citi NPL 1.0% — 美國銀行業整體信貸質量穩定。
  • 歐洲 2025 NPL 持續 normalize：DBK / Barclays / UBS NPL 1.5-2.5% 區間；歐洲 ECB ESM stability framework 已 mature。
  • 日本 2025 NPL ~1.0-1.5%：BoJ 升息對 borrower 是 testing，但 megabank 多為大企業 + 大型房地產（low risk）。
  • 中國 2025 信貸週期 stress：房地產 NPL 5-8%（vs 報告 3-4%）；LGFV 隱性 NPL ~10-15%（市場估）；Big 5 NIM sliding 是「主動壓縮 spread support real economy」+ 信貸質量壓力 dual-driven。
💡 §3 Insight
1. 「ROE = NIM + Fee + Cost + Capital + Asset Quality」5 個 component 的相對重要性因區域不同：US 大行 NIM + Fee 主導（70%+），歐洲 wealth 主導，日本 NIM 主導（BoJ 升息 catalyst），中國 Asset Quality 主導（房地產 + LGFV）。投資人要用區域不同 framework，不能用美國框架評歐日。
2. BoJ 從 0.75% → 1.0-1.25% 對日本 megabank 是 multi-decade structural transformation：MUFG sensitivity 顯示每 +25bp policy rate add ¥35-40B annual NII，2027-2028 升至 1.25% 將 add ¥75-150B 額外 NII = ¥2.8T NI 對 ¥3T+（vs FY25 ¥2.1T target）。
3. Citi 結構性 ROE 受限 — 不是 cyclical mispricing：C/I 70%+ 結構高（全球 footprint 龐大，難快速 simplify）；ROTCE 5-7% FY25 仍未到 mid-teens 是「too big to fix in 4 年」確認。Citi 估值倍數結構性低於同業 30-40% 是合理而非 mispricing。

§12 Non-Consensus View（4 條共識差異）

🟡 Thesis #1：Citi ROTCE 5-7% 是「too big to fix in one cycle」結構性 cap — 不會回到 mid-teens [structural] [信心: 高]

共識（sell-side 主流）：Citi 2026 達 10-11% target，2027-2028 達 12-14%，Jane Fraser turnaround 是「working」。
本 ID 觀點：4 年 turnaround 仍未 mid-teens 是結構性確認；Citi 業務模型（trading-heavy + 全球 footprint + 消費銀行 weakness）是 multi-cycle 限制；2027-2028 ROTCE 仍 11-13% 而非 mid-teens。
差異：共識 12-14% vs 本 ID 11-13% = 2-3pp 差異（中度非共識）

  • Bear case: Citi 2027 ROTCE 8-9% / 估值再 -20%
  • Base case: Citi 2027 ROTCE 11-12% / 估值持平至 -10%
  • Bull case: Citi 2027 ROTCE 13-14% / 估值 +20-30%
⚠ 證偽條件：Citi 2026 兩季公開 ROTCE > 11% 持續 / 2027 達 mid-teens 13%+
🟡 Thesis #2：日本 megabank ROE multi-decade re-rating — BoJ 升息是 30 年 generational transformation [structural] [信心: 高]

  • Bear case: BoJ 暫停 / MUFG 維持 ROE 11-12%
  • Base case: BoJ 達 1.25% 2028 / MUFG ROE 13-14%
  • Bull case: BoJ 達 1.5% 2030 / MUFG ROE 15%+
⚠ 證偽條件：BoJ 2026-2027 兩年仍維持 0.75% / Yen reverse 至 130-140 強勢迫使停止升息
