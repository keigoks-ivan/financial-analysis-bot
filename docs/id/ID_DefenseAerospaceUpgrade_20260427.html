💡 3 條核心 insight（預告）
1. 歐洲國防擴產比美國快 5-10 倍：Rheinmetall 2026 rev +40-45% YoY、Saab Q1 2026 +24%、BAE +7-9% — 而美國 LMT 5-7%、NOC 4-5%；Trump 紅利已 priced in，歐洲 NATO 5% 路徑才剛啟動。
2. Anduril $60B 估值是「defense tech disruption」訊號：傳統 prime 用 10-20 年週期、Anduril 用 18-24 月迭代；Replicator $20B 合約 + Bolt-M 巡飛彈 + Lattice OS 軟體；如果 IPO 將是 2026-2027 重大 catalyst。
3. 彈藥（155mm）+ GBSD Sentinel 是結構性而非週期性：Rheinmetall 21X 擴產 + NOC Sentinel 從 6-7% → 10% rev；這些是「填洞 + 換代」的結構性產能上修，2030+ 仍持續。

💡 Insight §1
1. 本 ID 與 CNTW S7 重疊但聚焦不同：CNTW S7 是「中台衝突地緣事件驅動」；本 ID 是「全球結構性國防 capex 上修」 — 即使中台和平，NATO 5% + 美國 modernization 仍持續。
2. Tier 1（傳統 prime）vs Tier 2（disruptors）不是 zero-sum：傳統 prime 主導大平台（飛機 / 艦艇 / 飛彈），disruptors 主導低成本消耗品（無人機 / 巡飛彈 / 軟體）— 兩者並存。
