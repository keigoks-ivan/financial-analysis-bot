II 結構性超級週期
★ 本 ID 核心非共識 Thesis 市場用「美國國防四巨頭主導」框架看 defense — 認為 LMT / RTX / NOC / GD 是主軸，歐洲玩家是 secondary。本 ID 認為：① 歐洲玩家是真正的結構性 alpha（NATO 5% 啟動 + Rheinmetall 訂單 +200% YoY + €135B backlog 9.5 yr 覆蓋；vs 共識 focus 美國四巨頭）；② 無人機 / loitering munitions 是 sector 第二春（AVAV Switchblade $250M+ backlog + Anduril $40B valuation）；③ 彈藥短缺是 multi-year fix（Patriot / 155mm / GMLRS 都在 2027+ 才回到 surge demand 滿足）；④ NATO 5% 是 paradigm shift 非政策變動（從 2% 到 5%，意味著新增 multi-trillion 支出，現有產業基礎無法供應）。
💡 §0 4 條核心 insight（預告）
1. RTX $271B record backlog（Q1 26 創新高，從 FY25 末 $268B）+ FY25 organic +11%：是「結構超級週期」訊號 — 訂單持續創新高 + 連續成長 = paradigm shift。
2. Rheinmetall FY26 guide +40-45% rev / backlog €135B（9.5 yr 覆蓋）：歐洲 alpha 證據；NATO 5% Hague 2025 是直接催化劑。
3. NATO 5% 從 2% 跳升 = multi-trillion 累計支出缺口：no existing industrial base can fill — 結構性 supply moat 與 cruise 4 yard 類似。
4. 無人機 / loitering munitions 增速 +50-100% vs Tier 1（LMT/RTX）+10-15%：2nd derivative alpha；AVAV / Anduril / Shield AI 是受益者。

💡 §1 Insight
1. 歐洲國防 prime（Rheinmetall + BAE + Saab + Leonardo）是被忽略的結構性贏家：NATO 5% 主要受惠者。
2. Anduril / AVAV 無人機段是「Tier 1.5」獨立利潤池：增速比四巨頭快 5-7x。
3. 邊界正在被 NATO 5% 重塑：歐洲 capacity 不足 → 美國 outsource / EDF 補貼 / 在地產能擴張三引擎。

結論

國防航太升級進入 Phase II 晚（2022+ NATO 5% paradigm shift + 4 戰線結構超級週期）。核心非共識三條：（1）歐洲玩家是真正結構性 alpha 非美國主導；（2）無人機 / loitering munitions 是 sector 第二春；（3）彈藥短缺是 multi-year fix 非循環。

結構性受益層次：（A）歐洲 prime（RHM.DE + BAE + Saab + Leonardo）；（B）美國 Tier 1（RTX + NOC + LMT）；（C）無人機 Tier 1.5（AVAV + Anduril 私募）；（D）彈藥 / 推進劑（OLN + HEI）。結構性壓力層次：4 戰線 ceasefire 風險 + 美國 budget 政治週期。
