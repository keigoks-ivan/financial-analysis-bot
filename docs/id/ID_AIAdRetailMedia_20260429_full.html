§1 產業白話定義 + 歷史脈絡

白話版：廣告這門生意過去靠「找代理商幫你買版位、調優化」。AI 把這層人工自動化了——你只要給目標和素材，平台 AI 就把廣告投好。結果不是大家一起好，而是錢往「握資料、握版位、看得到成效」的少數集中。

白話開場：先說三個關鍵詞。零售媒體（RMN）是零售商（Amazon/Walmart）用自家的購物資料賣廣告——它最值錢的地方是「閉環衡量」：能看到「你投的廣告有沒有變成購買」。AI 自動投放（Meta Advantage+、Google PMax）是「給目標 + 素材、AI 包辦投放優化」。agentic ads是能跨多個平台自動投放的 AI 代理。本報告問的是：當這兩件事同時發生，廣告這塊大餅的錢，被重分配給了誰。

邊界界定：本報告聚焦「AI 如何重分配廣告價值 + 誰被商品化」。納入圍牆花園（META/GOOGL）、零售媒體（AMZN/WMT/TGT）、開放網路/CTV（TTD/ROKU/NFLX/APP）、廣告代理商（WPP/OMC/IPG/PUB）。Apple Search Ads 詳見 AppleIntelligence 子題。灰色地帶是「代理商能否轉型」——若從賣工時轉成賣不可取代的策略/創意/第一方資料服務，可能上岸。

歷史分三段。第一段：2010s programmatic 自動化——即時競價把「人工買版位」自動化，錢往 Google/Meta 與資料層移。第二段：2020–2024 零售媒體崛起 + 隱私變局——Cookie 退場、ATT 衝擊，廣告主轉向有第一方資料的零售媒體；Amazon/Walmart 廣告高速成長。第三段：2025–2026 AI 自動化 + 零售媒體雙軸——Meta Advantage+ 目標 2026 底全自動、Google PMax/AI Max + AI Mode Direct Offers；US 零售媒體 $71B（+17.8%）、Amazon 79.7%；代理商崩（WPP -10.4%、OMC 併 IPG）。AI 把手動投放商品化、價值集中。

歷史類比一：programmatic 自動化——錢往平台與資料移。上一波廣告自動化把「人工買版位」自動化，價值不是平均分配，而是推向握流量與資料的平台（Google/Meta）。量化錨：programmatic 占美國數位展示廣告比例從 2010 年代初的個位數升到後來的 ~90%，人工買版位的中介被大幅壓縮。對 bull 含義：AI 自動化（Advantage+/agentic）是 programmatic 下一階，把錢進一步推向握第一方資料 + 庫存者。歷史類比二：去中介化——手動中介的宿命。技術讓買賣兩端直連，靠「人工居中」收費的中介被壓縮（旅行社被 OTA、經紀被券商）。量化錨：當 Advantage+「給 URL + 預算、AI 包辦」，代理商的「人工媒體規劃 + 投放優化」被商品化——WPP 2025 營收 -10.4%、Omnicom 併 IPG 砍約 4,000 人、成本目標上修 $1.5B，是當期證據。教訓：代理商要活，得從「賣工時」轉成「賣不可取代的策略/創意/第一方資料服務」。

本主題無單一價格 cycle，用「零售媒體增量集中 + 自動化採用」當時序錨。

關於 cycle 統計表的省略：廣告隨總經有週期，但本主題是結構性重分配（自動化 + 零售媒體），非 ASP cycle，故不立 cycle 統計表；以「廣告自動化三段」（2010s programmatic → 2020–24 零售媒體/隱私 → 2025–26 AI 自動化）的編年呈現時序。

這段歷史的可操作含義：AI 廣告同時是「programmatic 下一階」（錢往平台/資料）與「去中介化」（手動投放被消滅）。bull 的底是握資料/庫存的圍牆花園 + 零售媒體 + TTD，bear（對代理商）是手動優化被商品化——WPP/OMC 營收下滑已是早期證據。判別一家公司站哪邊，看它賣的是「工時」還是「資料/庫存/閉環」。

§2 技術成熟度 + S 曲線

為什麼自動化是現在？因為 AI 跨過「自動投放優於人工」的門檻（Advantage+ +22% ROAS）、零售媒體閉環資料成熟、生成式創意把製作成本壓到趨零——三者讓「手動投放」這層人工失去存在理由。

技術拐點有三。AI 自動投放優於人工：Advantage+「給 URL + 預算、AI 包辦創意/受眾/版位/出價」比手動高約 22% ROAS、低約 14% CPL；Google PMax/AI Max（Gemini 生成標題/描述）同向。閉環資料：零售媒體能看到「廣告→購買」全鏈，AI targeting 交出 6x ROI。生成式創意：Walmart Connect Automated Creative Generation 把製作時間砍 80%，創意製作成本趨零。三者交會，「手動媒體規劃 + 投放優化」（代理商核心業務）被自動化取代；競爭優勢移到「策略輸入品質 + 第一方資料深度 + 創意素材多樣性」。

§3 供給側：現在與未來 + 利潤池

AI 廣告的利潤往「握第一方資料 + 庫存 + 閉環衡量」集中——圍牆花園與零售媒體贏，賣手動優化的代理商被商品化。

供給端可拆四層。圍牆花園（META/GOOGL）：握用戶資料 + 流量 + AI 自動化（Advantage+/PMax），把投放自動化即贏。零售媒體（AMZN/WMT/TGT）：握購物資料 + 閉環衡量，AI 強化（自動創意 -80% 工時、targeting 6x ROI、Rufus/Sparky 購物助手）。開放網路 + CTV（TTD/ROKU/NFLX/APP）：圍牆外的獨立第三勢力。手動中介（WPP/OMC/IPG/PUB）：賣人工媒體規劃 + 投放優化，被 AI 自動化商品化。利潤往「資料 + 庫存 + 閉環」三層，手動中介失血。

利潤池遷移

整條鏈的利潤正從「手動中介」往「資料 + 庫存 + 閉環」遷移：AI 把投放優化自動化（人工成本趨零）、零售媒體用閉環資料交出可歸因 ROI、圍牆花園握用戶資料把 AI 內建即贏；被搶的是賣工時的代理商與無資料/次規模玩家，搶到的是圍牆花園、零售媒體、開放網路 TTD。Advantage+ +22% ROAS 與代理商營收 -10.4% 是利潤遷移的一體兩面。（依 QC-18，廣告利潤池逐段精確百分比難拆，以定性 + 份額數呈現。）

Inference · 供給
自動化商品化手動投放，利潤往資料+庫存+閉環集中
前提：AI 把投放優化自動化（+22% ROAS）+ 零售媒體握購物資料閉環 + 圍牆花園握用戶資料/流量 + 手動中介只賣工時 → 利潤往資料/庫存/閉環三層，代理商被商品化

當「給目標 + 素材、AI 包辦投放」變成預設，廣告主不需要付代理商買「人工調優」；賺得到的是握第一方資料（圍牆花園 + 零售商）、握庫存（CTV/開放網路 TTD）、握閉環衡量（零售媒體看得到「廣告→購買」）的人。Advantage+ +22% ROAS、Amazon Ads Agent -18% CPM 是「自動化贏過人工」的硬證據；代理商營收下滑是另一面。

可證偽條件：若代理商成功轉成「不可取代的策略/創意/第一方資料服務」、或 AI 自動化遇品牌安全/創意瓶頸需大量人工回補，則「手動中介被商品化」論失效。

未來供給與因果閉合：成為贏家的門檻是「第一方資料 + 庫存 + 閉環衡量 + 自動化引擎」。純賣工時的代理商補不上資料/庫存——這是圍牆花園與零售媒體守得住的原因。§1 提出「賣工時 vs 賣資料」是站哪邊的判準——未來 3–5 年仍 binding：AI 自動化只會更強、人工投放更被商品化；唯一鬆動是代理商轉型成不可取代的服務——這是 §7 的反方。
