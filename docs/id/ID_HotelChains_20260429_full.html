單位與口徑：RevPAR = Revenue per Available Room = ADR × Occupancy（owner 的房收口徑，chain 是 royalty base）；ADR = Average Daily Rate（每房單日房價）；NUG = Net Unit Growth = (新開房 − 退出房) / 期初房數（asset-light franchise 的房數成長 KPI，≠ RevPAR）；Pipeline = 已簽約未開幕的 rooms；Conversion = 既有 hotel 改掛新 brand 旗（無新建）；fee revenue = chain 抽的 royalty + management + reservation fee（≠ RevPAR、≠ NUG）；Co-brand royalty = chain 從信用卡發卡行依刷卡 spending 收取的非 RevPAR fee；Bonvoy / Honors / WoH = MAR / HLT / H 三大 loyalty programs；STR = short-term rental（Airbnb / Vrbo）；GBV = Gross Booking Value（ABNB / BKNG 用語，marketplace 交易額，≠ RevPAR）；Take rate（ABNB）= service fee revenue / GBV。三條成長線（RevPAR / NUG / fee revenue）口徑不同、不可互相代換。所有 financials = Q1 FY2026 actual（截至 2026-05 已公告）。

§1 產業白話定義 + 歷史脈絡

連鎖旅館賣的不是「房間」，而是「brand + loyalty + distribution」——付錢的不是住客（住客付給 owner / hotel），是 hotel owner 付給 chain 的 royalty 與 management fee，外加信用卡發卡行付給 chain 的共品牌 royalty。看懂「chain 不持有房、賺的是 fee」，才不會用 RevPAR 這把錯尺量它。

白話開場：1980 年以前，連鎖旅館是「hotel real estate operator」——Holiday Inn、Hilton、Marriott 自己蓋、自己持有、自己經營 hotel，賺的是房收。但 2008 金融危機之後，sector 徹底轉型成 asset-light：實體 hotel 多半由獨立 owner（個體 + 家族 + REIT + PE）持有，chain 只賣 brand + loyalty + 訂房系統，收 royalty + management fees。今天 MAR 75% / HLT 97% / WH 100% 的房都不是自己的。所謂「連鎖旅館投資」，買的其實是一個 capital-light、recurring fee 的平台生意，不是房地產。

邊界界定：本報告聚焦「以 brand 為核心、asset-light franchise / management 為主流」的全球住宿業集團（MAR / HLT / H / IHG / CHH / WH），並把短租平台（Airbnb 為核心）納為連鎖旅館的相鄰 / 替代品 / 競品三重身份。OTA（Booking / Expedia）是分銷層、與 chain 既合作又競爭，本 ID 視為相鄰、獨立另一份 OTA 母題 ID 處理；timeshare（VAC / HGV）屬度假權結構、casino（MGM / LVS）以博弈為主、hotel REIT（HST / RHP）屬 owner 端——三者皆 out-of-scope 或僅 mirror exposure 提及。灰色地帶在「Airbnb」：它的 leisure / 長住住宿與 chain 直接競爭 nights，列 🔴 核心；但它是 marketplace（賣 host inventory）而非 brand chain，所以是「替代品 + 平台」雙重身份。

歷史脈絡分四個轉折。第一次：1955~1980——asset-heavy chain 黃金期，Holiday Inn（1952）、Hilton 國際擴張，公司持有 hotel real estate，吃公路旅遊與商務黃金期。第二次：1980~1995——brand 多元化 + franchise 試點，1984 Courtyard by Marriott、1987 Hampton Inn 建立 brand portfolio 護城河。第三次：1993——Marriott 拆 Marriott International（運營）+ Host Marriott（REIT），是 asset-light 大轉型的里程碑事件；2007 Blackstone $26B 私有化 Hilton 後重塑 brand portfolio。第四次（現在）：2010~2026——loyalty platform + non-RevPAR fee 商務化，Bonvoy / Honors / WoH 上升至 ~283M / ~250M / 66M，共品牌信用卡 royalty 成為非 RevPAR 流，asset-light 比例逼近極限。

最關鍵的當期事件：2026 共品牌 royalty +35%——Marriott 揭露 2026 年共品牌信用卡 fee 將 step-up 約 35%，CFO Leeny Oberg 在法說明確說明這來自重新議定的 royalty rate（結構性、永久，每一塊 Bonvoy 刷卡 spending 的 fee 都上修）＋ 持卡人 spending 成長，且與仍在談判中的新卡約完全分開。這是連鎖旅館史上第一次把 loyalty 從「成本中心（給折扣換忠誠）」明確轉成「利潤中心（依 spending 收 royalty）」並公開量化。它與 1993 spin-off 同屬「收入性質被重新認知」的事件——只是這次發生在 fee 內部、而非公司結構。

  2. [T2]Skift — Marriott 規劃新卡約、CFO 證實 +35% 來自永久 royalty re-pricing（與新卡約分開）

§2 技術成熟度 + S 曲線

為什麼 fee re-rating 是現在發生、不是 2015 年？因為兩個結構 bottleneck 剛被解開——asset-light 比例終於 saturate（HLT 97% 逼近 ceiling，「再 asset-light」這條引擎用完），loyalty 規模終於到達可重定價的臨界量（Bonvoy ~283M），共品牌 royalty 才第一次能被當「payment rail」重新議價。

把連鎖旅館的「商業引擎成熟度」畫成 S 曲線，核心變數是 asset-light（franchise + managed）比例。過去 50 年它從 1975 約 25%（asset-heavy 黃金期）一路爬升，1990 約 45%（franchise 試點）、2000 約 55%、2010 約 70%（spin-off 加速期），到 2025 sector 均值約 85%，其中 HLT 97% / WH 100% 已 saturate。這條曲線的關鍵啟示是：asset-light 化這條多年的「margin step-up + 倍數 re-rating」引擎，對 sector 龍頭已經基本用完——HLT 沒有更多 owned 可以 spin off，MAR 剩約 25% owned/managed 是僅存的小空間。下一波 alpha 必須來自曲線之外的新變數：fee mix（共品牌 royalty / loyalty 商務化 / 直訂比例）。

§3 供給側：現在與未來 + 利潤池

供給端的真稀缺不是房間數（pipeline 都在創高），而是「能把房間轉成高品質 capital-light fee」的能力——這由 brand + loyalty + 黃金地段 + 直訂通路決定。asset-light 已飽和，下一個利潤池在 fee mix，不在繼續 spin off。

供給有兩條軸。第一條是房間供給（NUG），2026 仍健康且創高：HLT pipeline 創高 527k rooms（占全球在建房約 20%、近半已在建）、MAR ~618k rooms 創高、WH 259k+ 創高、IHG 343k rooms。但房間是「量」，conversion（既有 hotel 改掛 brand 旗、無新建）讓 NUG 在 cycle 末段仍能加速——owner 想換 brand 救援時 conversion 反而增加，是「無資本 NUG 引擎」。第二條、也是真護城河的是fee 品質供給：把房間轉成 capital-light、recurring、可重定價 fee 的能力。這道門檻已把 supply 結構性分層——MAR 的 ~283M Bonvoy（高 spending cohort）能把刷卡變成 payment-rail royalty，WH/CHH 的經濟型 cohort（人均 spending 低）即使 NUG 不差也拿不到同樣的 fee 品質。

用「asset-light 飽和度 × loyalty 規模 × Q1 動能」三維看誰在 fee 品質上領先、誰只是房間多。

成本曲線（省略說明）：連鎖旅館的 chain 端無實體產能 / 邊際成本曲線——fee margin 普遍 ~30~40%（chain 端），價格戰的終局不由單位成本決定，而由 brand 強度與 loyalty 鎖定決定。實體 capex 由 owner 承擔（每房 $100k~500k+），這正是 asset-light 模式把週期風險外推給 owner、chain 收 fee 的結構優勢。

未來供給與進入門檻：新進入者要成為 fee 品質贏家，門檻是「全 segment brand portfolio + 大規模高 spending loyalty + 黃金地段 + 直訂通路」，這幾乎不可能內生快速複製——所以新供給的主要路徑是 brand 收購與 conversion，而非新建。Hyatt 的 ALG（2021）+ Bahia Principe JV（2024）+ Playa（2025-06，$2.6B）整併把它推上 all-inclusive 全球前段，就是「靠併購補 segment」的範例。供給彈性低（不能快速「蓋」出一個 ~283M 會員的 loyalty 平台）正是 fee 品質領先者議價權的來源。

因果閉合（回應 §1 的結構變數）：§1 提出「fee 性質改變（RevPAR-linked → spending-linked）」是新護城河——未來 3~5 年它是否仍 binding？答案是很可能更 binding：loyalty 規模越大、人均 spending 越高，共品牌 royalty 的重定價籌碼越強，後進者越難追。唯一能鬆動門檻的是「AI 直訂 / 中立分銷讓 loyalty 鎖定失效」或「經濟型循環長期不翻轉讓 fee base 受壓」——這是 §7 要檢驗的反方。

12~36 個月內，供給端不會出現新的 fee 品質級競爭者（門檻太高、只能靠併購與 conversion）。對投資的含義：押「fee 品質領先者」（MAR loyalty / H/HLT NUG）比押「房間量大但 fee 品質低」（WH/CHH 經濟型）的結構勝率高——除非你押的是「經濟型 cycle 翻轉」這個獨立 contrarian 變數（見 §5/§7）。

§4 需求側：現在與未來 + 三角驗證

需求最不可替代的一塊是「商務 + 高端 + loyalty 鎖定」——chain 在這三塊 dominate；最大的結構性流失是「leisure + 30 天以上長住」，這塊 chain 已永久讓給 Airbnb。看懂「各贏各的」才不會把 K 形分化誤讀成 chain 全面敗退。

現需結構：需求由兩個分化定義。第一是K 形 segment 分化——高端與經濟型走兩條路。Q1 2026 攤開：Hyatt（高端 / all-inclusive 集中度最高）system RevPAR +5.4%、all-inclusive Net Package +7.4%；Marriott 全球 +4.2%（APAC +7%）；IHG +4.4%（Greater China +5.7%）；而經濟型的 Wyndham 美國 RevPAR 只回到 0%（雖大幅優於自家 -2~3% guide）、Choice 受颶風拖累微降。全球 luxury hotel CAGR（約 10%）顯著高於整體（約 5~7%），是嬰兒潮 + 高端 demographic 的多年順風（見姊妹 ID_GlobalLuxury / ID_BoomerTravelSupercycle）。第二是chain vs Airbnb 通路分化——ABNB Q1 2026 GBV $29.2B（+19% / ex-FX +13%）、nights +9%、ADR ex-FX +4%，在 leisure 與長住續強，但這不是搶 chain 的商務客，而是各吃各的需求池。

因果閉合（回應 §1 的需求驅動）：§1 提到 fee 商務化是當期核心——未來 5~10 年它會強化還是消退？很可能強化：loyalty 鎖定創造重複需求（H 揭露會員占近半 occupied nights、人均 spending ~2x 非會員），共品牌 royalty 隨 spending 與重定價持續上修。需求拐點風險不在「fee 商務化方向」（很難逆轉），而在「RevPAR base」（美國 occupancy plateau + 商務差旅 cycle）與「Airbnb 是否從 leisure 跨界吃商務」。

最有定價權的需求是「高端 + loyalty 鎖定 + 共品牌 spending」的綁定——客戶買的不只是房，是 status、點數、刷卡回饋的整套生態。對投資的含義：fee 商務化方向幾乎沒爭議、很可能持續，所以「fee 會 re-rate」不是風險點；真正要盯的是 RevPAR base 是否守正（美國 plateau）與經濟型循環是否翻轉（WH/CHH contrarian 的成立條件）。

§5 供需裁決 + 短中長期推估 + 投資時鐘

⚠️ 本章含 judgment：🟡 bullet 代表推論，附事實鏈與信心度 + ⚠ 證偽條件。
綜合 §3（asset-light 飽和、fee 品質是新門檻）與 §4（K 形分化、商務+高端+loyalty 是 chain 的 dominate 區、leisure 長住讓給 Airbnb），本產業未來 3~5 年的裁決是「fee 商務化續行、RevPAR plateau、高端 / NUG 領先者跑贏」。

框架逃逸（§5 必架橋）：傳統「過剩 / 短缺」的商品供需框架不直接適用——連鎖旅館的 chain 端無實體產能、無 ASP peak-to-trough cycle（不像記憶體 / 面板）。要對應的話，這是需求型 + asset-light 結構：稀缺不在房間（pipeline 都創高），而在「能把房間轉成高品質 capital-light fee 的能力」——稀缺資源是 brand + loyalty + 黃金地段。所以「裁決」不是「供需誰大」，而是「fee 品質的稀缺性能否維持 + RevPAR base 是否守正」。若硬要對應商品框架：高品質 fee 的供給（具備 ~283M loyalty + 重定價籌碼的玩家）遠少於市場需求，等價於 fee 品質的結構性短缺——只是稀缺性來自準入門檻（brand/loyalty/地段）而非實體產能。

資本週期證據（替代指標）：連鎖旅館的 chain 端無實體 capex cycle，改用三個「資本是否過度湧入」的替代訊號：① pipeline 與 conversion 比例——pipeline 創高但多為 conversion（無資本 NUG）與 asset-light 簽約，非 owner 端的過度新建（owner 端 capex 受高利率約束、新建受抑），顯示「房間供給」未過熱；② brand M&A 估值——Hyatt Playa $2.6B 反映「all-inclusive 拼圖」溢價，但收購方有現金流，非槓桿堆砌；③ asset-light 飽和——sector 已過 asset-light spin-off 的「資本結構重塑」階段（HLT 97% saturate），資本不再無差別湧入。三者指向「理性的 fee 商務化」而非「泡沫式擴張」。[A: 假設利率不再大幅上行壓 owner 端新建；若急升，pipeline 兌現會延後]。

§3 + §4 → 供需裁決

依 §3（asset-light 飽和、fee 品質是新門檻、只能靠 loyalty / 併購 / conversion）與 §4（K 形分化、商務+高端+loyalty 是 chain dominate 區）的推導，未來 3~5 年產業結構是 fee 品質領先者跑贏（loyalty / NUG 領先擴張、經濟型 fee base 觸底），因為需求側的高端 / loyalty 鎖定與供給側的 fee 品質門檻彼此強化。這不是「過剩 / 短缺」的商品框架（chain 端無產能），而是「fee 品質集中度」的結構框架——稀缺的是能 payment-rail 化的 loyalty，不是房間。

估值傳導機制：連鎖旅館的估值對「fee 性質」與「RevPAR 動能」雙敏感。傳導鏈一：RevPAR 動能 → fee base → 倍數，這是傳統的「週期股」傳導（美國 RevPAR plateau 會壓這條）。傳導鏈二、也是非共識的：fee 性質改變（RevPAR-linked → spending-linked）→ 估值框架改變 → 倍數 step-up，這是 1993 spin-off 式的「性質 re-rating」（MAR 共品牌 +35% 走的是這條）。敏感度錨點：1993 spin-off 後運營體 P/E 在 5 年內從 ~8~10x → ~18~20x（性質 re-rating），對照本次若市場接受「Bonvoy = payment rail」，MAR 從 hotel ~27x 往 payment 區間移動的空間。這就是為什麼「fee 性質」對 MAR 的殺傷力 / 助力遠大於單季 RevPAR。

估值現在 price 了什麼：HLT 的 ~33x price 了「asset-light + NUG 雙王」（已充分）；MAR 的 ~27x 仍用「hotel chain」框架、沒有 price 共品牌 royalty 往 payment-rail re-rate；WH/CHH 的 ~16~19x price 了「經濟型週期風險」（cycle 翻轉未 price）。會觸發 re-rate 的是 MAR 共品牌 fee 被市場重新框架（payment 而非 hotel）或經濟型 cycle 翻轉；會觸發 de-rate 的是美國 RevPAR 轉負或 +35% 被證明一次性。對 PM：RevPAR 高低不是進出唯一依據，fee 性質的 re-framing 才是 MAR 的關鍵變數。

  • 共識說 X：ABNB GBV 雙位數成長 = chain 持續輸 share。🟢 [F: T1 — ABNB Q1 2026 GBV +19%（ex-FX +13%）、nights +9%、STR share ~44%]。
  • 我們說 Y：這是 K 形雙贏而非零和——chain 在商務 + 高端 + loyalty 鎖定 dominate（H：會員占近半 nights、人均 spending ~2x），ABNB 在 leisure + 30 天以上長住贏，chain 已主動把長住讓出。🔵 [X: base 很可能 — 雙贏並存；bear 可能 — ABNB 從 leisure 跨界吃商務 / 短住]。
  • Priced-in：ABNB ~32~35x Fwd P/E price 了 leisure / 長住的續強；chain 的倍數則隱含「商務 + 高端 base 守得住」。市場常把 ABNB 雙位數誤讀成 chain share 流失，但兩條線 Q1 同時往上（H RevPAR +5.4% & ABNB nights +9%）——所以押「chain 商務 base 不會被 ABNB 吃」是對的且部分未 priced，但押「ABNB 會繼續吃 chain」則已 priced 在 ABNB 高倍數、不可加碼操作。
→ 推論邏輯：ABNB 吃 leisure 長住、不吃 chain 商務核心，雙贏並存。⚠ 證偽：若企業差旅大量轉向 STR、或 chain 直訂 / loyalty 占比下滑，分歧 2 翻轉成真正的 share 之爭。
🟡分歧 3：「asset-light 還有第二春（fee 永續線性加速）」信心：中
  • 共識說 X：chain asset-light fee 會像 2010~2020 那樣線性加速。🟢 [F: T1 — HLT 97% / WH 100% asset-light]。
  • 我們說 Y：asset-light 已飽和（HLT 97% 逼近 ceiling），「再 asset-light」引擎用完；下一個 fee alpha 必須靠 mix shift（共品牌 royalty / loyalty / 直訂），不是線性延伸。🔵 [X: base 很可能 — fee 靠 mix shift step-up（MAR +35%）；bear 可能 — mix shift 兌現不如預期、fee 增速向 RevPAR 收斂]。
  • Priced-in：HLT ~33x（sector 最高）隱含「asset-light + NUG 雙王」續航，幾乎沒 price「asset-light 引擎已用完、增量要靠 NUG / loyalty」的天花板風險——這是 HLT 最未被 price 的下行；反觀 MAR ~27x 沒 price mix shift 的上行。兩者剛好相反：HLT 把飽和當續航（上行已 price 滿）、MAR 把 mix shift 當未知（上行未 price）。
→ 推論邏輯：asset-light 飽和把 alpha 逼到 fee mix；HLT 上行已 price、MAR 上行未 price。⚠ 證偽：若 HLT NUG 持續 6~7% 且 loyalty 變現超預期，「飽和天花板」論對 HLT 失效。

Steel-man 反方（最有力的 3 條）

  • 反方 1（RevPAR 地心引力）：美國 occupancy ~62% 已 plateau，若商務差旅 / 宏觀走弱把 RevPAR 推入負區，chain fee base 直接受壓，sector 高倍數（HLT ~33x）de-rate 空間大。當前證據：美國 RevPAR Q1 僅 0~4%、HLT FY guide 2~3%。證偽窗口：美國 system RevPAR 連兩季轉負。
  • 反方 2（+35% 是一次性）：MAR 共品牌 fee +35% 可能是 royalty rate 重定價的一次性 step（2026 增量），2027 增速回落到 spending growth（~中個位數），fee re-rating 論落空。當前證據：+35% 是 2026 年的 step-up 數字，2027 續強未驗證。證偽窗口：MAR 2027 共品牌 fee 增速回落到 RevPAR 同步。
  • 反方 3（AI 分銷 disintermediate）：若 AI 訂房（ChatGPT / Perplexity）繞過 chain 直訂與 loyalty、把住客導向比價，chain 的 loyalty 鎖定與直訂溢價被稀釋，共品牌 royalty 的 spending base 受壓。當前證據：AI 訂房仍早期、但對 OTA SEO 已有衝擊。證偽窗口：chain 直訂 / loyalty 占比連續下行。

市場最低估的風險：不是 RevPAR plateau（人人都知道美國 mature）、也不是 Airbnb（已被廣泛討論），而是反方 2 的「+35% 是一次性」——市場若把 MAR fee re-rating 當 cycle booster（而非永久），則 fee 商務化的整個 thesis 落空。這是本報告認為最該獨立監測的關鍵驗證點（2027 是裁決年）。

三條分歧裡最可操作的是分歧 1（看 fee 性質而非 RevPAR）——它最未被 price（MAR ~27x 仍用 hotel 框架），且有明確領先指標（共品牌 fee 增速 vs RevPAR）可提前驗證。分歧 3（asset-light 飽和）次之、HLT 上行已 price 滿是其下行風險；分歧 2（雙贏非零和）方向對但在 ABNB 高倍數上已 part-priced。
