★ 本 ID 核心非共識 Thesis 市場把重型機械當「成熟 cyclical 工業股」追蹤 — 看 PMI、商品價、基建週期。本 ID 認為：① CAT 不是 cyclical 機械股而是「AI Data Center + 銅超級週期」隱性受益者 — Power & Energy segment（含發電 + 油氣 + AI DC）是 CAT 2026 成長最快段；CAT $51B backlog（+71% YoY）反映 AI DC 對備援電源 / 燃氣輪機 / oil & gas equipment 的爆發需求 — 不是傳統建設 cycle。② 自動採礦（autonomous haulage）是被嚴重低估的 5-10 年 secular shift — Komatsu 1,000 自動卡車里程碑 + CAT 700+ 卡車（目標 2030 翻 3 倍）；retrofit 一台 400-噸卡車成本 $500-800K，但每 1,000 台 fleet 自動化可省 $300M+ 人事 + 運營成本 5 年；2030+ 礦業設備將進入「自動化標配」階段。③ John Deere「2026 是 cycle 底部」是被低估的 contrarian 訊號 — Q1 FY26 EPS $2.42 大幅 beat（vs $2.02 預期）+ 公司管理層將 2026 net income guide 上修到 $4.5-5B（從 $4-4.75B）；如果 2027 農機 cycle 復甦 + See & Spray 軟體 ARR 加速，DE 估值有 +1 turn 重估空間。
💡 3 條核心 insight（預告）
1. CAT 是「AI Data Center 隱性受益者」：$51B backlog（+71% YoY）+ Power & Energy segment 最快增長 + 燃氣 / 備援發電 + 油氣 equipment 爆發；類比 GEV / VRT 但被市場當「重型機械週期股」估值。
2. Komatsu / CAT 自動採礦競賽是 5-10 年 secular shift：Komatsu 1,000 trucks 領先 + CAT 700+ trucks 跟進；retrofit 經濟性已驗證（5 年 ROI）；2030 全球自動採礦 fleet 可能 5,000+ trucks。
3. DE「2026 cycle 底部」是 contrarian buy 訊號：Q1 FY26 EPS 大幅 beat + 上修 2026 全年 guide；See & Spray + 自動曳引機軟體 ARR 是結構性成長 leg；類比工業自動化從 cyclical → secular。

§1 產業定義與邊界

本 ID 涵蓋三大子市場：① 建設 / 基建機械（CAT 主導 + Komatsu / 三一 / Volvo Group）；② 礦業設備（CAT / Komatsu 雙寡占 + Sandvik / Atlas Copco / Epiroc 鑽探）；③ 農業機械（DE 主導 + AGCO / CNH）；④ 動力系統（Cummins 主導 + Daimler / Volvo Group）。四者透過共同的「重型 ICE 引擎 + 自動化整合」connect。

💡 Insight §1
1. 「重型機械」是個錯誤的合併分類：建設 / 礦業 / 農業 / 動力系統四個子市場 driver 完全不同 — 建設受基建週期、礦業受商品週期、農業受農作物價格、動力系統受 powertrain 轉型；不能合併追蹤。
2. 礦業 + 自動化是被低估的「下一個 EV」軌道：自動採礦從 niche 變主流，類比 2010-2020 EV 從 niche 變主流；Komatsu / CAT 是受益最大者。
3. 動力系統正在轉型期：Cummins HELM 平台支持 diesel + H2 + 多燃料；2030+ 重型機械動力會從純 diesel → 多元化。
