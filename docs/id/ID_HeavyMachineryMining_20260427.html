§12 Non-Consensus View（共識差異）

🟢 #1 [structural] CAT P&E (Power & Electrification) 是 sector 最被低估的 AI DC alpha [信心: 高]
共識（Wall Street）：CAT 是 mining/construction cycle stock；fwd PE 18-20x 已 fully priced。本 ID 看法：P&E $51B backlog +71% YoY = AI DC gas turbine + 大型 reciprocating engines surge demand 直接證據；P&E 段 CAGR 14% > sector 6%；如 P&E 段獨立估值，fwd PE 應升至 22-25x（mirror power infra peer）。
事實 1: P&E $51B backlog +71% YoY (2025-2026 最快 segment)
事實 2: 大型 power CAGR 14% (sector 6%)
事實 3: AI DC capex MSFT/GOOGL/META +$200B+ /yr 結構持續
→ 推論：CAT 是 sector 結構性 alpha；fwd PE 22-25x 2027+。
⚠ 證偽：若 P&E backlog 連 2 季 YoY -10% OR AI DC capex -50%。
🟡 #2 [event-triggered] DE FY26 EPS 上修 = cycle bottom 確認 = 早段 entry 機會 [信心: 高]
共識：DE 是 deeply cyclical 農業股；2024-2025 cycle 仍未 confirm bottom。本 ID 看法：Q1 FY26 EPS $2.42 vs Street $2.02 + FY26 guide 上修 $4.5-5B = cycle bottom 從 2024 H2 反彈；歷史 cycle 早段 5 yr +200% (2003-2008 / 2010-2014)；2026-2028 是 mid-cycle 進場 entry 窗口。
事實 1: Q1 FY26 EPS $2.42 vs Street $2.02 (beat)
事實 2: FY26 guide $4.5-5B 上修
事實 3: precision agriculture moat (See & Spray)
→ 推論：DE 是 cycle 早段 alpha；2026-2028 cycle EPS CAGR +10-15%。
⚠ 證偽：若 Q2 / Q3 FY26 EPS miss OR 商品價格深度跌 -20%+。
🟢 #3 [structural] Komatsu 1,000 autonomous truck = 5-10 yr secular shift first mover [信心: 高]
共識：autonomous mining 是 niche 技術；CAT 跟進後 Komatsu first mover alpha 將 erode。本 ID 看法：Komatsu 1,000 trucks verified（首家 OEM）+ 6+ 年 deployment 經驗 + autonomous switching cost 高（mine plan 整合）= first mover moat 結構性；CAT 跟進需 3-5 年才能達 share parity；autonomous truck premium +25% = margin alpha 直接。

結論

重機械礦業進入 Phase II 晚（2022+ AI DC 隱性受益 + autonomous mining secular shift + cycle bottom 三引擎）。核心非共識三條：（1）CAT P&E (Power & Electrification) $51B backlog +71% YoY 是 sector 最被低估的 AI DC alpha；（2）DE Q1 FY26 EPS beat + FY26 guide 上修 $4.5-5B = cycle bottom 確認，2026-2028 是早段 entry；（3）Komatsu 1,000 autonomous truck verified = 5-10 yr secular shift first mover，CAT 跟進落後 1-2 年。

結構性受益層次：（A）CAT（P&E AI DC + 全球 mining + 建築多引擎）；（B）DE（cycle bottom + precision moat）；（C）Komatsu（autonomous mining secular）；（D）URI（DC + infra 雙引擎 + fleet moat）；（E）三一重工（中國基建 + 一帶一路 high-beta）。結構性壓力層次：全球衰退 + 商品價格暴跌 + AI DC capex pull-back。
