§0 TL;DR — 一句非共識 thesis

本 ID 一句 thesis：穀物 / 油籽 2026 是「南美超產 + 中國脫鉤美豆 + 全球小麥 5 年最寬鬆」三重 supply pressure 期；市場誤把 BG-Viterra 整合視為 ABCD 全行業重估的開始，但這是整合潮的句點而非起點 — Bunge synergies 紅利 8-12 月被消化、ADM SEC 和解後信任重建拖到 FY27、Cargill / LDC 不可能 IPO 或被併購。真正非共識的贏家是(a) N 肥寡占（CF / NTR）受益於中東 LNG 中斷 + 西方氮重建（urea +20% 兩週至 $650/ton），(b) DAR / INGR / AGM 三檔 non-obvious 二次受益者受益於 45Z 國產 feedstock 重排 + 飼料 cost down，(c) BG 短期 + ADM 中期但兩者反向（BG raise guide 早消化、ADM low-end $3.60 才是真實 base）。結構性 loser：MOS（純磷鉀無 N）、AGCO（農機被 Deere 碾壓）、純 commodity 中盤商若無 vertical integration。

§1 產業定義與邊界（in-scope / out-of-scope）

本 ID 涵蓋全球三大穀物（玉米 / 小麥 / 大豆）+ 主要油籽（菜籽 / 葵花籽 / 棕櫚籽 cross link）+ 其上下游價值鏈，重點放在「能交易的 public ticker」+「USDA WASDE / FAO / IGC 數據可追蹤」的環節。核心邊界判定：(a) 必須是 row crop（field crop）非園藝（fruits / veg），(b) commodity 屬性（同質化 + 集中交易所定價）非 specialty crops，(c) 全球可運輸（grain elevator + Panamax / Cape vessel）非區域消費。

為何要切「穀物 + 油籽」一起而非分開：玉米 + 大豆是美國 row crop 90%+，輪作關係（同一塊田一年玉米一年大豆 = 修復土壤氮）+ 同一條 ABCD 物流通道（大豆中盤商必同時做玉米）+ 同一肥料市場 N+P+K（玉米需 N、大豆需 P+K + 固氮）+ 同一農機（DE 玉米播種機與大豆收割機共平台），因此分析時必須一體看待。歐洲菜籽 + 烏克蘭葵花籽 + 加拿大菜籽屬於「油籽 substitutes」與大豆同 channel 在 ABCD 中流轉，但與玉米 / 小麥脫鉤，因此放入灰色地帶（油籽 substitutes）。

§1.B 灰色地帶 Insight

邊界爭議本身就是 insight 來源。本 ID 三個最重要的灰色地帶：(a) 棕櫚油 — Indonesia + Malaysia 占全球食用油 35%，是大豆油的主要 substitute；當 Indonesia B50 mandate 從 axed 復活時，棕櫚油從食品搬往 biofuel 即會推高大豆油價（因 substitute 拉漲），這是 ADM / BG 大豆油毛利的外生變數。(b) DAR Darling Ingredients — 純 RD（renewable diesel）feedstock 公司，跨「動物脂 (tallow / UCO) + 植物油」兩條原料，是 45Z 國產 feedstock 重排的最大受益者；本 ID 把 DAR 列入「non-obvious 二次受益者」而非 ABCD 同類，因為 DAR 不做 grain trading 只做 feedstock 提煉。(c) AGM Farmer Mac — 美國農地抵押二級市場，是農業金融的純 play；當 USDA 2026 政府支付提高 +$13.8B 至 $44.3B 時，農民信用 + 違約率改善 = AGM 直接受益，但市場很少把 AGM 與 grain ID 連結。

💡 §1 Insight
三大灰色地帶（棕櫚油 / DAR / AGM）是本 ID 與「純 ABCD basket」差異化的關鍵；任何只配 ADM + BG + CTVA 的 portfolio 等於押注「ABCD 重估」單一 thesis，而本 ID 主張這個 thesis 已被 FY25 H2 priced in（BG +35% from $80 to $108、ADM 從 SEC settlement 後 +18%），剩下 alpha 在三個灰色地帶（棕櫚 substitute 套利、DAR feedstock 重排、AGM 信用週期）。投資意義：任何 grains/oilseeds basket 應至少 25-30% 配置在這三個 non-obvious 名字。

§2 技術 / 產業演進 + S 曲線 + 歷史類比

§2.A 產業演進四階段（過去 → 現在 → 未來）

全球穀物 / 油籽產業並非單一線性技術 S 曲線，而是四個獨立技術 / 結構波疊加：(a) 育種 + GMO + 基因編輯（生產端）、(b) 精準農業 + AI + 衛星（操作端）、(c) ABCD 全球物流整合（中盤端）、(d) Biofuel 二次需求弧（下游端）。每個波的 S 曲線位置不同，造成投資 timing 複雜。

波 (a) 育種 + GMO + CRISPR：1996 Roundup Ready 商業化是 GMO 1.0 起點；2010s 是 GMO 2.0（多基因 stacked traits），到 2025 已達 Phase Late，傳統 transgenic 育種商業化空間有限；CRISPR 基因編輯是 GMO 3.0（不導入外源 DNA，許多國家 deregulate），Corteva 已 launch CRISPR 大豆，是下個 10 年的真正成長引擎。S 曲線位置：S 曲線中段尾，正在轉到第二條 S 曲線（CRISPR）的早期。

波 (b) 精準農業 / AI：2000s GPS guidance + 變率施肥是 1.0；2010s Climate FieldView（Bayer 250M acres）+ Deere Operations Center（400M acres）是 2.0；2020s See & Spray + AI weed detection 是 3.0（減少 60-90% 除草劑）。S 曲線位置：Phase II 中段，平台 winner-take-most 正在加速集中至 Deere（市占與資料壁壘最強）。

波 (c) ABCD 物流整合：1980-2000 是 ABCD 各自全球擴張期；2020-2025 是合併潮高峰（Bunge-Viterra $18B / 2025-07 closed 是史上最大 grain merger）；本 ID 主張 2026+ 進入 Phase Late，剩下整合空間有限（COFCO 國有、LDC 家族控股、Cargill 私有不 IPO）。S 曲線位置：S 曲線頂部進入 plateau，整合紅利 8-12 月被 priced in 完畢。

波 (d) Biofuel 二次需求弧：玉米乙醇 1.0（2000-2015 RFS 推力，玉米 40% 流向乙醇）已成熟；大豆油 RD/SAF 2.0（2022 IRA 起步，soyoil 50%+ 流向 biofuel）正進入 Phase II 早期；45Z（2025-2027）+ OBBBA（2025-07-04 簽，延長到 2029）是政策催化劑。S 曲線位置：第二弧的早中期，但被 45Z 第二批 guidance 模糊期（2026 Q2-Q3）暫時 stuck。

§2.D 歷史類比 反例（避免自我確認偏差，QC-I28）

反例 1：2010-2012 玉米乙醇推力的失敗類比 — 當年市場以為玉米乙醇 RFS 能撐玉米 $7+ 永久，事實證明 2014 後 RFS 政策遇瓶頸（blend wall），玉米乙醇用量 plateau。本次大豆油 RD/SAF 是否會重演同樣 plateau？45Z 2026 退至 $1/gal + SAF 從 $1.75 砍至 $1，已部分反映此風險；若 2027-2028 RD capacity 6B gal vs 需求 4-5B gal 形成過剩，大豆油 RD demand 同樣會 plateau。

反例 2：2008 商品 super-cycle 的「永遠 commodities」失敗類比 — 當年 Jim Rogers 等預測 commodities 進入 30 年大牛市，事實上 2014-2020 是 6 年熊市。本次「N 寡占永遠 + Biofuel 永遠」相同型態：中東 LNG 中斷若 2027 H2 解除 + 美國頁岩氣回升，N 寡占 Pricing 可能 -30%（CF 從 $90 ASP 跌回 $60）。

💡 §2 Insight
本 ID 對 2026 三大類比的綜合判斷是「不重演 1972 / 2008 / 2010-12 但也不重演 2014-16」 — 而是「中盤」狀態：ABCD EBITDA 比 2014-16 谷高 30-40%（受 Biofuel 弧 + N 寡占下檔保護）但比 2010-12 峰低 50%+（因中國需求換檔）。投資意義：避開「重演 supercycle」的單向押注（如 long ADM 2x leverage 期待玉米 $7+），改採「pair trade」結構（long N 寡占 + 中性 ABCD + short MOS）；對應 portfolio 預期 alpha vs SPX +3-5%/yr 而非 +10%+/yr。

§5 Value Chain（上游 → 中游 → 下游 SVG + 利潤池分布）

SVG 解讀：本價值鏈呈現典型「啞鈴形」利潤分布 — 上游（N 肥 + 種子）+ 下游（RD/Biofuel + 食品 mix）兩端議價權高、毛利 25-50%；中盤 ABCD 反而是利潤池中段，物流毛利薄、壓榨毛利受 crush margin 週期波動。這個結構與 2010-12 supercycle 完全相反 — 當年中盤 ABCD 是議價中心（中國需求飆升 + 物流瓶頸），現在中盤反而是 weakest link。

§5.A 議價權近 5 年轉移軌跡

從 2020 → 2026 的 5 年議價權轉移是本 ID 的結構性核心 thesis：

2020-2021（疫情 + 中國採購狂潮）：中盤 ABCD 議價權高峰 — 中國恐慌囤糧 + 蘇伊士運河受阻 + Mississippi 物流瓶頸，ADM FY21 EPS $5.13、BG FY21 EPS $13.62 雙雙創高，N 肥相對被動。議價權分布：中盤 (high) > 下游 (medium) > 上游 (medium-low)。

2022 俄烏戰爭 / 中東能源衝擊起點：N 肥議價權翻轉開始 — 俄烏佔全球 N 出口 30%+，戰爭使 urea 從 $300 跳至 $900；CF FY22 EPS $13.94 創史高，但 ADM/BG 同時受惠（高糧價）。議價權分布：中盤 (high) ≈ 上游 N (high)。

2023-2024（中國經濟換檔 + 巴西超產）：中盤 ABCD 議價權崩跌 — 巴西 2023/24 大豆 153 MMT 創高、中國 +6.5% 進口 Brazil 占比升至 73.6%、ADM SEC 醜聞（2024-01 stock -24%），中盤被夾擊。N 肥維持高位但開始正常化。議價權分布：上游 N (high) > 下游 (medium-high) > 中盤 (medium-low)。

2025-2026（南美超產 + 45Z 模糊期）：當前狀態，中盤 ABCD 議價權持續壓縮但 Biofuel 弧支撐有限；真正贏家是上游 N 寡占 + 下游 vertical integrators。CF + NTR Q1 2026 NI $619M (+59% YoY) 是直接證據；BG Q1 2026 raise EPS guide $9.00-9.50 但主因是 Viterra 整合 synergies $190M 一次性紅利，非系統性議價權回升。

2027-2030 預期：本 ID 預期議價權持續往兩端集中 — 上游 N 寡占（CF/NTR）若中東 LNG 持續中斷 → urea $700+/ton 成新常態；下游 RD/SAF + 食品 mix shift（INGR 高蛋白 + DAR feedstock）持續吸收 dollar；中盤 ABCD EBITDA 進入 plateau，相對 sell-side 共識 EPS revision -10-15%。
