★ 本 Master ID 核心非共識 Thesis 市場用 5 種錯誤框架看 AI 衝擊：① AI 是單一「科技股」故事，主要影響半導體 + LLM 廠；② OpenAI 仍是 horizontal AI 唯一贏家；③ Google 是 AI 受害者（搜尋廣告被取代）；④ Apple 在 AI 落後永久；⑤ vertical SaaS 全面被 horizontal AI 吞噬。本 Master ID 認為：① AI 是跨 6 大 vertical 的 platform layer 重塑（搜尋 / 媒體 / SaaS / 旅遊 / 零售 / 廣告），不是單一產業。每個 vertical 的勝負已開始分化 — 搜尋（GOOGL 雙刃）/ 媒體（publishers 大幅受害）/ SaaS（horizontal 雙頭 + vertical AI 整合）/ 旅遊（OTA 雙寡占成熟）/ 零售（AMZN + SHOP 受惠 / 部分受擠壓）/ 廣告（retail media + AI 重塑）。把 AI 當「半導體 + LLM」單一故事是 2024 年觀念；2026 真正 alpha 在 vertical-by-vertical 分析。② Anthropic vs OpenAI 雙頭分化已完成（2026-04 inflection）。Anthropic ARR $30B 超 OpenAI $25B 是首次；Anthropic 80% enterprise + Claude Code $1B / 6 月 + 1,000 enterprise $1M+ 客戶 = 「enterprise-first」勝出 OpenAI「consumer-first」。MCP 97M installs（2026-03）是 Anthropic 重塑 agent 經濟學的 strategic moat — 比 ChatGPT app 更開放、更 inventory-friendly。市場仍把兩者當「同一類 LLM 廠」是錯誤。③ Google 是 AI 雙刃，但 net 受惠。Google 是 publisher 上游 throttle 主謀（publisher Google 流量 -33% / Chegg -49%）但同時 Gemini 整合 Workspace + Cloud + Search 廣告 + Travel + Hotels = 自家全 stack 受惠；publisher 失去的流量很多 redirected 到 Google AI Overview 自身廣告，Google 廣告 revenue 仍成長。市場誤判「Google 衰退」是混淆 publisher 衰退 vs Google 自身。④ Apple Intelligence WWDC 2026-06 Siri 2.0 是 binary catalyst — 但實際進場 vertical 衝擊在 2027-2028。Siri 2.0 fully agentic + on-screen context + 多步任務 = 對 OTA / 零售 / payment / dating apps 是直接威脅；但 Apple 的 "invisible AI" 戰略 + privacy-first 意味著 Apple 不直接競爭 OpenAI/Anthropic，而是把 AI 整合進 OS 層；2026 WWDC 是預告，2027-2028 實際 vertical 進場才是真正衝擊。⑤ Vertical SaaS 不是被全面吞噬，而是分層存活：(a) regulated data vertical（WDAY / INTU / NOW / DOCU）安全；(b) communication vertical（HUBS / ZM / TWLO）被擠壓；(c) commerce vertical（SHOP / TOST / ABNB）受惠 AI 整合。這個三分法是本 ID 對 vertical SaaS 的核心判斷，與「vertical SaaS 全面死亡」共識相反。⑥ 2026 是 horizontal AI 的「分發 + 應用層」勝負年。分發層贏家：MSFT（Copilot Operator）+ Google（Gemini）+ Apple（Siri 2.0 - 2027 加入）；應用層贏家：CRM / NOW / PLTR / NET / Cloudflare（agentic infra）；模型層贏家：Anthropic（enterprise）+ OpenAI（consumer）+ Google Gemini；商品化 vertical：BKNG（OTA）+ TOST（餐飲）+ Salesforce（CRM）。三層分立但相互依賴。⑦ Vertical-specific 損益對照是本 ID 最大附加價值：跨 6 個 vertical 系統性對照 winner / loser，是 portfolio 配置核心工具；單 sub-vertical ID 只能看局部，Master ID 才能看全局結構。

§1 母題定義與邊界

AI 衝擊跨域母題的範圍是2024-2030 期間 AI（特別是生成式 AI / agentic AI / Apple Intelligence）對全球多個 vertical 的 platform-layer 重塑，跨 6 大 vertical：(1) 搜尋 / SEO；(2) 媒體 / publishers；(3) Vertical SaaS；(4) 旅遊 / OTA；(5) 零售 / e-commerce；(6) 廣告 / retail media。本 ID 是 master overview — 既有 5 份 AI 子議題（AgenticAIPlatform / OTAandAITravel / ProductivityCopilot / TokenEconomics / AICybersecurityDoubleEdge）+ 新增 vertical-specific 損益對照層；目的是把分散的 sub-vertical AI 故事 unify 成單一 platform layer 框架。

邊界判斷的關鍵是「是否被 AI platform layer 衝擊」與「是否在 6 大 vertical 之內」兩條軸：horizontal AI 平台（OpenAI / Anthropic / Google Gemini / MSFT Copilot / Apple Intelligence）— 全部 in-scope；vertical-specific AI 應用（Salesforce Agentforce / Toast IQ / NOW workflow）— in-scope 作為 vertical 整合 case；半導體（NVDA / AVGO / TSMC）— 部分 in-scope 作為 backbone，但詳細在獨立 ID（AIAcceleratorDemand 等）；data center 物理層（電力 / 冷卻 / 變壓器）— out-of-scope（已有獨立 IDs）；機器人（人形 / 工業）— out-of-scope；自動駕駛 — out-of-scope（不是 platform layer 重塑而是 vertical 應用）。

§2 AI 跨域演進史 + 滲透曲線 + 歷史類比

AI 跨域衝擊過去 4 年歷經 4 個結構性轉折：(1) 2022-11 ChatGPT 公開 — generative AI 大眾化起點；(2) 2023-2024 horizontal AI 平台戰（OpenAI vs Anthropic vs Google）+ 早期 vertical 整合；(3) 2024-05 Google AI Overview 全面推出 — 真正 cross-vertical 衝擊起點；(4) 2025-2026 agentic AI commercialization + Apple Intelligence WWDC 2026 預告 + Anthropic 超越 OpenAI。本次循環最特別的是 「分發 + 商業化 + 應用層」三層同時重塑：(a) 分發層 — Google AI Overview / ChatGPT / Apple Intelligence / MSFT Copilot 競爭；(b) 商業化層 — Anthropic vs OpenAI enterprise vs consumer 路線分化；(c) 應用層 — vertical SaaS 整合 AI（Salesforce Agentforce / NOW agentic SOC 等）。

本次革命最特別的是 「速度遠超預期 + 分化結構同步」：(a) Anthropic ARR 從 2024 ~$1B → 2026-04 $30B = 30x 兩年；(b) MCP installs 從 2025-12 launch → 2026-03 97M = 4 個月百萬級；(c) Apple Intelligence 從 2024-06 預告 → 2026-06 真正 agentic Siri 2.0 = 2 年產品化；(d) publisher 流量從 2024-05 AI Overview 推出 → 2025-11 全球 -33% YoY = 18 個月實質衝擊。這個速度比 2010-2015 mobile 革命快 2-3x，意味著 cross-vertical 分化的時間表壓縮 — 投資人沒有「等等看」的奢侈，必須在 2026 年內 commit cross-vertical position。

★ 歷史類比（2 個正類比 + 1 個反類比）

正類比 1：2007-2012 行動革命（iPhone + Android 對 PC 軟體的衝擊）。當年 iPhone (2007) → Android (2008) → 行動 web 主流（2010-2012）= 5 年從 0 → 70%+ 滲透；過程中桌面 search（Yahoo / 桌面 SEO）受擠壓 / 桌面廣告衰退 / 行動原生 app 公司爆發（Instagram / Uber / Airbnb）。量化對照：2007-2012 全球行動 internet 用戶從 5% → 50% / mobile commerce 從 0 → $50B；本次 AI cross-domain 從 2024-2030 預估從 5% → 70-80%，速度更快。

正類比 2：2010-2015 雲端革命（AWS / Azure 對 on-prem 軟體的衝擊）。當年 AWS (2006) + Azure (2010) → enterprise cloud 主流（2015-2018）；過程中 on-prem 軟體（Oracle / SAP 部分 / Sun）受擠壓 / 雲原生 SaaS 公司爆發（Salesforce / Workday / ServiceNow / NetSuite）。量化對照：2010-2015 enterprise cloud spending 從 $40B → $200B / on-prem CAGR -5%；本次 AI 跨域比例不同 vertical 不一致，但 horizontal AI （OpenAI / Anthropic）+ vertical 整合（Salesforce Agentforce）類比明確。

反類比：2014-2019 區塊鏈 / 加密貨幣的「全面顛覆」失敗。當年市場預期區塊鏈將取代金融基礎設施 / 物聯網 / 供應鏈 / 投票系統等多 vertical，2014-2019 大量 ICO + 平台投資；實際結果是除了 native crypto + 金融服務局部，多數 vertical 應用沒有真正商業化。反類比警示：AI 是否可能類似「過度預期 + 真正商業化局部」？特別是 vertical 之間勝負速度差大；某些 vertical（如教育 / 法律 / 醫療）可能比預期慢。多條件對照：區塊鏈當年 (a) 技術 mature 度低，(b) 用戶體驗差，(c) 監管不確定。AI 目前 (a) 技術已 mature 至能商業化，(b) 用戶體驗顯著好於 2018-2019 區塊鏈，(c) 監管雖不確定但不阻礙。多條件不成立但仍是 cautionary tale — 各 vertical 的商業化速度可能不一致，整體 cross-vertical 配置應該分散。
