Thesis（本 ID 核心非共識） NVDA 2026-Q1 發布「800V HVDC Architecture」，宣告 2027 Kyber rack-scale 全面切換，這不是單純的技術升級 —— 是電力電子產業的「7nm 代際轉換」。54V 架構 2027 起被淘汰，server PSU ASP 從 $2K → $3K+（+40%）。市場把 Delta、Vertiv 當一般 AI 受惠股，但真正的結構性論點是：(1) NVDA 把電力電子變成「co-develop 生態」、(2) 800V 認證 2 年以上 lead time，Delta 已領先 18-24 個月、(3) 2308 更像「電力電子界的 ASML」（關鍵寡占，非 TSMC 量級）—— Delta server PSU 市佔 40-45%，寡占但非單一壟斷（2026-04 patch：原「電力電子界的 TSMC」類比過強，ASML 的「關鍵寡占」定位更貼切）。每一顆 NVDA B200/GB300/Rubin 仍要搭 Delta 認證的 PSU，護城河在於認證 lead time 而非市佔量級。

灰色地帶

  1. 「Power 半導體（GaN/SiC）」算不算 power electronics？算，因為 800V 架構需 GaN / SiC；但本 ID 主軸是「系統整合」而非半導體，將 Navitas / Onsemi 列為 supporting 輔助角色。
  2. Rack-level 整合 vs Server 整合：800V 架構把電力管理從 server 內部上移到 rack / row 級，這是本 ID 的核心結構轉變。
  3. NVDA co-develop partner 算不算「獨家」？不是獨家，但 2-3 年 lead time 讓 Delta / Vertiv 領先 2 年，實質性壟斷 2027-2028 首批認證產品。
💡 Insight §1
市場常把 Delta 當「電源公司」、Vertiv 當「UPS 公司」，實際上兩者在 AI DC 產業已升級為「rack-level 整合方案商」。差別在於：做組件賣 3-5x BOM ASP 、做方案賣 2-3x system ASP。Delta 2024 還是元件廠，2026 已是方案廠。估值倍數應 re-rate 2-3x。
投資意義：Delta / Vertiv 估值的參照系不再是傳統電源 / UPS 公司（PE 15-20x），而應對標先進 rack infra 玩家（PE 30-40x）。
💡 Insight §1（二）
NVDA 800V HVDC 不是「選配」而是「必配」。從 2027 Kyber 世代開始，所有 1MW+ rack 都必須用 800V；54V 架構將被淘汰。Delta 2025-10 已在 OCP 展示 72kW + 90kW 整機方案，量產認證 2026 Q4，首批出貨 2027 H1。

💡 Insight §3（Kingmaker）
真正的 kingmaker 是「系統整合 know-how」，而非單一元件。Delta 之所以領先不是因為 GaN 技術比 Navitas 強，而是因為能把 AC-DC + DC-DC + 冷卻 + 監控組合成 rack 一體方案。NVDA 要的是 1 個交鑰匙方案，而不是 20 個元件。
投資意義：壓 Delta / Vertiv 比壓 Navitas / TI 更有 upside，因為系統整合的護城河比單一元件深。

§9.5 Kill Scenario / Steel-man 反方

反方 1：54V 架構延壽，800V 推遲到 2029

路徑：NVDA 2027 Rubin 首發選擇保守，繼續用 54V；hyperscaler 不急於轉型，因為 ROI 不明顯。Delta 2027-2028 預期的 800V 出貨大量延後。

成真時窗：2026 Q4 NVDA Rubin spec 揭示。

反方 2：NVDA 把 800V 設計「公開」，Taiwan 二線廠一起吃

路徑：NVDA 2026-2027 推動 OCP 開放 800V 規格，Lite-On / Chicony / Acbel 追上，Delta 先發優勢壓縮為 12 月。

當前證據：NVDA 2026-03 已公開部分白皮書。

反方 3：AI 資料中心成長 2027 H2 遇瓶頸

路徑：hyperscaler capex 2027 H2 放緩（類似 2022 半導體 cycle 短暫回調）；Delta / Vertiv 訂單 2028 YoY 僅 +15%。

💡 Insight §9.5
最有殺傷力的是反方 2（NVDA 開放規格）—— 因為 Delta 的溢價建立在「認證壁壘 + 先發 2 年」上，若規格被開源會壓縮溢價 30-40%。但 NVDA 目前的策略是「co-develop 式開放」而非 100% 公開，中度風險。2026-Q4 NVDA OCP 議程是關鍵觀察窗。
