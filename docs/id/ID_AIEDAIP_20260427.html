★ 本 ID 核心非共識 Thesis 市場把 EDA + IP 當成「半導體景氣 second derivative」追蹤 — 與 wafer fab 設備（AMAT / LRCX）並列 cyclical 看待。本 ID 認為：① EDA + IP 已從「per-tapeout 一次性費」結構性轉為「per-chiplet × per-tape-out × NRE 上修」三維乘數成長；chiplet 拼接讓每顆 AI ASIC 的 EDA / IP 投資從原本 $50-200M 飛升到 $400-600M（2028+ 同代雙拼），這是 sell-side 完全沒在 model 的 unit economics revolution。② Synopsys 收購 Ansys（$35B 完成 2025 年中）打開「multi-physics 整合 EDA」新類別 — 從 logic / verification 擴展到 thermal / EM / mechanical 全域模擬，這是 Cadence / Siemens 短期內無法跟進的護城河深化，但市場把它當成 M&A 整合風險而非新類別創造。③ Arm CSS（Compute Subsystems）授權模式從「IP 賣藍圖」進化為「賣 pre-validated subsystem」 — Q3 FY26 royalty +27% YoY、licensing +56% YoY 的 outlier 成長，是 hyperscaler 大規模採用 CSS 的直接證據；Armv9 + CSS 雙引擎在 2026-2030 結構性放大 per-chip royalty rate，是 Arm 從「mobile pure play」升級為「AI infra 必經 toll」的關鍵年。
💡 3 條核心 insight（預告）
1. 每顆 AI ASIC 的 EDA + IP 投資正在指數成長：從 2020 年 single die ~$50-200M（含 EDA tool license + IP licensing + design service NRE），到 2026 年 chiplet 跨代拼接 ~$200-400M，再到 2028+ 同代 leading-edge 雙拼 ~$400-600M（A14-on-A14）。Synopsys + Cadence + Arm 是這個成長的直接受益者，三家合計 EDA + IP 收費佔每顆 chip BOM 約 5-10%。
2. Synopsys-Ansys $35B 整合在 2026-2027 會兌現 multi-physics EDA 新類別：AgentEngineer 已展示 2-5X 設計生產力提升、Multiphysics-Fusion 解決 voltage drop / thermal / EM coupling 全域問題；Cadence Cerebrus + ChipStack AI 雖快但限於 logic 域；Synopsys 在 multi-physics 全 stack 整合領先 18-24 個月。
3. Arm CSS 模式 Q3 FY26 licensing +56% YoY、royalty +27% YoY 是「賣半完成品 IP」的結構性轉折：傳統 IP licensing 模式是「客戶買 license 自己整合」，CSS 是「客戶買整套已驗證 subsystem 直接用」 — per-deal value 從原本 $5-10M 飛升到 $50-100M，hyperscaler 的 ASIC 設計時間從 18 個月壓到 9-12 個月。這個結構性轉變使 Arm 從 royalty volume play 變成 royalty rate × volume 雙引擎成長。

§1 產業定義與邊界

「EDA + IP」這個主題在過去 10 年屬於半導體鏈相對 quiet 的 segment — 三家寡占、現金流穩定、但成長被視為 GDP-like。2024-2026 這波 AI 浪潮把它徹底重塑為兩個獨立的成長軌道：① EDA tool 端從 cyclical chip cycle 工具變成 AI generative chip design 平台（Synopsys.ai、Cadence Cerebrus、Siemens Aprisa AI）；② Semiconductor IP 端從 license once 變成 per-chiplet royalty 多次收費。本節把這兩條軌道的七個子市場逐一定義 — 因為每個子市場的客戶結構、競爭強度、估值倍數都差異極大。

💡 Insight §1
1. EDA + IP 邊界正在被「generative AI chip design 平台」重新定義：傳統 EDA tool（per-license / per-seat）+ design service（per-project NRE）+ IP（per-license）三段模式正在合併為「平台級 AI 輔助設計」 — Synopsys AgentEngineer、Cadence ChipStack 都是這個方向；2027-2028 可能出現按 per-tapeout / per-chiplet 收費的新 SaaS 模式。
2. Arm CSS 模式正在改變半導體 IP 的 unit economics：傳統 IP per-deal value $5-10M、CSS per-deal $50-100M（10X 提升）；hyperscaler ASIC 設計時間從 18 月壓到 9-12 月。這個轉變對 Arm 是結構性 ASP 上修。
3. Synopsys-Ansys 整合創造的「multi-physics EDA」是新子類別：Cadence + Siemens 短期（2026-2027）無法跟進；Synopsys 在 voltage drop / thermal / EM coupling 全域模擬有 18-24 月領先窗口。

§2 技術演進 + S 曲線 + 歷史類比

EDA + IP 產業的技術演進有兩條主線：第一是 EDA tool 從「規則驅動」演化為「機器學習驅動」，2018 年 Synopsys DSO.ai 是第一個生產級 ML EDA 工具，到 2026 年 Cadence ChipStack / Synopsys AgentEngineer 已演化為 multi-agent generative AI 平台；第二是 IP 模型從「per-license 一次費」演化為「per-chiplet 多次收費 + CSS pre-validated subsystem」，這個轉變使 Arm royalty 從「mobile chip 量大但單價低」變成「AI infra chip 量小但單價高」結構。這兩條主線疊加，造就了 2024-2026 EDA + IP 廠商營收 +15-30% YoY 的非線性成長。

§5 供應鏈 Value Chain

⚠ 證偽條件：若 2028 末中國 RISC-V 中高階 CPU 市佔仍 < 15% → RISC-V thesis 緩慢；若 > 30% → Arm 中國市場結構性退潮
💡 Insight §9
1. AI capex 2027 放緩是 EDA + IP 全鏈最 binding 風險：但 EDA + IP 的 lead time 較短（接 NRE 訂單 2-3 個月轉收入），下行 buffer 較 ATE / 製程設備短。
