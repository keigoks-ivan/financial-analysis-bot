§1 產業白話定義 + 歷史脈絡

白話版：中國運動服飾過去由 Nike / Adidas 主導高端、本土只佔三線。新疆棉事件後，Anta 用「多品牌 + 反向收編高端」把國際品牌的份額永久切走——但要看清楚，它真正的成長引擎是「買來的品牌」，不是自己長出來的。

白話開場：先說三個關鍵詞。国潮（Guochao）是 Z 世代對國產品牌的偏好轉向——從性價比到文化認同，2018 萌芽、2021 新疆棉事件後爆發。反向收編高端是 Anta 獨有的 playbook：用收購把外國品牌（FILA / DESCENTE / Amer）變成「Anta 旗下」並在中國本地化，賺高端段的高毛利。有機 vs 無機是本報告最重要的切口——有機是品牌內生成長（Anta 主品牌 +3.7%），無機是收購整合（Other Brands +59.2% 是買來的）。本報告問的是：這塊份額轉移是永久的還是循環的，以及 alpha 來自哪一段。

邊界界定：本報告聚焦「中國本土運動服飾如何永久切走國際品牌份額 + alpha 的有機/無機口徑」。納入國潮龍頭（Anta/Li-Ning）、國潮細分（361°/Xtep）、Anta 反向收編高端（FILA/Other Brands/Amer）、通路代理（Topsports）、國際對照（Nike/Adidas/LULU）。美國本土運動（NKE/LULU/ONON）詳見母題 ApparelFootwear。灰色地帶是「FILA China 算 Anta 的還是國際品牌」——Anta 2009 買的是「Greater China 永久經營權」，財報計入 Anta segment，但商標屬韓國 FILA Holdings；本報告將 FILA 列「Anta 反向收編高端」段。

歷史分三段。第一段：1990–2015 OEM 起家 + 本土通路擴張——裕元 / 寶成為 Nike / Adidas 代工發家；Anta（1991 創、2007 IPO）/ Li-Ning（1990 創、2004 IPO）拓三線專賣店，但高端仍由國際主導。第二段：2015–2021 国潮萌芽 + 反向收編開始——Anta 2009 收 FILA → 2019 收 Amer Sports $5.2B；Li-Ning 2018 NYFW「悟道」viral；Z 世代國產偏好首次過半。第三段：2021–2028E 永久反超 + 全球化啟動——2021 新疆棉事件加速、2022 北京冬奧 Anta 主贊助、2024 Amer NYSE IPO、2025 Anta 收 Jack Wolfskin + Puma 29% + SE Asia 1,000 店計畫。Anta 23% > Nike 20.7%（2025 confirmed）。

歷史類比一：三星 / 現代——本土份額過半後的全球擴張。韓國電子 / 汽車 1990–2010 從 OEM → 自主品牌 → 全球品牌，本土份額從 ~35% → 65% 後再向海外擴張至全球前三。量化錨：Anta 集團中國 share 從 2018 ~12.5% 到 2025 ~23%（+10.5pp），對應 Nike 同期 22.9% → ~16% 的鏡像流失——這是「本土過半前」的軌跡，類比三星預期 Anta 2030 中國 share 28–30%。差異：Anta 全球擴張靠收購 + 投資（Amer 持股 / Puma 29%）而非直接 brand build，capital-efficient 但聚合性較弱。歷史類比二：HUL 印度本地化 + LVMH 收購操盤。Unilever 透過 HUL 收本土品牌 + 本地化經營印度 OPM 25%+——FILA China 同結構（外國商標 + Anta 本地化 + Op margin ~26%），FILA 從 2009 RMB 0.6B 到 2025 RMB 28.5B（44x/15 年）是「反向收編可複製」範本；DESCENTE 2025 零售破百億、成集團第三個破百億品牌（前兩個 Anta、FILA）是「下一個 FILA 曲線」第一個實證。LVMH 收 Bulgari $5.2B 後 OPM 翻倍——Anta 收 Amer $5.2B（2019）→ IPO → ~$20B（~4x/5y），Salomon 破 $2B（+35%）。教訓：三個 playbook 同時運作，但 LVMH 類比提醒——這是無機成長，別當有機高成長付倍數。

本主題無單一價格 cycle，用「市占演變 + 三引擎成長」當時序錨。

關於 cycle 統計表的省略：運動服飾隨中國消費有週期，但本主題是結構性份額轉移（永久反超 + 反向收編），非 ASP cycle，故不立 cycle 統計表；以「中國本土崛起三段」（1990–2015 OEM/通路 → 2015–2021 国潮/收編 → 2021–2028 永久反超/全球化）的編年呈現時序。

這段歷史的可操作含義：中國本土運動同時是「三星式本土→全球份額路徑」+「HUL/LVMH 式反向收編」。bull 的底是份額未到頂（28–30% 可期）+ FILA playbook 可複製到 Other Brands；bear（對估值）是 LVMH 式無機成長若被當有機高成長定價，倍數會錯。判別一家公司，看它賣的是內生品牌力（有機）還是收購整合能力（無機）。

§2 技術成熟度 + S 曲線

為什麼是現在？因為國潮的「品牌建構棧」跨過了三道門檻：中端產品性能追平國際、文化敘事（新疆棉站隊）爆發、線上通路（Tmall/Douyin）讓本土品牌觸及一二線——三者讓 Nike/Adidas 在中端段失去「品牌溢價即護城河」的存在理由。

結構拐點有三。產品性能追平（中端）：本土跑鞋 carbon plate（Li-Ning 飛電 / Xtep 160X）、籃球 cushion（Anta KT）在中端段已追平 NB / ASICS，但頂級競技段仍弱於 Nike Vaporfly / Adidas AdiZero。文化敘事爆發：2021-03 新疆棉事件——Nike/Adidas/H&M 反棉聲明引發抵制，Anta/Li-Ning 同日宣告留用新疆棉、一夜搶下品牌新粉，Z 世代國產偏好從 2018 ~41% 升到 2024 ~78%（+37pp）。通路打通一二線：Anta 線上滲透 35%+、FILA 直營 95%+、Tmall 雙 11 sportswear top 3 全為中國品牌。三者交會，「品牌溢價即護城河」（國際品牌的中端護城河）失效；競爭優勢移到「文化認同 + 性價比 + 收購整合能力」。

§3 供給側：現在與未來 + 利潤池

中國運動服飾的利潤往「國潮多品牌（Anta/Li-Ning）+ Anta 反向收編高端（FILA/Other/Amer）」雙引擎集中——國際品牌（Nike/Adidas）與其通路代理（Topsports）失血。但利潤增量核心（Other Brands/Amer）多為無機，口徑要分。

供給端可拆四層。上游 OEM（裕元/寶成）：GM 5–12%、議價權低，本土訂單上升部分對沖 Nike 訂單下滑。中游國潮品牌（Anta/Li-Ning/361/Xtep）：2025 國產合計中國 share ~32%，GM 41–49% / OPM 11–24%；2018–2025 +20pp share gain。中游反向收編高端（FILA/Other Brands/Amer）：GM 56–72% / OPM 25–30%，是利潤池增量核心——FILA OPM ~26% / Other Brands OPM 27.9% / Amer GM 57.8%。下游通路（Topsports/Pou Sheng）：GM 40–42% / OPM 5–8%，議價權隨 Nike/Adidas 中國 share 從 42% → ~30% 崩跌。利潤往國潮 + Anta 高端 segment 集中，國際品牌與其通路代理失血。

整條鏈的利潤正從「國際品牌（Nike/Adidas）+ 通路代理（Topsports）」往「國潮多品牌（Anta/Li-Ning）+ Anta 反向收編高端（FILA/Other/Amer）」遷移：新疆棉（2021-03）把份額一夜重分配、Anta 用「私有化 → 中國本地化 → 全球反向擴張」滾起品牌矩陣、Nike 大中華連 7 季 -ve 把利潤往本土推。Other Brands OPM 27.9% > FILA 26% > Anta brand 21% 是 mix shift 向高端的證據；Topsports 淨利 -9.7% 是利潤流失的一體兩面。口徑提醒（依無機揭露原則）：Other Brands/Amer 多為 M&A 驅動的併購成長，與主品牌有機 +3.7% 不同口徑——前者反映收購整合能力，後者反映品牌內生健康。

§4 需求側：現在與未來 + 三角驗證

需求最剛性的是「Z 世代國產偏好 + 高端段可炫耀性」——國潮不只是愛國主義、是 5 層 stack 的結構現象；高端段在中國呈 K-shape（FILA/Arc'teryx/LULU 共享、mass 段被白牌價格戰侵蝕）。

第一引擎：国潮結構棧（現需）。國產崛起是 5 層 stack：L1 產品性能（中端追平、頂級弱）、L2 文化敘事（新疆棉站隊）、L3 通路（線上滲透 35%+、Tmall/Douyin 雙 11 本土 top 3）、L4 KOL 明星（王一博/谷愛凌/鄭欽文）、L5 反向收編高端（FILA/Amer/Jack Wolfskin）。Z 世代國產偏好從 2018 ~41% 升到 2024 ~78%（+37pp）是結構訊號。第二引擎：高端段 K-shape（現需）。高端 athleisure（>RMB 800/件）由 FILA + Arc'teryx + LULU 共享——LULU 中國大陸 FY25 +28%（從 +35% decel）是高端段唯一外國倖存者；Amer 大中華 +43.4% / Salomon XT-6、Arc'teryx Beta 用 wait list + 稀缺供應複製 LVMH 路徑。需求把預算推向「國潮性價比 + 高端可炫耀性」兩端，mass 段中間被 Pinduoduo/白牌價格戰侵蝕。

份額（top-down）：Anta 23% / Nike 20.7% / Li-Ning 9.4% / Adidas 8.7%（2025，Euromonitor）。偏好（消費者）：Z 世代國產偏好 ~41%（2018）→ ~78%（2024，第三方調查，量級）。高端 K-shape：LULU 中國 +28%（decel）/ Amer GC +43.4%（無機）。三邊一致——需求結構性轉向國產 + 高端兩端，mass 中間被白牌侵蝕。口徑警語：Amer +43.4% 是無機口徑（它證明 Anta 收購整合能力），不直接證明品牌有機需求；中國 sportswear 各家口徑分歧（2024 ~RMB 580B / ~$80B，2030E ~$130B，CAGR ~7.5%），以 range 呈現。

  • Priced-in：市場把「高殖利率」當 cushion、沒 price「股息正被砍」的 dividend trap——分歧 3 的股息續砍最未 priced。
→ 推論邏輯：高殖利率 + 衰退 = dividend trap，股息已砍是實證。⚠ 證偽：母公司 Pou Sheng 私有化/M&A 救援、payout 止穩。

Steel-man 反方（最有力的 3 條）

  • 反方 1（Anta share gain 已過峰）：Anta core +3.7%（從 FY22 +28% 顯著 decel）+ FILA +6.9% decel，2026–2028 進 plateau；Adidas Gulden turnaround 重啟 share gain。當前證據：主品牌有機 decel。證偽窗口：Anta 集團 FY26 H1 YoY < +5% + Other Brands <+20%。
  • 反方 2（Other Brands/Amer 是 cyclical 不是 structural）：Maia/Jack Wolfskin 是 small base、+59% 是 base effect；Salomon Sneakers 中國 hype 可能 cycle。當前證據：無機段 base 效應。證偽窗口：Other Brands FY26 H1 <+30% + Salomon GC +60%→+20%。
  • 反方 3（頂級競技盲點 + Z 世代偏好穩定性）：本土未進 World Marathon podium = brand premium 真盲點；Z 世代偏好 78% 是否延續至 α 世代未確認。當前證據：頂級 podium 100% Nike/Adidas。證偽窗口：本土跑鞋首次 podium（可能 2027+）+ Z 世代偏好 <60%。

市場最低估的：最低估的機會是分歧 2 的「Other Brands sum-of-parts rerate」（但被無機口徑限制 upside）；最低估的風險是反方 1+2 同時成真（Anta 集團 plateau + 無機段 decel，概率約「可能」）與分歧 3 的「Topsports dividend trap」。三者都該獨立監測。

三條分歧裡最可操作的是分歧 1（永久反超非循環）+ 分歧 2（Other Brands rerate，含無機口徑限制）——前者有明確領先指標（Nike 大中華 YoY），後者是 Anta sum-of-parts 重算的關鍵。分歧 3（Topsports dividend trap）是 cleanest short read。最大尾部風險是反方 1+2 同時成真。
