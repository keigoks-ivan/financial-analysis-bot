通路權力反轉（Apparel Channel Power Reversion）

發布日：2026-04-27（首篇，ID_ApparelFootwear 子題 #2）｜ 涵蓋 ticker：9 檔（3🔴 / 5🟡 / 1🟢）｜ 🟡 占比：18% ｜ 投資時鐘：Phase II → III（品牌 DTC peak 過後 → 通路反撲確認） ｜ industry-analyst v1.8

Unit Glossary：DTC（Direct-to-Consumer）= 自有店面 + 自家電商；Wholesale = 售予 DKS / FL / Amazon / Macy's / Kohl's 等通路再轉賣；1P（First Party）= Amazon 直接 vendor 進貨並控制 pricing / fulfillment；3P（Third Party）= 第三方賣家 marketplace；SSS（Same-Store Sales）= 同店銷售；FY：DKS / FL / KSS / M / JWN = Jan/Feb 結算（FY25 結束於 2026 初）；NKE = May 結算；JD Sports = Feb 結算；FY26 表示已公告 H1 / Q1 結果但全年未結算；EV = Enterprise Value（含負債）；本 ID 「通路反轉」= 利潤池從中游品牌（NKE / LULU 等）回流至下游通路（DKS-FL / AMZN / JD）的結構性現象。

★ 本 ID 核心非共識 Thesis 市場用「品牌 vs 通路」二元對立框架看 NKE / LULU 等品牌與 DKS / AMZN 等通路 — 認為通路只是 commodity 環節，brand equity 才是 alpha 來源。本 ID 認為：① DKS-FL 整併不是「兩家拼大」是「美國運動 retail 結構性 #1 確認」（DKS-FL combined ~$22B vs JD US $5.9B 領先 4x；FL 不是 DKS 的「拖累 segment」是「sneaker culture 通路 + DKS 一站式」雙引擎）；② AMZN 16.4% apparel share + ad $14.6B/Q 是被低估的雙引擎（AMZN 不只是 commodity 通路，是「廣告 + retail margin」雙引擎；ad GM 70% vs retail GM 25% 倍數差結構性；Nike 2025-05 1P 直銷回流是品牌「跪求通路」的 confirmation）；③ Nike DTC mix 從 44% peak → ~40% lock-in 是利潤池下移 confirmation 而非 cyclical（FY25 wholesale +8% > DTC -4%，Hill 上任 18 個月確認 Donahoe-era 結束；DTC 不會回到 44%）；④ 利潤池正從中游品牌（NKE / LULU / VFC）→ 下游通路（DKS-FL / AMZN / JD）+ ad ecosystem 結構性轉移。
💡 §0 4 條核心 insight（預告）
1. DKS-FL 整併產生「美國運動 retail 結構性 #1」：合計 $22B 領先 #2 JD US ($5.9B) 4x；3,200+ stores 跨 20 國是首次跨入國際；客群互補（DKS suburban / FL urban sneaker culture）。
2. AMZN apparel share 從 8.5%(2019) → 16.4%(Q2 25) 接近翻倍 + Nike 1P 直銷 2025-05 回流是「品牌跪求通路」的 confirmation。AMZN 不是「服飾通路」是「廣告 + retail margin」雙引擎 — ad GM 70% vs retail GM 25% 倍數差結構性。
3. Nike DTC 44% peak 已過，永久 lock 在 ~40%：FY25 wholesale +8% > DTC -4% 是 Hill turnaround 真正 alpha；NKE DTC 不會回到 44% Donahoe-era 高點。
4. JD Sports 是「歐洲 dominant + 美國 fastest-growing」雙身份：Hibbett 2024-07 整併後 NA 達 39% group sales；歐洲是 HOKA / On / NB emerging brands 最關鍵新通路；若 DKS-FL 整併失誤，JD 是 positioned to "pounce" 的 backup #1。

★ 灰色地帶

  • JD Sports (LSE) 是「英國股」還是「美國運動通路」？：JD 法律是 LSE 上市 UK 公司，但 NA 已達 39% group sales（FY26 H1）+ 4 年計畫 ~700 新店，已是 effectively 美國運動通路 #2。本 ID 將 JD 列核心三檔之一。
  • AMZN 是「電商 platform」還是「服飾零售」？：AMZN 自 2018 已是美國 #1 服飾 retailer（apparel + footwear share 16.4% Q2 25），超過 WMT 2x。本 ID 視 AMZN apparel & footwear segment 為 sportswear 核心通路（非單純電商 platform）。
  • NKE 「重啟 wholesale」是 cyclical 反彈還是 structural confirmation？：本 ID 視為 structural confirmation — DTC 44% peak 是 Donahoe-era 過度推進（Hill 2024-10 接手後重新尊重 wholesale），DTC mix 永久 lock 在 ~40%。詳 §10 投資時鐘論證。
  • Foot Locker：是 DKS 的子品牌還是獨立通路？：DKS CEO Hobart 確認維持 stand-alone business unit + 各 banner（FL / WSS / Champs / atmos）獨立；但 procurement / sourcing 整合（$100-125M synergies）已啟動。法律是 wholly-owned subsidiary，商業是「兩個獨立 brand」運作。
  • JWN 私有化後是否還算「通路反轉」案例？：JWN 2025-05 完成 $6.25B 私有化退市，已不是 public ID 標的；但作為 dept store 退潮 case study 仍有 historical 價值。本 ID 將 JWN 列「對照組」（已 parking）。
💡 §1 Insight
1. 「通路」不是單一概念：至少 4 個結構性差異 sub-segments — 運動專賣 / 電商 / 百貨 / off-price（不在本 ID）— 用單一「dept store decline」框架掩蓋了內部 K-shape（DKS / AMZN / JD 集體贏 vs M / KSS / JWN 集體輸）。
2. DKS-FL 整併創造「無 #2」局面：合計 $22B 領先 JD US $5.9B 4x，這是 sector 結構性 oligopoly 確認；JD 是唯一可能威脅但需 4-5 年 catch up。
3. AMZN apparel + ad 雙引擎是被忽略的隱性贏家：當市場仍把 AMZN 當「電商 platform」評估，其 apparel 16.4% share + ad $14.6B/Q +19% 是 sector 內最 underappreciated 的雙重 compounding。

💡 §2 Insight
1. NKE DTC 44% peak 已是「過熱反噬」 case study：類比 Levi's 永遠未推到此高點 — 顯示 mall-based 運動 specialty 通路（FL / DKS）對 sneaker culture 仍是 essential，純 DTC 模式無法替代。
2. Foot Locker 2018-2024 是 dept store 級的衰退：與 Macy's 同樣需要外部「整併解」（M = Reimagine 200 內部解 / FL = DKS 收購外部解）；通路本身不能 turnaround。
3. DKS-FL 整併路徑類比 Best Buy 2014-2018：3-5 年同店 SSS 才能體現整合 alpha；2026 Q3 是「Fast Break pilot 250 stores」進度檢視關鍵節點，2027-2028 是真正考驗。

§12 Non-Consensus View（共識差異）

本章列出 3 條本 ID vs 市場共識的具體分歧（QC-I12 強制可驗證）

分歧 1 — DKS-FL 整併不是「兩家拼大」是「結構性 #1 確認」 [structural]

共識（JPM / Citi 2026 Q1）本 ID
DKS 收 FL 是「rescue acquisition」 — DKS 為了搶 sneaker culture 補上 FL 拖累；FL 是 DKS 的「拖累 segment」需要 5-7 年才能止損；JD Sports 是真正威脅DKS-FL 整併創造「美國運動 retail 結構性 #1」 — combined $22B 領先 JD US $5.9B 4x；FL 不是 DKS 的「拖累」是「sneaker culture 通路 + DKS 一站式」雙引擎；synergies $100-125M 不是 cost saving 是「結構性 oligopoly 確認」
分歧原因共識把 FL FY24 EBITDA 6% 預期 vs 實際 ~3% 視為 fundamental issue；本 ID 視為「DKS 整合 + supply chain 重組」可解。共識看 short-term EPS dilution；本 ID 看 5-7 年結構性 #1 lock-in
事實基礎§4 通路 share 演變表（DKS-FL combined 11.5% → 15.5%）；§5 議價權 7 年轉移（2024-2025 通路反撲確認）；§9.5 反方 (A) 反駁論述

分歧 2 — AMZN apparel + ad 是 sector 內最 underappreciated 的雙引擎 [structural]

共識（Goldman / Morgan Stanley 2026 Q1）本 ID
AMZN 是「電商 platform」評估 — 主要由 AWS / e-commerce retail 貢獻 valuation；apparel 部分視為 mature segment（mass essentials）；ad revenue 視為 retail margin 補貼但非獨立 thesisAMZN apparel + ad 是 sector 內最 underappreciated 的雙引擎 — apparel share 8.5%(2019) → 16.4%(Q2 25) 接近翻倍 + Nike 1P 直銷回流 = 「品牌跪求通路」confirmation；ad GM 70% vs retail GM 25% 倍數差使 AMZN 「實際運動 retail 利潤率」遠超 DKS / JD
分歧原因共識用 sum-of-parts 評估 AMZN — apparel 是 retail segment 一部分 + ad 是另一條業務線。本 ID 認為兩者相互強化（ad 補貼 retail margin → apparel 競爭力提升 → 規模效應 → ad pricing power）
事實基礎§4 AMZN apparel share 翻倍；§5 AMZN ad ecosystem ~70% GM；§7 AMZN 對 vendor 議價權最高（mass scale + ad leverage）

分歧 3 — Nike DTC 44% peak 已永久 lock 在 ~40% [structural]

共識（Bof A / Citi 2026 Q1）本 ID
NKE Hill turnaround 中期目標仍是恢復 DTC 領先地位；DTC mix 將回升至 42-44%；wholesale 重啟是「短期戰術」非長期戰略NKE DTC mix 44% peak 已永久 lock 在 ~40%；FY25 wholesale +8% > DTC -4% 是 Hill turnaround 真正 alpha；Donahoe-era 過度激進是 case study，未來 5+ 年不會重演
分歧原因共識把 DTC 44% peak 視為「peak 狀態正常」 — 認為 mature DTC 應該是 sector standard；本 ID 認為 DTC 44% 是 over-extension，與 LULU 100% DTC 不同（LULU 是純 DTC 品牌 vs NKE 是 multi-channel global brand）
事實基礎§2 S 曲線 NKE DTC mix 演變；§5 議價權 7 年轉移；§9.5 反方 (B) 反駁論述
💡 §12 Insight
1. 3 條 thesis 對應 3 個獨立 long pair：(1) Long DKS / Short JD（如 DKS-FL synergies 達標 ahead of schedule）；(2) Long AMZN（apparel + ad hidden value）/ Short KSS（dept store 結構性衰退）；(3) Long DKS / Short NKE-DTC narrative（DTC 永久 lock-in confirms 通路反轉）。
2. 非共識最強的是 (2) AMZN apparel + ad 雙引擎：當 Nike 1P 回流 confirms「品牌-通路」議價權反轉，AMZN apparel 加速 share gain + ad ecosystem 補貼 retail margin = 雙引擎 compounding。
3. 本 ID 對 (3) NKE DTC ~40% lock-in 仍有 30% 概率錯誤：若 Hill 後繼者再次推激進 DTC（5+ 年後），thesis 重新校準。但短期內（2025-2027）共識認為 DTC 回到 44% 是 base case 是錯誤的。
