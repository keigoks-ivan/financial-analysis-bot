§1 產業白話定義 + 歷史脈絡

GLP-1 治療藍圖賣的不是「一個更會減肥的藥」，而是「一整個 incretin 治療生態系」——同一條荷爾蒙路徑被改造成單 / 雙 / 三激動劑、注射 / 口服雙劑型，同時打 T2D、肥胖、心血管、腎、肝（MASH）、睡眠呼吸中止、骨關節炎等多種代謝相關疾病。付錢的是 payer + Medicare + 自費患者，他們現在最痛的不是「沒有有效藥」，是「藥太貴、access 不夠廣」。

白話開場：傳統上，肥胖與 T2D 是兩條分開的治療線——T2D 用 metformin / insulin / SGLT2，肥胖幾乎沒有有效藥（早年 phentermine / orlistat 效果差、副作用大）。GLP-1 改寫了這件事：它原是腸道在進食後分泌的荷爾蒙，能促胰島素、抑食慾、延緩胃排空。把它做成長效藥（每週注射、近期到口服），就同時解決了血糖與體重兩件事，再加上 outcome trial 證明它降心血管事件、護腎、改善肝纖維化——一個分子打開了一整片代謝病的治療版圖。所謂「治療藍圖」（treatment roadmap），就是這條路徑從 mono → dual → triple、從注射 → 口服、從 T2D → 6 種適應症的整個演進地圖。

邊界界定：本報告聚焦「incretin-based therapeutics 的治療藍圖、玩家分化、定價演進」這條主軸。納入 GLP-1（semaglutide / orforglipron）、GLP-1+GIP（tirzepatide / VK2735 / CT-388 / MariTide）、GLP-1+GCG（mazdutide / survodutide / pemvidutide）、GLP-1+GIP+GCG triple（retatrutide）、GLP-1+amylin（CagriSema / amycretin），以及上游 API 製造、CDMO 填裝、device delivery 三大 picks-and-shovels 環節。下游消費 / 餐飲 / 食品衝擊由兩個兄弟 ID 涵蓋、本 ID 不重複。out-of-scope 是純 amylin mono、MC4R agonist、leptin analogue、手術治療、phenotype 藥。灰色地帶在「amylin combo 是否算 incretin」（CagriSema / amycretin 列入、但 amylin mono 排除）與「MariTide 的 GIPR antagonist」（與 tirzepatide 的 GIP agonist 機制相反，列入但標註待 Ph3 驗）。

歷史脈絡分四個轉折。第一次轉折：1986–2005——1986 年 Joel Habener 發現 GLP-1，1996 年 exenatide 從 Gila monster 唾液分離，2005 年 Byetta（exenatide）成為 first-in-class FDA 核准——但只是 niche 注射 T2D 藥，user < 100K。第二次轉折：2005–2019——liraglutide（Victoza 2010 每日）→ semaglutide（Ozempic 2017 每週）→ Rybelsus（2019 oral）逐代迭代，NVO 主導第一代 mono GLP-1，仍主要是 T2D。第三次轉折：2020–2025——Wegovy（2021 第一個 obesity-only）+ Mounjaro（2022 第一個 dual GIP+GLP-1，LLY 進場）+ SELECT/FLOW/ESSENCE outcome trial 證 CV/CKD/MASH 效益，user 從 1M 衝到 ~11M；供給（藥能不能造）是 binding constraint。第四次轉折：2026 起——口服 Foundayo 核准 + triple agonist retatrutide TRIUMPH-1 讀出 + MFN/Medicare pricing 重塑，進入「口服切換 + 次世代讀出 + 定價重塑」三因子並行的 Phase III。

2026 上半的硬事實基準：與一季前相比，三件 catalyst 已從「預測」變「已落地」——2026-04-01 FDA 核准 orforglipron（Foundayo），首個無食水限制的 oral GLP-1 small molecule（不像 Rybelsus 有空腹 30 分鐘限制）；2026-05-21 LLY 公布 retatrutide TRIUMPH-1 樞紐試驗 −28.3%（80 週），是史上 Phase 3 obesity 最大減重；2026-01-05 NVO 的 Wegovy 口服在美上市、首季 ~1.3M 處方。這些把「GLP-1 會不會繼續長」這個問題基本上回答完了——剩下的是分化與定價。

§2 技術成熟度 + S 曲線

為什麼分化是現在發生、不是 2022 年？因為兩個技術 bottleneck 剛被解開——「triple agonist」讓減重效力突破 −28% 天花板（mono ~15% → dual ~20% → triple ~28%），「oral small molecule」讓 access 成本斷崖式下降（注射 cartridge 寡占 vs 口服一錠）。兩者同時兌現，才把競賽從「同一機制比劑量」推到「跨機制 + 跨劑型」的多軸分化。

GLP-1 的 S 曲線正處陡峭段中段。過去「減重藥」的效力天花板被機制鎖死——mono GLP-1（semaglutide）約 −15%，要再往上得加機制。技術拐點一是 多激動劑（multi-agonist）：tirzepatide 加 GIP 到 ~−20%，retatrutide 再加 GCG（glucagon，提升能量消耗）到 −28.3%（TRIUMPH-1 已讀出）——每加一個受體靶點，效力階梯式上一階。拐點二是 口服 small molecule：Foundayo（orforglipron）是非胜肽小分子，可口服且無 Rybelsus 那種空腹限制，自費 $149/月——把「想用但用不起、或怕打針」的 access barrier 大幅拉低。三是 適應症外擴：每個 outcome trial（CV / CKD / MASH / OSA / OA）開啟一個新的 reimbursement source，把「減重藥」重新定位成「代謝病平台藥」。

把治療藍圖的演進畫成 user 累積 S 曲線——看現在在採用曲線的哪一段，決定是早買還是追高。

§3 供給側：現在與未來 + 利潤池

供給端的結構是「第一陣營雙寡占 + 第二陣營 differentiated 挑戰者 + device 4 寡占」。瓶頸已不是產能（美國急性短缺 2024–25 解除），而是「誰握有 best-in-class 臨床數據 + 適應症 dossier + cartridge 寡占產能」這三道稀缺——利潤被有數據 + 有產能的玩家牢牢吃住。

現供結構：美國的 tirzepatide 短缺於 2024-12-19 解除、semaglutide 於 2025-02-21 解除（FDA 移出 shortage list），隨後 compounding 503A/503B enforcement discretion 在 2025 年中陸續終止——「藥造不出來」這個 2021–24 的 binding constraint 已消失。2026-04-30 FDA 進一步提案把 semaglutide / tirzepatide / liraglutide 排除於 503B bulk drug substances list（無短缺即無臨床需求），意見徵詢至 2026-06-29，等於關上合法大量 compounding 的門。新的 binding constraint 改由「臨床數據護城河」與「borosilicate cartridge 寡占」定義。

§4 需求側：現在與未來 + 三角驗證

需求最不可替代的一塊是「肥胖 + 代謝病的龐大未滿足醫療需求」——美國成人約 12% 在用 GLP-1，潛在 pool > 100M；最能放大需求的引擎是 access 打開（口服核准 + Medicare 覆蓋 + D2C 降價），但同一批工具把 blended ASP 結構性下壓——量與價在 2026–2030 拉鋸。

現需結構：採購主體是 payer + Medicare + 自費患者，需求 price inelastic 程度高（肥胖與 T2D 是慢性、剛性需求）。美國成人約 12% 在用 GLP-1（任何用途，KFF），自報用於減重者約占成人 ~12.4%（≈ 30M+，含品牌 + compounded + 自報），其中兩大藥廠品牌 user 約 10M、潛在 pool > 100M。兩個需求引擎：①access 打開——Foundayo 口服 2026-04 核准（自費 $149/月、商保 $25、Medicare bridge $50 Jul–Dec 2026）、Wegovy 口服 2026-01 上市、MFN $245/月、TrumpRx D2C ~$350（起始劑量 $199）、Medicare Part D 肥胖試點 ~2026-07；②適應症外擴——CV/CKD/MASH/OSA/OA 各開新 reimbursement。但同一批 access 工具把 blended ASP 從 2025 ~$700/月 → 2030E ~$370–400/月（約 −45%）。

§7 Non-Consensus + Priced-in 檢驗 + Kill Scenario

⚠️ 本章含 judgment：每條分歧附 [F:] cornerstone + [X:] 三情境 + priced-in 檢驗。

🟡分歧 1：「patent cliff 2031–32 將毀滅 LLY/NVO」信心：高
  • 共識說 X：semaglutide 美國專利 2031–32 到期，biosimilar 入場將如他汀式 −85% 崩。🟢 [F: T2 — sema US 專利 2031-12 / 2032]。
  • 我們說 Y：GLP-1 有 5 重 defense（vs 他汀只有 2 重），衝擊更可能溫和 ≤ 20%——(a) tirzepatide US 保護到 2036+；(b) retatrutide / Foundayo / amycretin / CagriSema 次世代梯次接棒；(c) 適應症 outcome trial 各別 dossier 門檻；(d) cartridge 4 寡占供應壁壘；(e) formulary 位置。🔵 [X: base 很可能 — 衝擊 ≤ 20%；bull 可能 — 次世代 portfolio 完全接棒、衝擊更小；bear 可能 — 政治疊加 LOE 放大]。

  • 共識說 X：oral efficacy 低於注射（Foundayo ATTAIN-1 −12.4% vs tirzepatide ~−20%），患者一旦 tolerate 注射切換動機弱，oral 2030 < 25%。🟢 [F: T1 — ATTAIN-1 36mg −12.4%（NEJM 2025-09）]。
  • 我們說 Y：Foundayo $149 自費 + 商保 $25 + Medicare bridge $50 已 break access ceiling，加上 VK2735 oral + CT-996 第二波，oral 切換比注射第一波快——本 ID 判 oral 2030 ~35–40%。🔵 [X: base 很可能 — ~30–35%；bull 可能 — ~40%+；bear 可能 — < 25%（efficacy gap 主導）]。
  • Priced-in：LLY 現價部分 price 了 Foundayo $5–10B revenue forecast；若 oral 滲透僅 15–25%（反方），Foundayo 下修至 $2–4B → LLY 2030 EPS −3–5%，且 device（cartridge）受惠更高（注射持續主導）。這條分歧 part-priced，可操作性中等。
→ 推論邏輯：access 比 efficacy 更能驅動 oral 切換速率。⚠ 證偽：若 Foundayo 上市後季 run-rate 連兩季 < $500M（vs base $1.5B），oral 滲透 35–40% 論下修、device 受惠論上修。
🟡分歧 3：「MFN $245 是 LLY/NVO 營收末日」信心：中
  • 共識說 X：MFN $245/月 + TrumpRx $350 把 net price 砍崩，是 LLY/NVO destruction。🟢 [F: T1/T2 — MFN 已簽、TrumpRx 2026-01 上線 ~$350、起始 $199]。
  • 我們說 Y：MFN 不是 destruction 而是 mix shift——Medicare 5–7M user × $245 是純新增 volume 而非替代既有 net price；blended ASP −45% 被 user 11M→25M 倍增抵銷，雙巨頭 2030 incretin 營收仍 +30%。🔵 [X: base 很可能 — 量抵銷價；bull 可能 — 適應症全開 TAM 上修；bear 可能 — 2027 二次砍至 $150 量不足抵銷]。
  • Priced-in：LLY Q1 26 +56% 量能已證明「量在抵銷價」，市場部分 price 了；最未被 priced 的是「2027–28 政治二次砍」的下行風險——現價隱含 blended ASP 守 $400–500，若政治把 Medicare 砍至 $150、commercial 至 $300，blended 2030 跌到 ~$260，高 P/E 無緩衝。這是最該獨立監測的尾部。
→ 推論邏輯：MFN 是 mix shift 非末日，市場只 price 了「量抵銷」一面、沒 price「二次砍」。⚠ 證偽：2027 commercial net price 破 $300 或 Medicare 二次 negotiation 涉 GLP-1 = 分歧 3 的 bear 成立。

Steel-man 反方（最有力的 3 條）

  • 反方 1（pricing 二次砍）：MFN $245 只是 first step，類比 insulin 2018–24 net price −55%，2027–28 民主黨選舉週期把 GLP-1 當 affordability 主議題再砍至 $150。當前證據：2026-04-21 CMS 已取消 BALANCE Medicare Part D 2027 部分（政策搖擺）。量化影響：blended ASP 2030 → $260；LLY 2030 incretin $65B → $45B、NVO $45B → $32B；股價 −25–35%。證偽窗口：2027 commercial net price 破 $300 或 Medicare 二次 negotiation。
  • 反方 2（discontinuation + muscle loss narrative）：1 年 discontinuation 已 ~50%、muscle mass loss 報告 25–40% lean body mass——當 25M user 規模化，narrative 進主流可能觸發 FDA labeling 警告 + prior auth restrict，壓 user 至 18–22M。證偽窗口：2027–28 retatrutide / CagriSema / amycretin 三個次世代以「solve discontinuation + muscle」差異化框定 narrative。
  • 反方 3（次世代 binary miss 連鎖）：CagriSema FDA CRL 或 VANQUISH / MARITIME 多個 readout miss，引發 sector de-rate。當前證據：2024-11 CagriSema REDEFINE-1 −22.7%（vs 25% target）曾使 NVO 一週 −27%。證偽窗口：CagriSema FDA ~2026-10 過 + VANQUISH-2 ~2026-Q3 ≥ 18%。

市場最低估的風險：不是 patent cliff（人人都知 2031–32）、也不是 oral 滲透（已被廣泛討論），而是分歧 3 的 pricing 二次砍——市場把 MFN 當「已知一次性」，最沒準備好「2027–28 政治再砍一輪」這個情境。這是本報告認為最該獨立監測的尾部風險。

三條分歧裡最可操作的是分歧 1（LOE 衝擊溫和）——最未被 priced（高 P/E 隱含沒崩）且有 5 重 defense 支撐；分歧 3（MFN mix shift）次之、量抵銷已 part-priced 但「二次砍」尾部未 priced；分歧 2（oral 滲透）方向對但 part-priced。對 PM：押 LLY「主題 + 韌性」是分歧 1 的表達，但要盯分歧 3 的 pricing 尾部當降 conviction 觸發。
