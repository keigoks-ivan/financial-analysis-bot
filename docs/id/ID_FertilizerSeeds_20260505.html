§1 產業定義與邊界（in-scope / out-of-scope）

本 ID 是 GrainsOilseeds ID 的上游 deep-dive，聚焦在「投入端」（farm inputs）— 肥料 NPK + 種子 + 作物保護（CP）。三段共享同一個下游客戶（農民）+ 同一個 demand cycle（年度 acreage），但結構上是三個獨立寡占市場（N+P+K 各自不同玩家、種子 5 強、CP 集中在四大化學廠），因此分開深度分析。

為何要切「肥料 + 種子 + CP」一起而非分開：(a) 三者都是農民「投入端 OPEX」，同一個 budgeting cycle（春耕前 12-24 月決定）；(b) Nutrien Retail 是同時銷售三段的 #1 北美零售平台，公司財務上不可分；(c) Bayer Crop Science + Corteva + BASF 各自橫跨「種子 + CP」雙引擎（CTVA 種子 70% + CP 30%；BAYN.DE Crop Science 為 Bayer 三段之一）；(d) 同樣的環境法規（45Z / EU CBAM / 中國 export ban）同步影響三段。但三段「結構重塑邏輯」獨立 — N 是 LNG / 中東地緣、P 是中國退出、K 是 Belarus 制裁、種子是 GMO/CRISPR + Roundup 訴訟 — 因此分章節寫。

§2 技術 / 產業演進 + S 曲線 + 歷史類比

§2.A 產業演進（NPK + 種子四段 S 曲線）

肥料 / 種子產業是 100+ 年歷史的成熟產業（Haber-Bosch 1913 / GMO 1996），但 2025-2030 是多重新 S 曲線疊加期：(a) 綠氨 / 藍氨（N 段第二弧），(b) 中國 P 退出 + OCP / Saudi 接位（P 段重塑），(c) BHP Jansen + Belarus 制裁（K 段格局重組），(d) CRISPR + Vylor split（種子第二弧）。每個弧時間尺度 12-36 月不同，造成投資 timing 各異。

波 (a) 綠氨 / 藍氨：1913 Haber-Bosch SMR 製氨成熟；2020-2025 藍氨 (CCS) 起步；2026-2030 綠氨（renewable H2 + Haber）商業化；2030+ 海運 marine fuel + 亞洲電力 default。S 曲線位置：第二弧的 Phase I 末 → II 早期，Yara + CF + Air Products 是 first movers，全球 marine fuel adoption 2030+ 才達臨界量。

波 (b) 中國 P 退出 / 全球 P 重塑：1990s-2024 中國以 5% 全球磷礦儲量 + 30% 全球 P 產量主導全球 P 出口；2024-12 業界共識 suspend exports till 2026-08；硫磺成本 +200% + 中國糧食安全戰略推動，2026-2030 中國 P 出口將永久 retreat 30-50%；OCP Morocco（70%+ 全球儲量）+ Saudi Ma'aden + MOS（美國本土）接位。S 曲線位置：「中國 P 退」是 Phase I 早期（2024-12 業界宣告 → 2026-08 review → 2027+ 結構性變化），全球 P 定價權重塑進行中。

波 (c) Belarus 制裁 + Jansen 上線：1990s-2021 Belaruskali（一國占全球 K 約 20%）+ Uralkali 主導全球 K 出口；2021-08 美 + EU + 立陶宛先後制裁；2025-12-13 美國解除 Belarusian K 制裁但 EU 仍封；Belarus 仍依賴 Russia Baltic 終端 + 中國鐵路。同時 BHP Jansen Stage 1 (4-5 MMT/yr capacity) 2026 H2 投產。S 曲線位置：K 段在 Phase Late + 新進入者（BHP），玩家格局 2026-2027 重組。

波 (d) CRISPR + Vylor：1996 Roundup Ready GMO 1.0；2010s stacked traits GMO 2.0；2024 Corteva CRISPR 大豆 launch；2026-Q4 Vylor split 後成全球唯一純種子 public play。S 曲線位置：第二弧 Phase II 早期 — CRISPR 商業化中，但中國 / 歐盟 deregulation 進度將決定全球 ramp 速度。
