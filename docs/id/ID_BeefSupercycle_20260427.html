美國牛肉超級循環產業鏈（Beef Supercycle Value Chain）

發布日：2026-04-27 ｜ 涵蓋 ticker：21 檔（7🔴 / 7🟡 / 7🟢）｜ 🟡 占比：19% ｜ 投資時鐘：Phase II（trough 年 + heifer paradox + Mexico screwworm 三重壓力） ｜ industry-analyst v1.8

Unit Glossary：Cattle herd = 全美牛數（USDA NASS Jan inventory；含 cow + calves + bulls）；Beef cows = 母牛繁殖群（決定 calf crop 上限）；Heifer retention = 母小牛留作繁殖（vs 上市屠宰）；Feeder cattle = 500 lb 待育肥小牛（feedlot 入廠價）；Live cattle = 1,400 lb 育肥完成牛（屠宰價）；Boxed beef cutout = 屠宰後分切批發價；Packer spread = 牛肉批發 - 牛隻採購（packer 毛利核心）；Cattle on feed = 進入 feedlot 等屠宰的牛（slaughter 提前指標）；USDA NASS = National Agricultural Statistics Service；RLM = restaurant-level margin。所有 financials = Q4 FY25 + Q1 FY26 actual（4 月已公告）。

Today Phase

II（trough + 三重壓力）
★ 本 ID 核心非共識 Thesis 市場用兩種錯誤框架看美國牛肉週期：① 「standard 10 年週期 = 2027 H2 應已開始 rebuild」（多數 sell-side 用歷史 reverse engineer）；② 「全 beef chain 都壞」（看 TSN / JBS 巨額損失就推 entire space avoid）。本 ID 認為：① 這是 supercycle 不是 normal cycle — 4 重壓力疊加（heifer retention paradox + Mexico screwworm 1B lbs 損失 + Brazil tariff 反復 + 2011-2014 drought 殘留）= 2028-2030 才見產量回升；② 「packer 損失 = restaurant 機會」是反向（packer trough 2026 確認 → restaurants 2026 H2 / 2027 開始 see 牛肉成本 plateau → TXRH/DRI traffic 強勢 + RLM trough 後 2027-2029 multi-year EPS expansion）；③ PPC 是 cleanest long（chicken/beef parity 5 年極端 + FY25 record EBITDA $2.3B 已驗證 + JBS 旗下集團內 hedge 架構）；④ YUM Taco Bell Cantina Chicken Menu + 2030 chicken $5B 目標是最被低估的 chicken pivot alpha（從 ground beef 結構撤退）；⑤ ZTS 是 picks-and-shovels 隱性贏家（牛群縮小 → 每頭價值升 → 動物保健支出上升）。
💡 §0 4 條核心 insight（預告）
1. 「Heifer retention paradox」是 supercycle 核心機制：高 calf 價格（$400/cwt feeder）反而更激勵賣 heifer（$1,800-2,000/頭即時 cash）— 留作繁殖機會成本 $3,000+ + 2 年 forage 等待。USDA July 2025 報「最小 beef replacement heifer inventory 1973 以來」 — 復原延後至 2028-2030。
2. Mexico screwworm 從 2025/05 起鎖死美墨邊境：~1B lbs 牛肉損失（TX/OK/NM 三州 2026 估），相當 5% feedlot placements。Tamaulipas 2026/01 detection 60 miles 距德州 — 2026 邊境短期內無法重開。
3. Brazil 40% tariff 2025/11/20 拿掉：partial relief 但結構性 supply shortfall 不變；imports 2025 +16% 至 5.4B lbs，2026 +3%。imports 不是 silver bullet — 美國 grades / cuts demand 與 Brazil supply 結構不對齊。
4. PPC FY25 EBITDA $2.3B record + revenue $18.5B record：CEO 直白「QSR traffic was impacted by beef prices」 + 「chicken-centric QSRs leveraging affordability to drive traffic」 — chicken 是已驗證 beneficiary，不是預測。

★ 灰色地帶（5 個 boundary call）

  • Pilgrim's Pride（PPC）算 chicken 還是 beef 受惠 chain？：PPC 是 chicken pure-play 但 JBS 旗下；JBS 集團 hedge 架構（Beef NA loss + chicken 補）。本 ID 列為 🔴 chicken 但 cross-tier 受益 beef supercycle。
  • Cargill 是 Big-4 但 private — 算入嗎？：算入 segment 分析但無 stock。Cargill FY25 May 5 年最低 profit（Reuters）— 等同 TSN/JBS。
  • BYND 是受惠還是受害？：原以為「beef 替代品」應 +，但實際 FY25 -15.6% revenue / Q4 GM 2.3% — plant-based meat 結構性失敗，本 ID 列為「failed thesis」對照組。
  • HRL 算受惠還是中性？：Hormel pork + 加工肉是 beef trade-down 受惠，但 beef trim cost +12% 是 input pressure（Spam / Black Label bacon 成分）— net 受惠但 magnitude 弱。
  • Tariff / 政治變數放哪？：本 ID 在 §9 風險矩陣放 Brazil tariff（已 11/20/25 部分 rollback）+ Mexico screwworm 邊境 — 因兩者直接影響 supply 而非單純 macro。
💡 §1 Insight
1. Beef supercycle 是 cross-tier 衝擊 — 不能單看 packer 或 restaurant：value chain 11 個 segment 同時 affected 但方向相反。stockpicker 需要 cross-tier 視野（packer short / chicken long / cold chain 中性 long / 牛排館看 traffic 強弱）。
2. 「Big-4 packer 寡占」是核心結構特徵：85% fed cattle / 67% hogs concentration → 但這次循環 packer 仍虧損證明 oligopoly pricing power 在 supply shortage 時失效（cattle bid 戰爭）。

★ 歷史類比（3 個前例）

前例核心 pattern本次相似 / 差異
類比 #1：1973-1975 牛群高峰崩跌（132M peak → 1979 109M）1972 大豆禁運 + 1973-74 corn 通膨 → cow-calf operator 全面 liquidate；歷時 5 年下行；之後也是 5 年 rebuild結構相似：cow liquidation + 高 input cost。差異：1970s 是 single-driver（feed cost）；2021-2026 是 4 重 driver（drought + feed + COVID + Mexico screwworm）— 復甦時間更長 + 不對稱。
類比 #2：1996-2014 Texas drought 雙週期1996 第一次 drought / 2011-2014 第二次 — 重複壓力造成 cow-calf 永久流失（1996 之後 cow base 從未完全回到 peak）高度相似：本次 cow base 比 2014 trough（29M）還低（27.6M） — 證明 drought 殘留 +後續壓力導致 secular decline。差異：2014 後 5 年內反彈到 31.7M（94M total），但這次後可能無法回到 31M+ — 結構性 lower。
類比 #3：1980s Detroit 三大車廠 + 2008 GM Ch.11美國成熟產業 + 工會 + 結構性高成本 → 一旦衝擊（油價 / 進口）就 multi-year 復甦（GM 2008-2010 Ch.11，2011 才 IPO）類比有限：beef packer 不是 unionized to GM 程度；但 packer top-4 寡占 + plant fixed cost 結構 → 衰退期 plant 關（TSN Lexington 2025）+ multi-year capacity rationalization 確相似。
💡 §2 Insight
1. 本次 supercycle vs 1973-1979 normal cycle 最大差異是「multi-driver compound shock」：1970s 一個 trigger 5 年復甦；2020s 4 個 trigger 同時 → 復甦不可能 5 年內完成，2030+ 才見初步 rebuild。
2. cow base 永久 lower 是真正的 supercycle 含義：每次 trough 都比上次低（29M → 27.6M）— US beef production capacity 永久 step-down，imports 結構性吃進更大 share。
3. 類比 GM Ch.11 啟示：packer plant rationalization 是 healthy 訊號：TSN Lexington 關 + Amarillo 縮減 = 「fewer packers chasing fewer cattle」 = 2027-2028 turn 後 packer margin 結構 healthier。

💡 §3 Insight
1. Heifer retention paradox 是 self-perpetuating cycle：高價時不留 → 後續供給更緊 → 價格更高 → 再更不留。要打破這循環需要 calf 價跌至 $300 以下 + corn 仍便宜 — 至少 2027-2028 才可能。
2. Mexico screwworm 不僅是「短期供給衝擊」是「長期 trust deficit」：即使邊境重開，US 進口商會謹慎；Mexico 復原通常 18-24 個月。
3. Imports 5.4B lbs 不是 silver bullet：美國 domestic shortfall（含 screwworm 1B lbs）達 ~3-4B lbs — imports 最多補 1-1.5B lbs，剩下要靠 chicken 替代 + retail price 上升 destroy demand。

§12 Non-Consensus View（共識差異）

🟡分歧 #1：「不是 normal 10 年週期 — 是 4 重 driver supercycle，2030+ 才見 rebuild」信心：高
  • 共識：sell-side 多用「歷史 cycle 10 年」reverse engineer 預測 2027-2028 rebuild（Bank of America / Stephens / Goldman 2024-2025 多份報告）
  • 本 ID 看法：4 重 driver compound shock（heifer paradox + Mexico screwworm + Brazil tariff 反復 + 2011-2014 drought 殘留）— 不是 single-trigger cycle；2028-2030 才見産量回升 / 2030-2033 完整 rebuild
  • 分歧事實基礎：本 ID §3 表 + USDA July 2025「最小 beef replacement heifer inventory 1973 以來」+ Mexico screwworm 2026/01 Tamaulipas 60 mi
⚠ 證偽條件：USDA 2027 Jan inventory ≥ 88M（normal cycle）+ Mexico 邊境重開 + Brazil tariff 全撤 — 三條件同時 confirmed 則 supercycle thesis 部分作廢。
🟡分歧 #2：「PPC 是 cleanest long supercycle / 不是 cyclical chicken」信心：高
  • 共識：sell-side 把 PPC 當 cyclical chicken commodity / 12x P/E 是「fair multiple for cyclical」；忽略 chicken/beef substitution 結構性 alpha
  • 本 ID 看法：PPC FY25 record EBITDA $2.3B 不只是 cycle 高點 — 是「chicken QSR mix 結構性 shift」初期；Taco Bell 2030 $5B chicken + MCD 加速 = 結構性 demand 上升至少 5 年；JBS 集團 hedge 架構降 governance 風險
  • 分歧事實基礎：本 ID §6 + §8 — PPC 14x P/E vs TXRH 24x / SHAK 50x 顯著低估
⚠ 證偽條件：FY26 H1 PPC EBITDA YoY ≤ +5%（cycle revert）+ Taco Bell Cantina Chicken comp +3% 以下（chicken pivot under-deliver）— 雙條件 confirmed 則 PPC structural alpha 部分作廢。
🟡分歧 #3：「TXRH FY26 menu 漲 +1.9% 是 traffic-bet 不是漲價失策」信心：中
  • 共識：部分 sell-side 認為 TXRH FY26 漲價 +1.9% 過低 — 不能 cover 7%+ beef inflation → margin 持續壓縮 → multiple 應壓
  • 本 ID 看法：管理層紀律性「賭 traffic 守住 + 等 trough 後 multi-year EPS expansion」是正確 trade-off；TXRH +1.9% traffic Q4 FY25 + +4.9% FY = pricing power 強而 disciplined；MCD / WEN 漲價 +14% / +38% since 2020 destroy demand 是反例
  • 分歧事實基礎：本 ID §7 ASP 表 — TXRH +25% since 2020 vs MCD +44% / WEN +38% 對照
⚠ 證偽條件：FY26 H1 TXRH SSS 跌至 +1% 以下 + Q2 RLM 跌至 12% — 「紀律 backfire」confirmed。
