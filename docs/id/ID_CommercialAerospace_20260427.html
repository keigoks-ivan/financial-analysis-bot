商業航太復甦（Commercial Aerospace Recovery）

發布日：2026-04-27（v2.0 深度重寫，整合 Boeing Q1 26 143 jets reverse + GE LEAP 520 deliveries / +63% YoY + RTX GTF Advantage EASA 4/17 認證 + GE Aerospace Q1 26 +29% rev 最新數據） ｜ 涵蓋 ticker：12 檔（4🔴 / 5🟡 / 3🟢）｜ 🟡 占比：12% ｜ 投資時鐘：Phase II 中 — OEM-engine-aftermarket 三段同時 inflection ｜ industry-analyst v1.8

Unit Glossary：OEM = Boeing / Airbus 主機廠；narrowbody / widebody = 單走道 (737/A320) / 雙走道 (787/A350)；LEAP = GE/Safran CFM 共同開發 narrowbody 引擎（A320neo + 737 MAX 唯一引擎）；GTF = Pratt & Whitney Geared Turbofan（A320neo + A220 + E-Jet E2）；shop visit = 引擎進廠大修（每 8-12 年一次，aftermarket 收入主來源）；RPK = Revenue Passenger Kilometres（航空旅運需求指標）；backlog = OEM 未交機訂單；book-to-bill = 訂單 vs 出貨比；FAA / EASA = 美國 / 歐洲航空認證機構；FOD = Foreign Object Debris（Boeing 品質事件起因之一）；737 MAX 7/10 = 待認證的 MAX 變體。所有 financials = Q1 2026 actual。

★ 本 ID 核心非共識 Thesis 市場用「Boeing 是受傷王者 / Airbus 結構領先」框架看商航 — 認為 Boeing 永遠 lag，Airbus 是 default winner。本 ID 認為：① Boeing 翻轉是真實的（Q1 26 143 jets / 137 commercial / 7 年來首次 quarterly deliveries 超 Airbus；737 rate 42→47/mo；MAX 7 + MAX 10 2026 認證 in pipeline；vs 共識仍 deeply skeptical）；② GE Aerospace LEAP aftermarket 是 sector 最強單一 alpha（installed base 2024-2030 三倍 + shop visit volume 2025 +27% + aftermarket margin 2028 接近 CES service margin）；③ RTX GTF 從 disaster 變 quietly compounding alpha（groundings Q1 26 -15% + GTF Advantage EASA 4/17 認證 = inflection；market 仍將 GTF 視為 BA-tier 永久 overhang）；④ Aircraft cycle 是 multi-year structural tightness 非循環（OEM combined backlog 14,000+ / 8 yr 覆蓋 + production rate 受 supply chain 限制 = 是「想增也增不快」非「想增不想增」）。
💡 §0 4 條核心 insight（預告）
1. Boeing Q1 26 143 jets / 137 commercial / 約 114 是 737 MAX：是 7 年來首次 quarterly deliveries 超 Airbus；不是「lucky quarter」而是 production rate 從 38 升到 42 的直接結果。
2. GE LEAP Q1 26 deliveries 520（+63% YoY from 319）+ shop visit volume 2025 +27%：installed base ramp 已啟動，aftermarket 2nd derivative 進入收割期；2028 aftermarket margin 接近 CES service。
3. RTX Pratt GTF groundings Q1 26 -15%（QoQ 改善）+ GTF Advantage EASA 4/17 認證 (A320neo)：2024-2025 disaster narrative 已開始反轉；market 仍 deeply skeptical = pricing inefficiency。
4. Boeing + Airbus combined backlog 14,000+ / 8 yr 覆蓋：cycle 不是「想增不想增」而是「想增也增不快」 — supply tightness 結構化。

★ 灰色地帶

  • RTX 是「商航」還是「國防」？：Pratt + Collins 是 commercial 主場（佔 RTX revenue ~55%），Raytheon 段是國防；本 ID 與 ID_Defense 雙列（差異化在 Pratt GTF + Collins）。
  • GE Aerospace 是純 commercial 嗎？：~75% commercial（LEAP + Trent + 9X aftermarket）+ 25% defense (F414 / T700)；是 commercial-dominant pure-play。
  • Spirit AeroSystems (SPR) 是「Tier 1」還是「Boeing 子公司」？：Boeing 2025 宣布回購 — 灰色地帶；本 ID 仍列 Tier 2，但 catalyst 是 2026 deal close。
  • Honeywell aerospace 是 industrial 還是 aerospace？：Honeywell 2026 H2 分拆 → Honeywell Aerospace 將獨立 listing；屆時升級為 Tier 1 commercial player（見 ID_IndustrialAutomation）。
  • 商務機（Gulfstream / Bombardier）是否獨立週期？：商務機週期與商航 ~70% 同步但對 ultra-rich wealth effect 高度敏感（與 luxury 同步）。
💡 §1 Insight
1. 引擎 OEM 是被低估的 sub-sector：GE Aerospace + Safran + RTX + Rolls-Royce 共佔 sector profit pool 60%+，但市場 attention 過度集中 OEM 主機廠。
2. aftermarket parts (TDG/HEI) 是 high-multiple 隱形玩家：TDG 75%+ EBITDA margin / HEI 25-30% EBITDA — 受惠 fleet aging 結構化。
3. 邊界正在被 Honeywell 分拆 + Boeing 回購 SPR 重塑：2026 H2 後 sector composition 將變化。

結論

商業航太進入 Phase II 中（2025-2028 OEM-engine-aftermarket 三段同時 inflection）。核心非共識三條：（1）Boeing 翻轉是真實的，Q1 26 7 年首次 quarterly 超 Airbus = 訊號非偶然；（2）GE Aerospace LEAP aftermarket 是 sector 最強單一 alpha，類比 1995-2001 widebody supercycle GE 5 年 +500%；（3）RTX GTF 從 disaster 變 quietly compounding alpha，groundings -15% + GTF Advantage 認證 = inflection underpriced。
