★ 本 ID 核心非共識 Thesis 市場把商業航太當「Boeing 復甦故事」追蹤 — 多看 Boeing 自己的交付率與品質。本 ID 認為：① 真正的 alpha 不在 Boeing 而在引擎 aftermarket 黃金期 — LEAP（CFM）+ GTF（Pratt & Whitney）兩大新世代窄體機引擎都有「teething issues」（早期可靠性低於上一代），這個結構性「設計缺陷」反而推動 aftermarket 維修需求爆發 — 引擎 OEM aftermarket 毛利 70%+，每飛 5-7 年的維修週期擴大到 3-5 年，RTX / GE Aerospace 2026-2032 aftermarket 是史上最大利潤池。② Howmet（HWM）是「全行業 toll booth」最被低估的 pure play — 90%+ 的 LEAP + GTF 結構件 / 旋轉件來自 HWM；FY26 rev $9.1B（+10% YoY）+ EBITDA margin ~30%（業界最高）+ FCF $1.6B；類比於 ATE 鏈中 Advantest 的「最後一哩 toll booth」角色。③ Boeing 復甦不確定性高，但 Airbus + 引擎雙寡占受益是 hedge against Boeing 風險的關鍵 — 即使 Boeing 再次發生品質問題，Airbus backlog 9,031 架 + 引擎 aftermarket 仍能持續成長，這是配置上「捕捉產業順風但不押 Boeing 單一公司執行」的最佳結構。
💡 3 條核心 insight（預告）
1. 引擎 aftermarket 是商業航太最大被低估利潤池：LEAP + GTF 因「teething issues」維修週期短於上一代 5-7 年；RTX 2026 commercial aftermarket 占 segment 成長 70%；GE Aerospace 服務段 mid-teens YoY 成長；高毛利（70%+）+ 結構性需求 = 5-10 年黃金期。
2. HWM 是「行業 toll booth」最 pure play：90%+ 結構件 / 旋轉件來自 HWM（沒有第二替代）；EBITDA margin ~30% 業界最高；2026 FCF $1.6B；類比 Advantest 在 AI ATE 鏈、ASML 在 EUV 設備的角色。
3. Airbus + 引擎雙寡占是 Boeing 風險 hedge：Airbus backlog 9,031 架 + Pratt 仍持續 GTF 供應緊張時 Airbus 增加配給 LEAP — 即使 Boeing 再次危機，Airbus + 引擎廠仍受惠；商業航太投資不應 single-bet Boeing。

§1 產業定義與邊界

商業航太是個「OEM 雙寡占 + 引擎雙寡占 + 結構件少數幾家」的高度集中產業。本 ID 把焦點放在 commercial（商業客機）— 排除國防 / 太空（屬獨立 ID）。涵蓋四個子市場：① 飛機 OEM（Boeing + Airbus 雙寡占）；② 引擎廠（LEAP CFM = GE+Safran 50/50 + Pratt & Whitney GTF + Rolls-Royce 寬體）；③ 結構件 / 旋轉件（HWM 主導）；④ Aftermarket / MRO（OEM 自身 + 第三方）。

💡 Insight §1
1. 商業航太是個「四層雙寡占」結構：飛機 OEM（BA + Airbus）+ 窄體引擎（CFM + GTF）+ 結構件（HWM + ATI）+ aftermarket（OEM 自有 + TDG / HEI）— 每層都極度集中，這是 ROIC 持續高的根本原因。
2. 引擎 aftermarket 是利潤集中地：飛機 OEM 毛利 5-15%（OE 賠錢賺 aftermarket），引擎廠 OE 也是 15-25%，但 aftermarket 70%+ 毛利是業界共識。
3. HWM 是「結構件 toll booth」最 pure play：類比 ATE 鏈中 Advantest 的角色 — 90%+ 結構件市佔 + EBITDA 30% margin + 跨 GE/RTX 雙引擎客戶。

§2 技術演進 + 歷史類比

商業航太的「技術演進」其實是產品 + 引擎世代交替的故事。1990-2010 是上一代窄體（737NG / A320ceo）+ CFM56 / V2500 引擎黃金期；2015-2020 進入新世代（737 MAX / A320neo）+ LEAP / GTF 引擎；2020-2025 因 Boeing 危機 + COVID + 引擎可靠性問題而推延 ramp；2026-2032 才進入「真正的 ramp 黃金期 + aftermarket 爆發」。

💡 Insight §2
1. CFM56 1995-2005 aftermarket 黃金期是當前最佳類比：本次 LEAP/GTF 因 teething issues 更嚴重，aftermarket 利潤池可能比上一代大 1.5-2X。
2. Boeing 危機 2018-2024 信用修復是「需要時間但已啟動」過程：FAA 2025-09 自我認證放鬆是關鍵訊號；但市場 PE 仍偏低反映 lingering 風險。
3. 反類比警示：航空業歷史 cyclical（2001 / 2008 / 2020 三次大衰退）但這次 backlog 9,000+ 架是 strong buffer，下一次衰退可能延後或減弱衝擊。

3. Airbus 是 Boeing 的「hedge play」：即使 Boeing 復甦不及預期，Airbus backlog 9,031 架 + 強 FCF 仍能承擔產業 70% rev growth。

§5 Value Chain

議價權結構：飛機 OEM 對航空公司強勢（10+ 年 backlog 幾乎沒得議價）；對引擎廠是 50/50（雙方互相依賴）；對結構件廠（HWM）反而較弱（HWM 90%+ 市佔接近獨家）。HWM 的議價權因為「多 OEM 客戶分散 + 90% 市佔」極強 — 這是 HWM EBITDA margin 30% 業界最高的根本原因。
