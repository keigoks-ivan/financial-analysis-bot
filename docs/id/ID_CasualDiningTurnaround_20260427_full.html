§1 產業白話定義 + 歷史脈絡

白話版：休閒正餐（casual dining）就是「有服務生、坐下來吃、平均一餐 $18–28、用餐 60–90 分鐘」的全服務連鎖餐廳——Olive Garden、Outback、Chili's、Texas Roadhouse 這類。過去 20 年它從美國家庭外食首選，變成被快休閒（FC）+ 自家煮兩面夾擊的衰退 segment。但「衰退」不等於「全死」，內部正在發生暴力分化。

白話開場：先說三個關鍵詞。K 形分化指的是同一個 segment 內，贏家往上、輸家往下，平均值掩蓋了極端分歧——休閒正餐 segment 整體 -0.9%，但 TXRH Q1'26 +7.1%、RL 接連關店，這就是 K 形。value architecture（價值架構）是 Brinker/Chili's 做的事：不是單純漲價或促銷，而是設計一個「入門價格錨點（$10.99 三件套）+ 一個社群病毒英雄品項（Triple Dipper）+ 砍掉 25% 菜單」的整套結構，用 traffic 而非 check 驅動成長。PE-owned implicit short指的是：被私募股權買下、做了 sale-leaseback（賣斷店面再租回）+ dividend recap（抽走現金）的 chain，幾乎注定走向 Chapter 11——這不是個案，是系統性模式。本報告問的是：當休閒正餐 segment 整段萎縮，這塊縮小的餅，被重分配給了誰。

邊界界定：本報告聚焦「休閒正餐如何 K 形分化 + 翻身樣板換代」。納入全服務 casual（TXRH/DRI/EAT/BLMN/CBRL/CAKE）、作為對照與樣板移植來源的 best-in-class FC（CMG/CAVA/SHAK/WING/BROS/FWRG/SG/KRUS）、已/將 Ch.11 的 PE-owned 群組（RL/TGIF/Hooters/Buca/Boston Market/Rubio's）、以及非顯而易見二次受益（O 餐飲房東、TOST 餐飲 SaaS）。排除 QSR 漢堡/雞/咖啡（MCD/SBUX/YUM，商業模型不同、僅在對照時提及）、Fine Dining（規模小 ~$35B、獨立 luxury hospitality 主題）、以及把 restaurant tech 當主題（TOST 僅作二次受益、不深入其 SaaS metrics）。GLP-1 對餐飲衝擊詳見 GLP1RestaurantImpact 兄弟 ID，本報告在 §9 風險提及。

五個 boundary call（灰色地帶）是這份 ID 的判斷核心。其一，TXRH 屬 casual 還是 prevention？表面是 casual（有 server、用餐 60-90 分），但 unit economics（16-17% RLM / $8.4M AUV）接近 best-in-class FC，本報告認為應與 CMG/SHAK 比 multiple、不與 Outback 比（這是 §7 非共識核心）。其二，WING 屬 FC 還是 CDR 翻身樣板？WING asset-light 加盟 + 平均 check $20+ + no servers = 技術上是 FC，但本報告把它列為「數位 + chicken sandwich + Smart Kitchen」翻身教科書，與 CMG/DPZ 並列移植來源。其三，DPZ 屬 QSR 還是翻身樣板？傳統分類為 QSR pizza，但 DPZ 2009「pizza turnaround」是公開市場餐飲史上最完整的翻身案例（HBR/HBS case），不收則 ID 缺角。其四，CBRL 屬 casual 還是 family dining？分類 ambiguous（早午晚 + 零售 country store），本報告列為 casual 但標註「家庭式 mid-tier」。其五，BLMN 是「待翻身」還是「待 Ch.11」？仍 public + Outback ~30% 自有地產 + 2.3x leverage = 還不到 RL/TGIF 那種絕境，但 Spanos+Starboard 任期內若 traffic 不回 → refranchise 或 brand 拆售，本報告列為 🟡 mid（不確定、偏 implicit short）。

歷史分三段。第一段：1980–2007 黃金擴張——Applebee's/TGIF/Outback/Olive Garden 全國連鎖化，suburban strip mall 大爆發，休閒正餐是「全家出去吃」首選，占美國餐飲營收份額 peak ~30%。第二段：2000–2014 PE LBO 收購潮——Bain/Carlyle/Sun Capital/Apollo/Roark/Golden Gate/TriArtisan 大舉吃下 casual chains，推 sale-leaseback + dividend recap，把單店利潤被 SLB 租金 + debt service 蛀空，開始 deferred maintenance。第三段：2015–2026 FC 崛起 + 暴力分化——CMG 食安後重生、CAVA/SG/SHAK 上市、WING 加速；2020 COVID 砍掉 dine-in、2021-2022 勞動/食材通膨爆炸；2024 起 RL/TGIF/Hooters/Buca/Boston Market/Rubio's 接連 Ch.11，同期 EAT $10.99 翻身、TXRH 持續強勢——K 形定型。

歷史類比一：1990s-2010s 美國百貨死亡——倖存者擴大市佔。JCPenney/Sears/KMart 走的路徑是大量 PE LBO + REIT SLB + suburban mall 人流真實流失 = 全部 Ch.11；少數倖存者（Walmart/Target/Costco）反而擴大市佔。量化錨：百貨占美國零售營收份額從 1990 年代 peak ~10% 跌到 2020 年代 ~2-3%，但 Walmart/Costco 同期市佔翻倍。對本次含義：suburban 休閒正餐走同樣路徑——RL/TGIF/Hooters/各 PE-owned chain 死亡，TXRH/DRI/EAT 擴大市佔（贏家 group from ~38% → ~50% within CDR by 2030）。差異：餐飲頻次比百貨高約 20 倍，崩潰速度更快，但現場用餐體驗無法被線上替代，所以倖存者不會像百貨那樣連 Walmart 都被 Amazon 蠶食——餐飲是「死亡 chain 的店面 50%+ 不會被任何餐飲業者接手，改成住宅/醫療/倉儲」。歷史類比二：Domino's 2009「honest admission」翻身——翻身可量化。2009 之前 DPZ 是 pizza 三強最弱（comp 連年負）；CEO 用 4 分鐘紀錄片直播焦點團體罵 pizza「像紙箱」+ 全面 reformulation，2010 Q1 US comp 跳 +14.3%（QSR 史上最大跳升之一），接著做出 28+ 季連續正 SSS streak；同期 Pizza Hut（YUM）沒做翻身、SSS 連年負至今。教訓：翻身核心是「主動重做產品 + 公開承認失敗」，這正是 EAT 菜單瘦身 25% 的源頭，也是 BLMN/CBRL 至今未做的動作。

§2 技術成熟度 + S 曲線

為什麼翻身樣板正在換代？因為過去十年的 kingmaker（數位 + throughput）已跨過「成熟普及、人人都做」的門檻——CMG/WING FY25 首次負成長就是訊號：當每家 chain 都做了數位/loyalty，它就不再是 alpha 來源，而是 table stake。新的差異化往「value architecture」與「balance-sheet reset」上移。

翻身技術棧有六個 pattern，成熟度各異。已商品化（table stakes）：①數位 + loyalty database（DPZ Pizza Tracker / CMG Rewards 40M / WING MyWingstop 60M），2010-2018 是真 alpha，2025 起每家都做 → CMG FY25 -1.7%、WING -8.7% 是「數位 alpha 耗盡」訊號；②throughput redesign（CMG second make-line + Chipotlane，AUV +10-15%），需 12-18 個月 + $50-200K/店 capex，2,500+ 店時是系統工程。正成為新 kingmaker：③value architecture（EAT $10.99 + Triple Dipper + 菜單瘦身 25%），通膨後消費者 sticker shock 嚴重、digital 已商品化、capex 極低（~$50M behind-the-line vs CMG $200K/店 × 4,000 店 = $800M），12 個月做出 +27% comp；④balance-sheet reset（RL post-Ch.11 拒絕 ~$650M leases、debt 從 $1B → $350M），對 SLB 鎖死的 PE-owned chain 是唯一現實路徑。永遠是 prevention 而非 turnaround：⑤manager equity（TXRH 1993- 的 $25K 押金 + 10% 單店利潤分成），需 30 年文化建構、無法移植到危機中的 chain。第六個 pattern⑥honest admission（公開承認失敗）是所有翻身的前提——DPZ 2009 公開承認 pizza 難吃才有後續 reformulation，但這個 pattern 沒有專利、卡點在 CEO 自我與 Board 政治：多數 chain 死撐到 Ch.11 也不肯公開承認食物有問題，這是為何 BLMN/CBRL 的翻身遲遲不啟動的根本原因。

每個 pattern 的結構性瓶頸不同，這決定了哪些可移植、哪些不可。數位（pattern ④原 #4）已商品化、不再 differentiation，但缺它必死（是 table stake 而非 alpha 來源）；throughput 需重大 capex + 加盟商同意（asset-light chains 卡在加盟商不願出資）；value architecture 需要管理層願意短期接受 mix-down + GM 壓縮——這正是 PE-owned chain 無法做到的（不能接受 EBITDA 下滑 12 個月去換 traffic 回升）；balance-sheet reset 需經過 Ch.11 痛苦（brand equity 短期重大損失 + 部分 store 關閉）；manager equity 需 25-30 年文化建構（無法用 capex 或 hire 解決，25 年來無人成功模仿 TXRH）。從這個瓶頸地圖可以推導出一個關鍵判斷：2024-2028 真正的翻身 alpha 集中在 pattern ③（value architecture）與 ④（balance-sheet reset）——前者 capex 低見效快、後者是 PE-owned chain 唯一現實路徑，而過去十年的 kingmaker（數位 + throughput）已淪為人人都做的 table stake。誰能在這兩個 pattern 上執行，就是下一個 alpha 製造者。

§3 供給側：現在與未來 + 利潤池

餐飲價值鏈的利潤被上下游兩面 squeeze——餐廳業者（中游）被上游食材通膨 + 下游房東/平台/SaaS 抽走利潤，只有 unit economics 結構最強者能在 squeeze 中保 RLM ≥ 15%。這是 K 形分化暴力的根本原因。

供給端可拆三段。上游（食材 / 設備）：Sysco/US Foods/PFG 分銷（毛利 17-20%、EBITDA 4-5%）+ Middleby/Welbilt 設備（毛利 30-35%），議價權中；弔詭的是 chain 集中度上升反而壓縮分銷商毛利（採購方議價權上升），SYY EBITDA margin 已從 5.5% 壓至 ~4.0%。中游（餐廳營運）：勞動 30-36.5% + 食材 28-32% + 租金 6-10% + 行銷 3-5%，RLM 12-28%（K 形），議價權低（被房東 + 勞動 + 食材三面夾擊）。下游（房東 / 平台 / SaaS）：Realty Income（O）等 REIT lease 鎖 10-20 年 + 1-2% escalator、~95% NOI margin、議價權極高；DoorDash/Uber Eats take rate 20-30%、EBITDA 近年轉正；Toast/Olo POS switching cost 極高（換 POS = 全面停擺）、ARR +30-40%/年。利潤往「下游 + 上游」兩端集中，中游餐廳是被 squeeze 的一段。

估值傳導機制：產業變數 → 公司估值的傳導鏈——① FAFH CPI 收斂 → 全 CDR comp +1-2pp（+1pp comp ≈ +0.5x P/E expansion）；② 勞動 +1pp（如 tipped wage 廢除）→ CDR RLM -100bp（≈ -2x P/E）；③ 牛肉 +10% → TXRH RLM -200bp（≈ -1.5x P/E）；④ PE-owned chain Ch.11 釋放 sales → 贏家 comp +0.5-1pp。當前 multiple 對照：TXRH ~24x（unit economics 接近 best FC，合理）、EAT ~16x（comp 強度 ≥ CMG 但 multiple 折一半，低估）、CMG ~30x（FY25 -1.7% 後仍 pricey）、WING ~50x（FY25 -3.3% + Q1'26 -8.7% 後 multiple 應壓）、CAVA ~80x（IPO 蜜月退坡訊號）、BLMN ~9x（cheap for a reason）、CAKE ~14x（合理、hidden Brinker candidate）。

估值現在 price 了什麼：TXRH 的 prevention premium 已部分 price（~24x）；EAT 的 value architecture upside 未充分 price（~16x，comp 強度 ≥ CMG 卻 multiple 折一半）；CMG/WING/CAVA 的高估值未充分反映數位樣板退潮；BLMN 的 Ch.11 尾部風險未充分 price（市場仍當「cheap 翻身 candidate」）。會 re-rate 的是 EAT（若 FY26 守正）；會 de-rate 的是高估值 chains（WING/CAVA）與 PE-owned（BLMN）。對 PM：盯 EAT 退坡軌跡與 TXRH RLM，不是盯「餐飲景氣」。

§7 Non-Consensus + Priced-in 檢驗 + Kill Scenario

⚠️ 本章含 judgment：每條分歧附 [F:] cornerstone + [X:] 三情境 + priced-in 檢驗。
市場對這主題的三個共識：①「休閒正餐整段死亡 → 全砍」；②「CMG/DPZ 數位翻身樣板是永久 alpha」；③「PE-owned cheap stock 是翻身機會」。三個我們都部分不同意——分歧能否賺錢取決於市場 price 了多少。
🟡分歧 1：「休閒正餐整段死亡」是錯的，是「暴力分化」信心：高
  • 共識說 X：CDR -0.9% YoY + 10 件 PE-owned Ch.11 → 整段死亡、全砍（Goldman/MS sector reports 2024-2025 多用 underweight CDR）。🟢 [F: T2 — Technomic/NRA segment -0.9%]。
  • 我們說 Y：CDR 是 K 形暴力分化——TXRH Q1'26 +7.1% / DRI Q3FY26 +4.2% / EAT Q3FY26 +3.3%（贏家 group 38%→50% within CDR by 2030），同期 BLMN US +0.9% / RL 續關店。投資 thesis 是 buy 4-5 winners 非 buy/short 整段。🔵 [X: base 很可能 — K 形深化；bear 可能 — 第二波 Ch.11 拖累連贏家]。
  • Priced-in：市場把「CDR」泛泛 underweight、沒充分 price「贏家擴大市佔」——分歧 1 的 EAT mispricing（~16x vs CMG ~30x）最未 priced。
→ 推論邏輯：把「CDR underweight」拆成贏家（TXRH/DRI/EAT 擴市佔）vs 輸家（PE-owned 死亡）。⚠ 證偽：FY26 H1 EAT comp 跌至個位數 + TXRH Q3 出現負成長 + DRI EBITDA 跌至 13% 以下（連 3 個贏家失靈）→ 整段死亡共識 confirmed。
🟡分歧 2：數位翻身樣板已退潮，新樣板是 EAT value architecture信心：高

  • Priced-in：市場仍給 CMG/WING 高 multiple（~30x/~50x）、沒 price「數位 alpha 耗盡」——分歧 2 的 CMG/WING de-rate 風險未充分 priced。
→ 推論邏輯：數位是 table stakes、value architecture 是新 alpha。⚠ 證偽：CMG FY26 H1 comp 反彈至 +3%+ 且 EAT FY26 comp 跌至 +5% 以下（樣板對調）→ 本 thesis 部分作廢。
🟡分歧 3：PE-owned 是 implicit short signal，不是「cheap 翻身機會」信心：高
  • 共識說 X：PE-owned chain 出問題（如 BLMN 跌至 $9-10）= cheap stock + activist 觸發翻身 → buy at trough。🟢 [F: T3 — Starboard ~10% 入股 BLMN]。
  • 我們說 Y：PE 結構（LBO debt + SLB rent overhang + 抽 dividend 不投 capex）是系統性 alpha 摧毀者——2014 RL→2024 RL Ch.11；2017 TGIF→2024 TGIF Ch.11；2023 Hooters→2025 Hooters Ch.11，模式 100% 重複。BLMN 是下一邊緣（2.3x lev + Q1'26 +0.9%）。投資人應把「PE-owned」當 short，買點在 post-Ch.11（如 Adamolekun-led RL）。🔵 [X: base 很可能 — BLMN 走向重整/拆售；bull 不太可能 — Spanos+Starboard 翻身成功]。

  • 反方 1（EAT 翻身是 lap math）：EAT 2023 base 極低（Chili's traffic 5 年累計 -25%）→ +27%/+31% 部分是 base effect；Triple Dipper TikTok viral 不可重複；$10.99 是 promotional pricing 長期不可持續。當前證據：Q3FY26 已退坡至 +3.3%（從 +31%）。證偽窗口：FY26 H1 comp 是否守正 vs 跌至個位數 ≤ +5%。

市場最低估的：最低估的機會是分歧 1 的「EAT mispricing（~16x vs CMG ~30x）」與分歧 2 的「數位高估值 chains de-rate（CMG/WING/CAVA）」；最低估的風險是分歧 3 的「BLMN 走向 RL 路徑」與 tipped wage 廢除（CDR 結構性 -200 to -300bp RLM 的潛在 black swan）。三者都該獨立監測。
