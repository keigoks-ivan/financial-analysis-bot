「AI 推論越來越便宜」常被誤讀成兩種極端：要嘛是 LLM 廠的壞消息（賣 token 賺不到錢），要嘛是用 AI 的軟體公司的純利多（成本下降）。兩種都只說對一半。真相是：每百萬 token 的單價確實在崩——一年內由 ~$10 跌到 ~$2.50、三年累計跌幅超過 99%（早期 GPT-4 的 $25–60 到 GPT-5.2 的 $1.75/$14）——但這不是需求見頂，而是 Jevons 悖論：價格越便宜，用量爆炸得更快。

證據很直接：Silicon Data 的代幣支出指數自 2025 年底以來已翻倍，即使單一 token 價格較 2023 跌了約 90%；推論（而非訓練）現已吃掉 AI 資料中心算力的約 2/3 到 90%；2026 年 hyperscaler AI capex 達 ~$660–765B、AWS 積壓訂單 $244B（+40% YoY）、微軟 Azure 還有 ~$80B 訂單因 GPU/電力不足無法交付。單價在跌，總支出卻在飆。

所以本報告不押「token 會不會降價」（會），而是拆解單價趨零下，賣 token 的錢被誰賺走。結論：價值流向三處——自有矽的垂直整合者（GOOGL TPU / AMZN Inferentia / META MTIA，推論毛利不漏）、收推論租金的算力層（NVDA / AVGO 賣鏟子）、與把 token 重新包成 outcome 定價的應用層；而 AI 正把軟體毛利從 80–90% 壓到 50–60%，純賣 token 或 per-seat 的中間商被 Jevons 救了量、卻被壓了利。

PART I · DECISION SUMMARY

決策摘要層

三句話看完推論經濟的現狀、走向與行動，後接 PM 級監測點。這一層是報告的全部重點——下面各節只是把這一層的每個判斷補上證據。

現在 / NOW

AI 推論進入單價崩跌 + 用量爆發並存期——主流 model 每百萬 token 一年內由 ~$10 跌到 ~$2.50（3 年累計崩 ~99%），但 Silicon Data 代幣支出指數自 2025 末已翻倍（Jevons：價跌 90% 用量更大）；推論已占 AI 資料中心算力約 2/3–90%。2026 AI capex ~$660–765B。

未來 / NEXT

3–5 年 token 單價續跌但用量續爆（量補價），價值往「自有矽垂直整合（GOOGL TPU/AMZN Inferentia/META MTIA，毛利不漏）+ 收推論租金的算力（NVDA/AVGO）+ outcome 定價的應用層」集中；賣純 token 的 API 中間商被 Jevons 救量、毛利被壓。

行動 / ACTION

押自有矽垂直整合者 GOOGL/AMZN/META（推論毛利不漏）+ 收推論租金的 NVDA/AVGO + 推論基礎設施 NET；理解 token 降價不是 bear（Jevons 量補價），真風險是 SaaS 毛利壓縮（80–90%→50–60%）與外購矽者漏出；端側 AAPL/QCOM 省 token 是另一條。

The Key Call · 本報告最重要的一個判斷

市場把「token 降價」當成 LLM 廠的壞消息或軟體股的純利多，真正的非共識是價值捕獲——單價趨零是必然（3 年崩 99%），但 Jevons 讓用量爆發（代幣支出指數翻倍），所以「賣純 token」是商品化生意、毛利被壓；賺到錢的是三處：① 握自有矽的垂直整合者（GOOGL/AMZN/META——推論成本最低、毛利不漏出給外部）；② 收推論租金的算力層（NVDA/AVGO——賣鏟子、量爆即受益）；③ 把 token 重新包成 outcome/per-action 定價的應用層（避開 SaaS 毛利從 80–90% 被壓到 50–60%）。誰只會「賣 token」或「per-seat 吃到飽」，誰就被 Jevons 救了量、卻被壓了利——這才是推論經濟的核心賭注。

三道裂縫具體是：①SaaS 毛利壓縮——每次 model 呼叫都吃真實算力，AI 把軟體毛利從 80–90% 壓到 50–60%，per-seat 吃到飽的定價撐不住（per-seat 採用一年內由 21% 降到 15%）；②自有矽 vs 外購——GOOGL/AMZN/META 用自有 TPU/Inferentia/MTIA 把推論成本與毛利留在內部，外購 GPU 者把一塊漏給 NVDA；③定價模式遷移——hybrid（底價 + 用量）一年內由 27% 升到 41%、outcome-based 興起，誰先把 token 包成「按結果收費」誰守得住毛利。盯這三條，比盯「token 又降價了」更能看清錢的流向。

PART II · HISTORICAL ANALOGS

頻寬崩價與雲端降價，框住多空

歷史不必重演，但兩個對照各給一個教訓：頻寬崩價 + 流量爆發給出「量補價、贏家是握流量的平台」的 bull 底，雲端持續降價給出「規模 + 垂直整合者贏」。

類比一：頻寬崩價 → 串流巨頭誕生（量補價）。2000 年代網路頻寬單位成本崩跌約 99%，但這沒讓賣頻寬的賺更多——流量爆增千倍，贏家是握住流量與內容的平台（YouTube、Netflix），不是賣裸頻寬的電信。量化錨：頻寬成本一路崩、流量卻增千倍，這正是「單價趨零、總支出爆發」的歷史先例。對本次的 bull 含義：token 降價不會殺死 AI 經濟，反而靠 Jevons 把總支出做大（代幣支出指數已翻倍）；賺到錢的是握「算力 + 自有矽 + 應用」的人，不是賣裸 token 的。

怎麼讀：兩個對照都指向「單價趨零不是末日、量補價是常態，贏家是握流量/整合的人」。bull 的底很實（Jevons + capex $765B），bear 不是「token 沒人用」（用爆了），而是「賣純 token 的賺不到、SaaS 毛利被壓、外購矽者把利潤漏給 NVDA」。

類比二：雲端運算降價 → 垂直整合者贏。AWS 上市以來降價超過 100 次，但營收與利潤暴增——因為降價刺激用量（Jevons）+ 規模經濟攤平成本 + 垂直整合（自研晶片 Graviton/Inferentia）把毛利留在內部。量化差異：對本次的含義很直接，GOOGL（TPU）、AMZN（Inferentia/Trainium）、META（MTIA）正複製這條路——自有矽讓它們在 token 降價戰中既能跟價、又把推論毛利留住；外購 GPU 的純 LLM 廠則把一塊利潤付給 NVDA。判別訊號：看自有矽佔內部推論的比例、以及 AI 業務的毛利是否守得住。

PART III · SUPPLY SIDE

供給側：推論價值鏈，利潤往矽與整合集中

反直覺但可操作的結論：token 價值鏈的利潤往「握自有矽 + 收推論租金 + outcome 應用」集中——賣裸 token 的中間商被商品化，端側則把成本搬離雲端。

供給端可拆成五層：①矽層（NVDA GPU + 收 Groq 後的 LPU、AVGO 客製 ASIC、AMD 挑戰者——收推論租金）；②自有矽垂直整合（GOOGL TPU、AMZN Inferentia/Trainium、META MTIA——成本最低、毛利不漏）；③hyperscaler 分發（MSFT/AMZN/GOOGL——賣推論服務 + 經銷 LLM）；④推論基礎設施（NET routing/caching/edge、CRWV neocloud——降低 token 成本的中間層）；⑤端側（AAPL/QCOM——把推論搬到裝置、省雲端 token 費）。利潤往「自有矽 + 收租算力 + outcome 應用」集中；賣裸 token 的純 LLM 中間商被多模型壓價。

Inference · 供給
token 商品化，利潤往自有矽、收租算力與 outcome 應用集中
前提：token 單價被多模型壓價（商品化）+ 推論用量爆發（Jevons）+ 自有矽把成本/毛利留內部 + 算力受 CoWoS/電力限（賣方有定價權）→ 利潤往「矽/自有矽端 + outcome 應用端」集中，純 token 中間商被擠

當「一個 token」變成可替換、年年崩價的商品，賣裸 token 賺不到溢價；賺得到的是：握自有矽的垂直整合者（GOOGL/AMZN/META——跟得了價又留得住毛利）、收推論租金的算力層（NVDA/AVGO——量爆即受益、且 CoWoS/電力受限讓它有定價權）、與把 token 包成 outcome 的應用層。Jevons 把餅做大（代幣支出翻倍），但餅怎麼切由「誰握矽、誰握客戶結果」決定。

可證偽條件：若推論算力供給（CoWoS/電力）大幅鬆綁、算力定價權消失，或開源高效 model + 通用 ASIC 讓自有矽的成本優勢被抹平，則「利潤往矽/整合集中」論的某一面失效。

PART IV · DEMAND SIDE

需求側：Jevons 量補價與算力供給瓶頸

需求最反直覺的特徵是「越便宜、用越多」——token 單價崩 90%，總支出反而翻倍（Jevons）；最硬的瓶頸不在需求，而在供給（CoWoS、電力）跟不上。

第一引擎：Jevons 量補價（現需）。推論單價崩跌不會減少總支出，反而放大它——Silicon Data 代幣支出指數自 2025 末翻倍，即使單一 token 價格較 2023 跌約 90%；業界共識是「API 成本降 98%，企業不會少花 98%，而是多蓋 50 倍應用」。agentic AI（見子題「Agentic AI 平台」）更是推論放大器：一個 agent 任務要呼叫 model 數十次。推論已占 AI 資料中心算力約 2/3（Deloitte）到 80–90%（部分研究）。第二引擎：供給瓶頸（未需轉現需）。需求不是問題，供給才是：2026 AI capex ~$660–765B、→2031 ~$1.6T；AWS 積壓訂單 $244B（+40% YoY）、Azure ~$80B 訂單因 GPU/電力不足無法交付；TSMC CoWoS 由 2024 ~9,000 片/月擴到 2026 年底估 ~130,000 片/月仍「sold out through 2026」。電力與先進封裝是真瓶頸。

怎麼讀：單價（崩 99%）與用量（支出翻倍）方向相反、卻同時成立——這正是 Jevons；供給端 capex/backlog/CoWoS 全部 sold out，說明瓶頸在供給不在需求。採信「代幣支出指數 + backlog」作需求真實性的最硬證據，而非「token 又降價」的表象。

Inference · 需求
需求被 Jevons 放大、供給受限——成長品質看「誰的毛利不被降價戰侵蝕」
前提：token 降價刺激用量（Jevons）+ 推論占比升至 2/3–90% + 供給（CoWoS/電力）受限 → 總支出爆發、算力有定價權，但賣裸 token 者毛利被壓

需求的「量」由 Jevons 保證（單價越低、用越多、總支出越大），這幾乎沒爭議；變數在「誰的毛利不被降價戰侵蝕」——握自有矽者（GOOGL/AMZN/META）成本最低、跟得了價；收租算力（NVDA/AVGO）因供給受限有定價權；賣裸 token 的中間商被夾在「降價 vs 成本」之間。

可證偽條件：① 代幣支出指數成長停滯（量補不過價、Jevons 失效）；② 供給瓶頸快速鬆綁使算力定價權消失、推論變純價格戰。

多空為何都對

多頭測的是總需求（Jevons 量補價、capex $765B、推論占比升），這幾乎沒爭議；空頭測的是毛利與分潤（SaaS 毛利壓縮、外購矽漏出、賣裸 token 賺不到）。兩者可同時為真：推論總支出爆發（多頭對），但這份支出的利潤被矽/自有矽/應用層切走、純中間商吃虧（空頭對）。本報告把重心放在「單價趨零下誰捕獲價值」，而非「推論會不會成長」。

PART V · VERDICT & SCENARIOS

裁決與三情境（24–36M）

「Jevons 量補價」判 INTACT（支出指數翻倍），「價值往矽/自有矽/應用集中」判 INTACT，但「賣裸 token 與 per-seat SaaS 的毛利」判 AT_RISK（80–90%→50–60% 壓縮）——故結構看多推論總量、但要選「握矽或 outcome 定價」者。如硬要對應傳統框架：推論算力對「便宜且充足」的需求遠大於合格供給（CoWoS/電力受限），等價於結構性短缺型，稀缺來自先進封裝與電力產能而非單價——token 單價跌與算力短缺並存。

Base Case主觀機率 ~50%

PART VI · TAXONOMY

個股特徵與 conviction tier

本框架只輸出「特徵」與「分類」，買哪一檔由個股 DD 系統與 Pure MA 訊號決定。以下 tier 為產業推論的特徵歸類，提及公司僅為「該特徵的代表案例」，不附帶買賣建議。

CORE 🔴

NVDA / AVGO

收推論租金的算力層——NVDA（GPU + Groq LPU）、AVGO（hyperscaler 客製 ASIC）。token 用量爆發的最大賣鏟子，且 CoWoS/電力受限給它定價權。特徵是「量補價的直接受益者」，推論超級循環的核心曝險。conviction：高（量爆 + 定價權）· 估值看週期
CORE 🔴

GOOGL

垂直整合典範——TPU + Gemini + GCP + caching，推論成本最低、毛利不漏出給外部。特徵是「自有矽護城河」的最純代表，token 降價戰中既跟得了價又留得住毛利。conviction：高（自有矽 + 整合）
CORE 🔴

MSFT / AMZN / META（自有矽 + 分發）

MSFT：hyperscaler + LLM 經銷（Azure backlog ~$80B 受 GPU/電力限）。AMZN：Inferentia/Trainium 自有 ASIC + AWS backlog $244B +40%。META：MTIA + 內部推論超大用戶（省成本、毛利不漏）。特徵是「分發規模 + 自有矽」的結構占優者。conviction：高（規模 + 整合）
EDGE 🟡

NET / AMD / CRWV / AAPL / QCOM

NET = 推論 routing/caching/edge 收費站；AMD = GPU 挑戰者（多模型壓價下第二供給）；CRWV = neocloud 承接外溢；AAPL/QCOM = 端側推論（把 token 成本搬離雲端）。特徵是「降 token 成本 / 第二供給 / 端側替代」，第二層曝險。conviction：中 · 各看放量

估值面總結：本主題的核心張力在「Jevons 量補價是真（支出指數翻倍）vs 毛利在哪一層守住未定（SaaS 80–90%→50–60%、自有 vs 外購矽）」。收租算力（NVDA/AVGO）與自有矽整合者（GOOGL/AMZN/META）是最高純度；NET 是推論基礎設施；端側（AAPL/QCOM）是替代路徑；受壓的是賣裸 token 的中間商與 per-seat SaaS。現處 Phase II（量補價），轉空條件 = 代幣支出指數停滯（Jevons 失效）或 供給瓶頸鬆綁使算力定價權消失；de-rating window 在「市場發現量補不過價 + SaaS 毛利崩」那一刻。OpenAI/Anthropic 為私有、不可直接投資。買賣由個股 DD + Pure MA 系統決定。

The One Line

token 單價趨零不是末日、是 Jevons 量補價——真正的賭注是「賣 token 的錢被誰賺」，答案是握自有矽、收推論租金、與 outcome 定價的人。

這份報告刻意不做的三件事

  1. 不把「token 降價」當 bear——明確點出 Jevons 量補價（支出指數翻倍），降價放大而非縮小總支出。
  2. 不把「成本下降」當軟體股純利多——同時標出 AI 把 SaaS 毛利從 80–90% 壓到 50–60%、per-seat 撐不住。
  3. 不告訴你哪一檔該買——CORE/EDGE 是「特徵的代表案例」，買賣由你的個股 DD 與 Pure MA 系統決定。
