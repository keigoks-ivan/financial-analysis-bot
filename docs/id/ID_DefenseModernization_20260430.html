★ 4 個歷史類比（含正反例對照，QC-I28）

類比 1：Reagan 1980s 國防大擴張（1980-1989）— 正類比。Carter 末期國防 GDP 占比 5.0% / Reagan 1986 達 6.2% (peak) / 推動 Star Wars (SDI) + 600-ship Navy + B-1 / B-2 bomber + Pershing II missile。對應 capex S 曲線：4 年從低點到 peak（1981-1985）/ 5 年 plateau（1985-1990）/ 後 7 年 wind-down（1990-1997）。重點：Reagan 模式是「總統 + 國會雙重 push + 戰略對手明確」三軸驅動；當前 Trump 第二任 + Republican Congress + 4 戰線同步 = 結構與 Reagan 模式相似但 multi-region 更廣。本 ID 認為當前 phase 對應 Reagan 1983-1985 中期，仍有 3-5 年結構超級週期。

類比 2：9/11 反恐 + Iraq War 國防增量（2001-2010）— 正類比。9/11 後 美國國防 capex 從 $309B (2001) → $691B (2010) = +124% 9 年累積；推動 MRAP $40B+ / Predator drone 第一代 / 衛星偵察大幅擴產。對應「drone 第一代爆發」+「intelligence / surveillance / reconnaissance (ISR)」結構性新類別；當前的「drone 第二代 + autonomous + space」三引擎是 9/11 的 second wave。重點：9/11 之後 US 國防 surplus 持續 8 年（2001-2009），即使 Iraq surge 結束後仍 sustained / 證明 capex super-cycle 的 sustainability。

類比 3：1990s post-冷戰 peace dividend（1989-2001）— 反例類比（QC-I28 強制）。Berlin Wall 1989 後美國國防 capex 從 6% GDP → 3% (2001 9/11 前) / 大幅 underinvest 持續 10 年 / 歐洲 NATO 平均降至 1.5%。反例警示：當前的「結構超級週期」假設 4 戰線 multi-year sustained；若 Ukraine 2027 達成 ceasefire + Iran-Israel de-escalate + Taiwan crossing 不發生 → 可能進入「mini peace dividend」phase；NATO 5% 2035 目標若無實際 enforcement → 達標機率仍低。當前 vs 1989 差異：1989 是「敵人消失」/ 2026 是「敵人增加」+「supply chain reset」+「中俄合作明顯」/ 結構觸發因素強度顯著高於 1989。

類比 4：Iraq Surge 2007-2010 與後續 wind-down（2010-2014）— 反向類比。Iraq Surge 帶動國防短期 capex peak（2007-2010）但 2011 Iraq 撤軍 + 2014 Sequester / Budget Control Act 強制 Pentagon -10% 5 yr / 國防股經歷 30%+ drawdown。反例警示：本 ID 假設「Phase IV 結構超級週期」持續至 2031；若 OBBBA 2025-07 supplemental 用完 + 美國 budget deficit 失控觸發類似 2014 Sequester → 國防 capex 短期 -15-20%；但當前差異：4 戰線同步 + 中俄威脅 + drone / autonomous / space 三引擎都是 multi-year program，比 Iraq 單地理 surge 結構更穩固。

💡 §2 Insight（含 QC-I28 反例）
1. 2 正類比 (Reagan / 9/11) + 2 反例類比 (1990s / 2014 sequester) 框架收斂於：當前 Phase IV 結構超級週期觸發強度介於 Reagan 與 9/11 之間，但 sustainability 取決於 4 戰線 multi-year 維持。S 曲線 2030 plateau 機率高於 wind-down，但 2031-2035 wind-down 風險不可忽視。
2. 反例類比是本 ID Falsification 核心警示：1990s peace dividend 是最強 falsification 對照；若 2027-2028 出現 multi-front de-escalation + budget pressure 同步 → Phase IV 提前 wind-down，國防股 -20-30% drawdown 風險真實存在。

§3 三引擎拆解 — Drone × Autonomous × Space-Defense

本 ID 的核心母題 framework — 三個 engine 是當前 capex 增量的真 alpha 源。三引擎不是各自獨立 sub-sector，而是「整合作戰系統」三個 layer：drone（hardware）→ autonomous（C2 軟體）→ space（sensor / network），三者必須共同部署才能形成 next-generation 戰場架構。本章節 5 段敘述 + 1 張三引擎 capex 對比表。

引擎 1：Drone（無人機系統）。涵蓋 (a) loitering munitions（Switchblade $250M+ AVAV backlog / Hero / Spike）；(b) tactical drones（KTOS Valkyrie XQ-58A / MQ-9 ramp）；(c) attritable drones (Anduril ALTIUS / Roadrunner / Switchblade 600)。市場數據：全球 drone defense 市場 2024 $13B → 2030 $35-45B（CAGR 18-22%）；Ukraine 戰場驗證 drone-to-soldier ratio 從 1:50 (2022) → 1:5 (2026)，是 next-decade 軍事革命。Anduril Arsenal-1 mid-2026 Ohio 廠投產針對 attritable drones / 月產能 target 數萬量級。本 channel 的 alpha 已部分 priced（AVAV +60% YoY 2025）但 backlog visibility 5+ yr 仍高。
