三句話結論：本篇整合子 1-4 所有訊號，建構「經濟 × 軍事 × 政治」三線 × 「D-180 / D-30 / D-7」三時點的 9 格儀表板。每格列出 3+ 可量化、有資料源的 tripwire + false positive 控制方法。當前（2026-04）整體警戒級為 Level 2 （灰色地帶常態 + 單線訊號異常）；觸發 Level 3-4（多線同時異常）是倉位大幅調整 tripwire；觸發 Level 5（全部三線 D-30 同時異常）代表 6 個月內動武機率上修至 30%+。投資應用：Level 1-2 維持 Base case 組合；Level 3 加碼國防 / 稀土 / 減碼台灣本廠半導體；Level 4 增加戰爭險保險 / 能源；Level 5 進入 Tail case 現金 / 黃金 / 短期美債配置。

§1 核心判斷

★ 非共識 Thesis 1：單一訊號意義有限，「訊號組合」才是關鍵 市場常犯的錯誤是把單一訊號（如 PLA 演習、中國軍費增速、Trump 推文）視為 tripwire，但這類單一訊號 false positive 率極高（60-80%）。真正的 tripwire 是「跨線組合訊號」——例如「經濟線糧食儲備異常 + 軍事線 PLA 075 艦隊異常集結 + 政治線 Xi 公開視察前線」三者同時出現。本篇 9 格儀表板的設計核心是「多訊號合併 → 降低誤警率」。2022-2024 年曾有 3 次單線軍事訊號觸發（裴洛西訪台後演習、2023 聯合利劍、2024 聯合利劍 A/B），但全部未升級為實戰——這驗證了單線訊號的可容忍性，以及「多線組合」才是真正觸發。
★ 非共識 Thesis 2：政治線訊號比軍事線更早、更可靠 傳統情報分析偏重軍事線（PLA 部署、飛彈、艦艇動向），但這些是作戰準備的結果而非決策。真正的決策源於政治線——Xi 的公開行程、中央軍委會議、政治局密集互動、對內宣傳調性。歷史上蘇聯 1979 入侵阿富汗、伊拉克 1990 入侵科威特、俄羅斯 2022 入侵烏克蘭，軍事部署都在戰爭前 2-4 個月就位，但政治決策是前 6-9 個月。對 Xi 而言，政治線 D-180 的關鍵是「公開行程異常 + 中央軍委擴大會議 + 國務院 / 中央統戰部高層表態改變」。本篇特別強調政治線訊號的敏感度與可解釋性。
★ 非共識 Thesis 3：D-180 預警窗口是投資倉位調整的黃金時機 市場反應 D-30 或 D-7 訊號時，資產已大幅定價——這時調整成本高、流動性差。真正的 alpha 來自 D-180 訊號——「早半年看到」的訊號能讓投資者以常態價格建倉（國防 / 稀土加碼）或減碼（台灣本廠半導體 / 中國 ADR）。本篇三線 × D-180 的 9+ 個 tripwire 是基金經理人、家族辦公室、策略顧問的主要工具。例如 2022-05 俄羅斯軍費突然 +30% 增長 + 糧食儲備異常 + Putin 公開行程變化，這些訊號在烏克蘭戰爭爆發 6 個月前已足以預警；但大多數資產管理公司未能識別，2022-02 戰爭爆發時已難以調整。
💡 3 條最高價值 insight
1. 三線組合 level 系統是實用決策框架：Level 1（全部常態） / 2（1 線異常） / 3（2 線異常） / 4（3 線異常，但 D-180 範圍內） / 5（3 線 D-30 全異常）。這個 5 級系統對應不同倉位調整強度
2. 衛星圖像 + 開源情報（OSINT）是 D-180 的核心工具：Sentinel-2 / Maxar / Planet Labs 的商業衛星圖像 + 海事追蹤（MarineTraffic / Global Fishing Watch）+ 航班追蹤（FlightRadar24）+ 社交媒體分析。關鍵機構 / 智庫（CSIS China Power Project、US Naval Institute、Taiwan INDSR、RAND、CIMSS）提供的分析是免費或訂閱即可獲得
3. 「不訊號」本身是訊號：若 Xi 重要儀式缺席（北戴河會議、春節團拜、外交場合）或 PLA 突然停止例行演習（「過於安靜」），這些「應該有但沒出現」的訊號反而值得注意——通常意味著作戰前的「隱匿期」

§2 結構背景：三線 × 三時點 設計邏輯

§2.1 三線劃分邏輯

  • 經濟線：反映中國戰爭財政空間、戰備儲備、市場反應。變數包括軍費增速、糧食 / 能源儲備、黃金進口、人民幣匯率、特定產業動員跡象。相對結構性、數據公開性較強
  • 軍事線：反映 PLA 實戰準備程度。變數包括兵力集結、艦艇 / 飛機部署、彈藥配送、網攻 / 電戰演訓、海底電纜異常。需依賴衛星 / OSINT / 專業軍事分析機構
  • 政治線：反映 Xi 決策意圖 + 共產黨體制狀態。變數包括 Xi 公開行程、中央軍委會議、中央統戰部活動、官方媒體調性、外交措辭、對內宣傳重點。相對難以量化但高敏感度

§2.2 三時點設計

§3 核心儀表板：三線 × 三時點 9 格

經濟線 D-180

  • PLA 軍費突破兩位數增速（從 7% → 12-15%+）；2027 兩會公布
  • 中儲糧儲備從 18 個月 → 24+ 個月（月度公告可查）
  • SPR 儲油從 110-120 日 → 180+ 日覆蓋
  • 醫療戰備（血漿、繃帶、麻醉劑）集中採購量 +50% 以上
  • 稀土、鋰、銻、鎢戰略資源向沿海 / 大倉庫集中調度
  • 人民幣匯率波動區間收窄；央行 NDF 異常操作

軍事線 D-180

  • PLA 建制級部隊（集團軍 / 戰區）異常調往福建、廣東、浙江沿海
  • 075 + 076 型兩棲艦全數集結到同一海域 / 港口
  • 民用滾裝船、集裝箱船演練規模超過歷史單次 3 倍
  • H-6N 轟炸機攜帶 YJ-21 反艦飛彈出海頻次翻倍
  • 太平洋 / 南海近海水雷布設 + 掃雷演練顯著增強
  • 台灣東岸海底電纜連續異常（3 個月內 3+ 次）

政治線 D-180

  • Xi 公開行程異常——重大國際會議缺席、或連續視察前線部隊
  • 中央軍委擴大會議頻率 +50% 以上
  • 《人民日報》/《解放軍報》社論從「統一不可避免」轉為「統一是最後期限」調性
  • 中央統戰部 / 國台辦高層公開場合降溫 / 消失（表示內部「文路線」被擱置）
  • 外交部發言人對台措辭硬化（「任何形式」→「軍事手段保留」）
  • 駐外使館大量召回高級官員「述職」

經濟線 D-30

  • 資本管制實質強化（境外匯款凍結、外幣匯兌限制）
  • 黃金進口單月 300+ 噸（2 倍常態）
  • 重要 H 股 / A 股突然停牌（防凍結 / 防空襲準備）
  • 中國遊客 / 留學生 / 商務人員大規模撤僑
  • 央行突然大幅降準 / 向大型國企注資 1 兆+ RMB
  • 跨境電商 / 出口訂單異常快速結案（存活資金回流）

軍事線 D-30

  • PLA 部隊進入一級戰備（公開演習轉為實戰部署）
  • 空中加油機出動 2-3 倍常態（延伸作戰半徑訓練）
  • 055 型 + 航母打擊群集結第一島鏈東岸
  • PLA 戰地醫院 / 醫療部隊前推部署
  • 電子戰部隊高強度演習（GPS 干擾、通訊壓制測試）
  • 大規模飛彈燃料 / 彈藥配送（衛星可觀察）

政治線 D-30

  • Xi 本人停止所有國際露面 / 外交活動
  • 政治局常委全體集中北京，無分散活動
  • 國台辦發言人發表「最後勸告」級聲明
  • 中共中央 / 國務院緊急文件密集發布（戰時動員預備）
  • 軍隊反腐 / 紀檢突然停止新立案（政治穩定需求）
  • 駐外武官、經濟商務辦事處高階人員大量撤離

經濟線 D-7

  • 北京 / 上海 / 深圳 / 廣州電力、交通「應急演練」頻率翻倍
  • 中國國際貿易促進委員會異常發照加速（結案 outstanding 合同）
  • 央企 / 國企 CEO 突然失聯 / 緊急開會
  • 人民幣跨境結算系統（CIPS）異常運作（戰時轉型測試）

政治線 D-7

§4 外部變數整合：Iran + Venezuela 對預警儀表板的影響

§4.1 Iran 戰爭（2026-02-28）對儀表板基線的調整

Iran 戰爭改變了儀表板的某些基線假設：

  • 經濟線基線調整：中國 SPR 覆蓋日數原本 110-120 日是「常態」，但在 Iran 戰爭後，原油進口成本上升 + Venezuela 斷供壓力下，SPR 從 110 日 → 90 日（實際消耗累積）。原本「SPR 180+ 日」的 D-180 訊號閾值，需要重新校準為「SPR 從當前低點快速回補」的相對變化
  • 軍事線基線調整：Iran 戰爭期間 PLA 演習 / 擾台軍機頻率並未明顯變化，顯示 PLA 不受 Iran 事件干擾。這是基線「靜默」。若 Iran 停戰後 PLA 活動突然升級，才是真正訊號
  • 政治線基線調整：Xi 在 Iran 戰爭爆發後表態克制（2026-03-01 僅呼籲「對話」），政治線基線保持。若 Xi 突然對 Iran 問題發表「中國支持 Iran 抵抗西方霸權」式強硬表態，可能是轉向強硬軌的訊號

§4.2 Venezuela 政權轉移對儀表板的影響

  • 軍事線：中國海軍 2026 Q1 意外加強南海 / 西沙 / 南沙演訓，可能是受 Venezuela 事件刺激的「區域自信重建」，而非對台灣動武準備。但需觀察後續軌跡
  • 政治線：俄中 + 伊朗 + 北韓 在 Maduro 被捕後 UN 密集聯合聲明（抗議美國「軍事霸權」），這是政治線上「四國軸心」激活的訊號。若此聲明密集度持續升高，可能是戰略對抗強化的前奏
  • 經濟線：中國 2026 Q1 加速 Brazil / Angola / Guyana 石油進口談判，這是能源備選重建。若中國開始大規模囤積（而非漸進替代），才是 D-180 訊號

§4.3 Iran + Venezuela 對「美國反應速度」變數的影響

預警儀表板的關鍵假設：美國能否在 D-180 時點識別訊號 + D-30 前反應。Iran / Venezuela 事件顯示：

  • Trump 政府可快速決策：Iran Fordow 空襲決策週期從「情報到行動」僅 14 天；Venezuela 突襲決策僅 21 天。這證明 Trump 個人決策速度遠高於 Biden 政府
  • 美軍特種部隊 / 精準打擊能力驗證：若 Xi 決定動武，美軍從訊號識別到實質干預的反應速度可能在 48-72 小時內。這對 Xi 的「快速決勝」計畫是重大威懾
  • 美國情報體系對 Xi 決策層的滲透程度：Iran Khamenei 擊殺 + Venezuela Maduro 捕獲顯示美國特定情報來源滲透對手決策層核心。Xi 的政治線訊號可能在被 Xi 自己意識到之前就被美國情報掌握

§6 預警訊號：如何實操監測

§6.1 經濟線監測工具

  • 中國官方數據源：NBS（國家統計局）、PBOC（央行）、Ministry of Finance、中儲糧 / 中儲油月度公告
  • 國際機構：IMF、World Bank、Reuters 經濟數據庫、Bloomberg 中國經濟儀表板
  • 研究機構：Rhodium Group、Gavekal Dragonomics、Peterson Institute、CEIC 數據
  • 替代數據：港口吞吐量、貨運量（Baltic Dry Index）、PMI、衛星夜間燈光（NOAA）
  • 監測頻率：月度更新，突發訊號即時反應

§6.2 軍事線監測工具

  • 衛星圖像：Planet Labs、Maxar、Sentinel Hub（Copernicus）、PlanetScope（每日重訪）
  • 海事追蹤：MarineTraffic、VesselFinder、Global Fishing Watch（海警 / 民兵船識別）
  • 航班追蹤：FlightRadar24、ADS-B Exchange（軍機異常可見）
  • OSINT 社群：@dmitriyrt、@nrg8000、@Aurora_Intel、@RALee85（烏戰俄軍分析）對中國做類似監測的 @Quocai999、@RupprechtDeino、@detresfa
  • 智庫報告：CSIS China Power Project、USNI News、RAND Corporation、Taiwan INDSR、Project 2049 Institute
  • 監測頻率：日度關鍵訊號，週度綜合分析

§6.3 政治線監測工具

  • 官方媒體：《人民日報》、《解放軍報》、新華社、央視（調性變化）
  • Xi 行程追蹤：新華社「時政新聞」、央視《新聞聯播》開場「XXX 同志」序列分析
  • 政治局 / 中央軍委會議記錄：中央網信辦、國務院發布
  • 外交措辭：外交部發言人記者會、UNGA / UNSC 中國發言
  • 國台辦 / 中央統戰部動態：國台辦月度新聞發布
  • 研究機構：China Leadership Monitor (Hoover Institution)、Sinocism (Bill Bishop)、China Digital Times
  • 監測頻率：週度深度分析 + 重大事件即時

§6.4 False Positive 控制方法

  1. 多訊號合併：單一訊號觸發不調整，2+ 訊號 + 跨線才觸發
  2. 歷史對比：與 2020（COVID）、2022（裴洛西訪台）、2024（聯合利劍 A/B）等事件對比，濾掉「歷史常態」信號
  3. 時間視窗：訊號需在 30-60 天視窗內持續 / 重複，單次孤立訊號視為噪音
  4. 共識參照：2+ 獨立機構 / 智庫同時預警
  5. Xi 個人行為加權：政治線權重 >軍事線 > 經濟線（因為政治線是決策訊號）

§8 風險與結論

§8.1 三個最大風險

  1. Data 風險：衛星 / OSINT 覆蓋不均——中國許多關鍵基地（西藏、新疆、四川山區）衛星覆蓋有限或被雲遮蔽；民兵、海上準軍事船隊不在 AIS 上可見。本篇儀表板對結構性訊號敏感，對隱匿訊號弱
  2. 前提風險：訊號「欺敵」問題——PLA 可以刻意製造 false positive（大規模演習但不戰）或 false negative（隱匿真正準備）。本篇多訊號合併降低但不能完全消除
  3. Tail risk：「黑天鵝」事件——偶發事件（如賴清德意外、美台突發事件、中國意外事故）可能快速升級，繞過 D-180 / D-30 時點直接進入 D-7。這時儀表板失效

§8.3 本篇核心結論（3 句話）

1. 三線 × 三時點 9 格儀表板是實用決策框架：經濟 × 軍事 × 政治 × D-180/D-30/D-7 構成完整預警系統，每格 3-6 個可監測 tripwire，搭配 false positive 控制方法。

2. 政治線訊號比軍事線更早、更可靠：Xi 的個人行程、中央軍委擴大會議、官方媒體調性、外交措辭是真正的戰略決策訊號，通常比軍事部署早 3-6 個月。

3. Level 1-5 警戒系統對應明確倉位調整：當前 Level 2（灰色地帶常態）基本 Base case；Level 3 啟動 Stress 前期（加碼國防 / 稀土 / 減碼台廠）；Level 5 啟動 Tail 避險（現金 / 黃金 / 瑞郎）。基金經理人應每月監測，重大事件即時調整。
