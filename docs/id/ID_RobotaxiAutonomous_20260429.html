§1 產業定義與邊界

Robotaxi / 自動駕駛 vertical AI 在本 ID 的範圍是商業化 L4+ 自動駕駛 ride-hailing 服務 — 包含載人 robotaxi（Waymo / Tesla / 中國四家 / Zoox / Mobileye-powered）、商用 trucking（Aurora — vertical adjacent）、以及上游 supplier ecosystem（NVIDIA compute / Hesai+RoboSense LiDAR / 整車 OEM 如 Hyundai / Toyota / Geely / Zeekr）。本 ID 與 ID_AICrossDomainImpact 互鎖 — 跨域母題明確將 robotaxi 列為「out-of-scope vertical 應用，獨立 ID 待建」[T1: ID_AICrossDomainImpact §1 灰色地帶]，本 ID 即為填補此 gap。

邊界判斷的關鍵是「是否商業化 L4+ 並有 driverless miles」與「是否在 ride-hailing TAM 內」兩條軸：載人 robotaxi（Waymo / Apollo Go / Pony / WeRide / Zoox / Mobileye Drive / Tesla Robotaxi 服務）— 全部 in-scope；商用 trucking（Aurora L4 truck）— in-scope（vertical adjacent，share AI stack 但 ODD 不同）；ADAS L2/L3（Tesla FSD supervised customer cars / Mercedes Drive Pilot / GM Super Cruise）— out-of-scope（不是 driverless ride-hailing）；無人外送 / 巴士 / shuttle（Nuro / EasyMile / May Mobility shuttle）— 部分 in-scope 作為 vertical 整合 case；機器人計程車 startup 已退場（Cruise / Argo / Motional）— 註記但不深入；AI 平台層（OpenAI / Anthropic / Google Gemini）— out-of-scope（在 ID_AICrossDomainImpact 母題）。

另一個關鍵邊界是地理：美國（Waymo / Tesla / Zoox / Mobileye-Lyft / Aurora 主場）、中國（Apollo Go / Pony / WeRide / Didi / Caocao / 新興 Geely Eva Cab 主場）、海外擴張（Waymo Tokyo + London、Pony Dubai、WeRide 12 國 Switzerland + Abu Dhabi + Dubai）。本 ID 不單獨處理歐洲市場 — 歐洲落後 18-24 個月，Mobileye Drive + 中國出口為主。

第四個邊界爭議 — AI 平台層 vs vertical 應用層：Robotaxi 的 AI stack（end-to-end NN / VLA / world model）與 generic LLM 平台層共用基礎架構（NVIDIA Blackwell / Alpamayo VLA），但 vertical 訓練 data + 法規 + 硬體 constraints 完全不同；本 ID 視 robotaxi 為 vertical AI 中最 capital-intensive + regulation-bound 案例，類似 GLP-1 之於 generic LLM — 同樣 compute backbone 但完全獨立 monetization path。

💡 §1 Insight
1. 「Tesla Robotaxi」是當前 sector 最 mislabel 的玩家：Austin 14 crashes / 安全員實質駕駛 / Bay Area 完全 supervised — 真實 driverless miles < 1M vs Waymo 170M+；市場用「Tesla = robotaxi 軟體贏家」框架是 narrative 不是 evidence；事實基礎在 §2 + §6 玩家矩陣。投資意義：TSLA SOTP 中 robotaxi segment 應給 zero or negative 估值直到 FSD v15 + 100K-vehicle scale 同時兌現，2027 H2 前最早。

§2 技術演進 + S 曲線 + 歷史類比

Robotaxi 從 2009 Google Chauffeur 計畫起步至今 17 年，經歷 3 個明確階段。Phase 0（2009-2018）是技術 PoC 期 — Google / Cruise / Uber ATG / Argo / Aurora / Tesla Autopilot 競相 demo，但無 commercial 規模；2018 Tempe Uber AV 撞死行人事件 + Cruise / Argo 重組是 phase 結束訊號。Phase 1（2018-2024）是早期 commercial 試點 — Waymo One Phoenix 2018 開放 / 2020 完全 driverless / Apollo Go Beijing 2021；但 Cruise SF 2023 撞傷行人事件後 GM 直接 fold，$10B 損失證實「technical demo ≠ commercial scale」[T2: TechCrunch 2024-10]。Phase 2（2024-2030）是規模化期 — Waymo 50K weekly（May 2024）→ 500K（Mar 2026）→ 1M target（end-2026）的 10x → 20x 跳躍是 sector 進入 Phase II 的明確訊號[T1: Waymo 2025 Year in Review]。

當前位置（2026-04-29）是 Phase II 早段。三個明確訊號：(1) Waymo 從「single-city demo」轉為「12 cities + 20 more 2026 含 Tokyo/London」國際擴張[T1: Automotive World 2026-02]；(2) China Goldman 5K → 14K fleet 2026 是 commercial 跳躍 + Apollo Go RT6 -60% cost 是 unit economics inflection[T2: Goldman China Robotaxi 2026]；(3) NVIDIA DRIVE Hyperion + AGX Thor design wins（Pony / SWM / Hesai / Magna / ZF）是 sensor + compute ecosystem 規模化證據[T1: NVIDIA DRIVE Hyperion]。但 Phase II 不是均勻分布 — 美國 / 中國 / 歐洲 三軌差距 12-24 個月。

歷史類比（v1.7 必須含正反例）

正類比 #1（Waymo 規模化 ≈ 2010-2014 Smartphone iPhone vs Android）：iPhone 2007 上市但 2010 才開始大規模 — 同樣 robotaxi 2009 Google Chauffeur PoC 但 2024-2026 才規模化；當年 Nokia / BlackBerry / WindowsPhone 都是 me-too，最終 Apple + Google Android 雙寡占；同樣現在 Cruise / Argo 死亡，Waymo / Apollo Go 雙寡占明確[T2: Contrary Research]。差異：Robotaxi 是 capital-intensive + regulation-bound，scale 速度比 smartphone 慢 5-10x。

正類比 #2（中國 robotaxi 主場規模 ≈ 2015-2020 中國移動支付 Alipay/WeChat Pay）：當年中國移動支付規模一次到位（無美國 Visa/Mastercard 路徑依賴），2020 中國移動支付 user base 5x 美國；同樣現在 Goldman 預測中國 robotaxi 2028 fleet 50-100K vs 美國 30-50K，主因（a）中國地理一次到位（無 50 州監管碎片化）+（b）Hesai+RoboSense LiDAR 88% 寡占 + 政府公開鋪路。差異：移動支付 cross-border 受限（支付寶在美國基本無使用），但 robotaxi 中國技術可出口（WeRide 已 12 國 / Pony Dubai）。

反類比 #1（Tesla camera-only ≠ 2007 iPhone）：Tesla narrative「特斯拉 = AV 時代的蘋果」是 misleading — iPhone 上市時 hardware（multi-touch）+ software（iOS）+ App Store 同時 step-function，但 Tesla 2026 unsupervised launch 14 NHTSA crashes 證明 hardware（pure camera + HW3/HW4）+ software（FSD v14.3）尚未到 step-function；本次與前例的量化差異：iPhone 2007 上市後 12 月內 1M units sold + 持續 2007-2010 +50%/year，Tesla Robotaxi 2025-2026 fleet 從 0 → 1,500+ 但 driverless miles < 1M 是 demo scale 不是商業 scale。[T3: CBS News 14 crashes]

反類比 #2（Robotaxi ≠ 2000-2002 光纖 / .COM 過度建設）：對 sector「過度樂觀 / TAM 高估」的反方常用 .COM 類比；當年 .COM 光纖 2001 capex / 實際 utilization < 5%，但 robotaxi 2026 Waymo 500K weekly rides + 91% safer 是 actual rides not capex — 與 .COM 「shipped but unused」結構不同。但類比 partial valid：ARK $10T TAM 2030 是 .COM 級樂觀；Goldman $25B 2030 較實際；本 ID 採 base case。

反類比 #3（Cruise 死亡 ≠ Tesla 必死）：Cruise 2023 SF 撞傷行人後 GM 直接 fold + $10B 損失，常被引為「robotaxi 是 capital trap」；但 Cruise vs Tesla 結構不同 — Cruise 完全 supplier-dependent + 無 vertical 整合 + GM 不肯 commit capital；Tesla 有 vertical 整合（製造 + AI + fleet）+ 強現金 + 軟體更新 OTA 路徑。差異：Tesla 死亡機率雖低於 Cruise，但「永遠卡 100K-vehicle」機率仍 50-60%（見 §9.5 Kill Scenario）。

§3 核心技術棧拆解

Robotaxi 技術棧分 5 層 — Sensor / Compute / AI Model / Map / Operations。每層都有獨立瓶頸 + 獨立贏家結構。當前最重要的「kingmaker」子技術是低成本 LiDAR + Vision-Language-Action（VLA）model兩者，前者解決 BOM 經濟學，後者解決 long-tail edge cases。

第 1 層 Sensor是 robotaxi BOM 最大成本中心 — 2018 全感測器套件 $250K（Waymo 5th gen），2026 降至 $30-50K（Waymo 6th gen + Hesai $1K LiDAR），2028 預估再降至 $15-25K[T2: PatentPC AV Components Cost]。Waymo 6th-gen Driver target sub-$20K（含 LiDAR + camera + radar + 計算）是 sector 中最 aggressive cost-down。Tesla 「camera-only」邏輯是 BOM 領先 — 但代價是 redundancy 缺失 + edge case 死路；當 LiDAR < $500 時，camera-only 不再有 meaningful 成本優勢。Hesai 60% L4 share + RoboSense Q1 2026 +204% YoY 是 sensor 規模化具體訊號[T1: RoboSense Q1 2026 PR]。

第 2 層 Compute是 NVIDIA 生態主導 — DRIVE Hyperion + AGX Thor 雙 Blackwell SoC 提供 2,000 FP4 TFLOPS[T1: NVIDIA DRIVE Hyperion]。Pony.ai 新世代 platform / SWM-Lenovo AD1 / Hesai / Aeva / AUMOVIO / Bosch / Magna / Sony / ZF 都是 design wins[T1: Pony.ai NVIDIA DRIVE Press Release]。Tesla 是唯一 vertical 整合（Dojo + HW3/HW4/HW5），但 Dojo 訓練 cluster 規模仍遠落後 NVIDIA-powered 的 hyperscaler；中國（Baidu Kunlun）部分 vertical 整合但仍多 NVIDIA-mix。

第 3 層 AI Model是 sector 中技術迭代最快層。三類路線並行：(a) End-to-end NN（Tesla FSD v12+，單一 video transformer 取代 300K 行控制碼）[T2: FredPope Tesla FSD v12]；(b) Modular + HD map（Waymo / Apollo Go，感知 / 規劃 / 控制分離 + 厘米級地圖每週更新）[T2: Contrary Research]；(c) Vision-Language-Action（VLA）（NVIDIA Alpamayo open-source / Wayve LINGO-2 / AutoVLA NeurIPS 2025）— 將 LLM-style chain-of-thought reasoning 帶入駕駛決策，特別針對 long-tail edge cases[T1: NVIDIA Alpamayo Newsroom]。當前共識認為 VLA 是 2027-2030 主流路線 — Tesla end-to-end 是 VLA 的純文本變體，Waymo 已開始實驗 VLA + HD map hybrid。

第 4 層 Map是 Tesla vs Waymo 路線的核心分歧。Waymo / Apollo Go 採 HD map（Waymo 12 cities 已 mapped + 20 more 2026 mapping），優點是 known-road redundancy + edge case fallback；缺點是 mapping CAPEX 高 + 新城市 24-36 個月 ramp。Tesla 採「No HD Map」依賴 real-time perception — 優點是任何道路可開但缺點是每個交叉口視為「新」[T2: Tesla FSD vs Waymo perception]。VLA 的興起讓「中間路線」可能 — Wayve LINGO-2 + Alpamayo 是「light HD map + VLA reasoning」結合。

第 5 層 Operations是 Phase II 真正 unit economics 突破點。當前所有 robotaxi（Waymo + Apollo Go + Tesla unsupervised）都需遠端操作員 (remote operator) 干預 0.05-0.5 events / hour[T3: Understanding AI]。當前比例約 1 operator : 5-10 vehicles；Phase II 真正突破是 1:50（規模化 + AI 自主決策升級）。Waymo 6th-gen + VLA 路線預計 2027 達 1:30，2030 達 1:50；Tesla 因 supervised 階段更靠人工監控比例可能更糟。Apollo Go RT6 spec 不公開但 Baidu 暗示 1:20。

Robotaxi TAM 是 sector 中分歧最大的 forecast — 從 Goldman base case $25B 2030 / $415B 2035 到 ARK 極端 bull $10T 2030，差距 400x。本 ID 採 三層 base case，並指出過去 3 次 TAM forecast 實際偏差。

§6 玩家矩陣

本章為敘述 + 多類玩家比較，先按 5 類分列（規模冠軍 / 軟體挑戰者 / niche / demand layer / supplier），再列 2 檔結構性 loser（QC-I32），最後 4 檔 non-obvious 二次受益者（QC-I33 ≥ 3 檔）。

類別 A：規模冠軍 — 美國 Waymo

Waymo (Alphabet GOOG) 是 sector 中規模 + 安全 + 國際擴張的三冠王。2024 May 50K weekly rides → 2025 December 400K → 2026 March 500K → 2026 end target 1M weekly（10x → 20x 兩年）[T1: Waymo 2025 Year in Review]；累計 driverless rider-only miles 2025 Dec = 170.7M（vs human 150 lifetimes）[T1: Waymo Safety Impact]。城市覆蓋從 5 → 12（Phx / SF / LA + Austin / Atlanta / Miami / Dallas / Houston / SA / Orlando），2026 加 20+ 含 Tokyo + London 國際首發[T1: Automotive World Waymo 2026]。Fleet 從 5th-gen 3,067 vehicles（Dec 2025 NHTSA filing）轉到 6th-gen Driver — 將 deploy 在 Hyundai Ioniq 5 + Zeekr Ojai 兩 OEM；6th-gen Driver hardware target sub-$20K（vs 5th gen $50K+）。$16B 2026 capex（Waymo + Alphabet 直接投資）。Khosrowshahi（Uber CEO）公開稱「Waymo on Uber Austin/Atlanta 比 99% 人類 Uber 司機 productive」[T1: Uber Newsroom Atlanta]。

本 ID 對 Waymo 的核心 thesis 是「Driver-as-a-Service licensing platform」— 6th-gen Driver 將授權給 Hyundai / Zeekr / 未來 OEM；Waymo 不擁車隊，是 software/AI/sensor/operations 平台；類似 Mobileye Drive 但比它早 5 年 + 規模 100x；2027-2030 將收 $1-3K/車 monthly licensing fee。GOOGL 估值因 Waymo 應加 SOTP $100-200B（市場給 ~$50-80B），是市場最未 price in 的 step-function。

類別 B：軟體挑戰者 — 美國 Tesla

Tesla (TSLA) 在 2025-2026 經歷 4 段 launch — (1) June 22, 2025 Austin：safety monitor in passenger seat, $4.20/trip 補貼；(2) Aug 2025 Bay Area：safety monitor in driver seat（實質人駕計程車 + FSD supervised，無 robotaxi 牌照）；(3) Dec 2025 Austin no safety driver：首次 unsupervised but 限定區域；(4) April 18, 2026 Dallas + Houston unsupervised：573 vehicles 真正 driverless launch[T1: Tesla Robotaxi 573 vehicles]。但 Austin 自 June 2025 累計 14 NHTSA-reported crashes，包含進入錯誤車道 / 中段下客 / 突然剎車 / 逆向行駛 / 上人行道[T1: CBS News 14 Austin Crashes]。Cybercab production Apr 2026 Giga Texas，50+ vehicles 在 facility（含 crash test）[T2: Teslarati Cybercab]。FSD v14.3 powering current Robotaxi，Tesla 自承「FSD v15 是 large-scale expansion 必要 architectural rebuild」延至 2026-Q4 + 「dozen US states by end-2026」目標延後[T2: TeslaNorth FSD v15 delay]。

§7 Unit Economics + ASP 動態

Robotaxi unit economics 是 sector 中分歧最大也最關鍵的章節。Phase II 真正 inflection 不是「無人駕駛」而是「per-vehicle profitable + per-ride margin positive」。當前 Waymo 是 sector 中接近此 inflection 唯一玩家，Apollo Go RT6 路徑是第二。

第一個 lever — 車輛 BOM 成本：Waymo 5th-gen vehicle $250K（2018-2024），6th-gen target sub-$150K total（含 sub-$20K Driver hardware）。Apollo Go RT6 「-60% manufacturing cost」具體數字未公開但估 $30-50K（vs Apollo Go 5th-gen $100K+）。Tesla Cybercab target $30K（極激進 but vertical 整合 + 製造 scale）。Pony.ai / WeRide 仍依賴 OEM partner（Toyota / Geely Farizon）整車價 $40-60K + sensor stack。2028 sector 平均 BOM 預估 $30-50K（vs 2024 $250K = 5-8x cost-down）[T2: 01core driverless costs]。

第二個 lever — sensor BOM：2018 全感測器套件 $75K-$250K（Waymo 5th-gen），2026 降至 $30-50K（Waymo 6th-gen + Hesai $1K LiDAR），2028 預估 $15-25K。Hesai SR1 2026 Q3 target $200/unit（vs Hesai AT128 2024 $1K，2018 cutting-edge $75K = 380x cost-down 8 年）[T1: Yole Automotive LiDAR 2025]。

第三個 lever — 每車 daily revenue：Waymo 平均 daily revenue per vehicle = 500K weekly rides / 7 / 3,067 vehicles ≈ 23 rides/day/vehicle × $19.69 avg ≈ $450/day，年化 ~$165K/vehicle/year[T2: TechCrunch Waymo Uber pricing]。對比 Apollo Go 2025 ~$280M revenue / 1000 fleet ≈ $280K/vehicle/year（中國高 utilization 但 ASP 低）。Tesla per-trip $4.20 / per-mile $3.20 vs Waymo $9-10 = Tesla 補貼 50-70%；vehicle utilization 數據不公開但估 < 10 rides/day（vs Waymo 23）。
