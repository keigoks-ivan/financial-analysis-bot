市場把 robotaxi 當成「一個會贏的長期主題」在討論，焦點卻錯位了。真正的分歧不在「自駕計程車會不會成真」（會），而在「誰已經把 demo 變成可規模化的車隊」——這條線把檯面上的玩家切成天差地別的兩半。Waymo 已做到約 50 萬筆週付費訂單、橫跨 10–11 城（較 2024 年 5 月增約十倍）；Tesla 同月開放了整個 Austin metro 給無人車，卻只有約 20 台車在跑。同樣是「里程碑」，量級差了四個數量級。

這是一個典型的早期滲透型產業：產業 TAM 從 2023 年約 $0.4B 衝向 2030 年約 $45B（CAGR ~90%），但「早期高成長」的另一面是估值對里程碑高度敏感——一個城市的法規准入、一季的車隊兌現、一次重大安全事故，都能讓估值大幅 re-rate。Tesla 約 $1.5T 市值、本益比逾 300 倍，已經把 robotaxi 的未來 price 進去大半（多家模型估 robotaxi 占其企業價值 ~60%，ARK 更估 2029 達 ~90%），這意味著它買的不是現況、是兌現。

所以本報告不站隊「robotaxi 泡沫 yes/no」。方向幾乎沒有爭議，分歧在每一檔的「已驗證 vs 待兌現」落差，以及這落差被估值 price 了多少。判斷重心放在「曝險該放在規模已驗證端，還是規模未驗證的故事端」——以及賣鏟者（NVDA 算力、Hesai 感測）為何在這個階段風報常比車隊運營商更均衡。

PART I · DECISION SUMMARY

決策摘要層

三句話看完 robotaxi 的現狀、走向與行動，後接 PM 級監測點。這一層是報告的全部重點——下面各節只是把這一層的每個判斷補上證據。

現在 / NOW

Phase II 早段、規模差極大。Waymo 約 50 萬週付費訂單（10–11 城、較 2024-05 增約十倍）領跑；Tesla 已開放全 Austin 但無人車隊僅約 20 台、規模未現；中國 Apollo Go 約 30 萬週訂單（武漢單車獲利）、Pony.ai 首見季度 GAAP 獲利、WeRide 毛利約 35%。GM Cruise 已退出（虧約 100 億美元）。

未來 / NEXT

未來 3–5 年看「無人車隊能否從數十／數千跨到數萬」與法規准入擴張。Waymo 目標年底破百萬週訂單並出海倫敦／東京；Tesla 全押 unsupervised FSD、硬體已鋪、車隊兌現是賭注；中國靠成本（RT6／Hesai 約 $200 LiDAR）搶滲透。最大裂縫是估值對里程碑高度敏感。

行動 / ACTION

早期高成長＝高波動，非全進場區。曝險走「規模已驗證」端（Waymo→GOOG 隱含、UBER 需求層、Hesai 感測賣鏟）優於「規模未驗證的故事」端（TSLA robotaxi 已被估值 price 多數、容錯薄）。中國三雄高 beta、政策敏感。GM／純美系 LiDAR 結構承壓。

The Key Call · 本報告最重要的一個判斷

市場把 robotaxi 當「一個會贏的長期主題」在估值，真正的非共識是——這個階段最該定價的不是「方向」（自駕計程車會成真，幾乎無爭議），而是「已驗證的車隊規模 vs 待兌現的故事」之間的巨大落差。Waymo 約 50 萬筆週訂單對上 Tesla 全 Austin 卻只有約 20 台車，是同一個「里程碑」標籤底下、量級差四個數量級的兩個世界。在早期高成長產業，估值對里程碑兌現極度敏感——Tesla 約 $1.5T 市值已 price 了 robotaxi 占企業價值 ~60% 的未來。分歧不在「robotaxi 會不會贏」（會），而在「你買的是已驗證的規模，還是估值已透支的兌現」。

三道裂縫具體是：①估值已透支兌現——Tesla 本益比逾 300 倍、robotaxi 占其企業價值多數（ARK 估 2029 達 ~90%），等於把「unsupervised 車隊規模化零失誤」全 price 進去，而現況是全 Austin 僅約 20 台、且車隊不增反縮；②法規准入＝真正的供給瓶頸——Tesla 一邊擴地理範圍一邊背著 NHTSA 兩條調查（PE25012 涉 58 起 FSD 事件、EA26002 涉約 320 萬輛視覺系統），任何重大事故或退牌都能瞬間關掉一座城；③規模口徑陷阱——週訂單、車隊台數、營收三者不可混用比快慢（Tesla 地理覆蓋廣 ≠ 車隊大；中國營收高成長但基期低）。盯這三條，比盯任何單一城市的「開通」新聞都更早看到拐點。

PART II · HISTORICAL ANALOGS

兩個歷史類比，框住早期高成長的天花板與地板

歷史不必重演，但兩個對照各給一個可操作教訓：2010 年代電動車早期框出「方向對但時程與燒錢遠超想像」，2007–2012 智慧手機框出「平台與規模一旦領先就贏家通吃」。

類比一：電動車早期（2012–2018）——方向沒錯，但「兌現的時程與資本消耗」遠超市場預期。2012 年 Tesla Model S 上市時，多數人同意「電動車是未來」，但低估了量產地獄、充電基建、單位經濟轉正所需的年數與燒錢規模——特斯拉燒了將近十年、瀕臨破產數次才達到規模化獲利。對 robotaxi 的含義是對稱的：方向（自駕計程車會成真）幾乎無爭議，真正會傷人的是「規模化的時程比敘事慢、資本消耗比模型大」。GM Cruise 就是這個教訓最新的犧牲者——花了約 $10B、卻在「規模化所需的時間與資本」面前選擇退出（2024 年底停止注資，併回 Super Cruise 個人車 ADAS）。這把「早期高成長」的天花板框得很清楚：CAGR ~90% 的產業曲線是真的，但個別玩家會在「兌現前的燒錢谷」裡分化出生死。

怎麼讀：前兩項說明「方向不是分歧、瓶頸在准入」；第三項是地板——燒錢谷會淘汰跟不上的玩家（Cruise 已驗證）；後兩項是天花板——估值對里程碑敏感、且早領先者（Waymo 的數據飛輪）護城河會擴大。空頭若純押「robotaxi 太貴」會像當年純押「電動車不可能賺錢」一樣過早；要盯的是個別玩家的「規模兌現 vs 燒錢速度」。

類比二：智慧手機平台（2007–2012）——規模與數據一旦領先，就是贏家通吃的飛輪。2007 年 iPhone 問世、2008 年 Android 開源，五年內把功能機巨頭（Nokia、BlackBerry）從市佔過半打到幾乎歸零——關鍵不是單一功能，而是「裝置量 → 開發者 → 應用 → 更多裝置」的網路效應飛輪。Robotaxi 的飛輪是數據版的：跑越多無人里程 → AI 越安全 → 法規越敢放行 → 更多城市 → 更多里程。Waymo 已累積約 1 億英里以上純無人載客里程，這個數據領先正是它能在一年內把週訂單衝十倍（5 萬 → 50 萬）的底氣。教訓清楚：在早期，規模領先不是線性優勢，而是會自我強化的護城河——這把「地板」框住（早領先者很難被後進者線性追上），但也提醒「載體若再轉移（如更便宜的純視覺方案大規模驗證），現任領先者的飛輪也可能被繞過」，這正是 Tesla bull case 的賭注所在。

PART III · SUPPLY SIDE

供給側：稀缺不是產能，是法規准入＋AI 數據＋車隊規模

反直覺但可操作的結論：robotaxi 不是「造夠車就有供給」的商品產能問題——車子本身不稀缺，稀缺的是「能在某座城市合法跑無人車」這張准入門票，而它由三道門檻共同決定。

供給端的結構是「少數能跑無人載客的領先者 + 一票還在安全駕駛或測試的追趕者」。一個 2026 年合格的 robotaxi 供給者要同時跨過三道門檻——①法規准入（某城／某州的無人載客許可，如 Texas L4 法 5/28 生效、加州 CPUC 許可、中國各市牌照）；②AI／數據護城河（足夠的無人里程把 AI 訓練到法規敢放行的安全水準）；③車隊規模 + 單位經濟（把單車成本壓到能規模化獲利，如 Apollo Go RT6、Hesai 把 LiDAR 從數萬美元壓到約 $200）。能三道全跨的，全球僅 Waymo、Apollo Go、Pony.ai、WeRide 等少數；Tesla 跨了法規（自我認證）與部分 AI，但車隊規模端尚未兌現（全 Austin 僅約 20 台）。

Inference · 供給（框架逃逸 · 架橋）
這不是商品供需框架——稀缺來自准入門檻，不是產能曲線
前提：車輛本身不稀缺（整車產能充足）＋ 真正的限制是「合法跑無人載客」的三道門檻（法規准入 + AI/數據 + 車隊規模單位經濟）＋ 門檻有強數據飛輪正回饋

傳統供需框架問「產能會不會過剩、價格跌到誰先停產」——但 robotaxi 沒有實體產能的成本曲線可言，車子要多少有多少。真正的供給瓶頸是准入：你能在幾座城市合法跑無人車，由法規 + 累積無人里程的安全紀錄決定。所以這是早期滲透型產業（看「滲透到哪、誰拿到准入」），不是商品週期（看「過剩／短缺」）。稀缺性來自三道門檻的交集，而非產能；領先者的數據飛輪讓門檻越堆越高（跑越多里程 → 越安全 → 越多准入），後進者越難線性追上。

可證偽條件：若出現「更便宜的純視覺方案大規模通過法規 + 安全驗證」（如 Tesla unsupervised 在多城零重大事故規模化），讓不需高累積里程也能拿准入，則「數據飛輪 = 護城河」論失效、供給門檻被重訂、領先者優勢收斂。

怎麼讀：規模指標三種口徑並列（週訂單 / 車隊台數 / 毛利）但不交叉比快慢——Waymo 與 Apollo Go 用週訂單（已商業化規模），Pony.ai / WeRide 用車隊台數（規模兌現中），Tesla 用「地理覆蓋 vs 車隊台數」的落差揭露其「規模✗」。成本曲線省略說明：robotaxi 無實體產能的邊際成本曲線；車子不稀缺，價格戰不由單位停產點決定，而由「准入 + 單位經濟（誰先把單車成本壓到獲利）」決定——這也是中國靠 Hesai 約 $200 LiDAR + RT6 低成本車，能在成本端領先的根本原因。

PART IV · DEMAND SIDE

需求側：載客需求 vs 規模兌現的口徑陷阱

需求端最不可替代的是「比人類司機更便宜 + 更安全 + 隨叫隨到」的剛性——但本節最該防的不是「需求夠不夠」，而是「用什麼口徑衡量需求，才不會被快慢假象騙了」。

終端需求其實不缺：全球載客出行（ride-hailing + 計程車）是兆美元級市場，robotaxi 的價值主張是「拿掉司機成本（約佔每趟 60–70%）」，理論上可大幅壓價搶量。需求引擎有兩個：①單位成本下降——Waymo $30/月 Premier 會員、中國 Apollo Go 補貼價，都在測試「比 Uber 便宜」的彈性；②需求層整合——Uber（$10B+ AV capex、10+ 國、Lucid-Nuro / Wayve / Apollo Go / WeRide 多家整合）與 Lyft 把 robotaxi 接進既有 app，等於替車隊運營商現成導流。需求方向不是問題，問題在衡量口徑。

Bottom-up 規模（車隊）Pony.ai >1,400 / WeRide ~1,000 / Tesla ~20無人車隊台數口徑（≠ 訂單 ≠ 營收）

怎麼讀：這張表故意把三種口徑分行列出且標明「不可交叉比快慢」——「Tesla 全 Austin」（地理覆蓋）不能拿來跟「Waymo 50 萬週訂單」（需求規模）比，因為前者車隊僅約 20 台；「中國營收 +395% YoY」（Pony.ai）不能拿來跟絕對規模比，因為基期極低。bottom-up 的週訂單 + 車隊規模仍遠小於 top-down TAM，正說明這是早期滲透（滲透率個位數），缺口是「成長空間」而非「口徑誤差」——與成熟產業的三角驗證邏輯相反。

Inference · 需求
需求結構性存在、但「量」要用對口徑、「價」由單位經濟決定
前提：載客出行兆美元市場 + 拿掉司機成本（每趟 ~60–70%）的價值主張 + Uber/Lyft 需求層現成導流 → 需求方向不缺，缺的是規模化的單位經濟

需求的「量」幾乎無上限（出行剛需），所以這個階段真正的變數不是「有沒有人要搭」，而是「車隊運營商能否把單位經濟做正、且規模化」。Waymo 已用約 50 萬週訂單證明需求承接得住；中國三雄用「首見獲利 / 武漢單車獲利」證明單位經濟可轉正；Tesla 則尚未用規模驗證。需求引擎沒熄，但要盯的是「規模兌現的速度」與「單位經濟的持續性」，不是需求本身。

可證偽條件：① 主要市場出現重大安全事故導致法規收緊、需求被監管壓抑（量的風險）；② 補貼退場後單車經濟由正轉負、規模化即虧損放大（價的風險，中國補貼依賴是早訊）。

多空為何都對

多頭測的是方向與規模領先者（自駕成真、Waymo 數據飛輪、中國成本下降），這幾乎沒爭議、很可能持續；空頭測的是估值與兌現時程（TSLA P/E >300 已 price 兌現、法規退牌風險、燒錢谷）。兩者可同時為真：方向確實是順風（多頭對），但「早期高成長」不代表現價已 price 對兌現速度（空頭對，Cruise 是活教材）。所以本報告把判斷重心放在「已驗證規模 vs 估值透支的兌現」，而非「robotaxi 會不會發生」。

PART V · VERDICT & SCENARIOS

裁決與三情境（24–36M）

產業方向判 INTACT（自駕成真、規模在加速），規模領先者護城河判 INTACT（Waymo 數據飛輪 + 中國成本下降），但「估值對里程碑兌現的敏感度」判 AT_RISK（TSLA 已透支兌現、法規退牌風險未排除）——故 thesis 結構看多、但 conviction 卡在 mid、進場點與標的選擇極敏感。

Base Case主觀機率 · 較可能
規模續行、領先者吃滲透，估值高位震盪

PART VI · TAXONOMY

個股特徵與 conviction tier

本框架只輸出「特徵」與「分類」，買哪一檔由個股 DD 系統與 Pure MA 訊號決定。以下 tier 為產業推論的特徵歸類，提及公司僅為「該特徵的代表案例」，不附帶買賣建議。

CORE 🔴

GOOG（Waymo）

規模冠軍、數據飛輪最深的代表。約 50 萬筆週付費訂單、10–11 城、約 1 億英里以上無人載客里程，是「規模已驗證」端最純的特徵；但 Waymo 在 Alphabet 內占比極小，robotaxi 價值隱含在整體估值裡、非純 play——特徵是「規模最實、但難以單獨 price」。conviction：中高（規模面最硬、但曝險被稀釋）
CORE 🔴

TSLA

unsupervised / 純視覺路線的最大賭注。2026-06-03 開放全 Austin metro 但無人車隊僅約 20 台、且不增反縮；硬體已鋪、車隊兌現是賭注。估值已 price robotaxi 多數（市值約 $1.5T、P/E >300，多家模型估 robotaxi 占企業價值 ~60%、ARK 估 2029 ~90%）——容錯空間極薄，背著 NHTSA 兩條調查。特徵是「最大上行想像、但估值已透支兌現」。conviction：中（bull/bear 兩極、里程碑敏感最重）
EDGE 🟡

BIDU / PONY / WRD（中國三雄）

成本領先 + 首見獲利的滲透型玩家。Apollo Go ~30 萬週訂單、武漢單車獲利；Pony.ai 首見季度 GAAP 獲利、車隊 >1,400 → 目標 >3,500；WeRide 毛利 ~35%、中東＋歐洲出海。特徵是「單位經濟先轉正、但高 beta、政策／地緣敏感（含美國上市監管風險）」。conviction：中 · 視政策與獲利持續性
EDGE 🟡

UBER / NVDA / HSAI（需求層＋賣鏟者）

不賭單一車隊輸贏的「中立受益」特徵。UBER 是 AV 需求層（多家整合、現成導流）；NVDA（DRIVE/Thor 算力）與 HSAI（LiDAR 約 $200、2025 首見全年獲利、中國 51% share）是賣鏟者——robotaxi 誰贏都用算力與感測。特徵是「風報較均衡、不押單一技術路線」。conviction：中（賣鏟者隨產業 beta，較不押勝負）
EDGE 🟢

GM / LAZR / INVZ（結構承壓端）

早期燒錢谷與中國寡占的受壓端，非 robotaxi long。GM Cruise 已退出（虧約 $10B、併回 Super Cruise）；LAZR / INVZ 美系 LiDAR 被 Hesai / RoboSense 中國寡占擠壓。特徵是「被燒錢谷淘汰或被成本擠壓」。conviction：低 · 結構承壓

估值面總結：本主題的核心張力在「方向很清楚、但兌現很貴、且分布極不均」。Tesla（市值約 $1.5T、P/E >300）把 robotaxi 占企業價值多數的未來 price 進去——reverse 推導：若 robotaxi 真占其企業價值約 60%，等於市場已隱含「unsupervised 車隊在數年內規模化到數十萬台、單位經濟轉正、零重大退牌」這一連串里程碑全部兌現；現況全 Austin 約 20 台、Morgan Stanley 估年底約 1,000，落差就是這檔最大的下行槓桿（PT $415 vs ARK $4,600 的極端分歧本身即訊號）。Waymo 規模最實但曝險被 Alphabet 稀釋；中國三雄單位經濟領先但高 beta、政策敏感；賣鏟者（NVDA／Hesai）風報較均衡。現處 Phase II 早段（商業化驗證），轉空的條件 = 重大事故導致法規退牌 或 規模兌現連兩季落後敘事；de-rating window 在「市場從定價敘事改為定價兌現」那一刻。GM 為結構性退出者（特徵歸類為「燒錢谷淘汰」而非 play）。

The One Line

robotaxi 會不會贏不是分歧——Waymo 約 50 萬筆週訂單對上 Tesla 全 Austin 卻只有約 20 台車，真正的問題是「你買的是已驗證的規模，還是估值已透支的兌現」。

這份報告刻意不做的三件事

  1. 不告訴你哪一檔該買——CORE/EDGE 是「特徵的代表案例」，買賣由你的個股 DD 與 Pure MA 系統決定。TSLA「最大想像」不等於「最佳風報」，GOOG「規模最實」也不等於「最純曝險」。
  2. 不預測股價走勢——只給「進入哪個情境」的觸發訊號（Waymo 週訂單、Tesla 車隊台數、NHTSA 調查、中國單車經濟），不給點位。
  3. 不假裝知道純視覺方案會不會大規模驗證——Tesla unsupervised 能否在多城零事故規模化、重訂供給門檻，目前無定論，已誠實標為 2027 的裁決年觀察。
