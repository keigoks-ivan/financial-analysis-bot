II 初（特定 use case 商業化）
★ 本 ID 核心非共識 Thesis 市場用「通用 humanoid 普及就是大爆發」框架看人形機器人 — 認為 Tesla Optimus 量產 + Figure 03 起量 = sector 整體 ramp。本 ID 認為：① 2026-2028 是「特定 use case 商業化」階段（非通用機器人 / 非永遠 5 年後）—— Boston Dynamics Atlas 2026 Hyundai 部署 booked + Figure 03 BMW 已在線 + Tesla Optimus 「幾百台」內部 demo（仍 R&D）；② Unitree R1 $5,900 改變結構但不等於商業化（仍「玩具級」缺整合 AI）；③ 真正 alpha 在零件供應鏈（Harmonic Drive 50%+ 全球諧波減速機 / Nabtesco RV reducer 雙寡占 / NVDA Jetson — OEM 多未上市），非 OEM；④ Tesla Optimus 估值反映 narrative 而非 fundamentals — Gen 3 2026 Q1 仍「R&D and learning phase」，Boston Dynamics 已超車。
💡 §0 4 條核心 insight（預告）
1. Boston Dynamics Atlas 2026-01 CES 量產版發表 + Hyundai 部署 booked：是 humanoid 業界首個「production-ready + commercial deployment」，超越 Tesla Optimus（仍 R&D）。Hyundai (HMC) 是被忽略的 humanoid 主場 — 持有 Boston Dynamics 100%。
2. Figure 03 BMW 部署 + BotQ 12,000/yr 產能 + OpenAI partnership：是 humanoid OEM 第二贏家；2026-2027 加速；估值仍偏低（私募 $40B Series C 待）。
3. Tesla Optimus Gen 3 仍「R&D and learning phase」（Tesla Q1 26 confirm）：mass production 目標 2027，比共識估期晚 1-2 年；Boston Dynamics 已超車 18-24 個月。
4. 真正 alpha 是零件供應鏈雙寡占：Harmonic Drive (6324.T) 50%+ 全球諧波減速機 + Nabtesco (6268.T) RV 減速機；無論誰勝出 (Atlas / Optimus / Figure / 中國)，都會買這兩家的減速機。

  • NVDA 是 chip 還是 humanoid OS？：NVDA Isaac + GR00T + Jetson 是 humanoid AI 標準棧；雖然主業 chip，但 humanoid 領域是「OS 角色」 — 估值溢出 alpha。
  • Tesla 估值多少 humanoid 已 priced in？：Tesla market cap ~$1.3T，Optimus narrative 估占 $200-400B（基於 Cathie Wood / Musk 自稱「more valuable than Tesla auto」）；Gen 3 仍 R&D 意味 narrative 風險高。
  • Unitree R1 $5,900 是「玩具」還是「破壞性」？：價格擊穿但缺整合 AI / industrial-grade reliability；本 ID 視為「中國低端 cost competition」非 mainstream commercial。
  • Symbotic / SYM 是 humanoid 還是 mobile robot？：Symbotic 是 warehouse mobile robot（不是 humanoid），但與 humanoid 市場部分重疊（warehouse use case）— 列為對照。

結論

人形機器人進入 Phase IV（2026+ 特定 use case 商業化 — Boston Dynamics + Figure 領跑 / Tesla 落後）。核心非共識三條：（1）2026-2028 是 specific use case 階段非通用 humanoid；（2）HMC 是被忽略的 humanoid pure-play；（3）真正 alpha 是零件供應鏈（picks-and-shovels）非 OEM。

結構性受益層次：（A）OEM 領跑（HMC Boston Dynamics + Figure pre-IPO）；（B）零件雙寡占（Harmonic Drive + Nabtesco）；（C）AI 賦能（NVDA Jetson + GR00T）；（D）工業機器人 stable cash flow（FANUC / Yaskawa）。結構性壓力層次：Tesla Optimus narrative 風險 + 中端 OEM 被中國 cost competition 擠壓。
