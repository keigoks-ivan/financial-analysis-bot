II（暴力分化 + 樣板驗證）
★ 本 ID 核心非共識 Thesis 市場用兩種錯誤框架看 casual dining：① 「整段死亡」（CDR -0.9% → 全砍）；② 「等景氣回升」（macro 反彈大家都好）。本 ID 認為：① casual dining 不是死亡是暴力分化（TXRH +4.9% / EAT +14% vs Outback -1.5% / 10 件 PE-backed Ch.11）；② CMG/DPZ 1.0 翻身 playbook 在 2025 退潮（FY25 -1.7% / WING -3.3%）— 數位/loyalty/throughput 已成 table stakes，新 alpha 是「value architecture」（EAT $10.99 + Triple Dipper）；③ 下一波 turnaround 不是 brand 是 balance-sheet：PE-owned (RL/TGIF/Hooters) 的 Ch.11 重置 → 乾淨資產負債表 + 重議租約 = 投資 alpha（Adamolekun 後 RL 已從 $1B → $350M debt）；④ TXRH 不是 casual 是「prevention pattern」— 應與 best-in-class FC（CMG/SHAK）比 unit economics，不應與 BLMN/CBRL/CAKE 比 multiple（市場 mispricing）。
💡 §0 4 條核心 insight（預告）
1. CMG-Niccol 翻身樣板 2025 開始失靈：Niccol 走後 Boatwright 任內 -37% / FY25 -1.7% 首次負成長 — 顯示「digital + throughput + loyalty」已 table stakes，再造不出 alpha。
2. Brinker $10.99 + Triple Dipper 是新 turnaround 樣板，且可複製：Hochman（ex-KFC）做的是 menu cut 25% + 一次只一個 hero LTO + value architecture — 這比「digital 數位轉型」便宜 10 倍且見效快 12 個月。
3. TXRH 的 managing partner 10% equity 制度是 casual dining 真正的 moat：$25K 押金 + 5 年合約 + 10% 單店利潤分成 — 這制度從 1993 沿用至今，導致 TXRH 是 CDR 唯一 16 連年 SSS 正成長者。模仿成本 = 接受結構毛利 200bp lower（沒人模仿）。
4. PE-owned 是 implicit short signal：2014 RL → 2024 RL Ch.11；2017 TGIF 證券化 → 2024 TGIF Ch.11；2023 Hooters → 2025 Hooters Ch.11；BLMN 是下一個邊緣（2.3x lev + Starboard）。

  • CBRL 屬 casual 還是 family dining？：classification ambiguous（早午晚 + 零售 country store），本 ID 列為 casual 但標註「家庭式 mid-tier」。

💡 §1 Insight
1. casual dining 的死亡定義不能再用「table service vs counter」：分類已過時，實際應用「unit economics 是否能在 36% labor 下生存」。TXRH 能（because $8.7M AUV）；BLMN 不能（$3-3.5M AUV）。
2. 真正的「翻身樣本」必須跨 segment 學習：不能只看 casual 內部（多數已殘）。CMG/DPZ/WING 雖屬 FC/QSR，但 playbook 可移植；EAT 是唯一純 casual 樣板。

★ 歷史類比（3 個前例）

前例核心 pattern本次相似 / 差異
類比 #1：1990s 美國百貨 → 2010s 死亡（JCP / Sears / KMart）大量 PE LBO + REIT SLB + suburban mall 真實人流流失 = 全部 Ch.11；少數倖存者（Walmart / Target / Costco）反而擴大市佔高度相似：suburban casual dining 走百貨同樣路徑（Red Lobster / TGIF / Hooters / 各 PE-owned chains 死亡）；倖存者（TXRH / DRI / EAT）擴大市佔。差異：餐飲市場頻次比百貨高 20x，崩潰速度快但不會像百貨那樣連 Walmart 都被 Amazon 蠶食 — 餐飲現場體驗無法線上替代。
類比 #2：1980s GM / Chrysler 危機 → 1990s Toyota Lean 取代 Detroit美國老牌 incumbent 在勞動 / 工會 / 結構性成本固化 → 被「設計時就 lean」的新進入者（豐田）顛覆，而不是 incumbent 自我改造成功部分相似：CMG / DPZ / EAT 在公開市場的 turnaround 確實成功（Niccol / Doyle / Hochman），但 BLMN / CBRL 自我改造仍未驗證；新進入者（CAVA / SG / SHAK）「設計時就 lean」（22-28% RLM）但目前只取代 share，不取代 incumbent。差異：餐飲護城河比汽車弱（switching cost ~0），所以 K-shape 比汽車更暴力。
類比 #3：1990s-2000s Pizza Hut / Papa John's → DPZ 統治2009 之前 DPZ 是 pizza 三強最弱（comp 連年負）；用「公開承認 pizza 難吃」+ 數位（pizza tracker）+ fortressing 做出 28+ 季正 SSS streak；同期 Pizza Hut（YUM）SSS 連年負成長至今本 ID 直接引用：DPZ 是 §3 翻身六大模式 #1（Honest Admission）的源頭。當前對照組：BLMN / CBRL 至今未做「公開承認自己 brand 出問題」的動作（CBRL Masino 透過「transformation」反而觸發 Trump Jr. 反彈，更糟）。
💡 §2 Insight
1. 美國 casual dining 走百貨「死亡 → 倖存者擴大」的路：但比百貨快 5-10 倍（餐飲頻次高、switching cost 低）。倖存者（TXRH / DRI / EAT）獲取的市佔是「殘羹」而非「奪取」— 死亡 chain 的店面 50%+ 不會被任何餐飲業者接手，而是被改成住宅 / 醫療 / 倉儲。
2. 新進入者（FC）的 unit economics 顛覆，本身不會殺死 incumbent；殺死 incumbent 的是 PE 結構：類比 1980s Detroit — Lean 沒「殺」GM，工會 + pension 才殺的；casual dining 同理，CMG 沒殺 RL，SLB + 12 月 endless shrimp 才殺的。

哪個子技術成為 kingmaker？

2010-2018 期間 Pattern #2 + #4（throughput + digital）是真正 kingmaker — CMG / DPZ / WING 全靠這兩條獲得結構毛利擴張。但 2024-2026 起 Pattern #3 + #6 成為新 kingmaker：

  • Pattern #3（value architecture）成為 kingmaker 因為：① 通膨後消費者 sticker shock 嚴重（FAFH +31.9% cum）；② digital / loyalty 已商品化（每家都做不再是 alpha）；③ EAT 已驗證單一 LTO + 菜單瘦身 12 個月就能拉 +27% comp，且 capex 極低（~$50M behind-the-line）。
  • Pattern #6（balance-sheet reset）成為 kingmaker 因為：① ~10 件 PE-owned chains Ch.11 形成「試驗組」；② Adamolekun-led RL 是 first proof of concept（Adamolekun ex-P.F. Chang's 也是 turnaround 經驗）；③ 對於 SLB 結構性鎖死的 chain（RL / TGIF / Hooters），這是唯一能恢復 RLM 至 12-15% 的路徑。

💡 §3 Insight
1. Pattern #4（digital）退潮是 2025 最大 narrative 變化：CMG / WING FY25 首次負成長不是隨機 — 是「digital alpha 已 table stakes」的訊號。投資人不該再對 digital-led turnaround story 給 premium。
2. Pattern #3 + #6 是 2026-2028 alpha：EAT-style value architecture（看 Hochman 是否能 sustain Q3 FY25 +31% 進入 FY26）+ post-Ch.11 reset（看 Adamolekun-RL / Hooters franchisee group 是否能復原 RLM 至正）。誰能在這兩個 pattern 上執行 = 下一個 alpha 製造者。
3. Pattern #5 仍然是 prevention 而非 turnaround：TXRH manager equity 不可移植到已陷入危機的 chain（要 30 年文化建構）。但對於 fresh build / new concept，這是「永遠不需要 turnaround」的 cheapest moat。
