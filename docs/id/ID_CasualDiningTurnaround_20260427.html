關於「休閒正餐死亡」，市場最常見的偷懶看法有兩種：①「整段死亡」（CDR -0.9% → 全砍 underweight）；②「等景氣回升」（macro 反彈大家都好）。兩種都漏看了真正在發生的事——休閒正餐不是死亡是暴力分化：TXRH Q1'26 SSS +7.1%、DRI Q3FY26 +4.2%、EAT Q3FY26 +3.3%（雖從 +31% 高峰退坡），同期 BLMN US +0.9%、RL/TGIF/Hooters 接連 Ch.11 並持續關店（RL 2026 年 6 月關掉時代廣場旗艦店）。

更重要的非共識是翻身樣板的換代：CMG/DPZ/WING 的「數位 + throughput + loyalty」playbook 在 2025 退潮（CMG FY25 -1.7% 首次負、WING FY25 -3.3%、Q1'26 domestic -8.7%）——數位已成 table stakes，不再產 alpha。接棒的新樣板是 EAT 的「value architecture」（$10.99「3 for Me」+ Triple Dipper 占 16% 銷售 + 菜單瘦身），capex 比數位轉型便宜一個量級。第三條線是 balance-sheet reset——PE-owned 餐飲是 implicit short signal（2014 RL→2024 RL Ch.11 模式 100% 重複），買點在 post-Ch.11。

所以本報告不押「buy CDR」也不押「short CDR」，而是畫出誰擴大市佔、誰系統性死亡。結論：押 K 形贏家 TXRH/DRI/EAT（unit economics 已對齊 best-in-class FC），把 PE-owned（BLMN 是下一邊緣）當 short signal；EAT 是 CDR 內最大估值 mispricing。

PART I · DECISION SUMMARY

決策摘要層

三句話看完休閒正餐翻身的現狀、走向與行動，後接 PM 級監測點。這一層是報告的全部重點——下面各節只是把這一層的每個判斷補上證據。

現在 / NOW

休閒正餐 K 形暴力分化加速——TXRH Q1'26 SSS +7.1%（traffic +4.5%）、DRI Q3FY26 +4.2%（OG +3.2%/LH +7.2%）、EAT Q3FY26 +3.3%（從 Q3FY25 +31% 高峰退坡仍正）；同期 BLMN US +0.9%、CBRL 反例轉緩但體質弱、RL/TGIF/Hooters Ch.11 後持續關店。CMG Q1'26 回正 +0.5%、WING domestic -8.7%——數位翻身樣板退潮、value architecture 接棒。

未來 / NEXT

3–5 年休閒正餐萎縮至 ~$110B，3–5 家公開 chains（TXRH/DRI/EAT）主導；alpha 從數位/throughput（已 table stakes）移到 value architecture（EAT $10.99）與 balance-sheet reset（post-Ch.11）；PE-owned 結構性死亡延續、BLMN 是下一邊緣；GLP-1 + tipped-wage 廢除是長尾結構風險。

行動 / ACTION

押 K 形贏家 TXRH/DRI/EAT（unit economics 對齊 best-in-class FC）；EAT 是 CDR 最大估值 mispricing（~16x vs CMG ~30x）；把 PE-owned（BLMN）當 implicit short、買點在 post-Ch.11；非顯而易見二次受益 O（餐飲房東）/ TOST（POS）。買賣由個股 DD + 系統決定。

The Key Call · 本報告最重要的一個判斷

市場用「整段死亡」或「等景氣回升」看休閒正餐，真正的非共識是暴力分化 + 樣板換代——休閒正餐 segment 絕對衰退（CDR -0.9%），但內部 K 形：TXRH/DRI/EAT 擴大市佔，PE-owned chain 系統性死亡。同時翻身 alpha 從「數位 + throughput」（CMG FY25 -1.7%、WING -8.7% 已退潮、成 table stakes）換代到「value architecture」（EAT $10.99 + Triple Dipper 12 個月做出 +27% comp）與「balance-sheet reset」（post-Ch.11）。投資 thesis 不是 buy/short 整段 CDR，是 buy 4–5 winners + 把 PE-owned 當 implicit short。

三道裂縫具體是：①暴力分化——贏家 group（TXRH/DRI/EAT）市佔 from ~38% → ~50% of CDR by 2030，輸家（RL/TGIF/Hooters/Buca/Boston Market/Rubio's）腰斬消失；②樣板換代——數位/loyalty 已商品化（CMG FY25 首次負成長是訊號），新 alpha 是 value architecture（capex 便宜一個量級、見效快 12 個月）；③PE-owned 系統性 short——LBO debt + SLB rent overhang + 抽 dividend 不投 capex = 結構性 alpha 摧毀者，模式 100% 重複，BLMN 是下一邊緣。盯這三條，比盯「macro 何時回升」更能看清錢的流向。

PART II · HISTORICAL ANALOGS

百貨死亡與 Domino's 翻身，框住多空

歷史不必重演，但兩個對照各給一個教訓：1990s-2010s 百貨給出「PE+REIT SLB 結構殺死 incumbent、倖存者擴大市佔」，Domino's 2009 給出「公開承認失敗 + 重做產品的翻身可以量化」。

類比一：1990s-2010s 美國百貨死亡——倖存者擴大市佔。JCPenney / Sears / KMart 走的路徑是：大量 PE LBO + REIT sale-leaseback（SLB）+ suburban mall 人流真實流失 = 全部 Ch.11；少數倖存者（Walmart / Target / Costco）反而擴大市佔。量化錨：百貨占美國零售營收份額從 1990 年代 peak ~10% 跌到 2020 年代 ~2-3%，但 Walmart/Costco 同期市佔翻倍。對本次含義：suburban 休閒正餐走同樣路徑——RL/TGIF/Hooters/各 PE-owned chain 死亡，TXRH/DRI/EAT 擴大市佔（贏家 group from ~38% → ~50% within CDR）。差異：餐飲頻次比百貨高約 20 倍，崩潰速度更快，但現場用餐體驗無法被線上替代，所以倖存者不會像百貨那樣連 Walmart 都被 Amazon 蠶食。

怎麼讀：休閒正餐同時是「百貨死亡」（PE-owned 走 SLB+LBO 的 incumbent 死亡路徑）與「Domino's 翻身」（贏家可用公開承認 + 重做產品翻身）。bull 的底是握 unit economics 的倖存者 + 能執行 value architecture 者，bear 是 PE-owned 結構性死亡——RL/TGIF/Hooters 的 Ch.11 已是早期證據。

類比二：Domino's 2009「honest admission」翻身——翻身可以量化。2009 之前 DPZ 是 pizza 三強最弱（comp 連年負）；CEO 用 4 分鐘紀錄片直播焦點團體罵 pizza「像紙箱」「醬像番茄醬」+ 全面 reformulation，2010 Q1 US comp 跳 +14.3%（QSR 史上最大跳升之一），接著 2010-2019 做出 28+ 季連續正 SSS streak。量化錨：同期 Pizza Hut（YUM）沒做翻身、SSS 連年負成長至今。教訓：翻身的核心不是「等景氣」而是「主動重做產品 + 公開承認失敗」——這正是 EAT $10.99 + 菜單瘦身 25% 的源頭（FY25 +14%、Q3 +31%），也是 BLMN/CBRL 至今未做的動作（CBRL Masino「transformation」反觸發 logo 醜聞 + Trump Jr. 反彈，更糟）。判別訊號：管理層是否敢公開承認 brand 出問題 + 重做產品，而非漲價硬撐。

PART III · SUPPLY SIDE

供給側：餐飲價值鏈，利潤被上下游 squeeze

反直覺但可操作的結論：餐廳業者（中游）是價值鏈利潤 squeeze 點——上游食材通膨 + 下游房東/平台/SaaS 兩面抽走利潤，只有 unit economics 結構最強者能在 squeeze 中保 RLM ≥ 15%。

供給端可拆三段。上游（食材 / 設備）：Sysco/US Foods/PFG 分銷 + Middleby 設備，毛利 17-35%、議價權中；chain 集中度上升反而壓縮分銷商毛利（SYY EBITDA margin 已從 5.5% 壓至 ~4.0%）。中游（餐廳營運）：勞動 30-36% + 食材 28-32% + 租金 6-10%，RLM 12-28%（K 形）、議價權低（被房東 + 勞動 + 食材三面夾擊）。下游（房東 / 平台 / SaaS）：Realty Income（O）等 REIT lease 鎖 10-20 年 + escalator、毛利 95% NOI；DoorDash/Uber Eats take rate 20-30%；Toast/Olo POS switching cost 極高、ARR +30-40%/年。利潤往「下游 + 上游」兩端集中，中游餐廳是被 squeeze 的一段。

怎麼讀：下游（房東/平台/SaaS）議價權永遠最高、上游分銷次之，中游餐廳是 squeeze 點。K 形贏家（TXRH $8.4M AUV / EAT value architecture）在 ↑、輸家在 ↓。這就是為何分化暴力——只有 unit economics 結構性最強者能在三面夾擊中保 RLM ≥ 15%。

Inference · 供給
中游餐廳被上下游 squeeze，只有結構最強者能在 K 形上半生存
前提：上游食材通膨（牛肉 +14.7% YoY）+ 下游房東 lease 鎖死 + 平台 take rate 20-30% + 勞動 36.5% → 中游餐廳 RLM 被兩面抽走 → 只有 $8M+ AUV 或 value architecture 能保 RLM ≥ 15%

當上游通膨抽走毛利、下游房東/平台/SaaS 抽走利潤，中游餐廳只有兩條生路：①結構性高 AUV（TXRH $8.4M，攤薄固定成本）；②value architecture（EAT 用 traffic +20% 補回 mix-down）。其餘 mid-tier chains（BLMN/CBRL）AUV $3-3.5M + 漲價彈性已飽和 → RLM 壓至個位數 → 走向 Ch.11。TXRH Q1'26 RLM 16.3%（-36bp，牛肉壓力但仍守 16%+）vs BLMN EBITDA ~9% 衰退中，是 squeeze 結構的硬證據。

可證偽條件：若中游餐廳取得議價權（如自有 delivery 普及繞過平台、或 tipped wage 維持不變壓低勞動），則「中游被 squeeze」論部分鬆動。但 25 年趨勢未見逆轉。

非顯而易見的供給含義：休閒正餐 chain 死亡時，下游受益者（O / TOST）的暴險方向不對稱。Realty Income（O）持有大量 RL/TGIF SLB 資產——chain Ch.11 時 lease 是 senior 債權，O 仍收 100% 租金（即使 box 被 reject 也能 6-12 個月內 re-tenant 給 dollar store / urgent care / 物流）。SLB 對 PE-owned chain 是 short signal，對 REIT 是「永久 cash cow」signal。反向，Toast（TOST）ARR 與 chain 數量連動——只有 K 形贏家擴店才是真 alpha，chain 死亡時 TOST 也失客。這是「同一個 SLB 結構，對 PE-owned 是死亡、對房東是現金牛」的二次受益判讀。

PART IV · DEMAND SIDE

需求側：K 形分化與翻身樣板換代

需求最剛性的是「消費者要可感知的 value——但休閒正餐 ASP 已 +27% cum（sticker shock），漲價 leverage 結束」。下一輪 alpha 必須來自 traffic 而非 check，這正是 EAT 突出、CMG/WING 退潮的根本原因。

第一引擎：K 形分化（現需）。休閒正餐 segment 絕對衰退（CDR -0.9% 2024），但內部贏家擴大市佔——TXRH Q1'26 SSS +7.1%（traffic +4.5%）、DRI Q3FY26 +4.2%（OG +3.2%/LH +7.2%）、EAT Q3FY26 +3.3%；同期 BLMN US +0.9%、PE-owned chain 接連 Ch.11。輸家釋放的 ~$10B sales 不會留在 CDR，而流向 FC（CMG/CAVA/SHAK）+ 自家煮 + delivery（DPZ）。第二引擎：翻身樣板換代（現需）。CMG/DPZ/WING 的「數位 + throughput + loyalty」playbook 在 2025 退潮——CMG FY25 -1.7% 首次負（Q1'26 才回正 +0.5%）、WING FY25 -3.3% 且 Q1'26 domestic -8.7%、DPZ Q1'26 US +0.9%（低於 3% 內部目標）。數位已成 table stakes，新樣板是 EAT 的 value architecture（$10.99 + Triple Dipper 占 16% 銷售 + 菜單瘦身 25%），用 traffic 而非漲價驅動。

怎麼讀：贏家持續正成長 + 數位樣板退潮 + PE-owned 系統性死亡——三者一致指向「buy 4-5 winners + 把 PE-owned 當 short、樣板從數位換到 value architecture」。採信「贏家公司級 SSS + PE-owned Ch.11 速度」當大分流的最硬證據，禁用「餐飲很慘 / 餐飲要回升」這種模糊詞。

多空為何都對

空頭測休閒正餐 segment（CDR -0.9% 絕對衰退、PE-owned Ch.11），這幾乎沒爭議；多頭測誰倖存擴大（TXRH/DRI/EAT unit economics 結構強）。兩者可同時為真：segment 確實絕對衰退（空頭對），但 K 形贏家市佔擴大、value architecture 翻身可複製（多頭對）。本報告把重心放在「誰擴大市佔、誰系統性死亡」，而非賭 macro 何時回升。

PART V · VERDICT & SCENARIOS

裁決與三情境（24–36M）

「休閒正餐 K 形分化」判 INTACT，「value architecture 是新樣板」判 INTACT，但「EAT 翻身是 structural 還是 lap math」判 AT_RISK（Q3FY26 已退坡至 +3.3%）——故結構看「buy 4-5 winners + short PE-owned」、conviction 卡 mid-high。如硬要對應傳統框架：對「能保 RLM ≥ 15% 的休閒正餐供給」需求遠大於合格供給（要 $8M AUV 或 value architecture），等價於結構性短缺型，稀缺來自 unit economics 門檻而非店面數量。

PART VI · TAXONOMY

個股特徵與 conviction tier

本框架只輸出「特徵」與「分類」，買哪一檔由個股 DD 系統與 Pure MA 訊號決定。以下 tier 為產業推論的特徵歸類，提及公司僅為「該特徵的代表案例」，不附帶買賣建議。

CORE 🔴

TXRH / EAT / DRI（K 形贏家）

unit economics 對齊 best-in-class FC——TXRH（Q1'26 +7.1%/traffic +4.5%/$8.4M AUV/managing partner 10% equity，prevention pattern）、EAT（Brinker/Chili's，FY25 +14%/Q3 +31% 已驗證，Q3FY26 +3.3% 退坡仍正，value architecture 新樣板）、DRI（Q3FY26 +4.2%/OG +3.2%/LH +7.2%/EBITDA 16%+，多品牌 capital allocation）。特徵是「K 形上半倖存擴大」最高純度贏家。conviction：高（TXRH/DRI）/ 中高（EAT，退坡軌跡）
CORE 🔴

CMG / DPZ / WING（翻身教科書，樣板退潮中）

數位/throughput 翻身樣板——CMG（2015 食安→Niccol playbook，FY25 -1.7% 首次負後 Q1'26 回正 +0.5%）、DPZ（2009 教科書翻身，Q1'26 US +0.9% 低於 3% 目標）、WING（20+ 年連勝後 FY25 -3.3%、Q1'26 domestic -8.7%）。特徵是「數位 alpha 已 table stakes、需重新加速證明」。WING 列受害（連續負成長 + 高估值）。conviction：中（樣板退潮、需 catalyst 驗證）
EDGE 🟡 / LOSER 🟢

DRI*/BLMN/CBRL/CAKE/CAVA/SHAK/BROS ｜ FWRG/SG/KRUS/O/TOST

🟡 受益/掙扎混合：CAVA（FC Mediterranean Q1'26 +9.7% 再加速、~150x 估值風險）、SHAK（drive-thru 推進）、BROS（drinks-led +5.6%）。🟡 待翻身/反例：BLMN（Q1'26 flat、Starboard ~10%、PE-owned 邊緣）、CBRL（Q3FY26 beat + EBITDA 指引 $120-125M，反例轉緩但體質弱）、CAKE（steady mediocrity、hidden Brinker candidate）。🟢 非顯而易見二次受益：O（餐飲房東、SLB cash cow）、TOST（POS、K 形贏家擴店連動）、KRUS（never-need-turnaround 對照組）；🟢 受害：SG（結構性掙扎）。conviction：中（受益方）/ 偏空（BLMN/PE-owned）

估值面總結：本主題的核心張力在「休閒正餐 segment 絕對衰退 vs K 形贏家擴大市佔」。最高純度贏家是 TXRH/DRI/EAT（unit economics 對齊 best FC）；EAT 是 CDR 內最大估值 mispricing（~16x Fwd P/E vs CMG ~30x、TXRH ~24x，但 comp 強度 ≥ CMG）；結構性輸家是 PE-owned（BLMN 是下一邊緣、買點在 post-Ch.11）。現處 Phase II（暴力分化 + 樣板換代），轉空條件 = EAT FY26 comp ≤ +5%（翻身被認定 lap math）或 TXRH 出現負季（prevention pattern 失靈）；de-rating 風險主要在高估值 chains（WING ~50x / CAVA ~80x）與 PE-owned（BLMN）。買賣由個股 DD + Pure MA 系統決定。

The One Line

休閒正餐不是死亡、是暴力分化——買 4-5 個 unit economics 倖存者（TXRH/DRI/EAT），把 PE-owned 當 implicit short；翻身 alpha 從數位換到 value architecture。

這份報告刻意不做的三件事

  1. 不說「整段死亡 → 全砍」——明確畫出 K 形：segment 絕對衰退，但贏家擴大市佔。
  2. 不把 PE-owned cheap stock 當「翻身機會」——標出 PE 結構（LBO+SLB）是 implicit short signal、買點在 post-Ch.11。
  3. 不告訴你哪一檔該買——CORE/EDGE/LOSER 是「特徵的代表案例」，買賣由你的個股 DD 與 Pure MA 系統決定。
