§1 產業定義與邊界

「AI 資料儲存」這個主題在 2025 年之前並不是一個明確的子產業 — 過去儲存系統與 AI 的接點只是「企業伺服器底下的 SSD」。但 2024-2026 這波 AI 浪潮把儲存推到台前，並把它切成五個彼此獨立的子市場：① 企業 NVMe SSD（資料湖底層）；② AI-optimized storage 平台（Pure Storage / NetApp）；③ AI-native parallel file system（VAST / WEKA / Hammerspace）；④ Storage-Class Memory（Z-NAND / XL-Flash 用於 KV cache）；⑤ NAND/3D NAND wafer 製造端。每個子市場有獨立的 unit economics、客戶結構、競爭強度，不能合併分析。

本 ID 的核心觀察是：「企業 SSD」與「消費 SSD」在 2025 之後正式分裂為兩個獨立市場。過去 NAND 廠（Samsung / SK hynix / Micron / Kioxia / SanDisk）依產品線比例分配 wafer，價格傳導同步；但 AI 出現後，wafer 大幅向「8TB+ 企業 QLC」傾斜，企業 SSD ASP 結構性高於消費 SSD（每 GB price 約 1.3-1.5X premium），且短缺優先排擠消費端 — 這個動態在 §4 TAM 與 §7 ASP 章會具體量化。

1. 「企業 SSD」與「消費 SSD」分裂為兩個獨立市場是 2025-2026 結構性轉變：分析 SanDisk / Samsung NAND 業務必須拆段看；DC SSD GM 65-67%（SanDisk Q3 FY26 guide）vs 消費 SSD GM 30-40%。財報整體加總會掩蓋真相。

3. SCM（Z-NAND / XL-Flash）是 LLM 推論時代被低估的新 segment（但商業化非必然）：NVDA ICMS 平台（CES 2026）暗示 KV cache offloading 將是 inference TCO 改善的核心 — 2027-2028 SCM market 可能從現在 ~$1B 增長至 $3-7B（base case；下修自原 $5-10B）。需注意兩個歷史/結構性風險：① Intel 2022 終止 Optane（3D XPoint）已證明 SCM 在「DRAM 太貴 / NAND 太慢」夾縫的商業脆弱性；② CXL 2.0/3.0 DRAM 池化（Samsung 1PB CXL module、各家 CXL switch）是 KV cache 的另一條替代路徑。Bear case 若 CXL DRAM 普及則 SCM 成長落在 $1-2B 區間。

§2 技術演進 + S 曲線 + 歷史類比

儲存產業的技術演進有兩個重要規律：第一，NAND 製程從「每代縮 die size + 加 layer」轉為「每代換 cell type + 加 die」 — 從 2D NAND 到 3D NAND（layer 64 → 200+），再到 QLC（4-bit per cell）+ 多 die stacking，每一代的容量倍增方式不同；第二，儲存層級從單一 NAND tier 演化為「DRAM + SCM + NAND + HDD」四層階層，AI 推論的 KV cache 把 SCM 從 niche 變成必選層級。這兩條規律疊加，造就了 2024-2026 企業 SSD 容量單顆 30TB → 245TB（8X）的非線性曲線。

💡 Insight §2
1. 「企業 NVMe」進入結構性成長軌道（CAGR 27%+），與「消費 NAND」的 cyclical 完全脫鉤：Samsung / SK 財報必須拆段才能看到真相 — DC SSD GM 60%+，消費 SSD GM 30-40%。
2. AI-native storage 估值結構正在 SaaS 化：VAST $30B 估值（cash-flow positive、tripling revenue）走的是 SaaS PE 30-40x 路徑，而非傳統儲存 OEM 的 cyclical PE 12-15x — 這個估值結構轉換是 AI 儲存最大 alpha。
3. 反類比警示：Samsung HBM +47% 擴產若引發 NAND wafer 紀律失守會是 2027-2028 trap：過去 NAND 廠在 supply-tight 後 18-24 個月通常陷入「FOMO 擴產」陷阱；本次必須監控 capex 紀律是否持續。

★ Kingmaker 子技術判斷

從「未來 5 年最稀缺、最難 displace 的子技術」角度：

  • 第一名：NAND wafer 製造（Samsung / SK / Kioxia / SanDisk / Micron 五寡占） — 受限於 wafer fab 折舊週期 + capex 紀律 + 集團層級 HBM/NAND capex 博弈（注：HBM 用 DRAM 廠，與 NAND fab 不互換實體 wafer，但共享集團資本支出預算）；五家集中度 90%+，是整個 AI 儲存堆疊的根基稀缺資源。
  • 第二名：AI-native 平台軟體（VAST / WEKA / Pure / NetApp） — 客戶切換成本極高（資料遷移 + 軟體 lock-in + GPU 整合 NRE）；護城河是「客戶 customer-side capex 投入」。
  • 第三名：SCM 製程（Samsung Z-NAND） — 過去 8 年 niche 但 know-how 累積深厚；KV cache offloading 浪潮使這個積累一夕成為 ATE 鏈的「下一個 leg」。

1. §5 供應鏈 Value Chain

AI 儲存的價值鏈與一般半導體鏈不同 — 它從 NAND wafer（最稀缺資源）到 AI-native 平台軟體，跨越「製造設備」、「半導體」、「儲存系統」、「軟體」四大產業類別。利潤池的轉移正在從 NAND wafer 端向兩個端點同時擴散：上游（製程設備）+ 下游（AI-native 平台軟體）— 這是 ATE 鏈的反鏡。

  • 最強：AI-native 平台軟體（VAST / WEKA）— 客戶切換成本極高（資料遷移 + GPU 整合 NRE + 軟體 lock-in）；SaaS 級毛利 60-75%；SaaS 級估值（VAST 50x P/Sales）。

  • 弱：消費 SSD / commodity NAND — 持續面臨 commoditization 壓力；ASP 仍受 cyclical 壓力。
  • 最弱：傳統 HDD（被 QLC 大容量 SSD 取代）— 結構性下行 segment。
