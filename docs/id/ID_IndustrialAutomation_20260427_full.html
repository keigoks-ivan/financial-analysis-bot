§1 產業白話定義 + 歷史脈絡

工自賣的不是「一台更快的機器」，而是「讓一整條產線自己跑、自己量測、自己修正」的控制大腦與其上的軟體。付錢的是製造業的工廠主與工程部門，而他們最痛的不是某個元件不夠強，是「換一個控制器等於整線停產 + 重新驗證」的高轉換成本——這正是寡占能維持數十年的根。

白話開場：傳統製造要靠大量人力與固定機械；工業自動化就是把「控制決策」交給機器——用 PLC（可程式邏輯控制器，工廠的 brain）寫好離散製造的動作邏輯，用 DCS（分散式控制系統）管煉油廠、化工廠這類連續流程，用 servo/drive 精準驅動馬達與機器人，再用工業軟體（MES/SCADA/copilot）把整廠資料串起來做排程與優化。所謂這幾年的「轉型」，核心是兩件事：一是把一次性的「賣硬體」生意，加上經常性的「賣軟體訂閱（ARR）」；二是把 AI copilot 放進工程師的工作流。

邊界界定：本報告聚焦「跨多檔個股共用的工自 prime + 其供應鏈與挑戰者」這條主軸。納入離散自動化（PLC/HMI/motion）、過程自動化（DCS/SCADA）、servo/drive、工業機器人 OEM、工業軟體 ARR 的上市廠商；不深入純人形機器人（那是姊妹 ID_HumanoidIndustrialRobotics）、不展開純重機械（ID_HeavyMachineryMining）、不展開純資料中心基建（ID_AIDataCenter，但其 DC 自動化需求是本 ID 的順風來源）。灰色地帶在「Honeywell」與「Schneider」——HON 6/29 分拆後留下的才是純 automation；Schneider 雖有 EcoStruxure 工自，但主場其實在 DC power，這個雙重身份本身就是 §7 的 insight 來源。

歷史脈絡分四代。第一代：1970–1990——PLC 取代繼電器，離散製造自動化興起，Allen-Bradley（後被 Rockwell 收）、Siemens、三菱建立離散主場。第二代：1990–2010——DCS/SCADA 讓過程工業數位化，Honeywell PMT、Emerson 建立過程主場；同期日本機器人（FANUC、Yaskawa）取得全球 dominance。第三代：2010–2020——Industry 4.0 + IoT + 雲端，Siemens Digital Industries、Rockwell FactoryTalk、Schneider EcoStruxure 把軟體放進來、開始 ARR 化。第四代：2022+——Industrial AI copilot + 中國平價化 + 結構性分拆（HON）三股力量重塑競爭格局。

最硬的結構事件：2025–2026 Honeywell 三段分拆。Honeywell 先於 2025-10-30 分拆 Solstice Advanced Materials（材料，Nasdaq: SOLS），再於 2026-06-29 完成 Aerospace（HONA）分拆（1 HONA : 2 HON、tax-free），留下 Building / Industrial / Process Automation 三段的純 automation 主體。這是 sector 數年來最大的結構動作——它不靠營運改善，而靠拆解 conglomerate 把多年折價一次釋放。這也是為什麼本 ID 把 HON automation 段升為 Tier 1：分拆後它從「大到看不懂的工業集團」變成可被同業 automation 倍數重估的純 play。

歷史類比：GE 三段分拆（2021–24）。當年 GE 被當 conglomerate 折價，拆出 GE Aerospace 後純航太段約 +200%，conglomerate discount 被結構釋放。本次的量化差異是 HON 起始估值較高（fwd P/E ~21x）、體質本就乾淨，所以「可釋放的折價」明顯小於 GE——HON 分拆是「中等 upside、高確定性」事件，不是 GE 級暴漲。多頭錯在「把 HON 當下一個 GE +200%」，空頭錯在「以為分拆沒用」——正解在中間：折價會釋放，但幅度有限。

工自沒有半導體那種可統計的價格 cycle，改用「自動化滲透率 + 軟體 ARR 占比」的擴散曲線替代——看結構遷移到哪。

關於 cycle 統計表的省略：本產業沒有記憶體/面板那種 ASP peak-to-trough 的供需價格 cycle（毛利結構穩定、靠黏性而非單位成本競爭），故不立歷史 ASP cycle 表；改以 §2 的自動化滲透 + 軟體 ARR S 曲線呈現結構遷移階段。可參考的時序錨點是：軟體 ARR 化的兌現是「年」不是「季」——ROK 的 ARR 從第三代起步到 2026 仍只 +6%、占比中個位數，這個傳導時滯約 5–10 年，比敘事慢。

這段歷史的可操作含義是：真正該怕的不是「工自股太貴」這種市場情緒，而是「把三條口徑不同的結構線混成一句工自漲/跌」。分拆是事件（看確定性與幅度）、軟體 ARR 是年級遷移（看占比與增速）、DC 自動化是當下訂單（看週期）——分開計，才不會被「ROK 執行強但 ARR 只 +6%」這種口徑落差誤導。

§2 技術成熟度 + S 曲線

為什麼軟體 ARR 化與 AI copilot 是現在發生、不是 2015 年？因為兩個技術 bottleneck 剛被解開——「邊緣 + 雲端的工業資料層」讓裝機基數第一次能持續產生可訂閱的數據服務，「工業大語言模型 copilot」讓工程知識能被自動化，平台價值才從 PPT 變成可量化 ROI。

過去「工業軟體」多半是賣斷的工程工具；真正的技術拐點是工業資料層 + 邊緣運算：當 PLC/DCS 的遙測能持續上雲、跨產線做預測維護與優化，廠商才能把「賣硬體」變成「賣裝機基數上的訂閱」——這是 Rockwell FactoryTalk + Plex/Fiix、Siemens Xcelerator、Schneider EcoStruxure 的底層邏輯。第二個拐點是工業 copilot：把工程師的設計、除錯、維護知識用 LLM 自動化——Siemens 已推出 9 個 Industrial Copilot、與 NVIDIA 共建 AI 控制產線，這讓平台價值從「省人力」升級到「省工程 know-how 累積時間」。

把「自動化滲透 + 軟體 ARR 占比」畫成 S 曲線——看現在在採用曲線的哪一段，決定是早布局還是追高。

§4 需求側：現在與未來 + 三角驗證

需求最不可替代的是「工廠一旦自動化就回不去」——勞動成本與良率壓力讓滲透率單向上行；但最大的誤判是把「data center / 電網投資熱」當成「純廠房自動化也熱」，這兩者目前是雙速。

現需結構：採購主體是製造業工廠主與工程部門，需求由三個引擎推動但強弱差異極大。①data center / 電網自動化（最熱）——AI 資料中心的供電、散熱、配電需要大量電氣化與過程控制，ABB 訂單 +24%、Schneider 能源管理 +12.8%、Emerson Ovation（電廠/utility 控制）訂單 +74%，這是當下真正在燒的需求。②reshoring 蓋廠（溫吞）——美國製造業建廠 capex 表面看好，但扣掉半導體 fab（自 2024 年中 -44%）後其餘製造業建廠只小幅成長（~+5.6%），「工廠蓋好才買自動化」的傳導比敘事慢。③AI-in-factory（早期）——製造業用 AI 的比例從 2023/9 的 1.8% 升到 2026/2 的 13.9%（四倍），但 ~87% 工廠還沒導入、只有 ~20% 自認準備好。

§7 Non-Consensus + Priced-in 檢驗 + Kill Scenario

⚠️ 本章含 judgment：每條分歧附 [F:] cornerstone + [X:] 三情境 + priced-in 檢驗。
市場主流對工自的三個共識：①「HON 是穩定 conglomerate，分拆 nice-to-have」；②「ROK 軟體 ARR 化值得追高」；③「中國機器人全面取代日本」。三個我們都部分不同意——而分歧能不能賺錢，取決於市場 price 了多少。
🟡分歧 1：「HON 是穩定 conglomerate，分拆是 nice-to-have」信心：高
  • 共識說 X：HON 是 stable 工業集團，分拆只是錦上添花、非 transformational。🟢 [F: T1 — HON 6/29 完成 Aerospace（HONA）分拆、tax-free、留三段純 automation]。

  • Priced-in：HON fwd P/E ~21x、2026 年至今股價約 +10%，市場已部分 price 分拆預期，但「純 automation 段被重估到同業倍數」這一步尚未完全 price——所以分歧在「分拆無用」是錯的，但 upside 幅度有限（不是 GE +200%）。reverse 推導：若 automation 段重估到 ~26–28x（ROK ~36x 的 70–80%），對應的 upside 約中個位數到雙位數，與「事件型、中等幅度」一致。
→ 推論邏輯：分拆釋放折價、確定性高但幅度有限。⚠ 證偽：若 6/29 後留存 HON automation 倍數未往同業靠攏（停在集團折價），分拆 catalyst 已被 price、分歧 1 失效。
🟡分歧 2：「ROK 軟體 ARR 化值得追高」信心：中
  • 共識說 X：ROK 軟體 ARR 化 + 美國 reshoring 純度高，~36x 合理、值得追。🟢 [F: T1 — ROK Q2 sales +12%、adj EPS +32%、上修 guide；ARR +6%]。
  • 我們說 Y：方向對（軟體 ARR 化是真 re-rate driver），但口徑與現價都要打折——ROK ARR 只 +6%、占比中個位數，「sales +12%」含併購/匯率（有機 +9%），而 ~36x 已把「ARR 達標 + reshoring 完美兌現」price 進去，容錯薄。🔵 [X: base 很可能 — ARR 緩升、報酬靠 EPS 兌現非 re-rate；bull 可能 — ARR 加速 + SaaS 溢價；bear 可能 — 製造遞延 + de-rate]。
  • Priced-in：ROK ~36x（約工業中位數 21x 的 1.7 倍、GuruFocus 標「significantly overvalued」、GF Value 隱含逾 40% 下行）已 price 樂觀情境，沒有 price「製造遞延 + ARR 停滯」——所以這條分歧若押「ROK 質佳」是對的，但押「現價追高有 alpha」則分歧已 priced、不可操作。正解是「等回調」而非追 52 週高。
→ 推論邏輯：把「ROK 好不好」拆成「質佳但現價貴」，等回調而非追高。⚠ 證偽：若 ARR 增速重新加速至雙位數 + 製造業建廠 capex 轉強，現價樂觀情境兌現、追高論成立。
🟡分歧 3：「中國機器人全面取代日本」信心：高
  • 共識說 X：中國 servo/機器人全面取代日本是 multi-year 結構問題、FANUC/Yaskawa 全線受壓。🟢 [F: T2 — 中國國產機器人 2025 占中國市場 >51%、Inovance 出貨 >5M 關節伺服]。
  • 我們說 Y：中國平價化擠的是中段（mass-market servo / 通用機器人），不是高階（CNC / 高精度 / 半導體 fab automation）；高階仍由日系/歐美主導，門檻是軟體 + 高精度 + 安全認證。🔵 [X: base 很可能 — 高階日歐美守、中段被中國取；bull（對日系）不太可能 — 中國卡在中段；bear（對日系）可能 — 中國突破高階]。
  • Priced-in：FANUC/Yaskawa 估值已 price「中段壓力」（日系機器人 multiple 多年壓縮），但沒有 price「中國突破高階」——所以押「FANUC/Yaskawa 中段受壓」是對的且已部分 priced（不可操作），真正的非共識是「高階守得住」這個區分：避中段曝險、別把「FANUC 是機器人 #1」當全面護城河。
→ 推論邏輯：結構性 split——高階日歐美守、中段被中國取；定位是「避中段、保高階 + 軟體 ARR」。⚠ 證偽：若中國國產高階機器人 share +10pp 或突破美國高階 chip 管制，高階護城河鬆動、日系全線受壓成立。

Steel-man 反方（最有力的 3 條）

  • 反方 1（製造衰退）：2026–2027 全球 GDP 衰退 → 製造業 capex -10% → ROK/SIE/EMR -15–20%、FANUC/Yaskawa -25%。當前證據：美國 ISM 仍在 50 附近、歐洲製造 PMI 偏弱、中國 mixed。證偽窗口：全球 GDP 維持 2%+ 且 ISM 持續 50+。
  • 反方 2（DC capex 消化 + reshoring 落空）：AI 資料中心 capex 進入消化期、DC 自動化訂單見頂，而 reshoring 接棒落空（建廠 capex 扣半導體後溫吞）→ 當下唯一在燒的引擎熄火、無接棒者。當前證據：電子/半導體 fab capex 自 2024 年中 -44%。證偽窗口：製造業建廠 capex（扣半導體）連兩季加速接棒。
  • 反方 3（中國蔓延高階）：2028+ Inovance/Estun 從中段升至高階 → SIE/ROK/FANUC 高階段也受擠、prime margin -3–5pp。當前證據：中國高階 chip 受美國管制（上限），Inovance 仍以中段為主。證偽窗口：中國 chip 管制持續 + prime 軟體 ARR 結構穩固。

市場最低估的風險：不是製造衰退（人人都知道是 cyclical risk）、也不是中國中段（已被廣泛討論），而是反方 2 的「DC 見頂但 reshoring 沒接棒」——市場把 DC 自動化熱當成「純工自全面復甦」的訊號，最沒準備好「DC 是當下唯一引擎、純廠房 reshoring 兌現慢」這個雙速真相。這是本報告認為最該獨立監測的尾部風險。

三條分歧裡最可操作的是分歧 1（HON 分拆釋放折價）——它有明確的事件（6/29）與領先指標（留存 HON automation 倍數）。分歧 2（ROK 等回調）次之、部分 priced。分歧 3（中國只擠中段）方向對但在日系估值上已部分 priced，操作價值在「避中段曝險」而非「做空日系」。
