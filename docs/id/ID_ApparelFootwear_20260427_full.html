§1 產業白話定義 + 歷史脈絡

白話版：服飾這門生意過去靠「品牌做爆款 + 多元化分散風險」當護城河。現在消費者已分叉成高低兩端——高端買 elevation 與 brand heat、低端買 value，中段被三面夾擊。結果不是大家一起好或一起壞，而是錢往「品牌動能稀缺的兩端」集中。

白話開場：先把「服飾」這個詞拆開。市場習慣把它當成一個 sector（一支 XRT ETF 蓋過去），但它至少是四個結構性差異的子象限——Athletic / Performance（NKE / LULU / ONON / DECK / UAA）、Specialty Retail / Mall-based（ANF / GAP / AEO / URBN）、Heritage / Wardrobe Staples（LEVI / VFC / PVH）、Premium / Affordable Luxury（RL / GOOS / ROOT）。本報告問的是：當消費者分叉成 K 形，這四個象限的錢被重分配給了誰、為什麼這條分化不會收斂。

邊界界定：本報告聚焦「affordable luxury（< $1000 ASP）以下的品牌服飾」。納入上述 15 檔；不納入 Hard Luxury（Hermès / LV，見 GlobalLuxury）、運動通路（DICK's / Foot Locker，§5 提及但非主角）、Beauty、Off-price（TJX / ROST）、Department Store、Fast Fashion（H&M / Zara / Shein / Temu，§9 風險變數提及）。灰色地帶：HOKA / On 已從 pure athletic 跨入 lifestyle（HOKA 60-70% 賣給非競技跑者、ON Apparel 進 lifestyle），但本 ID 仍歸 Athletic（品牌 origin + 通路）；Vuori（$5.5B IPO filing）/ Alo（$10B 私募）無 public ticker、列 §9 風險章重點討論。

歷史分四段。第一段：1990–2005 傳統 mall + dept store 主導——Tommy Hilfiger 1997 IPO 暴起、ANF 1996-2005 mall king。第二段：2005–2015 Athleisure 1.0 + Fast Fashion——LULU 2007 IPO + UAA 2005 IPO peak（UAA $52 in 2015）+ H&M / Zara。第三段：2015–2022 DTC 革命 + Athleisure 2.0——Nike DTC 推到 44%、LULU $10B、Hoka / On 從利基到主流、中國消費爆發（NKE FY22 $46.7B 高點）。第四段：2023–2028E K-shape mature + Tariff Reset——athleisure 滲透率達 mature、新進者持續搶份額、China local-for-local、Trump tariff 2.0（後被 2026-02 SCOTUS 推翻為 10% 統一稅）。

歷史類比一：LULU 2018–2022 成熟曲線——新進者的宿命。所有 athleisure 新進者最終都會碰到 derate point，問題不是會不會，是何時。量化錨：LULU FY18-22 revenue +25% CAGR、FY22 Q4 北美 SSS +28%，到 FY26 Q1（2026-06）北美 comp 已 -6% c/c——maturation 用了約 7-8 年走完 +28% → -6% 的全曲線。對本次含義：HOKA FY26 +16%（vs 前一年 +28%）已是第一道 deceleration、ON 仍 +26% c/c 但 mix 往 lifestyle 走、derate cycle 已啟動。反向類比警示：HOKA / On 全球滲透率仍 < 5%（vs Nike ~25%），TAM 空間大；On 性能定位深、Strava-active runner 滲透高，maturation 路徑可能偏向 ASICS（多年 mid-single-digit growth）而非 LULU 式陡降。

歷史類比二：Adidas 2017 Yeezy 過熱 vs NKE 2024 Donahoe-era reset。每次品牌把爆款餵爆通路，brand fatigue 接著來、份額被對手切走。量化錨：Adidas 2017 Yeezy boost 350 + Stan Smith / Superstar 過熱售罄 → 2018-19 NKE Air Force 1 / Dunk 反撲，Adidas NA share 從 11% → 6%。本次：Donahoe 把 NKE classics（AF1 / Dunk / AJ1）餵爆導致 brand fatigue，Hill 上任後刻意抽掉 $4B+ Classics 庫存（約 5pp 營收 headwind 落在 Q3 FY26）。差異：Adidas 2018 反彈靠 NKE 自家爆品；NKE 2026 反彈卻被 HOKA / On / Anta 三方夾擊，無法重演 Adidas 2018——Hill turnaround 上限是把營收回到 FY25 的 95-100%，不是 reach FY22 高點。

本主題無單一價格 cycle，用「K 形子象限增速分叉」當時序錨。

關於 cycle 統計表的省略：服飾隨總經有週期，但本主題是結構性 K 形分化（需求二分叉 + 新進者份額切割 + 通路權反轉），非單一 ASP cycle，故不立 cycle 統計表；以「服飾四代演進」（1990-2005 mall → 2005-15 athleisure 1.0 → 2015-22 DTC + athleisure 2.0 → 2023-28E K-shape + tariff reset）的編年呈現時序。

這段歷史的可操作含義：服飾 K 形同時是「LULU 成熟曲線」（新進者終 derate、中段失速）與「Adidas 過熱」（NKE classics 被餵爆 → 份額永久流失）。bull 的底是高端 elevation + 純度新進者 + URBN inflection，bear 是中段 mass 持續失速——LULU 北美 -6%、Vans 連衰已是當期證據。判別一家公司站哪邊，看它賣的是「brand heat / 高端 elevation」還是「中段 mass commodity」。

§2 技術成熟度 + S 曲線

為什麼 K 形是現在？因為 athleisure 滲透率跨過 mature 門檻（美國 ~30-32%）、performance fabric 護城河被 dupe 商品化、China local-for-local 確認——三者讓「整體循環反彈」這個敘事失去基礎，分化變成結構。

「技術成熟度」在服飾不是矽晶圓而是品類滲透 + 護城河子棧成熟度。三個拐點同時發生。Athleisure 滲透率 mature：美國 athleisure 占 apparel 已 ~30-32%（2026），龍頭曲線（LULU 北美 comp -6%）已轉折早於整體 sector。Performance fabric 被 dupe：LULU Align legging 被 Vuori / Alo / Costco Kirkland / Amazon Essentials 全面 dupe，毛利從 GM ~60% 降至 54.9%（近季），LULU Align $98 hold flat 5 年（不敢漲、因 dupe 覆蓋太高）。China local-for-local：Anta + Li-Ning 中國份額從 22%（2020）→ 36%（2025），NKE 大中華 FY19 ≈ FY25 雙 $7.5B（6 年零成長）。三者交會，「服飾整體循環反彈」失去基礎、分化變成結構。

ASP 抗 commoditization 動態（決定誰守得住 elevation）

S 曲線進 mature 後，分勝負的不是「滲透率還能漲多少」，而是「ASP 能不能漲、能不能不流失客」。這裡的對照最能看出 K 形上端與中段的分野。RL Polo Shirt 從 2020 的 $98 漲到 2025 的 $110-125（5 年 ASP +25%），靠 Made in Italy + 縮減 outlet 把 GM 撐到 ~69%——這是 elevation 戰略的範本，抗性最強。對照 LULU Align legging 5 年 hold flat 在 $98（不敢漲、因 Vuori / Alo / Costco / Amazon Essentials 全面 dupe），抗性最弱、是 fabric 護城河失守的直接量度。HOKA running shoe（$130-145 → $155-165）與 UGG Classic II（$170 → $200）抗性中，但都進 maturation——HOKA 靠 Vibram + carbon plate 差異化、UGG 被 Z 世代 Tasman / Goldenstar 替代正在 commoditize。ANF jacket 是危險訊號：Comfortable Casual elevation 把 ASP 在 4 年內推 +50%（$98 → $148），但 namesake 增速已降到 +2%、Hollister comp 翻負——暗示中段 elevation 已碰消費者支付意願天花板，ASP 不能持續高於 mall 段客的承受度。

§3 供給側：現在與未來 + 利潤池

服飾的利潤往「高端 elevation pricing + 高 GM DTC mix」集中——high-GM tier（RL / DECK / LULU / GOOS）擴張、low-GM tier（UAA / VFC / GAP / PVH）壓縮；同時利潤池正從中游品牌往下游通路遷移。

供給端可拆三層 value chain。上游（原料 + 製造）：Cotton（US / India / China 60%）+ Polyester；製造在 Vietnam（NKE FW ~50%）/ Bangladesh / Cambodia / Indonesia / India，GM 5-12%、議價權極低，10% 統一 tariff + 工資漲 7-65% 雙重壓力。中游（品牌，本 ID 15 檔）：GM 40-70%、議價權核心，但內部 K 形已下沉——high-GM tier（RL 69% / DECK 57.7% / LULU 54.9% / GOOS 71.3%）vs low-GM tier（UAA 44% / VFC ~52% / GAP 38% / PVH ~58% 但 OPM 低）。下游（通路）：DTC + DICK's-FL 整併 + Amazon（NKE 2025 回流），GM 25-40%、議價權上升。利潤往「中游高 GM tier + 下游整併通路」集中。

  • 我們說 Y：Tariff 是 GM permanent reset 但量級已大幅收斂——2026-02 SCOTUS 推翻 IEEPA、改 10% 統一 Section 122 稅（footwear，2026-02-24 生效）；稅率仍是 2024 零基期的永久上移（DECK FY26 GM -80bps、ON 估 -4pp GP），但不再是先前估的 -215~-550bps 量級。🔵 [X: base 很可能 — 10% 統一稅維持、GM lock 在 -50~-100bps；bear 可能 — tariff 再升級]。
  • Priced-in：市場把 tariff 不確定性 part-price 在高曝險者（PVH 5.9x / NKE）；但「10% 統一稅是利空收斂」對高 Vietnam 曝險者（ON ~90% / NKE ~50%）的正面含義未充分 price——分歧 2 在 SCOTUS 後從「permanent reset 利空」變「量級收斂」，這個方向反轉是 GM 估值的關鍵變數。
→ 推論邏輯：tariff thesis 在 2026-02 後需修正——仍是永久上移但量級收斂、高曝險者反而受惠。⚠ 證偽：tariff 再升級回 country-specific、或 SCOTUS 後 Trump 另尋法律基礎重課。
🟡分歧 3：「VFC turnaround alpha 在 Vans inflection」信心：中
  • 共識說 X：VFC turnaround 過半 / Vans inflection 即將來臨 / Sun Choe（ex-LULU）上任後 Vans「衰退率減速」= 即將轉正。🟢 [F: T1 — Vans 衰退率減速]。
  • 我們說 Y：VFC 真 alpha 是 TNF + Timberland（NA TNF +5% c/c / Timberland +9% Americas）+ <2% China sourcing 抗 tariff；Vans 的「衰退率減速」是「continued decline at slower rate」、不是 inflection——Sun Choe 需 12-18M 才看到 SSS 轉正。🔵 [X: base 很可能 — TNF/Timberland 帶 EPS；bear 可能 — Vans 拖累擴大]。
  • Priced-in：市場把 Vans inflection 預期 price 進 VFC（估 FY27 EPS $1.50 vs $1.20 共識）；沒 price「Vans 連衰是 brand fatigue 需 product reset、不是行銷 reset」——分歧 3 的 Vans 持續失血未充分 priced，但對 VFC 整體不致命（TNF + Timberland 可帶 EPS）。
→ 推論邏輯：VFC turnaround 真 alpha 在 TNF + Timberland、不在 Vans。⚠ 證偽：Vans Q SSS 轉正 +ve、product reset 成功。

Steel-man 反方（最有力的 3 條）

  • 反方 1（athleisure peak + cyclical reset）：HOKA / On 同時 SSS 個位數、LULU 北美 -3% 連 2 季、新進者搶份額同時放緩，全 sector 進 cyclical reset。當前證據：HOKA FY26 +16%（vs +28%）已減速；Vuori IPO / Alo $10B 暗示新進者搶份額。證偽窗口：HOKA/On SSS + LULU 北美 comp。成真時間窗：2027 H1。
  • 反方 2（China local-for-local 不可逆）：Anta + Li-Ning 中國份額 22% → 36% +14pp、NKE 大中華 6 年零成長 + Q4 約 -20%、FILA + Anta GM 已超 NKE China。當前證據：已成真，前瞻問題是「能否從零成長 → -5% 衰退」。證偽窗口：NKE 大中華 YoY。成真時間窗：已部分成真。
  • 反方 3（高端也被拖下水）：若美國失業率破 5.5% + China 整體消費再降溫，連 RL / 高端段也會 derate（K 形上端塌）。當前證據：RL China +40% 仍強、但暖冬 + GOOS China -4.7% 暗示高端 outerwear 已有裂縫。證偽窗口：RL Asia comp + 失業率。

市場最低估的：最低估的風險是分歧 1 的「中段 mass 結構性 derate」（仍被當 cyclical）與分歧 3 的「Vans 持續失血」；最低估的正面是分歧 2 的「10% 統一稅是利空收斂、高曝險者反而受惠」（SCOTUS 後 thesis 方向反轉）。三者都該獨立監測。

三條分歧裡最可操作的是分歧 1（K 形結構性）+ 分歧 2（tariff 量級收斂）——前者有明確領先指標（兩端 vs 中段 comp 差距），後者是 GM 估值關鍵變數（SCOTUS 後方向反轉）。分歧 3（VFC Vans）次之、且對 VFC 整體不致命。
