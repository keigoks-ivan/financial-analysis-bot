治療已 priced，真正的 alpha 在下游六域

市場把這當「三個重點 sector（治療 / 餐飲 / 食品）」在框 GLP-1 衝擊，焦點窄了。真正的結構是一場六域串聯式 cascade——一個 GLP-1 user growth 訊號（11M → 25M），同時是六個下游產業的 leading indicator，而擴散順序與市場預期顯著不同。

治療側已不是分歧所在：Eli Lilly 2026 Q1 全公司營收 $19.8B（+55.5%）、Mounjaro 與 Zepbound 合計約 $12.8B、GLP-1 市占約六成，Novo Nordisk 同季 Wegovy 口服首發雙倍預期、注射端仍兩位數增長但 Ozempic 轉跌——雙寡占與 K 形分化（食品 / 餐飲）都已被大量定價。真正 underpriced 的是三個擴散域：旅遊（Jefferies 算出航空年省 $580M、約 4% EPS）、健康（Planet Fitness 公開約 50% GLP-1 用戶考慮上健身房）、保險（RGA 模型估 2045 前美國死亡率降 3.5%）。三條 quantified data 已存在，但都還沒進 sell-side 的 EPS model。

所以本報告不站隊「GLP-1 泡沫 yes/no」。治療誰贏很清楚（LLY/NVO 雙寡占）、且已 price；判斷重心放在跨域時序梯度——同一個 catalyst 對「未被 priced 的三域」邊際 alpha 大於對「已 priced 的成熟域」，而最大的單一裂縫是 cohort 行為持續性（約 2/3 用戶一年內停藥）。

PART I · DECISION SUMMARY

決策摘要層

三句話看完六域 cascade 的現狀、走向與行動，後接 PM 級監測點。這一層是報告的全部重點——下面各節只是把這一層的每個判斷補上證據。

現在 / NOW

治療母題在加速、雙寡占格局確立——LLY Q1 2026 全公司 $19.8B（+55.5%）、Mounjaro+Zepbound 合計約 $12.8B、GLP-1 市占約 60%；NVO 同季 Wegovy 口服首發 $354M（約 2x 預期）、注射 +12% CER 但 Ozempic -14%。治療 + 食品已 priced，旅遊 / 健康 / 保險三域 leading-edge data 已出但未進 EPS model。

未來 / NEXT

未來 3–5 年六域依時序梯度展開：治療 0-3 月即時、食品餐飲 6-18 月、旅遊 12-24 月、保險 mortality 24-60 月最慢但 magnitude 最大。2026-Q3 至 2027-Q1 是跨域 catalyst peak window（Foundayo ramp + retatrutide TRIUMPH-2/3 + Medicare Bridge）。最大裂縫是 cohort 24 月行為持續性。

行動 / ACTION

核心配置直接受惠且已驗證的治療側（LLY / NVO / STVN）作 base；alpha 加碼方向放 underpriced 三域——旅遊（DAL 等 top-4 航空）、健康（PLNT / HIMS）、保險（MET / PRU / RGA long 對 health insurer 短期 short 的 pair trade）。買賣由個股 DD + 系統決定。

The Key Call · 本報告最重要的一個判斷

市場用「三個重點 sector」框架看 GLP-1，真正的非共識是——這是六域串聯式 cascade，且擴散順序與共識顯著不同：治療 + 食品 + 餐飲已被 sell-side 大量分析、Phase III-IV 已被 priced；旅遊 + 健康 + 保險仍 Phase I-II 嚴重 underpriced，但三條 quantified leading-edge data 已存在——Jefferies 算出航空年省 $580M / 約 4% EPS、RGA 估死亡率降 3.5%、Planet Fitness 公開約 50% 用戶考慮上健身房。三條都還沒進 sell-side 的 EPS model。分歧不在「GLP-1 會不會擴散」（會），而在「哪些域已被 price、哪些還沒、各域的時滯多長」——同一個 catalyst 對未 priced 三域的邊際 alpha 大於對成熟域。

本 ID 的三道裂縫具體是：①cohort 行為持續性——六域 thesis 全部建立在「用戶行為改變且持久」之上，但實際約 2/3 用戶一年內停藥（RGA 引述產業研究），若 24 個月 cohort 顯示食慾恢復 / 體重回升，整個 cascade 退化為「藥物自身」而非「跨產業革命」；②口徑陷阱——LLY $19.8B 是全公司營收、Mounjaro+Zepbound 約 $12.8B 才是 GLP-1 franchise，NVO 報告營收 +32% 含 340B 沖回（調整後 -4% CER），任何「LLY 快過 NVO」的論斷都要對齊分子量 vs 全公司口徑，否則錯估市占傳導；③sell-side incorporation lag——underpriced 三域的 alpha 仰賴 sell-side 把 Jefferies / RGA / PLNT 數據放進 EPS model，歷史 baseline 是 6-12 月，但若拖到 18 個月以上，alpha 仍在但 timing 拉長、conviction 須降一級。盯這三條，比盯任何單季處方數字都更早看到拐點。

PART II · HISTORICAL ANALOGS

兩個歷史類比，框住速度與廣度

歷史不必重演，但兩個藥物擴散史各給一個可操作教訓：他汀（statin）框「shadow 產業反向受益要等多久才被認知」，吸煙 cessation 框「mass behavior change 的下游 cascade 有多寬」——再用一個反例（SSRI）框住 bear 的天花板。

他汀擴散（1987–2010）——shadow 產業的反向受益，要等 5–7 年才被 sell-side 認知。Lovastatin 1987 年 FDA、Lipitor 1996 上市，美國他汀用戶到 2008 年達約 3,200 萬（橫跨 22 年）。它的下游 cascade 不只是心血管支架與繞道手術需求下降，還包括蛋類、補充劑這類「shadow demand」結構移轉——但這些反向受益方向，sell-side 平均花 5–7 年才放進 EPS model。對 GLP-1 的含義是直接的：「protein pivot」（Chipotle / CAVA / 高蛋白飲品）就是同一 pattern——替代不是完全取代，而是需求結構移轉，而保險端的 mortality 改善正是他汀當年對壽險的翻版，只是這次有 RGA 把數字 quantified 出來。教訓清楚：cascade 的財務體現有結構性時滯，這個時滯本身就是 alpha 來源。

吸煙 cessation（1964–1990）——mass behavior change 的下游 cascade 可以很寬。1964 年 Surgeon General report 後，美國吸煙率從約 42%（1965）降到約 25%（1990），這個行為改變的 cascade 橫跨菸草、餐飲（酒吧 / 速食）、醫療（肺癌治療）、保險（壽險 mortality）多個產業，而非局限於菸草股本身。關鍵差異是速度：GLP-1 的用戶成長（11M → 25M、約 5 年）比吸煙 cessation（42% → 25%、約 25 年）快 5–10 倍，cascade 的傳導速度應同步加速——意味著歷史上要 25 年才走完的下游重定價，這次可能 8–12 年壓縮完成。這把六域 cascade 的廣度框得很清楚：只要行為改變的 magnitude 夠大且持久，下游受影響的不會只有「吃飯這件事」。

怎麼讀：兩個正類比（他汀 / 吸煙）證明「cascade 真實且夠寬」，反例（SSRI）框住 bear 天花板——SSRI 當年也被預期觸發「酒精消費下降 / 旅遊增加 / 保險 mortality 改善」，結果全部落空，因為「mood 改善」的行為 magnitude 遠弱於「食慾抑制」。GLP-1 與 SSRI 的關鍵差異是 GLP-1 直接改變食慾 hormone、且 weight loss 是可見的身體變化，必然觸發 wardrobe / fitness / social 連鎖；SSRI 只調整 serotonin uptake、是內在症狀緩解。所以 bear 不是「cascade 不存在」，而是「若 24 個月 cohort 行為 reverse，cascade 縮回 SSRI pattern」——這正是 PART I 監測點 1 的由來。

PART III · SUPPLY SIDE

供給側：臨床護城河與六域傳導時滯

反直覺但可操作的結論：GLP-1 母題的「供給」稀缺不在產能，而在「臨床護城河 + 跨域傳導時滯」——這把競爭從單一商品供需，變成「誰先把行為改變傳導到下游、誰的下游時滯最長最持久」。

治療側的供給結構是「兩家寡占 + picks-and-shovels 瓶頸」。LLY 與 NVO 雙寡占並非靠產能，而是靠 tirzepatide / semaglutide 的臨床數據護城河——LLY 美國專利保護到 2036 年以後、retatrutide（三激動劑）與 orforglipron（口服小分子）拉開第二代差距，NVO 則以 CagriSema 與 amycretin 回應。真正的產能瓶頸落在 picks-and-shovels：Stevanato、SCHOTT 這類 cartridge / 注射裝置寡占廠，scarcity 議價權隨著注射筆量級擴張而上升。但母題的關鍵稀缺不在這裡——而在「行為改變如何傳導到六個下游域」，每個域的傳導 channel 與時滯都不同，這才是跨域 alpha 的供給端。

Inference · 供給
稀缺＝臨床護城河 × 跨域傳導時滯，不是產能
前提：LLY/NVO 臨床數據 + 專利保護到 2036+（治療護城河）× 行為改變經六種不同 channel 傳導下游（治療 0-3 月 / 食品餐飲 6-18 月 / 旅遊 12-24 月 / 保險 24-60 月）× sell-side 平均 6-12 月才把跨域數據放進 EPS model

母題的供給端框架不能用「單一商品的供需循環」描述——GLP-1 不是一個會 over-build 產能、然後價格崩跌的 commodity。它的稀缺是雙層的：治療側是臨床 + 專利護城河（短期不會被新進者打散），下游各域是「傳導時滯」本身——時滯越長的域（保險 mortality 24-60 月），alpha 持續越久，因為 sell-side 越晚 incorporate。這把競爭從「誰產能大」改寫成「誰的下游時滯最長且最持久」，所以利潤池不會像 commodity 那樣被打散，而是沿時序梯度逐域釋放。

可證偽條件：若 2027-Q1 出現第一家 GLP-1 biosimilar 核准、且 MFN / TrumpRx 把 net price 結構性壓低，導致 LLY/NVO 縮減 CapEx 與 R&D，則 picks-and-shovels 需求與跨域 spillover 動力同時弱化、「臨床護城河」論降級——但 LLY tirzepatide 美國保護到 2036 以後，此情境機率不高。

PART IV · DEMAND SIDE

需求側：用戶量級驅動與 priced/underpriced 分域

需求最不可替代的驅動，是「一個用戶量級訊號同時是六個下游域的 leading indicator」——而需求端的真正分歧，不在量級會不會成長（很可能會），而在哪些域的下游影響已被 price、哪些還沒。

需求由一個共同引擎驅動：GLP-1 用戶量級。從約 6M（2024）到約 11M（2025）再到本 ID 估的約 25M（2030），這條曲線同時是治療處方收入、食品 volume drag、餐飲菜單結構、航空載重、健身房入會、壽險 mortality 假設六條下游曲線的源頭。2026 年 4 月 orforglipron（Foundayo）口服核准，把可及性從注射筆的價格與門檻層級，下拉到「任何時間、不限食水」的口服層級——這是 user growth 的加速器，也是 PART I 監測點 2 的核心。需求端的「量」很穩（合規 + 慢病負擔 + 口服可及性），真正的變數在「下游各域被 price 到什麼程度」。

怎麼讀（priced-in reverse 推導）：以航空為例——Jefferies 已把「10% 體重下降 → 約 2% 飛機總重 → 約 1.5% 燃料成本 → 約 4% EPS」這條鏈 quantified，但 DAL 現價的隱含 EPS 路徑裡看不到這 4%，等於市場 reverse-implied「GLP-1 對航空燃料的效益＝0」；保險同理，MET 以遠低於市場的倍數交易，現價隱含「死亡率假設不變」，而 RGA 已把 -3.5% 模型化。缺口的解讀：這不是市場錯，而是 sell-side incorporation lag——歷史 baseline 6-12 月，alpha 視窗就在這個 lag 內。三域共同特性是 quantified data 已存在但 EPS model 未反映，這是與「治療已 fully reflect」最大的口徑差。

Inference · 需求
同一用戶量級訊號，對六域的邊際 alpha 不對稱
前提：治療 multiple 已大量 expansion（LLY 已 re-rate）× 旅遊 / 保險仍以週期股 / 無聊 sector 倍數交易 × 同一 catalyst（如 Foundayo ramp）確認的是「全域 user growth」

當一個 catalyst 確認 user growth，它對六域的 multiple 影響並不均勻：已 priced 的治療，catalyst 只確認「多年 EPS 能見度」（10-20% upside）；underpriced 的旅遊 / 保險，同一 catalyst 同時觸發「multiple re-rating + EPS upside」（合計可達 30-50% upside）。所以 catalyst 前的 portfolio 結構應該重 underpriced 域、而非平均分布——這是跨域時序梯度的需求端對應。

可證偽條件：① 2026-Q4 至 2027-Q2 的 24 個月 cohort 數據顯示行為 reversal（食慾恢復 / 停藥率升至 60%+）→ 六域需求引擎同時熄火、退化為 SSRI pattern；② 2027-Q1 catalyst 達標但 underpriced 域（DAL / MET / PLNT）未 re-rate → 「邊際 alpha 不對稱」論部分推翻。

多空為何都對

多頭測的是擴散方向（user growth、cascade 廣度、underpriced 域終將 re-rate），這幾乎沒有爭議、很可能持續；空頭測的是行為持續性與口徑（約 2/3 用戶一年內停藥、24 個月 cohort 未驗、LLY 全公司口徑 vs franchise 口徑）。兩者可以同時為真：cascade 確實在發生（多頭對），但若行為不持久、cascade 會縮回藥物自身（空頭對）。所以本報告把判斷重心放在「哪些域已 price、行為是否持久」，而非「GLP-1 會不會擴散」。

The One Line

GLP-1 是一場六域串聯的 cascade——治療與食品已 priced，真正的 alpha 在旅遊、健康、保險三條 quantified-but-unpriced 的下游，和它們各自不同的時序梯度上。

這份報告刻意不做的三件事

  1. 不告訴你哪一檔該買——CORE / EDGE 是「各域特徵的代表案例」，買賣由你的個股 DD 與 Pure MA 系統決定。LLY「最確定受惠」不等於「最佳邊際 alpha」（已 priced），DAL / MET「最 underpriced」也不等於「最佳進場價」（須驗 sell-side incorporation 進度）。
  2. 不預測股價走勢——只給「進入哪個情境」的觸發訊號（24 個月 cohort、Foundayo run-rate、sell-side incorporation、RGA update），不給點位。
  3. 不假裝知道行為會不會持久——約 2/3 用戶一年內停藥是已知事實，24 個月 cohort 是否 reverse 目前無定論，已誠實標為母題級 kill switch（監測點 1）；若 reverse，六域 cascade 同時受打擊。
