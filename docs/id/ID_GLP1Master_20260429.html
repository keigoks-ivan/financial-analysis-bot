§1 產業定義與邊界

「GLP-1 跨域擴散母題」是 — 所有因 GLP-1 medication 用戶行為改變而產生 P&L impact 的下游產業集合，本 ID 為母題框架，不重複各子題已有的 deep-dive。範圍邊界依據 (a) 是否有 quantified financial data（Numerator 30K consumer / Cornell-JMR / RGA mortality study / Jefferies airline study / PLNT 公開 commentary 五大 anchored data source）；(b) 是否有 listed equity 受影響可投資；(c) Phase 階段（Phase 0 純研究無 earnings impact 的 out-of-scope）。

母題 ID 的 結構性 leverage 在於：(1) 一個 GLP-1 user growth 訊號（11M → 25M 2030）同時是 6 個下游產業的 leading indicator；(2) cross-tier ticker（如 LLY 本身 / KO Coca-Cola 同時 receive sugary -7% volume drag + fairlife protein +20% / UNH 短期 cost 高但長期慢病 save）需在母題層才能完整評估；(3) Phase 不一致創造「跨域時序梯度套利」機會（治療股已 Phase III 但保險股仍 Phase I = 同一 catalyst 對保險股邊際 alpha 大於治療股）。

§2 跨域擴散 S 曲線 + 歷史類比

GLP-1 的跨域擴散不是 sudden event 而是「user growth driver → 6 domain serial cascade」過程：(1) Phase I 機制驗證（2017-2024）— Ozempic 2017 FDA / Wegovy 2021 obesity-only / SELECT 2024 CV outcome；user 從 0 → 6M；(2) Phase II 行為固化（2025-2026）— Numerator 30K consumer 證實 -10% 跨類別消費 / Cornell-JMR 6-month cohort 數據 / 11M user / Foundayo 2026-04 oral 啟動；(3) Phase III 跨域擴散爆發（2026-2030）— Medicare BALANCE 2026-07 + retatrutide 2026-Q3-Q4 + 25M user 2030 / 治療 + 食品 + 餐飲 + 旅遊 + 健康 + 保險 6 domain 同步 cascade；(4) Phase IV 結構固化（2030-2040）— US adult 滲透 25-35% / 慢病 burden 結構釋放 / mortality reduction 體現 reinsurer reserve / health insurer cost burden 完成 reset。

★ 4 個歷史類比（含正反例對照，QC-I28）

類比 1：吸煙 cessation cascade（1965-1985）— 正類比。1964 Surgeon General report 後吸煙率 42% (1965) → 25% (1990)；正好 25 年從 user 高峰到結構衰退。對應產業 cascade：菸草股（PM/MO）/ 餐飲 (酒吧 + 速食) / 醫療（lung cancer treatment）/ 保險（life mortality）。重點：行為改變的下游 cascade 通常需要 15-20 年完整體現，但 leading-edge data 在 5-7 年內就可量化。GLP-1 的 user growth 速度（11M→25M 5y）比吸煙 cessation（42%→35% 25y）快 5-10 倍，cascade 速度應同步加速。

類比 2：Statin diffusion（1987-2010）— 正類比。Lovastatin 1987 FDA → Lipitor 1996 → 美國 32M 用戶 2008（22 年）；對應產業 cascade：心血管支架 / bypass surgery / 蛋類 / 保險（life mortality）。重點：藥物降低慢病風險後，shadow industry（如蛋類補充劑）反向受益但 take 5-7 年才被 sell-side 認知。GLP-1 的「protein pivot」（CMG/CAVA/Premier Protein）是同一 pattern — 替代不是完全取代，而是 shadow demand 結構移轉。

類比 3：Antidepressant SSRI（1987-2005）— 反例類比（QC-I28 強制）。Prozac 1987 FDA / Zoloft 1991 / Paxil 1992 / SSRI 美國 12.5% 成年使用率 2010（25 年）。但 SSRI 的「下游產業 cascade」遠比預期溫和 — 預期的「酒精消費下降 / 旅遊增加 / 保險 mortality 改善」全部沒有顯著實現，因為 SSRI 的行為改變 effect 比 GLP-1 weak（mood 改善 ≠ 食慾抑制）。反例警示：若 GLP-1 的「行為改變」magnitude 被誇大（如 cohort 12-24 month 數據 reverse），跨域擴散 thesis 將大幅縮水。但目前 Numerator + Cornell + RGA + Jefferies 4 條 anchored data 都顯示 magnitude 大於 SSRI；GLP-1 與 SSRI 的關鍵差異是 GLP-1 直接 alter 食慾 hormone 而 SSRI 只調整 serotonin uptake — mechanism 強度不同。

類比 4：PPI / 胃酸抑制劑（1989-2010）— 反向類比（部分反例）。Prilosec 1989 FDA → 美國 PPI 用戶 25M+（隨後成 OTC）；對應產業 cascade：胃藥 / 抗酸劑 / antacid（Tums / Pepcid）。但 PPI 的下游 cascade 限於「同類藥物 substitution」，沒有跨產業 cascade（吃辣 / 酒精 / 旅遊 都沒受影響）。反例警示：若 GLP-1 的「行為改變」局限在「吃飯量」而沒有 spillover 到 leisure / fitness / travel / sleep，則旅遊 + 健康擴散域 thesis 將被推翻。但 PwC 2025 survey + Numerator data 已顯示 sporting goods +24% / formal-wear +80% / fitness 50% 考慮 = spillover 是真實的；GLP-1 vs PPI 的差異是 weight loss 是 visible body change，PPI 是 internal symptom relief，前者必然觸發 social / wardrobe / fitness 連鎖效應。

💡 §2 Insight（含 QC-I28 反例）
1. 2 正類比 (吸煙 / Statin) + 2 反例類比 (SSRI / PPI) 框架收斂於：GLP-1 = 強度介於 statin 與吸煙 cessation 之間的 mass behavior change。Cascade speed 比吸煙快 5-10 倍但比純藥物 substitution（PPI）廣 — 預期跨域擴散覆蓋 6+ domain 而非 2-3 domain。
2. 反例類比警示是本 ID 證偽核心：若 GLP-1 cohort 12-24 month behavior data reverse（如食慾恢復 / discontinuation 60%+），則跨域擴散 thesis 退化為 SSRI pattern（局限於藥物自身）；本 ID 在 §13 列具體 falsification metric 包含 cohort persistence 數據。
3. 跨域擴散的 sell-side incorporation lag 5-7 年是歷史 baseline：吸煙 cessation 1965 報告 → 1972 公衛保險開始 reprice（7 年）；statin 1987 → 1995 cardiovascular hospitalization rate 進入 EPS model（8 年）；GLP-1 11M user 2024 + leading data 2025-2026 → 2030-2032 是預期 sell-side EPS reset 時點 — 對 PM 是「2026-2028 build position before consensus catches up」窗口。

§3 6 個 Domain 的擴散機制拆解

母題 ID 的關鍵在跨域 mechanism 整合 — 每個 domain 有不同的 transmission channel + lag。本章節 5 段敘述 + 1 張機制對比表，建立母題的 unifying framework。

Mechanism 1：直接受惠（治療 + 健康部分）。LLY/NVO/STVN/SCHOTT 是 user growth 直接受惠 — 每個新 user = 直接 prescription revenue / cartridge demand。Lag = 0-3 月（每季 prescription 數據即時體現）。健康 domain 的 PLNT/HIMS 是 Phase I user 行為固化的 direct beneficiary（GLP-1 user 的 strength training + telehealth needs），lag = 6-12 月。本 channel 的 alpha 速度最快但已 priced 程度最高（治療 fully priced / 健康 partially priced）。

Mechanism 2：消費結構轉移（食品 + 餐飲）。GLP-1 user 食量 -20-30% + 偏好高蛋白 → 食品/餐飲 channel 的 volume drag + protein pivot alpha 雙效應。Lag = 6-18 月（cohort 6-month behavior data 開始顯現）。Cornell-JMR + Numerator 2025 數據已 anchored 此 channel；KHC/HSY/MDLZ 受害 + CMG/CAVA/Premier Protein 受惠是該 channel 已經被 priced 的部分。但 Phase IV 加速（25M user 2030）尚未在 EPS guidance 完整體現 — 是 partially priced。

Mechanism 3：燃料 / 體積 / 重量 leverage（旅遊）。GLP-1 user 體重 -10% → 航空 fuel cost -1.5% / cruise food cost -10-15% / hotel wellness 設施需求增。Lag = 12-24 月（fleet-wide 重量分布需 18-month 數據累積）。Jefferies 2026-01 study 已 quantify $580M/yr top-4 carrier saving / 4% EPS boost — 但 sell-side 尚未把 EPS uplift 加進 model，underpriced 程度高。對 PM 是 leading-edge alpha。

Mechanism 4：Mortality / morbidity 結構重估（保險）。GLP-1 user 心血管 / 腎臟 / MASH 慢病 risk 顯著降低 → life insurance mortality -3.5% by 2045 (RGA central scenario) / health insurance 短期 cost peak 2026-2028 但 5-10 yr 慢病節省 ($173B obesity-related cost / yr) 反轉。Lag 雙向：life insurance reserve repricing = 2-5 yr / health insurance cost burden = 1-3 yr / disability insurance claims = 3-7 yr。本 channel timing mismatch 創造「long life-short health short-term」pair trade alpha。

Mechanism 5：Spillover behavior（旅遊 leisure + 健康 fitness）。GLP-1 user 體重 -10% 後產生 wardrobe（formal wear +80%） / sporting goods +24% / fitness 50% 考慮 gym / wellness retreat 需求 spillover。Lag = 6-18 月（user 第 6-12 月開始 spillover）。本 channel 是「body change → social / lifestyle change → consumption shift」三段傳導；對 PLNT / Equinox / 旅遊 + 健身房 + 服飾 zinc tail 受惠。Sell-side 共識仍把這些 sector 當「週期股」，spillover thesis underpriced。
