## 寫稿 lessons（pilot critic 學到，B2–G 第一版就套用、省 critic 來回）

1. **無機成長必揭露**：任何 headline 成長率若被併購灌大，必須在第一版就標「報告值 +X%（有機 ~+Y%，含 {標的} $Z 併入）」。pilot 抓到 PANW NGS ARR +60% 其實有機僅 ~+28%（CyberArk/Chronosphere $1.63B），原稿誤當「PANW 成長快過 CRWD」momentum，實際有機 ~+28% ≈ CRWD +24%——edge 要改寫成「平台最寬 + 估值便宜」而非「成長更快」。
2. **§7 priced-in 必附量化分位 + 來源**：不可只寫「估值在歷史高位」，要給「EV/S ~Nx 約在 {年}–{年} band 的 ~XX 百分位（來源）」+ 每條分歧現價隱含的成長假設（reverse）。
3. **§5 框架逃逸要架橋**：非商品型產業若不用「過剩/平衡/短缺」，加一句對應傳統框架（如「需求 > 合格供給 = 結構性短缺型，稀缺來自準入門檻非產能」）。
