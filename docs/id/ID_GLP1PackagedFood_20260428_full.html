核心 Thesis
GLP-1 對包裝食品是「終端需求的相變」——減重藥從生理上壓抑食慾，把每個用戶的胃容量永久砍掉一塊，而且不平均地砍在 indulgent 零食糖果餅乾上。 🟢 [I: 用戶層 grocery −5.3% / snack ~−10%（需求被搬走）× user 13M→25M（滲透放大）→ 包裝食品 indulgent 是 secular LOSER；但低估值不是 floor、「便宜的防禦股」反而是 value trap 風險最高的一群，受惠在優格 / 蛋白]。
KEY CALL — 本報告最重要的一個判斷

市場把包裝食品當「便宜的防禦股」在估值，真正的非共識是——這是一場終端需求的相變：藥理性食慾壓抑不靠意志力、難以 fade，Cornell 150K 樣本確認用戶 grocery −5.3%、savory snack ~−10%，砍得最深的正是高毛利 indulgent。所以「便宜」反而最危險——KHC fwd ~10x、殖利率近 7%、HSY ~29x，估值光譜兩端都還沒把 secular 量縮 modeling 進去。分歧不在「GLP-1 會不會傷」（會），而在「低估值與高股息是 value 還是 trap」——這把選股焦點從「買不買防禦股」移到「分清結構受害 vs 受惠，零食糖果是前者」。

GLP-1 衝擊 mosaic：本 ID（包裝食品）只是 GLP-1 對消費端衝擊的一塊。完整圖像需 cross-reference 兄弟題：飲料能量飲（含糖飲料受害 / fairlife 蛋白奶受惠）、GLP-1 對餐飲衝擊（限服餐飲客流 −8%）、GLP-1 治療藍圖（藥物管線 / pricing / user 數的上游錨）。

§1 產業定義與邊界

先把研究對象的邊界劃清楚——本 ID 看的是「美國包裝食品（CPG 食品）受 GLP-1 滲透率上升的中長期需求 reshape」，不是減重藥本身、也不是餐飲與飲料（各有兄弟題）。

「GLP-1 對包裝食品衝擊」這個主題的 in-scope 是美國包裝食品的八個核心 category（savory snack / sweet snack 與糖果 / cookie 與 cracker / cereal / dairy 與 yogurt / frozen / shelf-stable 與 soup / protein 與 nutrition）＋食品 input 上游（McCormick 香料、Smucker 醬料、Tyson 蛋白、fairlife dairy）。out-of-scope 是三塊：(a) GLP-1 medication 本身（屬 ID_GLP1Master 治療藍圖，涵蓋藥物管線與 pricing）；(b) GLP-1 對餐飲衝擊（兄弟 ID_GLP1RestaurantImpact）；(c) GLP-1 對飲料衝擊（兄弟 ID_BeverageEnergyDrink）。

關鍵的 boundary call 是「食品 vs 飲料邊界」——fairlife、Premier Protein 是 RTD（即飲）飲料形態但屬 protein 範疇，本 ID 簡略提及、深度在飲料 ID 避免重複；Vital Pursuit（Nestle）是 frozen meal 屬本 ID 範圍。本 ID 主要解析八個 packaged food category 的個別 GLP-1 受影響程度 + 主要 incumbent（KHC / MDLZ / GIS / HSY / CAG / CPB / NSRGY / TSN）的結構性 winner / loser 分布。

§2 GLP-1 衝擊演進史與 S 曲線

衝擊不是 sudden event，而是四階段累積——從 2017 的臨床 niche，到 2024–2026 的早期主流化（Cornell 確認量縮），再到 2026 起口服 Foundayo 觸發的加速主流化。

GLP-1 對包裝食品的衝擊是滲透率 S 曲線驅動的四階段過程。2017–2020 早期：Ozempic（semaglutide）僅 type 2 diabetes 用藥，100K–1M user，包裝食品公司視為臨床 niche、影響不可測。2021–2023 中期：Wegovy（2021）、Mounjaro（2022）上市，user 從 1M 到 5M，但供給限制 + 商保價格 $1,000+/月把市場 cap 住，全業 effect < 0.5%。2024–2026 早期主流化：user 達約 13M（2025 末）、約 1/5 美國成人試過，Cornell-JMR 確認用戶 grocery −5.3% 的 basket effect、產業層量 −2%。2026–2030 加速主流化（現在）：口服 Foundayo（2026-04-01 FDA）+ Medicare $50/月（7/1）+ 商保 $25/月，三重 access 擴張把 user 推向 2030 ~25M、滲透 ~10%。

本次循環最特別的是 口服切換。注射 GLP-1 受三個限制 cap 住：患者心理門檻、供給、商保 $1,000–1,500/月。口服 orforglipron 完全 break 這三個限制——Medicare $50/月 + 商保 $25/月 + 自費 $149/月，都是大幅 access 擴張；且免食物水限制、任何時間服用。2027 預期是口服滲透躍升年，包裝食品衝擊從「漸進」變「加速」。

§3 供給側：reformulation 是部分對沖、不是解藥

這是一個沒有實體產能短缺問題的供給側——大廠有的是工廠與貨架。真正的供給變數是「能不能把組合從受害類別重配到受惠類別」。

包裝食品不是供給過剩、也不是供給短缺的故事——大廠產能充裕、貨架關係穩固。供給側的結構是「indulgent 重倉的純玩家 vs 已重配到蛋白/優格的玩家」之間的分化。對受害方，唯一的供給側武器是 reformulation（蛋白化、縮 portion、功能化）與併購（補健康定位缺口）。實際動作 2025–2026 大量出現：KHC 新 CEO Steve Cahillane 撤掉分拆、投 $5 億以上轉向蛋白避糖；HSY 投資 VitaKey 做時間釋放營養、擴 Fulfil 蛋白棒；MDLZ 靠 Clif／Builders 蛋白棒對沖；GIS 推 Cheerios Protein、Annie's Super Mac（14g protein）。但業界統計顯示——Campbell's / Conagra / GIS / HSY / Smucker / KHC / MDLZ / PEP 的 16 項 premiumization／reformulation／降價投資中，只有 7 項讓品牌回到成長。

§5 框架逃逸 · 架橋 本產業的供給側框架要跳脫「過剩 vs 短缺」的慣性——它兩者都不是。減重藥侵蝕的是終端需求，是「需求結構性萎縮」型；稀缺的反面不是供給太多、而是需求被外生力量永久搬走。鐵證是反差：受害類別（零食糖果）產能過剩，受惠類別（優格 / 蛋白）卻 capacity-constrained——Danone 美國高蛋白優格缺貨、Chobani 擴廠。需求被搬到別處，不是供給多了。這個架橋是理解 §4 需求破壞與 §5 裁決的關鍵：傳統「供需平衡」尺規在這裡量錯了東西。

§4 需求側：胃容量被永久砍掉的破壞引擎

這是本報告的核心章節——需求不是疲軟、是被外生力量結構性破壞。減重藥從生理上壓抑食慾，把每個用戶的總食量永久下調，且不平均地砍在 indulgent 類別。

需求破壞由一條清楚的因果鏈推動：GLP-1 藥物延緩胃排空、作用於下視丘食慾中樞 → 用戶總熱量攝取下降 → 反映在 grocery 帳單上是 6 個月內 −5.3%（高收入家庭逾 8%、戶均減 $416、高收入戶 $690）→ 而且砍得不平均：savory snack 約 −10%、甜點烘焙與餅乾 −6.7~−11%，主食（麵包/肉/蛋）小幅下降，只有優格（最大增幅）、生鮮水果、蛋白棒、肉類零食逆勢上升。這是 Cornell × Numerator 對 150K 家庭面板的研究——2,623 個含至少一位 GLP-1 user 的家庭（從 22,712 戶篩出）、樣本窗 2023 末–2024 中、2025-12-18 發表於《Journal of Marketing Research》——不是預測、是已觀測到的行為。

需求破壞三角驗證 · 用戶層 vs 產業層 vs user 數放大（口徑不可混用）

怎麼讀：三個視角口徑不同、嚴禁直接相加。用戶層 −5.3% 是「個別用戶 vs 用藥前的自己」，產業層 −2% 是 user 佔比 ~5% × 用戶量縮 + PL/通膨混合稀釋的結果。隨 user 從 13M→25M、滲透 5%→10%，產業層拖累會從 ~−0.5~−1%/年放大到 ~−1.5~−2.5%/年——這是「現在溫和、未來放大」的算術根據，也是公司口徑（MDLZ 0.5–1% 十年）被本 ID 視為 floor 的理由。

重點在「永久」二字。注射用戶有 30–40% 在 12 個月內停藥（保險、副作用、成本），這是衝擊的上限剎車；但口服 Foundayo 把門檻打低、把 user 池放大，淨效果是衝擊深化而非收斂。2027 起第二代藥（GIP/GLP-1 dual、triple agonist retatrutide）減重幅度更大、食慾壓抑更深，會把 indulgent 量縮從 base 的 −5~−8% 推向 bear 的 −8~−10%。

受害方營收模型最脆弱的不是價（仍有定價權）、而是量（被永久搬走）。一般景氣下行是「想吃但沒錢」、可用降價/小包裝救回；GLP-1 是「有錢但吃不下」、降價無效。所以零食糖果純玩家的低估值不是安全墊——當終端需求 secular 萎縮、市場仍當它「景氣底部便宜貨」時，估值會「便宜得有理由」並一路便宜下去（value trap）。

§5 供需裁決與三情境（24–36M）

把供給（reformulation 部分對沖）與需求（永久搬走）兩側合起來裁決：indulgent 是 secular LOSER、蛋白優格是 winner，分歧只在衝擊的「速度與深度」。

需求破壞方向判 INTACT（藥理性食慾壓抑 + 口服加速），受害類別 secular 萎縮判 INTACT（Cornell 證據 + HSY 已現量縮），但「衝擊的速度與深度」判 AT_RISK（停藥率 + reformulation 回補 + access 擴張節奏未定）——故 thesis 結構性看空 indulgent、結構性看多蛋白優格，但 conviction 卡在 mid-high。三情境：

Base Case — 衝擊漸進放大、K 形分化加深 很可能

🟢 [X:] user 2027 達 17–20M、2030 達 ~25M。零食糖果餅乾 5y 累積量縮 −5~−8%，靠提價維持名目營收但量持續探負；HSY/KHC/MDLZ multi-year EPS 緩慢下修，reformulation 補回部分。優格/蛋白棒/雞肉蛋白雙位數成長。

→ 量化影響：包裝食品全業量 ~−1.5~−2.5%/年（K 形掩蓋）；indulgent 純玩家 EPS multi-year 緩跌、便宜估值不構成 floor 但不崩盤——緩跌肥股息型。受惠三軸 EPS 2030 ＋10~15%。
Bull（受害方）— 停藥率高 + reformulation 起效，衝擊收斂 可能

🟢 [X:] 停藥率升向 40–50%、user 池停在 15–18M，Cornell follow-up 24 月顯示 effect 縮至 −3% 以下；reformulation 蛋白 SKU 銷售達原 SKU 50%+。HSY/KHC/MDLZ 從「結構受害」重估為「便宜的穩定股」。

→ 量化影響：indulgent 量縮回補大半、EPS 下修見底、估值修復；KHC/HSY 反彈 ＋15~25% vs base。受惠三軸增幅收斂但仍正。
Bear（受害方）— 口服超預期 + 第二代藥更強，衝擊提前加深 可能

🟢 [X:] Foundayo + Medicare $50/月把 access 打得比預期快、2027 user 衝破 20M；第二代 dual/triple agonist 食慾壓抑更深。零食糖果量縮提前到 −8~−10%/年，PL 同步加速，reformulation 來不及。

→ 量化影響：HSY/KHC/MDLZ EPS 大幅下修、便宜估值仍 de-rate（成長轉負 + 股息覆蓋承壓），跌幅放大 −25~−35%。受惠三軸加速 ＋20%+。

§6 估值傳導：「便宜」price 了多少需求破壞

priced-in 量化——零食糖果純玩家的低估值，到底反映了多少 secular 量縮？答案是：很少，而且是兩端 mispricing、方向相同（multi-year EPS 下修）。

包裝食品的估值張力在「受害方便宜得有理由、受惠方共識仍不足」。受害端 fwd P/E 光譜：KHC ~10x（殖利率 ~6.85%）、GIS ~10.4x、MDLZ ~20x、HSY ~29x。用 reverse-DCF 角度推導：以 GIS/KHC ~10x（vs 必選消費歷史 band 中位約 16–18x）來看，市場確實已把「低成長甚至零成長」price 進去一部分——但 price 的是「景氣循環低成長」，不是「secular 量縮加速」。差別在於：循環低成長假設 EPS 會回升（mean-revert），secular 量縮假設 EPS 持續下修不回頭。

priced-in 三角推導 · 受害端估值對照與隱含假設
玩家fwd P/E / 殖利率估值隱含假設本 ID 判斷 vs 隱含
KHC~10x / ~6.85%低成長但 EPS 穩定、股息可持續未 price secular 量縮；股息覆蓋是後續壓力點
GIS~10.4xcereal stagnant 但 reformulation 撐住部分 price；reformulation 成效是關鍵
MDLZ~20x新興市場成長 + 0.5–1% 十年量縮（公司口徑）公司口徑被視為 floor，實際 −3~−5%
HSY~29xcocoa 緩解後 EPS 回升、GLP-1 minimal最未 price；高估值 + secular 量縮 = 下修空間最大

怎麼讀：估值光譜兩端都有 mispricing——KHC ~10x 便宜得有理由（value trap）、HSY ~29x 高估值未反映 secular，但方向相同：未來幾年是 multi-year EPS 下修。reverse 推導的結論是：沒有任何一檔受害股的估值「充分」price 了 Cornell −5~−10% 的 secular 量縮，因為市場仍用「景氣循環會回升」的框架。

🟢 [I:] 「防禦＝便宜＝安全」是包裝食品最危險的 priced-in 誤判。低估值不是 floor（KHC ~10x 可以更低，香煙股模板）、高估值有下修空間（HSY ~29x）。受惠端（DANOY 優格 / BFI 蛋白棒 / TSN 雞肉蛋白）sell-side 仍用傳統 CPG 框架、對 GLP-1 user 蛋白攝取上升 modeling 不足，是反向的 misaligned alpha——但 DANOY 為 ADR、流動性有限。

§7 分歧與證偽（非共識 view）

把本 ID 與市場共識的差距列清楚——共識依賴公司口徑（「no material impact」/ 0.5–1%），本 ID 依賴學術 evidence（Cornell −5~−11%），差距是 directional。

Thesis #1 — 公司口徑「0.5–1% 十年量縮」是 floor，實際 −3~−5% 信心高
  • 共識（MDLZ CEO 法說）：GLP-1 對 MDLZ volume drag 是 0.5–1% / 10 年。
  • 本 ID：Cornell 餅乾/甜點 −6.7~−11% × 10 年滲透累積 + 口服加速 + reformulation 部分 offset = 實際 −3~−5%。公司口徑是「reassure investor」非真實預測。
  • 差異：共識 0.5–1% vs 本 ID 3–5%（約 4 倍，顯著非共識）。
→ Base：MDLZ 2030 量縮 −3~−5%；Bear −7%；Bull（reformulation full offset）−1%。
Thesis #2 — 「便宜的防禦股」是 value trap，低估值非 floor 信心高
  • 共識：KHC fwd ~10x、殖利率近 7% = deep value，景氣底部便宜貨。
  • 本 ID：終端需求 secular 萎縮下，低估值會「便宜得有理由」並持續便宜（香煙股 1995–2015 模板）；股息覆蓋是後續壓力點。
  • 差異：共識「便宜＝安全」vs 本 ID「便宜＝trap」= directional。
→ Base：KHC 緩跌肥股息、總報酬靠股息；Bear：成長轉負 + 股息承壓 de-rate；Bull：停藥率高 + reformulation 起效，估值修復。
Thesis #3 — 優格/蛋白棒/雞肉蛋白三軸是直接 alpha，共識仍 underweight 信心高
  • 共識：DANOY / BFI / TSN 是 niche pure play，sell-side 用傳統 CPG 框架。
  • 本 ID：Cornell 四大 winner 中優格/蛋白棒/肉類零食三類直接對應；user 食優格 3x average + Danone 美國高蛋白缺貨 = 結構受惠 hard evidence。
  • 差異：共識 niche/mid-cap vs 本 ID 結構性 alpha + multi-year EPS 上修 = directional。
→ Base：三軸 2030 EPS ＋10~15%；Bull ＋20%+（口服全面普及）；Bear ＋5%。
⚠ 無機成長揭露：本 ID 提及的 Mars-Kellanova $36B 整合、各公司併購（如 Smucker 收 Hostess）屬無機；MDLZ 新興市場 organic 與 GLP-1 量縮須分開看。受惠端 BFI 的 RTD 成長含通路擴張（部分無機）；引用 organic 量/mix 時已剔除併購、匯率、剝離。
