§1 產業白話定義 + 邊界

航空公司賣的早已不只是「一個座位」——對 4 大網絡航空而言，最賺錢的產品是「一張綁在你錢包裡的信用卡」。付錢買里程的不只是旅客，更是願意付高 royalty 換取高消費 cardholder 的 Amex / Chase / Citi；而他們最在意的不是航線多寡，是這個航空的常客有多高消費、premium 產品撐不撐得起里程的兌換價值。

白話開場：傳統上，航空是教科書級的 commodity——每家都飛差不多的航線、用差不多的飛機、燒一樣的油，票價戰打到誰先撐不住誰破產（過去 50 年美國航空破產名單長到嚇人）。但拼到 2026 年，4 大網絡航空（DAL / UAL / AAL，加轉型中的 LUV）做了一件 commodity 公司做不到的事：把「常客里程」做成一個接近 payment platform 的 fee 流——Delta 一家光是賣里程給 American Express，一年就收 $8.4B，而且這筆錢的成長跟它飛多少座位（capacity）幾乎脫鉤。所謂「航空 + Loyalty 重定價」，就是這個利潤引擎從「賣座位」（commodity）轉向「賣里程」（fee 流）的結構轉折。

邊界界定：本報告聚焦「有 premium + loyalty 平台的網絡航空」與「被 K 形分化擠壓 / 出局的 ULCC / 中型 niche」這條主軸，並橫切「跨 vertical 受益的信用卡發卡行」。納入美國 4 大（DAL / UAL / AAL / LUV）、中型分化（ALK / JBLU）、結構出局的 ULCC（SAVE 已停飛 / FRO 高 leverage）、相鄰簡略的歐洲 / 拉美 long-haul（LHA / IAG / AF.PA / CPA / RYAAY），以及 co-brand 發卡行（AXP / JPM / C）。不深入純貨運（FDX / UPS）、私人航空（Wheels Up）、飛機租賃（AerCap）、機場 concession，飛機 OEM（BA / AIR.PA）僅簡略（詳見獨立 ID_CommercialAerospace）。

四個灰色地帶（boundary call）：① LUV 算 LCC / ULCC / hybrid？ 視為「正在轉型成 hybrid network carrier」——2025–2026 全面轉型（bag fees / 指定座位 / premium）已到位，Q1'26 op margin +8.1pt 是兌現訊號，但仍非 full-service。② ALK 算中型還是 4.5 大？ Hawaiian 整合 + 西岸 hub 領先 + 高端 leisure exposure＝「中大型」+ boomer 受惠。③ JBLU 算 loser 還是 turnaround？ 失去 Spirit 併購後失去 strategic option、中端 niche 受 DAL/UAL 擠壓——視為 🟡 結構性受擠壓。④ 歐洲航空算 in-scope 還是獨立？ 跨大西洋 boomer 是 cross-link，但歐洲 cycle / cost 結構不同，視為「相鄰簡略」。

歷史脈絡可分三個轉折。第一次：2008–2014——美國航空大整合（DAL+NWA / UAL+CO / US+AA / SW+ATA），sector 從 fragmented 8 家收斂成 4 大寡占，EBITDA margin 從 ~5% 升到 12–14%，估值從 cycle 4–6x 升到 8–10x。第二次：2020–2024——COVID 後 reopen，K 形分化開始萌芽，DAL premium 連年加速、Amex co-brand 從 2024 ~$7.4B 走向 2025 $8.4B。第三次（現在）：2024–2030——premium 永久取代 main cabin（DAL Q4'25 premium 已首超）、co-brand royalty 重定價擴散（DAL 領先、UAL/AAL 跟進）、ULCC 結構性死亡（Spirit 2026-05 全面停飛）三因素同時。本次與第一次整合 cycle 的關鍵差異：上一輪靠「供給整合」推升 margin，本輪靠「fee 流重定價 + premium 結構升級」——成長與 capacity 第一次脫鉤。

§2 產業演進 + 歷史類比

為什麼利潤改寫是現在發生、不是 2015 年？因為兩個條件剛同時成熟——「premium 需求引擎」（嬰兒潮 + affluent millennial 的 wealth transfer 推 premium leisure）與「co-brand 重定價窗口」（hotel MAR +35% 已驗證、發卡行願付更高 royalty 換高消費 cardholder），讓航空第一次能把利潤從座位移到里程。

過去航空的「常客計畫」多半是行銷工具——送里程刺激回購，本身不是利潤中心。真正的拐點是里程被金融化：航空把里程「批發」賣給信用卡發卡行（Amex / Chase / Citi），發卡行用里程吸引高消費 cardholder，再把刷卡手續費的一部分回饋給航空當 royalty。當高消費常客基數夠大、premium 產品讓里程兌換有價值，這個 royalty 流就從「marketing 補貼」變成「接近 payment rail 的穩定 fee 流」。Delta 把這件事做到極致——FY25 Amex co-brand $8.4B，且契約綁定朝 $10B（2029）。

S 曲線的關鍵啟示：4 大 co-brand royalty 從 2024 ~$13B → 2030E ~$30B+（~2.3–2.7x），而這個 step-up 不依賴座位 capacity 成長，而是 royalty rate + cardholder 卡消費（+10–15%）——是航空第一次「成長與 capacity 脫鉤」的非 RevPAR-style fee 流。DAL 領先 UAL/AAL 約 12–18 個月。

正類比：MAR Bonvoy royalty +35% 重定價（帶量化錨）

Marriott 在 2026 從 Chase / Amex 重新議價 co-brand royalty rate +35% 永久 step-up（詳見 ID_HotelChains、ID_CobrandCardEcosystem）。量化對照：MAR 共品牌 royalty 從 ~$800M–1B 一年跳到 $1.1–1.4B（+30–40%）。航空是「下一個 vertical 重定價」，且絕對規模更大——4 大 airline co-brand ~$18B（2026E）約 4 大 hotel chain（~$5–7B）的 3 倍。傳導已在發生：DAL Amex 實質重定價（FY25 $8.4B 朝 $10B）、AAL 2026-01 換 Citi 獨家發卡（收 Barclays portfolio、10 年延約）、UAL 改持卡差異化並洽簽新 Chase 約。傳導時滯量化：從「換約 / 改 benefit」到「remuneration 數字完整顯影」約 12–24 個月，所以不能用單季數字論成敗。

反類比：2018–2020 ULCC 興起（low cost ≠ 永遠 winner）

當年市場曾期望 ULCC 永久搶走 4 大客源，2018 年 Spirit / Frontier 估值 P/E 達 15–18x。實際結局：2020 COVID 後永遠沒 recover——Spirit 2024–25 兩次破產，最終 2026-05-02 全面停飛清算、~17,000 人失業、拍賣 LGA slots 與 loyalty program。多條件對照：當年 ULCC 假設 (a) 客戶永遠選最低價、(b) loyalty 不重要、(c) cycle 性復甦會持續；實際 (a) 客戶在景氣好時選 brand + premium（K 形）、(b) loyalty 是 sticky moat（DAL co-brand $8.4B 即證據）、(c) ULCC 在景氣弱時受擠壓更嚴重。LUV 完全轉型成 hybrid + Spirit 死亡＝ULCC 模式失敗的雙重確認。這把 bear 天花板框得很清楚：押「ULCC 觸底反彈」是押已被證偽的模式。

§3 供給側：座位 commodity vs 里程準入戰 + 利潤池

供給端有兩種截然不同的結構：「座位」是有實體產能、受油價 / capacity / 機隊週期決定的 commodity（無護城河）；「里程」是無實體產能、由高消費會員 × premium 產品 × 信用卡夥伴談判力決定的準入戰（有護城河）。利潤正從前者流向後者。

成本曲線（雙軌說明）：座位端有油價驅動的成本曲線——油價漲到某 level，高成本 ULCC（CASM 低但無 premium 緩衝）先撐不住，這正是 Spirit 在油價 + capacity 壓力下出局的部分原因；里程端則無實體產能、無邊際成本曲線，royalty 的終局由準入門檻（高消費會員 × premium × 談判力）決定。這也是 DAL 能在油價週期裡靠 co-brand fee 流平滑利潤、而 ULCC 只能裸露在油價下的根本差異。

未來供給與進入門檻（因果閉合，回應 §1）：§1 提出「里程準入門檻」是護城河來源——未來 3–5 年它是否更 binding？答案是很可能更 binding：Spirit 退場後 capacity 被消化、4 大 capacity discipline 維持（AAL 已因高油價砍 6 條航線、不擴張），而 co-brand 重定價把 DAL 的領先轉成契約鎖定（朝 $10B 2029）。後進者要複製 DAL 的 $8.4B，需要高消費常客基數 + premium 產品 + 發卡行談判力，這三者單靠加 capacity 砸不出來。唯一能鬆動門檻的是「中立里程聯盟」讓 ULCC 也能分潤高消費客——這是 §7 要檢驗的反方。

12–36 個月內，供給端不會出現新的「里程平台級」競爭者（門檻太高、Spirit 退場 supply 收斂），所以議價權持續集中在 DAL（規模王）與 UAL（第二）。對投資的含義：押「有里程平台 + premium」的航空（DAL/UAL/LUV）比押「純座位」的 ULCC 安全——後者即使估值看似便宜，cycle 裸露 + 無護城河，終局常是出局（Spirit 即案例）。

§4 需求側：K 形需求 × co-brand 重定價 + 三角驗證

需求最不可替代的兩塊是「affluent 常客的旅行剛性」與「信用卡夥伴的分潤承諾」——這是利潤改寫的兩個引擎，但脆弱點不同：前者敏感於宏觀消費信心，後者敏感於重議談判。

現需結構：採購主體分兩層——旅客（買座位 + 里程）與信用卡發卡行（批發里程）。第一引擎是 K 形需求：嬰兒潮 + affluent millennial 的 wealth transfer 推 premium leisure（DAL premium +9–14%、UAL premium +13.6%），而 main cabin 的價格敏感客增長平緩。第二引擎是 co-brand 重定價：發卡行願付更高 royalty 換高消費 cardholder 的卡消費（DAL Amex 卡消費雙位數成長）。兩個引擎方向一致，都把利潤推向「高端常客 + 里程 fee 流」。但需求型態不同（見架橋）。

§5 框架架橋 · 航空需求不能用單一框架

航空客運整體是週期型需求（GDP-sensitive、與油價 / 票價戰共振），不能單用「結構成長」框架；但本 ID 的核心 loyalty 段是結構型 fee 流——co-brand royalty 的稀缺性來自「高消費常客 + 信用卡夥伴談判力」這個準入門檻，而非座位 capacity。所以本報告對「座位需求」用週期框架（情境含宏觀 / 油價 trigger）、對「里程需求」用結構框架（看 co-brand 重定價軌跡）。兩個框架疊在同一產業上，正是 K 形的本質：上半截（premium + loyalty）結構成長、下半截（main + ULCC）週期擺布甚至出局。loyalty 段稀缺＝高消費常客 + 信用卡夥伴談判力，這是本 ID 的核心護城河。

§7 分歧與證偽

本 ID 的多空分歧不在「K 形會不會發生」（共識），而在「結構利潤改寫能不能扛過 cycle 逆風」。以下三個 non-consensus（NC）各附證偽條件，閾值用 current actual 校準。

NC#1：利潤從座位轉向里程是結構性、非循環（判 INTACT）。多頭認為 co-brand fee 流是 secular re-rating；空頭認為 premium 增長是 reopen 紅利、會 mean-revert。本 ID 站多頭——DAL premium 票收已逼平百年 main cabin（Q4'25 已首超）、co-brand $8.4B 契約朝 $10B、Spirit 死亡騰出 supply，這不是循環反彈。🟡 [I: premium 永久結構升級 + co-brand 契約鎖定 → 結構性]。⚠ 證偽（current-actual 校準）：若 DAL/UAL premium RASM 對 main 的 spread 從現在的 >4pt 連兩季收斂到 <2pt，或 Amex remuneration 同比增速從 +10% 掉到個位數，則「結構性」論降級為「循環性」。

NC#2：結構紅利對宏觀消費走弱脆弱（判 AT_RISK）。premium leisure 是本輪成長引擎，但它最敏感於 affluent 消費信心。🟡 [X: 若高端消費降溫，premium 溢價（DAL 每座位貴 ~20%）先收斂]。⚠ 證偽 / 觸發：若 4 大下修 premium guidance、或 affluent 消費信心指標轉弱同時 premium bookings 走軟，bear case 兌現。current actual：AAL Q2 約 80% booked、corporate +13%、leisure 強勁——目前未觸發，但這是「需求得撐住」的前提。

NC#3：油價是座位端的利潤雙殺風險（判 AT_RISK）。jet fuel 從 Q4'25 ~$2.60/加侖升到夏季 2026 ~$3.40（+31%）。目前 disciplined capacity 讓票價能 pass-through，但這是「需求撐得住」的條件。🟡 [X: 若油價漲 + 需求軟，無法 pass-through → 利潤雙殺]。⚠ 證偽 / 觸發：若 jet fuel 破 ~$3.50–4.00 同時 bookings 走弱（AAL 已砍 6 條航線是早訊），座位端虧損會蓋過 co-brand 重定價紅利。current actual：油價 ~$3.40、需求仍強、票價 +20% 可 pass-through——尚可控但需密盯。

反方檢驗（§3 留的伏筆）：里程準入門檻會不會被「中立里程聯盟」鬆動？目前無證據——里程的價值高度綁定發卡行對「該航空高消費常客」的競爭，中立聯盟反而稀釋稀缺性，發卡行沒有動機支持。所以「里程門檻」論目前 binding。
