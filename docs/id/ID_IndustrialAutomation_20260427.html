🟡 #1 [event-triggered] HON 2026 H2 分拆是 sector 最大 catalyst [信心: 高]
共識（Wall Street）：HON 是 stable 工業 conglomerate；分拆是「nice to have」而非 transformational。本 ID 看法：類比 GE 三段分拆 → GE Aerospace +200%；Honeywell Automation 純 industrial AI play 將獨立 listing；fwd PE 從 22x → 25-28x（保守 +30% upside）；Aerospace + Energy 段同時 unlock multi-segment value。

事實 2: GE 分拆後 GE Aerospace +200%
事實 3: Honeywell Forge industrial AI 已建立
→ 推論：HON 分拆 catalyst 可能比 GE 更 leveraged（因 Aerospace 段更純）。
⚠ 證偽：若分拆延 1+ 年 OR Aerospace 段 multiple 低於 BA / RTX peer 50%。
🟢 #2 [structural] ROK 軟體 ARR 14% → 25% 是 underpriced PE re-rate [信心: 高]
共識：ROK 是 cyclical 工業股，PE 25-27x 已 fully priced。本 ID 看法：軟體 ARR 14% → 25% (2030) 觸發 SaaS-like re-rate；fwd PE 應升至 30x+（mirror SAP / CRM 軟體 ARR 化）；FactoryTalk + Plex + Fiix 完整 stack + Microsoft partnership = sticky moat。

事實 2: 軟體 ARR 14% → 25% target by 2030
事實 3: Plex + Fiix M&A 完成軟體 stack
→ 推論：ROK 是 sector PE re-rate 最 leveraged play；fwd PE 30x+ by 2028。
⚠ 證偽：若 ARR 增速 stagnate < 15% 或 Plex / Fiix 整合 fail。
🟡 #3 [structural] 中國 cost competition 擠的是 FANUC / Yaskawa「中段」非「高端」 [信心: 高]
共識：中國機器人 / servo / drive 全面取代日本玩家是 multi-year 結構問題。本 ID 看法：中國 Inovance / Estun 主要擠中段（mass-market servo / 通用 cobot）；高階段 (CNC / 高精度機器人 / 半導體 fab automation) 仍歐美 + 日本 dominant；FANUC / Yaskawa 高階 stable 但中段受害；建議避中段，保高階。

結論

工業自動化進入 Phase II 中（2022+ 軟體 ARR + AI copilot + 中國平價化 + HON 分拆四引擎重塑）。核心非共識三條：（1）HON 2026 H2 分拆是 sector 最大 catalyst，類比 GE Aerospace +200%；（2）ROK 軟體 ARR 14%→25% 是 underpriced PE re-rate trigger；（3）中國 cost competition 擠的是 FANUC / Yaskawa 中段非高端，sector 結構性 split。

結構性受益層次：（A）HON 分拆 catalyst（2026 H2 unlock）；（B）ROK 軟體 ARR re-rate；（C）SIE Industrial Copilot AI 化；（D）EMR / ABB 高階段 stable；（E）中國 Inovance / Estun high-beta share gain。結構性壓力層次：FANUC / Yaskawa 中段 + 美國 manufacturing capex 衰退風險 + 歐洲 manufacturing 持續疲弱。
