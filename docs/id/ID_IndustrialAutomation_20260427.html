工業自動化 / Industrial AI（Industrial Automation）

★ 本 ID 核心非共識 Thesis 市場把工業自動化當「成熟 cyclical 工業股」追蹤 — 看 PMI、製造業 capex 推估營收。本 ID 認為：① 「工業自動化」正在從 cyclical → secular 結構轉變 — re-shoring + CHIPS Act + AI 整合 + 老化 workforce 替換四重 driver 共振，把工業自動化從「製造週期股」結構性升級為「智慧製造平台股」 — 類比 SaaS 在 2010-2020 從 IT 工具升級為「企業作業系統」。② AI 整合是利潤池重新分配的關鍵 — Siemens / Schneider / Rockwell 都在「automate automation」（用 AI copilots 自動產生 PLC code、SCADA 自動 troubleshooting），這個 stack 整合會讓大廠寡占地位結構性深化（中小型工控廠商被擠出）。③ Honeywell 分拆 Aerospace + Automation + GBS 是 Industrial sector 最大事件 — 純 Honeywell Automation 將是 industrial AI pure play，估值有 +1 turn 重估空間（從 conglomerate discount 25-30% → pure play 估值）。
💡 3 條核心 insight（預告）
1. re-shoring + CHIPS Act 是 ROK 結構性順風：ROK 拿到 20% 美國 CHIPS Act 公告 fab 專案；FY26 EPS guide $11.2-12.2 + Q1 +12% YoY 是真實兌現訊號；數據中心 + 半導體 + 倉儲自動化三軌驅動。
2. AI 整合改變工業自動化商業模式：Siemens 投 €1B+ 建 20+ industrial copilots、Schneider + Microsoft GenAI partnership、ROK 把 software ARR 做為核心成長 — 從 hardware-heavy 變 software-led，毛利結構性上修。
3. Honeywell 分拆是 industrial sector 最大 event：Aerospace + Automation + GBS 三家公司 2026 H2 完成；Honeywell Automation 變純 industrial AI play；類比 GE 2024 分拆後 GE Aerospace 大幅 re-rate；HON Automation 估值有 30%+ upside。

§1 產業定義與邊界

工業自動化是個橫跨「硬體（PLC/SCADA/DCS/感測器/驅動器）+ 軟體（MES/ERP/digital twin/AI copilots）+ 服務（顧問/integration/maintenance）+ 機器人（傳統 + cobot）」四層 stack 的產業。本 ID 把焦點放在「工業端的 AI / automation 整合」（人形機器人 / 工業機器人單獨在 ID_HumanoidIndustrialRobotics）。

💡 Insight §1
1. OT vs IT 邊界正在融合：傳統 PLC 是 OT（工廠機台層）、ERP 是 IT（管理層）；2026-2030 透過 industrial cloud + digital twin 兩層融合，創造「IT/OT convergence」新利潤池 — Siemens / Schneider / ROK 都在卡位。
2. AI 整合「從附加功能變平台核心」：Siemens 20+ industrial copilots + Schneider AVEVA + ROK FactoryTalk Design Studio AI — 「automate automation」是 secular shift。
3. cobot 是工業機器人成長最快子段：傳統機器人 CAGR ~6-8%；cobot ~25%+；中國優必選 / Unitree 在 cobot 段切入快速。

Siemens — 歐洲 + AI 投資領先

Siemens（SIEGY 美 ADR / SIE.DE 法蘭克福）是歐洲工業自動化 + AI 領先者。3 年內投 €1B+ 在 AI 能力上、發佈 20+ industrial copilots。商業模式是「digital + automation 整合銷售」 — Xcelerator 平台 + Simatic PLC + AI copilots 套裝銷售。其優勢是歐洲本土 + 全球化覆蓋，劣勢是多元業務 conglomerate discount。

Schneider Electric（SU.PA）— 配電 + 自動化雙引擎

Schneider 是配電 + 工控雙引擎玩家。AVEVA 收購（2023 完成 100%）後成 industrial software 第三大；與 Microsoft GenAI 戰略合作；Data Center 配電 + 工控雙軌成長強勁。EcoStruxure 平台是其競爭優勢。

Honeywell（HON）— 分拆 catalyst

Honeywell 2026 H2 完成分拆為 3 家公司：① Honeywell Aerospace（航太）；② Honeywell Automation（工業）；③ Honeywell GBS（材料 / quantum）。對於本 ID 最重要的是 Honeywell Automation 將變成 industrial AI pure play（從 conglomerate discount 25-30% → pure play 估值 PE +1 turn）。Experion / Forge / Sentience 三套件聚焦 process industry（油氣 / 化學 / 製藥）。

§9.5 Kill Scenario

→ ROK / Siemens / Schneider EPS -15-20%、股價 -25-30%；訂單 backlog 仍是 buffer；持續性 1-2 季

🟡 反方 3：中國本土工控廠商崛起（Inovance 匯川 / Hollysys）擠壓全球大廠中國市佔 信心：中（25%）
