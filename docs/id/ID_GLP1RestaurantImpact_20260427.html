市場曾用兩種極端框架看 GLP-1 對餐飲：一端是「整段末日」（早期估卡路里 -20-30% → 誤推餐飲 -10%+ 海嘯），另一端是「不會發生」（高停藥率 → 一筆勾銷）。兩個都錯。真相是中度而分散的結構逆風：用戶並沒有不上餐廳——Circana 訂單級資料顯示每趟訂單僅 -1%——他們上餐廳但點得不一樣，少點甜點/麵包/配菜、多點主菜與高蛋白。

所以衝擊不是「全餐飲 -X%」，而是品類暴力分化：同一個 GLP-1 驅動下，CAVA 2026 Q1 同店銷售 +9.7%（牛排 + 客流 +6.8%）、SBUX 北美 +7.1%（七季來首次正成長）、Chipotle +0.5%（打破三季連負），但 Wingstop 國內 -8.7%、Krispy Kreme 營收 -2.2%。受惠端到受壓端的 alpha 差距高達 15 個百分點——這是 stock-picker 的黃金分化，不是 underweight 整段餐飲的理由。

本報告不站隊「GLP-1 對餐飲是大利空 yes/no」。逆風是真的、且 2026-04-01 口服 Foundayo 獲批會溫和加深它；但 alpha 不在「平均」、在「選對品類」——long 結構受惠（高蛋白天然 alignment）+ pivot 兌現者，avoid 無 pivot 路徑的甜食單品與低薪客群裸曝險。判斷重心放在「誰能把逆風轉成菜單 alpha，誰只能挨打」。

PART I · DECISION SUMMARY

決策摘要層

三句話看完 GLP-1 對餐飲的現狀、走向與行動，後接 PM 級監測點。這一層是報告的全部重點——下面各節只是把這一層的每個判斷補上證據。

現在 / NOW

GLP-1 用戶 ~10-11M（2025），仍上餐廳但點得不一樣（Circana 每趟訂單僅 -1%）；品類分化已實證——CAVA Q1'26 +9.7%、SBUX NA +7.1%（七季首正）、CMG +0.5%（破三季連負）、MCD +3.8%，但 WING 國內 -8.7%、DNUT 營收 -2.2%。是中度逆風、非海嘯。

未來 / NEXT

未來 3-5 年逆風溫和加深但仍分散：2026-04-01 口服 Foundayo（orforglipron）獲批拉低 access 門檻、retatrutide 2027 把減重從 -15% 推向 -29%，用戶 base 朝 ~25M（2030）走；但 operator response（42% 業者考慮加 GLP-1-friendly 菜）讓 drag 被 menu pivot 部分對沖。

行動 / ACTION

不要 underweight 整段餐飲——long 結構受惠（CAVA/SHAK/TXRH 高蛋白天然 alignment）+ pivot 兌現者（SBUX/CMG），avoid 無 pivot 路徑的甜食單品（DNUT）與低薪客群裸曝險（WING/WEN/JACK）。買賣由個股 DD + 系統決定。

The Key Call · 本報告最重要的一個判斷

市場曾把 GLP-1 對餐飲想成「海嘯」——但真正的非共識是：這不是全餐飲下沉，而是品類內部的暴力重排。同一個食慾抑制機制，造成 DNUT 受壓、CAVA +9.7%；用戶沒有離開餐廳（每趟訂單只少 -1%），他們把錢從甜點/配菜挪到主菜/高蛋白。所以「業界平均 -1.5% to -3%」是個會誤導的數字——alpha 在 15 pp 的分化裡，不在平均。分歧不在「GLP-1 會不會傷餐飲」（會、但溫和），而在「誰能把它轉成菜單 alpha」。這把選股焦點從「避不避餐飲」移到「選對品類與 pivot 能力」。

三道值得盯的裂縫具體是：①口服拐點——2026-04-01 Foundayo 獲批是第一顆無需注射、無食水限制的 GLP-1 藥丸，把 access 門檻（注射恐懼 + 製造瓶頸）打掉，用戶 base 的成長斜率可能 step-up，溫和加深受壓品類的逆風；②低薪客群入場——CMS BALANCE 把 Medicaid（2026/05）與 Medicare Part D（2026-07，$50/月）覆蓋進 GLP-1，過去「老人/低薪 chain 不受影響」的 buffer 會消失，QSR 與家庭餐廳的裸曝險上升；③pivot 商品化——當高蛋白菜單變成全行業標配（42% 業者已考慮），結構受惠者的 alpha 可能被稀釋（但有 brand permission 的 CAVA/SHAK/TXRH 較難被追上）。盯這三條，比盯任一季 SSS beat 都更早看到 thesis 轉折。

PART II · HISTORICAL ANALOGS

三個歷史類比，框住逆風的真實量級

歷史不必重演，但三個對照各給一個可操作的量化錨：香煙框住「藥理+健康+政策三重夾擊的永久性」、歷次飲食潮流框住「靠意志力的潮流終究 fade」、醫療技術 S 曲線框住「滲透節奏」。

類比一：1990s 香煙警語 + 高稅 → 2000s 消費永久 -40%。價格、健康敘事與政策三重夾擊讓 consumer behavior 永久改變（酒類 1980s peak 後也 -25%）。機制相似——GLP-1 是「藥理 + 健康敘事」雙重，對糖飲/甜點的消費是永久而非暫時改變。但量化差異很關鍵：GLP-1 速度快 3-5 倍（5-10 年 vs 香煙 30 年）、用戶 demographic 更廣（不限癮君子）。這把逆風的「永久性」框住——不會像飲食潮流那樣 fade，但量級遠小於「全餐飲 -10%」的海嘯預期，因為香煙是「整段品類消失」，GLP-1 是「品類內部重排」。

類比二：1980s low-fat / 2000s low-carb / 2010s plant-based 飲食潮流——每波都被預測「永久改變美國飲食」但都證偽（BYND plant-based meat 2025 -78%）。差異在 sticky 程度：前三波靠「自願 + 意志力」（持續率 < 10%），GLP-1 靠「藥理 + 食慾抑制」（行為改變 post-discontinuation 仍保留 40-55%）。所以類比有限——市場若用「歷史飲食潮流終究 fade」一筆勾銷 GLP-1 是錯的（它更 sticky）；但若用它當「海嘯不可避免」的論據也錯（它仍是品類分化、不是整段崩）。

怎麼讀：五項全部把框架從「海嘯」拉回「分化」——海嘯論若整段 underweight 餐飲會錯失受惠端的結構 alpha（CAVA/TXRH），又高估受壓端的崩跌幅度（DNUT 早已 -70%、price in 大半）。真正可操作的是 long/short 品類，不是 underweight 整段。

類比三：1990s 雷射視力矯正 / 2010s mRNA 疫苗——醫療技術改變消費行為。新醫療技術滲透率走 S 曲線（LASIK 1995→2010 達峰）；GLP-1 從 2017（早期 T2D）→ 2025（~11M users）→ 2030（~25M）是經典 S 曲線。關鍵量化錨：2026-04-01 口服 Foundayo 上市 = S 曲線第二段加速器，類似 LASIK 的 femtosecond 技術帶來第二波擴張——它不會讓逆風變海嘯，但會把溫和逆風的斜率往上推一檔，這是受壓品類 5y drag 從 -3% 走向 -4~-5% 的主要驅動。

PART III · SUPPLY SIDE

供給側：菜單 pivot 能力是分水嶺，利潤往高蛋白集中

反直覺但可操作的結論：餐飲的供給側不是「產能」問題，而是「菜單能不能跟著 GLP-1 user 的胃口重排」——pivot 能力（菜單 R&D + 供應鏈 + 品牌許可）才是真正的準入門檻。

供給端的結構是「能 pivot 的 vs 不能 pivot 的」。GLP-1 user 的需求重排（少甜點/配菜、多主菜/高蛋白）對所有 chain 都一樣，差別在誰能把菜單跟著改。高蛋白/健康碗（CAVA/SHAK/TXRH）有天然 alignment——牛排與雞肉碗本來就是高蛋白，幾乎不需 pivot；翻身者（SBUX/CMG）靠主動 menu R&D 補位（CMG 2025/12 High Protein Menu、SBUX protein push）；而 100% indulgent 的甜食單品（DNUT）沒有 pivot 路徑——你無法把甜甜圈變高蛋白。這道「能不能跟著胃口重排菜單」的門檻，把餐飲分成結構受惠、可兌現翻身、與只能挨打三類。

Inference · 供給
pivot 能力（菜單 R&D + 供應鏈 + 品牌許可）是真準入門檻
前提：GLP-1 user 需求重排對全行業一致 + 高蛋白天然 alignment 無需 pivot + 甜食單品無 pivot 路徑 + 42% 業者已動手加 GLP-1-friendly 菜

當需求往高蛋白重排，能提供它的 chain（天然 alignment 或快速 pivot）吃下流入的客流與 check，不能的（甜食/低薪裸曝險）則承受流出。pivot 的前兩重能力（菜單 R&D + 供應鏈）約 12 個月可複製，但第三重「品牌許可」（CAVA 的地中海 DNA、TXRH 的牛排定位、SHAK 的高端定位讓 Good Fit 不像 special diet）需要 5+ 年品牌建構——這是結構受惠者 alpha 較難被追平的原因。

可證偽條件：若 2027-2028 全行業完成 protein pivot 後，CAVA/SHAK SSS 仍能維持 +3-5%（結構 alignment > pivot 商品化），則「pivot 商品化稀釋 alpha」論失效、結構受惠論升級；反之若 CAVA/SHAK 連兩季 SSS 跌至 ≤ +1%，則 alpha 被商品化。

PART IV · DEMAND SIDE

需求側：用戶仍上餐廳但點得不一樣

需求側最被誤解的一點：GLP-1 user 沒有離開餐廳。Cornell-JMR 訂單級資料與 Circana 都顯示，restaurant spending 12 個月後甚至 +0.9%——真正改變的是「點什麼」，不是「去不去」。

需求由兩個事實定義。第一是 用戶沒離開：Cornell-JMR 數萬戶訂單級資料顯示 GLP-1 user restaurant spending 12 個月後淨 +0.9%（先 dip 後恢復），Circana 顯示每趟訂單僅 -1%——用戶在體重穩定後恢復社交飲食。第二是 消費內部重排：fast food / limited-service -8%、savory snacks -10.1%、糖飲與甜點下滑，但 full-service / 高蛋白 fast-casual 持平到正成長。兩個事實合起來＝「品類 share shift」而非「全餐飲淨流失」——這正是 CAVA/SHAK/TXRH 結構 alpha 與 DNUT/WING drag 的共同來源。

Inference · 需求
逆風溫和加深的斜率由口服拐點與低薪入場決定，但仍分散
前提：用戶仍上餐廳（每趟 -1%）+ 內部重排（fast food -8% / full-service 正）+ 口服 Foundayo 2026-04-01 上市 + CMS BALANCE 低薪/老人入場

用戶 base 從 ~11M（2025）走向 ~25M（2030）由兩個加速器推動——口服 Foundayo 打掉注射門檻與製造瓶頸、CMS BALANCE 把低薪（Medicaid 2026/05）與老人（Medicare 2026-07 $50/月）覆蓋進來。這把受壓品類（QSR/甜食/低薪裸曝險）的 5y drag 從 -3% 溫和推向 -4~-5%，但受惠品類同步被更多高蛋白需求加成——逆風加深但分化結構不變。

可證偽條件：① 2028 KFF/IQVIA currently user 仍 ≤ 7%（無加速）→「停藥救援」反方成立、整體 drag 下修；② Cornell follow-up 顯示 user 5y 累積 restaurant spend 轉 -10%+（用戶真的開始離開餐廳）→ 分化論升級為部分海嘯。

為何海嘯派與「不會發生」派都對一半

海嘯派對的是「GLP-1 確實對甜食/低薪 QSR 造成永久逆風」（DNUT/WING 確實受壓），錯的是「全餐飲均勻 -10%」（忽略受惠端與用戶沒離開）。「不會發生」派對的是「高停藥率讓 active base 小於恐慌」（64.8% 12M 停藥屬實），錯的是「所以不要緊」（行為改變 post-discontinuation 仍保留 40-55%、且 base 仍在成長）。兩者可同時為真：逆風是真的且永久（海嘯派對一半），但它溫和而分散、可被 pivot 對沖（「不會發生」派對一半）。所以本報告把判斷重心放在「品類分化」，而非「整段方向」。

PART V · VERDICT & SCENARIOS

裁決與三情境（品類分化）

逆風方向判 INTACT（用戶重排是永久的、口服拐點加深），但「全餐飲海嘯」判 FALSE（用戶沒離開、分化暴力）——故 thesis 結構性看「分化」而非「整段空」，conviction 在 high（分化論本身），低在「平均量級」這個會誤導的指標。

PART VI · TAXONOMY

個股特徵與 conviction tier

本框架只輸出「特徵」與「分類」，買哪一檔由個股 DD 系統與 Pure MA 訊號決定。以下 tier 為產業推論的特徵歸類，提及公司僅為「該特徵的代表案例」，不附帶買賣建議。22 檔全集見 §9 _full 表與 id-meta。

受惠 🟢

CAVA / TXRH

結構受惠端，高蛋白天然 alignment、幾乎不需 pivot。CAVA Q1'26 SSS +9.7%（牛排 + traffic +6.8%）是「同一 GLP-1 機制造 alpha」最純證據；TXRH 牛排 = 天然高蛋白、customers trading up to bigger steaks（customer-led mix，不需 chain initiative）。特徵是「品牌許可護城河 + 結構 alignment」——pivot 商品化最難追平的一類。conviction：高（結構受惠）· 但受惠 narrative 部分 priced
兌現 🔴

SBUX / CMG

主動 pivot 兌現者。SBUX Niccol 翻身教科書——Q2 FY26 NA SSS +7.1%（七季來首次正成長）、protein push 已驗證；CMG Q1'26 +0.5% 破三季連負、High Protein Menu（2025/12）兌現中。特徵是「menu R&D 把逆風轉成 alpha」——但屬 binary（若 traffic 再轉負，市場會重標 GLP-1 為頭號 bear）。conviction：中-高 · 視 pivot sustain
防禦/中性 🟡

DPZ / MCD / DRI / EAT / CAKE

有結構防禦或 value/multi-brand hedge。DPZ「dinner sharing 不受影響」、結構防禦最強 pizza；MCD $5 McValue 救 low-income trade-down（Q1'26 +3.8%）；DRI 多品牌 + Olive Garden 2026 smaller portions + LongHorn 牛排 hedge；EAT value-anchor、CAKE 250-item buffer。特徵是「逆風被 value/廣度/防禦對沖」。conviction：中 · pivot/防禦兌現決定
受壓 🔴

DNUT / WING / WEN / JACK / PZZA

整併逆風的受壓端——無 pivot 路徑（甜食單品 DNUT）或低薪客群裸曝險（WING 國內 -8.7% / WEN / JACK）。DNUT 100% indulgent 無法高蛋白化、過去 12M -70% 已 price in 大半但 fade 未完；WING 持續惡化且無 explicit hedge；PZZA 重整中。特徵是「受壓且難自救」——可能是 short signal 或避開對象。conviction：低（受壓承壓）· DNUT 已大跌、注意 priced-in

估值與 priced-in 總結：本主題的核心張力在「分化已部分被 price、但市場仍用錯指標」。受壓端 DNUT 過去 12M -70%、估值已 price 多數 GLP-1 受害（reverse 推導：-70% 隱含的 5y drag 遠超本 ID 估的 -12~-16%，部分是過度恐慌；但 leverage 高 + 無 pivot 路徑讓 fade 仍未完），WING 持續惡化中尚未見底。受惠端 CAVA/SHAK 受惠 narrative 已部分 priced（CAVA IPO 後高估值），結構 alpha 真實但進場價偏貴。中間防禦 chain（DPZ/MCD/DRI）的 GLP-1 風險相對 mispriced 較少。現處 Phase II（機制驗證 + pivot 展開），thesis 轉折的條件 = 2028 user 停滯（逆風 fade）或 CAVA/SHAK SSS 跌破 +1%（pivot 商品化）。LLY/NVO 雖列 🟢 但是 cross-sector context（直接受惠製藥端），非餐飲 play。

The One Line

GLP-1 沒讓人離開餐廳——它讓人點得不一樣；所以真正的問題不是「餐飲完不完」（不會），而是「誰能把高蛋白的胃口轉成菜單 alpha」。

這份報告刻意不做的三件事

  1. 不告訴你哪一檔該買——受惠/兌現/防禦/受壓是「特徵的代表案例」，買賣由你的個股 DD 與 Pure MA 系統決定。CAVA「結構受惠」不等於「現價值得買」（受惠 narrative 已部分 priced）。
  2. 不用「業界平均 -1.5%~-3%」當操作指標——那是個會誤導的數字，alpha 在 15 pp 的品類分化裡，不在平均；不要 underweight 整段餐飲。
  3. 不假裝知道逆風會多快加深——口服 Foundayo 與 retatrutide 的上量斜率、停藥率的演進，都還有不確定性，已誠實標為 2027-2028 的監測點而非定論。
