GLP-1 對美國餐飲業的長期衝擊（GLP-1 Restaurant Impact）

發布日：2026-04-27 ｜ 涵蓋 ticker：22 檔（9🔴 / 7🟡 / 6🟢）｜ 🟡 占比：19% ｜ 投資時鐘：Phase II（行為機制已驗證 + 2027 oral GLP-1 拐點 + operator response 全面展開） ｜ industry-analyst v1.8

Unit Glossary：GLP-1 = glucagon-like peptide-1 receptor agonist（如 Ozempic / Wegovy semaglutide）；GIP/GLP-1 = dual-agonist（Mounjaro / Zepbound tirzepatide）；triple agonist = GIP + GLP-1 + glucagon（retatrutide，Phase 3）；oral GLP-1 = 口服小分子（orforglipron，Phase 3 已遞交）；BALANCE = CMS 2025/12 公告 model：Medicaid 2026/05 + Medicare Part D 2027/01 覆蓋（$50/月）；discontinuation = 12M 內停藥比例；SSS drag = 5y 累積 GLP-1 isolated SSS impact（vs 反事實）；FAFH = food away from home（餐飲）。所有 financials = Q4 FY25 actual + Q1 FY26 actual（4 月已公告）。

II（機制驗證 + 2027 oral 拐點）
★ 本 ID 核心非共識 Thesis 市場用兩種錯誤框架看 GLP-1 對餐飲：① 「整段末日」（Morgan Stanley 早期 -20-30% 卡路里→誤推 -10%+ 餐飲衝擊）；② 「不會發生 / discontinuation 救援」（GS 2025 「market may prove smaller」一派，認為 64.8% 12M 停藥 = 不要緊）。本 ID 認為：① GLP-1 對餐飲是「品類暴力分化 + 中度 -1.5% to -3% 累積 5y headwind」（Cornell-JMR 訂單級資料證實 user restaurant 12M 後 +0.9% 但結構重排）；② 2027 oral GLP-1 + retatrutide 三重 agonist 雙叉是真拐點（口服把 base 從 25M 推向 40-50M / triple-agonist 把 weight loss 從 -15% → -29%）— 餐飲 stock 仍未 price in 這個 2030+ 真正壓力測試；③ 「protein pivot」是 GLP-1 era 真 alpha — CMG Honey Chicken 公司史上最高 LTO 訂購率、SHAK Good Fit Menu first-mover、SBUX Niccol 翻身教科書、TXRH 牛排天然 alignment（這些 ticker 5y SSS drag 0 to +3% / 結構性受惠）；④ DNUT-style「100% indulgent + 高 leverage + 無 pivot 路徑」是真 short signal（過去 12M -70% 已部分反映，但 fade 仍未完成）。
💡 §0 4 條核心 insight（預告）
1. SBUX Niccol「家庭 9% → 20% 一年翻倍」是公開市場最強 acceleration proof：這數字來自 SBUX CEO 公開發言，意味 GLP-1 已進入主流家庭 — 不是「未來威脅」是「當前現實」。
2. CMS BALANCE Model 是過去 18 個月最大政策催化：2026/05 Medicaid + 2027/01 Medicare $50/月覆蓋 — 解開保險端最後 bottleneck，把預估 2030 user 從 GS bear 15M 推向 JPM bull 30M 中性 22-25M。
3. 「品類暴力分化」是 thesis core，不是「全餐飲 -X%」：DNUT 5y -12-16% / WING -4-6% / CAVA +1-3% / SHAK 0-+1% — 同一驅動下 alpha gap 高達 18-19 pp。stock-picker 黃金時代。
4. 2027 oral + retatrutide 雙叉是真拐點：本 5y（2025-2030）僅暖身；下個 5y（2030-2035）才是真壓力測試 — 餐飲 stock 多重未 price in。

★ 灰色地帶（5 個 boundary call）

  • 「Compounded GLP-1（grey market semaglutide）」算不算 GLP-1 user？：FDA 2024 強化規範後 compounded use 大幅萎縮，但 user 流向 branded 也增 access bottleneck — 本 ID 把 compounded 算入 11M / 22-25M base。
  • SBUX 是受害還是受惠？：糖飲菜單 mix 高 → 受害；但 Niccol 主動 protein pivot 是教科書翻身 → Q1 FY26 NA SSS +4% / traffic +3% 兩年來首次正成長。本 ID 列為 🔴 核心但 beneficiary=true（pivot 已驗證）。
  • CMG 是 winner 還是 loser？：FY25 -1.7% 首次負成長 → 看似受害；但 Honey Chicken 公司史上最高 LTO 訂購率 + High Protein Cup 2026/04 推出 → protein pivot 走得比 SBUX 還深。本 ID 列為 🔴 + 受惠（transition story）。
  • DPZ 是 GLP-1 中性還是受害？：Russell Weiner Q4 2025 explicit 「dinner sharing 不受影響」 + 2026 guide US +3% — 結構性防禦。但 pizza 本身糖 + 碳水高 → 仍有風險。本 ID 列為 🔴 + 受惠（防禦最強的 pizza chain）。
  • Cracker Barrel 是 underexposed 還是 delayed exposure？：早餐重碳水 + Medicare 不保 GLP-1（直到 2027/01 BALANCE 才入）→ 短期 underexposed 但 2027+ 加速 hit。本 ID 評為「2026 中性 → 2028+ -2 to -4%」。
💡 §1 Insight
1. 「GLP-1 對餐飲」邊界本身就是 thesis 的一部分：受害 / 受惠分類取決於「menu pivot 速度」+「客群 demographic」 — 不是「pizza vs salad」這種菜系分類。Pizza Hut 失敗 vs DPZ 防禦差別 = 後者「dinner sharing」結構勝過前者「lunch buffet」。
2. Medicare BALANCE 2027/01 是「老人客群」chain（CBRL / CAKE / Olive Garden）的時間窗終結：50-64 歲 KFF 22% 已 currently，65+ 僅 9%（Medicare 不保）— BALANCE 進來後這個 buffer 消失。

★ 歷史類比（3 個前例）

前例核心 pattern本次相似 / 差異
類比 #1：1990s 香煙警語 + 高稅 → 2000s 香煙消費永久 -40%價格 + 健康 narrative + 政策三重夾擊 → consumer behavior 永久改變；alcohol consumption 1980s peak 後也類似 -25%機制相似：GLP-1 是「藥理 + 健康 narrative」雙重 → 對糖飲 / indulgent 食物消費永久改變。差異：GLP-1 速度更快（5-10 年 vs 香煙 30 年），且 user demographic 更廣（不限癮君子）— 衝擊速度比香煙快 3-5x。
類比 #2：1980s low-fat / 2000s low-carb / 2010s plant-based 飲食潮流每波都被預測「永久改變美國飲食」但都證偽 — McDonald's / Coke / Pepsi 仍佔主導；plant-based meat（BYND）2025 -78%差異重要：前 3 波都靠「自願 + 意志力」（持續率 < 10%）；GLP-1 靠「藥理 + 食慾抑制機制」（持續率 35-50% post-discontinuation 行為保留）。類比有限—GLP-1 比這 3 波都更 sticky。
類比 #3：1990s 雷射視力矯正 / 2010s mRNA 疫苗 — 醫療技術改變消費行為新醫療技術滲透率 S 曲線通常 10-15 年 — LASIK 從 1995 到 2010 達峰值；mRNA COVID 2-3 年完成全民化（特殊事件）高度相似：GLP-1 從 2017（早期）→ 2026（11M users）→ 2030（22-25M）→ 2035（30-35M active） 是經典 S 曲線。差異：GLP-1 oral form 上市 = 第二段加速，類似 LASIK 的 femtosecond technology 帶來第二波擴張。
💡 §2 Insight
1. GLP-1 比歷次飲食潮流（low-fat / low-carb / plant-based）都更 sticky：前 3 波靠意志力（< 10% 持續率）；GLP-1 靠藥理（35-50% 行為保留 post-discontinuation）。市場若用「歷史飲食潮流終究 fade」推 thesis 是錯的。
2. 2027 oral GLP-1 拐點是 LASIK femtosecond 級的第二段加速：base 從 25M 推向 40-50M。餐飲 stock 多重在 2026 估值多數 priced 「current 6%」 base — 對 oral 後 base 翻倍尚未 price in。

💡 §3 Insight
1. 「Cornell user 12M 後 +0.9%」是 thesis 核心反證：market 預期「user 不上 restaurant 了」是錯的 — 真實是「user 上 restaurant 但點不一樣」。這就是 CAVA / SHAK / TXRH 結構性 alpha 來源。
2. 糖反感 + 蛋白偏好是 alpha + drag 的「對偶」：同一機制造成 DNUT -70% / TXRH +6.1%。stockpicker 不是賭「GLP-1 in or out」，是賭「menu pivot 速度」。
3. 行為持久性 40-55% post-discontinuation：意味即使 64.8% 12M 停藥，習慣 shift 不會歸零 — 這是「stock + flow」雙重累積，餐飲業面臨的是 cumulative 結構變化而非 transient shock。

§12 Non-Consensus View（共識差異）

🟡分歧 #1：「GLP-1 = 餐飲末日」是錯的，是「品類分化 + 中度 -1.5% to -3% headwind」信心：高
  • 共識：MS 2024 早期估卡路里 -20-30% → 推「全餐飲 -10%+」末日；Wall Street Journal / Bloomberg 多篇報導 framed 「GLP-1 將 transform restaurant industry」如海嘯
  • 本 ID 看法：Cornell-JMR 訂單級資料顯示卡路里實際 -10-15% / restaurant spending 12M 後 +0.9% / 業界平均 -1.5% to -3% 累積（不是 -10%+）
  • 分歧事實基礎：本 ID §3 表 + §4 量化計算（25M user × $3000 × -5-8% = $3-6B / $1.1T FAFH = -0.5%/年 → 5y -2.5%）
⚠ 證偽條件：2028 年 IQVIA 顯示 active user 達 14%+ + Cornell follow-up 顯示 user 5y 累積 restaurant spend -10%+，本 thesis 部分作廢。
🟡分歧 #2：2027 oral + retatrutide 雙叉是市場低估的 30%+ tail event信心：高
  • 共識：sell-side（GS / MS / JPM）多用 2030 25-30M user base — 但這個 base 假設 oral 上市已 priced；實際多重在「current 6%」base
  • 本 ID 看法：oral 把製造瓶頸消失 + cost 大降，base 翻倍 5 年內；retatrutide weight loss -29% vs Wegovy -15% 是 2x 衝擊；2027 拐點 + BALANCE 老人入局 = 三重催化
  • 分歧事實基礎：Lilly Phase 3 已揭曉（ATTAIN-1 -27.3 lbs / TRIUMPH-4 -71.2 lbs），FDA timing 已 visible — 不是「未來不確定事件」是「2 年內 confirmed」
⚠ 證偽條件：Orforglipron 2026 FDA 失敗 + Retatrutide 2027 Phase 3 readout 1+ 失敗 → 拐點推遲至 2030+，本 thesis 大部分作廢。
🟡分歧 #3：「Protein pivot」是真 alpha 不是 commodity — 至少未來 3 年信心：中
  • 共識：sell-side 認為 SBUX Niccol / CMG Honey Chicken / SHAK Good Fit 都是「temporary marketing」，alpha 會在 12-24 個月被全行業 commoditize
  • 本 ID 看法：protein pivot 需要 3 重 capability — menu R&D（CMG Honey Chicken）+ supply chain（高蛋白成本）+ brand permission（SHAK 高所得 skew 才能標 Good Fit 不是 special diet）。前 2 重 12 個月可複製，第 3 重需要 5+ 年品牌建構（CAVA Mediterranean DNA、TXRH 牛排定位、SHAK 高端定位）
  • 分歧事實基礎：CMG Honey Chicken 公司史上最高 LTO 訂購率（不是 marketing 是消費者真實偏好）；SBUX Q1 FY26 traffic +3% 兩年首次正成長
⚠ 證偽條件：2027 H2 全 mid-tier chains 完成 protein pivot 後，CAVA / SHAK / CMG 結構性 alpha 消失 → CAVA SSS 跌至 ≤ +1% + SHAK SSS 跌至 ≤ +1%（vs current +3-5%），本 thesis 部分作廢。
