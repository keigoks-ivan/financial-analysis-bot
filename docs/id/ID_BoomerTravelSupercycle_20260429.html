II 早中段（supercycle 啟動）
★ 本 Master ID 核心非共識 Thesis 市場用 5 種錯誤框架看嬰兒潮旅遊：① Boomer 旅遊只是「短期 reopen 紅利」即將結束；② Wealth transfer 開始 = Boomer 旅遊支出萎縮；③ K 形分化是循環性（reopen 後會 mean-revert）；④ Cruise / luxury hotel / airline premium 是各自獨立子題；⑤ 嬰兒潮 demographic 是 sell-side 都看到的 consensus，沒有 alpha。本 Master ID 認為：① Boomer 旅遊是 2025-2035 的 10 年「壓縮窗口」demographic supercycle，不是 cycle。Boomer 73M 中 ~50M 仍在 Go-Go phase（age 62-75，2026），健康可用旅遊窗從 2025 起再 10 年；75+ 後降頻 + 短程，整體 spend 下降 15%。這是物理級不可逆的「demographic 壓縮」— 與循環無關，與 Fed / 利率 / GDP 弱相關。② Wealth transfer 啟動反而推升 Boomer 旅遊支出，不是萎縮。$80T 從 Boomer → Gen X / Millennials 是 25 年期（2025-2050），其中 65-75 歲的 Go-Go phase 是「Boomer 自己花掉」的 peak window — 多數 Boomer 已退休、健康仍佳、財力 peak、有時間，這是「use it or lose it」決策心理；wealth transfer 反而強化 Boomer 自己花錢動機（給子女但不能等到自己沒法享用）。③ K 形分化是 demographic 結構性而非循環性。DAL premium revenue +14% / main +1%（差 13pp）是 K 形最直接證據；premium cabin Q1 2025 已逼近 main cabin（差 $41M）— 這個結構在 2030 完全取代 main cabin 主導 airline economics。Boomer + affluent Millennials 雙引擎共振推動 premium leisure 結構性 ceiling。④ Cruise / luxury hotel / airline premium / 機場 concession 不是獨立子題，是一個 demographic 母題的 5 個受惠 vertical — supply 端都已 lock 至 2030+（4 家造船廠 / 寬體機 backlog / 高端 hotel pipeline / 機場長期 concession）= demand 與 supply 結構性錯位 = 持續高 ADR / yield / RevPAR 整體共振 5-10 年。⑤ 「Boomer 旅遊」是 sell-side 已知但持續低估規模與持續性：共識認為 Boomer 旅遊 2027-2028 開始降溫；本 ID 認為 2030-2032 才見頂。差距 3-4 年的 EBITDA flow-through 是現有 RCL / VIK / DAL / H / HLT 估值倍數仍未完全 price in 的 alpha 來源。⑥ 2030+ supply lock 是本 supercycle 與其他 demographic plays 最大差異：cruise（4 家造船廠 backlog 至 2031）+ 寬體機（Airbus A350 / Boeing 787 backlog 至 2030+）+ 高端 hotel（HLT/MAR/H luxury+lifestyle pipeline 5 年）+ 機場 concession（Aena 50 年）= 任何單一 vertical 都無法靠新進場玩家衝擊。⑦ Master ID 的最大實用性是「跨 vertical pair trade」：基於同一 demographic thesis 配置 RCL（cruise）+ DAL（airline premium）+ H（luxury hotel）+ AENA（concession）+ AXP（co-brand）— 這個 5-stock basket 是 2026-2030 最佳 demographic-driven multi-vertical 組合（vs 單一 sector ETF）。
💡 §0 5 條核心 insight（預告）
1. Boomer 旅遊是 2025-2035 的「壓縮窗口」：Go-Go phase（65-75）的人口從 2025 ~50M → 2030 ~55M（peak）→ 2035 ~40M（已部分進入 Slow-Go）；2025-2030 是 peak 5 年，2030-2035 是降頻 5 年，2035+ 進入永久衰退。投資人應該緊密追蹤 2030 inflection — 這是本 thesis 的時間 bound。
2. "Use it or lose it" 心理是 wealth transfer 加速 Boomer 自己花錢的反向證據：估算 Boomer 中 60% 已決定不會把財富全部留給子女（vs 矽谷 / 紐約 affluent 的 estate planning culture）— 這是 Boomer 與前一代（沉默世代）最大行為差異。
3. DAL Amex co-brand $8.2B (+11%) 2025 是 sector 中最高 quality non-RevPAR fee 流：管理層目標 $10B 中期；類比 MAR Bonvoy royalty +35% 但更高量級；本 ID 認為這是 DAL 估值倍數 step-up 從 cycle 8-10x → payment platform 12-15x 的最強 catalyst。
4. VIK 2026 86% booked + $6B advanced bookings 是 demand visibility 範例 — sector 中沒有可比物：相當於 18 個月 forward booked，遠超 hotel chain 12-week / OTA 90-day visibility；意味著 cruise sub-segment 的 thesis confidence 最高。
5. 本 Master ID 的「pair trade basket」（RCL + DAL + H + AENA + AXP）是 2026-2030 最佳 cross-vertical alpha：5 檔基於同一 demographic thesis 但跨 5 個 sub-vertical，相關係數中等（0.4-0.6 historical）— 這是 portfolio diversification + thesis concentration 的最佳配置形式。

§1 母題定義與邊界

嬰兒潮旅遊超級循環（Baby Boomer Travel Supercycle）的範圍是美國 + 歐洲 + 日本嬰兒潮世代（1946-1964 出生，2026 年 62-80 歲）的旅遊與相關高端 leisure 消費，跨 cruise / 高端 hotel / airline premium / 機場 concession / wellness retreat / luxury 自助旅 / 高端 tour 等多個子產業。本 ID 是 master overview — 既有 4 份子議題 ID（HotelChains / LuxuryTravelCruise / OTAandAITravel / GlobalLuxury）+ 1 份待發布（Airline）的整合層；目的是把分散的 sub-vertical 故事 unify 成單一 demographic thesis 框架。

邊界判斷的關鍵是「是否核心受益人口為 Boomer + 高 net worth」與「是否 supply 端有結構性 lock-up」兩條軸：cruise（核心 Boomer + 4 家造船廠 lock）/ luxury hotel（核心 Boomer + pipeline lock）/ airline premium（雙引擎 Boomer + affluent Millennial + 寬體機 lock）/ 機場 concession（boomer 跨大西洋 + 50 年 concession lock）— 全部符合；wellness retreat / 高端 tour package / 高端 cruise 是相鄰 in-scope；mass airline / mass hotel / 經濟型 cruise / RTD 旅遊（Airbnb leisure）是部分 in-scope（boomer 占比較低但仍有曝險）；Las Vegas casino / 賭場 hotel 是邊界爭議（boomer 高重疊但 gambling 不是純 leisure）— 本 ID 列為相鄰簡略提及。

§2 嬰兒潮 demographic 演進史 + 壓縮窗口曲線 + 歷史類比

美國嬰兒潮世代（Baby Boomers，1946-1964 出生）是現代美國消費史中最具經濟影響力的 demographic 集合 — 73M 人口（占美國 22%）、$83.3T 淨資產（占美國家庭淨值 51.4%）、年消費支出 $7T+（占美國消費支出 50%+）。Boomer 經歷了 4 個結構性消費階段：(1) 1946-1980 是「成長期」— 戰後嬰兒潮全國 73M 出生 / 教育消費爆發 / consumer goods 1.0；(2) 1980-2000 是「家庭形成期」— Boomer 在 16-54 歲，房屋 / 教育 / 子女是 main spend；(3) 2000-2020 是「peak earning + 退休準備期」— Boomer 在 36-74 歲，401(k) 累積 + house equity 上升 + medical 消費起飛；(4) 2020-2035 是「退休 + leisure peak 期」 — Boomer 56-89 歲，這是 Go-Go / Slow-Go / No-Go 三階段組合。2025-2030 是 Go-Go peak 5 年，2030-2035 是 Slow-Go 5 年，2035+ 進入 No-Go 結構性衰退。

本次循環最特別的是 「壓縮窗口 + supply lock 」雙因素共振：(a) 需求端壓縮 — Go-Go peak 5 年（2025-2030）的 ~55M Boomer + ~10-15M affluent Millennial 共振 = peak 旅遊需求 vs 歷史；(b) 供給端 lock — cruise / 寬體機 / 高端 hotel pipeline 都 lock 至 2030+ 無法快速擴產。這是 supply / demand 雙重結構性錯位，意味 ADR / yield / RevPAR 在 2025-2030 會持續 stable-to-rising，2030+ 才會見 supply 鬆綁與 demand 降頻同時發生。本 ID 認為 sector EBITDA peak 在 2029-2030，比共識預期的 2027 晚 2-3 年。

★ 歷史類比（2 個正類比 + 1 個反類比）

正類比 1：1980-2000 嬰兒潮第一次消費爆發（家庭 + 房屋 + 教育）。當年 Boomer 在 16-54 歲，driving 家庭 / 房屋 / 教育消費，造就 1980-2000 美國家庭消費 CAGR +6% / housing 大牛市 / 大學教育擴張。量化對照：1980-2000 全美 housing market cap +600% / 教育支出 +500%；本次 2025-2030 旅遊 / luxury / cruise CAGR 預估 +9-12%（5 年），雖然規模較小但增速接近 — 同一 demographic 在不同生命階段的不同消費 pattern。

正類比 2：2010-2020 Boomer 退休準備期 cruise + luxury 起步。當年 Boomer 在 56-66 歲，cruise CAGR +5% / 全球 luxury hotel CAGR +6% / 國際長途 airline 復甦；那是 demographic supercycle 的 setup phase。量化對照：2010-2020 RCL EPS CAGR +18% / Hilton 上市以來累計 +400%；本次 2025-2030 預估 RCL EPS CAGR +20% / Hilton 累計 +60-80%（5 年）— 比上一個 setup phase 更強因為 demographic cohort 已進入 peak window。

反類比：1990-2000 日本嬰兒潮（Dankai 世代）旅遊消費。當年日本 Dankai 世代（1947-1949 出生，~6.7M 人口）退休後本應推動日本 luxury 旅遊 boom，但實際被 1990-1995 房地產泡沫 / 資產通縮壓制；2000 年後 Dankai 旅遊消費長期低迷。反類比警示：若美國 2026-2030 經歷類似資產通縮（housing -30%+ / 401(k) -40%），Boomer 旅遊消費可能突然萎縮 — 是本 thesis 最大下檔風險。多條件對照：日本當年 (a) 房地產泡沫破裂直接衝擊 70%+ Boomer 資產，(b) 通縮持續 20+ 年，(c) demographic 規模較小（6.7M vs 美國 73M）。美國目前 (a) housing 仍處 cycle 中後段但無明顯泡沫，(b) Fed 政策有相對 flexibility，(c) Boomer 規模 11x — 多條件不完全成立，但反類比警示仍有效，特別是 2026-2027 美國 housing 是否進入下行 cycle。

💡 §2 Insight
1. 本 thesis 的時間 bound 是 2030（Go-Go peak）+ 2035（Slow-Go 結束）：2026-2030 是加速期 buy-and-hold；2030 開始需要 monitor demographic decay；2035+ 應該主動 reposition。投資 horizon 應該 align demographic curve 而非單純 cycle 概念。
2. 日本 Dankai 反類比警示是最大下檔風險 — 監控指標是 美國 housing + 401(k) drawdown：若 2026-2027 美國 housing market 累計下降 -15%+ 連 4 季 OR 401(k) median balance 下降 -25%+，Boomer 旅遊消費可能急速萎縮；這是本 thesis 第一級 falsification metric（§13）。

§9.5 Kill Scenario / Steel-man 反方

反方論證 #1：美國 housing 2026-2027 -20%+ 大幅下行（Dankai 反類比觸發）

路徑：(a) Fed 2026-2027 結構性升息壓制 housing；(b) inventory 供給上升 + 空房率上升；(c) 401k cohort 退休前 mass de-risking → 餘 cohort 心理 fear。Boomer 65-75% 淨值在 housing + 401(k)，若兩者同時下行 -20%+，旅遊支出可能 sudden contract -30-40%。當前證據：(a) 美國 housing inventory 仍緊缺；(b) 401(k) median balance 在 historic high — 但 (c) Boomer cohort 在 retirement 前 5 年的 risk averse 上升是真實事實。量化影響：sector revenue -25-35% / RCL 股價 -40-50% / DAL premium revenue -20% / 估值倍數 sector 整體 -3-5x。falsification 事件：2026-Q3 Q4 Case-Shiller -10% YoY 連 2 季 OR 401k median balance -25%+。最壞情境 drawdown：5 檔 basket -35-45%。

反方論證 #2：Boomer 旅遊 demographic 已 plateau，2027-2028 開始降溫（vs 本 ID 認為 2030 才頂）

路徑：Boomer Go-Go 65-75 phase 在 2025 已開始 plateau（Boomer 1946-1949 出生已 76-79 歲，正進入 Slow-Go）；本 ID 過度延伸 demographic peak 至 2030。當前證據：(a) 2024-2025 cruise / luxury hotel 數據仍加速 — 反方證據；但 (b) 1946-1949 Boomer 已 76-79 歲，是 Slow-Go cohort —反方證據。量化影響：sector EBITDA peak 提前 2 年 → RCL / VIK / DAL 估值倍數 -15-20%。falsification 事件：2026-2027 cruise nights / luxury hotel ADR 連 2 季 negative growth。最壞情境 drawdown：5 檔 basket -15-20%。

反方論證 #3：Affluent Millennial 不接棒 — 2030+ demand cliff

路徑：Boomer 2030+ 進入 Slow-Go，但 Affluent Millennial 規模太小（前 1% 約 ~10M）+ FIRE 運動傾向反消費 + 不像 Boomer 把 luxury travel 當 status symbol；2030+ 沒有接棒 cohort，sector demand cliff 顯著。當前證據：(a) Affluent Millennial 規模仍小；(b) Gen X 接棒 cohort 在 60-70 歲（2030+）但人口少（~65M vs Boomer 73M）；但 (c) Affluent Millennial wealth 增長中（科技股 + 房產）。量化影響：2030+ sector growth 從 +9% → +3-5% / 估值倍數 -2-3x。falsification 事件：2030-2031 Millennial 高端 cohort 資產分布或 luxury travel 統計 — 是長期 monitoring metric。最壞情境 drawdown：2030+ -25-30%。

💡 §9 + §9.5 Insight
1. 3 條反方論證中，#1（housing 大幅下行）是 base rate 最高且影響最大（25-35% 機率 × -35-45% drawdown）；應該緊密追蹤 Case-Shiller + 401k median balance。
2. #2（提前降溫）和 #3（無接棒）是長期 monitoring metrics，不是 short-term traders；本 ID 的 2030 demographic peak 假設應該每年 review。
