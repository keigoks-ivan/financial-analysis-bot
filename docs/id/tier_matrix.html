📐 Framework v3.1 — 結構凍結、數據活水
產業 DD（ID）描述產業 framework，不替個股下投資結論（§11 玩家矩陣的 🔴🟡🟢 是「產業曝險深度」非「投資建議」）。本頁把跨 ID 累積的「個股投資 conviction」獨立出來，做兩軸手評 + 四 Tier 分級，並把不同半衰期的資訊分層——讓頁面可放 60-180 天不腐爛：

慢層（手寫·凍結） Tier 歸屬＋結構性理由（賣鏟子還是賣 seat、守門人還是挑戰者）、以下兩軸手評分數、核心 ID/DD 連結、證偽條件。60 天只複審這一層。
快層（前端 live 注入） 訊號、裁決、live Fwd PE、5Y 分位＋估值燈、護城河趨勢、Priced 軸——載入時同源 fetch dd-screener latest.json 即時填入，永遠是最新一次快照。

兩條手評軸（1-5；數字越大 = 風險越高，沿用 v3.0 定義）：
• AI 顛覆（自家 AI 替代核心業務 + 競對 AI 直擊 use case 的淨曝險。1=AI 是需求 driver / 5=核心被替代路徑明確）
• 週期（接 AIComputeCapexCycle 框架：capex digestion / 產業自身週期反轉時，這檔在「見血順序」的位置。1=需求結構性無週期 / 5=cycle 反轉時最先且最重跌 — 上游晶片、memory、WFE 高分；買方 hyperscaler 低分）

一條 live 軸：
• Priced（快層，不再人工打分）：由 JS 從 screener 的 live 5Y PE 分位自動映射為圓點——<20→1、20-40→2、40-70→3、70-90→4、≥90→5；screener 未覆蓋者顯示「—」。

四 Tier 分級邏輯（Tier 反映結構性 conviction，不是 entry timing — timing 看 live 欄的訊號燈與 Priced）：
• Tier 1 真 Moat-core：不可替代的 infra / 矽片 / 平台 / 稀缺資產 + AI 顛覆免疫或受益；三軌組合的核心持倉池（複利軌；Priced 高低不影響 Tier，只影響 entry timing）
• Tier 2 Conviction Long：結構強 + execution 強，但已 priced 或 thesis 已獲數據驗證但結構強度未達 moat-core；衛星結構級配置
• Tier 3 Thesis-pure / 週期觀望：押 thesis 反轉 / 週期 timing / turnaround；嚴守對應 ID §13 證偽條件——含衛星循環（QC-42 循環鏡頭）候選；不該無腦 Long
• Tier 4 Avoid / 結構受損：增速、競對、估值多軸不利，或結構性受害 / 高槓桿週期溫度計；除非重大 catalyst 反轉，不該 Long
🧭 三頁分工：本頁不是唯一入口，與另兩頁協同——
• 精選清單 /picks/＝可執行短名單（GRP 排序、席位人工治理，動作快）；
• Tier Matrix（本頁）＝全覆蓋 conviction 底圖（慢層 60 天只複審 Tier 與證偽觸發，快層數據每次載入即刷新）；
• dd-screener＝live 數據源（本頁快層即取自它）。
先在本頁定位一檔的結構 conviction，再去 picks 看它今天能不能 act。

週期定位：Phase II（build-out 中後段，late-build-out 早期裂縫）— 非過剩週期的頂。12M 供需 = 短缺（GPU lead time 36-52 週、CoWoS full allocation 至 mid-2027、H100 一年租約價半年反漲 +40%）；3Y = 平衡偏緊但變現缺口同步擴大（上游晶片營收 ~$450-500B vs 已驗證 AI app 營收 ~$55-70B ≈ 7-9x）。本輪的 kill 是 financial 過剩（變現追不上 + 2025-09 起集體轉外部發債，2026 AI 新債估 ~$570B），不是 physical 過剩。估值結構與 2000 年相反：「倍數安全、E 危險」 — 四大 hyperscaler 倍數都在歷史 band 中下緣，樂觀全部 price 進上游晶片商的 E（NVDA FY27 consensus 建基 capex 100% 兌現）。bear 觸發時殺傷力集中在上游晶片股（高週期分）而非 hyperscaler。

📌 重點觀察（v3.1）

觀察意義
「倍數安全、E 危險」仍是最重要的單一句子與 2000 年「倍數瘋狂」相反：NVDA 與四大 hyperscaler 倍數都在歷史 band 中下緣，樂觀全部藏在上游晶片商的 E（capex 兌現假設）。所以週期軸高分集中在上游（NVDA/AMD/WFE/memory），低分在買方（MSFT/GOOGL）——digestion 來時殺傷不對稱。
結構凍結、數據活水：這一版把腐爛面外包了角色欄與 Tier 是慢層（60 天複審）；訊號／裁決／估值／Priced 是快層（screener 每次載入即刷新）；近期 binary 日曆移到 /catalyst/。頁面因此可放 60-180 天不腐——過期的只會是點進去的 DD，不是這張底圖。
Tier 1 核心池維持 15 檔——只留瓶頸守門人與真稀缺資產賣鏟子／賣 seat、守門人／挑戰者是這張表的分層主軸。AAPL（大腦外包 Gemini）、VRT/ETN（gap closed／非獨家 moat）、NET（被 PANW 正面攻擊）都不在核心池——結構理由而非估值。
APH 入 T2＝AI interconnect 的純鏟子股全球連接器＃1、decentralized M&A 飛輪＋NVDA 直供、A 級護城河；6/23 與 6/26 DD 皆裁決進場、picks 正式榜長熬五檔之一。與 CRDO/MRVL 並列 networking 組，但護城河寬度更接近守門人。
UBER 升 T2＝v3.0 降級理由已被推翻v3.0 列 T3 的理由是「訊號 X 為 timing」；7/03 DD 裁決進場、訊號 A，AV 平台聚合者（不押單一技術路線）的結構論點沒變、timing 障礙消失——升入衛星結構。
