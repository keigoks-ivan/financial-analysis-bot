📐 Framework — 為何要這個獨立頁？
產業 DD（ID）描述產業 framework，不替個股下投資結論（§11 玩家矩陣的 🔴🟡🟢 是「產業曝險深度」非「投資建議」）。本頁把跨 ID 累積的「個股投資 conviction」獨立出來，做三軸打分 + 四 Tier 分級：

三軸打分（1-5；數字越大 = 風險越高）：
• AI 自我蠶食風險（1=幾乎無 / 5=核心被自家 AI 替代）
• 競對 AI 直擊威脅（1=幾乎無 / 5=直接 use case 被取代）
• 估值 reset 度（1=已大幅 reset 便宜 / 5=仍 priced 樂觀情境）

四 Tier 分級邏輯：
• Tier 1 真 Moat-core：不可被替代的 infra / 矽片 / 平台 / 稀缺資產 + AI 顛覆免疫；可作組合核心持倉（reset 分數高低不影響 Tier，只影響當前 entry timing）
• Tier 2 Conviction Long but Priced：strong moat + execution 強，但 PE 已 priced；要追蹤執行不偏，satellite 級配置
• Tier 3 Thesis-pure / Reset 觀望：押 thesis 反轉 / 估值 reset；嚴守對應 ID 的 §13 證偽條件，不該無腦 Long
• Tier 4 Avoid 重新評估：增速、競對、估值三軸全不利；除非有重大 catalyst 反轉
