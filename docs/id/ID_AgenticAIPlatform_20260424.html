市場談「Agentic AI」最容易犯的錯，是把「LLM」「Agent」「Copilot」「基礎設施」攪成一團、當成一個賭注。其實它有四個截然不同的層次：底層的 frontier model（LLM）、把 model 組裝成自主流程的 agent 平台、嵌進業務情境的應用/copilot、以及讓 agent 連工具與彼此的協議/網路基礎設施。每一層的贏家邏輯、護城河來源、估值倍數都不同——把它們混為一談，就會買錯層。

2026 年的關鍵變化是：Agentic AI 從 hype 進入商業化證明年。Salesforce Agentforce 累計 29,000 筆交易、ARR 衝到 $800M、累計逾 24 億 agentic work units（AWU、單季 +57%）、客戶平均 ROI ~171%（第三方研究）；Microsoft Copilot Studio 已有 逾十萬個組織（第三方估）、40 萬以上自建 agent；Palantir AIP 把營收推到單季 $1.63B、+85% YoY。這些不是 demo，是付費的營收。

但同一年，Gartner 對 3,400 家投資中的企業民調後預測：逾 40% 的 agentic AI 專案會在 2027 年底前被取消（成本失控、價值不明、治理不足）；它還點名「agent washing」——數千家號稱 agentic 的廠商裡，只有約 130 家提供真正的自主能力。所以本報告不押「Agentic AI 會不會贏」（會），而是拆解四層棧裡誰捕獲價值、誰會在 40% 夭折潮裡活下來。

PART I · DECISION SUMMARY

決策摘要層

三句話看完 agentic 棧的現狀、走向與行動，後接 PM 級監測點。這一層是報告的全部重點——下面各節只是把這一層的每個判斷補上證據。

現在 / NOW

Agentic AI 從 hype 進入商業化證明年——CRM Agentforce 累計 29,000 筆、ARR $800M、累計逾 24 億 AWU、平均 ROI ~171%（第三方研究）；MSFT Copilot Studio 逾十萬組織（第三方估）、40 萬+ 自建 agent；PLTR AIP Q1'26 營收 $1.63B +85%。但 Gartner 估 >40% agentic 專案 2027 前取消。

未來 / NEXT

3–5 年價值往應用層（專屬資料+workflow+分發）+ 基礎設施層（MCP 協議/agent 網路）兩端集中；LLM 層商品化、純 agent 工具被 agent washing 淘汰（~130/數千家才是真 agent）。MCP（Anthropic 發起、已捐 Linux Foundation）成 agent-to-tool 標準。

行動 / ACTION

押有專屬資料+workflow+分發的應用層 CRM/NOW/PLTR/MSFT、開放生態 GOOGL；基礎設施層盯 NET（agent 網路）+ MCP 標準化；理解 40% 專案會夭折、選「真 agent + 有 ROI 證明」者。

The Key Call · 本報告最重要的一個判斷

市場把「Agentic AI」當一個賭注，真正的非共識是四層棧的價值捕獲——底層 LLM 正被商品化（多模型壓價）、純 agent 工具淹沒在 agent washing（數千家號稱、僅 ~130 家是真 agent）、Gartner 估 40% 專案 2027 前取消；真正握護城河的是「有專屬資料 + workflow 嵌入 + 分發的應用層」（CRM/NOW/PLTR）與「協議/網路基礎設施層」（MCP/NET）。商業化已過 hype、進入證明年（CRM $800M ARR、PLTR +85%、MSFT 40 萬 agent），但 governance/ROI 缺口讓「誰活下來、估值能否撐住」未定——這才是 agentic 的核心賭注。

三道裂縫具體是：①agent washing——數千家號稱 agentic、僅約 130 家提供真自主能力，純工具型新創會在洗牌中被淘汰或併購；②40% 專案夭折——Gartner 估逾 40% agentic 專案 2027 前因成本/價值/治理被取消，需求是真的、但落地門檻高；③LLM 層商品化——底層模型被多家壓價，價值往「應用層（資料/workflow）+ 基礎設施層（協議/網路）」兩端集中，中間純 agent 工具被擠。盯這三條，比盯「Agentic AI 火不火」更能看清錢的流向。

來源結構說明

公司級數字（CRM Agentforce ARR/deals/ROI、MSFT Copilot Studio 組織數、PLTR 營收）引一手法說/IR T1；市場規模與 CAGR 各家分歧大（automation 口徑 $9.89B→$167B ~42% vs Grand View enterprise $24.5B by 2030 ~46%），以 range 呈現並標第三方 tier。Gartner 40% 取消為調查預測（3,400 家民調）。OpenAI/Anthropic 為私有公司、無上市 ticker。機率用「很可能 / 可能 / 不太可能」非百分比。

PART II · HISTORICAL ANALOGS

SaaS 上移與「貼名泡沫」，框住多空

歷史不必重演，但兩個對照各給一個教訓：SaaS 取代 on-prem 給出「握資料+workflow 者贏」的 bull 底，「貼名泡沫」（.com / blockchain）給出 agent washing 的 bear 天花板。

類比一：SaaS 取代 on-prem——握資料 + workflow + 分發者贏。上一個平台換代（雲端 SaaS 取代套裝軟體），贏家不是「技術最炫的」，而是「離客戶最近、握住資料與工作流」的應用層（Salesforce、ServiceNow 由此長成數百億美元巨頭）。agentic 正重演：agent 要有用，必須接到企業的專屬資料與既有工作流——這正是 CRM（Agentforce 嵌 CRM 資料）、NOW（嵌 IT/企業流程）、PLTR（嵌作戰/營運資料）守得住的原因。量化錨：Agentforce 上線 6 個月做到 ARR $800M、平均 ROI ~171%（第三方研究），是「應用層握資料 = 真護城河」的早期證據。

兩個對照 · agentic 像 SaaS 上移還是貼名泡沫
對照SaaS 取代 on-prem（應用層贏）貼名泡沫（.com / blockchain）agentic 像哪個？
價值來源握資料 + workflow + 分發蹭題材、無實質應用層握資料（像 SaaS）
贏家CRM / NOW 長成巨頭泡沫後 <50% 存活CRM $800M ARR（像 SaaS）
雜訊少數真 SaaS無數「.com」貼名~130/數千真 agent（像泡沫）
淘汰機制不黏者出局泡沫破、貼名歸零40% 專案取消（兩者皆有）
結論應用層可獨立成巨頭純題材會歸零真 agent 像 SaaS、agent washing 像泡沫

怎麼讀：agentic 同時有「SaaS 上移」的真（應用層握資料的 CRM/NOW/PLTR）與「貼名泡沫」的假（agent washing、~130/數千家才真）。bull 的底是真 agent 的商業化證明，bear 不是「agentic 不重要」（重要），而是「純題材/純工具被 40% 夭折潮洗掉、估值溢價收斂」。

類比二：貼名泡沫——agent washing 的歷史鏡像。1999–2000 年無數公司在名字加「.com」估值即翻倍，泡沫後僅少數活下；2017 年 Long Island Iced Tea 改名「Long Blockchain」股價單日 +289%（後遭下市）。今天的「agent washing」是同一齣戲：數千家把舊聊天機器人/自動化貼上「agentic」標籤，Gartner 估只有約 130 家是真的。教訓對 bear 含義很直接：分辨「真 agent（有自主性 + ROI 證明）」與「貼名」是這主題能不能賺錢的關鍵；買「貼名」者在 40% 夭折潮裡會虧。判別訊號：看是否有可量化的 ROI（如 Agentforce 171%）、是否嵌入專屬資料、專案存活率。

PART III · SUPPLY SIDE

供給側：四層價值鏈，利潤往兩端集中

反直覺但可操作的結論：agentic 棧的利潤往「兩端」集中——握客戶資料+workflow+分發的應用層（一端）+ 協議/網路的基礎設施層（另一端），中間的 LLM 商品化、純 agent 工具被擠。

供給端是四層：①LLM 模型層（OpenAI/Anthropic/Google，被多模型壓價、商品化中）；②agent 平台層（組裝 model 成自主流程——MSFT Copilot Studio、Google Vertex、AWS Bedrock Agents、PLTR AIP）；③應用/copilot 層（嵌業務情境——CRM Agentforce、NOW、WDAY、INTU、ADBE）；④基礎設施層（agent 連工具與彼此——MCP 協議、NET agent 網路、身份/治理）。利潤往「應用層（握專屬資料+workflow+分發）+ 基礎設施層（協議/網路收費站）」兩端集中；LLM 層被壓價、純 agent 工具淹沒在 agent washing。

Inference · 供給
LLM 商品化、純工具被淘汰，利潤往應用與基礎設施兩端
前提：LLM 多模型壓價（商品化）+ 純 agent 工具易被平台內建/agent washing 淹沒 + 應用層握專屬資料與 workflow（高轉換成本）+ 基礎設施層握協議/網路（標準鎖定）→ 利潤往「應用端 + 基礎設施端」集中，中間被擠

當「呼叫一個 model」變成可替換的商品、當「做一個 demo agent」變成人人能貼名，價值就不在這兩處。賺得到溢價的是：握客戶專屬資料 + 既有工作流 + 分發通路的應用層（CRM/NOW/PLTR——高轉換成本），與握協議標準 + agent 網路的基礎設施層（MCP/NET——標準鎖定）。CRM Agentforce 的 $800M ARR 與 171% ROI 證明應用層能把 agentic 重新定價。

可證偽條件：若某純 agent 工具型新創用「AI-native 平台」獨立長成大廠（不被併、自建分發），或開源 agent 框架讓應用層的資料/workflow 優勢被抹平，則「兩端集中」論的某一面失效。

PART IV · DEMAND SIDE

需求側：商業化證明 vs 40% 夭折的治理缺口

需求最剛性的是「把人力流程交給 agent、有可量化 ROI」——CRM/PLTR 的客戶 ROI 證明需求是真的；但最大的變數是「40% 專案因治理/成本/價值夭折」，需求真實但落地門檻高。

第一引擎：商業化證明（現需）。需求已從 demo 變成付費營收：Agentforce 累計 29,000 筆、ARR $800M、平均 ROI ~171%（第三方研究）；Copilot Studio 逾十萬組織（第三方估）、40 萬+ 自建 agent；Gartner 預測 2026 年底前 40% 企業應用會內建任務型 AI agent。企業要的是「可量化的 ROI + 嵌進既有工作流」。第二引擎：治理缺口（反需求/篩選器）。同時 Gartner 估逾 40% agentic 專案 2027 前被取消——主因不是技術不行，而是「沒有治理結構去管 agent 出錯時的後果」、成本失控、價值不明。這個治理缺口既篩掉貼名者、也創造「agent governance/observability」的新需求（與子題「LLM 廠商資安經濟學」重疊）。

怎麼讀：商業化營收（CRM $800M ARR、PLTR +85%）證明需求真實、且應用層握資料者吃得到；但市場規模估計分歧極大（42% vs 46%、automation vs enterprise 口徑不同），且 40% 專案會夭折——所以「需求是真的」與「多數專案會失敗」可同時成立。採信「有 ROI 證明的應用層營收」作需求真實性的最硬證據，而非市場 TAM 估計。

多空為何都對

多頭測的是商業化需求（CRM $800M ARR、PLTR ROI、Gartner 40% apps by 2026），這幾乎沒爭議；空頭測的是落地與分化（40% 專案夭折、agent washing、估值已 priced 完美執行）。兩者可同時為真：商業化是真增長（多頭對），但多數專案會失敗、贏家高度集中於少數握資料/分發者（空頭對）。本報告因此把重心放在「四層棧裡誰捕獲價值、誰活下來」，而非「Agentic AI 會不會成長」。

PART V · VERDICT & SCENARIOS

裁決與三情境（24–36M）

「商業化是真增長」判 INTACT（CRM $800M ARR、PLTR +85%），「四層分化、價值往兩端」判 INTACT，但「誰活下來 + 高估值能否撐住」判 AT_RISK（40% 夭折 + near-perfect execution 已 priced）——故結構看多 agentic、conviction 卡 mid。如硬要對應傳統框架：對「能落地、有 ROI 的真 agent」的需求遠大於「合格供給」（~130/數千家真 agent），等價於結構性短缺型，稀缺來自 execution/governance 門檻而非產能。

PART VI · TAXONOMY

個股特徵與 conviction tier

本框架只輸出「特徵」與「分類」，買哪一檔由個股 DD 系統與 Pure MA 訊號決定。以下 tier 為產業推論的特徵歸類，提及公司僅為「該特徵的代表案例」，不附帶買賣建議。

CORE 🔴

PLTR

agent 作業系統本體型（AIP/AOP）——Q1'26 營收 $1.63B +85%、Rule of 40 145%（Q1'26：85% 成長 + 60% adj op margin；與 NVDA/Micron/SK hynix 並列 AI 基建頂尖）。特徵是「最純的 agentic 平台 play」，但估值最極端（near-perfect execution 已 priced），需個股盯增速續航與估值。conviction：高（賭 agentic OS）· 估值風險最大
CORE 🔴

CRM

Agentforce 商業化規模最大——29,000 deals、ARR $800M、累計逾 24 億 AWU、平均 ROI ~171%（第三方研究）。特徵是「應用層握 CRM 專屬資料 + workflow」的代表，商業化證明最硬的一檔。conviction：中高 · 視 ARR 續成長
CORE 🔴

MSFT / NOW / GOOGL

MSFT：Copilot Studio 分發王（逾十萬組織（第三方估）、40 萬+ agent）+ 自有 model。NOW：IT/企業 workflow 深 moat、#1 Gartner Critical Capabilities 2025。GOOGL：Gemini + 開放生態（MCP/A2A）、自有 model 不付外部。特徵是「分發 + 專屬 workflow + 自有 model」的結構占優者。conviction：高（分發/workflow 護城河）
EDGE 🟡

NET / WDAY / INTU / ADBE / TEAM

NET = 基礎設施層真贏家（Workers + MCP host，「agents are new users of the web」）；WDAY/INTU/ADBE/TEAM = 各自握 regulated/專屬資料的垂直 agent（HR/稅務/創意/dev workflow）。特徵是「協議網路 + 垂直專屬資料」，第二層受益者。conviction：中 · 各看垂直放量

估值面總結：本主題的核心張力在「商業化是真增長（CRM $800M ARR、PLTR +85% 證明）vs 誰活下來+估值未定（40% 夭折 + near-perfect execution 已 priced）」。應用層握資料者（CRM/NOW/PLTR）與分發/自有 model 者（MSFT/GOOGL）是最高純度；NET 是基礎設施層;受害方是被擠壓的單點工具（HUBS/ZM）。現處 Phase II（商業化證明年），轉空條件 = 40% 夭折擴大 或 應用層 ARR 增速放緩 + 高估值 de-rate；de-rating window 在「市場確認 agentic ROI 普遍落空」那一刻。OpenAI/Anthropic 為私有、不可直接投資。買賣由個股 DD + Pure MA 系統決定。

The One Line

Agentic AI 不是一個賭注、是四層棧的價值捕獲——商業化已過 hype（CRM $800M ARR、PLTR +85%），真正的賭注是握資料+workflow+分發者能否在 40% 夭折潮裡活成贏家。

這份報告刻意不做的三件事

  1. 不把「Agentic AI」當一個賭注——明確拆成四層，每層贏家邏輯/護城河/估值都不同。
  2. 不把商業化證明當無腦利多——同時標出 40% 專案會夭折、agent washing 充斥，分辨真假是賺錢關鍵。
  3. 不告訴你哪一檔該買——CORE/EDGE 是「特徵的代表案例」，買賣由你的個股 DD 與 Pure MA 系統決定。
