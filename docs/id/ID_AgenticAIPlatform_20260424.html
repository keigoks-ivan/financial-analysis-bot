★ 本 ID 核心非共識 Thesis（3 條） ① Agentic AI 真正贏家不是 LLM 原廠（Anthropic / OpenAI），是 Horizontal 平台的 distribution + Vertical SaaS 的 regulated data moat：CRM Agentforce $800M ARR +169%、PLTR US comm +137% YoY、NOW Now Assist ACV >$600M（FY26 目標 $1B）、MSFT AI run rate $13B — 這四家 2026 agentic revenue 可驗證且每季加速，是市場「看到但未充分 price in」的結構 tailwind。LLM 原廠 $30B(Anthropic) + $25B(OpenAI) run rate 裡資安 vertical < 3%、企業應用 vertical 也類似 — 真正收企業 SaaS ARR 的還是 Horizontal + Vertical SaaS；
② Framework 層（LangChain $1.25B 估值 vs $12-16M ARR = 75-100x multiple）已結構性商品化，真贏家是 Agent Infra（Cloudflare Code Mode 壓 99.9% token / $42.5M ARR 單合約）：Anthropic 2025-12 捐 MCP 給 Linux Foundation 是放棄鎖定主動升級為「開放協議 + 付費 SDK / Claude Code $2.5B ARR」三層堆疊；LangChain Series B $1.25B 估值是在 price-in「agent engineering platform」未來而非當下 revenue，風險極大。**投資應避 framework，押 infra（NET 最純）+ 協議受益（OKTA XAA 已覆蓋於 M11）**；
③ Vertical SaaS 被 Horizontal 吞食是分層勝負，不是全滅 — regulated data + workflow schema 是唯一可守護城河：Intuit（$100M OpenAI 合約、3M+ AI 用戶、稅務 regulated data）、Workday（Illuminate 12 agents、75% 核心客戶採用、payroll ledger 不可取代）、ServiceNow（IT CMDB 深 workflow）= 最抗 Agentforce / Copilot 吞食者；反面 HubSpot 2026-04 被迫從 $1 降至 $0.5/resolved conversation、Zoom 推 $10 AI Companion standalone 搶 user、Twilio CPaaS 被 Voice AI-first 玩家侵蝕 = 已開始被擠壓。**投資分層**：🔴 INTU/WDAY/NOW/DOCU 安全；🟡 TEAM/ADBE 中性；🟢 HUBS/ZM/TWLO/APPF 高被吞食風險（2027-2028 可能 PE 化或併購）。
💡 本 ID 3 條最高價值 insight（預告）
1. PLTR 65-80x EV/Sales 估值已 price-in FY28 +40% CAGR continuation：任一季 US commercial < 100% YoY 都會壓縮 multiple；ontology + FedRAMP + Maven $1.3B ceiling 組成 regulated moat 是 CRM/MSFT/GOOGL 短期不可複製，但估值容錯極低

★ 歷史類比（3 個前例）

前例議題贏家 / 輸家多空誰錯本次異同
1. 1995-2005 ERP 雲端轉型Salesforce.com 取代 Siebel / Oracle CRM贏：CRM Salesforce（SaaS 顛覆）
輸：Siebel（被 Oracle 2005 收 $5.85B）		空頭低估 SaaS 取代速度；多頭低估 Siebel 客戶 sticky同：Horizontal 平台（Agentforce）顛覆 vertical legacy（類比 Siebel）；異：Vertical SaaS 如 WDAY/NOW 已是 SaaS-native 護城河，非 on-prem；數字：CRM 2010 市佔從 0 → 2020 20%；Agentforce 類似軌跡
2. 2015-2022 Salesforce Einstein / SAP Leonardo / Oracle AI第一代 AI copilot 為何失敗？贏：無（都被重新定位或放棄）
輸：SAP Leonardo 2022 放棄；Einstein 1.0 重新包裝成 Einstein Copilot		多頭低估「人工智慧從 ML → LLM」的典範轉移；空頭則低估 ChatGPT 2022 重啟動能異：這次 GPT-4 / Claude 3.5 基礎能力遠勝 2018 Einstein；同：70% human-in-the-loop 限制仍適用；數字：Einstein 2018-2022 無公開 ARR；Agentforce 2025-2026 ARR $800M +169% 是量級不同
3. 2008-2018 雲端 infra（AWS 崛起）vs SaaS 應用基礎設施廠會不會吃應用層？贏：Snowflake / MongoDB / Databricks 獨立
輸：被 AWS 直接複製的新創		空頭誇大 AWS 吞食；多頭忽略 data gravity moat同：Anthropic 類 AWS 策略 push primitives（MCP / Agent SDK）讓上層應用繁榮而非自吃；Claude Code 是 exception（$2.5B ARR 直接應用）；異：Amazon $25B + Google 14% 結構限制 Anthropic vertical；數字：AWS 2015-2020 未完全吃 SaaS，平均 vertical 仍以 30%+ CAGR 成長
⚠ 反向類比警示：類比 1（Siebel）常被用來預測「所有 vertical SaaS 會被 Agentforce 殺」。量化差異條件不成立：(1) NOW/WDAY/INTU 擁有 regulated data + workflow schema（Siebel 沒有）；(2) CRM 本身是 vertical SaaS 而非通用 database；(3) MSFT M365 distribution 與 CRM Data Cloud 雙龍頭格局 ≠ 2000 年 Siebel 單一玩家；(4) Outcome-based pricing 讓 vertical SaaS 有抵擋 horizontal 的機制。**四條全破 → 不可直接套用 Siebel 崩盤劇本**，但 communication / 輕 CRM vertical（HUBS/ZM/TWLO）確實在被擠壓。
💡 Insight §2
1. Einstein 1.0 失敗的真正原因是 LLM 能力不足，而非 Salesforce 策略錯：2022 ChatGPT 後 Agentforce 才有商業化可能；當前類比應是「ERP 雲端轉型」而非「Einstein 崩盤」
2. AWS-SaaS 類比最接近 Anthropic 策略：push primitives（MCP）讓上層繁榮而非直接吃
3. Claude Code $2.5B ARR 是 Anthropic 自打臉的 exception：雖然說「推 platform 不做 vertical」但 coding 是唯一例外 — 這個例外是否擴展到 security / legal / medical 是 2027 關鍵觀察

💡 Insight §3

3. Outcome-based pricing 是 Vertical SaaS 反守為攻的關鍵：HUBS / Harvey 先行者優勢

§9.5 Kill Scenario / Steel-man 反方

⚠️ 本章強制列 3 條最有力反方論證（QC-I8）

反方 1：PLTR 65-80x EV/S 在 FY27 US comm 減速至 < 80% 後壓縮 30-40%

  • 路徑：PLTR FY26 guide US comm +115% → 2027 Q1 公告 YoY +90% → 市場恐慌「減速 narrative」→ EV/S 從 80x 壓至 45-50x → 股價 -40%

  • 成真時間窗：2026 Q4 - 2027 Q2
  • 本 ID 反駁：(1) US comm 從 $1.465B FY25 到 $3.14B FY26（+115%）意味從 < 1,000 客戶擴張至 2,000+ 客戶，TAM 未飽和；(2) Maven 擴張至 $1.3B ceiling 為 government 錨；(3) ontology 深度換算 5-7 年合約鎖定

反方 2：Agentforce $800M ARR 裡 60%+ 是「rebrand existing Einstein」貼牌

  • 路徑：CRM 把舊 Einstein 使用者重新計為 Agentforce 深度用戶 → $800M ARR 誇大 → FY27 實際 organic agentforce < $400M → market re-rate
  • 當前證據：(1) 60% Q4 bookings 來自現有客戶擴張（非 new logo）；(2) Salesforce 未拆 new logo vs expansion 比例；(3) Informatica $1.1B 混入 $2.9B Data 360 + Agentforce 總和
  • 成真時間窗：2027 Q1-Q2
  • 本 ID 反駁：(1) 29,000 deals / 2.4B agentic work units 是真實執行數據；(2) Anthropic 擴大合作 regulated industry 是 organic 新用例；(3) Flex Credits $2/conv 的消費數據無法偽造

反方 3：Vertical SaaS 被 Horizontal 吞食是持續減速的結構風險，INTU/WDAY 2028 後同受影響

  • 路徑：MSFT Copilot Studio + CRM Agentforce 進軍 HR / Finance vertical → 2027 開始搶 WDAY 大客戶 → INTU TurboTax 被 ChatGPT 吃中小端 → WDAY / INTU 2028 AI ARR 成長放緩

  • 成真時間窗：2028-2029
