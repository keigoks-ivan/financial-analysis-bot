GLP-1 對包裝食品
不是景氣回落，是需求結構性被吃掉

把母題報告濃縮成一張投資決策卡：減重藥把包裝食品的「胃容量」永久性砍掉一塊——零食、糖果、餅乾是 secular LOSER，優格與蛋白是 winner，而「便宜的防禦股」恰恰是 value trap 風險最高的一群。

市場把包裝食品當「無聊但安全的防禦股」在看，焦點錯了。GLP-1（減重藥）對這個產業不是又一輪通膨或景氣的量價波動，而是一場終端需求的相變——藥理上壓抑食慾，把每個用戶的「胃容量」永久性砍掉一塊，而且砍得最深的正是高熱量、indulgent 的零食、糖果、餅乾。這不是供給過剩、也不是短缺，是需求被外生力量結構性侵蝕。

證據是硬的：Cornell × Numerator 對 150K 家庭面板的研究（2025-12 發表於《Journal of Marketing Research》）顯示，用戶開始用藥 6 個月內 grocery 支出降 5.3%（高收入家庭逾 8%），savory snack 掉約 10%、甜點烘焙與餅乾掉 6.7～11%；只有優格、生鮮水果、蛋白棒、肉類零食逆勢上升。Hershey 2026 第一季量已 −2pt、整體包裝食品量自 2026 起維持 −2%——這已經不是預測，是在財報上發生的事。

所以本報告的核心張力不是「GLP-1 會不會傷包裝食品」（會，而且在加速），而是「便宜」是不是安全。KHC fwd P/E ~10x、殖利率近 7%、HSY ~29x——估值光譜兩端都還沒把 secular 量縮 modeling 進去。判斷重心：辨認哪些是「便宜有理由的 value trap」、哪些是結構受惠的真 alpha。

PART I · DECISION SUMMARY

決策摘要層

三句話看完需求破壞的現狀、走向與行動，後接 PM 級監測點。這一層是報告的全部重點——下面各節只是把這一層的每個判斷補上證據。

現在 / NOW

需求結構性萎縮、且在加速——GLP-1 user 已約 13M（2025 末）、佔成人 ~5%，Cornell-JMR 確認用戶 grocery −5.3%／savory snack ~−10%／甜點烘焙餅乾 −6.7~−11%；HSY Q1 2026 量 −2pt、整體包裝食品量自 2026 起維持 −2%，零食糖果餅乾是直接受害類別。

未來 / NEXT

未來 3–5 年衝擊放大、K 形分化加深——口服 Foundayo（$50 Medicare／$25 商保，7/1 上路）把 user 推向 2030 ~25M、滲透 ~10%，零食糖果餅乾 5y 累積量縮 −5~−8%；優格／蛋白棒／肉類零食結構受惠。最大反向變數是停藥率 30–40% 與 reformulation 部分回補。

行動 / ACTION

包裝食品「防禦＝便宜」是 value trap 風險最高的一群——零食糖果純玩家（HSY／MDLZ／KHC 的 indulgent 組合）非結構 long，便宜（KHC fwd ~10x／GIS ~10x）不等於有 alpha；結構受惠在優格（DANOY）／蛋白棒（BFI）／雞肉蛋白（TSN）。買賣由個股 DD 決定。

The Key Call · 本報告最重要的一個判斷

市場把包裝食品當「便宜的防禦股」在估值，真正的非共識是——這是一場終端需求的相變：減重藥把消費者的胃容量永久砍掉一塊，砍得最深的正是 indulgent 零食糖果餅乾。Cornell 150K 樣本的 −5.3% grocery、savory snack ~−10% 不是景氣波動、是 secular 結構。所以「便宜」反而最危險——分歧不在「GLP-1 會不會傷」（會），而在「現在的低估值與高股息，是 value 還是 trap」。這把選股焦點從「買不買防禦股」移到「分清結構受害 vs 結構受惠，零食糖果是前者」。

三道讓「便宜＝陷阱」的裂縫具體是：①口服把市場打開——注射 GLP-1 受心理門檻、供給與 $1,000+/月商保價格 cap 住，Foundayo（orforglipron）2026-04-01 獲批、$25 商保／$149 自費／$50 Medicare（7/1），三重 access 擴張把 2027 起的 user 成長從漸進變加速；②公司口徑落後數據——MDLZ 自承的 0.5–1% 十年量縮、HSY 過去「no material impact」說法，與 Cornell 的 −6.7~−11% 甜點數據明顯落差，意味未來幾年是 multi-year EPS 下修而非已 price；③reformulation 只是部分回補——蛋白化、縮 portion 能補回 30–50% 量縮，不是完整解。盯這三條，比盯任一季 organic beat 都更早看到拐點。

PART II · HISTORICAL ANALOGS

兩個歷史類比，框住衝擊的天花板

歷史不必重演，但兩個量縮對照各給一個可操作的尺度：2003–2004 的 Atkins 低碳水熱潮框住「主動限食的量縮幅度」，1995–2015 的香煙 30 年衰退框住「結構性減用的股價路徑」。

正類比 #1：2003–2004 Atkins／South Beach 低碳水熱潮——量縮幅度的標尺，但持續性差一個量級。2003–2004 年高峰時約 3,000 萬美國成人在試低碳水飲食，bread／pasta／cereal 等碳水類別 volume 一度掉 10–15%，Kraft（KFT 前身）、GIS 等大廠用低碳水 reformulation 救援，但 2005 起熱潮 fade、量回升。教訓清楚：當消費者主動限制某個 macronutrient，包裝食品的 category volume 可以掉到雙位數——這給了 GLP-1 衝擊一個量縮上限的參照。但兩者差一個量級的關鍵是持續性：Atkins 是 fad diet、12–24 個月後 fade；GLP-1 是 medication，Cornell 12 個月以上數據顯示 effect persists（magnitude 縮小但不消失）。所以包裝食品不能套 Atkins playbook——「撐著、等熱潮過去、reformulation 過渡」這套對 GLP-1 不成立，因為它不會過去。

正類比 #2：1995–2015 美國香煙 30 年衰退——結構性減用的股價路徑模板。美國吸菸成人佔比從 1990 年代中期約 25% 降到 2015 年約 14%，香煙 volume 累積腰斬；Big Tobacco（DA／RAI／PMI）用提價、reformulation（heat-not-burn）、多元化與高股息撐住現金流，股價是「multi-decade 緩跌 + 肥股息」的型態。這對 KHC／HSY／MDLZ 的含義是雙面的：好的一面是 indulgent 食品有定價權與品牌黏性、可以靠提價與股息延緩；壞的一面、也是本 ID 的警示——「便宜 + 高股息」正是香煙股 1995–2015 的長相，買它的人很多年都在賺股息、賠股價。KHC 殖利率近 7%、fwd P/E ~10x，看起來是 deep value，但若 secular 量縮像香煙一樣 multi-decade，便宜會一直便宜下去（value trap）。差別在於：香煙 incumbent 維持 80%+ 品類份額、沒有 disruption；包裝食品 incumbent 同時面對 private label（PL）與健康新創雙重夾擊，競爭強度數倍於香煙——衰退路徑可能比香煙更顛簸。

PART III · SUPPLY SIDE

供給側：reformulation 是部分對沖、不是解藥

這是一個沒有實體產能短缺問題的供給側——大廠有的是工廠與貨架。供給端的真正變數是「能不能把組合從受害類別重配到受惠類別」，而 reformulation 與併購是這場重配的工具，不是萬靈丹。

包裝食品不是供給過剩、也不是供給短缺的故事——大廠產能充裕、貨架關係穩固。供給側的結構是「indulgent 重倉的純玩家 vs 已重配到蛋白/優格的玩家」之間的分化。對受害方，唯一的供給側武器是 reformulation（蛋白化、縮 portion、功能化）與併購（補健康定位的缺口）：KHC 新 CEO Cahillane 撤掉分拆、投 $5 億以上轉向蛋白、避開糖；HSY 投資 VitaKey 做時間釋放營養、擴 Fulfil 蛋白棒；MDLZ 靠 Clif／Builders 蛋白棒對沖；GIS 推 Cheerios Protein、Annie's 14g protein。但業界數據顯示——16 項 premiumization／reformulation／降價投資中，只有 7 項讓品牌回到成長。reformulation 是部分回補（補回 30–50% 量縮），不是完整解。

Inference · 供給
沒有供給短缺，所以「稀缺溢價」不存在——重配能力才是分水嶺
前提：產能充裕（無實體短缺）+ 受害類別需求 secular 萎縮 + reformulation 僅部分回補（業界 16 項僅 7 項回成長）+ 受惠類別 capacity-constrained（優格/蛋白）

這個產業的供給側框架要逃逸「過剩 vs 短缺」的慣性——它兩者都不是：終端需求被藥物永久侵蝕，是「需求結構性萎縮」型，稀缺的反面不是供給太多、而是需求被吃掉。所以決定勝負的不是誰產能效率高、而是誰能把組合從受害類別（零食糖果餅乾）重配到受惠類別（蛋白、優格、肉類零食）。值得注意的反差是：受害類別產能過剩、受惠類別卻 capacity-constrained（Danone 美國高蛋白優格缺貨、Chobani 擴廠）——這正是「需求被搬走、不是供給多了」的鐵證。

可證偽條件：若零食糖果純玩家透過 reformulation 在 2027–2028 把 volume 拉回持平、且新蛋白 SKU 銷售達原 SKU 的 50%+（即 reformulation 從「部分」變「大致完整」回補），則「reformulation 非解藥」論鬆動、受害論降級。

PART IV · DEMAND SIDE — DESTRUCTION

需求側：胃容量被永久砍掉的破壞引擎

這是本報告的核心章節——需求不是疲軟、是被外生力量結構性破壞。減重藥從生理上壓抑食慾，把每個用戶的總食量永久下調，而且不平均地砍在 indulgent 類別上。

需求破壞由一條清楚的因果鏈推動：GLP-1 藥物延緩胃排空、作用於下視丘食慾中樞 → 用戶總熱量攝取下降 → 反映在 grocery 帳單上是 6 個月內 −5.3%（高收入家庭逾 8%）→ 而且砍得不平均：savory snack 約 −10%、甜點烘焙與餅乾 −6.7~−11%，主食（麵包/肉/蛋）小幅下降，只有優格、生鮮水果、蛋白棒、肉類零食逆勢上升。這是 Cornell × Numerator 對 150K 家庭面板的研究（樣本期 2023 末–2024 中、2025-12 發表於 JMR）——不是預測、是已觀測到的行為。重點在「永久」二字：注射用戶有 30–40% 在 12 個月內停藥（保險、副作用、成本），這是衝擊的上限剎車；但口服 Foundayo 把門檻打低、把 user 池放大，淨效果是衝擊深化。

Inference · 需求
這是「需求被永久搬走」、不是景氣回落——所以提價救不了量
前提：藥理性食慾壓抑（生理、非偏好）+ Cornell 12 月以上 effect persists + 口服把 user 池放大 + 停藥率 30–40% 是上限剎車

包裝食品的需求破壞與一般「景氣下行、消費降級」有本質差別：景氣回落時消費者是「想吃但沒錢」，可以靠降價/小包裝救回；GLP-1 用戶是「有錢但吃不下」，降價無效——這把需求的「量」直接從結構上下調。所以受害方的營收模型最脆弱的不是價（仍有定價權）、而是量（被永久搬走）。需求的「質」也在劣化：留下的需求往蛋白/優格/肉類零食集中，indulgent 那塊回不來。

可證偽條件：① Cornell follow-up 24/36 月數據顯示 effect 縮至 −3% 以下（用戶適應後恢復部分 indulgent 攝取）；② 停藥率升破 50%、user 池停在 13–15M（access 擴張失敗），則「需求永久搬走」降級為「需求暫時下移」、受害論的時間軸延後。

為何「便宜的防禦股」最危險

包裝食品的低估值（KHC fwd ~10x、殖利率近 7%；GIS ~10x）通常被當「安全墊」——但這恰是 value trap 的典型結構：當終端需求 secular 萎縮、而市場仍把它當「景氣循環底部的便宜貨」，估值會「便宜得有理由」並一路便宜下去（香煙股 1995–2015 模板）。反過來，HSY ~29x 的高估值也還沒把 secular 量縮 modeling 進去——光譜兩端都有 mispricing，但方向相同：未來幾年是 multi-year EPS 下修。所以本 ID 把判斷重心放在「分清結構受害 vs 受惠」，而非「估值高低」。

PART V · VERDICT & SCENARIOS

裁決與三情境（24–36M）

需求破壞方向判 INTACT（藥理性食慾壓抑 + 口服加速），受害類別 secular 萎縮判 INTACT（Cornell 證據 + HSY 已現量縮），但「衝擊的速度與深度」判 AT_RISK（停藥率 + reformulation 回補 + access 擴張節奏未定）——故 thesis 結構性看空 indulgent、結構性看多蛋白，但 conviction 卡在 mid-high。

PART VI · TAXONOMY

個股特徵與 conviction tier

本框架只輸出「特徵」與「分類」，買哪一檔由個股 DD 系統與 Pure MA 訊號決定。以下 tier 為產業推論的特徵歸類，提及公司僅為「該特徵的代表案例」，不附帶買賣建議。

LOSER 🔴

HSY / KHC

結構性受害最純的代表。HSY 巧克力純玩家、Q1 2026 量已 −2pt，純度高 = 衝擊最集中，但 fwd ~29x 估值尚未 modeling secular 量縮；KHC indulgent 重倉（Lunchables/Oscar Mayer）、撤分拆 + $500M 轉蛋白救援中、fwd ~10x 殖利率近 7%——「便宜 + 高股息」正是 value trap 的長相。特徵是「結構受害、估值便宜不等於安全」。conviction：受害論高 · 估值非 floor
LOSER 🔴

MDLZ / GIS

受害但有緩衝。MDLZ 餅乾甜點主導（Oreo/Ritz）、自承 0.5–1% 十年量縮（本 ID 視為 floor、實際偏 −3~−5%），靠 Clif/Builders 蛋白棒對沖、fwd ~20x；GIS Cheerios 對 PL 價差 35%、Cheerios Protein reformulation 部分 offset 但 cereal 結構 stagnant、fwd ~10x。特徵是「受害 + reformulation 部分回補」。conviction：中 · 視 reformulation 成效
中性 🟡

CAG / CPB / SJM / NSRGY

混合曝險。CAG frozen 部分受惠 healthier mix／shelf-stable 持平；CPB soup 結構脆弱（multi-decade 衰退 + 關稅 + GLP-1 mild 三壓）；SJM Hostess sweet baked 受害、咖啡中性；NSRGY Vital Pursuit GLP-1 friendly first mover、hybrid 受惠。特徵是「非純受害也非純受惠、子類別互抵」。conviction：中 · 子類別淨效果
WINNER 🟢

DANOY / BFI / TSN

結構受惠三軸，Cornell 四大 winner 類別（優格/蛋白棒/肉類零食）的直接對應。DANOY 高蛋白優格美國 capacity-constrained（GLP-1 user 食優格 3x average）；BFI Premier Protein RTD/bar 純玩家、small-cap leverage；TSN Chicken Cups 30g protein + Simple Ingredient Nuggets 23g protein、chicken seg adj OI +45%。特徵是「需求被搬到這裡」。conviction：受惠論高 · DANOY 流動性偏低

估值與配置面總結：本主題的核心張力在「受害方便宜得有理由、受惠方共識仍不足」。受害端（HSY ~29x／KHC ~10x／MDLZ ~20x／GIS ~10x）的估值光譜兩端都還沒把 secular 量縮充分 price——低估值不是 floor、高估值有下修空間，方向都是 multi-year EPS 下修。受惠端（DANOY 優格／BFI 蛋白棒／TSN 雞肉蛋白）sell-side 共識仍以傳統 CPG 框架看待，對 GLP-1 user 蛋白攝取上升的 modeling 不足，是 PM 可關注的 misaligned 配置——但 DANOY 為 ADR、流動性有限。MKC（香料 flavor attach rate 升）是 stealth 中性候選。現處 Phase IV（口服上路、K 形加深）；轉空 indulgent 的條件 = user 加速破 20M 或 第二代藥更強；受害論鬆動的條件 = 停藥率破 40% + Cornell follow-up effect 收斂。所有 ticker 為「特徵代表案例」，不附帶買賣建議，買賣由個股 DD 與 Pure MA 系統決定。

The One Line

這不是景氣回落、是胃容量被永久砍掉——零食糖果是 secular LOSER；真正的陷阱不是「貴」，而是「便宜的防禦股」把 multi-year 量縮當成了循環底部。

這份報告刻意不做的三件事

  1. 不告訴你哪一檔該買或該空——LOSER/WINNER 是「特徵的代表案例」，買賣由你的個股 DD 與 Pure MA 系統決定。「結構受害」不等於「現在就該空」（估值、股息、停藥率反向變數都在）。
  2. 不預測股價走勢——只給「進入哪個情境」的觸發訊號（user count、HSY/Frito-Lay volume、Cornell follow-up、停藥率），不給點位。
  3. 不假裝知道用戶會不會適應——GLP-1 user 5 年後是否恢復部分 indulgent 攝取、停藥率會走多高，目前無定論，已誠實標為 base/bull/bear 的關鍵分歧。
