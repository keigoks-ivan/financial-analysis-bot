§1 產業白話定義 + 歷史脈絡

Robotaxi 賣的不是「一輛更好的車」，而是「把人類司機從計程車裡拿掉」——付錢的是每天要移動的乘客，他們在意的不是技術多炫，是「比 Uber 便宜、不用等、夠安全」。能不能合法在某座城市跑無人車，才是整個生意的關卡。

白話開場：傳統計程車與 ride-hailing（Uber / Lyft / 滴滴）的成本結構裡，司機薪資佔每趟約 60–70%。Robotaxi（自動駕駛計程車，車內無人類駕駛、由 AI 開車的載客服務）的整個價值主張，就是把這塊成本拿掉——理論上能大幅降價搶量、或在同價下賺更高毛利。但這不是「造一台會自己開的車」就成立的生意：你還要拿到某座城市／某州的「無人載客」法規許可，而監管只在你累積足夠的無人里程、證明 AI 夠安全後才敢放行。所以 robotaxi 的本質是「AI 安全紀錄 → 法規准入 → 車隊規模 → 更多里程」的滲透遊戲，不是單純的硬體製造。

邊界界定：本報告聚焦「以載客為目的的 L4 自動駕駛計程車服務」與其關鍵供應商。納入車隊運營商（Waymo / Tesla robotaxi / Apollo Go / Pony.ai / WeRide / Zoox）、需求層平台（Uber / Lyft）、關鍵賣鏟者（NVDA 車端算力、Hesai / RoboSense / Luminar / Innoviz 的 LiDAR 感測）與整車夥伴（Toyota / Geely）。不深入 L2/L3 個人車輔助駕駛的純 ADAS 滲透（那是另一條敘事，雖然 Mobileye / Tesla FSD 橫跨兩者），也不展開 Aurora 的無人卡車（vertical adjacent 但載貨非載客）。灰色地帶在「Tesla」——它既是 unsupervised 純視覺路線的最大賭注，又因 FSD 同時是個人車功能而身份雙重，這個雙重身份本身就是 §7 的 insight 來源。

歷史脈絡分三個轉折。第一次轉折：2009–2016——Google 自駕車計畫（後分拆為 Waymo）起步、Cruise 被 GM 收購（2016），「自駕計程車」從科幻變成大廠重押的工程專案，但當時普遍低估了「最後 1% 安全性」需要多少年與里程。第二次轉折：2020–2024——Waymo 在鳳凰城、舊金山逐步開放無人載客，證明「方向可行」；但同期 Cruise 因 2023 年舊金山拖行行人事故停運、GM 於 2024 年底停止注資退出（累計燒約 $10B），給整個產業上了一堂「規模化的時程與資本遠超想像」的課。第三次轉折：2025–2026——商業化規模真正分化：Waymo 週訂單一年內從 5 萬衝到 50 萬（10x）、中國三雄首見獲利、Tesla 於 2025 年 6 月在 Austin 推出 robotaxi 並於 2026-06-03 開放全 Austin metro 給 unsupervised 無人車。

最硬的壓力測試：GM Cruise 的退場——這是 robotaxi 史上最貴的一課。GM 自 2016 年收購 Cruise 起累計投入約 $10B，卻在「規模化所需的時間與資本」面前選擇退出，2024 年底停止注資、將其併回內部 Super Cruise 個人車 ADAS 團隊，預計每年省下逾 $1B。一家有底特律製造底氣、燒了 100 億美元的玩家都選擇放棄無人計程車——這比任何樂觀敘事都更能量化「方向對 ≠ 個別玩家活得下來」的早期高成長殘酷性。

歷史類比一：電動車早期（2012–2018）。當年「電動是未來」是共識，但 Tesla 燒了近十年、數次瀕臨破產才達規模化獲利——多頭錯在「以為方向對就能快速賺錢」、空頭其實對了一段但對的理由是「兌現時程慢、燒錢久」而非「方向錯」。本次的量化對照是：robotaxi 的方向確定性同樣高，但瓶頸從「量產地獄」移到「法規准入 + 無人車隊規模化」，Cruise 燒 $10B 退出就是這個瓶頸的最新印證。

Robotaxi 沒有半導體那種可統計的價格 cycle，改用「滲透率擴散曲線」替代——看自駕計程車滲透到哪、誰拿到准入。

關於 cycle 統計表的省略：本產業是早期單向滲透成長、無明顯供需價格 cycle（不像記憶體／面板有 ASP peak-to-trough），故不立歷史 cycle 統計表；改以 §2 的滲透 S 曲線呈現階段。可參考的時序錨點是：Waymo 週訂單從 2024-05 的 5 萬增至 2026 的約 50 萬（約 18 個月增 10x），顯示「商業化規模的傳導」一旦數據飛輪啟動可以非線性加速——但這個加速高度集中在少數領先者，不是全產業普漲。

這段歷史的可操作含義是：真正該怕的不是「robotaxi 太貴」（那是市場情緒），而是「個別玩家在兌現前的燒錢谷裡分化出生死」（Cruise 已示範）+「估值透支兌現速度」（TSLA 風險）。只要產業滲透曲線（CAGR ~90%）成立、且你押的是規模已驗證 + 數據飛輪領先者，方向型 bear 就不成立——這把 bear 的天花板從「整體估值」重新定位到「個別玩家的兌現 vs 燒錢」。

§2 技術成熟度 + S 曲線

為什麼商業化是現在發生、不是 2020 年？因為三個技術 bottleneck 剛被解開——AI 模型把「最後 1% 長尾場景」的安全性推到法規敢放行、感測成本（LiDAR 從數萬美元降到約 $200）讓單車經濟有機會轉正、車端算力（NVIDIA Thor 級）讓即時推理夠快。三者到位，滲透才從 demo 變成可規模化的服務。

過去「自駕」多半卡在「99% 場景能開、但最後 1% 長尾（罕見路況、惡劣天氣、邊緣案例）不夠安全」，而監管只看那 1%。真正的技術拐點有三：①AI 模型成熟——端到端神經網路 + 大量真實里程訓練，把長尾安全性推到「無人載客比人類司機更安全」的可驗證水準（Waymo 公布的事故率數據是其法規准入的基礎）；②感測成本崩跌——Hesai 把車規 LiDAR 從早年數萬美元壓到 ATX 約 $200（靠自研晶片 + 量產），讓「裝得起 LiDAR 的安全冗餘」與「單車經濟轉正」第一次可以兼得；③車端算力——NVIDIA DRIVE Hyperion + AGX Thor 提供即時多感測融合推理的算力底座。三者缺一，robotaxi 就停在 PPT。

§3 供給側：現在與未來 + 利潤池

供給端的結構是「少數能跑無人載客的領先者 + 一票還在安全駕駛或測試的追趕者」。瓶頸不是產能（車子不稀缺），而是「能不能在某座城市合法跑無人車」這道准入門票——它由法規 + AI 數據 + 車隊單位經濟三道門檻共同決定。

框架逃逸 · 為何不是商品供需

傳統供需框架問「產能會不會過剩、價格跌到誰先停產」——但 robotaxi 沒有實體產能的成本曲線，整車要多少有多少，車子本身不稀缺。真正的供給瓶頸是准入：你能在幾座城市合法跑無人車，由法規 + 累積無人里程的安全紀錄決定。所以這是早期滲透型產業（看「滲透到哪、誰拿到准入」），不是商品週期（看「過剩／短缺」）。稀缺性來自三道門檻的交集（法規准入 + AI/數據護城河 + 車隊規模單位經濟），而非產能。若硬要對應傳統框架：合格供給（能合法跑無人載客）遠少於潛在需求，等價於結構性短缺型——只是稀缺來自准入門檻而非實體產能。

成本曲線（省略說明）：robotaxi 無實體產能的邊際成本曲線；單位成本由「感測（LiDAR 約 $200 vs 純視覺）+ 算力 + 營運（遠端協助、清潔、充電）」決定。價格戰的終局不由「誰先停產」決定，而由誰先把單車經濟做正決定——這正是中國靠 Hesai $200 LiDAR + Apollo Go RT6 低成本車，能在成本端領先、率先報出單車獲利的根本原因；而 Tesla 純視覺路線是賭「拿掉 LiDAR 把成本壓更低」，但代價是安全冗餘的爭議。

未來供給與進入門檻：新進入者要成為規模贏家，門檻是「累積足夠無人里程拿准入 + 把單車成本壓到獲利 + 拿到多城法規」，這三者單靠內生要很多年——這也是為什麼整車廠選擇與軟體商合作而非自建（Toyota × Pony.ai、Geely × WeRide），以及需求層（Uber）選擇「整合多家車隊」而非自建 AV stack（$10B+ capex 但分散押注 Lucid-Nuro / Wayve / Apollo Go / WeRide）。供給彈性低（不能快速「養」出一個有數據飛輪 + 多城准入的車隊）正是領先者議價權的來源。

因果閉合（回應 §1 的結構變數）：§1 提出「法規准入 + 數據飛輪」是供給瓶頸——未來 3–5 年它是否仍 binding？答案是很可能更 binding：跑越多無人里程 → AI 越安全 → 法規越敢放行 → 更多城市 → 更多里程，形成正回饋，後進者越難線性追上（Waymo 一年衝 10x 即此飛輪）。唯一能鬆動門檻的是「更便宜的純視覺方案大規模通過安全驗證」（Tesla 賭注），讓不需高累積里程也能拿准入——這是 §7 要檢驗的反方。

12–36 個月內，供給端不會出現大量新的規模級競爭者（門檻太高、燒錢谷淘汰），議價權會持續集中在現有少數領先者 + 賣鏟者。對投資的含義：押「賣鏟者」（NVDA/Hesai，誰贏都受益）與「規模已驗證的車隊」（Waymo/中國領先者）比押「規模未兌現的故事」（Tesla robotaxi）在風報上更均衡——因為後者即使方向對，也可能在「兌現比敘事慢」時被估值地心引力修正。

§4 需求側：現在與未來 + 三角驗證

需求最不可替代的一塊是「比人類司機更便宜 + 更安全 + 隨叫隨到」的剛性——載客出行是兆美元級剛需；最該防的不是「需求夠不夠」，而是「用什麼口徑衡量需求，才不會被快慢假象騙了」。

現需結構：終端需求不缺——全球載客出行（ride-hailing + 計程車）是兆美元級市場，robotaxi 拿掉司機成本（每趟約 60–70%）後理論上可大幅壓價搶量。需求 price elastic（價格彈性高）：降價會帶量。兩個需求引擎：①單位成本下降——Waymo $30/月 Premier 會員、中國 Apollo Go 補貼價，都在測「比 Uber 便宜」的彈性；②需求層整合——Uber（$10B+ AV capex、目標 2026 底 10+ 國）與 Lyft（Waymo 那什維爾、Mobileye 達拉斯）把 robotaxi 接進既有 app，替車隊運營商現成導流。需求方向不缺，問題在衡量口徑。

因果閉合（回應 §1 的需求驅動）：§1 提到「拿掉司機成本」是整併驅動——未來 5–10 年它會強化而非消退。隨 AI 與感測成本續降，單車經濟轉正的玩家越多、降價空間越大、需求被進一步釋放。需求拐點風險不在「量」（出行剛需很難消失），而在「價」與「准入」——補貼退場後單車經濟能否維持（中國依賴）、重大事故是否壓抑監管放行（見 §5/§7）。

最有定價權的需求是「比人便宜 + 隨叫隨到 + 安全紀錄可信」的綁定——客戶買的不只是便宜，是「不用跟人類司機互動 + 可預期的服務」。對投資的含義：需求引擎（出行剛需 + 降價彈性）幾乎無爭議、很可能持續，所以「需求夠不夠」不是風險點；真正要盯的是「規模兌現的速度」與「單位經濟的持續性」——以及別被「地理覆蓋」「營收成長率」這類口徑當成「規模領先」。

§6 產業經濟學與估值傳導

Robotaxi 的錢怎麼賺：拿掉司機成本後，每趟的毛利取決於「單車利用率 × 單趟淨費 − 感測/算力/營運攤提」。估值跟著「車隊規模兌現 × 單車經濟轉正」動——里程碑兌現會直接決定高倍數能否撐住。

robotaxi 占估值小（Alphabet 主體 search/cloud）~60%（ARK 估 2029 ~90%）主體即 robotaxi

怎麼讀：三類玩家的「估值 vs 兌現」關係完全不同——Waymo 規模最實但 robotaxi 占 Alphabet 估值極小（曝險被稀釋）；Tesla robotaxi 占估值多數但規模未兌現（透支最重）；中國三雄主體即 robotaxi、單位經濟領先但基期低、政策高 beta。所以「貴不貴」要分玩家、且要對齊「估值 price 了哪個尚未發生的里程碑」。

估值傳導機制（priced-in 的 reverse 推導）：早期高成長標的的估值對里程碑極度敏感。以 Tesla 為例做反推：市值約 $1.5T、若 robotaxi 真占其企業價值約 60%（多家模型）、甚至 ARK 估 2029 達 ~90%，等於市場已隱含「unsupervised 純視覺車隊在數年內規模化到數十萬台、單位經濟轉正、零重大退牌」這一連串里程碑全部兌現。但現況是全 Austin 約 20 台、Morgan Stanley 估 2026 底約 1,000 台（PT $415）——這個「估值隱含的數十萬 vs 現實的數十／一千」就是 robotaxi 最大的下行槓桿。傳導鏈是：里程碑兌現 → 確認高估值 → 撐住股價；反向，兌現一落後敘事，高倍數會被快速壓縮（de-rate），且因 robotaxi 占估值多數、跌幅會放大。敏感度錨點：ARK $4,600 vs Morgan Stanley $415 的近 11 倍價差本身，就是「市場對里程碑兌現假設分歧極大」的量化證據。

估值現在 price 了什麼：Tesla 的 P/E >300 price 了「unsupervised 車隊規模化零失誤 + 數十萬台 + 單位經濟轉正」；中國三雄的低估值 price 了「政策/地緣折價 + 兌現但基期低」；Waymo 的規模實質上沒被單獨 price（藏在 Alphabet 裡）。會觸發 re-rate 的是「Tesla 車隊實際放量 + 純視覺驗證」；會觸發 de-rate 的是「兌現落後敘事 + 重大事故」。對 PM：估值高低不是進出依據，里程碑兌現的拐點才是。

§7 Non-Consensus + Priced-in 檢驗 + Kill Scenario

⚠️ 本章含 judgment：每條分歧附 [F:] cornerstone + [X:] 三情境 + priced-in 檢驗。
市場對 robotaxi 的三個共識：①「robotaxi 是長期大主題、買就對」；②「Tesla 會用純視覺 + fleet 數據贏者通吃」；③「中國三雄便宜、必補漲」。三個我們都部分不同意——而分歧能不能賺錢，取決於市場 price 了多少。
🔴分歧 1：「robotaxi 是長期大主題、買就對」信心：高
  • 共識說 X：robotaxi 是確定的長期趨勢，買哪檔都會贏。🟢 [F: T3 — TAM 2030 ~$45B、CAGR ~90%]。
  • 我們說 Y：方向對，但「買就對」忽略了個別玩家兌現速度差四個數量級——Waymo 50 萬週訂單 vs Tesla 全 Austin 約 20 台，且早期高成長標的估值對里程碑極敏感。🔵 [X: base 很可能 — 規模已驗證端（Waymo/中國領先者/賣鏟者）穩；bull 可能 — 純視覺驗證使故事端也兌現；bear 可能 — 重大事故全產業 de-rate]。
  • Priced-in：Tesla robotaxi 占 EV ~60%、P/E >300，現價隱含「數十萬台規模化全兌現」，alpha 薄、下行槓桿大；賣鏟者（NVDA/Hesai）與中國三雄的估值沒有 price 這種完美兌現，留有空間。→ 分歧在「買就對」是錯的，正解是「選已驗證/賣鏟端、避透支端」。
→ 推論邏輯：把「robotaxi 買不買」拆成「哪一檔、賭哪個里程碑、什麼價」，賣鏟 + 已驗證端風報優於故事端。⚠ 證偽：若 Tesla 純視覺在多城零事故規模化，故事端的高估值被兌現追上、分歧 1 收斂。
🔴分歧 2：「Tesla 用純視覺 + fleet 數據贏者通吃」信心：中
  • 共識說 X：Tesla 數百萬輛在路上的 fleet 數據 + 純視覺低成本，終將碾壓 Waymo 的「貴 LiDAR + 慢擴張」。🟢 [F: T1 — Tesla 全 Austin metro unsupervised、Texas L4 自我認證]。
  • 我們說 Y：fleet 數據優勢是真的，但unsupervised 無人載客的安全冗餘仍未經多城規模驗證——全 Austin 僅約 20 台、車隊不增反縮，且背 NHTSA 兩條調查（PE25012 涉 58 起 FSD、EA26002 涉約 320 萬輛視覺）。🔵 [X: base 很可能 — Tesla 車隊緩步放量但慢於敘事；bull 可能 — 純視覺驗證重訂門檻；bear 可能 — 重大事故/退牌]。
  • Priced-in：Tesla 估值已 price「贏者通吃」（robotaxi 占 EV ~60%），幾乎沒 price「純視覺安全冗餘不足導致退牌/召回」的下行；反之 Waymo 的數據飛輪領先（約 1 億英里無人里程）沒被單獨 price（藏在 Alphabet）。所以押「Tesla 不會輕易退牌」是部分未 priced 的對賭，但押「Tesla 已贏」則分歧已 priced、不可操作。
→ 推論邏輯：純視覺 vs LiDAR 的安全冗餘之爭未定，市場只 price 了 Tesla 利多一面。⚠ 證偽：Tesla unsupervised 車隊連季倍增 + 安全紀錄乾淨 = 分歧 2 翻向 Tesla；或重大致命事故 + 退牌 = 翻向 Waymo。
🔴分歧 3：「中國三雄便宜、必補漲」信心：中
  • 共識說 X：Pony.ai / WeRide / Baidu 估值低、又首見獲利、成本領先，必然補漲。🟢 [F: T1 — Pony.ai 首見季度 GAAP 獲利、Apollo Go 武漢單車獲利]。
  • 我們說 Y：成本領先與首見獲利是真的，但「便宜」反映的是政策/地緣折價——美國上市 ADR 的監管風險、中美科技脫鉤、補貼依賴的單車經濟可持續性，都是 high-beta 來源。🔵 [X: base 很可能 — 隨產業滲透緩步重估；bull 可能 — 出海（中東/歐洲）+ 內生獲利證明，折價收斂；bear 可能 — 政策/退市風險引爆]。
  • Priced-in：中國三雄的低估值已 price 了部分政策折價，所以「便宜」本身不是純 alpha；真正未 priced 的是「單車經濟從補貼驅動轉內生 + 出海成功」——若這個兌現，折價收斂才是可操作的；但若只押「便宜必漲」而忽略政策尾部，分歧已 priced 且風險不對稱。
→ 推論邏輯：中國三雄是「高 beta、政策敏感的單位經濟領先者」，不是「無腦便宜」。⚠ 證偽：補貼退場後單車經濟轉負、或 ADR 退市風險升級 = 分歧 3 的 bear 成立。

Steel-man 反方（最有力的 3 條）

  • 反方 1（估值地心引力）：Tesla P/E >300、robotaxi 占 EV 多數，在任何兌現 miss 或利率正常化下都有巨大 de-rate 空間，且因 robotaxi 占比高、跌幅放大。當前證據：全 Austin 僅約 20 台、車隊不增反縮。證偽窗口：Tesla 任一季無人車隊台數停滯或事故升級。
  • 反方 2（法規退牌終局）：一次致命無人車事故就能觸發 NHTSA 召回/退牌，瞬間關掉一座城、需求被監管壓抑、滲透遞延。當前證據：NHTSA PE25012（58 起 FSD）+ EA26002（約 320 萬輛視覺）兩條調查在跑。證偽窗口：任一調查升級為強制召回或退牌。
  • 反方 3（燒錢谷淘汰）：規模化的時程比敘事慢、資本消耗比模型大，個別玩家在兌現前的燒錢谷裡破產/退出，方向對也救不了股東。當前證據：GM Cruise 燒約 $10B 退出。證偽窗口：任一車隊運營商現金燒盡或被迫大幅縮減。

市場最低估的風險：不是「robotaxi 會不會成真」（人人都信會）、也不是「Tesla 貴」（已被廣泛討論），而是分歧 2 的 Tesla 純視覺安全冗餘未經多城規模驗證 × 法規退牌的尾部——市場把 Tesla robotaxi 純當「贏者通吃」price 進去，最沒準備好「全 Austin 約 20 台、車隊不增反縮、背兩條 NHTSA 調查」這個現實落差被市場重新定價。這是本報告認為最該獨立監測的尾部風險。

三條分歧裡最可操作的是分歧 1（選已驗證/賣鏟端、避透支端）——它最直接、且有明確領先指標（Waymo 週訂單、Tesla 車隊台數、NHTSA 調查）。分歧 2（Tesla 未必已贏）次之、part-priced；分歧 3（中國三雄是高 beta 而非無腦便宜）方向對但政策折價已 part-priced。
