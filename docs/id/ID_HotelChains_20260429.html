Unit Glossary：RevPAR = Revenue per Available Room = ADR × Occupancy（產業核心 KPI）；ADR = Average Daily Rate（每房單日房價）；NUG = Net Unit Growth = (新開房 − 退出房) / 期初房數，asset-light franchise 模式的成長 KPI；Pipeline = 已簽約但未開幕的 hotels / rooms，未來 NUG 的可見性；Conversion = 既有獨立 hotel 或競品 brand 改掛新 brand 旗（無新建）；IMF = Incentive Management Fee（hotel EBITDA 達 hurdle 後 owner 付給 chain operator 的績效 fee，景氣領先指標）；Franchise fee = royalty + marketing + reservation fee 合計（典型 5-15% room revenue，median 11.8% — HVS）；Co-brand royalty = chain 從信用卡發卡銀行收取的 fee（基於每筆消費），是 non-RevPAR fee；Bonvoy / Honors / WoH = Marriott / Hilton / Hyatt 三大 loyalty programs；STR = short-term rental（Airbnb / Vrbo）；Take rate（ABNB）= service fee revenue / GBV（Q4 25 = 13.6%）；GBV = Gross Booking Value（ABNB / BKNG 用語）；Asset-light = chain 持有 brand + system 但不持有實體 hotel，靠 fees 為主（MAR 75% / HLT 97% / WH 100%）。所有 financials = Q4 FY25 + Q1 FY26 actual（截至 2026-04 已公告）。

★ 本 ID 核心非共識 Thesis 市場用 4 種錯誤框架看連鎖旅館：① RevPAR 是判斷 chain 表現的單一 KPI；② 「MAR vs HLT」是 1 對 1 雙寡占競爭；③ Airbnb 已永久搶走 hotel share；④ asset-light 模式還有第二春（fee growth 永續加速）。本 ID 認為：① RevPAR 已不是 chain 的核心 alpha — non-RevPAR fee 才是。Marriott 2026 共品牌信用卡 royalty +35% 重定價是 royalty rate 永久上修（不是循環性），這把 271M Bonvoy 會員的每筆消費轉化為非 RevPAR 流，估值倍數應從「hotel chain 25-30x」修正到「payment platform 35-40x」。② Hyatt 7.3% NUG 是 sector 結構性 alpha 而非短期 noise — Playa 收購 + luxury/lifestyle 集中度（30%+ vs MAR/HLT ~15%）+ 56M 會員小基數高黏性，未來 3 年 NUG 可維持 6-7%（共識給 5-6%）。③ Airbnb 對 chain 的真正威脅是「ADR / ad spend / direct booking」而非 share — chain 永久輸掉 leisure 30 天以上長住，但 chain 透過 Bonvoy/Honors/WoH 鎖定商務+忠誠度 cohort，2024-2025 的 K 形分化是「chain 在商務+高端贏 / ABNB 在 leisure+長住贏」雙贏結構，市場錯誤地認為 ABNB 還會繼續吃 chain。④ NYC Local Law 18 模式（90%+ Airbnb listings 消失，hotel ADR +50%）是「local hotel 短期 alpha」但「ABNB 全球壓力的天花板訊號」。LL18 之後 18 個月，沒有第二個 Tier-1 美國市場跟進，意味著 regulation backlash 已到頂；ABNB 2025 Q4 GBV +16% 重新加速是這個天花板已開始被 price in。⑤ conversion（既有 hotel 改掛 brand 旗）是真正的「無資本 NUG 引擎」，IHG 2025 conversion 占 52% openings + Wyndham 870 deals (+18%) 大多 conversion 是被低估的 alpha — 這個 KPI 被市場用一般 NUG 算總，但 conversion 風險低、cycle bottom 反而加速（owner 想換 brand 救援）。⑥ Wyndham + Choice 雙弱（US RevPAR -8% / -3%）是經濟型循環底部訊號 — 2027 大循環翻轉時這兩家 fee growth 從 +2% 跳到 +8%，是 sector 中最被低估的 contrarian buy。
💡 §0 5 條核心 insight（預告）
1. Marriott 共品牌信用卡 royalty +35% 重定價是永久 alpha 不是 cycle：這是 royalty rate 從 Chase / Amex 重新談下來的結果（不是 spending volume 的循環性增長），271M 會員 × 每年人均消費 $1,500-3,000 × 提升的 royalty bps = 永久增量 fee；market 把它當成 cycle 性 booster 是錯誤的，這應該推高 MAR 估值倍數從 27x → 32x+。
2. Hyatt 7.3% NUG 是 sector 第一名而非「最小玩家賽跑加分項」：HLT 6.7% / MAR 4.5-5% / IHG 4.7% / WH ~3% — Hyatt 連續 4 年領先；Playa $2.6B 全 cash 收購 2025-06 closed 是 all-inclusive vertical 的最後 piece；luxury+lifestyle 集中度（30%+）vs MAR/HLT 約 15% 是真正結構性 NUG quality 差異。
3. asset-light 飽和：HLT 97% / WH 100% / MAR 75% / H ~52% — 進一步 asset-light 已無空間，下個 alpha 是 fee mix shift。MAR 還有 25% managed/owned 拖累，2026-2028 若繼續 spin off，剩餘 fee margin 會 step-up；H 是 sector 中 owned 比例最高（Playa 收購反向擴大 owned），是潛在 swing factor。
4. NYC Local Law 18 模式 18 個月後沒有第二個 Tier-1 跟進，意味 regulation backlash 已到頂：LL18 (2023-09 生效) 之後，紐約 Airbnb listings -90% / hotel ADR +50% / 84% 占用率，但同時 housing 租金仍上漲 8%（vs 全美 3.5%）— 證明 Airbnb 不是 housing crisis 主因。SF/LA/Boston/DC 都未跟進，反向意味 ABNB 全球 share 風險已 capped。
5. Hyatt 的 Playa 收購是「all-inclusive 終極化」而非「逆勢擴張 owned」：$2.6B 含 $900M 債務，14 hotel + 已 manage 8 = 完成 ALG 整合 last piece；Bahia Principe JV (2024) + Playa (2025) 後 Hyatt 在 all-inclusive 是全球 #2 (Iberostar-Vivid 等品牌)，這個 segment 受 GLP-1 / luxury / 嬰兒潮三重順風是被 underprice 的。

§1 產業定義與邊界

連鎖旅館（hotel chains）在本 ID 的範圍是全球以 brand 為核心、asset-light franchise / management 為主流商業模式的住宿業集團。最關鍵的判斷是：「連鎖旅館」不能再以 1980-2010 年代的「hotel real estate operator」看待 — 2008 金融危機後 sector 已徹底轉型成「brand + loyalty + distribution + technology」platform business，實體 hotel 多半由獨立 owner（個體 + 家族 + REIT + PE）持有，chain 收取 royalty + management fees。本 ID 同時涵蓋短租平台（Airbnb 為核心）作為連鎖旅館的相鄰 / 替代品 / 競品三重身份。

邊界判斷的關鍵是「是否賣 brand 與 loyalty」與「是否經營 RTD inventory of rooms」兩條軸：傳統 chain（MAR/HLT/H/IHG/CHH/WH）兩條都符合；Airbnb 賣的是 marketplace 不是 brand（雖然 ABNB 本身是 brand），是 in-scope 因為跟 chain 競爭 nights stayed；Booking / Expedia（OTA）是分銷層，與 chain 既合作又競爭，本 ID 視為相鄰，會在 §5 value chain 詳細討論，但會獨立另一份 OTA ID（ID_OTA_TravelDistribution，2026-05 規劃）；timeshare / vacation ownership（如 MGM Resorts、Marriott Vacations Worldwide VAC）屬度假權結構，在本 ID 視為相鄰簡略提及；郵輪 / 機場 concession 屬兄弟 ID ID_LuxuryTravelCruise。

💡 §1 Insight
1. 「hotel vs Airbnb」二元框架是 2018 年觀念：2025 真實結構是 K 形雙贏 — chain 在商務 + 高端 + 集團出差 + 90 天以下旅遊 / ABNB 在 leisure + 30 天以上長住 + 第三世界 emerging market。MAR Homes & Villas 183k properties 表示 chain 已主動 hedge ABNB 領域；ABNB 引入 Experiences 表示 ABNB 反過來想吃 chain 的 ancillary。
2. conversion 是 cycle 末段最被低估的 alpha：IHG 52% conversion + 443 hotels openings 是「無資本 NUG」極致範例；conversion 風險低、cycle bottom 加速（owner 想換 brand 救援）— 這跟 cycle 末段景氣判斷反向。HLT Spark / Tempo（2023 launch）+ IHG Vignette / Ruby（2024 收購）+ MAR Autograph 都是 conversion-friendly soft brand，是未來 3 年 NUG 結構性順風。

§2 連鎖旅館演進史 + S 曲線 + 歷史類比

連鎖旅館過去 70 年歷經 4 個結構性轉折：(1) 1950-1970 是 asset-heavy chain 黃金期 — Holiday Inn (1952)、Hilton 國際擴張，公司持有 hotel real estate；(2) 1970-1995 是 brand 多元化 + franchise 試行 — MAR / HLT 開始試點 franchise，但 owned 仍占大宗；(3) 1995-2010 是 asset-light 大轉型 — 1993 MAR 拆 Marriott International（運營）+ Host Marriott（REIT）為里程碑事件，2007 HLT 被 Blackstone 私有化後重塑 brand portfolio；(4) 2010-2026 是 loyalty platform + non-RevPAR fee 商務化 — Bonvoy / Honors / WoH 上升至 270M+ / 200M+ / 56M 會員，共品牌信用卡 royalty 成為非 RevPAR 流，asset-light 比例已逼近極限（HLT 97% / WH 100% / MAR 75% / H 52%）。

本次循環最特別的是 asset-light 飽和 + loyalty 商務化：sector 已經沒有更多 asset 可以 spin off（HLT 97% 已是 ceiling），下一個結構性 alpha 必須來自 fee mix shift（co-brand royalty / experiential ancillary / paid search）。Marriott 2026 共品牌信用卡 royalty +35% 重定價是這個轉型的標誌性事件 — 從 Chase / Amex 重新談下來的 royalty rate 永久上修，這是 hotel chain 第一次把自己重新定位為「payment platform partner」。同時 Airbnb 用 marketplace + experiences 攻擊 chain 的 ancillary，雙向 K 形分化。

★ 歷史類比（2 個正類比 + 1 個反類比）

正類比 1：1993 Marriott 拆 Marriott International + Host Marriott（spin-off 開啟 asset-light 估值重定價）。當年 MAR 把運營與不動產分拆後，Marriott International 估值倍數從 8-10x P/E 跳到 18-20x（5 年內），主因是市場認知到「brand + management」是 capital-light + recurring fee business。本次 2026 共品牌信用卡 royalty +35% 的重定價類比於此 — 從 hotel cycle 倍數（25-30x）轉移到 payment platform 倍數（35-40x）。量化對照：當年 spin-off 後 5 年 EBITDA 倍數從 10x → 16x；本次預期 fee re-rating 將推 MAR 從 EV/EBITDA 20x → 24-26x（3 年內）。

正類比 2：2007 Hilton 被 Blackstone $26B 私有化（資本結構重塑）。Blackstone 收 Hilton 後重塑 brand portfolio + asset-light 加速，2013 IPO 時 Hilton 已是「brand-led」結構。2007-2013 6 年期間 Hilton 從 owned-heavy（~60% owned）轉成 asset-light 主導（~85%），EBITDA 倍數從 8-10x → 15-18x。量化對照：當年 6 年 fee margin 從 40% → 65%；本次 Hyatt Playa 收購類比反向（asset-light → 部分 owned），但 luxury all-inclusive 是高 ADR + 高 EBITDA margin 利基，預期 H 整體 EBITDA margin 5 年內從 15% → 18%（共識給 16-17%）。

反類比：2014 IHG 賣 InterContinental Hong Kong $938M（asset-light over-extension）。當年 IHG 賣掉旗艦 hotel real estate，市場 cheer 但實際 5 年後 RevPAR 表現不如 MAR/HLT（IHG 2014-2019 RevPAR CAGR 1.8% vs MAR 3.2% / HLT 3.0%）— 因 IHG 失去 lighthouse hotel 的 anchor effect。反類比警示：HLT 97% asset-light 已是 ceiling；MAR 若繼續 spin off 剩餘 25% owned，可能重演 IHG 失誤。MAR / HLT 的 owned hotels 是「showcase」與「new brand 試點」場域，不能完全 spin off。多條件對照：IHG 當年 (a) 賣的是旗艦級 owned，(b) 沒有 luxury brand pipeline 補位，(c) 賣後沒立即新建。MAR 目前 (a) 沒有計畫賣旗艦，(b) Ritz-Carlton + St. Regis pipeline 強勁，(c) 仍主動建 / 收 luxury — 多條件不成立，類比不適用，但仍是 cautionary tale。

💡 §2 Insight
1. 連鎖旅館的「下一波 alpha」是 fee 結構轉型，不是 asset-light 加速：HLT 97% 已是 saturation，sector 均值 ~85% 接近上限；2026-2030 若還有 alpha，必須來自 (a) co-brand royalty 重定價（MAR +35% 已啟動），(b) experiential ancillary 商務化，(c) AI direct booking 比例上升 — 這是估值倍數 step-up 的真正觸發點。
2. Hyatt Playa 收購是反 asset-light 趨勢但邏輯一致：sector 整體 asset-light 飽和的同時，luxury all-inclusive 是 owned 比例反向擴大的特例 — 因為 all-inclusive resorts 需要垂直整合（餐飲 + 活動 + 客房），純 franchise 模式無法保證體驗；H 在這個 niche 反向押注是合理戰略。但若 sector 整體 cycle bottom 來臨，Hyatt 增加的 owned 部位會放大波動 — 這是 H 短期波動性 vs MAR/HLT 較高的結構性原因。

💡 §5 Insight
1. chain 在 value chain 中的 sweet spot 是「Brand+Loyalty 雙引擎」：30-40% EBITDA margin + 低 capital intensity + 議價權持續上升，這是 sector 中最佳結構性位置。比 owner（asset-heavy）好，比 OTA（被 AI 衝擊）好，比 ABNB（regulation + take rate 低）好。
2. OTA 議價權在 2027+ 受 AI 分銷衝擊顯著弱化，間接利好 chain：當 ChatGPT / Perplexity / Genspark 等 AI 工具直接訂房（不經 BKNG / EXPE），chain 的「brand + direct booking」優勢被放大 — chain 不需付 OTA 15-30% commission；ABNB 因「marketplace = brand + 分銷一體」結構也相對受惠；BKNG / EXPE 是最大輸家。這是兄弟 OTA ID 將深入的議題。

§9.5 Kill Scenario / Steel-man 反方

本章列 3 條最有力的反方論證，每條附量化 EPS / margin 影響、falsification 事件日期、最壞情境 drawdown 估計。

反方論證 #1：「MAR co-brand royalty +35% 重定價是一次性事件，不是永久 re-rating」

路徑：Chase / Amex 在 2027 重新檢視 contract，發現 royalty rate 太高，反向 revised down 5-10%。Bonvoy 會員 churn 加速（2024-2025 ~3-5%，可能上升至 8-10%）。market 重新把 MAR 估值倍數從 32x 拉回 27x。

當前證據：(a) Chase / Amex 都在 fight 高端信用卡 attrition（Sapphire Reserve 升級 + Amex Platinum 升級都增加 fee），意味銀行 willingness to pay royalty 可能 cap；(b) Bonvoy 271M 會員中活躍會員 ~50-60%（推估），長期 active rate 下滑風險。

量化影響：若 royalty 反向 revised down 10% = MAR 2027 EBITDA -2-3% / EPS -3-5% / 估值倍數 27-30x（vs base case 32-35x）。

falsification 事件 / 日期：2027-Q2 MAR 共品牌 royalty trend disclosure（earnings call）；若連續 2 季 royalty growth slope 顯著放緩 = thesis 重創。
