💡 §0 Insight：母題的 leverage 在於「打破 training/inference 二元論」
事實基礎：Jensen Huang 2026-02 Q4 FY26 法說明確說「Compute equals revenue」、「Grace Blackwell with NVLink is the king of inference today」（[T1: NVDA Q4 FY26 transcript]），但同時 Sundar Pichai 在 GOOG Q1 2026 說「TPU 8i delivers 80% better performance per dollar than the prior generation」（[T1: GOOG Q1 2026 transcript]）— 兩家都在喊「inference 第一」，但意義完全不同：NVDA 喊的是「Blackwell 是 inference 王」（賣硬體 + 護住 ASP），GOOG 喊的是「TPU 不必跟 NVDA 比 absolute FLOPS，只要 perf/$ 贏就好」（賣 cloud + 內部消費）。
推論邏輯：當所有 stack 玩家都把自己定位成「inference winner」，市場必須拆「誰的 inference 是 commodity（拿不到 pricing power）vs 誰的 inference 是 sticky asset（多年 lifetime value）」。NVDA 推論是 commodity（任何人租 GPU 都可跑），但 GOOG TPU 推論是 sticky（綁 GCP + Gemini，Anthropic 1M 顆鎖三年）。這個區別是 §12 非共識的核心。
投資意義：Portfolio overlay 中，把 inference 母題視為「四層分流配置」：Layer 1 ASIC（AVGO 60-70% 重倉）+ Layer 2 vertical（GOOG/MSFT/META 各 50-70%）+ Layer 3 routing/caching（NET 過半）+ Layer 4 端側（AAPL/QCOM 中性偏多）。NVDA 維持但不加（已被 anchor 定價）、CRWV 結構性 short candidate、純 LLM API 廠（OpenAI / Anthropic equity 私募）valuation 風險。個股 conviction 分級 → Tier Matrix。

§1 產業定義與邊界

AI Inference Economics 母題涵蓋「從 hyperscaler 採購 silicon 到 token 計費完成」的完整資金流 × 性能流 × 毛利流三軸 framework。它不是新的子產業，而是把已有的四個子產業（推論需求、推論商業化、推論硬體供應鏈、推論基建）串成 PM 級的 cross-tier 分析框架。本 ID 的存在價值在於：當你做 portfolio decision 時，不是問「NVDA 該不該買」，而是問「同樣是看好 AI inference，配置 NVDA 30% / AVGO 30% / GOOG 30% / NET 10% vs 配置 NVDA 80% / 其他 20%，哪個 risk-adjusted 報酬更高？」這是母題 framing 的具體適用場景。

2026 年成為 inference economics 母題的觸發點，不只是「推論超過訓練」這個簡單翻轉，而是同時發生三個結構性轉折：（a）Capex 結構轉折 — hyperscaler 2026 capex $650B+，其中 ASIC 部分（GOOG TPU + AMZN Trainium + META MTIA + MSFT Maia）首次明確設計為 inference-dominant，AVGO 客製 ASIC 業務 +140% YoY 是供給端訊號；（b）毛利結構轉折 — OpenAI 推論毛利從 35% → 70%（2024-2025），但這是「享用 ASIC + caching 紅利」的暫時高點，2026-2027 純 LLM API 廠毛利會被新進者（Grok 4 $0.20/M、DeepSeek $0.07/M）拉回 33-45%；（c）Valuation 邏輯轉折 — 過去用「training capex 高峰將過」恐慌 NVDA，現在 capex 結構性遞延（GOOG 直接告訴市場「2027 capex 將顯著高於 2026」），舊的「cycle peak」估值框架失效。本 ID 即聚焦這三個轉折的交集。

邊界上，本 ID 必須清楚區分四個子題的責任分工：（1）需求與分配（哪些 hyperscaler 會買多少、買 NVDA 還是 ASIC）→ ID_AIAcceleratorDemand 已涵蓋；（2）token 計費與商業化（per-token vs per-action、毛利結構）→ ID_TokenEconomics 已涵蓋；（3）ASIC 設計服務供應鏈（AVGO / MRVL / 台系 Alchip）→ ID_AIASICDesignService 已涵蓋；（4）基建與電力（VRT / GEV / SMR）→ ID_AIDataCenter + ID_NuclearRenaissance 已涵蓋。本 ID 不重寫這些，而是把四個視角合成一張 capex/margin/valuation 框架，回答 PM 層的 cross-tier 問題。

§2 技術演進 + 歷史類比

AI 推論的技術演進可分四個時代，每個時代都對應一次 capex / 毛利結構的重整。2018-2022 訓練主導期：Transformer 架構成為主流，hyperscaler capex 以「能訓練多大模型」為勝負點，NVDA A100 是工業標準，推論被視為訓練的 by-product（既有 cluster 順便跑）。2023-2024 推論浮現期：ChatGPT 商業化後 inference token 量爆發，但 capex 仍 70% 訓練主導，因為訓練模型仍在「跑得越大越好」的 scaling 階段。NVDA H100 推出，hyperscaler 開始意識到「推論 token 的單位 cost 比訓練 wall-clock 重要」。2025 過渡期：DeepSeek R1 衝擊（推論便宜 27x）、reasoning model 出現（thinking token 5-10x output 量爆增），第一次出現「訓練停滯 / 推論加速」的市場恐慌（DeepSeek panic），但實際結果是訓練 capex 並未下降，而是推論 capex 從追趕者變成主力。2026+ 推論主導期：本 ID 的時間定位 —— hyperscaler 正式承認「inference dominant」，AVGO 客製 ASIC 業務 +140%、GOOG TPU v8 split、META MTIA 量產、MSFT FY26Q1 capex $34.9B、AMZN Trainium 3 主力。

第一階段的關鍵特徵是「NVDA 在訓練 + 推論雙吃」—— 客戶買的是 cluster 整體不分用途。第二階段出現第一道裂縫 —— GOOG TPU v4/v5 在內部訓練 + 部分推論 workload 成熟，AMZN Inferentia / Trainium 1.0 成型，但都還是 single-digit % 市佔，市場可忽略。第三階段 DeepSeek 與 reasoning 雙重衝擊出現，市場第一次開始質疑「training scaling law」，雖然這個 panic 很快被 NVDA Q1 FY26 業績反證（$26B DC + 80%+ YoY 成長），但結構訊號已埋下：市場對「token cost 下降」的反應比對「FLOPS 提升」更敏感，這個 framework shift 影響後續所有估值邏輯。第四階段 2026 是 capex 結構轉換的明確訊號 — Anthropic 1M TPUv7 deal（>$50B 金額、>1GW capacity，10/2025 公布）、META MTIA Gen 2 部署（5T inference ops/day）、Trainium 3 「nearly fully subscribed」（AMZN Q1 2026 法說），三個訊號同時出現代表 hyperscaler 用採購行動投票。

歷史類比：兩個有用、一個誤導（QC-I14 + I21）

有用類比 1：1995-2000 半導體 ASP 雙曲線（DRAM vs CPU）。1990 年代 DRAM ASP 從 $15/MB 跌到 $0.50/MB（30x 下降），同期 CPU ASP（Intel）反而上漲。為何同樣是半導體，價格走向相反？答案是「產品差異化軸不同」—— DRAM 是 commodity（規格一致 → 拼產能 → ASP 跌），CPU 是 differentiated（架構 + 軟體生態 → 護城河 → ASP 守得住）。今天推論市場也是雙曲線：純 GPU 租賃（CRWV、Lambda）= DRAM 化（ASP 跌）；NVDA Blackwell + CUDA + NIM 整合 = CPU 化（守 ASP）。對 NVDA 估值意義：別把 NVDA 當 DRAM 看（cycle peak 警示），它走的是 CPU 路徑（軟體護城河 + 換代週期 driver）。但同樣意義對 AVGO 更強—— AVGO 客製 ASIC 是「專屬 IC」（為單一客戶設計），ASP 結構上比 NVDA 還穩。

有用類比 2：2007-2014 AWS S3 / EC2 跌價 95%+ 但毛利不崩。S3 從 $0.15/GB-月 → $0.023/GB-月（85% 跌幅），EC2 從 $0.12/hour → $0.01/hour（92% 跌幅），但 AWS 毛利不只沒崩，反而從 ~25% 升到 ~30%+。為何？因為 AWS 同步推出 RDS / SageMaker / Bedrock 等 SaaS-化服務，把 monetization 從「commodity IaaS」mix-shift 到「differentiated PaaS」。今天 OpenAI / Anthropic 也在做同樣事：原始 API 跌價（GPT-4 $20 → GPT-5 nano $0.40 50x），但同步推出 ChatGPT Pro / Claude Pro / Enterprise / Opus 4 高階 tier 把高毛利留住，毛利結構從「per-token」shift 到「per-seat / per-user」。反向類比警示：但 OpenAI 跟 AWS 不同的是「Microsoft 30% 收入分成」這個結構性負擔 —— AWS 不必跟誰分成，OpenAI 必須。所以 OpenAI 走 AWS 路徑會比較辛苦，這也是 Anthropic（沒類似分成包袱）能 ARR $30B 超 OpenAI $25B 的部分結構原因（[T1: Anthropic Google Cloud expansion 2025-10]）。

誤導類比：Cisco 2000 年（網路設備 capex peak 警示）。共識常用 Cisco 1999-2001 經驗警告 NVDA「capex peak 將過 → revenue 崩 → 估值砍半」。反向類比警示（QC-I21 + I28）：（a）Cisco 客戶是 telecom（資本結構脆弱、利潤端對端薄、過半客戶後來破產）vs NVDA 客戶是 hyperscaler（手握合計 $1T+ 現金、自有現金流支撐、全球市值 top 5）；（b）Cisco 賣路由器（通用網路設備、多家替代）vs NVDA 賣的是「cost-per-token 下降 driver」（軟體 + 硬體整合，CUDA 護城河深）；（c）Cisco 2001 是「網路效應失敗」（dot-com 變現失敗）vs 2026 是「企業 AI 早期變現」（CRM Agentforce $800M ARR、Anthropic $30B、Microsoft 365 Copilot 9.4M 付費 seats）；（d）Cisco peak 時 PE 130x vs NVDA 今天 fwd PE 30x。四項條件都不對，所以 Cisco 類比是誤導，但市場一直在用 —— 這個誤用本身就是 §12 非共識的根源。

💡 §2 Insight：「ASIC 殺 GPU」是表象；真正的母題是 capex 從 cluster 整體 → 細分 use case
事實基礎：Phase I 客戶買「整套 cluster」不分訓練 / 推論用途；Phase IV 客戶買「哪一台跑哪個 workload」(GOOG TPU 8t 訓練 / 8i 推論分開設計)。
推論邏輯：當 capex 開始細分 use case，每個 use case 都會有 specialty 玩家出現。訓練仍是 NVDA + AMD 主場（HBM 容量 + 多卡互連），但推論分裂成五個子戰場：（1）大模型雲端推論 → AVGO ASIC + NVDA GB200/Rubin；（2）reasoning model（thinking token 量爆）→ NVDA + Cerebras 高 tok/s；（3）邊緣推論 → AAPL + QCOM；（4）routing/caching layer → NET + Anthropic SDK；（5）私有部署（on-prem）→ Groq + SambaNova。
投資意義：別用單一「inference winner」邏輯買股 — 五個子戰場有五個贏家。把 portfolio 分配對應每個子戰場才是 inference 母題的正確操作。個股 conviction 分級 → Tier Matrix。

§3 核心技術棧拆解

AI 推論的技術棧可拆成六層，每層都對應一段 capex 流向 + 毛利結構。第一層 silicon（NVDA / AVGO / AMD / Cerebras / Groq）：這是 capex 的入口，也是利潤池最深的位置。NVDA Q4 FY26 DC 收入 $62.3B（91.5% 收入占比，[T1: NVDA Q4 FY26]）+ 毛利 75%+；AVGO Q1 FY26 AI 收入 $8.4B +106% YoY、客製加速器 +140% YoY（[T1: AVGO Q1 FY26]）。第二層 memory + interconnect（HBM、NVLink、PCIe）：對推論的影響特別關鍵 —— KV cache 容量直接決定能 cache 多大 prompt（Anthropic 90% 折扣只能在大 KV cache 設備上成立），HBM4E 是 Phase IV 的關鍵賦能因子。第三層 serving framework（vLLM、TensorRT-LLM、SGLang）：這層快速演化，2026 重點是 reasoning model 的 thinking token 優化（speculative decoding 2.8x 加速、KV cache 壓縮）；NVDA 用 TensorRT-LLM + NIM 綁 GPU 銷售，是「軟體鎖入 + 硬體 monetization」的核心戰術。

第四層 model layer（OpenAI、Anthropic、Google Gemini、Meta Llama、DeepSeek）：商業競爭最激烈、定價權快速消蝕 —— DeepSeek R1 把推論價格打到 $0.07/M input（OpenAI equivalent 27x），Llama 4 是 open-weight（hyperscaler 可自部署不付 OpenAI）。對 inference economics 的意義：model layer 的 commoditization 把利潤從這裡擠出，流向上下游（硬體 / 雲端 / 應用）。第五層 orchestration / routing / caching：這層的關鍵能力是「讓客戶 token 消費下降」（caching + 模型切換 + 多 provider 比價）—— Cloudflare AI Gateway requests +1,200% YoY（Q1 FY25）、Anthropic prompt caching 90% 讀取折扣、GOOG Gemini implicit caching 自動 75% 折扣。商業模式特殊性：賺「節省的 token 費」而非「消費的 token 費」（與客戶利益對齊），是 token economics 中最穩定的層 —— 這層的 NET / Anthropic SDK 是結構性 alpha。第六層 application / SaaS（CRM Agentforce、MSFT Copilot、ADBE Firefly）：消化下方所有層的成本，毛利攤在這裡見真章 —— 能保毛利的（CRM $2/conversation 鎖價）vs 不能保的（ADBE Firefly fwd PE 從 30x 砍到 12x）。

💡 §3 Insight：第 5 層才是 inference 母題的 toll booth；第 1 層的真贏家是 AVGO 不是 NVDA

投資意義：母題 framing 下 inference 真贏家排序是 (1) AVGO（ASIC kingmaker）→ (2) GOOG（自有 TPU + Gemini + GCP 三層整合）→ (3) NET（routing/caching toll booth）→ (4) NVDA（仍漲但相對 ASIC 結構性 derate）。共識仍把 NVDA 排第一，這是 §12 非共識的核心。

§5 Value Chain 供應鏈

AI 推論的 value chain 可分上游、中游、下游三段：上游是 silicon + 製造（NVDA 設計 / TSMC 生產 / SK hynix HBM / ASML 設備），中游是 hyperscaler 採購 + 資料中心整合（AMZN / GOOG / MSFT / META + Oracle），下游是 model 廠 + SaaS 應用（OpenAI / Anthropic / Salesforce / Microsoft 365 Copilot / vertical SaaS）。三段中議價權近 5 年快速轉移，這是 inference 母題另一個結構訊號。

這張 value chain 的關鍵解讀有四：（1）議價權近 5 年從中游往兩端轉移 —— 2018-2022 中游（AWS / Azure / GCP）議價權最強（hyperscaler 是 silicon 大買家），2026 上游 NVDA + AVGO + HBM 三家寡占議價權結構性上升；下游能鎖價的 SaaS（CRM / MSFT / NOW）議價權也上升；中段純 IaaS 雲（CoreWeave 類）反而被擠壓。（2）毛利兩極化 —— 上游兩極（NVDA 75% / HBM 35-45%）、中游兩極（垂直整合 65% / 純 GPU 雲 個位數）、下游兩極（鎖價 SaaS 75% / 純 token 計費 35%）。（3）每段都有 commodity 端 + premium 端，差異不在「位置」而在「定價策略」。（4）NVDA 的「軟體護城河」是橫跨上中下游的特殊現象 —— CUDA 鎖在上游、NIM 鎖在中游、TensorRT 鎖在下游，這是其他玩家（包括 AVGO）都沒有的全棧鎖入。

這張議價權變遷表的後解讀：上下游的議價權都在升、中段大部分在跌，這是「啞鈴形議價權」結構 —— 跟 §3 講的「啞鈴形利潤池」吻合。對 portfolio 操作的意義：buy upstream（NVDA + AVGO + HBM）+ buy downstream lock-pricing（CRM / MSFT / GOOG）；avoid 中段純租賃（CRWV / Lambda）+ 純 token-based SaaS（ADBE Firefly 類）。

💡 §5 Insight：value chain 啞鈴化是結構性而非循環性
事實基礎：上游 NVDA + AVGO 議價權從 2021 強 → 2026 極強（hyperscaler 多角化採購反而 confirmed 寡占）；中段 hyperscaler 雲分裂（垂直整合 vs 純租賃）；下游鎖價 SaaS 結構性上升、純 token SaaS 結構性下降。
推論邏輯：value chain 議價權的轉移不是 cycle 性的（會回擺），而是結構性的（business model 差異）—— 軟體鎖入、垂直整合、定價權三個 driver 都是 5+ 年穩定的結構性訊號。
投資意義：portfolio 操作對應「啞鈴策略」—— 重倉上游（NVDA + AVGO 合計 30-40%）+ 重倉下游鎖價（GOOG + MSFT + CRM 合計 30-40%）+ 避開中段（CRWV / pure GPU rent / 純 token SaaS）。

§7 Unit Economics + ASP 動態

Inference 經濟學的 unit economics 由三個變數決定：(1) cost-per-million-tokens（單位推論成本）、(2) tokens-per-second-per-user（SLA 量度）、(3) gross margin per dollar token sold（毛利結構）。三者過去 5 年的變化決定了今天的玩家排序，未來 2 年的變化決定了 inference 母題的下半場。

(1) Token cost 歷史 + 預測（post critic Pass 2 修正）：GPT-4 2023-03 launch 時 $60/M output → 2025 末 GPT-4o ~$10/M（~6x 降幅，年複合下降 ~50%）；若比較 mini tier（GPT-4o-mini $0.60/M），則近 100x，但屬不同 quality tier — directional「token cost 暴跌」結論成立但 magnitude 取決於 quality tier 對齊。2026 GPT-5.2 $14/M output、GPT-5 nano $0.40/M、Gemini 3 Flash $3/M、Grok 4.1 $0.50/M、DeepSeek R1 $0.07/M input。預測 2027-2028 主流推論價格再降 50-70%（多數來自 ASIC 採用 + caching 普及，而非 model 演化）。

§9.5 Kill Scenario / Steel-man 反方

⚔️ 反方 #1：「ASIC 無法真正威脅 NVDA — TPU/Trainium 都只能服務自家工作流」
具體路徑：客製 ASIC 在「自家垂直整合」場景內贏 NVDA 30-40% TCO，但對外部第三方 enterprise / startup來說，TPU / Trainium / MTIA 都需要客戶用 GOOG / AMZN / META 自家 SDK + cluster orchestration，遷移成本巨大。CUDA 生態 4M+ 開發者 vs TPU XLA 開發者 < 50K，這個生態落差 5-10 年內無法閉合。

⚔️ 反方 #2：「agentic AI 商業化延後 2-3 年 — Cerebras / Groq / SambaNova 估值全崩」

§12 Non-Consensus View（共識差異 — 母題核心）

三條本 ID 與市場共識的具體分歧（QC-I12 + I30 強制三欄式）。每條附 Bear / Base / Bull 三檔（Gate 2.2）。

非共識 #1：「NVDA 不是 inference 母題的最大贏家，AVGO 才是」

市場共識
NVDA 是 AI infra 唯一 anchor（90% inference 市佔 + Blackwell/Rubin lock-in）；AVGO 是「另一個 AI infra 玩家」但次要。NVDA fwd PE 30x（cycle peak 警示）vs AVGO fwd PE 35x（已 fully priced）。
本 ID 觀點
真正贏家排序：AVGO > GOOG > META > NVDA > MSFT。AVGO 客製加速器 +140% YoY > NVDA DC +75% YoY；AVGO 60-65% ASIC 市佔（主導者，非壟斷）+ 2027 visibility > $100B 是 NVDA 之外最強基本面。AVGO 應 PE 40-45x、NVDA 維持 30x。
差異
排序差異 = NVDA 第 1 vs 第 4；AVGO PE 重估 +25%（35 → 44）；NVDA 維持 ±5%。差異程度：顯著非共識（>15% PE delta）。
Bear: AVGO ASIC visibility 砍至 $60B（hyperscaler 多源）→ PE 25x
Base: AVGO PE 40x（visibility $100B 兌現）
Bull: AVGO PE 50x（visibility 上修 $130B + 2028 ASIC capex 加速）

支撐事實：（a）AVGO Q1 FY26 客製加速器 +140% YoY vs NVDA DC +75%（增速差 65pp）；（b）AVGO 60-65% 客製 ASIC 市佔結構性穩（hyperscaler 多年 design-in 鎖死，主導者地位但非 monopoly）；（c）2027 visibility $100B vs 2025 ~$25B = 4x 增長 vs ASIC 整體 ~2.5x → AVGO 市佔 expand；（d）GOOG TPU v8 split 證實「ASIC 走向 use-case 細分」趨勢，AVGO 是 enabler。

分歧原因：市場仍以「training cycle」邏輯估 NVDA（pure-play AI infra → 高 anchor），把 AVGO 視為「半導體多元玩家」（VMware + 網路 + AI）— 但 AVGO Q1 FY26 後 AI 已是主導 segment，這個 anchor 還沒切換。切換時點將是 2026-09 AVGO Q3 FY26 法說（若 AI 收入 sequential up + visibility 上修）。

非共識 #2：「2026 capex peak narrative 是錯的 — 結構性遞延，不是 cycle」

Bull: 2027 capex +60%+ YoY（agentic AI ramp + 電力解決）→ NVDA PE 38x、AVGO PE 50x
