市場把 hybrid bonding 當成「HBM4 量產帶動的下一代封裝」，需求劇本搞錯了一代。HBM4 12-Hi（2026 量產）走的是 MR-MUF（SK Hynix）與 TC-NCF（Samsung / Micron），不是 hybrid bonding；甚至 HBM4E 16-Hi，SK Hynix 也明說維持 MR-MUF——因為 JEDEC 2025-04 把高度上限放寬到 775μm，讓微凸塊路線又續命一代。

真正已在量產的 hybrid bonding 引擎是 TSMC SoIC 邏輯堆疊（6μm pitch HVM、AMD MI400、Apple M5、3D V-Cache）。HBM 的 hybrid bonding 強制世代是 HBM5 20-Hi（2028-2029）——當層數超過微凸塊的物理極限，三家才會被迫收斂到 Cu-Cu。所以這不是「2026 爆發」，是「2026 SoIC 量產、2028 HBM 接棒」的兩段式滲透。

利潤往兩端集中：設備（BESI ~60-67% 市占 + AMAT 9% 股權與 CMP 近乎獨佔）與 IP（Adeia DBI 收租）。本報告不站「現在追」也不站「沒戲」，而判：thesis 結構性成立，但兩個純 play 的估值都已先 re-rate——時點比方向更重要。

PART I · DECISION SUMMARY

決策摘要層

三句話看完這個產業的現狀、走向與行動，後接 PM 級監測點。這一層是報告的全部重點——下面各節只是把這一層的每個判斷補上證據。

現在 / NOW

供需＝設備短缺——BESI Q1-26 訂單 €269.7M（+104.5% YoY）、hybrid bonder 單季訂單翻倍破 2024 峰值。但需求被誤讀：當前量產引擎是 TSMC SoIC 邏輯堆疊（6μm HVM、AMD MI400 / Apple M5），不是 HBM。

未來 / NEXT

HBM4 / HBM4E 16-Hi 仍走 MR-MUF / 先進 TCB（JEDEC 775μm 放寬續命一代）；hybrid bonding 的 HBM 強制世代是 HBM5 20-Hi（2028-2029 三家收斂）。2028+ 利潤池往 BESI+AMAT 設備 + Adeia IP 兩端集中，中游 OSAT 被擠。

行動 / ACTION

純 play 是 BESI（設備寡占）與 Adeia（DBI 收租），但兩者估值已 re-rate（BESI Fwd P/E ~56x、Adeia 半導體授權 +808% YoY 已部分定價）——現價非進場點。盯 HBM5 hybrid bonding 採用時點 / Samsung 16-Hi 良率 / 設備訂單 run-rate 任一轉弱降一級。

The Key Call · 本報告最重要的一個判斷

市場把 hybrid bonding 的爆發點押在「HBM4 取代 MR-MUF」上——這個前提是錯的。HBM4、甚至 HBM4E 16-Hi 都還在用 MR-MUF / TC-NCF，因為 JEDEC 2025-04 把堆疊高度上限放寬到 775μm，加上把 DRAM 晶圓磨到 30μm，微凸塊路線又能塞下 16 層，hybrid bonding 在 HBM 被往後推到 HBM5（2028-2029）那個「物理上塞不下」的世代。真正已在規模量產的 hybrid bonding，是 TSMC SoIC 把邏輯 die 疊邏輯 die——盯 HBM 量產時點的人會等錯一個 catalyst，盯 SoIC 邏輯堆疊滲透與 HBM5 世代切換的人，才盯對了戰場。

把這拆成可監測的三條線：①SoIC 滲透（當下引擎）——AMD MI400、Apple M5、NVDA Rubin Ultra（2027）/ Feynman（2028）的邏輯堆疊採用，是設備訂單的近期需求源；②HBM 世代切換（中期 step-function）——Samsung HBM4E 16-Hi hybrid bonding 良率（樣品報 ~10%）、與 HBM5 20-Hi 三家收斂時點；③IP 收租兌現（Adeia）——半導體授權營收從 Q1-25 的 4% 占比跳到 Q1-26 的 32%（+808% YoY），AMD 已於 2026-03 和解簽授權，Samsung / SK Hynix 的 per-stack 經濟要等 2027+ 續約。盯這三條，比盯任何單一估值倍數都更早看到方向。

PART II · HISTORICAL ANALOGS

兩個歷史類比，一個框上行、一個框收租

歷史不必重演，但兩個對照各給一個可操作教訓：2010-2018 EUV 雙寡占框「設備玩家能漲多久、為何 BESI 還在早期」，1995-2010 Dolby 框「收租模式值多少倍數」。

2010-2018 EUV 雙寡占——框上行的形狀，也框「現在是 2016 不是 2014」。ASML 獨家 EUV 光刻機，Carl Zeiss SMT（ASML 持股 ~25%）獨家光學系統——這是「設備 + 關鍵子系統」雙寡占的原型。BESI + AMAT 的結構同構：AMAT 2025-04 買 BESI ~9% 股權成最大股東、co-develop「業界首套全整合 die-based hybrid bonding 設備」，且 AMAT 在 hybrid bonding 必須的 CMP 製程近乎獨佔。差別在時點：ASML 的 EUV 在 2010-2014 是純投資期（無營收）、2016 才出首批客戶機台、2019-2020 才 HVM 放量；股價真正的非線性段在 EUV 從研發走向量產之後。BESI 現在約等於「2016 的 ASML」——hybrid bonding 首批客戶機台正在出、HBM 的 HVM 還沒到，這意味上行的形狀在、但時間軸還沒走完。值得提醒的反向警示：hybrid bonding 設備市場規模（~$1.3B by 2030）只有 EUV 市場的約 1/10，別把 ASML 的絕對市值天花板直接套上去。

1995-2010 Dolby 授權金——框收租模式的倍數天花板。Dolby AC-3 被 DVD Forum 訂為強制標準（1996），每台 DVD 機都得付 Dolby 授權金，創造「結構性、繞不開」的收租護城河；營收 IPO 年（2005）$328M → 2010 $923M，五年 2.8x。Adeia 的 DBI / DBI Ultra 在 hybrid bonding 是同構角色——任何用 Cu-Cu 直接鍵合的 die 在多數情境繞不開其專利族（除非自研避開，如 TSMC 走自有 IP）。Adeia 的位置約等於「1998-2002 的 Dolby」：技術已證明、首批商用（Sony CIS 2016）、大規模授權潮剛起（AMD 2026-03、UMC 2026-03 簽 / 擴授權，Samsung 的 per-stack 經濟仍懸）。差異是 Adeia 客戶集中度更高（5-7 家 memory + chip 大客戶），單一 deal 體量大但也更 binary。

PART III · SUPPLY SIDE

供給側：兩端收租，中段被擠

這個產業的「巴菲特式好生意」不在中段——不在跑製程的 foundry / OSAT，而在兩端：賣機台的 BESI+AMAT、收授權金的 Adeia。看清這個分界，是不過度押單一環節的關鍵。

供給端有三類玩家賺不同的錢：設備（BESI ~60-67% 市占的 Datacon 8800 CHAMEO 系列 + ASMPT 第二、AMAT 整合線與 CMP）、IP（Adeia DBI / DBI Ultra 收授權金，毛利近乎 100%）、製程整合（TSMC SoIC 自有 IP / Intel Foveros / Samsung X-Cube）。隱形鎖喉點是 CMP——hybrid bonding 要求介電層表面粗糙度 RMS 0.1-0.5nm、Cu pad 凹陷 ≤5nm，這種超平整化只有 AMAT 有量產資歷，等於 AMAT 在 hybrid bonding 供應鏈握了一個「不管誰做 bonder 都得先過 CMP」的結構性槓桿。

Inference · 供給
利潤往設備與 IP 兩端集中，中段被結構性擠壓
前提：BESI 設備毛利 ~63.5% + Adeia 授權毛利 ~100% + AMAT CMP 近乎獨佔 ＞ 中段 OSAT 毛利 ~20%

當價值往「賣機台」與「收授權」兩端集中，中段的 foundry / OSAT（除 TSMC 自有 IP 外）變成「量大錢薄」的承包方——這跟舊式封裝（OSAT 主導）的利潤分布相反。最硬的兩個 asymmetric bet 是 BESI（設備寡占）與 Adeia（IP 收租）；TSMC 受惠但被全公司業務稀釋，Intel 用 Adeia 技術還得付授權金、margin 結構更差。

可證偽條件：若 Korean 新進者（Hanmi、Hanwha、LG）或 ASMPT 在 2027-2028 搶走 BESI >10pp 份額，或 Samsung / SK Hynix 自研繞過 Adeia 成功，則「兩端收租」的議價權被稀釋、本論失效。

PART IV · DEMAND SIDE

需求側：SoIC 先量產、HBM 後接棒

最不可替代的需求是 TSMC SoIC 邏輯堆疊（已在 6μm HVM、AMD/Apple/NVDA 鎖定），最被誤讀的是 HBM——HBM4 不用 hybrid bonding、HBM5 才強制。這個一代之差，是把 thesis 從「2026 爆發」打成「兩段式滲透」的根源。

需求分兩條時間線。近端（已量產）是邏輯堆疊：TSMC SoIC 6μm pitch HVM，AMD MI400（432GB HBM4、12 顆 compute/IO die 用 SoIC-X 疊）、Apple M5 Pro/Max/Ultra（SoIC-mH，Apple 首個 chiplet 設計）、所有 3D V-Cache，再加 NVDA Rubin Ultra（2027）/ Feynman（2028）排隊。遠端（2028+ 接棒）是 HBM：當 HBM 堆到 20-Hi（HBM5），微凸塊在 JEDEC 高度限制下物理上塞不下，hybrid bonding 才從「可選」變「強制」。

Inference · 需求
兩段式滲透——盯錯時間線會等錯 catalyst
前提：SoIC 6μm 已 HVM + HBM4 16-Hi 仍走 MR-MUF（JEDEC 775μm）+ HBM5 20-Hi 物理強制 hybrid（2028-2029）

設備需求不是「HBM4 一爆全來」，而是「2026-2027 靠 SoIC 邏輯堆疊撐、2028-2029 靠 HBM5 接棒」的兩段。把全部押在 HBM4 catalyst 的人會在 2026 失望（因為根本沒用 hybrid bonding），但把 SoIC 滲透 + HBM5 世代切換當兩條獨立需求線盯的人，能看出設備訂單的真實節奏。

可證偽條件：若 fluxless TCB（ASMPT AOR、K&S APTURA）在 sub-10μm 良率與成本上勝出，把 HBM5 的 hybrid bonding inflection 再往後推；或 SoIC 邏輯堆疊需求被 AI capex digestion 砍單——則兩段引擎任一熄火。

多空為何都對

多頭測的是設備訂單與 SoIC 滲透（BESI 訂單 +104%、SoIC 6μm HVM、AMD/Apple 鎖定）——這是真的；空頭測的是HBM 接棒時點與估值（HBM4 不用 hybrid、Adeia 的 per-stack 經濟未鎖、BESI Fwd P/E 56x）——這也是真的。兩者測的不是同一件事，所以可以同時為真：當下 SoIC 需求確實緊（多頭對），但 HBM 接棒要等 2028 且 pure play 已先漲（空頭對）。本報告因此把判斷重心放在「兩段引擎的銜接時點」與「估值是否已 price 整個劇本」。

PART V · VERDICT & SCENARIOS

裁決與三情境（24–36M）

NC#1（利潤兩端集中）判 INTACT、NC#2（HBM 一代之差、量產引擎是 SoIC）判 INTACT、NC#3（Adeia 收租被低估）判 AT_RISK——Samsung 繞過 Adeia 與估值已先 re-rate 兩條 kill 未排除，故 thesis 評級 mixed、conviction mid。

PART VI · TAXONOMY

個股特徵與 conviction tier

本框架只輸出「特徵」與「分類」，買哪一檔由個股 DD 系統與 Pure MA 訊號決定。以下 tier 為產業推論的特徵歸類，提及公司僅為「該特徵的代表案例」，不附帶買賣建議。

CORE 🔴

BESI.AS

設備寡占純 play（~60-67% 市占、Q1-26 訂單 +104% YoY、hybrid bonder 訂單翻倍破 2024 峰值、AMAT ~9% 股東）。最像「2016 的 ASML」——上行形狀在但時間軸未走完。風險全壓在估值：Fwd P/E ~56x、EV/EBITDA ~57x，re-rate 已先發生，不是便宜的寡占。conviction：中高（估值是主要風險）
CORE 🔴

ADEA

IP 收租純 play（Dolby-style，授權毛利 ~100%）。半導體授權 Q1-26 +808% YoY 至 32% 營收占比，AMD 2026-03 和解簽授權驗證專利強度。最 asymmetric 也最 binary：Samsung / SK Hynix per-stack 經濟未鎖（2027+ 續約是 single catalyst），市值已從 ~$1.5B 漲到 ~$3.5B，部分劇本已定價。conviction：中（binary catalyst 未定）
CORE 🔴

AMAT

CMP 鎖喉點 + BESI ~9% 股東 + co-develop 整合線。hybrid bonding 是「不管誰做 bonder 都得先過我的 CMP」的結構槓桿。但 hybrid bonding 在 AMAT 全公司營收占比小、不單獨拆分——稀釋受益、非 pure play，是把 BESI 上行透過大盤股低 beta 持有的方式。conviction：中（稀釋受益）
CORE 🔴

2330.TW

SoIC 邏輯堆疊量產唯一規模玩家（6μm HVM、56X interconnect density / 5X power efficiency vs CoWoS 2.5D、自有 IP 繞過 Adeia）。是當前 hybrid bonding 真量產引擎的承載者，但 SoIC 對 TSMC 全公司是增量非主體——受惠但被稀釋，且本就是 AI 全鏈核心持股。conviction：高（但非本主題 pure play）
MID 🟡

INTC / 0522.HK / MU / AMD

次要曝險。INTC：Foveros Direct/Foveros-B 追趕（9μm→3μm），需付 Adeia、margin 稀釋。0522.HK ASMPT：設備第二玩家，但近期 HBM 主力是 TCB（+146% YoY）非 hybrid bonding，hybrid 是 2027+ 事件。MU：HBM4 採用者，hybrid bonding 專利組合最大但 HBM4 12-Hi 仍用 TC-NCF。AMD：MI400 / 3D V-Cache 採 SoIC-X，與 Adeia 2026-03 已和解簽授權。conviction：中（曝險真實但非主導）
EDGE 🟢

AAPL / 005930.KS / 000660.KS

邊緣連動。AAPL：M5 Pro/Max/Ultra 採 SoIC-mH（首個 chiplet 設計），但封裝對 Apple 估值是極小變數。005930.KS Samsung：HBM4E 16-Hi 押 hybrid bonding（樣品 ~10% 良率）+ X-Cube + foundry 追趕——多軸落後但 single binary。000660.KS SK Hynix：HBM 龍頭但 HBM4/HBM4E 16-Hi 維持 MR-MUF，hybrid bonding 對其反而是「被迫遷移」的成本端。conviction：低 · 觀察

估值面總結：兩個 pure play 都不便宜——BESI Fwd P/E ~56x / EV/EBITDA ~57x（10 年中位 ~27x 之上），Adeia 雖然 EV/EBIT ~18x 看似在 Dolby 區間低端，但市值已從 ~$1.5B 翻到 ~$3.5B、半導體授權 +808% 已部分 price。現處 Phase II（SoIC HVM），轉 Phase III（HBM 接棒）的條件 = HBM5 20-Hi 確認三家全採 hybrid bonding 且 Samsung per-stack 授權落地；de-rating window 在「SoIC 訂單降溫但 HBM 還沒接上」那個青黃不接的空檔。

The One Line

hybrid bonding 的真量產引擎不是 HBM4，是 SoIC 邏輯堆疊——盯 HBM 量產時點的人等錯一個 catalyst，盯 SoIC 滲透與 HBM5 世代切換、兼顧兩端收租估值的人，才盯對了戰場。

這份報告刻意不做的三件事

  1. 不告訴你哪一檔該買——個股 tier 是「特徵的代表案例」，買賣由你的個股 DD 與 Pure MA 系統決定，本份不是買入推薦。
  2. 不假裝 HBM4 已經是 hybrid bonding——它不是，2026 量產的 HBM4 走 MR-MUF / TC-NCF，hybrid bonding 的 HBM 強制世代誠實標為 HBM5（2028-2029）。
  3. 不假裝便宜——兩個 pure play（BESI / Adeia）的估值已先 re-rate，thesis 方向對不等於現價是進場點。
