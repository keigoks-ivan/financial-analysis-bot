市場對 commodity 記憶體只問一件事：「2027 會不會像 2022-23 那樣崩」。這把問題問窄了。供給端目前清楚是結構短缺——2026 commodity DRAM 合約價單季漲 55–60%、NAND 漲 70–75%，32GB DDR5 模組兩年內從 $100 漲到 $265；驅動力不只是 AI server，而是一個藏在中間的結構機制：HBM 吃掉約 23% 的 DRAM wafer、且每 GB 消耗約 3x wafer，把 commodity DRAM 的產能從內部抽走。這是「同步 cycle 被 HBM crowd-out 額外推升」的雙引擎，不是單一市場。

但本報告不站「永久成長」。Q1 2026 那種 +55–60% 的暴漲，在 1995、2017 兩次都是 late-cycle 的特徵而非起點訊號——需求被高價反向破壞、客戶提前拉貨、庫存悄悄回補。因此裁決是 Phase II 末段、late-cycle monitor active：12 個月內短缺無疑、賣方市場續行，但中期最大的兩道裂縫是「三寡占同步擴產的紀律」與「AI capex 增速降檔」。2027-28 的 trough 振幅是整個母題的核心辯論——本報告判 commodity −25–30% 的 mini cycle，而非共識的 −50% 大屠殺，因為寡占更深（5 家 vs 1995 二十多家）、HBM crowd-out 持續、中國替代僅在成熟段。

第三件事是把一個 legacy 錯誤糾正過來：WDC 不是結構性 loser。2026 的 AI HDD 是受益者——cold/warm tier 的 nearline 需求爆發、產能售罄、cloud 營收占比 ~89%、YTD 約 +170%。AI 讓 HDD 與 SSD「一起缺」，不是互相取代：訓練資料湖與推論快取需要的儲存層級不同，HDD 守冷層、QLC SSD 守溫層，兩者同時被同一股 AI 資料洪水推上去。盯 PC/mobile DRAM 出貨、hyperscaler capex 增速、庫存週數三盞燈的人，會比只盯「超級循環 yes/no」的人早 2–3 季看到轉折。

PART I · DECISION SUMMARY

決策摘要層

三句話看完記憶體母循環的現狀、走向與行動，後接 PM 級監測點。這一層是報告的全部重點——下面各節只是把這一層的每個判斷補上證據。

現在 / NOW

供需裁決＝結構短缺，但已進 Phase II 末段——2026 commodity DRAM contract +55–60% QoQ、NAND +70–75% QoQ、32GB DDR5 kit 兩年 $100→$265、HBM 吃 ~23% DRAM wafer 把 commodity wafer 抽走；三寡占毛利創歷史新高（SK hynix Q1 op margin 72%）。

未來 / NEXT

12M 短缺無疑；裂縫在中期——① 三寡占 2026 同步擴產，「2022-23 教訓使大家克制」的舊敘事開始鬆動；② AI capex 增速降檔（2026 四大 +77% 高基期、2027 估降檔）。2027-28 trough 是核心辯論：本報告判 commodity −25–30% mini cycle，非共識 −50% 大屠殺。

行動 / ACTION

三寡占（MU / SK hynix / Samsung）是最純 play，但 commodity-sensitive、已 re-rate——不追 basket；以「HBM insulation>NAND>commodity」風險排序加權。WDC 列受益（AI HDD 售罄）非 loser。PC/mobile DRAM 出貨連 2 季 <−10% / capex 增速破 +20% / 庫存破 8 週任一亮燈才是賣點。

The Key Call · 本報告最重要的一個判斷

市場吵的是「commodity 記憶體會不會像 2022-23 崩」，這是個庫存 / 需求問題；但這輪真正被低估的結構變化，是 HBM 用一個會計上看不見的機制，把 commodity 記憶體的 cycle 振幅永久壓縮了。因為 HBM 每 GB 吃約 3x wafer、且賺 ~5x margin，三寡占會理性地把有限的 wafer 優先給 HBM——這同時做兩件事：把 commodity DRAM 的有效供給結構性抽走（短缺底更硬），又讓三寡占在 cycle 反轉時有一塊高毛利的 HBM 收入墊著（trough 不會像純 commodity 時代那樣見骨）。判斷崩不崩的人盯錯了變數：下一輪 trough 的深淺，不取決於 commodity 需求掉多少，而取決於「HBM 把多少 wafer 鎖在高毛利區」與「三寡占擴產紀律守不守得住」。

中期三道裂縫具體是：①紀律裂縫——三寡占 2026 capex 同步放大（Samsung HBM 產能 +50%、Micron FY26 capex ~$25B、SK M15X），「克制」敘事鬆動，2027-28 oversupply 機率上修；②late-cycle 自我修正——commodity DRAM +55–60% 的單季暴漲，在 1995（4Mb $40→$10）與 2017（server +100% 後 2019 −40%）兩次都是 late-cycle 而非起漲訊號，高價會反向破壞 PC/手機需求並誘發提前拉貨；③capex 增速裂縫——AI server 需求建立在 hyperscaler capex 持續加速上，四大 2026 capex +77% 是高基期，2027 增速降檔是大概率。盯這三條，比盯 ASP 創新高更早 2–3 季看到轉折。

PART II · HISTORICAL ANALOGS

三個歷史類比，一個框天花板、兩個給 late-cycle 警示

記憶體是 50 年的 boom-bust 史，已歷至少 6 次 cycle，peak-to-trough −40% 到 −75%。2022-23 框出 bear 的下行深度；1995、2017 則提醒：commodity ASP 的單季暴漲，往往是 cycle 末段而非起點。

2022-23 庫存大屠殺——框住 bear 的天花板。後 COVID 的 PC＋手機需求大跌疊加 hyperscaler 暫停採購，DRAM 現貨價從 2022 Q1 peak 跌到 2023 Q3 trough 約 −50%、NAND 更深約 −65%；三寡占全進 OP 虧損，Samsung memory 2023 全年也虧。教訓兩層：致命的不是「需求消失」，而是「需求成長被高估＋三家同步擴產」兩者齊發——這正是本報告把「紀律裂縫」與「capex 增速」並列核心監測線的歷史理由。對本次的含義：記憶體是高 beta 週期股，一旦 digestion 啟動，下行空間遠大於一般成長股，所以 bear 情境不能只記「conviction 微降」，要記「股價腰斬級」。但本次的結構差異也真實：寡占更深、HBM crowd-out 把 wafer 鎖住、客戶能見度更高——這些把 trough 振幅從 −50% 拉回到本報告估的 −25–30%。

怎麼讀：三項「本次較安全」（需求底、HBM 鎖 wafer、寡占深）與兩項「本次同樣危險」（紀律、高檔起跌）並存——所以裁決是「12M shortage 無疑、但 2027-28 不可忽視」。當年純押「過剩」的空頭最終對了，但時點抓錯會被軋空一年；本次時點要看 PART I 監測點的領先指標（PC/mobile 出貨、capex 增速、ASP 轉負），而非「超級循環 yes/no」的直覺。

1995 PC bubble ＋ 2017 server boom——late-cycle 暴漲的兩個前車之鑑。1995 Windows 95 帶動 4Mb DRAM ASP 從 $40 一度走高，但 1996 供給過剩、年末跌到 $10（約 −75%）；2017-18 hyperscaler 第一波 server DRAM 牛市把 ASP 推 +100%、三寡占 OP margin 衝破 50%，但 2019 反轉、ASP 跌約 −40%。兩次的共同教訓：commodity 記憶體 ASP 的單季暴漲，常是 cycle 末段的特徵，不是起點——高價反向破壞需求（PC/手機客戶延後升級）、誘發下游提前拉貨與庫存回補，這些都在價格頂部累積反轉動能。本次的 2026 Q1 commodity DRAM +55–60% 看起來像超級循環的確認，但在 1995/2017 的框架裡，它同時是「late-cycle monitor 該開機」的訊號。差異點仍在結構：1995 有二十多家玩家共同 over-invest、2017 三家 capex 過度（共 $80B+），本次寡占更深、capex 更聚焦 HBM——所以本報告判「mini cycle」而非「大屠殺」，但 late-cycle 的方向不容否認，只是振幅較淺。

PART III · SUPPLY SIDE

供給側：三寡占之上，多了一條 HBM crowd-out 暗線

commodity 記憶體的供給故事過去是「Samsung / SK / Micron 三寡占誰擴得快」；2024 起多了一條暗線——HBM 把 DRAM wafer 結構性抽走，commodity 短缺不再只靠需求，而是被供給端的 crowd-out 鎖死。

供給端是 Samsung（全球 DRAM 第一、commodity exposure 最高）＋ SK hynix（HBM 龍頭、DRAM 第二）＋ Micron（DRAM/NAND 雙引擎＋HBM 突破中、美系唯一）的三寡占，合計約 93% DRAM share，毛利集體創歷史新高（SK Q1 op margin 72%、Samsung Q1 DS OP ₩53.7T、Micron FQ3 guide GM ~81%）。NAND 端則是三寡占＋ Kioxia（全球第二）＋ SanDisk（美系唯一純 NAND）的五玩家格局。瓶頸是HBM crowd-out：HBM 占約 23% DRAM wafer、每 GB 吃約 3x wafer、賺約 5x margin——三寡占理性地把 wafer 優先配給 HBM，commodity DRAM 的有效供給因此被結構抽走。這是 commodity 短缺的供給側物理基礎。

Inference · 供給
HBM crowd-out 把 commodity 的 cycle 振幅永久壓縮（但沒取消 cycle）
前提：HBM 占 ~23% DRAM wafer ＋ 每 GB 吃 3x wafer ＋ 賺 ~5x margin ＋ 三寡占擴產優先給 HBM

當三寡占把有限 wafer 優先配給高毛利 HBM，commodity DRAM 的有效產能被結構抽走——這把 commodity 短缺底拉硬（供給少了一塊），同時讓三寡占在 cycle 反轉時有 HBM 高毛利收入墊著，trough 不會像純 commodity 時代見骨。2022-23 式「三家 DRAM 產能齊發 → 崩」的劇本因此變難複製：產能想衝，也得先在「commodity vs HBM」之間做取捨，而理性選擇永遠偏 HBM。

可證偽條件：若 2027 AI / HBM 需求放緩、三寡占把 wafer 重新轉回 commodity（HBM wafer 占比回落、commodity 供給回升、contract price 連 2 季 < −10%），則「crowd-out 保護」失效、commodity 過剩風險前移——這正是 PART I 監測點 3 的設計理由。

PART IV · DEMAND SIDE

需求側：AI server＋DC NAND是結構的，commodity 補漲是循環的

最不可替代的是 AI server DRAM 與 DC NAND 的結構需求（DC NAND 2026 首次超越 mobile、15 年來頭一遭）；最脆弱的是「commodity ASP +55–60% 暴漲能否持續」這個 late-cycle 的隱含假設。

需求要拆兩塊看。結構塊：AI server DRAM（hyperscaler 採購占 server DRAM 60%+）＋ DC NAND（QLC enterprise SSD 升級，2026 首次超越 mobile NAND），這兩塊是 AI 資本週期的直接延伸，需求底盤深。循環塊：commodity DRAM（DDR5 PC/一般 server）＋ consumer NAND，這兩塊隨 PC/手機景氣動，目前因 HBM crowd-out 與 AI server 拉貨被推到 +55–60% / +70–75% QoQ，但這個漲幅本身建立在「需求持續＋庫存不回補」的假設上。下游出錢方仍是四大 hyperscaler，2026 capex 合計約 $725B（+77% YoY），其中約 75% 投 AI 基建——需求廣度稀釋了單一買家砍單的傳導，但增速本身在減速，這是裁決從「結構性」打成「mixed」的根源。

怎麼讀：三寡占 DRAM 加總對上寬口徑 DRAM TAM 大致對得起來，缺口 <20%、合理（口徑差＋HBM 計入）。真正要對帳的是「commodity ASP +55–60% 能不能持續」——這建立在 capex 持續加速＋下游不提前拉貨上；若 2027 capex 增速顯著降檔、或 PC/mobile 因高價需求破壞而砍單，commodity contract price 就會率先轉負（NAND 比 DRAM 更早，因 consumer 段占比高）。短期採信下游 capex（法說承諾最硬），但這是「採信」不是「確定」——late-cycle 的本質就是「看起來最強的時候，反轉動能正在累積」。

Inference · 需求
DC NAND 首超 mobile 是結構訊號，commodity +55–60% 是 late-cycle 訊號——兩者同時為真
前提：DC NAND 2026 首次超越 mobile（15 年來）＋ commodity DRAM +55–60% QoQ ＋ hyperscaler capex +77%（高基期）但 2027 增速降檔

需求的「結構塊」（AI server DRAM、DC NAND）給了短缺一個深底——DC NAND 占比首次超過 mobile，意味儲存需求重心從消費移向資料中心，這塊不隨 PC/手機景氣動。但需求的「循環塊」（commodity DDR5、consumer NAND）正在 late-cycle：+55–60% 的暴漲建立在客戶不提前拉貨、PC/手機需求不被高價破壞的假設上，而這兩個假設在 1995/2017 都被證明會在頂部失效。

可證偽條件：① 任一 Big-4 宣布 2027 capex YoY 負增長；② PC/mobile DRAM 出貨連 2 季 < −10% 且 consumer ASP 下跌；③ AI 變現缺口逼 CFO 砍實驗預算。任一發生，「commodity 補漲」由結構轉為 late-cycle 反轉。

多空為何都對

多頭測的是結構需求＋crowd-out（DC NAND 首超 mobile、HBM 抽 wafer、ASP 暴漲），空頭測的是commodity late-cycle（+55–60% 是 1995/2017 式的末段特徵、高價破壞需求）——兩者測的不是同一塊需求，可同時為真：結構塊確實在升級（多頭對），但 commodity 暴漲不代表能持續（空頭對）。本報告因此把判斷重心放在「late-cycle monitor 何時亮燈、紀律何時破」，而非「超級循環 yes/no」。

The One Line

commodity 記憶體的下一輪 trough 深淺，不取決於需求掉多少，而取決於 HBM 把多少 wafer 鎖在高毛利區——盯 ASP 創新高的人盯錯了變數，crowd-out 正悄悄把 cycle 振幅壓淺，而 late-cycle 的反轉動能也正在頂部累積。
