Thesis（本 ID 最核心非共識觀點） Intel 在 2026 年 CES 首度量產 Clearwater Forest（Xeon 6+）是全球第一款商用 glass-core CPU，這件事市場只當成「Intel 又一個 niche 技術 demo」。實際上 glass 是 Foveros 整合的一環，屬於 Intel 先進封裝生態護城河；但 18A 仍是 Intel 主戰場，節點競爭無法被 glass 替代（2026-04 patch：原「Intel 真護城河是 glass 非 18A」類比過強，18A 與 glass 是互補而非替代）。LPKF LIDE 專利到 2035+ 是隱形瓶頸；Absolics（SKC 子公司）獲 AMD MI400 訂單，2028 前獨家供應。本 ID 預測 glass 2028 佔先進封裝 7-10%（2026-04 patch：原 15%+ 過 aggressive，Clearwater Forest 量產不等於全面普及；共識 5-8% 本 ID 略偏上緣但不激進），2032 超越 organic 成為 chiplet 標配。

灰色地帶：

  1. TGV 孔徑標準未統一：Intel Clearwater Forest 採 < 50μm 孔徑，Absolics / 三星規格介於 60-75μm，TSMC CoPoS 規格未公開。誰的標準勝出，決定材料廠與設備商生意分配。
  2. Glass interposer vs glass-core CPU 區分：Intel Clearwater Forest 是「玻璃作為 core」（取代矽底板），AMD MI400 是「玻璃作為 interposer」（取代矽 interposer）。兩者市場獨立，本 ID 同時涵蓋。
  3. FOPLP（panel-level）與 glass substrate 關係：TSMC CoPoS 是 panel-level + glass 組合；Samsung 則先 glass 再 panel。本 ID 視 FOPLP 為 glass 的導入路徑之一。
💡 Insight §1
Glass substrate 的真正市場分野不是「材料 vs 封裝」而是「core 還是 interposer」：作為 core 直接取代矽底板（Intel 路線），取代整個矽成本；作為 interposer 僅取代 CoWoS-S 矽中介層（AMD / AWS 路線），與現有 CoWoS 共存。兩者投資邏輯完全不同：core 路線打擊 TSMC 2nm / 3nm 矽銷售，interposer 路線打擊 TSMC CoWoS 毛利（但不影響晶圓銷售）。
投資意義：2026-2027 Intel Clearwater Forest 代表「core 路線」，若量產順利是對 TSMC 最深威脅；Absolics-AMD 代表「interposer 路線」，主要搶 CoWoS 份額。
💡 Insight §1（二）
LPKF LIDE 專利壁壘橫跨整個 glass 生態：無論是 Absolics、Samsung、TSMC、Intel，要打出 defect-free TGV 都需 LPKF 技術或自研突破（後者 5+ 年）。這讓 LPKF 成為「玻璃世代的 ASML」— 不做終端產品但每顆 glass 晶片都付它費。

💡 Insight §2（歷史類比最精準）
2015-2022 CoWoS 週期是最接近的類比。當年：(a) TSMC 是「技術 + 產能」雙壟斷，2015-2019 毛利 55%+；(b) 設備鏈（BESI hybrid bonder）被埋沒 5 年才被市場認識，現為 AP 明星。本次類比預期：(i) Intel 2026-2028 的 glass 先發優勢毛利可達 50%+（Xeon glass-core 溢價 30-40%）；(ii) LPKF 是 glass 世代的「BESI + 早期 TSMC」—— 專利壟斷 + 被市場低估。
投資意義：LPKF（LPK.DE）是本 ID 最強 asymmetric bet；Intel 是受惠者但被自身 18A 焦慮拖累股價；Absolics 因在 SKC（011790.KS）之下，需買母公司間接受益。
💡 Insight §2（二）
市場普遍把 Intel Clearwater Forest 當「又一個 Intel niche 試驗」，類似 2013 年把 TSMC CoWoS 當 FPGA niche 應用。但Xeon 6+ 作為 server CPU flagship 訂貨量遠大於當年 Xilinx FPGA，Intel 已投入 Foveros Direct 3D + EMIB + Glass core 三合一整合，一次把先進封裝所有環節綁成 Intel 自有專利棧。若 2026 Q4 Clearwater Forest shipment 達 50K+ 單位，Intel foundry 前景（對抗 TSMC）將徹底改寫。

💡 Insight §3（Kingmaker 子技術）
TGV 鑽孔技術（LIDE）是全 glass 產業的 kingmaker。無 LIDE 或同等技術，無法商業化 glass substrate。LPKF 專利到 2035+，期間任何玩家（Absolics / 三星 / TSMC）都須買授權或支付 royalty。
次級 kingmaker：Intel 的 glass-core + Foveros + EMIB 三合一整合工藝 — 這非單一 patent，而是 complex process know-how，TSMC/Samsung 至少落後 2 年複製。
投資意義：Intel + LPKF 是兩個「獨占型」贏家；Absolics / SKC / AGC / Corning 屬「早期受惠」玩家。

投資意義：2026-2028 是 glass 最陡斜率階段，但絕對規模小 → 小市值 alpha 空間大；2028-2033 絕對規模放大，中大型標的開始分杯羹。

投資意義：2026-2027 核心配置 Intel + LPKF + SKC；2028+ 再觀察 TSMC CoPoS 進度決定是否減碼前兩者。

§9.5 Kill Scenario / Steel-man 反方

反方 1：Intel 18A 失敗，Clearwater Forest 量產受阻

具體路徑：Intel 18A 良率 2026-H2 仍 < 50%，Clearwater Forest 受 18A 製程拖累，glass-core 無法如期 ramp，Panther Lake 2026-Q4 延後，2027 市場重審 Intel foundry thesis。

當前證據：Intel 18A 內部 yield 50-60%（2025-Q4）；Nova Lake 已延至 2027，代表 18A 出貨壓力 shift 到 Panther Lake。

成真時間窗：2026-Q4 財報 → 2027-Q1 yield 更新。

反方 2：TSMC 2027 提前推出 CoPoS，Intel 獨占期縮至 1 年

具體路徑：TSMC 壓力下加速 CoPoS 研發，2027-Q4 提前量產，與 Intel 同步競爭 AMD / Apple 訂單，Intel 先發優勢溢價時間縮為 18 個月而非 36 個月。

當前證據：TSMC 2026 Q1 宣布 CoPoS mini line 先導計畫；技術能力客觀領先。

成真時間窗：2027-Q1 TSMC Symposium 揭示 CoPoS 具體時程。

反方 3：Hybrid bonding 跳過 glass 路線，2028+ 架構轉向

具體路徑：BESI + TSMC + AMAT 合作加速 hybrid bonding（die-to-die 直接），跳過 glass interposer 層。Apple / TSMC / Samsung 的 SoIC 產線直接進入全 3D，無需 glass 層。

當前證據：BESI 2025 Q3 發表 Chameleon hybrid bonder 升級版；AMAT 9% BESI 股份已確認；2025-Q4 多 hyperscaler qualifying hybrid bonding 路線。

成真時間窗：2028 HBM4 代際轉換時決定路線。

反方 4（v1.1 新增）：TSMC SoW-X 把 substrate 路線推到 >40-reticle，玻璃「必走時程」從 2028 推到 2030+

具體路徑：TSMC 在 2026 北美技術論壇對外揭示 SoW-X（System-on-Wafer-X）>40-reticle / 64xHBM 將在 2029 量產，且整套 wafer-scale assembly 仍在 substrate 路線（不需要玻璃 interposer / 玻璃面板）。配合既有 CoWoS interposer 路線圖（2026 5.5x → 2027 9.5x → 2028 14x → 2029 >14x），TSMC 證明 substrate scaling 可以一路扛到 64xHBM 規模。這意味原本本 ID thesis「玻璃基板是 2028+ 大封裝唯一解」需要降級為「玻璃基板是 2030+ 才會成為 mainstream 必要技術」。Ibiden / Shinko / Unimicron 等 substrate 路線廠商反而在 SoW-X 階段（2029 起）獲得新一波訂單動能；玻璃路線（LPKF / Corning / SKC）的時程被推遲。

當前證據：TSMC 2026 NA Symposium roadmap 明確 SoW-X 在 substrate 上做到 >40-reticle、2029 量產；CoPoS（pilot 2026 Q2 → mass 2028-2029）改稱「並行選項」而非「必走」。同時 5.5-reticle CoWoS 2026 yield 已過 98%，substrate scaling 節奏可信度高。

成真時間窗：2027 Q4 TSMC 公告 SoW-X 客戶名單（hyperscaler 採用），且 CoPoS mass production 推遲 → 玻璃必走時程從 2028 推到 2030+。本反方對玻璃 thesis 的傷害最大 — 不是「玻璃失敗」而是「玻璃時程被推遲 2 年」，這對 LPKF / SKC 的 cash flow ramp 是直接 -2 年的 IRR 折損。

💡 Insight §9.5
最有殺傷力的反方是「TSMC CoPoS 提前 + Samsung interposer 提前」雙壓，而非單一技術失敗。本 ID 的 Intel / SKC / LPKF 3 極格局核心假設是 Intel 1-2 年獨占；若縮為 6 個月，Intel 溢價 thesis 破產。
投資意義：2027-Q1 TSMC Symposium 是關鍵 catalyst；若 TSMC 宣布 CoPoS 提前到 2027，需重新 size Intel / SKC 配置。

§12 Non-Consensus View（共識差異）

🟡 非共識 1：Intel 真正護城河是 glass substrate（不是 18A）[信心: 高]
共識：賣方（Goldman、Morgan Stanley）多看 Intel 價值錨定在 18A 製程能否追上 TSMC 3nm。18A yield 是 Intel 估值主變量。
本 ID 看法：Intel 的結構性優勢在 glass-core + EMIB + Foveros Direct 三合一整合，不是 18A。理由：(a) 18A 即使追上 TSMC 3nm，foundry 體系慢形成；(b) Glass-core 是 Intel 目前 only competitor is TSMC（2028 量產）；(c) Clearwater Forest 已量產，證明 Intel glass 領先執行 2 年。
分歧原因：共識把 Intel 看作「製程落後者」，忽略其先進封裝整合能力。2024-08 Intel Foundry 白皮書已揭示此戰略。
→ 投資意義：若本 ID 正確，Intel 2027 foundry 營收 target 可能從 $15B 提升至 $25B；股價 +40-60%。
⚠ 證偽條件：2026 Q4 Clearwater Forest 出貨 < 50K 單位（不足證明先發優勢）；或 2027 Q1 TSMC CoPoS 提前宣布
🟡 非共識 2：LPKF LIDE 是隱性贏家，類比 2015 年 ASML [信心: 中-高]
共識：LPKF 被歸類為「小型雷射設備公司」，歐洲小型股，股價覆蓋少。2025 年股價平淡。
本 ID 看法：LPKF LIDE 專利 2017 授予，2035+ 到期，是 glass substrate 產業的唯一獨占技術。SKC / Samsung / Intel 都需授權。類比 2015 ASML 在 EUV 前的狀態。2026-2030 授權收入 CAGR 40%+。
分歧原因：共識低估 LIDE 壟斷性，將其視為替代技術可得。實際上 glass TGV 無同等商業化替代（2025-Q4 數據）。
