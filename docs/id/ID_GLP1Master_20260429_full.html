§1 產業定義與邊界

「GLP-1 跨域擴散母題」＝所有因 GLP-1 medication 用戶行為改變而產生 P&L impact 的下游產業集合。本 ID 是母題框架，不重複各子題的 deep-dive，重點在六域整合的 timing 與 magnitude。

把母題定義清楚很重要，否則邊界會無限擴張。本 ID 的範圍依三條準則收斂：(a) 是否有 quantified financial data——治療側的季度處方營收、食品側的 Numerator / Cornell consumer 數據、旅遊側的 Jefferies 航空研究、保險側的 RGA mortality 模型、健康側的 Planet Fitness 法說 commentary，五大 anchored data source 是各域的入場券；(b) 是否有 listed equity 受影響可投資；(c) Phase 階段——純研究、尚無 earnings impact 的域（Phase 0）排除在外。母題 ID 的結構性 leverage 在於：一個 user growth 訊號（11M → 25M）同時是六個下游域的 leading indicator，而 cross-tier ticker（同一家公司在不同域有相反 exposure）只有在母題層才能完整評估。

§2 技術成熟度 + 歷史類比

GLP-1 的跨域擴散不是 sudden event，而是「user growth → 六域 serial cascade」過程；用兩個正類比（他汀 / 吸煙）框 cascade 的速度與廣度，一個反例（SSRI）框 bear 天花板。

擴散分四個 phase：Phase I 機制驗證（2017-2024）——Ozempic 2017 FDA / Wegovy 2021 obesity / SELECT 2024 CV outcome，user 從 0 到約 6M；Phase II 行為固化（2025-2026）——Numerator / Cornell consumer 數據證實跨類別消費下降、約 11M user、Foundayo 2026-04 口服啟動；Phase III 跨域擴散（2026-2030）——Medicare Bridge + retatrutide 讀出 + 約 25M user，六域同步 cascade；Phase IV 結構固化（2030-2040）——美國成年滲透約 25-35%、慢病負擔結構釋放、mortality reduction 體現於 reinsurer reserve。

歷史類比（含正反例對照）

類比 1：他汀擴散（1987-2010）— 正類比。Lovastatin 1987 FDA、Lipitor 1996，美國他汀用戶 2008 達約 3,200 萬（22 年）。下游 cascade 包含心血管支架 / 繞道手術需求下降，以及蛋類、補充劑這類 shadow demand 結構移轉——但 sell-side 平均花 5-7 年才把反向受益放進 EPS model。對 GLP-1：protein pivot（CMG / CAVA / 高蛋白飲品）是同一 pattern，保險端 mortality 改善是他汀對壽險的翻版，只是這次有 RGA quantified。教訓：cascade 的財務體現有結構性時滯，時滯本身就是 alpha。

類比 2：吸煙 cessation（1964-1990）— 正類比。1964 Surgeon General report 後吸煙率約 42%（1965）降至約 25%（1990），cascade 橫跨菸草 / 餐飲 / 醫療 / 保險多域而非局限菸草股。關鍵差異是速度：GLP-1 user growth（11M → 25M、約 5 年）比吸煙 cessation（約 25 年）快 5-10 倍，cascade 傳導應同步加速——歷史上 25 年走完的下游重定價，這次可能 8-12 年壓縮。這把六域 cascade 的廣度框清楚。

類比 3：SSRI 抗憂鬱（1987-2005）— 反例。Prozac 1987 / Zoloft 1991，美國成年使用率約 12.5%（2010、25 年）。但 SSRI 的下游 cascade 遠比預期溫和——預期的「酒精消費下降 / 旅遊增加 / 保險 mortality 改善」全部沒顯著實現，因為「mood 改善」的行為 magnitude 遠弱於「食慾抑制」。反例警示：若 GLP-1 的行為改變 magnitude 被誇大（24 月 cohort reverse），cascade 退化為 SSRI pattern（局限藥物自身）。但目前 Numerator + Cornell + RGA + Jefferies 四條 anchored data 都顯示 magnitude 大於 SSRI；關鍵差異是 GLP-1 直接改變食慾 hormone、weight loss 是可見身體變化、必然觸發 wardrobe / fitness / social 連鎖。

🟡 §2 收斂判斷：GLP-1 = 強度介於他汀與吸煙 cessation 之間的 mass behavior change 中-高
事實 1：兩正類比證明 cascade 真實且夠寬（他汀 shadow demand / 吸煙跨四域）
事實 2：反例（SSRI）框住 bear 天花板——行為 magnitude 不足則縮回藥物自身
事實 3：cascade speed 比吸煙快 5-10 倍，但比純藥物替代（PPI）廣
事實 4：sell-side incorporation lag 歷史 baseline 5-7 年（吸煙 1965→1972 保險 reprice / 他汀 1987→1995 進 EPS model）
→ 推論：預期跨域擴散覆蓋 6+ domain 而非 2-3，速度比歷史快；對 PM 是「2026-2028 build position before consensus catches up」窗口。
⚠ 證偽：若 24 月 cohort behavior data reverse（食慾恢復 / 停藥 60%+），cascade 退化為 SSRI pattern；本 ID §8 列具體 falsification metric 含 cohort persistence。

§3 供給側：臨床護城河與六域傳導時滯

母題的「供給」稀缺不在產能，而在「臨床護城河 + 跨域傳導時滯」——這是框架逃逸的關鍵：GLP-1 不是會 over-build 產能然後價格崩跌的 commodity。

治療側供給是「兩家寡占 + picks-and-shovels 瓶頸」。LLY 與 NVO 雙寡占靠的是臨床數據護城河——tirzepatide 美國專利保護到 2036 以後、retatrutide（三激動劑）與 orforglipron（口服小分子）拉開第二代差距，NVO 以 CagriSema、amycretin 回應。真正產能瓶頸落在 picks-and-shovels：Stevanato、SCHOTT 這類 cartridge / 注射裝置寡占廠，scarcity 議價權隨注射筆量級擴張而上升。但母題關鍵稀缺不在這——而在「行為改變如何傳導到六個下游域」，每個域的 channel 與時滯都不同。

框架逃逸 · 為何不能用單一商品供需描述
跨域 cascade 不是「一種商品的供需循環」。GLP-1 的稀缺是雙層的：治療側是臨床 + 專利護城河（短期不被新進者打散），下游各域是「傳導時滯」本身——時滯越長的域（保險 mortality 24-60 月），alpha 持續越久，因為 sell-side 越晚 incorporate。所以利潤池不像 commodity 被打散，而是沿時序梯度逐域釋放。這把競爭從「誰產能大」改寫成「誰的下游時滯最長且最持久」。

§4 需求側：用戶量級驅動與三角驗證

需求最不可替代的驅動是「一個用戶量級訊號同時是六域的 leading indicator」；真正的分歧不在量級會不會成長（很可能會），而在哪些域的下游影響已被 price。

需求由一個共同引擎驅動：GLP-1 用戶量級。從約 6M（2024）到約 11M（2025）再到本 ID 估的約 25M（2030），這條曲線同時是治療處方收入、食品 volume drag、餐飲菜單結構、航空載重、健身房入會、壽險 mortality 假設六條下游曲線的源頭。2026 年 4 月 orforglipron（Foundayo）口服核准，把可及性從注射筆層級下拉到「任何時間、不限食水」的口服層級——這是 user growth 的加速器。需求端的「量」很穩（合規 + 慢病負擔 + 口服可及性），真正變數在「下游各域被 price 到什麼程度」。

🟡 §4 判斷：同一 catalyst 對六域邊際 alpha 不對稱 高
事實 1：治療 multiple 已大量 expansion（LLY 已 re-rate）
事實 2：旅遊（DAL）仍以週期股倍數交易、保險（MET）以「無聊 sector」低倍交易
事實 3：同一 catalyst（Foundayo ramp）確認的是「全域 user growth」
事實 4：已 priced 域 catalyst 只確認 EPS 能見度（10-20% upside）vs underpriced 域同時觸發 re-rating + EPS upside（30-50%）
→ 推論：catalyst 前 portfolio 應重 underpriced 域（50-60%）而非平均分布；這是跨域時序梯度的需求端對應。
⚠ 證偽：① 24 月 cohort reverse → 六域需求引擎同時熄火；② 2027-Q1 catalyst 達標但 underpriced 域未 re-rate → 「邊際 alpha 不對稱」論部分推翻。

§5 供需裁決 + 三情境 + 投資時鐘

擴散方向判 INTACT，治療護城河判 INTACT，但「underpriced 三域何時 re-rate」與「行為持續性」判 AT_RISK——故母題結構看多、conviction mid-high、進場以時序梯度排序。

三情境裁決：[X: Base（很可能）]——六域依時序梯度逐域 re-rate，underpriced 三域在 2026-Q3 至 2027-Q1 陸續進入 sell-side EPS model；cohort 24 月維持顯著行為改變。[X: Bull（可能）]——2026-Q3 至 2027-Q1 catalyst 集中達標（Foundayo + retatrutide TRIUMPH-2/3（TRIUMPH-1 −28.3% 已達標）+ Medicare Bridge），多 catalyst 同框確認主 thesis，alpha 非線性放大、underpriced 三域同步 re-rate。[X: Bear（不太可能但影響大）]——2026-Q4 至 2027-Q2 的 24 月 cohort 顯示食慾恢復 / 停藥率升至 60%+，cascade 縮回藥物自身（SSRI pattern），母題級結構打擊。

資本週期證據（治療側錨點 + 跨域注記）：本母題的資本週期主力落在治療側（詳見 ID_GLP1Treatment）——① LLY／NVO R&D 強度高（LLY R&D 約占營收 ~19%、絕對額居製藥前段）；② incretin 龍頭 ROIC 30%+ 遠高於 WACC ~9%，高報酬正吸引次世代（VKTX／ALT／RHHBY）與大廠 M&A（PFE ~$10B 收 Metsera）；③ 臨床 lead time 5–8 年構成天然數據／產能護城河。跨域擴散側（旅遊／健康／保險）本身無傳統 capex 週期，受益來自「需求外溢」而非資本投入——故母題的「資本過度湧入」風險集中在治療次世代估值，而非 cascade 下游。

§6 產業經濟學與估值傳導

跨域變數對個股估值的傳導不是線性——同一 catalyst 對不同域的 fwd PE 影響邊際不同，這是 priced-in 檢驗的核心。

本章用 priced-in reverse 推導框 underpriced 程度：以航空為例，Jefferies 已把「10% 體重下降 → 約 2% 飛機總重 → 約 1.5% 燃料成本 → 約 4% EPS」這條鏈 quantified，但 DAL 現價的隱含 EPS 路徑裡看不到這 4%——等於市場 reverse-implied「GLP-1 對航空燃料效益＝0」。保險同理：MET 以遠低於市場的倍數交易（「無聊 sector」估值），現價隱含「死亡率假設不變」，而 RGA 已把 2045 前 -3.5% 模型化。這個缺口不是市場錯，而是 sell-side incorporation lag——歷史 baseline 6-12 月，alpha 視窗就在這個 lag 內。

🟡 估值傳導：每個域的現價隱含成長 vs leading data 缺口 高
治療（LLY）：fwd 估值隱含多年高成長續航——已 reflect、容錯薄；FY26 guide 上調至 $82-85B 確認量級
旅遊（DAL）：現價隱含「無 GLP-1 燃料效益」——Jefferies 約 4% EPS 完全未進 model
保險（MET）：以「無聊 sector」低倍交易——mortality 結構 alpha 未進 reserve repricing
健康（PLNT）：部分 priced（被當週期股）——structural multi-year tailwind 未充分；HSY 公司聲稱 vs Cornell 數據矛盾＝EPS revision risk 未 priced
→ 推論：multiple expansion 邊際 alpha 不對稱——已 priced 域 catalyst confirmation 主要是 EPS 能見度（10-20%）；underpriced 域同時觸發 multiple re-rating + EPS upside（30-50%）。
⚠ 證偽：若 2027-Q1 catalyst 達標但 underpriced 域（DAL / MET / PLNT）未 re-rate（multiple 未 expansion），本 thesis 部分推翻；2027-Q3 仍未 → 整體推翻。

§7 Non-Consensus + Priced-in + Kill Scenario

四條非共識，至少一條是「跨域擴散順序與市場預期不同」；後接三條 steel-man 反方。

🟢 NC#1：旅遊 / 健康 / 保險擴散領先共識認知 12-24 個月 高
共識：sell-side 仍重點在治療 + 食品 + 餐飲，旅遊 / 健康 / 保險視為「邊際 spillover」。
本 ID：三域 anchored leading data 已 2025-2026 出版（Jefferies / RGA / PLNT），擴散時序與共識認知差 12-24 月。
事實 1：Jefferies 2026-01 publish $580M/yr top-4 fuel saving——sell-side 預估 2026-Q3 才 incorporate
事實 2：RGA 2025-11 publish mortality -3.5%——MET / PRU reserve assumption 尚未 reflect
事實 3：PLNT 公開約 50% GLP-1 用戶考慮 gym + Ro Perks 史上最成功——sell-side 視為「短期 marketing」
→ 推論：alpha 在 catalyst 前 1-2 季 build position；2026-Q3 至 2027-Q1 是 sell-side mass incorporation 期。
⚠ 證偽：若 2026-Q4 主要 sell-side report 仍未 incorporate 旅遊 / 保險 thesis → 部分推翻；2027-Q2 仍未 → 整體推翻。
🟢 NC#2：保險 timing mismatch ＝「long life-short health」pair trade alpha 高
共識：保險 sector GLP-1 影響「淨 mixed」（life 受惠 + health 受害互抵 → 中性）。
本 ID：timing mismatch（life 5-10 yr 慢 vs health 1-3 yr 快）使「long MET/PRU/RGA + short UNH/CVS short-term」是結構性 pair trade；雙端 timing 錯位使兩側都 underpriced。
事實 1：RGA mortality -3.5% by 2045 → life insurer reserve release 結構 alpha
事實 2：health insurer 2026-2028 GLP-1 cost peak window（cost 雙倍）
事實 3：5-10 yr 慢病 saving 反轉是結構 EPS 上行 driver
事實 4：MET 以低倍「無聊 sector」交易 vs UNH 相對高倍——mortality 與 cost peak 兩 timing 都未 priced
→ 推論：兩段 timing 錯位是 structural 而非 cyclical，在 6 月 timeframe 都不在 sell-side 框架內，所以兩側 underpriced 且方向相反。
⚠ 證偽：若 2027 RGA update 下調 mortality 至 -2% 以下 + health insurer 公開承認 cost burden 不嚴重 → pair trade 雙向 alpha 同時失效。
🟡 NC#3：跨域 binary catalyst 對「未 priced 擴散域」邊際 impact 大於「已 priced 成熟域」 中
共識：投資人視 Foundayo / retatrutide catalyst 主要影響治療 + 食品 + 餐飲，對旅遊 / 健康 / 保險「邊際 spillover」。
本 ID：treatment + 食品已大量 priced，catalyst 邊際 alpha 中等；旅遊 + 健康 + 保險 underpriced，同一 catalyst 邊際 alpha 顯著大（+30-50%）。
事實 1：治療 multiple expansion 2025-2026 已大量發生
事實 2：旅遊仍以週期股倍數交易——Jefferies 進 model 後預估 re-rating
事實 3：保險（MET）仍「無聊 sector」倍數——RGA 進 sell-side 後預估 re-rating
→ 推論：catalyst 前 portfolio 結構應重 underpriced（50-60%）而非 priced（30-40%）。
⚠ 證偽：若 2027-Q1 catalyst 達標但 underpriced 域未 re-rate → 部分推翻；2027-Q3 仍未 → 整體推翻。

Kill Scenario / Steel-man 反方

🟡 反方 #1：cohort 24 月停藥 60%+ → cascade 退化為 SSRI pattern 中
路徑：2026-Q4 Numerator / Cornell follow-up 24 月 data 顯示 GLP-1 user 60%+ discontinue / 食慾恢復 / 體重回升 → 跨六域行為改變 magnitude 大幅縮水。
當前證據（反 thesis）：RGA 引述產業研究——約 2/3 患者一年內停藥；24 月 attrition 可能更高；月費 $500-1000+ 是 adherence 障礙。
量化影響：跨六域同時 -10-15% EPS / 食品 winner alpha 萎縮 / 保險 mortality 從 -3.5% 退化至 -1.5%。
最壞 drawdown：跨域組合 -25-35%。falsification date：2026-Q4 Numerator 24 月 update。
→ 路徑：mass-market 2y cohort attrition 是 GLP-1 era 結構性 reset 訊號；如 SSRI 從「mood 革命」回歸「個別 mood disorder treatment」。但 Foundayo 口服可及性（不限食水、價格層級下拉）可能改善 adherence，是反方的反方。
🟡 反方 #2：MFN + biosimilar 壓低 net price → picks + spillover 失動力 中
路徑：2027-Q1 第一家 GLP-1 biosimilar approve + net price 結構下行 → LLY/NVO CapEx/R&D 縮減 → picks（STVN/SCHOTT）demand 與 spillover driver 弱化。
當前證據（反 thesis）：LLY tirzepatide 美國保護到 2036+ 未受影響 / biosimilar peptide 合成 capacity 受限。
量化影響：LLY -15-20% / NVO -25-30% / 跨域 spillover -20-30%。falsification date：2027-Q1 biosimilar approve。
→ 路徑：類比他汀 2001-2003 LOE 後 net price 下行衝擊 picks（LDL test panel）；但 obesity drug 結構抗 commoditization 較強（多重 LOE defense）。
🟡 反方 #3：跨域 thesis 是 selection bias——真實 spillover 局限 1-2 sector 低
路徑：四條 anchored data 是「研究機構選擇性發表」+「sell-side selection bias」——真實 effect size 局限治療 + 食品，旅遊 / 健康 / 保險 spillover 是 marginal noise。
當前證據（反 thesis）：RGA 3.5% mortality 是 long-tail 預估、2 年內無實質 EPS impact。
量化影響：跨域組合 -10-15% / 但治療 + 食品 thesis 仍維持。falsification date：2027-Q3 四條 data updated 後 sell-side EPS model 仍未 incorporate。
→ 路徑：若 2027-Q3 sell-side 仍未把跨域 thesis 進入 model（lag 比預期長），可能不只 lag、而是 thesis 本身過度外推。
