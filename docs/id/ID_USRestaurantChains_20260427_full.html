§1 產業白話定義 + 歷史脈絡

白話版：外食是剛需、連鎖規模化——但過去十年的「fast casual = 贏家、casual dining = 在死」這張地圖已經翻面。真實的地圖是一個 K 形：高端體驗反撲、中段通用 fast casual 失速，價值往兩端集中。

白話開場：先說三個關鍵詞。同店銷售（SSS）是「開超過一年的老店」銷售年增——它把展店灌水剔掉，是看一家餐廳「真的有沒有更多人來、買更多」的核心。K 形分化是指同一個產業裡，上端與下端往相反方向走、中段被排空。cap-light franchise是「母公司不持有店面、只對加盟商的全系統銷售抽 royalty（4-5%）」的商業模式——它最值錢的地方是把牛肉漲價、工資上升這些 P&L 風險推給加盟商，自己收的是穩定抽成。本報告問的是：當這三件事同時發生，美國連鎖餐飲這塊 $1.55T 大餅的錢，被重分配給了誰。

邊界界定：本報告聚焦「美國公開上市的連鎖餐飲 K 形分化 + 誰捕獲價值」。納入 QSR Mega（MCD/YUM/QSR/WEN）、QSR Pizza（DPZ/PZZA）、fast casual（CMG/CAVA/SHAK/WING）、casual dining（TXRH/DRI/EAT/CAKE/BLMN）、coffee（SBUX）。不納入未上市的 In-N-Out／Raising Cane's（無投資工具）、第三方 delivery 平台（DASH，獨立 ID）、食材通路（Sysco／US Foods，獨立 ID）。灰色地帶是 TXRH（客單 $25-30、有服務生但體驗品質接近 polished casual）、SBUX（80%+ drive-thru/mobile，operating model 近 QSR 但客戶心智屬 specialty coffee），本報告分別歸 casual dining 與 QSR/coffee 雙重定位。

歷史分五代。第一代：1955–1980 QSR 速食革命——franchise + drive-thru + 統一 ops（Ray Kroc 模式），MCD／Burger King／KFC 主導。第二代：1980–2007 casual dining 黃金年代——嬰兒潮 sit-down + chain 規模 + 廣告，Olive Garden／Outback／Chili's／Applebee's 主導。第三代：2007–2019 fast casual 興起——Millennials + 健康 + customizer + 食材敘事，CMG IPO 後 5 年 +500%、Panera／SHAK 跟進。第四代：2020–2024 COVID 加速——off-premise + drive-thru + delivery + loyalty 數位化，DPZ／MCD／SBUX／WING 為高滲透贏家。第五代：2025+ K 形分化 + 牛肉超級循環——casual dining 反撲 + fast casual 退潮 + 牛肉／工資結構成本，TXRH／Chili's／Olive Garden／MCD／DPZ／CAVA 為贏家。

歷史類比一：1990s-2007 casual dining 黃金期——體驗規模是護城河。嬰兒潮 sit-down 文化把 Olive Garden、Outback、Chili's 推上高峰，2007 前 casual SSS 領先 fast food。量化錨：當年 casual dining AUV $4-5M vs fast food $1.5-2M，規模差是體驗護城河的根。對本次含義：當前 casual 反撲的 AUV 已更高——TXRH 約 $8.4M、Olive Garden $5.5M——當 fast casual SSS 滑、casual 反而 +4~7%，顯示「有服務生 + 體驗」在 K 形分化時代是 alpha 而非舊框架認定的劣勢。關鍵差異：1990s driver 是經濟擴張（cyclical），本次 driver 是 fast food 結構成本（CA $20）+ casual 主動 reposition（$10.99 直接價格戰），持久性可能更強。歷史類比二：2008-2014 fast casual 興起與 2014 cattle 低點循環。CMG IPO 後 +500%、Olive Garden／Applebee's 連年負成長，Millennials 拋棄 sit-down——但那一波 driver（customizer + 食材敘事）已被全業態同化。量化錨：當年 fast casual AUV $2M、如今 $3M+（成熟）；2014 cattle herd 88.5M（當年低點）時 TXRH commodity inflation 一度 18%、margin -3~5pp，但只持續約 1.5 年；本次 herd 86.2M（更低）+ 2028 前無 expansion，結構更久。教訓：fast casual 押注要找「子段別獨立 driver」，casual 押注要把牛肉循環當可量化 known cost、用 portfolio hedge（DRI）或 cap-light royalty 規避。

本主題無單一商品 cycle，用「段別 SSS 分化 + 牛肉/工資成本」當時序錨。

關於 cycle 統計表的省略：餐飲隨總經與通膨有週期，但本主題是結構性 K 形分化（業態 reposition + 成本墊高），非單一商品 ASP cycle，故不立 cycle 統計表；以「餐飲五代演進」（QSR 速食 → casual 黃金 → fast casual → COVID 加速 → K 形分化）的編年呈現時序，並把牛肉 cattle 低點循環（2014 vs 2026）當可對照的成本錨。

為什麼這一輪 casual 反撲與 1990s 的持久性不同：1990s casual dining 黃金期最終在 2007-2014 崩解，根因是「嬰兒潮老化 + Millennials 拋棄 sit-down + fast casual 用更快更便宜的高品質食材搶走年輕客」——那是一場「需求結構搬家」。本輪不同在於三點。第一，driver 不是經濟順風而是 fast food 自己的成本墊高：CA $20（覆蓋 60+ 店連鎖）讓 fast food 與 fast casual 的價格優勢被侵蝕，casual dining 用 $10.99 體驗反過來變成「相對划算」。第二，casual 主動做了 menu engineering：Chili's 把 SKU 砍半、聚焦 3 For Me 引流 + margarita 高毛利飲品交叉銷售，Olive Garden 用「無限續杯義大利麵 + Never Ending Pasta Bowl」鎖住 value 心智，這是 1990s 那一代被動守菜單所沒有的主動 reposition。第三，TXRH 證明「等位文化 + 現切牛排體驗」能在 GLP-1 時代把消費者「少而精的 splurge」承接住——traffic +4.5% 是這套體驗護城河的硬證據，而非單純靠漲價撐 SSS。三點疊加，使本輪 casual 反撲的持久性論述比 1990s 那一輪更可辯護——但這也正是 §7 反方 1「revert 成 cyclical bounce」的攻擊面：若 2027 macro 回穩、價格優勢回到 fast food 手上，這套敘事仍可能 mean-revert。

這段歷史的可操作含義：當前 K 形同時是「1990s casual 黃金期的迴響」（體驗 AUV 在分化期是 alpha）與「fast casual 進入 maturity」（通用配方被同化）。判別一家公司站哪邊，看它賣的是「通用配方」（被同化、PE compress）還是「體驗反撲／cap-light royalty／子段別獨立 driver」（K 形兩端贏家）。而判別 casual 反撲是否 structural，看它靠的是「主動 reposition + 體驗 traffic」還是「被動漲價」——前者持久、後者 cyclical。

§2 技術 / 業態演進成熟度 + S 曲線

為什麼 K 形是現在？因為 fast casual 滲透率衝過 ~22% 段別 share 的 peak、通用配方被同化進 maturity，同時 CA $20 把 fast food 成本墊高、逼出 casual dining 的「value + 體驗」反撲——三者交會把曲線往兩端拉開。

業態演進拐點有三。fast casual 滲透 peak：第三代 fast casual 2019-2023 達約 22% 段別 share，2025-2026 增速從盛期 +12% 跌至 +3%、2025 全段別流量負成長，通用配方（無服務生 + 高品質食材 + customizer）已被全業態同化。casual dining 反撲拐點：Chili's「$10.99 3 For Me」+ TXRH 體驗 + Olive Garden 雙星菜單，直接 price competition with QSR／fast casual，Q1'26 SSS +4~7% vs fast casual 整體失速。成本墊高：CA AB 1228 $20 工資（覆蓋 60+ 店連鎖）讓 fast food 成本結構性上升、研究估 -18,000 jobs。三者交會，價值往 K 形兩端（casual 體驗 + cap-light royalty）集中、把中段通用 fast casual 排空。

§3 供給側：現在與未來 + 利潤池

2026-27 餐飲利潤往兩端集中——上游牛肉端（herd 1951 低）與母公司 royalty 端（cap-light），被擠的是夾在中間的 1P-heavy 自營店。

供給端可拆三層。上游食材 / 勞工：牛肉是 sector 最大隱稅——herd 86.2M（1951 低）、beef cow herd 27.6M（1961 低）、ground beef >$6.70/lb（+72% vs 2020）、2026 wholesale beef +6.9%（前一年 +13.9%），且 Mexico 邊境因 New World screwworm 封鎖、每年少進 120-150 萬頭 feedlot 牛，2028 前無 expansion；CA $20 工資讓 fast food 成本墊高。中游連鎖餐廳（核心）：cap-light QSR Mega（MCD royalty 4-5%、DPZ／WING 100% franchise）對 systemwide $ 抽成、與 commodity／wage margin 風險脫鉤；1P-heavy（CMG／CAVA／SHAK 自營、casual dining 大店）直接承擔牛肉 + 工資。下游消費者 / 通路：低收入 trade-down（MCD CFO 指 traffic 受 gas price 壓抑、靠 check 撐 SSS）+ GLP-1 mix-shift + 餐廳 first-party app 反撲第三方 delivery。利潤往「royalty 母公司 + 上游牛肉商」兩端，中間 1P-heavy 失血。

Steel-man 反方（最有力的 3 條）

  • 反方 1（casual 反撲是 cyclical bounce）：Chili's $10.99 + TXRH 體驗是 2024-25 macro tailwind 下的特殊現象，2027 macro 回穩 + GLP-1 + Millennials 成熟後 casual revert、重演 2008-14 的 5 年衰退。當前證據：Olive Garden 已從 #1 跌到 #3、Maggiano's 持續衰退（Q3 -4.6%）。證偽窗口：casual 段別 SSS 連 2 季 < +3%。

  • 反方 3（CAVA 高 multiple 撐不住）：CAVA 未驗證跨 region scale，若 new AUV 從 $3M 滑回 $2.7M + SSS < +2% + unit opening 滑落，fwd PE 80-150x collapse 至 50x。當前證據：fast casual PE compression 是結構必然（CMG 2010 60x → 現 30x）。證偽窗口：CAVA new class AUV + SSS。

市場最低估的：最低估的機會是分歧 1 的「EAT structural alpha（fwd PE 17x、20 連季 SSS 正）」；最低估的風險是分歧 3 的「牛肉 + GLP-1 + CA 工資同步性」與分歧 2 的「CMG catering 失敗」。三者都該獨立監測。

三條分歧裡最可操作的是分歧 1（EAT valuation gap）+ 分歧 3（三重壓力同步 → cap-light pair-trade）——前者有明確催化（structural 兌現使 re-rate），後者有明確 hedge（cap-light royalty 脫鉤）。分歧 2（CMG 觀望）次之，是轉折期的決定變數。
