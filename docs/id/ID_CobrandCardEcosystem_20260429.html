Unit Glossary：Co-brand credit card = 銀行（Issuer）+ brand（airline / hotel / retailer）合作發卡，brand 收 royalty fee；Royalty rate = 銀行付給 brand 的 fee%（典型 80-200 bps of spending）；Co-brand remuneration（airline 用語）= airline 從 issuer 銀行收到的總 fee（含 royalty + 點數 sale + bonus）；Member spending = co-brand 持卡人總刷卡 spending（DAL Amex 估算 $60-80B 2025）；Closed-loop network = issuer 同時擁有 network（如 Amex / Discover / 現在 Capital One-Discover）/ 對立 open-loop（V / MA）；Issuer = 發卡銀行（AXP / JPM Chase / Citi / COF / BofA）；Network = 支付網路（V / MA / Amex / Discover）；Cross-vertical（本 ID 用語）= co-brand partner 同時擁有 hotel + airline + retail 多 vertical 曝險；Premium cohort = 高端持卡人（Sapphire Reserve / Amex Platinum / Venture X 等 $400+ 年費卡）；Sapphire Reserve = JPM Chase 旗艦旅遊卡（$550 年費，2026 升級加 WoH Explorist）；Bonvoy royalty +35% = MAR 2026 從 Chase / Amex 重新議價 royalty rate 永久上修。所有 financials = Q4 FY25 + 2026-04 actual。

★ 本 Master ID 核心非共識 Thesis 市場用 5 種錯誤框架看 Co-brand 信用卡 ecosystem：① Co-brand royalty 是穩定但平淡的 fee 流，不是 alpha；② AXP / JPM / Citi 估值倍數已 fully price in 旅遊復甦；③ 信用卡 issuer 是「傳統金融」業，cycle 倍數 10-15x 合理；④ MAR Bonvoy +35% 是單一公司 / 單一 sector 事件；⑤ Capital One-Discover 合併是傳統 retail 信貸事件，與 cobrand ecosystem 無關。本 Master ID 認為：① Co-brand 信用卡 ecosystem 是 2025-2030 sector 中最大「橫切」alpha — MAR royalty +35% / DAL $8.2B → $10B / UAL +12% / AAL Citi 獨家 / LUV Chase amended / Hyatt Sapphire Reserve Explorist 是同一 secular re-rating 事件的不同 manifestations。royalty rate 從歷史 80-150 bps 系統性升級至 150-200 bps = sector 整體 fee 流 2030 達 $65-75B（vs 2024 ~$30B），CAGR 13-15%。② AXP / JPM / Citi / COF 估值倍數應 step-up 至「payment platform」倍數，不是傳統金融 cycle 倍數。AXP 22x ✓ 合理；JPM 13x cycle 倍數仍折讓；Citi 10x 嚴重折讓；COF Discover 整合後應 10-12x（vs 目前 ~9x）。banks 「cobrand growth 鎖定 cohort」+「premium spending 可預期」兩特性接近 V/MA payment rail 結構。③ 共品牌信用卡 spending ≈ 1% US GDP（DAL CEO Bastian 2026 評估）是被嚴重低估的整體經濟規模。Costco-Citi $80B billed 也是 1% US GDP；2 個單一 partner 加總 ~2% US GDP；全 sector 推算 5-7% US GDP，是「影子支付網路」結構性巨大但 fragmented 認知。④ MAR Bonvoy +35% 不是孤立事件 — 是 secular re-rating 起點，HLT / IHG / H 跟進 2027-2028 機率 70-80%；DAL Amex $10B 目標 達成後 UAL Chase 跟進 / AAL Citi 重議 / Hyatt 新卡 launch。每個事件提供 portfolio 加碼機會。⑤ Capital One-Discover $35.3B 合併是「第三極崛起」改變競爭格局事件。COF 從 #5 issuer 變 #2 / 收 Discover network 取得「closed-loop」能力 vs V/MA / Venture X 高端旅遊卡 + Discover hotel 網路 = 對 AXP（Membership Rewards）+ JPM Chase（Sapphire Reserve）形成第三勢力。⑥ Sapphire Reserve + WoH Explorist 2026 mid-year 是 sector 首個明確 binary catalyst — $75k spending 門檻 + Hyatt 新卡 + 6 個月 ramp。預期 6-12 個月 H WoH 會員從 56M → 65-70M / 高端 cohort spending +30-40% / Chase + Hyatt 雙引擎 alpha 兌現。⑦ 本 Master ID 推薦 4 檔 cross-vertical pair trade basket（AXP 30% + JPM 25% + Citi 25% + COF 20%）是 sector 中最完整 co-brand 母題 portfolio 工具，2026-2030 預期 alpha vs SPX +5-8% 年化。
💡 §0 5 條核心 insight（預告）
1. 共品牌信用卡 spending ≈ 1% US GDP（單一 DAL Amex partnership）：是「影子支付網路」級別規模，但 sell-side 仍把 cobrand 當「airline / hotel ancillary」單獨報告 — 整合視角看，sector 整體規模超越多個傳統 sector。
2. MAR royalty +35% 不是孤立事件 — 是 sector-wide secular re-rating 起點：DAL $10B 中目標、UAL +12% FY、AAL Citi 獨家 + Globe Mastercard、LUV Chase amended、Hyatt Sapphire Reserve Explorist 都是同一個重議浪潮的不同表現；HLT / IHG 跟進 2027-2028 機率 70-80%。
3. AXP 是 sector 中 cross-vertical 集中度最高玩家：DAL（airline 王 $8.2B）+ MAR（hotel 王 royalty +35%）+ HLT（hotel 第二）三引擎；royalty 重議全面同分潤；估值倍數 22x 已部分 price in 但 ramp 後仍有 +20-30% upside。
4. Capital One-Discover $35.3B 合併創造「第三極」改變競爭格局：COF 22% US 信用卡 balance 市占 / Discover network「closed-loop」/ Venture X 高端 / 對 AXP + JPM Chase 形成第三勢力；2027-2028 entry into 旅遊 + 高端 segment 是被低估 catalyst。
5. Sapphire Reserve + Hyatt Explorist 2026 mid-year + 新 Hyatt 卡是首個明確 binary catalyst：每年 $75k spending 門檻 = 美國 ~3-5M 高端 cohort 鎖定 / 對 H WoH 會員 conversion 加速；建議 2026-Q3 Q4 兌現 monitoring。

§1 母題定義與邊界

Co-brand 信用卡 Ecosystem 母題的範圍是美國 + 全球主要 cobrand credit card partnerships，跨 4 大 vertical（hotel / airline / retail / travel general）+ 4 大 issuer 銀行（AXP / JPM Chase / Citi / Capital One-Discover）+ 2 大 network（V / MA + Amex/Discover closed-loop）。本 ID 是 Master overview — 整合既有 6 份子議題（HotelChains / AirlineLoyalty / Boomer / CardNetworkDuopoly + 部分 LuxuryTravel + Restaurant Chain partial），unify cobrand 故事在 cross-vertical 框架。

邊界判斷的關鍵是「是否 cobrand partnership 結構」與「是否 partner 銀行的 royalty fee 流」兩條軸：airline cobrand（DAL Amex / UAL Chase / AAL Citi / LUV Chase / ALK BofA）+ hotel cobrand（MAR Bonvoy / HLT Honors / H WoH / IHG）+ retail cobrand（COST Citi / AMZN Chase / Apple Goldman / Target / Best Buy）+ travel general（Sapphire Reserve / Amex Platinum / Venture X）— 全部 in-scope；純信用卡網路（V / MA）— 部分 in-scope（cobrand 部分）；純 issuer 銀行業務（一般信用卡）— out-of-scope（除非與 cobrand 整合）；BNPL（Affirm / Klarna）— out-of-scope（非 cobrand 結構）；store cards（單一 retailer 卡）— out-of-scope（規模小）；crypto cards — out-of-scope。

💡 §1 Insight
1. 本 Master ID 整合 6 份既有子議題 + 3 個新增 sub-vertical（retail cobrand / travel general / premium cohort）：完整覆蓋 cobrand ecosystem 全 stack；單獨看每份子題都會 underprice cross-vertical alpha。
2. Premium Cohort（$400+ 年費卡）是 cobrand ecosystem 中最被低估的單一 alpha：高 ARPU + 高 sticky + 跨多 vertical 兌換；Chase Sapphire Reserve + Hyatt Explorist 2026 mid-year 是 sector 首個明確 catalyst。

§2 Co-brand Ecosystem 演進史 + 重議曲線

Co-brand 信用卡 ecosystem 過去 30 年歷經 4 個結構性轉折：(1) 1985-1995 是早期單純 mile / point cards；(2) 1995-2015 是 dual-network 整合 + 大整合（Citi Costco 取代 Amex 2016；AXP-Delta 1995 起；Chase 多 partnership 累積）；(3) 2015-2024 是 premium cohort 興起（Sapphire Reserve 2016 launch / Amex Platinum 升級 / Venture X 2021 launch）；(4) 2024-2030 是 royalty 重議擴散 + Capital One-Discover 合併 + cross-vertical re-rating。本次循環最特別的是 「royalty rate 永久 step-up + sector-wide 同步 + closed-loop 第三極崛起」三因素同時。

本次循環的關鍵驅動：(a) MAR Bonvoy royalty +35% 重議（2026）— sector-wide 範本；(b) DAL Amex $10B 中目標 + UAL Chase +12% + AAL Citi 獨家 + LUV Chase amended — airline 4 大 同步重議；(c) Sapphire Reserve + Hyatt Explorist（2026 mid-year）+ 新 Hyatt 卡 — premium cohort consolidation；(d) Capital One-Discover $35.3B 合併（2025-05 closed）— 第三極崛起。本 Master ID 認為 sector EBITDA 從 cobrand royalty 流 2024 ~$30B → 2030 $65-75B（CAGR 13-15%），比傳統金融 sector 增速快 2-3x。

§3 4 大 vertical 整合（cross-ID 引用）

① Hotel Cobrand（MAR royalty +35% 起點）

2025 規模：~$5-7B / 2030 預估：~$15B / 對應子題：ID_HotelChains

§9 風險矩陣 + Kill Scenario

Kill Scenario 反方論證 #1：Credit Card Competition Act 通過 — Durbin 擴大

路徑：2026-2028 Credit Card Competition Act 通過 → cobrand royalty 結構性壓縮 -20-30%。量化影響：4 大 issuer 都受傷 / AXP -25-30% / JPM -10-15% / Citi -15-20% / COF -10%。最壞情境 drawdown：basket -20-30%。

反方論證 #2：嬰兒潮 + 高端 cohort spending 反轉（housing 大幅下行）

路徑：housing -20%+ → 高端 cohort spending 萎縮 → cobrand royalty 下降；premium cohort 持卡萎縮。最壞情境：basket -25-35%。

反方論證 #3：Capital One-Discover 整合失敗 / 規模化困難

路徑：COF 整合 Discover network 失敗 → 高端 segment 進場無法達成 → COF 估值倍數 -20%。最壞情境：COF -15-20% / 其他 issuer 不受影響。
