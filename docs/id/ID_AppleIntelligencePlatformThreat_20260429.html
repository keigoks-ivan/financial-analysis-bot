II 早段（Sherlock 第一年）
★ 本 ID 6 條核心非共識 Thesis 市場用 5 種錯誤框架看 Apple Intelligence：① 「Apple AI 落後 OpenAI / Google」（fact: Apple 1.5B install + privacy + ecosystem 是 platform 護城河，不是輸 AI race）；② Apple Intelligence 對 Apple 估值是 net positive（fact: 主要 disrupt 第三方 app rev 是 Sherlock side effect，但對 Apple 估值倍數 step-up 30-50%）；③ Match / Bumble 是 dating fatigue 不是 Apple disruption（fact: dating fatigue + Apple Sherlock 雙重夾殺）；④ Yelp / OpenTable 是中型 niche 安全（fact: 餐廳預訂是 iOS 26.4 + Siri 直接 sherlock target）；⑤ Apple 對 Google Search $20B/年 收入是穩定（fact: Apple Intelligence Siri 取代部分 Search 是 GOOG 結構性 risk）。本 ID 認為：① Apple Intelligence 是「distributor」threat 比「product」threat 大。Apple 不需要 GPT-5 級模型 — 只需 OS-native + privacy + 1.5B install base，就能 sherlock OTA / dating / 餐廳預訂 / 部分零售；機制是「app discovery + invocation」由 Siri 主導，不再需要 third-party app brand。② Match / Bumble 結構性 decline，AI 是延緩非逆轉。MTCH paying users -5% / BMBL -16% Q4 2025 / 投 $60M AI / Concierge 翻轉 sentiment +47pp 但 paying 持續流失；類似 Yahoo 2010 「投 search + mobile」但仍被 disrupted；2027-2030 dating sector consolidation + Sherlock 雙重壓力。③ Yelp / OpenTable 是 sector 最直接 Sherlock victim。iOS 26.4 + Siri「order me lunch nearby」直接 bypass Yelp / OpenTable；Apple Maps + 餐廳評論 + 預訂整合是 2026-2027 直接 sherlock target；YELP 估值倍數 collapse 30-40% 是 base case。④ OpenAI ChatGPT 平台是 Apple 真競爭對手 — 不是 Google。Google Search 是 Apple Intelligence 「結盟者」（iOS 默認 search engine $20B/年）；OpenAI ChatGPT App Store 300+ integrations 是直接挑戰 Apple App Store；2027-2030 OpenAI 平台 vs Apple OS-native 雙頭結構。⑤ Apple Search Ads $5B → $14-20B 2030 是 Sherlock 廣告母題。Sherlock 第三方 app 後 + Apple App Store 高 intent + 設備上 attribution = ad pricing 升級 + take rate 升級；廣告 sector ID（ID_AIAdRetailMedia）的 Apple 收益。⑥ Apple Intelligence 「OS-native」缺 platform network effect 是結構劣勢。OpenAI / Google / Meta 都有 multi-end + 多開發者 network effect，Apple 只有 device-end；iOS 26.4 Siri 重啟 是 step-function 但 platform network effect 永遠落後 OpenAI；2030 OpenAI ChatGPT MAU 可能超過 iOS install base。
💡 §0 5 條核心 insight（預告）
1. Sherlock 不是 product threat 是 distribution threat：Apple 透過 OS-level integration + Siri 重啟，能不需自建 best-in-class AI 而 sherlock 第三方 app；機制是「app discovery + invocation」由 Siri 主導；對 Match / Bumble / Yelp / OpenTable / 部分 OTA 是直接威脅。
2. Match / Bumble 是「dating fatigue + Apple Sherlock + iOS 26.4 Siri」三重夾殺：paying users -5% / -16% / Q4 2025；AI Concierge 是延緩 decline 不是逆轉；2027-2030 dating sector consolidation + private equity buyout 是 base case。
3. Yelp / OpenTable 是 sector 中最直接 Sherlock victim：iOS 26.4 + Siri「book me a table」直接 bypass；YELP 估值倍數 collapse 30-40%；OpenTable（Booking 旗下）相對 BKNG 多元保護。
4. Google Search 收 Apple $20B/年 是雙刃刀：iOS 默認 search engine 對 Apple 是 services rev / 對 Google 是流量；但 Apple Intelligence Siri 取代部分 search query → Google query volume -10-20% 在 iOS 是 base case；GOOG -3-5% 估值影響。
5. Apple Search Ads $5B → $14-20B 2030 是 Sherlock 母題的 Apple 收益：sherlock 第三方 + 高 intent + 設備上 attribution = ad pricing 升級 + take rate 升級；對 Apple Services rev 加 layer。

§1 產業定義與邊界

Apple Intelligence Platform Threat 的範圍是Apple 透過 OS-native AI（Apple Intelligence + iOS 26.4 + Siri 重啟 + Apple Search Ads）對第三方 app vertical 的 Sherlock 威脅。本 ID 不是 stock-analyst 對 AAPL 的 DD（已在 docs/dd/INDEX.md），而是「橫跨 5 vertical 的 platform threat 母題」— 對應 ID_OTAandAITravel §9.5 反方論證 #2 的母題化。

邊界判斷的關鍵是「是否被 Apple OS-native AI 直接 sherlock」與「是否依賴 iOS install base 1.5B」兩條軸：直接 sherlock target（Match / Bumble / Yelp / OpenTable / 部分 OTA / 1Password / Truecaller / 翻譯 app）— 全部 in-scope；間接受影響但有 platform 護城河（BKNG / EXPE / META / Google）— 部分 in-scope；不受 Apple Intelligence 直接威脅（B2B SaaS / Salesforce / Microsoft enterprise / 中國市場全部）— out-of-scope。

歷史類比（QC-I14 + QC-I28 含正反例）

正類比 #1（Apple Intelligence Sherlock ≈ 1990s Sherlock 取代 Watson + 2014 iCloud Drive 取代 Dropbox）：Apple 過去 30 年連續 sherlock 第三方 app — Sherlock OS 工具 / iCloud / Music / Maps / 1Password。每次都從 Apple 的「OS-native」+ pre-installed 優勢出手，不需要 best-in-class 但需要 good enough；類似這次 Apple Intelligence 對 5 vertical 的 sherlock。

正類比 #2（OpenAI ChatGPT App Store ≈ 2008 Apple App Store）：Apple App Store 2008 launch 顛覆 PC 軟體生態；OpenAI ChatGPT App Store 2025 launch + 300+ integrations 是 super app 雛形；2027 H2 inflection 是 base case；OpenAI 平台 rev $5B → $30-50B 2027-2030。但 OpenAI 規模化比 Apple 慢 12-18 個月（缺 hardware lock-in）。

反類比 #1（Apple Intelligence ≠ 必然成功的 iPod 2001）：iPod 2001 launch 後 5 年 100x growth + iTunes 整合 / 是 Apple 「整合 hardware + software + service」鼻祖；但 Apple Intelligence 落後 OpenAI 12-18 個月 + Siri 多次延期 + 「OS-native」缺 platform network effect。本 ID 認為 Apple Intelligence 「成功」是 0.5x iPod success — 對 Apple Services rev step-up 但無法逆轉 OpenAI 平台領先。

反類比 #2（Match / Bumble AI revival ≠ Yahoo 2010 search + mobile）：Yahoo 投 search + mobile 但仍被 disrupted；Match / Bumble 投 AI 是「延緩」decline 不是「逆轉」；類似 dating sector 結構性 fatigue + Apple Sherlock 雙夾殺。

💡 §2 Insight
1. Apple Intelligence 是 Sherlock 30 年歷史的延續，不是新策略：1990s Sherlock / 2014 iCloud / 2024-2026 Apple Intelligence 都是同套劇本；這次規模 + AI 階段 step-up = Apple Services rev +$30-40B 2030。
2. OpenAI ChatGPT App Store 是「2008 Apple」moment 但時程慢 12-18 個月：2027 H2 inflection 是 base case；OpenAI 平台 rev $5B → $30-50B 2027-2030；OpenAI valuation $300B → $500B+。
3. Apple Intelligence 「成功」是 0.5x iPod — 對 Apple 估值倍數 step-up 30-50% 但無法 disrupt OpenAI：Apple Services rev step-up + Apple Search Ads $14-20B 2030；但 OpenAI 平台 network effect 結構性領先 Apple OS-native。

§3 5 大 Vertical Sherlock 機制深度

Vertical 1：Dating（最直接 sherlock 受害）

當前狀態：MTCH paying users -5% YoY → 13.8M Q4 2025 / Tinder -8% / BMBL -16% Q3 → 3.6M[T2: Columbian dating fatigue]；MTCH 投 $60M AI / Bumble Concierge sentiment 翻轉 +47pp 但 paying 持續流失。Sherlock 機制：iOS 26.4 + Siri 將整合 Apple-native dating UI（從 contacts / location / 興趣推薦匹配），預估 launch 2027 H2；Apple 不需要 best dating algo — 只需 OS-level integration + privacy + 1.5B install。

Vertical 2：餐廳預訂 / 評論（YELP 最 vulnerable）

當前狀態：YELP 評論 + 預訂 / OpenTable（BKNG 旗下）/ Resy / DoorDash + Grubhub。Sherlock 機制：iOS 26.4 + Siri「book me a table」直接 bypass third-party app；Apple Maps 已整合餐廳評論 + 預訂 + Apple Pay payment；2027 H2 launch Siri-driven booking。

Vertical 3：OTA / 旅遊（BKNG 多平台保護）

當前狀態：BKNG 入駐 ChatGPT App + 自家 AI / EXPE / ABNB host-side 保護。Sherlock 機制：Siri「book a flight to Tokyo」+ Apple Maps + 直接 booking；但 BKNG 多平台分散 + ABNB host-side 結構保護。

Vertical 4：零售 search / shopping（platform 保護）

當前狀態：AMZN Rufus / SHOP / Etsy。Sherlock 機制：Siri「buy this iPhone case」+ App Clip + Apple Pay；但 AMZN Prime + Walmart pickup + SHOP 商家 SDK 是 platform 保護。

Vertical 5：廣告 / 推薦（GOOG Search 結構性 risk）

當前狀態：iOS Safari 默認 Google Search $20B/年 收 Apple；Meta Ads 受 ATT 影響 priced in。Sherlock 機制：Apple Intelligence Siri 取代部分 search query → Google query volume -10-20% 在 iOS；Apple Search Ads $5B → $14-20B 2030 受益。

§9.5 Kill Scenario
