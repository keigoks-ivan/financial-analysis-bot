需求被軍備競賽推高，定價被 AI 自己通縮

市場把「AI 利好資安」當成無差別的順風，焦點錯了。AI 對資安是雙刃——它讓攻擊更便宜、更快、更可規模化（offense 被民主化：AI 釣魚量暴增、deepfake 兩年增 16 倍、agentic 攻擊能自主入侵），也讓防禦更快（agentic SOC 把 tier-1 分析師的調查自動化）。攻防同時 AI 化，結果是一場結構性的軍備競賽，把資安需求往上推。

但同一把刀的另一面，是定價相變。當 agentic SOC 自動化掉人力密集的安全營運，「按分析師人頭/seat 計價」的 SaaS 劇本就被動搖——SentinelOne 已用 Singularity Credits 把計價改成按用量（consumption）。AI 一面推高需求的量，一面可能通縮每單位的價。

所以贏家不是「最早喊 AI」的廠商，而是有最大資料飛輪 + frontier model 整合 + 平台分發的少數（CRWD / PANW / S / ZS / MSFT）——AI 看的攻擊越多、模型越準、護城河越深。本報告的判斷重心：軍備競賽把需求推高（很可能），但定價相變與「attacker 也用 AI」會抵銷多少淨優勢，是最未被 price 的一面。

PART I · DECISION SUMMARY

決策摘要層

三句話看完 AI 雙刃的現狀、走向與行動，後接 PM 級監測點。這一層是報告的全部重點——下面各節只是把這一層的每個判斷補上證據。

現在 / NOW

AI 雙刃已啟動、攻擊端領先——48% 視 agentic AI 為頭號攻擊向量、AI 釣魚點擊率 4x、deepfake 50萬→800萬（2023→25）、Arup $25M deepfake 詐騙；防禦端 agentic SOC 才剛全面開放（SentinelOne Purple AI 2026-06、CRWD Charlotte SOAR、MSFT Security Copilot）。

未來 / NEXT

3–5 年攻防軍備競賽結構性推高需求；最大裂縫是定價相變——agentic SOC 自動化 tier-1，seat-based 被 consumption/credits 取代（Singularity Credits 領先），對 per-seat ARR 是雙刃；securing-AI（AI-SPM/非人身份）開新 TAM。

行動 / ACTION

押有資料飛輪 + frontier model 整合 + 平台分發的 CRWD / PANW / S / MSFT；S 是 agentic SOC 純度最高的小盤、波動大；定價模式轉變盯 NRR / consumption mix。純單點（CHKP/FFIV）非主軸。

The Key Call · 本報告最重要的一個判斷

市場把「AI 利好資安」當無差別利多，真正的非共識是——AI 是 thesis 的雙刃：它放大需求（攻防軍備競賽），但同時通縮定價（agentic 自動化 tier-1，seat → consumption）。所以贏家不是最早喊 AI 的，而是有最大資料飛輪 + frontier model 整合的少數平台；而 AI defense 的用量計價（Singularity Credits）正在重寫 SaaS 的 per-seat 成長劇本——這是市場最沒 price 的一面。同時別忘了：attacker 也用同一把刀，淨防禦優勢能維持多少，是軍備競賽永遠的開放問題。

三道裂縫具體是：①定價相變——agentic SOC 把「賣分析師人力」變成「賣 AI 結果」，seat-based ARR 成長動能被 consumption 模型重訂（SentinelOne 已先行）；②淨優勢可被抵銷——同樣的 frontier model attacker 也在用（2025-09 中國國家級劫持 Claude Code 自主入侵 ~30 標的、AI 扛 80–90% 戰術），防禦領先不是單調遞增；③新攻擊面——AI agent / 非人身份是 securing-AI 的新 TAM，但也是新的攻擊入口（autonomous agents 已占 1/8 AI breaches）。盯這三條，比盯「AI 是不是利多」這種粗問題更早看到拐點。

PART II · HISTORICAL ANALOGS

兩場軍備競賽，框住 bear 的天花板

歷史不必重演，但兩場舊軍備競賽各給一個教訓：簽章 → 行為偵測（上一次 ML 進資安）給出「技術換代誰勝出」，反垃圾郵件給出「軍備競賽無終局、淨優勢會被抵銷」。

上一次資安 ML 換代（2013–2018）——技術領導者通吃。當攻擊從「已知病毒」演化到「無檔案 / 多型」攻擊，靠簽章比對的舊防毒失效，CrowdStrike、SentinelOne 用機器學習行為偵測（不靠簽章、靠模型判斷異常）崛起，把 Symantec/McAfee 架空。教訓清楚：資安每一次技術換代，先把模型做對、資料收最多的廠商通吃。這把本次 agentic AI 換代的 bull 框得很實——CRWD/S 這次又站在新技術（agentic SOC）的領導位，份額流失型 bear 不成立。

怎麼讀：前兩列「模型+資料贏」與上一次一致（bull 的底）；後三列「計價被重訂、attacker 對稱、需求更陡」是本次的新變數——這正是把 bear 從「份額流失」重新定位到「定價通縮 + 淨優勢抵銷」的依據。

反垃圾郵件軍備競賽（2000s 至今）——最該記住的 bear 教訓。spam filter 與 spammer 鬥了二十年，filter 越強、spammer 手法越進化，從未有一方「終結」對方——這是純軍備競賽的原型。教訓對本次的含義：AI 防禦的優勢不是單調遞增的，因為 attacker 拿到同一把刀。2025-09 中國國家級組織劫持 Claude Code 自主執行對 ~30 個國防/能源/科技標的的間諜行動、AI 獨立完成 80–90% 戰術操作、每秒數千次請求探漏，就是「attacker 也 agentic」的實證。所以 bull 不能假設「AI 防禦永遠領先」；可操作的 bull 是「軍備競賽本身把總需求推高」（無論誰暫時領先，雙方都得花錢），而非「防禦方淨勝」。

PART III · SUPPLY SIDE

供給側：資料飛輪是 kingmaker，利潤池往平台 AI 集中

反直覺但可操作的結論：AI 防禦沒有實體產能，決勝點是「誰看的攻擊最多 + 誰接 frontier model 最深 + 誰有平台把 AI 分發出去」——這三者構成新護城河，把 MSSP 的人力利潤池也一併吃掉。

供給端的競賽不是「誰先做 AI 助手」，而是「誰的資料飛輪最大」。agentic SOC 的準確度取決於訓練/推論時看過多少真實攻擊遙測——CRWD（Falcon 單一 agent 全球遙測）、ZS（全球最大資安雲流量）、PANW（Cortex）、S（Purple AI 多模型）都在拼這個飛輪。第二個決勝點是 frontier model 整合：SentinelOne Purple AI 直接組合 Anthropic Claude + OpenAI GPT + 自研 Ultraviolet 多模型；CRWD Charlotte AgentWorks 拉進 Anthropic/OpenAI/NVIDIA 當夥伴；Microsoft 自有 frontier model（OpenAI 投資）+ Security Copilot。第三個是平台分發——能把 AI 模組塞進既有平台的（CRWD/PANW）upsell 最快。

Inference · 供給
資料飛輪 + frontier model 整合是 agentic SOC 的雙重門檻
前提：agentic SOC 準確度 ∝ 看過的攻擊遙測量 + 接入 frontier model 的深度 + 平台分發觸角；三者皆規模/網路效應

當防禦的品質取決於「看過多少攻擊 × 模型多強 × 分發多廣」，這三項都是規模效應——大者越大。少數有最大資料飛輪 + 最深 model 整合 + 最廣平台的廠商會把 agentic SOC 的價值與 MSSP 的人力服務利潤池一併吃下。新進者沒有飛輪、追不上。

可證偽條件：若「開源安全模型 + 中立資料層」讓小廠也能做出同級 agentic SOC（飛輪不再稀缺），或 frontier model 廠商（OpenAI/Anthropic）直接下場做 SOC 繞過平台廠，則「飛輪門檻」論失效。

PART IV · DEMAND SIDE

需求側：攻擊統計 × 防禦 ROI的雙引擎

需求最硬的證據是「攻擊真的在 AI 化、而且見效」——AI 釣魚點擊率 4x、deepfake 詐騙單案 $25M；防禦端 ROI 也真實（AI+自動化早 98 天遏制、省 $2.22M/起）。攻防同時見效，需求自然軍備競賽式上升。

攻擊端（offense）：AI 把攻擊的成本與門檻打到地板——AI 釣魚量據報 +1,265%、82.6% 釣魚信含 AI 生成內容、AI 釣魚點擊率是傳統 4 倍；deepfake 線上量從 2023 ~50 萬增到 2025 ~800 萬（約 16 倍），Arup 工程公司一名財務員工在 deepfake CFO 視訊會議後匯出 $25M。30% 企業 2026 認為獨立身份驗證已不可靠；97% 資安專業人員擔心遭 AI 驅動攻擊、93% 預期每日 AI 攻擊。防禦端（defense）：用 AI + 安全自動化的組織比人工早 98 天遏制 breach、平均每起省 $2.22M（IBM 2024 報告；2025 版更新 ~80 天 / ~$1.9M，方向不變）——這是 agentic SOC 採購的硬 ROI。

Inference · 需求
軍備競賽推高「量」，但 AI 同時重訂「價」
前提：攻擊 AI 化（量↑）+ 防禦 ROI 可量化（採購剛需）+ agentic SOC 自動化人力（每單位人力成本↓ → 計價基礎變）

軟體類無實體庫存，以 NRR / consumption mix / AI 模組 attach rate 取代 book-to-bill。需求的「量」很穩（軍備競賽），真正的變數在「價」——當 AI 取代 tier-1 分析師，「按 seat / 按分析師人頭」計價的基礎被掏空，廠商改按用量（SentinelOne Singularity Credits）。這對 ARR 是雙刃：用量計價可能隨攻擊量上升而擴張（利多），也可能讓客戶「用多少付多少」而壓縮原本的 seat 底盤（利空）。

可證偽條件：① consumption 轉換導致平台廠 NRR 跌破 110%（seat 通縮壓過用量擴張）；② AI 模組 attach rate 停滯（AI 防禦無法轉成 ARR，只是 cost center）。

多空為何都對

多頭測的是軍備競賽推高需求量（攻擊統計、防禦 ROI），這幾乎沒爭議、很可能持續；空頭測的是定價相變與淨優勢抵銷（consumption 通縮 seat、attacker 也用 AI）。兩者可同時為真：資安總支出確實被 AI 推高（多頭對），但平台廠的 per-seat ARR 成長劇本可能被 AI 自己改寫（空頭對）。本報告因此把重心放在「量增 vs 價減」的淨效果，而非「AI 是不是利多」。

PART V · VERDICT & SCENARIOS

裁決與三情境（24–36M）

「軍備競賽推高需求」判 INTACT（攻防雙引擎），「AI 防禦轉成 ARR」判進行中、「定價相變淨效果」判 AT_RISK——故 thesis 結構性看多需求、但 conviction 卡在 mid，個股要選「飛輪 + model + 分發」三者齊備的。如硬要對應傳統商品框架：攻擊（需求）爆量、合格 AI 防禦供給稀缺，等價於結構性短缺型，稀缺來自資料飛輪門檻而非產能。

PART VI · TAXONOMY

個股特徵與 conviction tier

本框架只輸出「特徵」與「分類」，買哪一檔由個股 DD 系統與 Pure MA 訊號決定。以下 tier 為產業推論的特徵歸類，提及公司僅為「該特徵的代表案例」，不附帶買賣建議。

CORE 🔴

CRWD

agentic SOC 飛輪 + 平台分發最深（Charlotte AgentWorks 拉進 Anthropic/OpenAI/NVIDIA、Agentic SOAR 統籌整個 agent 生態）。特徵是「最大資料飛輪 + 最廣分發」，AI 雙刃中的防禦面最受益的代表案例；估值偏高需個股盯。conviction：高（估值面 caveat）
CORE 🔴

S（SentinelOne）

agentic SOC 純度最高的小盤——Purple AI 全面開放 zero-click 自主調查（多模型 Claude+GPT+Ultraviolet）、用 Singularity Credits 領先把計價改 consumption。特徵是「定價相變的領先實驗者」，雙刃兩面（最受益 AI 防禦、也最先試 consumption 風險）；小盤波動大。conviction：中高 · 高波動
CORE 🔴

MSFT / PANW / ZS

MSFT：自有 frontier model + Security Copilot + E5 最廣分發（~90% M365 可用）。PANW：Cortex agentic + 卡位 securing-AI（AI-SPM）。ZS：全球最大資安雲流量飛輪 + Red Canary。特徵是「飛輪/模型/分發各有一項頂尖」。conviction：中高
EDGE 🟡

NET / FFIV / VRNS

二階受益。NET：AI 推論流量 + bot/API 邊緣防護；FFIV：app/API 安全（LLM edge 推論流量）；VRNS：DSPM——保護 AI（Copilot）讀取的資料，securing-AI 的資料治理面。特徵是「AI 攻防的周邊受益」，非核心 agentic SOC play。conviction：中 · 視主題擴散

估值面總結：本主題的核心張力在「需求被軍備競賽推高（量增確定）vs 定價被 AI 通縮（價減未定）」。CRWD/S 是 agentic SOC 最純的代表，但 CRWD 估值高、S 是高波動小盤；MSFT 是「自有模型 + 最廣分發」的最穩受益但量體稀釋。現處 Phase II（agentic SOC 放量初期），轉空條件 = consumption 轉換讓 NRR 跌破 110% 或 attacker AI 持續跑贏（防禦淪軍備支出）；de-rating window 在「市場發現 AI 對資安是雙刃而非純利多」那一刻。買賣由個股 DD + Pure MA 系統決定。

The One Line

AI 對資安是雙刃——它用軍備競賽推高需求的量，卻可能用 agentic 自動化通縮每單位的價；贏家是有最大資料飛輪 + frontier model 整合的少數平台。

這份報告刻意不做的三件事

  1. 不告訴你哪一檔該買——CORE/EDGE 是「特徵的代表案例」，買賣由你的個股 DD 與 Pure MA 系統決定。
  2. 不把「AI」當無差別利多——明確標出 AI 對資安是雙刃（量增 vs 價減 + attacker 也用 AI），不報喜不報憂。
  3. 不假裝知道軍備競賽誰最終勝——attacker 與 defender 拿同一把 frontier model，淨優勢可持續性目前無定論，已誠實標為開放問題。
