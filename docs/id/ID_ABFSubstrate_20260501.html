半導體基板（ABF Substrate）— 寡占四強 × 缺貨第二段（2026-2028）

發布：2026-05-01 ｜ industry-analyst v1.8 ｜ 涵蓋 11 ticker（4🔴 / 4🟡 / 3🟢） ｜ 母題：AdvancedPackaging ｜ 姊妹：GlassSubstrate · HybridBondingSoIC
鮮度 · tech 🟢 · market 🟢 · judgment 🟢 · 下次 refresh：2026-Q3（Ibiden Phase 2 公告 + Rubin Ultra sample 確認） · 觸發 patch：AP ID §3 ABF 段（4 家分工 + ASP）
📌 子題定位：本 ID 是 AdvancedPackaging（AP 母題）的子章節深化，焦點在 有機 FC-BGA / ABF substrate 寡占四強的 2026-2028 缺貨循環、ASP / margin 軌跡、Rubin/Vera body size 物理瓶頸。AP 母題已涵蓋 ABF 整體歷史與 4 家市佔；本 ID 不重複，新增供需缺口量化模型、Rubin/Vera 物理拆解、私有化後 Shinko 國家隊 thesis、ASP pass-through 與上游卡位（Ajinomoto + Nittobo）的真實定價權歸屬。Glass substrate 替代論述全部委由 GlassSubstrate ID。

§0 TL;DR 卡片

主題

ABF Substrate（FC-BGA 有機基板）

主導期

2005-2032+（Rubin Ultra 後仍延續至 mainstream AI）

2025 規模

$5.26-5.42B（CAGR 10.6% → 2032 $9.55B）

關鍵玩家

Unimicron 22% / Ibiden ~18% / Shinko ~14% / AT&S ~13% / Nan Ya ~11%（Top 5 = 74%）

投資時鐘

Phase II 後期（缺貨第二段 + ASP 上行 + 寡占擴產）
本 ID 核心 Thesis（一句話） ABF substrate 缺貨從 2021-2023 的「mobile + server 重疊週期」進入 2026-2028 的「AI 結構性永續週期」，但真正的定價權已從 substrate 廠（中游）轉到 Ajinomoto + Nittobo（上游材料）—— Top 4 中游 ASP 上行多為 pass-through，毛利率擴張上限 25-28%（vs 共識認為可達 35%+）；中游真正能享受寡占溢價的是有「客戶綁定 + 政策補貼」雙重護城河的 Ibiden（Phoenix CHIPS Act 鎖 Intel/AMD）與 Shinko（JIC 私有化後鎖 Rapidus + 高階 server），而非 ASP 跟漲就能享受的 Unimicron / AT&S。

關鍵灰色地帶

(1) RDL Substrate： CoWoS-L 的 RDL 層原本算 foundry 製程，但 TSMC 把這層委外給 Unimicron / Ibiden 做 organic RDL（替代矽 interposer 中的部分功能）—— 算 substrate 廠的「上探」，但容易被 Hybrid Bonding ID 重複算。本 ID 把有機 RDL 載板算進來（屬 ABF 變體），但避開 die 級 RDL。

(2) Glass Core ABF Hybrid： AT&S 2026-Q1 開始把 glass core（玻璃中心 + ABF 兩面堆疊）作為「下一代 ABF 」推銷給 AMD / Intel [T1: AT&S 2026-04 Glass Core Press Release]。這條技術路線屬於「ABF → glass」的中間態，本 ID 在 §10 投資時鐘討論其過渡意義，但定位上仍歸 ABF（因 build-up layer 仍是 Ajinomoto 介電層）；純 glass interposer / glass-only 路線交給 GlassSubstrate ID。

(3) Body Size 邊界爭議： JEDEC 標準 substrate 上限為 120×120mm（FC-BGA）。NVDA Rubin Ultra 9.5-reticle 預計 120×150mm，已突破 JEDEC 標準需要 custom 規格。本 ID 把這類 oversize substrate 算進「ABF 高階」（仍由 Ibiden / Shinko 雙獨家），但未來如果 14-reticle (2028) 拉到 200mm+，技術上更接近 panel-level / glass，邊界會再切。

💡 §1 Insight：邊界爭議本身揭示「ABF 的物理天花板已被 Rubin Ultra 觸到」
1. JEDEC 120mm 上限就是 ABF 物理極限：Rubin Ultra 9.5-reticle 預計 120×150mm（已超 JEDEC），業界普遍當作「特殊規格」處理；但這意味 Rubin Ultra 之後只剩兩條路：(a) 繼續 customize 突破 JEDEC（warpage 暴增）/ (b) 換 glass core / panel level。本子題的 thesis 不是「ABF 永遠不被取代」，而是「ABF 在 mainstream（< 120mm）仍主導至 2032+」
2. RDL 載板 / glass core 都是 ABF 廠的「上探」與「往下相容」：Unimicron CoWoS RDL 載板營收 + AT&S glass core 都讓 ABF 廠在邊界外擴張收入，但真正的競爭壓力來自 OSAT（日月光）也在做 RDL，與 LPKF / SKC 也在做 glass —— 中游 substrate 廠面臨「兩面被夾」的競爭格局

💡 §2 Insight（從歷史類比提煉）
1. 每次 substrate 缺貨循環都有「3-4 年持續期 + 後段 capex 釋放破壞 ASP」結構。前兩次都符合：循環 1 持續至 2005 後 IDM 大舉擴產壓力；循環 2 持續至 2023 後手機 down-cycle。本次最大差異是 capex 高度集中在 Top 4，且新 capex 進度受上游 T-glass / Ajinomoto 限制 → 即使 Ibiden FY27 產能 2.5x，T-glass 只能滿足 60-70%，實質 effective capacity 增幅 1.5-1.7x，需求增幅 2.5x，缺口維持至 2028 H2
2. 循環 1 / 2 的贏家結構是「兩家獨大」（Ibiden + Shinko），但循環 3 是「四家齊漲」。差別在於 hyperscaler 都在做 multi-source（NVDA 用 Ibiden + Unimicron，AMD 用 AT&S + Unimicron，Intel 用 Ibiden Phoenix + AT&S Texas + Shinko），沒有單一玩家拿 > 30% 市佔的時代來臨。這抵銷了「寡占 → 高 ASP 漲幅 → 高毛利」的傳統劇本，毛利天花板會比循環 2 低（Unimicron 循環 2 peak GM 35% vs 本次預估 25-28%）
3. 反向類比警示：常見錯誤類比為「2018 DRAM 缺貨 → 2019 大跌」，但 ABF 與 DRAM 不同 ——（a）ABF 沒有 single product spec（每顆 GPU/CPU 都是 custom design，substrate 廠 lock-in 客戶後 1-2 年才能換 supplier）；（b）ABF demand 由 hyperscaler capex 驅動（vs DRAM 由消費電子驅動，敏感度高 5-10x）。用 DRAM 週期類比 ABF 是錯誤歸因

💡 §3 Insight：Rubin Ultra 是 ABF 物理極限的「壓力測試」，但不會是 ABF 的死刑
1. Layer count 從 18L → 26L 的 yield 損失非線性：每加 4L yield 損失從 5pp 跳至 15pp（22L+）。Rubin Ultra 早期 yield 可能僅 30-40%，意味單顆 substrate 成本將是 GB200 的 3-4 倍（$100-150 vs $30-40），這部分由 NVDA 吸收（Rubin Ultra ASP 估 $50-70K vs Blackwell $30-40K），對 substrate 廠是「ASP 跳升 + 損耗轉嫁」雙利。Top 4 在 22L+ 區段毛利可達 30%+（vs 14L 一般 server 僅 15-20%）
2. JEDEC 120mm 上限被 Rubin Ultra 突破，不等於 glass 立即取代：Rubin Ultra 9.5-reticle 雖超 JEDEC 但仍在 ABF 物理可做範圍內（warpage 用 stiffener + carrier 可控）；玻璃 core 在 14-reticle (2028) 才有真正必要性。2027-2030 是「ABF 與 glass 並行（hybrid）」的中間態，AT&S glass-core 路線是最可能贏家（Ibiden / Shinko 跟玻璃供應商配合慢，Unimicron 還在追 22L）
3. Vera CPU 的 PDN 設計是被忽略的新瓶頸：88-core / 176-thread 在 ~70×70mm 上做 power delivery，需要的低 inductance + 高 thermal dissipation 設計遠比 Grace 困難。這部分在 substrate ASP 中的「設計服務費」會獨立計（Ibiden / Shinko 各家收 NVDA $5-8M one-time NRE），市場目前只算量產 ASP，沒算 design service revenue
