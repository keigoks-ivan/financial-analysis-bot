快休閒餐飲暴力分化
Chipotle playbook 已商品化，新 alpha 是 format moat

把母題報告濃縮成一張投資決策卡：FC 段別不是一起好或一起壞，而是沿「格式護城河」暴力 K-shape——drive-thru/drinks-led/automation 贏家拉開，賣「digital+throughput+loyalty」舊樣板的退潮，urban-saturated 沒 drive-thru 的結構失敗。保留核心判斷、conviction tier、賣出訊號，砍掉考證厚度。

關於「快休閒餐飲（fast casual, FC）」這個過去十年最性感的成長 segment，市場最常見的偷懶看法是「FC 仍是最高 CAGR segment、龍頭都還能 compound」。這對 2010–2023 大致沒錯，但 2025 起整段邏輯換軌了——FC 段別 2024 連鎖店成長從 2023 的 +11.2% 暴跌到 +0.6%，2025 看似 +6% 但幾乎全靠開店（unit-driven），同店（SSS）約 0–1%，FC 不再是 share-gainer 主力。

更關鍵的是 alpha 來源換了。「Niccol playbook」（Chipotle 在 2018–2024 示範的 digital + throughput + loyalty 三件套）曾是 FC 的 alpha 配方，2025 起被全段別複製、商品化——CMG digital 衝到 38.6% plateau、FY25 出現自 2017 來首次年負同店（-1.7%）。新的 alpha 是「format moat」（格式護城河）：Shake Shack 的 drive-thru 重置、Dutch Bros 的 drinks-led 高毛利小 box、Kura Sushi 從 day-1 設計的 automation——這些是別人短期內抄不走的結構優勢。

所以本報告不押「FC = 成長利多」，而是畫出誰握 format moat、誰賣已商品化的 playbook、誰的 formula 結構性破裂。最新一季（2026 Q1）把分化講得更白：format moat 三家全勝（SHAK +4.6%、BROS sys +8.3%、CAVA +9.7%）；playbook 群內部再分裂（CMG +0.5% 觸底回升、WING dom -8.7% 持續惡化）；broken FC 加速崩（Sweetgreen SSS -12.8%、上市以來最差、restaurant-level margin 崩到 10%）。一個值得釘住的反向訊號：CAVA 不但沒「蜜月期 ending」，反而 re-accelerate 到 +9.7%——但代價是估值衝到 ~150x，priced for perfection 的風險反而最高。

PART I · DECISION SUMMARY

決策摘要層

三句話看完快休閒餐飲分化的現狀、走向與行動，後接 PM 級監測點。這一層是報告的全部重點——下面各節只是把這一層的每個判斷補上證據。

現在 / NOW

FC 段別暴力 K-shape：playbook 群內部再分化——CMG Q1'26 +0.5% 觸底回升（FY25 -1.7% 首次年負後）、WING Q1'26 dom -8.7% 持續惡化；format moat 全勝（SHAK +4.6% 三季連續 traffic+、BROS sys +8.3%/co-op +10.6%、rev +30.8%）；SG -12.8% 創上市最差、RLM 崩至 10%；CAVA +9.7% re-accel 但 ~150x P/E 把估值風險推更高。

未來 / NEXT

3–5 年 FC 段別 CAGR 降到 3–5%（vs 歷史 9–15%），不再是最高 CAGR segment（讓位 c-store foodservice）；alpha 來源永久從「playbook execution」移到「format moat」；Wonder 2027 IPO 可能重塑 multi-brand 格局；broken FC（SG/BurgerFi 已 Ch.11）持續退場或被併購。AB 1228 加速勞動成本分化。

行動 / ACTION

押 format moat winners（SHAK/BROS drive-thru/drinks-led 護城河；KRUS automation）；CMG 觸底但 ~32x 仍 priced for recovery、WING -8.7% 結構問題未解；CAVA +9.7% 強但 ~150x priced for perfection、估值風險最高；避開 broken FC（SG binary）。買賣由個股 DD + 系統決定。

The Key Call · 本報告最重要的一個判斷

市場看「FC = 成長 segment、龍頭都能 compound」，真正的非共識是alpha 來源換軌——「Niccol playbook」（digital + throughput + loyalty）2018–2024 是 alpha，2025 起被全段別複製、商品化（CMG digital 38.6% plateau、FY25 首次年負）；新 alpha 是別人抄不走的format moat（drive-thru 重置 SHAK、drinks-led 高毛利 BROS、day-1 automation KRUS）。stockpicker 該從「誰執行 playbook 最好」pivot 到「誰握真正的格式護城河」。最危險的兩個誤判：①把 CMG/WING 的 FY25 弱當「短期 lap」（WING Q1'26 -8.7% 證明部分是結構）；②把 CAVA 的 ~150x 當「成長理所當然」（+9.7% 很強，但估值已 price 永續高速）。

三道裂縫具體是：①playbook 商品化——digital/throughput/loyalty 三件套每家 FC 都有了，差異化跌到「誰能補貼折扣」，CMG/WING FY25 雙雙首次年負是 segment-level 訊號；②format moat 新 alpha——SHAK drive-thru 重置（build cost $2.6M→$2.2M target）、BROS drinks-led 85% liquid（小 box capex $1.0–1.2M）、KRUS conveyor automation，這些需要重新 box design 或 10 年品牌建構，短期抄不走；③broken formula 退場——urban-saturated + 沒 drive-thru + $14+ check sticker shock 的（SG/BurgerFi）結構性失血，SG 的 RLM 從 17.9% 崩到 10%、BurgerFi 已 Ch.11。盯這三條，比盯「FC 段別成長率」更能看清錢的流向。

PART II · HISTORICAL ANALOGS

零售 big-box 與書店去中介化，框住多空

歷史不必重演，但兩個對照各給一個教訓：1990s 零售 big-box 給出「新格式結構性吃 share」，書店去中介化給出「傳統大者會萎縮但未必消失」。

類比一：1990s 大型零售 big-box vs 中型 mall-based——新格式結構性吃 share。Walmart/Target 的 big-box 格式顛覆了 KMart/Sears 這類中型 mall-based 零售，後者衰退至破產（KMart 2002 Ch.11）。量化錨：1990–2005 間 Walmart 美國門市從約 1,500 家擴到超過 3,800 家、營收成長逾 5 倍，同期 KMart/Sears 份額崩塌。對本次 FC 含義：drive-thru-led（SHAK/BROS）+ automation（KRUS）是 FC 中的「big-box 新格式」——結構性吃 share。差異：餐飲消費頻次比百貨高約 20 倍、品牌切換成本低，崩潰更快但「大者恆大」未必成立，這也是為何 CMG 即使 FY25 首次年負、Q1'26 也能 +0.5% 觸底——它不會像 KMart 那樣歸零。

怎麼讀：FC 分化同時是「big-box 新格式吃 share」（format moat 拉開）與「書店式萎縮」（playbook 大者 multiple 壓但不死）。bull 的底是 format moat 三家，bear 是 broken FC（SG/BurgerFi 已是 KMart 結局）——SG 的 RLM 崩塌與 BurgerFi 的 Ch.11 已是早期證據。

類比二：1990s–2000s 美國書店 Borders/B&N → Amazon——傳統大者萎縮不死。傳統 superstore（Borders）被 specialty + tech-native（Amazon）顛覆，Borders 2011 Ch.11、Barnes & Noble 倖存但門市縮逾六成。量化錨：CMG/WING 是 FC 的「Borders/B&N」——規模最大、傳統格式、數位作為 layer 而非 native。Wonder/Crumbl/KRUS 是「specialty/tech-native」吃 share。差異與啟示：餐飲現場體驗無法線上替代，CMG 不會像 Borders 那樣消失（Q1'26 +0.5% 觸底為證），但會像 B&N 那樣萎縮 / multiple compress——WING 的 Q1'26 dom -8.7% 比 CMG 更深，更接近「結構萎縮」而非「短期 lap」。判別訊號：playbook 群同店能否回正 + multiple 是否被段別重新定價。

PART III · SUPPLY SIDE

供給側：FC 利潤池，被上游勞動與下游平台雙夾擊

反直覺但可操作的結論：FC 段別的利潤被「上游勞動/房東（AB 1228）」與「下游平台/aggregator（Toast/DoorDash/Wonder）」雙夾擊——只有 format moat 贏家守得住 restaurant-level margin，其餘被擠壓。

供給端可拆三層。上游（食材/設備/勞動/房東）：SYY/USFD 食材 + 勞動 + 房東，AB 1228 讓 CA 限服務餐飲價格 +13.1%、就業 -3.2%，議價權往上游勞動流。中游（FC 餐廳營運）：avg check $10–15、labor 22–28% / food 28–30%，restaurant-level margin 暴力分化（10%–29%）——format moat 贏家（BROS 29%、SHAK 22%+）vs broken（SG 崩到 10%）。下游（tech/platform）：Toast/Olo POS + DoorDash/Uber Eats delivery（take rate 20–30%）+ Wonder multi-brand，議價權往下游平台集中。利潤往「上游勞動 + 下游平台」兩端流，中游 FC 除 format moat 外被擠壓。

Inference · 供給
playbook 商品化 + 雙端夾擊，利潤往 format moat 集中
前提：digital/throughput/loyalty 已全段別複製（無 alpha）+ AB 1228 推升上游勞動成本 + 下游平台收 take rate + 只有 format moat 守得住 RLM → 利潤往 format moat 贏家集中，playbook 群與 broken FC 失血

當 playbook 三件套變成 table stake、每家都有，差異化就跌到「誰能補貼折扣」；同時 AB 1228 把 CA store 結構性 -100–200 bp RLM、下游平台抽 20–30% take rate。守得住的是握 format moat（drive-thru/drinks-led/automation）的——SHAK Q1'26 +4.6% 同店 + 三季連續 traffic+、BROS co-op +10.6% + RLM 29% 是「format moat 守得住」的硬證據；SG RLM 從 17.9% 崩到 10% 是「broken formula 守不住」的另一面。

可證偽條件：若 CMG/WING FY26 同店重新加速回 +3%+（證明 playbook 仍是 alpha 而非商品化），或 AB 1228 被回滾、下游平台 take rate 結構性下降，則「雙端夾擊 + playbook 商品化」論部分失效。

PART IV · DEMAND SIDE

需求側：playbook 商品化 vs format moat 新 alpha

需求最剛性的是「消費者要 value + convenience」——sticker shock（$14+ check）懲罰 broken FC、drive-thru/便利性獎勵 format moat；同店分化把這件事講得最白。

第一引擎：playbook 已商品化（無 alpha）。Niccol playbook 五大 lever 全 plateau——CMG digital 衝到 38.6% 已 saturate、loyalty 每家都有、throughput 不再是 bottleneck、Chipotlane 新建 AUV 比 base -10–15%、LTO comp lift 衰減。CMG FY25 -1.7%（首次年負 since 2017）+ Q1'26 僅 +0.5%、WING dom Q1'26 -8.7% 持續惡化——證明 digital alpha 已被段別複製、不再是差異化來源。第二引擎：format moat 是新 alpha（現需）。消費者用腳投票給格式：SHAK drive-thru 重置 Q1'26 +4.6%、三季連續 traffic+；BROS drinks-led 高毛利 + drive-thru-only 小 box，Q1'26 sys +8.3% / co-op +10.6% / rev +30.8%；CAVA Mediterranean bowl + GLP-1 protein alignment，Q1'26 SSS +9.7% re-accelerate、traffic +6.8%。需求把流量推向「有結構格式優勢」的 chain。

多空為何都對

多頭測FC 段別需求（format moat 三家高速同店 + 開店），這幾乎沒爭議；空頭測誰被商品化/結構失敗（playbook 群同店 plateau、broken FC RLM 崩）。兩者可同時為真：format moat 是真增長（多頭對），但 segment 整體 CAGR 降到 3–5%、playbook 群萎縮、broken 退場（空頭對）。本報告把重心放在「誰握 format moat、誰賣已商品化 playbook、誰結構破裂」。

PART V · VERDICT & SCENARIOS

裁決與三情境（24–36M）

「format moat 是新 alpha」判 INTACT，「playbook 已商品化」判 INTACT（WING Q1'26 -8.7% 加強、CMG +0.5% 觸底僅部分弱化），但「CAVA ~150x 估值能否撐」判 AT_RISK——故結構看「format moat 贏、playbook 萎縮、broken 退場」、conviction 卡 mid-high（CAVA 估值風險拖累）。如硬要對應傳統框架：對「能交出結構性 unit economics 的 FC 供給」需求遠大於合格供給（要握 format moat），等價於結構性短缺型，稀缺來自格式門檻而非店面數量。

Base Case主觀機率 ~50%

個股特徵與 conviction tier

本框架只輸出「特徵」與「分類」，買哪一檔由個股 DD 系統與 Pure MA 訊號決定。以下 tier 為產業推論的特徵歸類，提及公司僅為「該特徵的代表案例」，不附帶買賣建議。

CORE 🔴

SHAK / BROS（format moat 純度最高）

握別人抄不走的格式護城河——SHAK（drive-thru 重置、Q1'26 +4.6% / 三季連續 traffic+、build cost $2.6M→$2.2M target、1,500 unit LT）、BROS（drinks-led 85% liquid 高毛利、小 box capex $1.0–1.2M、Q1'26 sys +8.3%/co-op +10.6%、1,136→2,029 by 2029）。特徵是「format moat + 結構性 unit growth」最高純度贏家。conviction：高（format moat 守得住）
CORE 🔴

CAVA / KRUS（format moat 但估值/規模張力）

CAVA（Mediterranean + GLP-1 protein alignment、Q1'26 SSS +9.7% re-accel、RLM 25.1%（Q1'26）gold standard——但 ~150x P/E priced for perfection、估值風險最高）；KRUS（day-1 automation conveyor、AUV ~$3.9M、rev +23.3%、勞動改善——adjacent FC、偏貴 ~80x）。特徵是「unit-growth-with-margin」，但估值已 price 永續高速。conviction：中高（成長強）/ 估值風險高（CAVA ~150x）

CMG / WING / DPZ（playbook plateau）

playbook 商品化群——CMG（digital 38.6% plateau、FY25 -1.7% 首次年負後 Q1'26 +0.5% 觸底回升、~32x 仍 priced for recovery）、WING（Q1'26 dom -8.7% 持續惡化、Smart Kitchen turnaround 未證、~45–50x 應壓）、DPZ（FC-adjacent 防禦最強、dinner-sharing、~26x）。特徵是「萎縮但不死」（B&N 型）。conviction：中（CMG 觸底待證）/ 偏空（WING 結構未解）
LOSER / EDGE 🟢

SG / FWRG / 對照組（broken + 邊緣）

🟢 broken formula：SG（Q1'26 -12.8% 上市最差、traffic -11.2%、RLM 17.9%→10%、adj EBITDA 轉負、FY26 guide -4 to -2%、binary：Wonder M&A vs Ch.11）、FWRG（daypart-limited、+2.8% 但 traffic -2%）。對照組：MCD（$5 value down-trade hedge）、DRI（polished casual 從上方夾擊 upper-FC）、SBUX（Niccol 翻身樣板）、BFI/MOD（已 Ch.11/私募）。conviction：偏空（broken）/ 對照（其餘）

估值面總結：本主題的核心張力在「format moat 是真 alpha vs playbook 已商品化 vs broken 結構失敗」。format moat（SHAK/BROS）是最高純度贏家；CAVA/KRUS 成長強但估值已 price 永續高速（CAVA ~150x 是全段別估值風險最高）；playbook 群（CMG 觸底待證、WING 結構未解）是「B&N 型萎縮」；結構性輸家是 broken FC（SG binary、BurgerFi 已 Ch.11）。現處 Phase II（暴力分化），轉空條件 = CAVA 同店減速 + ~150x de-rate 或 消費降級從 broken 蔓延到 format moat；de-rating 風險主要在 CAVA（估值）與 WING（結構同店）。買賣由個股 DD + Pure MA 系統決定。

The One Line

FC 不是一起好或一起壞、是沿格式護城河暴力 K-shape——drive-thru/drinks-led/automation 贏家拉開，賣已商品化 playbook的萎縮，沒 drive-thru 的 urban formula 結構破裂。

這份報告刻意不做的三件事

  1. 不說「FC = 成長股齊漲」——明確畫出分化：format moat 贏、playbook 萎縮、broken 退場。
  2. 不把 CMG/WING 的 FY25 弱當「純短期 lap」——CMG Q1'26 +0.5% 觸底但 WING dom -8.7% 持續惡化，後者更像結構問題。
  3. 不把 CAVA 的 +9.7% 當「估值理所當然」——成長很強，但 ~150x 已 price 永續高速，估值風險反而最高；買賣由你的個股 DD 與 Pure MA 系統決定。
