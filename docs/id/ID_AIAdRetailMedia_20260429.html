💡 §0 5 條核心 insight（預告）
1. Meta vs Google 2026 首次逆轉是結構性 — META 估值倍數 step-up 25-40%：Reels + Threads + WhatsApp + AI 推薦三引擎 vs Google Search 受 ChatGPT + Apple Sherlock 雙夾殺；2027-2030 META 持續擴大 share。

§2 技術演進 + S 曲線 + 歷史類比

數位廣告從 2007 iPhone launch 進入 mobile era，2010-2015 social ads 爆發（Facebook / Instagram），2015-2020 programmatic + DSP 成熟（TTD），2020-2025 Retail Media 爆發（Amazon / Walmart）— 至今 18 年經 4 階段。當前 Phase II 中段是 AI / agentic commerce 整合期（2024-2030）：(1) Meta AI 推薦引擎優化 / (2) Amazon Rufus + sponsored products + AI / (3) Apple Search Ads + Sherlock / (4) WPP / OMC / IPG 結構性 collapse / (5) TTD UID2 + CTV 規模化。

歷史類比（QC-I14 + QC-I28 含正反例）

正類比 #1（廣告代理 HoldCo collapse ≈ 2003-2008 大型百貨 collapse）：1990s-2000s 大型百貨 Sears / JCPenney 被 Walmart / Amazon 顛覆 — 同樣現在 WPP / OMC / IPG 被 Big Tech + agentic AI 顛覆；都是 (a) scale 失去 leverage / (b) low margin / (c) 客戶 disintermediated。差異：HoldCo 還可能保留 niche service（client liaison），不會完全 dead 但 -50-70% scale。

正類比 #2（Meta 2026 超 Google ≈ 2010-2014 Facebook 超 Yahoo）：Yahoo 2010 仍是 #1 web property 但 5 年內被 Facebook 超越 in mobile / display；同樣 Google Search 受 ChatGPT + Apple Sherlock 雙夾殺，Meta Reels + Threads + WhatsApp 三引擎 + AI 推薦 = Yahoo → FB 重演。

反類比 #1（TTD ≠ 必然贏家的 Bloomberg 1990）：Bloomberg 1990 起步 + 2000-2020 完全主導 Wall Street 數據 — 但 TTD 面臨 walled garden（Meta / Google / Amazon）封閉 + UID2 標準化未完成 + Olympics programmatic 是 single event。本 ID 認為 TTD 是 sector 第三勢力但不是 Bloomberg 級 dominance。

反類比 #2（Retail Media ≠ 完全取代 search ad）：Retail Media 是 high-intent 但 funnel 中下段；Search 仍 awareness 階段；兩者互補非取代；2027-2030 RMN $200B 不會 disrupt Search $300B+。

💡 §2 Insight
1. Phase II 中段（2024-2030）是 sector 估值倍數 step-up 6 年最佳階段：Meta 超 Google + RMN $200B + TTD $7-10B + 廣告代理 collapse 同時發生；對應 META / AMZN / TTD / WMT 估值倍數 step-up 30-50%。
2. 廣告代理 HoldCo 是 「2003-2008 大百貨」重演：WPP / OMC / IPG / PUB collapse trajectory 已 confirm；2027 reset 後 sector 從 4 → 2-3 倖存。
3. TTD 是 sector 第三勢力但不是 Bloomberg dominance：CTV + UID2 + Open Internet + Olympics 多引擎；2027-2030 rev 2.4-3.5x；但 walled garden 封閉是天花板。
