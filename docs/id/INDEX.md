**欄位說明**：
- **涵蓋 ticker 數**：§11 關聯個股清單總數
- **深度分布**：🔴 核心（營收 >40% 依賴）/ 🟡 次要（10-40%）/ 🟢 邊緣（<10%）

- **🟡 比例**：judgment bullet 占全篇 bullet 的比例（QC-I1 上限 20%）
- **鮮度**：tech（§1-§3，半衰期 365 天）/ market（§4-§6、§10.5，90 天）/ judgment（§8-§13，60 天）
