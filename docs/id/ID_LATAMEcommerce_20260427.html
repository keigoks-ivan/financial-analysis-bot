拉美電商生態（LATAM E-commerce Ecosystem）

發布日：2026-04-27 ｜ 涵蓋 ticker：10 檔（4🔴 / 4🟡 / 2🟢）｜ 🟡 占比：18% ｜ 投資時鐘：Phase II（Mexico 滲透率超越美國 + Mercado Pago 飛輪 + Temu 攻擊已過峰） ｜ industry-analyst v1.7

Unit Glossary：GMV（Gross Merchandise Volume）= 平台總交易額；MAU / Active Buyers = 活躍買家；FX-neutral = 排除匯率影響的成長率；ARPAC（Avg Revenue Per Active Customer）= 每客戶平均收入；Pix = 巴西央行即時支付系統（2020 上線、2024 成全球最大即時支付網路）；NPL90+ = 90 天以上逾期率；de minimis = 美國 $800 以下進口免稅條款（2025-05 對中國取消）；cross-border = 跨境電商。所有財報 = Q4 2025 或 Q1 2026 最新可得。

Thesis（一句話非共識） LATAM e-commerce 不是「追趕北美 / 中國」的已開發市場 catch-up 故事（共識敘事）而是「leapfrog 跳躍式結構受益」：（1）Mexico 2026 滲透率 17.7% 反超美國 17% 是被市場低估的結構性訊號 — LATAM 跳過 BBC sales channel、直接進入 mobile-first + fintech-integrated e-comm，發展曲線比北美陡；（2）Mercado Pago 是 LATAM 獨佔的「fintech + e-comm 飛輪」—— 80M 月活 + $19B AUM +78% YoY + 87% AI 互動使 MELI 成為 LATAM 唯一具「e-comm × 支付 × 信貸」三引擎的 super-app；（3）Temu / 中國跨境攻擊已過峰，US tariff de minimis 關閉（2025-05）+ MELI / Shopee LATAM 在地物流護城河，使中國跨境威脅 GMV 增量 < 5%。市場仍以「美國電商比較 framework」看待 LATAM，但實際是結構截然不同的市場。
💡 §0 Insight：「Mexico 滲透率 17.7% 反超美國」是被低估的結構性訊號
事實基礎：2026 Mexico e-comm 滲透率預估 17.7% vs 美國 17%（eMarketer / PCMI）；MELI Mexico GMV +35% YoY；Nu Mexico 14M 客戶（15% penetration），相當於 Brazil 5 年前的位置。
推論邏輯：LATAM 不是 1990s 北美 e-comm 路徑的延遲版本，而是不同 stack 的 leapfrog—— 跳過信用卡普及階段直接 mobile + Pix 即時支付；跳過分散獨立電商階段直接 super-app；跳過低 ARPU 階段直接 fintech-integrated 高 ARPU。
投資意義：Brazil 仍是 e-comm 龍頭規模 (Brazil GMV +35% / Mexico +35%)，但 Mexico 增速從低基期更快爆發；NU Mexico 14M 客戶仍在早期擴張階段。押 LATAM e-comm 同時要押「Brazil 規模」+「Mexico 加速」雙重利好。

§1 產業定義與邊界

本 ID 涵蓋 LATAM 4 個結構性子市場：（a）純 e-commerce 平台 = MELI Mercado Libre（Brazil/Mexico/Argentina/Colombia 等 18 國）+ Sea Shopee（Brazil/Mexico）+ AMZN（Brazil/Mexico）+ BABA AliExpress + PDD Temu；（b）fintech 飛輪整合 = Mercado Pago（MELI 子業務）+ NU Holdings（純 fintech 但與 e-comm 競合）；（c）e-comm 賦能 SaaS = VTEX Brazil（賦能 1,000+ B2C 商家）；（d）傳統 retail 數位化 = Magazine Luiza Brazil / Falabella Chile / Liverpool Mexico。本 ID 主軸為（a）+（b）—— 平台 + fintech 飛輪是 LATAM 結構性 alpha 主源。

2026 年產業進入新階段的觸發點是「Mexico 滲透率反超美國 + Brazil Pix 完全成熟 + Temu 攻擊過峰三共振」：（a）2026 Mexico e-comm 占 retail 17.7% > 美國 17%（eMarketer 預估）；（b）Brazil Pix 即時支付已成全球最大網路，MELI / NU 客戶留存率因此結構性升；（c）2025-05 美國取消對中國 de minimis loophole，Temu 美國 GMV 大跌但已 redirect 到 LATAM/SEA — 但 MELI / Shopee 在地物流（MELI 6 大配送中心 / Shopee Brazil 日配送）已建護城河，Temu 攻擊增量 GMV < 5%。

💡 §1 Insight：Brazil + Mexico 雙引擎佔 LATAM 70%+ 規模
事實基礎：Brazil 占 LATAM e-comm 29% + Mexico 26% = 55%（2024 base）；Brazil 加 Mexico 跨境 e-comm 占 70%+。
推論邏輯：LATAM e-comm 投資不需分散到所有國家；押 MELI（all-in-one）或 NU（Brazil 龍頭 + Mexico 加速）已能 capture 70%+ 結構受益。
投資意義：避開 single-country plays（Argentina 通膨 / Colombia 政治不穩 / Chile 規模太小）；押有 Brazil + Mexico 雙重曝險的玩家。

§2 技術演進 + 歷史類比

LATAM e-commerce 演進四個世代。第一代（2000-2010）是「美國模型 LATAM 化嘗試」 — eBay LATAM 進入後撤退，MELI 1999 IPO 接手 niche；第二代（2010-2020）是「行動支付 + Marketplace 飛輪」 — MELI 推 Mercado Pago，逐步建立 LATAM 唯一全國覆蓋的 fintech；第三代（2020-2024）是「Pix 革命 + COVID 加速」 — Brazil 央行 2020 推 Pix 即時支付改變整個 LATAM 支付 stack，COVID 進一步推升 e-comm 滲透；第四代（2025+）是「leapfrog 確認期 + Mexico 加速 + 中國跨境攻擊過峰」 — Mexico 滲透率反超美國、NU Mexico 客戶 14M、Temu 攻擊已被在地物流護城河擋住。

第四代的核心特徵是「fintech-integrated e-comm 模式 vs 純 e-comm 模式」勝負分曉。MELI 透過 Mercado Pago 整合（80M 月活 + 12.5B credit portfolio + AI 處理 87%）形成「e-comm × 支付 × 信貸」三引擎，每年 ARPU 結構性增；純 e-comm 玩家（AMZN LATAM / Shopee LATAM）必須持續燒錢補足在地支付方案。NU 雖純 fintech 但結構性受益 LATAM 銀行業 disruption（NPL90+ 1.1% 穩定 + ARPAC $15 持續升）。

歷史類比：兩個有用、一個誤導

有用類比 1：1995-2010 中國 e-commerce + 行動支付雙起步。1995-2010 中國從信用卡未普及階段直接跳到行動支付（支付寶 / 微信支付）+ Marketplace（淘寶 / 京東），跳過北美「PC e-comm 時代」。LATAM 2020-2030 路徑類似 — Pix + 移動端 + Mercado Pago 整合 = 跳過 PC e-comm 時代。結論：LATAM e-comm CAGR 將比北美高 1.5-2x（北美 5-7% vs LATAM 10-13%）。

有用類比 2：2014-2024 SEA 東南亞 leapfrog（Sea / Grab / GoTo）。東南亞用 10 年從 e-comm 滲透 1% 拉到 8-12%，主要 driver 是行動裝置 + super-app（Grab / Sea Limited / GoTo）。LATAM 已在 2020-2025 走完 SEA 2014-2020 路徑（滲透 5% → 12%），下一步預期到 2030 達 25%+，類比 SEA 已 priced 完成的階段。

誤導類比：2010 北美 e-commerce vs 傳統 retail 整合。不少評論用 2010-2020 美國 AMZN vs Walmart / Target 整合的劇本看 LATAM，認為 MELI 終將被 AMZN / Walmart 收購或被打敗。反向類比警示：（a）LATAM 跨國物流 / 支付複雜度遠高於美國，AMZN 至今 Brazil/Mexico 整合不力（5%+ market share 但 GMV 落後 MELI 5x）；（b）MELI 在 LATAM 18 國均已建本地物流 + 支付 + 信貸生態，AMZN 全球玩家無法輕易複製；（c）LATAM 跨境 e-comm 仍 fragmented，每國仍需獨立 stack。誤用會錯估 MELI 護城河強度。

💡 §2 Insight：Mercado Pago 飛輪是 MELI 25 年護城河的核心
事實基礎：Mercado Pago Q4 2025 月活 80M+ / AUM $19B +78% / 信貸 portfolio $12.5B / AI 處理 87% 互動；NU 雖純 fintech 但無 e-comm 整合。
推論邏輯：MELI 的「e-comm GMV → 支付 → 信貸」飛輪在 LATAM 是獨佔結構—— AMZN 在美國有 same-day 物流但無自有支付飛輪；BABA 中國有支付飛輪但在 LATAM 無 e-comm 規模。MELI 是全球少數同時擁有 e-comm 規模 + 整合支付 + AI 信貸的玩家。
投資意義：MELI 的 fwd PE 估值 prem 是合理結構（vs AMZN 30x）—— 飛輪每年 ARPU 結構性升、competitor 5+ 年難複製。

§5 商業鏈 / Value Chain

LATAM e-comm 商業鏈特徵：fintech-integrated 模式佔結構優勢。MELI 把 GMV → 支付 → 信貸三層整合，每層都收費 take rate 1-3%；純 e-comm 玩家僅有第一層 take rate（5-10%），但無支付 / 信貸第二三層飛輪。NU 是純 fintech（無 GMV 層），靠銀行業 disruption 收費（信用卡 interchange + lending margin）。

L2 Payment（Mercado Pago / NU）MELI MP / NU / Visa / Mastercard1.5-2% transaction fee結構性受益（Pix 普及 + AI 自動化）

價值鏈解讀：（1）三引擎玩家（MELI）獨佔 L1+L2+L3，毛利結構最強；NU 主導 L2+L3 但無 L1；（2）L4 物流是 Temu 攻擊的真正護城河—— MELI 6 大配送中心 + Brazil 24h 配送是中國跨境玩家難複製；（3）L5 廣告層是 LATAM 未來 3-5 年新增 alpha—— AMZN US ad +19% YoY 是案例，MELI / Shopee 在 LATAM ads 業務剛起步。
