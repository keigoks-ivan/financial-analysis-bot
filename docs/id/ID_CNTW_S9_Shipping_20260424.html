四 chokepoint 同時壓力是歷史首次

三句話結論：2026 是「四 chokepoint 同時承壓」的歷史性時刻——Hormuz 被 US 海軍封鎖（Iran 戰爭中）、紅海 Bab-el-Mandeb Houthi 停攻但不穩、Suez 運河受紅海影響、巴拿馬運河受乾旱 + Trump 主權爭議（2025-01 Trump 公開討論收回），台灣海峽則是未來 6 年最大風險點。這意味全球航運市場進入「結構性高保費 + 繞道」常態。受益者：MSC（未上市）/ Maersk（MAERSK.B）/ CMA CGM（未上市）/ COSCO（雙刃）/ Evergreen（2603）+ 再保龍頭 Munich Re / Swiss Re / Hannover Re + 戰爭險專家 Lloyd's / AXA XL / AIG Specialty + 能源運輸 Frontline / Euronav / International Seaways + 美資 LNG 航運 Cheniere / NextDecade。台海危機情境下，台灣海峽戰爭險保費可暴漲 20-50 倍（0.05% → 2-5%），全球貨櫃運輸成本立即上升 40-80%。

§1 核心判斷

★ 非共識 Thesis 1：四 chokepoint 同時承壓是「結構性 paradigm shift」 歷史上單一 chokepoint 危機（如 1973 石油危機、1984 兩伊、2015 Bab-el-Mandeb）皆為短期擾動，市場 6-12 個月回歸。但 2024-2026 是四個 chokepoint 同時失控——Hormuz（Iran 戰爭 + 美封鎖）、紅海（Houthi 2023-2025 攻擊 + 停攻但不穩）、Suez（紅海連動 + Egypt 政治）、巴拿馬（2024 乾旱限船 + Trump 收回論）。當前台海雖穩定但已是「第五 chokepoint」的潛在威脅。這意味全球海事保險市場進入「多頭地緣政治風險」定價時代——Lloyd's 等承保人把戰爭險保費基本率從 0.025% 上升到 0.05-0.1%，更高風險區域（紅海、Hormuz）從 0.5% 上升到 1-3%，且不設回歸機制。這是結構性利多再保 + 專業險，不利貨櫃船公司毛利。
★ 非共識 Thesis 2：台海風險被航運保險業嚴重低估 當前（2026-04）Lloyd's JWC（Joint War Committee）高風險區域列表包括 Hormuz、紅海、Bab-el-Mandeb、Suez、黑海，但不包括台灣海峽。這意味航運業尚未對台海危機定價——船東繼續走台灣海峽航線、保費僅 0.05-0.1%、沒有替代航線準備。這是重大的mispricing 機會 + 結構性危機風險：如果 PLA 宣布「演習」封鎖台海 3-7 天，Lloyd's JWC 會緊急加入台海高風險，保費立刻跳至 1-3%，貨櫃船被迫繞道菲律賓 / 琉球（航程 +3-5 天，成本 +15-25%）。2024-12 「Justice Mission 2025」PLA 環台演習已是預警，但市場尚未結構性反應。建議 Stress case 配置中加入：再保股 + 戰爭險 ETF（若存在）+ 能源海運（替代路線受益）。
★ 非共識 Thesis 3：LNG 航運是被低估的結構牛市 台海衝突情境下，東亞（日韓台）能源進口全面轉向美洲 / 中東 LNG，繞過 Hormuz / 馬六甲高風險航線。美國 LNG 出口 2023 年超越 Qatar 成為全球第一，2026 年達 14 BCF/d（85M 噸/年），2030 目標 25 BCF/d（150M 噸/年）——項目包括 Cheniere Corpus Christi Stage 3、Venture Global Plaquemines、NextDecade Rio Grande、Sempra Port Arthur、Freeport LNG 擴產。相應 LNG 船運需求爆發：2024 全球 LNG 船 720+ 艘，2030 需 900+ 艘。LNG 船市場龍頭：GasLog（私有）、Flex LNG（FLNG）、Dynagas LNG、BW LPG（航運相關）、Golar LNG（GLNG）。除了船東，LNG 基礎設施（Cheniere LNG / CQP、EQT / Range Resources 天然氣生產、Excelerate Energy FLNG）也是受益者。Trump 政府 2025-01 解除 LNG 出口禁令，2026-2030 是 LNG 航運超級週期。
💡 3 條最高價值 insight
1. 戰爭險再保是投資而非商品：Munich Re / Swiss Re / Hannover Re 等 2026 配置結構已優化為「高保費低損失」區間，結構性 EPS CAGR 8-12%。若台海衝突發生，短期單點損失（個別船）會擠壓資本，但長期保費上漲覆蓋多倍於損失。這是長期複利機會
2. M1 船級社是被忽略的關鍵角色：ABS（American Bureau of Shipping）、DNV（挪威）、Lloyd's Register、Bureau Veritas、ClassNK（日本）五大船級社負責船舶分級。戰爭險理賠條件 + 新安全標準會讓船級社檢驗需求大增；DNV（Euronext OSL）、BV（BVI.PA）可投資
3. 油輪期貨對沖台海風險：Baltic Dirty Tanker Index (BDTI) + Baltic Clean Tanker Index (BCTI) 反映即期運費。若台海衝突，油輪繞道 Cape of Good Hope 航程 +15-20 天，BDTI 可能 +150-200%。Frontline (FRO)、Euronav (EURN)、International Seaways (INSW)、Scorpio Tankers (STNG) 是 pure-play

§2 結構背景：四大 chokepoint 現況

§2.1 Hormuz 海峽（伊朗 / 阿曼之間）

  • 流量：全球 20% 原油（16-18M bpd）、20% LNG 貿易（主要 Qatar）
  • 現況（2026-04）：US Navy 第五艦隊封鎖進行中，禁止 Iran 出口原油，Iran 扣押 3 艘商船並威脅「全面封鎖」。停火談判進行中
  • 風險：若 Iran 全面封鎖，油價可能到 $120-150/bbl，LNG 價格 +80%
  • 替代路線：經阿曼 Duqm 港 / 沙烏地 East-West 輸油管（5M bpd 容量）
  • 保險費：戰爭險 2024 0.5% → 2026 1-2%；某些船停止承保

§2.2 紅海 / Bab-el-Mandeb 海峽（葉門 / 吉布地）

  • 流量：全球 12-15% 原油、10% 貨櫃、Suez 運河的亞歐主航線
  • 現況：Houthi 2023-11 起攻擊西方船隻 + 以色列關聯船隻，2024 年共 80+ 次攻擊。2025-05 停火但仍不穩
  • 紅海分流：歐亞貿易 70% 已繞 Cape of Good Hope（航程 +10 天，成本 +20-30%）
  • 風險：Iran 戰爭期間 Houthi 可能復攻；Iran 談判失敗則攻擊頻率上升
  • 保險費：紅海戰爭險 2024 峰值 1-3%，2026-04 回落到 0.3-0.8%

§2.3 Suez 運河（埃及）

  • 流量：全球 12-13% 貨運（紅海 + 地中海 pass-through）
  • 現況：紅海風險讓 Suez 2024-2025 運量腰斬，Egypt 運河收入 2024 -50%。2026 隨紅海穩定回升但仍未回到 2023 水平
  • 風險：Egypt 內政 + 紅海連動
  • 替代：Cape of Good Hope（永久替代，但航程長）

§2.4 巴拿馬運河

  • 流量：全球 6% 貨運（美國東岸 / 西岸、拉美 / 亞洲）
  • 現況：2023-2024 乾旱使 Gatun 湖水位低，每日通行船隻從 36-38 艘減到 24-28 艘。2025-2026 水位部分恢復但仍脆弱
  • Trump 主權論：2025-01 Trump 公開表達「巴拿馬運河應回美國」，2026 尚未實際行動但增加政治張力
  • 風險：氣候（El Niño 年）+ 政治（Trump 施壓 / 巴拿馬大選）

§2.5 台灣海峽（第五 chokepoint）

  • 流量：全球 30% 貨櫃容量、50%+ 全球商船通過（統計方式不同）
  • 現況：尚未被列入 Lloyd's JWC 高風險區，但 PLA 2024-2025 「聯合利劍」演習已展示封鎖能力
  • 風險：母議題所有情境均涉及；Stress case 20-25% 機率
  • 替代路線：菲律賓 Luzon Strait + 巴林塘海峽（航程 +3-5 天）

§3 核心分析

§3.1 台海三情境下的運價 / 保費模型

情境 B2 有限封鎖（1-4 週）

  • Lloyd's JWC 將台海加入高風險清單，戰爭險從 0.05% 漲至 0.5-1.5%（10-30x）
  • 貨櫃船繞道（菲律賓海峽或琉球海峽），航程 +3-5 天，運費上升 20-35%
  • Shanghai Containerized Freight Index (SCFI) 1-2 週內從 1,500 漲到 3,000+
  • Baltic Exchange Dry Index (BDI) 小漲 10-20%
  • 油輪運費上漲 15-25%
  • 台灣集裝箱港（基隆 / 高雄 / 台中）通關延遲 + 部分船放棄靠泊

情境 B1 全面封鎖（1-3 個月）

  • 戰爭險升至 2-5%，部分船級社暫停認證
  • MSC / Maersk / CMA CGM 等大型貨櫃船完全改道（Cape of Good Hope）
  • SCFI 在 1 個月內達 5,000-8,000（vs 正常 1,500-2,000）
  • 全球供應鏈衝擊：半導體 / 機械 / 電子產品運輸延遲 3-6 週
  • 能源運輸（LNG / 原油）尋找替代航線
  • 保險理賠爆發 + 再保公司承受首次大額理賠

情境 Tail 全面戰爭

  • 台灣海峽成為戰區，保險完全退出
  • 全球貨運 SCFI 可達 10,000+（2021 Omicron 高點 2 倍）
  • 台灣本地港口無法運作，台灣進出口停擺
  • 全球 GDP 短期 -5% 級衝擊
  • 戰爭險保險公司大額理賠（部分可能破產）
  • 航運股票短期大漲（非台灣船公司受益），長期面對資本市場恐慌

§4 外部變數整合：Iran + Venezuela

§4.1 Iran 戰爭對航運市場的直接衝擊

  • Hormuz 封鎖：2026-02 起 US 海軍封鎖持續，Iran 原油出口 -1.38M bpd。全球原油價格上升到 $90-95/bbl 區間；Brent-Dubai 價差擴大
  • 油輪運費：VLCC 運費 2025-11 $40K/day → 2026-04 $65-80K/day，漲 60-100%
  • LNG 船運費：$120K/day → $180K/day，受 Qatar 部分轉向亞洲直運
  • 紅海緊張升級：Iran 戰爭期間 Houthi 雖停攻但威脅復攻；紅海戰爭險保費回升至 0.5-1%
  • 中國能源進口航線變化：從 Hormuz 繞道 Cape of Good Hope + 從俄羅斯 ESPO 管道 + 從拉美（巴西 / Guyana / Venezuela 被美國控制後減少）

§4.2 Venezuela 政權轉移對航運的影響

  • 原油運輸路線變化：Venezuela 原產 750K bpd（危機前），Maduro 倒台後美國重新規劃。Venezuela 原油主要輸往 Russia（繞道到中國）、印度、中國。新政府可能轉向美國 / 歐洲；中國需重新佈局拉美能源供應
  • 加勒比海運輸保險上升：Colombia 反美政局 + Venezuela 過渡期，加勒比海航運保費上升 20-30%
  • 巴拿馬運河影響：Venezuela 油輪原走巴拿馬運河到東亞，新路線可能降低巴拿馬 traffic 3-5%
  • 美國 LNG 出口佈局：Venezuela 穩定 + 美國產能擴張 → 美國 LNG 出口到歐洲 / 亞洲動力更強

§8 風險與結論

§8.1 三個最大風險

  1. Data 風險：戰爭險定價不透明——Lloyd's JWC 運作機制不完全公開，保費調整時機可能比本篇預估快或慢
  2. 前提風險：再保業對台海 exposure 不均——不同再保公司承擔比例不一，Munich Re / Swiss Re 可能比 Hannover Re 多擔
  3. Tail risk：Berkshire Hathaway 因 BNSF 鐵路 + General Re 雙重曝險——美國供應鏈衝擊對 Berkshire 有複合影響
