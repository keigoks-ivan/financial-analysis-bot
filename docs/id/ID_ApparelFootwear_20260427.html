發布日：2026-04-27（首篇）｜ 涵蓋 ticker：15 檔（6🔴 / 8🟡 / 1🟢）｜ 🟡 占比：18% ｜ 投資時鐘：Phase III → IV K 形分化 + Tariff Permanent Reset ｜ industry-analyst v1.8

Unit Glossary：SSS / comp = 同店銷售（連續經營 13 個月以上的零售場域）；DTC = Direct-to-Consumer（自有店面 + 自家電商）；Wholesale = 售予 Dick's / Foot Locker / Macy's / Kohl's 等通路再轉賣；AUR = Average Unit Retail（單品實收價）；Athleisure = 運動 + 休閒兼用穿搭；K-shape = 高低端分化（中段被擠壓）；c/c = constant currency 排除 FX；FY = fiscal year（NKE/LULU/RL = May/Jan 結算 → FY25 = Cal 2025；ANF/AEO/URBN/PVH/VFC/GAP = Jan/Feb 結算 → FY25 結束於 2026 初）；tariff = 關稅，本 ID 內所有 tariff 提及指 2025-04-02 Trump Liberation Day reciprocal tariff 體系。

💡 §0 4 條核心 insight（預告）
1. 同產業同時間 K 形分化已具雙位數百分點差距：Q4 FY25/Q1 FY26 同一報季 ANF Hollister +23% vs 自家 Abercrombie -10% 差 33pp、URBN UO +9.6% / Anthro +8.3% / FP +8.0% 三品牌全正成長 vs LULU 北美 -1% — K 形已下沉到單一公司內部。
2. NKE Q3 FY26 wholesale +5% > DTC -4% 是 Donahoe 時代結束的明確訊號：Donahoe 把 DTC 推到 44%，Hill 上任 18 月後 wholesale 第一次反超 DTC，意味通路權回流給 Dick's-FL / 亞馬遜。
3. RL Asia +22% / China +30% c/c 是 Affordable Luxury 在 China 的 alpha 證據：當 NKE Greater China -10% 連 6 季時，RL 在同一市場 +30% — 這不是「China 復甦」是「品類分化」（K-shape 也存在於 China）。
4. VFC 連衰 + Vans Q3 FY26 -10% 仍比 Q4 FY25 -22% 改善：「衰退率減速」是 turnaround 早期訊號，但 NA TNF +3% / Timberland +9% 已是 Darrell 真正的 alpha 來源（不是 Vans）。

💡 §2 Insight
1. HOKA + ON 的 Lululemon 化是 base case 不是 risk case：所有 athleisure 新進者最終都會碰 derate point，問題不是會不會，是何時。本 ID 估 ON 比 HOKA 晚 12-18 月（mix shift + lifestyle 較成功）。
2. ANF 2025-2028 cycle = 2003-2010 重演但財務體質強 2x：股價可能不會 -75%，但本季 -24% 已是 cycle 啟動訊號；Hollister 仍會像 2007-2010 一樣成獨立 alpha 引擎。
3. NKE 不會重演 Adidas 2018 反彈路徑：HOKA + ON 4 年內從零到 $4B 已永久切走 NKE 7-9% 美國 share；Anta + Li-Ning 永久切走 China 4-6pp share。Hill turnaround 上限是 NKE FY27 revenue 回到 FY25 的 95-100%（不是 reach FY22 高點）。

§12 Non-Consensus View（共識差異）

分歧 1 — K-shape 是結構性而非循環性 [structural]

共識（Wells Fargo / GS / Citi 2026 Q1）本 ID
市場用「服飾循環」框架：認為 2024-2025 LULU / NKE 弱勢 + ANF Abercrombie -10% 是「消費者支出回升 + China 復甦就會反彈」的循環性現象（GS Neutral RL "已在頂"）K-shape 是結構性：高端（RL Asia +22% / China +30%）+ 新進者（HOKA / ON / 全球 +30% YoY）+ specialty inflection（URBN UO +9.6% / Anthro +8.3% / FP +8.0% 三品牌全 +ve）三引擎同時存在 vs 中段（ANF Abercrombie -10% / Vans -10% / LULU 北美 -1%）集體失速 — 同一報季出現 30pp+ 內部差距已是歷史性
分歧原因共識把 LULU 北美 -1% 視為 cyclical（會回 +5%），本 ID 視為 structural derate（mature 滲透率 30%+ + Vuori-Alo 持續 dupe = 永久不會回 +14%）；共識把 RL +12% 當「短期反彈」，本 ID 視為「Affordable Luxury elevation 結構性 alpha」
事實基礎§4 athleisure CAGR mature derate；§7 RL ASP 5Y +25% vs LULU Align $98 hold flat；§6 子象限 1 內部 K-shape

分歧 2 — Tariff 是 GM permanent reset 而非 transitory [event-triggered]

共識（Bof A / JPM 2026 Q1）本 ID
tariff 是 cyclical / 1-2 年 hit；Vietnam deal 從 46% → 20% 暗示 baseline normalization；FY27 GM 將回到 FY24 高點Tariff GM permanent reset：FY26 PVH -215bps / DECK $150M / NKE -130bps 是長期 baseline；FY27 公司 GM 不會回到 FY24 高點，而是 lock 在 -50~-150bps 永久水平
