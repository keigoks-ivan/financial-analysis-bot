§1 產業白話定義 + 歷史脈絡

白話版：快休閒餐飲（FC）過去十年是「餐飲界的成長股」——介於速食與正式餐廳之間、櫃檯點餐、平均一餐 $10–15。它過去靠「digital 點餐 + 提升出餐速度 + 會員」三件套贏，但這套現在每家都會了，新的贏家靠的是「格式」本身——得來速、高毛利飲料、自動化——這些別人短期內抄不走。

白話開場：先說三個關鍵詞。快休閒（fast casual）是「比速食高級、比正式餐廳便宜」的中間段——你在櫃檯點餐、沒有服務生、一餐約 $10–15、坐 30–50 個位子、用餐 15–30 分鐘，Chipotle、Shake Shack、CAVA 是典型。「Niccol playbook」是 Chipotle 前執行長 Brian Niccol 在 2018–2024 示範的成長配方：把數位點餐做到三成以上、用第二條製作線提升出餐速度、加上會員 loyalty。format moat（格式護城河）是那些「不是靠執行、而是靠格式選擇」帶來的結構優勢——Dutch Bros 的「只賣飲料、只做得來速」、Kura Sushi 的「從第一天就用迴轉自動化」。本報告問的是：當 playbook 被每家複製、商品化之後，FC 段別的成長與利潤，被重分配給了誰。

邊界界定：本報告聚焦「FC 段別內部如何暴力分化 + alpha 來源換軌」，是美國餐飲連鎖母題的 fast casual 專屬視角。納入 playbook 群（CMG/WING/DPZ）、format moat 群（SHAK/BROS/CAVA/KRUS）、broken formula 群（SG/FWRG/BurgerFi/MOD）、以及 QSR/casual 對照組（MCD/DRI/TXRH/SBUX）。母題的全 segment 廣度（QSR、casual dining、咖啡）見 USRestaurantChains；casual dining 翻身樣板見 CasualDiningTurnaround；POS/delivery SaaS（Toast/Olo/DoorDash）見 RestaurantTechSaaS。灰色地帶有五個：WING 結構是 FC 但被列「playbook 群」（因 Smart Kitchen + 73% digital 是 Niccol-style）；DPZ 傳統算 QSR pizza 但 digital + $25 check 接近 FC dynamics，列「FC 防禦」對照；SBUX 是咖啡 chain，但「Niccol 翻身教科書」值得 FC 借鏡；DRI 的 polished casual（North Italia）是 FC 上方夾擊；Wonder 是 multi-brand ghost-kitchen，不是傳統 FC 但直接競爭 FC traffic。

歷史分三段。第一段：2010–2018 黃金擴張——CMG（IPO 2006）、Panera、SHAK（IPO 2015）、WING（IPO 2015）、CAVA 早期、SG 紐約 cult，段別 CAGR +10–15%，FC 是「最高 CAGR segment」的 narrative 主導。第二段：2018–2024 Niccol playbook 主導——Niccol 接 Chipotle（2018）推 Chipotlanes + 第二製作線 + Rewards、WING 推 Smart Kitchen、CAVA IPO（2023）、SG 上市（2021），「digital + throughput + loyalty」成為公認的 alpha 配方。第三段：2025–2026 playbook 退潮 + 暴力分化——CMG FY25 -1.7%（首次年負 since 2017）、WING FY25 -3.3%（20+ 年首負）；SHAK/BROS/KRUS 的 format moat 驗證；SG/BurgerFi 結構失敗（後者已 Ch.11）。2026 Q1 把分化講得最白：CMG +0.5% 觸底、WING dom -8.7% 持續惡化、SHAK +4.6%、BROS sys +8.3%、CAVA +9.7%、SG -12.8%。

歷史類比一：1990s 零售 big-box——新格式結構性吃 share。Walmart/Target 的 big-box 顛覆 KMart/Sears 中型 mall-based 零售，後者破產（KMart 2002 Ch.11）。量化錨：1990–2005 間 Walmart 美國門市從約 1,500 家擴到逾 3,800 家、營收成長逾 5 倍，同期 KMart/Sears 份額崩塌。對本次 FC 含義：drive-thru-led（SHAK/BROS）+ automation（KRUS）是 FC 中的「big-box 新格式」，結構性吃 share。差異：餐飲消費頻次比百貨高約 20 倍、品牌切換成本低，崩潰更快但「大者恆大」未必成立——這也是為何 CMG 即使 FY25 首次年負、Q1'26 也能 +0.5% 觸底，它不會像 KMart 歸零。歷史類比二：1990s–2000s 書店去中介化——傳統大者萎縮不死。傳統 superstore（Borders）被 specialty + tech-native（Amazon）顛覆，Borders 2011 Ch.11、Barnes & Noble 倖存但門市縮逾六成。量化錨：CMG/WING 是 FC 的「Borders/B&N」——規模最大、傳統格式、數位作為 layer 而非 native；Wonder/Crumbl/KRUS 是「specialty/tech-native」吃 share。啟示：餐飲現場體驗無法線上替代，CMG 不會像 Borders 消失（Q1'26 +0.5% 觸底為證），但會像 B&N 萎縮 / multiple compress；WING 的 Q1'26 dom -8.7% 比 CMG 更深，更接近「結構萎縮」而非「短期 lap」。

歷史 TAM 預測的系統性高估：要校準「FC 仍是高成長 segment」這個共識，最好的方法是看過去的 TAM 預測準不準。Technomic 2018 年的 forecast 預估 2025 年 FC 規模約 $120B（隱含 CAGR ~10%），實際只有約 $95B，偏差 -21%；NRA 2021 年的 post-COVID V-recovery 預估約 $110B，實際 ~$95B，偏差 -14%；Technomic 2023 年的預估約 $100B，偏差 -5%。三次預測系統性高估 5–21%，且偏差在「FC 黃金期 narrative 最盛」時最大。這個歷史 pattern 的教訓是：投資人應對 sell-side「FC 仍是 highest CAGR segment」的論述打 5–15% 折扣——FC 的真實成長一直比 narrative 慢，而 2024 的 +0.6%、2025 的 +6%（unit-driven）只是把這個長期高估一次性 reset 到現實。把這個量化錨放進估值，CAVA ~150x、KRUS ~80x 這些「price 黃金期延續」的 multiple 就顯得更脆弱。

本主題無單一價格 cycle，用「段別同店分化 + format moat 採用」當時序錨。

關於 cycle 統計表的省略：FC 隨總經消費有週期，但本主題是結構性分化（playbook 商品化 + format moat 換軌），非單一 ASP cycle，故不立 cycle 統計表；以「FC 五階段編年」（2010–18 黃金擴張 → 2018–24 playbook 主導 → 2025–26 暴力分化 → 2027–29 Wonder/平台 → 2030+ 新均衡）呈現時序。

這段歷史的可操作含義：FC 分化同時是「big-box 新格式吃 share」（format moat 拉開）與「書店式萎縮」（playbook 大者 multiple 壓但不死）。bull 的底是 format moat 三家（SHAK/BROS/CAVA），bear 是 broken FC（SG/BurgerFi 已是 KMart 結局）。判別一家公司站哪邊，看它賣的是「已商品化的 playbook」還是「抄不走的 format moat」。

§2 技術成熟度 + S 曲線

為什麼是現在分化？因為 Niccol playbook 的每個 lever 都到頂了（CMG digital 38.6% plateau、loyalty 每家都有、throughput 不再是瓶頸），同時 format moat（drive-thru/automation）跨過「能規模化複製」的門檻——舊配方失去 alpha、新配方開始驗證。

「技術拐點」在 FC 不是晶片而是營運格式。Niccol playbook 五大 lever 全 plateau：digital % of sales——CMG 衝到 38.6%（Q1'26）已 saturate、WING 73% plateau，再增加沒空間；throughput——第二製作線 / Smart Kitchen 已把出餐速度做到極限，瓶頸轉成尖峰時段需求本身；loyalty——CMG 40M+ 會員、WING 60M+ 資料庫，每家都有，差異化跌到「誰能補貼折扣」；Chipotlane 開店——新建仍佔約八成，但新店 AUV 比 base -10–15%（好點位用盡）；menu LTO——每個限時新品的 comp lift 在衰減。五個 lever 同時到頂，就是「playbook 商品化」的技術根據。對照之下，format moat 跨過「能規模化複製自身」的門檻：SHAK 的 drive-thru prototype 把 build cost 從 $2.6M 壓向 $2.2M target、BROS 的小 box 模型可複製到 2,029 家、KRUS 的迴轉自動化從 day-1 設計——這些是新 S 曲線的起點。

怎麼讀：三種格式決定誰贏——drive-thru 便利性（SHAK/BROS）、高毛利格式（BROS 液體 / CAVA bowl）、day-1 automation（KRUS）。只賣「已商品化 playbook」（digital+loyalty+throughput）的 chain（CMG/WING）失去 alpha；想 retrofit automation 的（SG Infinite Kitchen）證明既有 chain 無法回頭改造——這是 playbook 與 format moat 的分水嶺。

為什麼 SG 的 Infinite Kitchen 是「playbook 思維想解 format 問題」的失敗範本：Sweetgreen 的故事值得單獨拆解，因為它把「playbook 時代的執行思維」用在「format moat 時代的格式問題」上，結果兩頭落空。SG 在 2021 年以約 $70M 收購 Spyce 機器人沙拉組裝技術，想用「automation 作為 layer」加在既有的 urban-Manhattan 高租金店面上——但 automation 的真正價值是「從 day-1 設計進格式」（如 KRUS），事後 retrofit 進一個本身就 broken 的格式（沒有得來速、$14+ check、紐約高租金）並不能救它。四年後 SG 把 Spyce 以 $186M 賣給 Wonder，管理層公開講「operational transformation」——這其實是放棄「tech-restaurant」定位的告白。對照 KRUS 從第一天就把迴轉自動化當成格式骨架、Q1'26 仍有勞動成本改善，SG 的失敗證明了一個關鍵命題：automation 是 format moat 的一部分、不是可以外掛的 playbook lever。這也是為什麼本報告把 SG 列為 broken（格式問題）、而非單純的執行不力——它的 RLM 從 17.9% 崩到 10%、traffic Q1'26 -11.2%，是格式破裂而非短期逆風。

技術拐點對 entry timing 的含義：format moat 採用正從早期往主導走、playbook 群同店 plateau——這是「選邊站」階段。差異化往「drive-thru/高毛利格式/automation」上移，握這些者享最持久優勢；賣已商品化 playbook 的 chain（CMG 觸底待證、WING 偏空）面臨 multiple compress。

§3 供給側：現在與未來 + 利潤池

FC 段別的利潤被「上游勞動/房東（AB 1228）」與「下游平台/aggregator（Toast/DoorDash/Wonder）」雙夾擊——只有 format moat 贏家守得住 restaurant-level margin，其餘被擠壓。

供給端可拆三層。上游（食材/設備/勞動/房東）：Sysco/US Foods 食材 + 勞動 + 房東，AB 1228 讓加州限服務餐飲價格 +13.1%、就業 -3.2%，議價權往上游勞動流；牛肉/雞/海鮮通膨進一步壓中游。中游（FC 餐廳營運）：avg check $10–15、labor 22–28% / food 28–30%，restaurant-level margin 暴力分化（10%–29%）——format moat 贏家（BROS 29%、SHAK 22%+）vs broken（SG 崩到 10%）。下游（tech/platform）：Toast/Olo POS（ARR +30–40%/年）+ DoorDash/Uber Eats delivery（take rate 20–30%）+ Wonder multi-brand 平台，議價權往下游集中。利潤往「上游勞動 + 下游平台」兩端流，中游 FC 除 format moat 外被擠壓。

§4 需求側：現在與未來 + 三角驗證

需求最剛性的是「消費者要 value + convenience」——$14+ check sticker shock 懲罰 broken FC、drive-thru/便利性獎勵 format moat；2026 Q1 的同店分化把這件事講得最白。

第一引擎：playbook 已商品化（無 alpha）。Niccol playbook 五大 lever 全 plateau——CMG digital 38.6% 已 saturate、loyalty 每家都有、throughput 不再是 bottleneck、Chipotlane 新店 AUV -10–15%、LTO comp lift 衰減。CMG FY25 -1.7%（首次年負 since 2017）+ Q1'26 僅 +0.5%、WING dom Q1'26 -8.7% 持續惡化——證明 digital alpha 已被段別複製、不再是差異化來源。值得釘住的是兩家的分歧：CMG 觸底回升（Q1'26 +0.5%、digital 38.6%、rev +7.4%），WING 反而更深（dom -8.7%、儘管系統營收靠 +17% 開店維持 +5.9%）——同樣是 playbook 群，一個像「觸底」、一個像「結構性問題」。第二引擎：format moat 是新 alpha（現需）。消費者用腳投票給格式：SHAK drive-thru 重置 Q1'26 +4.6%、traffic +1.4%（三季連續 +）、system +14.1%；BROS drinks-led 高毛利 + drive-thru-only，Q1'26 sys +8.3% / co-op +10.6% / rev +30.8%；CAVA Mediterranean bowl + GLP-1 protein alignment，Q1'26 SSS +9.7% re-accelerate、traffic +6.8%、rev +32.2%。需求把流量推向「有結構格式優勢」的 chain。

估值傳導機制：format moat 估值對「同店與單店經濟能否 sustain」敏感（SHAK build cost 達 $2.2M target、BROS co-op SSS）；CAVA 額外對「~150x 能否被同店撐住」敏感——同店一旦從 +9.7% 減速到 low-single-digit，~150x 的 de-rate 幅度可達 40%+；playbook 群對「同店是 lap 還是結構」敏感（CMG 觸底待證、WING -8.7% 偏結構）；broken 對「結局（M&A/Ch.11）」敏感。傳導鏈：format moat 同店兌現 → 贏家 multiple sustain；CAVA 同店減速 → ~150x de-rate；playbook 同店回正 → multiple 反彈。敏感度錨點：CAVA 從 ~150x 壓到 ~80x 即 -45%，這是段別最大的 de-rate 風險源。

估值現在 price 了什麼：format moat（SHAK/BROS）的成長已部分 price（~50x）；CAVA 的永續高速幾乎全 price（~150x）、減速尾部未充分 price；CMG 的 recovery 部分 price（~32x）；WING 的結構同店問題部分 price（但 ~45–50x 偏高）；SG 的 distress 進行中、Ch.11 尾部部分 price。會 re-rate 的是 playbook 群觸底反彈；會 de-rate 的是 CAVA 同店減速 + WING 結構續弱。對 PM：盯 format moat 單店經濟與 CAVA 同店 vs 估值，不是盯「FC 段別成長率」。

  • 我們說 Y：+6% 幾乎全 unit-driven（+5% units），comp ~0–1%；Black Box 顯示 FC 是「only 2 segments with negative SSS」之一；2026–2030 實際 CAGR 將 +3–5%、讓位 c-store foodservice。🔵 [X: base 很可能 — 段別 mature；bull 可能 — format moat 把餅撐住]。
  • Priced-in：市場仍把「FC 段別」泛泛當高成長定價（CAVA ~150x、SHAK/BROS ~50x），沒 price「段別整體 mature、僅 format moat 高速」的分化——分歧 1 的段別降速最未 priced。對照當期：FC 段別 2024 +0.6%、2025 +6%（unit-driven）已是「降速」鐵證，但估值仍 price 黃金期延續。
→ 推論邏輯：把「FC 段別成長」拆成 format moat（高速）vs segment 整體（mature）。⚠ 證偽：2026 FC segment 全年 SSS +2%+ 且 2027 +3%+（段別仍 growing）。
🟡分歧 2：「CMG/WING 的 FY25 弱只是短期 lap」信心：中
  • 共識說 X：CMG/WING FY24 base 高（CMG +7–9% / WING +21% Q4），FY25 弱純 lap effect，FY26 會反彈。🟢 [F: T1 — FY24 高 base]。
  • 我們說 Y：部分是 lap、部分是結構——CMG Q1'26 +0.5% 觸底（lap 解釋部分弱），但 WING Q1'26 dom -8.7% 持續惡化（純 lap 解釋不了），更像「digital playbook 已商品化、不再帶 traffic」的結構問題。🔵 [X: base 很可能 — CMG 緩步回正/WING 續弱；bear 可能 — 雙雙永久 plateau]。
  • Priced-in：CMG ~32x part-price 了 recovery（觸底已 confirmed）；WING ~45–50x 沒充分 price「結構同店」（-8.7% 比市場預期深）——分歧 2 的 WING 結構問題最未 priced。對照當期：CMG +0.5%（觸底）vs WING -8.7%（惡化）的分歧，正好把「lap vs 結構」釘住——同是 playbook 群，一個觸底、一個結構。
→ 推論邏輯：CMG 是 lap（觸底）、WING 偏結構（惡化），不能一概而論。⚠ 證偽：WING FY26 H2 同店回 +1%+（lap math confirmed、結構論部分作廢）。
🟡分歧 3：「CAVA 的成長理所當然、估值不是問題」信心：高
  • 共識說 X：CAVA Q1'26 +9.7% re-accelerate、提高 guide、1,000 by 2032，~150x 估值由成長 justify。🟢 [F: T1 — +9.7% SSS、rev +32.2%]。
  • 我們說 Y：成長確實強（這點我們同意、且推翻了舊版「蜜月期 ending」判斷），但 ~150x 已 price 永續高速——同店一旦從 +9.7% 減速到 low-single-digit，de-rate 幅度可達 40%+。CAVA 不是「估值便宜的成長」，是「成長很強但估值已 price perfection」。🔵 [X: base 很可能 — 同店撐住但 multiple 溫和壓；bear 可能 — 減速 + de-rate 40%+]。
  • Priced-in：市場幾乎完全 price 了 CAVA 永續高速（~150x），沒 price「同店減速 → multiple 崩」的尾部——分歧 3 的估值風險最未 priced，且是全段別最大的 de-rate 源。
→ 推論邏輯：CAVA 成長強 ≠ 估值安全；~150x 把未來幾年的完美成長都 price 進去了。⚠ 證偽：CAVA 同店維持 high-single-digit 3+ 年、開店至 1,000+（~150x 部分 justify）。

Steel-man 反方（最有力的 3 條）

  • 反方 1（playbook 觸底反彈）：CMG Q1'26 +0.5% 已觸底、WING Smart Kitchen 全 system 2026 見效，FY26 H2 雙雙回 +3%+，「playbook 商品化」narrative 證偽、multiple 反彈。當前證據：CMG 觸底、WING +17% 開店。證偽窗口：CMG/WING FY26 H2 同店。
  • 反方 2（CAVA 同店撐得住 ~150x）：Mediterranean 是結構性 GLP-1 winner（高蛋白 bowl）、unit growth 至 1,000 by 2032，同店維持 high-single-digit，~150x 由成長 justify。當前證據：Q1'26 +9.7%、traffic +6.8%、提高 guide。證偽窗口：CAVA FY26–27 全年同店。
  • 反方 3（SG 不會 Ch.11、Wonder 併購是 likely path）：SG 已賣 Spyce 給 Wonder（$186M, 2025/12）、現金充足、~$700M revenue 是 Wonder bolt-on candidate，Wonder 2026–27 併購而非 SG 破產。當前證據：Spyce 賣 Wonder、SG 現金 +$100M。證偽窗口：2026–27 SG 結局。

市場最低估的：最低估的風險是分歧 3 的「CAVA ~150x 同店減速 de-rate」與分歧 2 的「WING 結構同店」；最低估的分化是分歧 1 的「FC 段別整體 mature、僅 format moat 高速」。三者都該獨立監測。

三條分歧裡最可操作的是分歧 3（CAVA 估值風險）+ 分歧 2（WING vs CMG 的 lap/結構分歧）——前者是全段別最大的 de-rate 源（~150x），後者有明確領先指標（CMG 觸底續正 vs WING 續弱）。分歧 1（段別 mature）次之、但對「FC = 高成長」的整段定價最具殺傷力。
