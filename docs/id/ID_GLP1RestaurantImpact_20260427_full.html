§1 產業白話定義 + 歷史脈絡

這份報告問的是一個很具體的問題：當數千萬美國人開始打 GLP-1（食慾被藥理抑制），他們的外食消費會怎麼變、哪些餐飲 chain 受傷、哪些反而受惠？付錢的是消費者，受影響的是上市餐飲公司的同店銷售與利潤率。

白話開場：GLP-1 是一類原本治糖尿病、後來證實能大幅減重的藥物（Ozempic / Wegovy / Mounjaro / Zepbound），機制是抑制食慾、延緩胃排空、讓人提早有飽足感。對餐飲的含義很直接——吃藥的人吃得少、而且口味偏好改變（厭糖、想吃高蛋白）。市場一開始的恐慌很簡單：「如果幾千萬人都吃得少了，餐飲不就完了？」這份報告要拆解的，正是這個直覺對在哪、錯在哪。

邊界界定：本報告聚焦「GLP-1 對美國餐飲（fast food / fast casual / casual dining / 牛排館）的同店銷售與利潤衝擊」這條主軸，按「受壓 vs 受惠 vs 防禦」分類 22 檔上市公司。納入下游糖飲（KO/PEP）作 cross-tier context；不深入製藥端公司估值（LLY/NVO 只作 macro context，是兄弟 ID 的範圍）；不展開非美國市場（採用率與機制不同）；不展開下游 grocery / packaged food（KHC/GIS，獨立 ID）。灰色地帶在「SBUX 是受害還是受惠」——糖飲菜單 mix 高 → 受害，但 Niccol 主動 protein pivot 已兌現 → 受惠；這種「分類取決於 pivot 速度而非菜系」本身就是 §3 的 insight 來源。

歷史脈絡分四個轉折。第一次：2017-2020——Ozempic 2017 FDA 批 T2D，醫師只開給糖尿病人，沒有餐飲 narrative，美國用戶約 1.5M。第二次：2021-2023——Wegovy 2021/06 批肥胖、Mounjaro 2022 / Zepbound 2023 接力，Elon Musk / Oprah / TikTok 名人效應引爆，2023/10 Walmart McMillon 公開「basket pullback」是 Wall Street 第一次警覺，用戶約 5M。第三次：2024-2026（現在）——Cornell-JMR 訂單級資料公佈確認機制、DNUT -70%、Red Lobster 破產、SBUX/CMG/SHAK menu pivot 全面展開、CMS BALANCE 2025/12 公告；用戶 ~10-11M。第四次：2026-04-01——口服 Foundayo（orforglipron）獲批，是第一個無需注射、無食水限制的 GLP-1 藥丸，把 access 門檻打掉，啟動 S 曲線第二段。

最硬的反證測試：如果市場「海嘯論」對，GLP-1 user 應該大幅減少外食。但 Cornell-JMR 數萬戶訂單級資料顯示——user restaurant spending 12 個月後淨 +0.9%（先 dip 後恢復），Circana 顯示每趟訂單僅 -1%。用戶沒有離開餐廳，他們改變了點什麼——少點 snacks / breads / sides、多點主菜與高蛋白。這個事實比任何單季 SSS 都更能定義 thesis：不是全餐飲下沉，是品類內部重排。

歷史類比：1990s 香煙警語 + 高稅 → 2000s 消費永久 -40%。當年也是「藥理/健康 + 政策」三重夾擊讓消費永久改變。量化差異很關鍵：香煙是「整段品類萎縮」，GLP-1 是「品類內部重排」（用戶沒離開餐廳、只是換點）；且 GLP-1 速度快 3-5 倍（5-10 年 vs 香煙 30 年）。所以「GLP-1 對糖飲/甜點是永久逆風」這點與香煙相似（不會 fade），但「全餐飲 -40%」這個量級完全不適用——這正是把恐慌從海嘯拉回分化的第一個量化錨。

餐飲沒有半導體那種價格 cycle，改用「GLP-1 用戶滲透」的 S 曲線替代——看現在在採用曲線哪一段。

關於 cycle 統計表的省略：餐飲業 SSS 受宏觀 / 通膨 / 天氣多因子干擾，無單一供需價格 cycle（不像記憶體有 ASP peak-to-trough）；本 ID 改以 §2 的 GLP-1 用戶滲透 S 曲線呈現衝擊階段。可參考的時序錨點：用戶從 2017（1.5M）到 2025（~11M）約 8 年，2026-04-01 口服上市是第二段加速器，預估 2030 達 ~25M（JPM 共識）。

這段歷史的可操作含義是：真正該怕的不是「全餐飲下沉」（用戶沒離開），而是「無 pivot 路徑的甜食單品 + 低薪客群裸曝險」這個窄帶（DNUT/WING）。香煙類比告訴我們逆風是永久的（厭糖不會 fade），但只打到特定品類；分化論告訴我們同一機制也造了受惠端（高蛋白）。把資金配置從「avoid 餐飲」移到「long 受惠 / avoid 受壓」才是對的操作。

§2 核心機制 + S 曲線

為什麼是品類分化、不是整段下沉？因為 GLP-1 的機制不只是「吃得少」，更是「口味偏好系統性改變」——厭糖、反油、想吃高蛋白。同一個機制同時造了受壓（甜食）與受惠（高蛋白）的對偶。

GLP-1 透過五條傳導路徑改變餐飲消費。第一是 食慾抑制（receptor 在腦幹/下丘腦，飽足感提早），總卡路里 -10-15%（Cornell 隱含；早期 MS 估 -20-30% 偏高）。第二是 糖反感 / 高脂反胃（鈍化 VTA dopamine reward circuit），糖飲、甜點、油炸最受傷（savory snacks -10.1%、confections 大幅下滑、酒類 wine -52% / beer -43%）。第三是 蛋白偏好上升（飽足需更高蛋白，user 主動追高蛋白食物，「trading up to bigger steaks」）。第四是 消費結構 reallocation（預算從 fast food -8% 轉向 full-service 持平到正）。第五是 行為持久性（停藥後 40-55% 行為保留）。前兩條造 drag、第三第四條造 alpha——這就是分化的機制根源。

§3 供給側：菜單 pivot 能力是分水嶺

餐飲的供給側不是「廚房產能」問題，而是「菜單能不能跟著 GLP-1 user 的胃口重排」。pivot 能力（菜單 R&D + 供應鏈 + 品牌許可）才是真正的準入門檻——它把餐飲分成結構受惠、可兌現翻身、與只能挨打三類。

現供結構：GLP-1 user 的需求重排（少甜點/配菜、多主菜/高蛋白）對所有 chain 一致，差別在誰能跟著改菜單。高蛋白/健康碗（CAVA/SHAK/TXRH）有天然 alignment——牛排與雞肉碗本來就高蛋白，幾乎不需 pivot；翻身者（SBUX/CMG）靠主動 menu R&D 補位；100% indulgent 甜食單品（DNUT）沒有 pivot 路徑——你無法把甜甜圈變高蛋白。這道門檻把餐飲分成三類。

看利潤往哪一段集中——高蛋白/健康碗、value/防禦、甜食/低薪裸曝險誰賺走重排後的需求。

§3 → 因果閉合（回應 §1 的結構變數）

§1 提出「pivot 能力」是分水嶺——未來 3-5 年它是否仍 binding？答案是部分鬆動但核心仍 binding：menu R&D 與供應鏈這兩重會被全行業複製（42% 業者已動手），所以「加高蛋白菜」本身不再差異化；但「品牌許可」這一重很難被追上——CAVA 標高蛋白碗是 native、DNUT 標高蛋白甜甜圈是矛盾。這是 §7 要檢驗的「pivot 商品化 vs 結構 alignment」的核心張力。

12-36 個月內，供給端會出現「全行業都有高蛋白菜」的新常態，所以單純的 menu pivot 不再是 alpha 來源——真正持久的是「品牌許可 + 客群 demographic」。對投資的含義：押「結構受惠端」（CAVA/TXRH/SHAK，高蛋白是 DNA）比押「pivot 兌現者」（SBUX/CMG，binary）更穩；避開「無 pivot 路徑」（DNUT）與「低薪裸曝險」（WING）。

§4 需求側：用戶仍上餐廳但點得不一樣 + 三角驗證

§6 產業經濟學與估值傳導

餐飲在 GLP-1 era 的錢怎麼賺：受惠端靠「高蛋白 check 上升 + 客流流入」做 margin accretive，受壓端靠「無 pivot 路徑 + 流量槓桿」被 margin compress。估值跟著「SSS 分化」走——受惠 re-rate、受壓 de-rate。

估值傳導機制（含 priced-in reverse 推導）：餐飲股估值對「SSS 趨勢」極敏感——市場用 NTM SSS 給 P/E band。傳導鏈：分化 → SSS 分化 → P/E 分化。受壓端 priced-in reverse 推導：DNUT 過去 12M -70%，若市場用「合理 P/E × EPS」反推，這 -70% 隱含的 5y SSS drag 遠超本 ID 估的 -12~-16%——意味部分是過度恐慌（市場把 DNUT 當「GLP-1 = 末日」的純空標的）；但 DNUT 高 leverage（~$950M 債）+ 無 pivot 路徑 + MCD partnership 暫停（關 ~2,400 門市），讓 fade 仍未完成，過度恐慌與真實惡化並存。受惠端 priced-in：CAVA IPO 後高估值（一度 ~80x）已 price 結構受惠 narrative，SSS +9.7% 驗證但進場價偏貴——受惠是真的、alpha 未必還在現價。中間防禦（DPZ/MCD）GLP-1 風險 mispriced 較少，是 mispricing gap 最小的一類。

估值現在 price 了什麼：受壓端（DNUT -70%）price 了「GLP-1 = 末日」（部分過度恐慌但 leverage 真實）；受惠端（CAVA ~高 P/E）price 了「結構受惠」（真實但進場價貴）。會觸發受壓端 re-rate 的是「停藥救援 / 意外 pivot」；會觸發受惠端 de-rate 的是「pivot 商品化 / SSS 跌破 +1%」。對 PM：分化是真的、但兩端的 priced-in 程度不同——受壓端注意過度恐慌反彈、受惠端注意估值，中間防禦 mispricing 最小。

  • Priced-in：受壓端 DNUT/WING 估值已部分 price 停藥救援（否則跌更深）；但若停藥救援成真（2028 user ≤7%），受壓 chain 仍有 re-rate 空間、受惠端 alpha 也壓縮——這條若押「停藥 = 沒事」是錯的且未完全 priced。
→ 推論邏輯：停藥率高 ≠ 衝擊歸零（行為保留 + base 成長 + 口服解決停藥主因）。⚠ 證偽：2028 KFF/IQVIA currently user 仍 ≤7%（無加速），分歧 2 翻轉、逆風 fade。
🟡分歧 3：「protein pivot 是 commodity」是部分錯的——至少未來 3 年信心：中
  • 共識說 X：SBUX/CMG/SHAK protein pivot 都是 temporary marketing，alpha 12-24 個月被全行業 commoditize（42% 業者已跟進）。🟢 [F: T3 — Datassential 42% 業者考慮加 GLP-1-friendly 菜]。
  • 我們說 Y：pivot 需三重能力——menu R&D + 供應鏈 + 品牌許可；前兩重 12 個月可複製，第三重需 5+ 年品牌建構（CAVA 地中海 DNA、TXRH 牛排、SHAK 高端讓 Good Fit 不像 special diet）。🔵 [X: base 很可能 — 結構 alignment > pivot 商品化；bear 可能 — 全行業 pivot 抹平 alpha]。
  • Priced-in：市場目前把 CAVA/SHAK 的結構 alpha 純當利多 price（高 P/E），幾乎沒 price「pivot 商品化稀釋 alpha」的下行——這是最未被 priced、也最可操作的一條：盯 CAVA/SHAK SSS，若跌破 +1% 市場會被迫重新定價。
→ 推論邏輯：menu pivot 會商品化，但「品牌許可 + 客群」這一重難追。⚠ 證偽：2027 H2 全 mid-tier 完成 pivot 後 CAVA/SHAK SSS 連兩季 ≤+1%，分歧 3 的 bear 成立。

Steel-man 反方（最有力的 3 條）

  • 反方 1（停藥救援）：64.8% 12M 停藥 + 成本壓力，5 年後 active base 僅 GS bear 15-20M，逆風 fade、受壓 chain re-rate。當前證據：Truveta 高停藥率屬實。證偽窗口：2028 currently user 仍 ≤7%。
  • 反方 2（歷史飲食潮流終究 fade）：low-fat / low-carb / plant-based 都被預測「永久改變」但都證偽（BYND -78%）。當前證據：消費者對極端飲食的長期不耐受。為何信心低：GLP-1「不靠意志力」是與歷次潮流的根本差異（藥理 sticky）。
  • 反方 3（pivot 商品化）：42% 業者加高蛋白菜，2027-2028 全行業完成後「protein menu」不再 differentiating，CAVA/SHAK 結構 alpha 流失。當前證據：mid-tier（Olive Garden/Cheesecake）已跟進。證偽窗口：CAVA SSS 仍 +3-5% 持續（structural > pivot）。

市場最低估的風險：不是「末日」（人人都知道甜食受傷）、也不是停藥（已被廣泛討論），而是分歧 3 的 pivot 商品化——市場把 CAVA/SHAK 結構 alpha 純當利多，最沒準備好「全行業 pivot 後 alpha 被抹平」這個情境。這是本報告認為最該獨立監測的尾部風險（有明確領先指標：CAVA/SHAK SSS）。

三條分歧裡最可操作的是分歧 1（分化 > 平均）——它最 actionable 且已部分驗證（CAVA +9.7% vs WING -8.7%）。分歧 3（pivot 商品化）最未被 priced、有明確領先指標（CAVA/SHAK SSS），是最該獨立監測的尾部。分歧 2（停藥不等於沒事）方向對但部分 priced。
