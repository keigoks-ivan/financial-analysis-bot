AI 基建投資的八個非共識（2026 版）

從 8 份獨立 Industry DD 合讀萃取的 8 條核心 thesis。每條都挑戰一個主流共識、都有具體事實支撐、都附證偽條件。合起來構成一份 AI 基建投資的系統性視角：從需求（Jevons）到運算（BSPDN）到 IO（CPO 並存）到基建（gas turbine），四個層次環環相扣。

這不是單檔推薦，是 「為什麼現在這些主題值得關注 + 什麼事件會打臉」的 thesis map。每條 thesis 連結對應的深度 ID，讀者可依興趣下鑽。

發布日：2026-04-19 ｜ 基於 8 份 v1.0.1 ID ｜ 下次 flagship refresh：2026-07 季報

八大非共識 目錄
  1. 推論是 run-rate 不是 capex（Jevons）
  2. BSPDN 是 2027-2028 真護城河
  3. Rubin Ultra 4→2 die 是物理極限訊號
  4. TSMC HBM base die 是隱性贏家
  5. Scale-up / Scale-out 是兩個市場非 zero-sum
  6. NVDA $4B 4 元素 deal 保 pluggable+CPO 並存
  7. GEV gas turbine 是 2030+ 主力非過渡
  8. CDU 是液冷的 hybrid bonding（kingmaker）

structural Layer 4 基建

八條 thesis 合起來的組合意義

這八條 thesis 不是獨立推薦，是一張相互驗證的結構圖：若 T1（推論 run-rate 需求）成立 → T2-T8 的受益者都有多年 runway；若 T1 失效（AI capex 2027 peak） → T2-T8 全部 recalibrate。

四個層次的因果鏈

需求（L1）觸發

Jevons 效應使推論成本下降反而擴大 TAM → 持續性算力消費

運算（L2）鎖定

BSPDN 護城河 + CoWoS-L 物理極限 → TSMC 2027-2028 結構性領先 + hybrid bonding 時程前移

記憶 / IO（L3）擴張

HBM base die 新業務 + Scale-up/out 雙市場 + CPO 並存 → 利潤池從晶片擴散到連接

基建（L4）瓶頸

Gas turbine 延長 + CDU kingmaker → 2030+ 電力 / 冷卻仍是硬瓶頸，非「服務器+GPU」故事

組合曝險暗示（非投資建議）

若八條全部或大部分成立，組合應有以下 structural 曝險（而非 tactical 追漲）：

  • 核心錨（受益最深、估值最堅實）：TSMC 2330、NVDA（但漸受 ASIC 侵蝕）、ASML
  • 上游稀缺（結構性定價權）：BESI、ONTO、Credo、Lumentum / Coherent
  • 基建轉型（從工業股變 AI 受益）：VRT、ETN、GEV、Nidec
  • ASIC / 客製（對沖 NVDA 壟斷風險）：AVGO、MRVL
  • 二次受益者（被主流忽略）：Ibiden、Shinko、Chemours、Asetek

主要 kill scenario（八條同時動搖的路徑）

  • T1 kill：2027 Q2 NVIDIA DC revenue YoY 低於 +15%（從 +60%+ 腰斬） → 推論需求 thesis 破 → 下游全崩
  • Hyperscaler FCF 2026 H2 惡化（Meta FCF -95%+） → 2027 capex 下修 → L1 → 全鏈收縮
  • 台海地緣事件 → L2 / L3 供應中斷（TSMC / SK hynix / ASML 荷蘭等） → 全局重新洗牌
