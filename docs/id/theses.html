AI 基建投資的八個非共識（2026 版）

從 8 份獨立 Industry DD 合讀萃取的 8 條核心 thesis。每條都挑戰一個主流共識、都有具體事實支撐、都附證偽條件。合起來構成一份 AI 基建投資的系統性視角：從需求（Jevons）到運算（BSPDN）到 IO（CPO 並存）到基建（gas turbine），四個層次環環相扣。

這不是單檔推薦，是 「為什麼現在這些主題值得關注 + 什麼事件會打臉」的 thesis map。每條 thesis 連結對應的深度 ID，讀者可依興趣下鑽。

發布日：2026-04-19 ｜ 基於 8 份 v1.0.1 ID ｜ 下次 flagship refresh：2026-07 季報

九大非共識 目錄
  1. 推論是 run-rate 不是 capex（Jevons）
  2. BSPDN 是 2027-2028 真護城河
  3. Rubin Ultra 4→2 die 是物理極限訊號
  4. TSMC HBM base die 是隱性贏家
  5. Scale-up / Scale-out 是兩個市場非 zero-sum
  6. NVDA $4B 4 元素 deal 保 pluggable+CPO 並存
  7. GEV gas turbine 是 2030+ 主力非過渡
  8. CDU 是液冷的 hybrid bonding（kingmaker）
  9. 變壓器 2-3 年 lead time + site-specific 是比晶片更硬的 bottleneck

ID_LeadingEdgeNode →

structural Layer 2 運算
3
Rubin Ultra 4→2 die 退讓 = CoWoS-L 物理極限訊號
ID_AdvancedPackaging →
共識Tom's Hardware / TechPowerUp 把 Rubin Ultra 4-die → 2-die 視為 NVIDIA「產品策略調整」，對性能無影響

事實基礎CoWoS-L 2024 起量、4-die 架構失敗意味 5.5-reticle 已達 warpage 上限；BESI（AMAT 已持股 9%）結構性稀缺仍對 ✅；hybrid bonding 接棒前移 ✅。2026-04 symposium 新事實：① 2026 5.5-reticle CoWoS yield 已過 98%（成熟驗證 substrate scaling 節奏）；② 2027 9.5-reticle / 12xHBM4E、2028 14-reticle / 20xHBM5、2029 >14-reticle / 24xHBM5E + SoW-X >40-reticle / 64xHBM；③ CoPoS pilot 2026 Q2 仍存在但定位改為「並行選項」。對玻璃 thesis 的修正：LPKF / Corning / SKC 等玻璃廠的「必走時程」推遲 2 年，IRR ramp 受影響

Layer 3 · 記憶 / IO 層

HBM、網路、光連接
3 條
4
TSMC HBM base die 是隱性贏家（市場尚未 price in）
ID_HBM_Supercycle →
共識TSMC 被視為 foundry / CoWoS 故事，HBM 是 SK hynix / Samsung / Micron 的舞台
本 ID（v1.2 事實修正）方向不變：TSMC 獨家代工 HBM base die、從 memory 廠地盤切一塊 ✅。但 base die 工藝是兩段式跳級而非單一節點：HBM4/4E (2026-2027) 用 N12FFC+（不是 3nm），HBM5/5E (2028-2029) 才升到 N3P。對 thesis 影響：① 收入規模兩段，2026-2027 N12FFC+ wafer ASP 較低（~$5-7K）、2028+ N3P 才升到 leading-edge ASP（~$20K+）；② 2028 HBM5 起 base die 接近「mini SoC」（含 memory controller + interconnect controller），HBM 廠對 TSMC base die IP service 的結構性依賴反而更深
事實基礎HBM4 每個 stack 需 base die ✅；TSMC 獨家代工地位確認 ✅。2026-04 symposium 修正：① HBM4 base die = N12FFC+（IO bus 從 HBM3/3E 1024-bit 翻倍到 2048-bit，是頻寬翻倍真正 enabler）；② HBM4E (2027) base die 仍 N12FFC+；③ HBM5 (2028, 20 stacks) + HBM5E (2029, 24 stacks) base die 升到 N3P；④ 2024 → 2029 一個 CoWoS 內 HBM bandwidth +34X。收入估算修正：原「2027-2028 TSMC 新增 HBM base die rev ~$3-5B」改為「2026-2027 ~$2-3B (N12FFC+ 段)、2028+ ramp 到 ~$5-8B (N3P 段)」
structural Layer 3 記憶
5
Scale-up vs Scale-out 是兩個市場 — NVDA 與 AVGO 非 zero-sum
ID_AINetworking →
共識媒體普遍將 NVDA Networking 和 AVGO AI 網通並列對比，視為 zero-sum 競爭
本 ID兩者是並存市場：NVDA 吃 Scale-up（rack-internal，~$30B 2030）/ AVGO 吃 Scale-out（rack-to-rack，~$100B 2030），共同擴大而非互吃
事實基礎NVDA Q4 FY26 Networking $11B（+263% YoY）主為 NVLink + Spectrum-X；AVGO 網通 ~$15-20B 主為 Tomahawk + Jericho — 不同產品線；UALink 商用 2027+ 實際比共識晚
structural Layer 3 IO
6
NVDA $4B 4 元素 deal 證明 pluggable 與 CPO 並存 — LITE/COHR 被保證
ID_SiliconPhotonicsCPO →
共識CPO 將取代 pluggable，Lumentum / Coherent 有被淘汰風險；視為 value trap
本 ID（v1.2 精準化）NVDA 2026-03-02 對 LITE + COHR 合計投 $4B（multiyear agreement + purchase commitment + capacity access rights + R&D support 4 元素結構）就是鎖長期供應 → 兩者並存，非取代。2026-04 symposium 精準化：「並存」要分兩個窗口：① Switch-level（pluggable + CPO 並存到 2030+） — LITE / COHR 在 switch-to-switch 連線是長期贏家；② Chip-level（2029+ Interposer 階段 pluggable 沒空間） — chip-to-chip 距離（<50mm）COUPE on Interposer 10X power / <0.05X latency 全面勝過電 SerDes，pluggable 在這個 segment 會被結構性取代
事實基礎deal 結構遠超純股權（4 元素 multi-layer）；Lumentum 毛利從 31% → 42.5%；InP laser 供應 Coherent 是物理瓶頸（MOCVD 產能）；NVDA 需要 ensure supply。2026-04 symposium 新事實：TSMC 揭示 COUPE 四階段 ladder — Cu Wire (1X) → COUPE on PCB (2X power, 2026 ramp) → COUPE on Substrate (4X power / <0.1X latency, 2027-2028 step change) → COUPE on Interposer (10X power / <0.05X latency, 2029+ chip-to-chip 強制)。對 LITE/COHR 長期意義：2030+ 角色從「成品 laser 模組」轉型為「上游 InP wafer / epitaxy 供應」 — 這是市場還沒 model 的長期結構轉變，是「短期 NVDA $4B 投資」之外更深的 thesis

ID_Transformers →

structural Layer 4 基建

九條 thesis 合起來的組合意義

這九條 thesis 不是獨立推薦，是一張相互驗證的結構圖：若 T1（推論 run-rate 需求）成立 → T2-T9 的受益者都有多年 runway；若 T1 失效（AI capex 2027 peak） → T2-T9 全部 recalibrate。T9（變壓器 bottleneck）是 2026-04 新增的第九條，把「基建（L4）瓶頸」從液冷 + gas turbine 擴展到「變壓器 / 電網」，讓結構圖更完整。

四個層次的因果鏈

需求（L1）觸發

Jevons 效應使推論成本下降反而擴大 TAM → 持續性算力消費

運算（L2）鎖定

BSPDN 護城河 2026-2029+ 連續四代（A16 → A14 → A13 IP 相容）+ CoWoS interposer 持續 substrate 主導（2026 5.5x → 2029 >14x、SoW-X >40x）→ TSMC 結構性領先；hybrid bonding 時程前移；玻璃 / 面板級推遲到 2030+

記憶 / IO（L3）擴張

HBM base die 新業務（HBM4/4E N12FFC+ → HBM5/5E N3P 兩段升級）+ Scale-up/out 雙市場 + CPO 四階段 ladder（switch-level 並存到 2030+ / chip-level 2029+ Interposer 強制）→ 利潤池從晶片擴散到連接

基建（L4）瓶頸

Gas turbine 延長 + CDU kingmaker + 變壓器 2-3 年 lead time → 2030+ 電力 / 冷卻 / 變壓器三合一硬瓶頸，非「服務器+GPU」故事；GSU 47% 缺口卡 hyperscaler 自建電廠

組合曝險暗示（非投資建議）

若八條全部或大部分成立，組合應有以下 structural 曝險（而非 tactical 追漲）：

  • 核心錨（受益最深、估值最堅實）：TSMC 2330、NVDA（但漸受 ASIC 侵蝕）、ASML
  • 上游稀缺（結構性定價權）：BESI、ONTO、Credo、Lumentum / Coherent
  • 基建轉型（從工業股變 AI 受益）：VRT、ETN、GEV、Nidec、Hitachi Energy（HVDC + GSU + 印度 exposure 三線受益）、Siemens Energy Grid Tech
  • ASIC / 客製（對沖 NVDA 壟斷風險）：AVGO、MRVL
  • 二次受益者（被主流忽略）：Ibiden、Shinko、Chemours、Asetek

主要 kill scenario（八條同時動搖的路徑）

  • T1 kill：2027 Q2 NVIDIA DC revenue YoY 低於 +15%（從 +60%+ 腰斬） → 推論需求 thesis 破 → 下游全崩
  • Hyperscaler FCF 2026 H2 惡化（Meta FCF -95%+） → 2027 capex 下修 → L1 → 全鏈收縮
  • 台海地緣事件 → L2 / L3 供應中斷（TSMC / SK hynix / ASML 荷蘭等） → 全局重新洗牌

下次 refresh（2026-07）必查

  • AVGO Q2 FY26、TSMC Q2 法說 — L2 / L3 產能分配

  • VRT Q1 FY26（4/22）、GEV Q1 2026 — L4 基建訊號
  • Rubin 正式出貨 H2 2026 — 所有 thesis 的 validation point

⭐ Portfolio Implications — 八大 thesis 導出的 long/short map

把 8 條 thesis 交叉比對，挑出「同時在 ≥ 2 條 thesis 受益且無反向影響」的 super winners，以及「受 ≥ 2 條 thesis 反向影響且無對沖保護」的 super losers。這不是單檔推薦，是 thesis map 導出的結構性長 / 空分佈。
