② MCP 2025-12 捐 Linux Foundation 是 LLM 原廠議價權崩跌的結構性事件：Anthropic 自己放棄「agent 綁 Claude」的鎖定能力，OpenAI / Google 才敢跟進；一旦 agent 可跨 model 遷移（Code Mode 壓 99.9% token），LLM 原廠只剩「model quality + token 單價」兩軸競爭 → 2027 後商品化加速。真正的結構受益者是 agent identity 中立層（OKTA XAA、CYBR-Venafi NHI）+ agent infra（Cloudflare Workers + Firewall for AI + MCP portal），而非 Anthropic 自己
③ 「Anthropic 吃應用層 vs 資安 SaaS 保住 vertical」是非 zero-sum — 真正的分野是 Tier A 結構 moat / Tier C AppSec 高風險：Claude Mythos 能自主多階段攻防、Claude Code Security 觸發 Snyk / Cycode flash crash、Accenture Cyber.AI 把 vulnerability scan 從 3-5 天壓到 1 小時 — 但 EDR sensor / kernel telemetry / human MDR label（CRWD OverWatch）/ cross-vendor data lake（S1 Singularity）這四層 LLM 原廠 5 年內不會複製。Anthropic Amazon $25B + Google 14% 持股使其無需下沉自建 SOC SaaS，Glasswing 反而證明「原廠選擇與頭部 SaaS 共生，擠壓中腰部 SOAR / AppSec 純軟體」。補充（2026-04 patch）：CRWD 投資優先序在本 flagship 經過 kernel moat erosion 重新校準後已從「core long」降為「moat 換軌觀察」，因為 MSFT 2024-09 Windows Resiliency Initiative 主動削 CRWD kernel 鎖定 moat，2027 前 CRWD 必須用 data lake + MDR + Charlotte + 非 Windows 覆蓋重建非 kernel moat；短期優先序調整為 PANW > MSFT > CRWD，且 CRWD 持倉 sizing 應該比原本小。詳見 軟體股 flagship Thesis #1。
💡 本 ID 3 條最高價值 insight（預告）
第一條是 Token cost 每年降 2-4 倍，但資安 AI ARPU 以 credit bundle 持有，這是未被重訂價的 margin 槓桿。CRWD Charlotte 走 credit cap 40/月、Flex 模式把 AI COGS 直接 pass-through 給客戶，代表成本下降完全轉為毛利擴張。S FY26 Q4 GM 被壓 100-150 bps 看起來像 headwind，但其實是「先攤 capacity expansion」的前置投資，CY27-28 GM 會重新擴張——這不是結構性 headwind，是結構性 tailwind，市場現在誤讀為負面。第二條是 MCP + Agent Protocol 的真正贏家是 OKTA XAA 加上 NHI 類，不是 Anthropic。2025-06 XAA 被 MCP 收為企業授權標準，這在重演 Auth0 當年在 OAuth 2.0 時代的歷史位置——每個 agent session 抽 identity tax，比 per-token 收費更黏著更穩定，OKTA 和 NHI 類將是最大受益者，而 Anthropic 自己反而把 MCP 捐出去。第三條是 垂直資安 LLM 新創（Dropzone、Crogl、Simbian）100% 呼叫 API，是 CRWD / S / PANW 的併購標的，不是獨立贏家。這些新創融資規模 $10-60M 不夠自建 70B 級別的模型，被併購後第一個動作就是改掛自家 model——這是資安廠「反吃模型廠」的主要渠道，也解釋為何不該把這些 pure-play 當 Wiz-like exit 來投資。

§1 產業定義與邊界

「LLM 廠商 × 資安生態系的經濟學分工」這個主題回答一個關鍵問題：當 CRWD、PANW、S 都付錢給 Anthropic / OpenAI 做 AI SOC，誰最終賺到最多錢？要回答這個問題，必須把整個生態系拆成四個層次：Token 經濟、應用層蠶食威脅、MCP 協議利潤重塑、以及資安 SaaS 自建 LLM「反吃」模型廠的能力。每一層的利潤分配邏輯都不同，這一節先定義四層範圍。

💡 Insight §1
1. 四層經濟分工是分析 LLM × 資安的正確框架：token / 蠶食 / 協議 / 自建；忽略任一層都會得出錯誤 thesis
2. MCP 中立化把傳統「LLM 原廠 vs 應用層」二元對立拆解為多邊生態：agent identity / infra 是被忽略的中間層贏家
3. MSFT 是 LLM × 資安交集的最特殊玩家：既是 OpenAI 最大投資者又自建 Phi-4 + MAI-1，對 OpenAI 的依賴正快速下降

💡 Insight §2
1. AWS 類比最接近 Anthropic 的生態策略：推 primitives 讓上層繁榮而非直接吃
2. Oracle 類比適用 AppSec / SOAR pure-play 但不適用 CRWD / PANW 平台
3. MCP + Claude Mythos + Glasswing 組合是新型態 moat：Anthropic 把 headline 頂級 model 給 < 40 家 exclusive 防禦夥伴，反向綁定頭部 SaaS
