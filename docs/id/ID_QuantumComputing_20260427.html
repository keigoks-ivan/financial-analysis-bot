💡 3 條核心 insight（預告）
1. fault-tolerant 不是 binary 事件而是分波次商業化：Wave 1（2026-2029，數十 logical qubits，特殊化學 / 材料 / QEC research）已展開；Wave 2（2030-2032，數百 logical qubits）是真正商業化轉折；Wave 3（2033+）才是廣泛應用。Quantinuum 加速 roadmap 到 2029 是 Wave 1 末期領先；PsiQuantum + Atom Computing 在不同 modality 並進。
2. 量子 modality 之爭尚未決出：trapped ion（fidelity 領先 99.9999%）+ 中性原子（neutral atom 24 logical qubits 紀錄）+ 光子（PsiQuantum 99.98% prep）+ 超導（IBM Heron R2 156 qubit、Google Willow below-threshold）+ topological（Microsoft 仍在實驗）— 五個 modality 沒有絕對贏家，但 trapped ion 與中性原子在 2026-2028 logical qubit 競賽領先。
3. NVDA NVQLink 是「量子產業 NVDA 化」的關鍵訊號：17 quantum builders + 9 scientific labs 採用 NVQLink；CUDA-Q 變成 hybrid quantum-classical 通用平台；Quantinuum Helios 整合 NVDA Grace Blackwell — 與 NVDA 在 AI 早期 CUDA + GPU 的角色高度類似。市場低估 NVDA 在量子產業的「平台王」地位。

§1 產業定義與邊界

「量子計算」是個邊界相對清晰的產業，但內部分為多個 modality（超導、trapped ion、neutral atom、photonic、topological）+ 多個產業層（hardware QPU、software/middleware、application、services）。本節定義五個 modality + 三個產業層，並標註各層的關鍵玩家。重要的觀察是：量子計算市場規模目前極小（2026 ~$2B），但對應的算力與應用 potential 遠超當前商業 AI 規模。

💡 Insight §1
1. Modality 競爭是量子產業最大的不確定性：與超導 vs 矽 chip（一個贏家）不同，量子可能 2028+ 出現「不同 modality 主導不同應用」的多贏家局面 — Trapped ion 主導 fidelity 高的應用，超導主導 scale，光子主導長距離互聯，中性原子主導大規模並行。
2. NVDA NVQLink 把量子計算「重新定義為 GPU 加速器」：17 quantum builders + 9 scientific labs 採用，CUDA-Q 變 hybrid 通用平台 — 類比 NVDA 在 AI 早期 CUDA 的角色；但量子市場規模目前太小（NVDA quantum 收入 < 1% 全公司），市場低估這個未來 leverage。
3. Post-quantum cryptography 是不可避免的長尾基建支出：NIST 標準已確立（2024）；美國聯邦政府 2030 強制 PQC 遷移；隱含一個 $50-100B 的長尾「密碼學遷移」市場 — 但這個是基建支出而非量子計算市場；會被 IBM / MSFT / AWS 等大廠合計吃下。

§2 技術演進 + S 曲線 + 歷史類比

量子計算的技術演進有兩條主線：第一是 從「物理 qubit 數量競賽」轉為「logical qubit 競賽 + 錯誤糾正」，2024 年 Google Willow 的 below-threshold 錯誤糾正是一個關鍵 inflection — 證明錯誤糾正能 work；第二是 從「modality 之爭」進入「混合架構」，2025-2026 NVDA NVQLink 把 GPU 與 QPU 結合，多家 quantum builder 共同採用。這兩條主線決定了 2026-2030 的競爭格局。

§3 核心技術棧拆解

量子計算技術棧有五個關鍵層次：① 物理 qubit hardware（modality 之爭）；② 錯誤糾正 / fault-tolerance；③ Hybrid quantum-classical 整合；④ Quantum software / SDK；⑤ Application 層。每層都有 kingmaker 與瓶頸。本 ID 重點看前三層（hardware + QEC + hybrid），這三層是 2026-2030 主要競爭與商業化前線。

★ Kingmaker 子技術判斷

💡 Insight §3
1. Quantinuum 在 trapped ion + QEC 雙領先（2:1 encoding ratio + 94 protected logical）：是 Wave 1 fault-tolerant 最有希望率先兌現的玩家；但 Quantinuum 是 Honeywell 子公司，未獨立上市，公開市場無 pure play 對應。
2. NVDA NVQLink 把量子產業導向「GPU + QPU 混合」標準：17 quantum builders 採用是「平台王」訊號；類比 CUDA 在 AI 的角色；量子市場雖小但 NVDA 已悄悄卡位。
3. Modality 分散投資是當前最 prudent 配置：核心倉應跨 IBM + IonQ + Atom（每家代表一個 modality）；單壓任一家 modality 在 2027-2028 modality 之爭塵埃落定前風險過高。

§4 市場規模 + CAGR

量子計算市場規模目前極小（2026 ~$2B），但成長率極高（CAGR 22-25%）。本節區分「整體量子計算市場」與「pure play 量子廠營收」兩個口徑 — 後者是當前可投資的真實 revenue。

§5 供應鏈 Value Chain

量子計算的價值鏈仍處早期，與成熟半導體鏈不同 — 多數 vertical integration（hardware 廠自己設計 cryostat / qubit / control electronics / software），少數專業上游（dilution refrigerator、雷射、光學元件廠）。下游主要是研究機構 + 大型企業（化學 / 金融 / 國防 / 製藥）。本節呈現這個獨特結構。
