★ 本 ID 核心非共識 Thesis 市場把「先進封裝」與「CoWoS 擴產」畫等號，視為 TSMC 獨占故事。本 ID 認為：2026-2027 的真正贏家不是 TSMC（已被 price in），而是① BESI（AMAT 已持股 9%，2025-04 SEC 8-K）的 hybrid bonding 設備、② KLA / Onto 的 AP 專用檢測（量 + 良率雙重槓桿）、③ 日系基板廠（Ibiden / Shinko）脫離 PC/手機週期進入 AI HPC 獨立週期。Rubin Ultra 4→2-die 退讓 + CoPoS 時程 2028-2029（一直是 TSMC 原訂 guide，非延期）共同暴露有機基板物理極限，強化 hybrid bonding / 面板級封裝 Phase II 時程已被 Street 低估。
💡 本報告的 3 個核心 insight（預告）
1. Rubin Ultra 架構退讓（4-die → 2-die）是 CoWoS-L 物理天花板的首個 data point，強化 hybrid bonding / 玻璃路線的 Phase II 時程
2. AP 設備商 ROIC 槓桿 > 封測廠（BESI FY25 實際 op margin 29.3%、long-term target 40-55% vs ASE ~20%），TAM 成長 9.5% 在設備段放大到 15-20%
3. NVIDIA 佔 60% CoWoS 是最大單一 counterparty risk（thesis 對 AI capex 高度非對稱），§13 有具體 falsification 條件

灰色地帶（★ 邊界爭議）

三個爭議點，處理方式決定 §4 TAM 大小：

  • CoWoS vs 純 interposer：是否把 silicon interposer 單獨計 TAM？本 ID 把 interposer 視為 CoWoS 的子部件，不重複計算。
  • HBM base die（3nm 邏輯）算不算 AP？Yole 將 HBM base die 計入 AP TAM，本 ID 遵循此定義。
  • CoPoS 是否屬於 AP？TSMC 官方將 CoPoS 歸在 advanced packaging 類別，但面板級製程與 OSAT SiP 界線模糊。本 ID 將之納入，但備註「與面板 PCB / OLED 面板商有重疊」。
💡 Insight §1
1. AP 的邊界定義正在 drift：從「封測後段」擴展到「系統整合」— HBM base die 被計入 AP TAM 意味 foundry（TSMC）已侵入純 memory 廠（SK hynix）的地盤，估值倍數外溢
2. CoPoS 歸類爭議揭示利潤池邊界不清：若 CoPoS 歸 AP，TSMC 直接吃下面板廠的 addressable market；若歸 panel level，Ibiden / Shinko 有機會分到一塊 — 取決於 2028 業界規格收斂方向
3. 對投資人的意義：邊界模糊的產業通常估值波動大，因不同口徑下 TAM 差 20%+。閱讀券商 primer 時必須看其 AP 定義是否含 HBM base die

💡 Insight §2
1. 每次封裝轉折的「市場錯估時間」平均 2-3 年 — 空頭通常低估結構性技術的 persistence（interposer 持續 7 年），多頭通常低估新技術的 adoption 速度（ABF 晚 2 年才普及）
2. 贏家通常不是「宣告擁有技術」的公司，而是「卡在關鍵瓶頸」的供應商 — 2001 年是基板（Ibiden），2015 年是 interposer foundry（TSMC），2026 年輪到 hybrid bonding 設備（BESI）與檢測（KLA）
3. Rubin Ultra 4→2-die 架構退讓揭露有機基板物理極限：CoPoS 2028-2029 時程本就如此（非延期，是 TSMC 原訂 roadmap）；重點是 CoWoS-L 壽命可能短於 CoWoS-S，hybrid bonding 接棒時程提前 — 現在市場只當作「NVIDIA 產品調整」

💡 Insight §3
1. Hybrid Bonding 是本波的 kingmaker：這是從 2.5D 跨到 3D 的唯一路徑，也是唯一一家公司（BESI）近乎壟斷的子技術 — AMAT / LRCX 併購 BESI 傳聞不斷，結構性稀缺
2. 「CoWoS 良率問題」是偽問題：真正瓶頸是 warpage，不是矽製程 — Rubin Ultra 退讓證實物理極限比想像近。這意味 CoWoS-L 是過渡技術，長青性比 CoWoS-S 短
3. TGV 還遠未成熟（良率 < 40%）：玻璃基板是 2028-2030 故事；當前投資 LPKF 等純玻璃設備標的為時過早，應先看 hybrid bonding

💡 Insight §4

💡 Insight §5
1. 利潤池正在從中游（foundry）回流到上游（設備 + 基板）：過去 3 年設備商佔比從 15% → 20%+，這是「鐵鋤而非金礦」的結構性轉折
2. OSAT（ASE、Amkor）是最弱一環：毛利僅 18-22%，因受 TSMC 吸走 high-value AP 市場。ASE 的 AP 成長 +60% 但毛利提升有限，是量不是質
3. NVIDIA 60% CoWoS 分配使 TSMC 反而失去一些議價權：單一客戶過度集中，TSMC 必須配合 NVIDIA 節奏。這對 TSMC 是「表面榮耀但結構風險」

🟡 分歧 2：Rubin Ultra 4→2-die 退讓不是「NVIDIA 產品調整」，是「CoWoS-L 壽命縮短」的首個訊號 信心：中
  • 共識觀點（T3）：Tom's Hardware / TechPowerUp 報導「NVIDIA 調整 Rubin Ultra 架構，對性能無影響」，視為產品策略
  • 本 ID 觀點：退讓反映有機基板 CTE 與矽不匹配的物理極限已達。CoPoS 延期（2028-2029）進一步證實。CoWoS-L 作為主流技術壽命可能短於 CoWoS-S（7 年）
  • 分歧事實：CoWoS-L 才 2024 起量，4-die 架構失敗意味 3-die+ 封裝需要玻璃 / 面板級 — 而這兩者最快 2028
→ 若 CoWoS-L 壽命僅 4-5 年（2024-2029），hybrid bonding + 面板級 packaging 的商業化視窗比共識假設早 1-2 年
⚠ 證偽條件：若 2027 GTC 宣告 6-die CoWoS-L 可量產（意味 warping 已解決）→ 共識觀點正確
🟡 分歧 3：ASE 的 AP 成長是「量」不是「質」，長期不是贏家 信心：中
  • 共識觀點（T3-zh）：DIGITIMES / 券商把 ASE 列為 AP 主要受益者（revenue 翻倍）
  • 本 ID 觀點：ASE 毛利 20-22%，即使 AP revenue 翻倍，ROIC 仍停 10% 區間。真正利潤在上游（BESI FY25 實際 op margin 29.3%、毛利 63.3%）與 foundry（TSMC 55%）
  • 分歧事實：ASE 2026 AP target $3.2B，若毛利維持 22% = gross profit ~$700M；BESI FY25 實際 €591M × 29.3% op margin = €173M op profit（2026 orders 年化預示 €1B+ revenue），ROIC 天差地別
→ ASE 是「跑量」玩家，長期不如「收費站型」設備商
⚠ 證偽條件：若 ASE AP 毛利 2027 升至 30%+（當前 22%），共識正確
