本 ID 一句 Thesis（非共識） WFE 五寡占（ASML / AMAT / LRCX / TEL / KLA + SCREEN）正在經歷一次「結構性重新定價」而非另一個 cyclical 高點：（1）GAA + High-NA EUV 把 2nm 的設備強度拉高到 7nm 的 1.4-1.5 倍 → 即使 wafer starts 持平，WFE TAM 仍上行；（2）中國成熟節點反制（28nm+）只衝擊 ICAPS 段，先進節點（5nm 以下）90%+ 仍由五寡占獨占，且 Naura/AMEC 永遠無法跨越 EUV 障礙；（3）KLA 的 process control intensity 因 GAA 良率挑戰而結構性上升，是五寡占中最被低估的「複利機」。本 ID 看 2027 cycle trough 不會跌破 $130B WFE（vs 共識 $100-110B），且五寡占合計 PE 應從目前 22-26x 重估至 28-32x。

§1 產業定義與邊界

「WFE（Wafer Fab Equipment）」指晶圓廠前段製程所使用的所有資本設備，從原始矽晶圓進入潔淨室開始，到完成電路圖案、絕緣層、金屬連線、阱離子佈植、化學機械研磨等步驟為止。SEMI 的標準分類把 WFE 與「後段組裝測試設備（Assembly + Packaging + Test）」明確分開 — 這個分割很重要，因為兩個市場的玩家、毛利結構、cycle 週期都不同。[T2: SEMI Dec 2025 Equipment Forecast]

本 ID 的範圍刻意縮在「五寡占 + 兩日商」這個 framework — ASML（litho）/ AMAT（沉積、離子佈植、CMP）/ LRCX（蝕刻、ALD）/ KLA（檢測、量測）/ TEL（塗佈顯影、蝕刻）+ SCREEN（清洗）+ Hitachi High-Tech（特殊蝕刻）。為什麼是這 7 家？因為在 5nm 以下的「先進製程」這個子市場，這 7 家合計市佔率是 95%+；中國的 Naura、AMEC、SMEE 在成熟節點（28nm 以上）有 5-10% 的份額，但在先進製程則是零。把這 7 家當成一個 framework，就能用一個「Phase 判斷」推導出 7 家的 cycle 同步性，這正是本 ID 與既有 ID（LeadingEdgeNode 看 wafer fabbers、AITestEquipmentATE 看後段測試）的差異化。

邊界灰色地帶有三個值得獨立討論：（1）BESI（hybrid bonding）原本屬於後段封裝設備，但 AMAT 在 2025-04 取得 9% 持股後，雙方「co-develop integrated hybrid bonding system」已模糊前後段邊界，本 ID 把 BESI 列為「次要曝險 🟡」；（2）Hitachi High-Tech（半導體蝕刻 ECR plasma etcher）是日本第三家 WFE 玩家，市佔不大但在某些 niche（如磁性材料、conductor etch）為龍頭，列為「次要曝險 🟡」；（3）矽晶圓材料商（Shin-Etsu、SUMCO、Globalwafers）雖然與 WFE 同步 cycle，但屬於「materials」不是「equipment」，本 ID 不涵蓋。

另一個重要邊界：本 ID 把「成熟節點 ICAPS（IoT + Communications + Auto + Power + Sensors）」與「先進節點 leading-edge」明確分開計算。AMAT 自己 2026 Q1 法說會把這兩個 segment 拆開報數 — ICAPS 預估「全年大致持平」，leading-edge 卻預估「+20% 以上」。[T1: AMAT Q1 FY26 Earnings Call] 這個分裂結構決定了本 ID §6 玩家矩陣的競爭分析方法 — 五寡占主導 leading-edge，中國 + 國際小廠在 ICAPS 競爭，兩個競爭格局不能混談。

💡 §1 Insight Bullet（3 條）
1. WFE 的「先進 vs 成熟」二元結構是分析五寡占的唯一正確框架。事實基礎：AMAT FY26 Q1 法說明確把 ICAPS（成熟）vs leading-edge（先進）分開報數，前者全年持平、後者 +20%+。推論邏輯：把 WFE 當成單一市場討論會錯過 90% 的 alpha — 中國替代 narrative 只衝擊 ICAPS，而 leading-edge 才是五寡占毛利率的主要來源。投資意義：分析 WFE 公司時必須拆出 leading-edge revenue 占比 — AMAT ~70%、LRCX ~80%、KLA ~85%、ASML ~95%（EUV 全部都是 leading-edge）；越偏 leading-edge 的，受中國反制影響越小。

2. AMAT-BESI 9% 持股暗示「五寡占」實際已演變為「五寡占 + 1.5 個附庸」。事實基礎：2025-04 AMAT 取得 BESI 9% 股權 + co-develop hybrid bonding；同時 SCREEN 與 LRCX 在 wet clean 上分工互補。推論邏輯：五寡占內部正出現「拉小弟」的策略 — AMAT 通過 BESI 把 hybrid bonding 收編、TEL 與 ASML 在 EUV coater/developer 上 100% 綁定。這暗示新進者（中國）即使突破單一設備，也很難打破生態系。投資意義：BESI 應作為 AMAT 的「衍生選擇權」進場 — 漲幅彈性大、若 AMAT thesis 兌現，BESI 跟漲不對稱。

3. SEMI 統計的「WFE TAM」實際被低估約 15-20%。事實基礎：SEMI 把 advanced packaging 的部分（hybrid bonding、glass substrate、SoIC 設備）歸到「Assembly + Packaging」，但這些設備的客戶與 WFE 客戶完全相同（TSMC、Samsung、Intel）。推論邏輯：當投資人用 SEMI 的 $145B 推導市佔，實際的「五寡占 + adjacencies 有效 TAM」應為 $170-180B（再加 15-20% 的 advanced packaging 設備）。投資意義：估值 model 用 $170B 推 ASP × 市佔，會比用 $145B 多算出 15-20% 的 revenue runway，這是支撐 PE 倍數擴張的數字基礎。

§2 技術演進 + S 曲線 + 歷史類比

WFE 設備的技術演進歷史，本質上是「光波長 vs 線寬」的競賽 — 從 1980 年代 g-line（436nm 波長）到 i-line（365nm）、KrF（248nm）、ArF dry（193nm）、ArF immersion（193nm 加水）、最後到 EUV（13.5nm）和 High-NA EUV（13.5nm 但 NA 從 0.33 升到 0.55）。每一次世代轉換，都伴隨一個「在位者死亡 + 新王者誕生」的故事 — 這個故事在過去 30 年發生了 4 次，本 ID 認為 2026-2030 將是第 5 次（High-NA EUV 主導期）。

第一個歷史類比：1990 年代 Nikon 與 Canon 的 i-line / KrF 雙寡占崩塌。當 ArF immersion 在 2002-2005 出現時，ASML 押注 immersion，Nikon 押注 dry ArF — 結果 ASML 在 2007-2010 完全主導 immersion 市場，Nikon 從 50%+ litho 市佔跌到不到 10%。教訓：litho 設備商每次世代轉換都是「贏家通吃」，沒有第二名。第二個歷史類比：2010-2015 年 ASML 對 EUV 的 $5B 賭注。當時 Intel / TSMC / Samsung 三家共同投資 ASML 取得優先供應權，結果 EUV 在 2018-2019 才真正商用，ASML 賭贏；同期 Veeco 等公司也曾嘗試 EUV-related niches，但全部退出。教訓：litho 的 R&D cycle 是 10-15 年；只有資本足夠深的玩家能撐到回收。第三個歷史類比（反向警示）：2015-2018 年 LCD 玻璃基板從 Generation 8.5 到 Generation 10.5 — 當年市場預期 G10.5 line 會驅動 Corning / AGC 的爆發，但實際上 G10.5 的玻璃量遠不如預期，因為 LCD 需求被 OLED 蠶食。教訓（對 WFE 有警示意義）：技術迭代不一定帶動 TAM 擴張，要看下游需求是否同步擴張 — 本 ID 認為 AI / HBM / GAA 三者同時爆發的概率大於「需求被替代」的概率，但這是 §13 的 falsification 條件之一。

第四個歷史類比：2008-2012 NAND 從 2D planar 進入 3D NAND — 當時 LRCX 因為 high-aspect-ratio etch 的 know-how 成為唯一能服務 3D NAND 的 etcher 廠，從 2008 的 etcher 市佔 ~30% 升到 2014 的 ~50%。教訓：當製程結構發生 step change（FinFET → GAA、planar → 3D NAND、Cu → Co 線、High-NA EUV），有 know-how 的玩家獲得結構性 share gain。本 ID 認為 GAA + High-NA 的 step change 將讓 KLA（process control）和 LRCX（high-aspect-ratio etch）再次受益 — 這是個「重複歷史而非新故事」的 thesis。

第五個（反例）歷史類比：1990 年代日本 DRAM 廠商崩塌。當時日本 6 家 DRAM 廠（NEC、Hitachi、Toshiba、Mitsubishi、Fujitsu、Matsushita）擁有 80%+ 全球 DRAM 市佔，看似不可動搖；但 1990 年代末韓國（Samsung、Hyundai）以低成本快速攻城掠地，到 2000 年日本 DRAM 市佔跌到不到 20%。反向警示：當前五寡占看似不可動搖，但「在位者神話」並非永恆 — 中國的 Naura / AMEC 雖然只在成熟節點，但 5-10 年後若取得更深的政策支持 + 製程突破，可能重演韓國 DRAM 的故事。差異點：（1）litho 不像 DRAM，需要的不只是資本而是 know-how 累積（ASML EUV 用了 15 年）；（2）EUV 設備所需的 Zeiss 光學 + Cymer 光源是極窄供應鏈，中國若沒有原 ASML / Zeiss 的人才轉移，技術躍遷時程是 10 年級別。所以本 ID 認為「中國替代論」在先進節點為錯誤類比，但在成熟節點是正確類比 — 這是本 ID 的核心 differentiation。

💡 §2 Insight Bullet（3 條）
1. ASML High-NA 不是「升級」，而是「ASML 從 monopoly 變成 super-monopoly」的轉折點。事實基礎：High-NA EXE:5200 單機 $380M（vs Low-NA $200M = 1.9x），ASML 2027 計畫出貨 10 台 = 全部由 ASML 收。推論邏輯：每一次 litho 世代轉換都產生「贏家通吃」，但這次贏家在「自己內部接棒」，沒有競爭者搶食 → ASML EUV 收入應從 Q1 26 的 €4.1B/季 (annualized €16-17B) 升到 2028 的 €25-28B。投資意義：ASML 的 PE 25x 還沒 price in High-NA 的「unit ASP 翻倍」效應，FY27 EPS 上修空間 15-20%。

2. GAA + High-NA 對 KLA 是「complexity = revenue」的雙倍催化，市場低估。事實基礎：2nm wafer 成本是 7nm 的 3.5 倍（$28K vs $8K），且 GAA nanosheet 結構讓良率挑戰升一級 + High-NA 多重 patterning 增加 mask defect 風險。推論邏輯：KLA 的 process control intensity（KLA revenue / 整體 WFE）歷史上是 8-10%，本 ID 認為 2027-2028 會結構性升到 11-13%；且 KLA 在 patterned wafer inspection 75-80% 寡占 + reticle inspection 80%+ 寡占無對手。投資意義：KLA 是五寡占中「最被低估的複利機」（thesis 詳見 §12 #2）；FY27 revenue 可能比 sell-side consensus 高 8-12%。[T3-A: Robert Castellano on KLA]

3. 中國 Naura / AMEC 在先進節點 0% 市佔，這個結構不會在 5 年內被打破。事實基礎：中國 SMEE litho 仍在 28nm DUV 努力中（2025 仍未量產 28nm DUV），距離 EUV 至少 10-15 年；Naura / AMEC etcher / depo 雖在成熟節點 +4-5pp，但先進節點 0% 訂單。推論邏輯：litho 是 EUV 之外最難自主化的設備（光源、光學、polymer photoresist 都需 ASML 生態系）；蝕刻 / 沉積在先進節點需要 < 1nm 的 atomic-scale 控制，這需要 10 年的 know-how 累積。投資意義：中國替代論衝擊的是 ICAPS 段（AMAT ICAPS 30% 中國 revenue 是受傷段），但對 ASML / KLA / LRCX 在先進節點的 90%+ 市佔幾乎無影響 — 這是市場高估的風險。

§3 核心技術棧拆解

WFE 設備可拆成 8 個主要 subsystem，每個 subsystem 都有自己的玩家結構與毛利特徵；下表的目的是把「五寡占在哪裡寡占」這件事量化。從中可以看到一個關鍵 insight — 五寡占並不是「7 家公司在每個 subsystem 都競爭」，而是「每家在自己 1-2 個 subsystem 上都是 80%+ 寡占，彼此井水不犯河水」。這個分工結構是 WFE 的護城河 — 新進者要進入 WFE 不只是要做出設備，還要選一個「沒被佔據的山頭」，但所有山頭都已被佔。

3.2 良率瓶頸（為什麼 GAA + High-NA 把五寡占的議價權推升）

當製程從 7nm FinFET 進入 2nm GAA，主要良率瓶頸從「litho overlay 對齊」轉變為「nanosheet 形貌控制」與「multiple patterning EUV defect」。這個轉變對五寡占的影響是：

  1. ALD 沉積的關鍵性大增（LRCX / TEL 受益）：GAA 每層 nanosheet 都需要 atomic-layer 沉積；FinFET 主要靠 PECVD 即可。LRCX 的 ALD（Vector / Striker 系列）在 GAA 客戶滲透率從 FinFET 的 30% 升到 GAA 的 60%。
  2. High-aspect-ratio etch 的難度上升（LRCX 受益）：3D NAND 從 200+ 層走向 400+ 層、GAA 的 inner spacer etch 都需要 100:1 以上的 aspect ratio；LRCX 的 Kiyo / Flex 系列是唯一被三家先進記憶體廠（Samsung / SK Hynix / Micron）共同認證的方案。
  3. EUV mask defect 風險倍增（KLA 受益）：High-NA 需要 multiple exposure，每多一次曝光就多一倍 reticle defect 風險；KLA 的 reticle inspection 80%+ 市佔在這個趨勢下成為「不可繞過」的供應商。
  4. Wet clean step 數量倍增（SCREEN 受益）：每一次 EUV 曝光都需要前後 wet clean；High-NA 多重曝光的 clean 步驟從 7nm 的 ~50 步增到 2nm 的 ~80-100 步。SCREEN SU-3400 的 24-chamber 高吞吐量設計直接對應這個需求。[T2: Persistence Wafer Cleaning Market]
💡 §3 Insight Bullet（2 條）
1. 五寡占在「8 個 subsystem 中佔據 6 個 kingmaker」，這是它們維持 90%+ 先進製程市佔的結構性原因。事實基礎：litho（ASML 100% EUV）、etch（LRCX 50% + TEL 25% = 75%）、process control（KLA 55-60%）、coater（TEL 90%）、deposition（AMAT 40% + LRCX 25% = 65%）。推論邏輯：新進者要打破五寡占，不能只突破單一設備 — 需要同時突破 3-4 個 subsystem；而每個 subsystem 都有 10-15 年的 know-how 累積。投資意義：「中國替代」如果只在單一 subsystem 突破（如 Naura 蝕刻 5%），無法撼動五寡占；除非中國同時突破 3-4 個 subsystem，這在 5 年內不會發生 — 所以五寡占 thesis 仍然成立。

2. GAA + High-NA 的良率挑戰升一級，把 KLA / LRCX 的「process complexity premium」結構性提升。事實基礎：2nm GAA inner spacer etch 需要 100:1 aspect ratio + High-NA multiple exposure 翻倍 reticle defect 風險 + ALD step coverage 要求 atomic-layer。推論邏輯：這些 step change 對 LRCX（high-aspect-ratio etch）和 KLA（reticle inspection）的議價權都是結構性提升；而對 commodity 化的 wet clean 沒有影響。投資意義：在五寡占內部，LRCX 和 KLA 應該得到「complexity premium」 — PE 倍數應比 AMAT 多 3-5x。

7.3 抗 commoditization 結構性分析

WFE 為什麼能持續 ASP +5-10% YoY？三個結構性原因：

  1. 客戶寡占 + 設備寡占的雙向綁定：全球 leading-edge fab 不到 10 座（TSMC + Samsung + Intel + SK Hynix + Micron + 少數 IDM），客戶端高度集中；五寡占在 subsystem 也寡占，雙向綁定使設備商可長期定價而非標案競爭。
  2. R&D 入場費極高：ASML EUV 從 R&D 到量產用了 15 年 + $5B+ 投資；High-NA 累積投資 $10B+。新進者光是 R&D 入場費就是 prohibitive；五寡占則攤提多年後變成 sunk cost。
  3. 製程升級 = ASP 升級：每一次製程從 7nm → 5nm → 3nm → 2nm 都需要新一代設備（更多 EUV 層、更精準 inspection、更高 aspect ratio etch）。下游晶圓廠不得不升級，否則就被淘汰。這個「強迫升級」週期是 ASP 的結構性 driver。

WFE 全產業沒有任何 tool 出現 ASP 下降，這在 5 年觀察期是極罕見的「結構性無 commoditization」現象。事實基礎：§7.2 表中 9 個 tool ASP 全部 5 年連續上行；最 commodity 的 wet cleaner 也從 $3M → $5M (+67% in 5 yrs)。推論邏輯：唯一可解釋的原因是「技術 + 客戶」雙寡占讓 pricing power 結構性鎖死；這個結構在 GAA + High-NA + 中國替代失敗的 catalyst 推動下會更強化。投資意義：分析師應拋棄「WFE 是 cyclical industry」的舊框架，改用「technology rent extractor」的新框架；前者 PE peak 22-25x，後者 PE peak 32-35x — 這個 framework switch 是五寡占 thesis 的最大 alpha 來源。
