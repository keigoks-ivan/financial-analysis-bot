§1 產業白話定義 + 歷史脈絡

白話版：折扣零售過去被當成「一群賣便宜東西的店」。但它其實是 K 的兩端——中高收入往下找品牌價值（off-price 接），低收入剛性求生（dollar store 撐）。錢不是平均分給所有便宜店，而是往兩端各自的最強者集中。

白話開場：先說三個關鍵詞。Off-Price（TJX 旗下 TJ Maxx + Marshalls + HomeGoods、ROST、Burlington）買的是品牌商過剩的 closeout——同樣一件名牌，它用 20-60% off MSRP 賣；它的 moat 是 treasure hunt（不固定 SKU + 隨機到貨，逼你常常逛、線上難複製）。Dollar Store（Dollar Tree、Dollar General）賣的是低絕對價格點的 essentials，靠 rural 地理覆蓋 + 急購便利。K-shape是收入兩極化——高收入增加 share、低收入萎縮，把消費者往兩端推。本報告問的是：當 K-shape 強化、百貨死亡、關稅升級同時發生，折扣零售這塊大餅的錢，被重分配給了誰。

邊界界定：本報告聚焦「折扣零售兩端的結構分流 + 誰捕獲價值」。納入 off-price 龍頭（TJX/ROST/BURL）、off-price niche（OLLI/FIVE）、dollar store 雙頭（DLTR/DG）、mass merchant reference（WMT/TGT/COST）、百貨 loser（M/KSS）。純 e-commerce 折扣（Temu/SHEIN）out-of-scope（不同模式、見 GlobalEcommerce 子題）。灰色地帶是 Walmart/Target——定位 EDLP 但價格點高於 dollar store、多樣性高於 off-price，本報告列 mass merchant reference 不是 discount。

歷史分三段。第一段：1976–2010 創立與規模化——TJX 1976 開第一家 TJ Maxx、Ross 1982、Burlington 1972、DG 1939、Dollar Tree 1986；2008 金融危機是第一波 share migration（中收入 trade-down + 百貨第一輪壓力）。第二段：2010–2020 百貨死亡 + e-commerce 共存——TJX/ROST 對 e-commerce「免疫」（treasure hunt 線上難複製）、Macy's/JCPenney/Sears 連年關店；DG 在 Vasos 2008-2018 任 CEO 期規模爆發；DLTR 2015 收購 Family Dollar（後證實是錯誤決定）。第三段：2020–2026 K-shape + 關稅 + 結構終局——通膨 stickiness + SNAP 收緊 + 中收入壓力 + 2025-2026 trade war 2.0；DLTR 2025-07 售出 Family Dollar 回歸 pure-play、multi-price 規模化；DG Vasos 2024 回任 turnaround。

歷史類比一：1970s-1980s stagflation——中產壓力催生 discount 巨頭。TJX、Ross、Walmart 規模化都誕生於 1970s 通膨 + 失業的環境——當中產被擠，「找價值」需求催生折扣零售。量化錨：當前 2025-2026 通膨 stickiness + SNAP 收緊 + 關稅推高物價，K-shape 比 1970s stagflation 結構更深（不只是週期性收入壓力，而是上下兩端結構性拉開）。對 bull 含義：DLTR/DG 接低收入剛性、TJX/ROST 接中高收入 trade-in，雙端同時受益。歷史類比二：百貨死亡——mall anchor 的宿命。1990s-2000s Sears/JCPenney 衰落、2008 危機第一輪 collapse、2010-2020 Macy's/Kohl's 持續關店——這是 mall anchor 模式的結構終結、不是週期。量化錨：Macy's 客戶把 27% dept 支出移到 TJX（vs 25% prior），是「share migration confirmed、加速中」的硬證據；off-price 全美成衣 share 從 ~18%（2020）升到 ~23-24%（2025）、目標 2030 27-30%。教訓：判別一家折扣零售商站哪邊，看它接的是「百貨流出的中高收入 trade-in」（off-price 贏）還是「結構性低收入 SNAP-dependent」（dollar store 底層承壓）。

本主題無單一價格 cycle，用「百貨客戶支出遷移 + off-price comp + 客群收入分流」當時序錨。

關於 cycle 統計表的省略：折扣零售隨總經有週期（衰退時 trade-down 強、復甦時可能 mean revert），但本主題是結構性分流（K-shape + 百貨死亡 + 關稅），非單一 ASP cycle，故不立 cycle 統計表；以「折扣零售三段」（1976-2010 創立規模化 → 2010-2020 百貨死亡/e-commerce → 2020-2026 K-shape/關稅）的編年呈現時序。注意 DLTR 2025-07 剝離 Family Dollar 後，其 comp/sales 是有機口徑、不含 Family Dollar（避免無機口徑混淆）。

歷史類比的量化錨對照：把 1970s 與當前並排看，可以校準兩端的強度。1970s stagflation 期通膨年增率一度衝到雙位數、失業率高，催生 TJX/Ross 創立——但那是「單一收入層被壓」的週期；當前 2025-2026 是「上下兩端結構性拉開」——高收入因資產增值（股市 / 房市）擴大消費 share、低收入因 SNAP 收緊與物價剛性被擠到生存線。這就是為什麼連 DG 都報告 above-$100K 客群成長最快——trade-in 不只是中產下沉，而是全收入層「重新找價值」的結構行為。對照之下，2008 金融危機是第一輪百貨壓力（Sears/JCPenney 開始崩），但當時 Macy's 仍有客流；2026 的不同在於 Macy's 自己的客戶都把 27% dept 支出移走——這是 anchor 模式從內部瓦解，不是外部衝擊。歷史告訴我們：discount 黃金期不是「便宜就贏」，而是「結構分流時，兩端各自的最強者贏」——1970s 是 TJX/Walmart，2026 是 off-price 龍頭 + DLTR multi-price。

這段歷史的可操作含義：當前同時是「1970s discount 興起的升級版」（K-shape 兩端受益）與「百貨死亡終局」（share 往 off-price 移）。bull 的底是 off-price + dollar store best-in-class，bear 是 K-shape 收斂（中收入回流）+ 百貨復興——但 Macy's 27% 客戶支出流出已是結構性、非週期。判別一家公司站哪邊，看它接的是中高收入 trade-in 還是底層 SNAP-dependent。歷史的鐵律：折扣零售不是防禦同質籃子，而是結構分流下「兩端各自最強者」的集中遊戲。

§2 技術成熟度 + S 曲線

為什麼結構分流是現在？因為三件事同時成熟：百貨死亡進入終局（Macy's 27% 客戶支出流出）、關稅讓 off-price 的 closeout 供給暴增、DLTR multi-price 跨過「客單價 step-up」門檻——三者把折扣零售推向 K 的兩端。

結構拐點有三。百貨死亡終局：Macy's/Kohl's 不是週期性疲軟、而是 mall anchor 模式結構終結；Macy's 客戶 27% dept 支出已流向 TJX（vs 25% prior），off-price 全美成衣 share ~23-24%、目標 2030 27-30%。關稅推升 closeout 供給：trade war 2.0 讓直接進口零售商被迫清庫存，過剩 inventory 流向 off-price——「pack-and-hold」的好時機（TJX 已 offset 全部關稅壓力、Ross Q4 tariff 成本「negligible」）。DLTR multi-price 客單價 step-up：從 $1 一價 → $1.25/$3/$5 multi-price，~5,900 店、客單價 ticket Q1 +4.5%——這是 dollar store 從「固定低價 commodity」升級成「可成長 format」的拐點。三者交會，折扣零售從「同質防禦」分裂成「兩端各自成長」的兩個 sub-sector。

§3 供給側：現在與未來 + 利潤池

折扣零售的兩端用完全不同的供應鏈，關稅對它們的作用方向相反——off-price 90% 第三方 vendor 反向受益、dollar store 30-40% 直接進口被擠。這是兩端結構分流最硬的供給端證據。

供給端可拆兩條鏈。Off-price（TJX/ROST/BURL）closeout 採購鏈：90% 來自第三方 vendor——製造商過剩、退貨、季末 inventory、加上百貨關店處置；vendor 處理關稅 + 海運，不固定 SKU + 隨機到貨創造 treasure hunt moat；毛利 ~32-33%、inventory turn 高（6-7x/年）。Dollar store（DLTR/DG）直接進口鏈：30-40% 直接進口（中國 / 越南 / 印度）+ 私有品牌 essentials + 部分 closeout（DLTR）；固定價格點（DLTR multi-price $1.25/$3/$5、DG $1-$10），rural footprint（DG 多數店在小人口城鎮）+ 急購便利 moat；毛利 DLTR ~35-39% / DG 31.6%（Q1 +65bps）。關稅是分水嶺：off-price 反向受益（製造商過剩 closeout 流入、TJX 已 offset 全部關稅壓力、Ross Q4 tariff 成本「negligible」），dollar store 被擠（直接進口 + 私品牌無法 fully passthrough）。

整條鏈的利潤正從「百貨 + 中段 mass」往「off-price + dollar store best-in-class」遷移：百貨關店把成衣 share（成衣 sector $300B+ × 5pp share gain ≈ $15B）直接移轉給 off-price；關稅把製造商過剩 closeout 推給 off-price（vendor 議價權上升）；DLTR multi-price 把客單價 step-up（dollar store 升級）。新增的隱藏利潤層是 DG Media Network（FY25 $170M）——dollar store 用自家客流賣廣告的高毛利 layer，TJX/ROST 仍 0、是潛在 EBITDA lever。被搶的是百貨（M/KSS）與中段 mass（TGT），搶到的是 off-price 龍頭 + DLTR multi-price。（依 QC-18，折扣零售利潤池逐段精確百分比難拆，以定性 + share 數呈現。）

vendor 議價權的結構強化（最被低估的 secular trend）：off-price 的供給端有一個容易被忽略的飛輪——隨著 TJX 達到 5,000+ 店、百貨持續關店，製造商過剩 inventory 的「去化通路」越來越集中到 off-price。對品牌商而言，要快速清掉一批季末或退貨 inventory，TJX/ROST 的全國規模是少數能「整批吃下、不傷品牌定位」的買家。這讓 off-price 龍頭的 vendor 議價權持續上升、毛利 floor 結構性墊高（TJX ~32-33% gross margin 是 secular floor 而非週期高點）。關稅只是把這個飛輪加速——直接進口零售商被迫清庫存時，過剩量更大、流向 off-price 的 closeout 更多。這就是為什麼 TJX 能在關稅環境下「offset 全部關稅壓力」還擴張毛利，而 Macy's/Target 卻被同一股關稅壓得喘不過氣：同樣的關稅，對「買別人過剩貨」的 off-price 是 supply tailwind，對「自己直接進口」的零售商是 cost headwind。

未來供給與因果閉合：成為贏家的門檻是「closeout 採購規模 + 客流 moat + format 彈性」。純固定低價、無 closeout 規模的弱 operator（BIG / 99 Cents Only）補不上——這是 off-price 龍頭與 DLTR multi-price 守得住的原因。§1 提出「接 trade-in 還是接 SNAP-dependent」是站哪邊的判準——未來 3–5 年仍 binding：百貨死亡只會更深、關稅只會推更多 closeout 給 off-price；唯一鬆動是 dollar store 靠 multi-price 升級突圍（DLTR）或 turnaround 兌現（DG）——這是 §7 的反方。值得注意的是，dollar store 端正在長出第二條供給側護城河：DG Media Network（FY25 $170M）用自家 rural 客流賣高毛利廣告，這是把「實體客流」變現成「廣告利潤層」的嘗試；若 DG 能把這條 layer 規模化到 Walmart Connect 的相對量級，它的 op margin 天花板可能被部分撐開——但目前仍是 hidden leg、未進入估值。

§4 需求側：現在與未來 + 三角驗證

§7 Non-Consensus + Priced-in 檢驗 + Kill Scenario

⚠️ 本章含 judgment：每條分歧附 [F:] cornerstone + [X:] 三情境 + priced-in 檢驗。
市場對這主題的三個共識：①「折扣零售都一樣、買哪個都行」；②「關稅對折扣零售是 net negative」；③「DLTR 剝離 Family Dollar 只是 pure-play 簡化」。三個我們都部分不同意——分歧能否賺錢取決於市場 price 了多少。
🟡分歧 1：「折扣零售都一樣、buy the sector」信心：高
  • 共識說 X：TJX/ROST/DLTR/DG 都是 discount retail，買 sector ETF 或買哪個都一樣。🟢 [F: T3 — 折扣零售防禦敘事]。
  • 我們說 Y：折扣零售坐在 K 的兩端——off-price 接中高收入 trade-in + 百貨 share + 關稅受益，dollar store 接低收入 SNAP-dependent + 關稅曝險，六個維度（客群/採購/關稅/SNAP/毛利/moat）都結構不同。最佳工具是 K-shape barbell（TJX + DLTR）。🔵 [X: base 很可能 — 兩端分流；bear 可能 — K-shape 收斂、分流弱化]。
  • Priced-in：市場把「折扣零售」泛泛當防禦利多，沒 price「兩端結構分流、價值集中 best-in-class」的分化——分歧 1 的分流未 priced（DLTR multi-price step-up 尤其未 priced，16-18x 在 5Y 下緣）。
→ 推論邏輯：把「折扣零售利多」拆成 off-price（關稅受益）vs dollar store（關稅曝險）兩端。⚠ 證偽：K-shape 收斂、兩端表現趨同。
🟡分歧 2：「關稅對折扣零售是 net negative」信心：高
  • 共識說 X：關稅推高進口成本、所有零售都受害，折扣零售也不例外。🟢 [F: T2 — 關稅推升零售成本]。
  • 我們說 Y：關稅對 off-price 是 net positive——90% 第三方 vendor 吸收關稅 + 製造商過剩 closeout 流入（pack-and-hold）；TJX 已 offset 全部關稅壓力、Ross Q4 tariff 成本「negligible」。只有直接進口的 dollar store/百貨受害。🔵 [X: base 很可能 — off-price 受益；bear 可能 — 關稅 retraction、closeout 受益逆轉]。
  • Priced-in：市場 part-price 了「off-price 中和關稅」，但「關稅 step-up = off-price 毛利反向擴張」未充分 price——分歧 2 是 off-price 估值的 upside option。
→ 推論邏輯：off-price 90% 第三方 + closeout 增是關稅反向受益。⚠ 證偽：關稅大幅 retraction、off-price gross margin 連 2 季 -50bps+。
🟡分歧 3：「DLTR 剝離 Family Dollar 只是 pure-play 簡化」信心：中
  • 共識說 X：DLTR 2025-07 賣掉 Family Dollar 只是擺脫拖累、估值維持 16-18x。🟢 [F: T1 — Family Dollar 售 $1.0B]。
  • 我們說 Y：剝離 + multi-price 規模化（~5,900 店、ticket +4.5%、gross margin +120bps、EPS +42%）= 估值倍數 step-up 16-18x → 22-25x（vs ROST 22-24x 同質地）；不只是簡化、是 format 升級的 re-rate。🔵 [X: base 很可能 — gap 收斂；bear 可能 — multi-price 轉換延後、comp < +2%]。
  • Priced-in：市場把 DLTR 當「固定低價 dollar store」估值（16-18x、5Y 下緣），沒 price「multi-price 升級成可成長 format + pure-play 簡化」的倍數重評——分歧 3 的 step-up 最未 priced。
→ 推論邏輯：DLTR multi-price 是 dollar store 唯一成長 lever、估值倍數應 step-up。⚠ 證偽：DLTR FY26 comp < +2% / multi-price 7,500 店 target 延後。

Steel-man 反方（最有力的 3 條）

  • 反方 1（K-shape 收斂 / off-price mean revert）：通膨明顯 cool + 失業率 < 4%、中收入回流，off-price trade-in 動能減弱、TJX 估值 26-28x → 22-24x（-15-20%）；Macy's revival narrative 啟動。當前證據：off-price 估值含 share gain 溢價。證偽窗口：TJX comp 連 2 季 < +2% + Macy's comp turn positive。
  • 反方 2（DG turnaround 失敗 / Walmart rural 擠壓）：DG comp < +2%、Walmart 加碼 rural delivery、op margin 反轉至 5%、SNAP 11% 持續 headwind；Vasos 第二輪 turnaround 失敗。當前證據：SNAP $ 收緊、Walmart rural 加碼。證偽窗口：DG comp 與 op margin、Walmart rural footprint。
  • 反方 3（關稅大幅 retraction / off-price closeout 受益逆轉）：政治反轉、關稅 retraction、製造商不再過剩 inventory 流向 off-price、TJX gross margin -1-2pp；同時 dollar store 成本下降受益。當前證據：2025 關稅政策已多次反轉、部分國家豁免。證偽窗口：關稅政策方向 + off-price gross margin。

TJX vs ROST 是質地 vs 週期的差距，不是同一檔的兩個版本：兩家 Q1 都超預期（TJX comp +6%、ROST comp +17%），但市場容易誤把 ROST 更高的 comp 讀成「ROST 更好」。本報告的判讀是：TJX 是 30 年 compounder + 多 banner（TJ Maxx + Marshalls + HomeGoods + Sierra + HomeSense + TK Maxx Europe）的 diversified quality，comp 雖溫和但跨品類、跨地理分散風險；ROST 的 +17% 帶有更強的週期性彈性（單一市場、banner 較集中、10 年 track vs TJX 30 年）。換句話說，ROST 在 K-shape 強化期的 beta 更高（漲時漲更多），但在 K-shape 收斂期的回撤風險也更大；TJX 是 lower-beta、更耐週期的 core holding。這對 barbell 的含義是：off-price 端用 TJX 當 anchor（穩健 + 關稅 immune + Macy's share gain）、ROST/BURL 當 cyclical satellite，而非把三者當同質替代。

市場最低估的：最低估的upside是分歧 2 的「關稅 step-up = off-price 毛利反向擴張」與分歧 3 的「DLTR multi-price step-up」；最低估的風險是反方 2 的「DG SNAP/Walmart 天花板」與反方 1 的「K-shape 收斂」。三者都該獨立監測。

三條分歧裡最可操作的是分歧 1（兩端結構分流 → barbell）+ 分歧 3（DLTR multi-price step-up）——前者是工具選擇（barbell 而非 ETF），後者有明確領先指標（multi-price 店數 + 客單價）且最未 priced。分歧 2（關稅 net positive）是 off-price 的 upside option。
