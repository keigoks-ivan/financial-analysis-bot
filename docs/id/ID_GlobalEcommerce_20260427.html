全球 E-commerce 整合（Global E-commerce Consolidation）

發布日：2026-04-27（v2.0 深度重寫，整合 AMZN Q1 26 / SHOP Q4 25 / Temu post-de-minimis） ｜ 涵蓋 ticker：11 檔（3🔴 / 5🟡 / 3🟢）｜ 🟡 占比：18% ｜ 投資時鐘：Phase IV 廣告利潤池主導 + AI 推薦 + 區域 leapfrog ｜ industry-analyst v1.8

Unit Glossary：GMV（Gross Merchandise Volume）= 平台總交易額；1P / 3P（first-party / third-party）= 平台自營 vs 第三方賣家；de minimis = 美國 $800 以下進口免稅，2025-08-29 對中國取消；T01 customs clearance = 中國跨境物流 in-bond formal entry，繞 de minimis 限制；B2B2C = 平台批發給商家再零售；OS 模式（operating system）= SHOP 為商家提供 e-comm OS；Marketplace = AMZN/MELI/Shopee 集合 sellers；Headless commerce = 前台 UI 與後端 commerce 分離。所有 financials = Q4 2025 / Q1 2026 actual。

★ 本 ID 核心非共識 Thesis 市場用「AMZN vs Temu」單一框架看 e-commerce — 認為跨境 dumping vs 在地龍頭是主旋律。本 ID 認為：① 廣告利潤池才是 e-commerce 真結構性 alpha（AMZN Q1 26 ad rev $14.6B+ / 占 retail 利潤 50%+；ad GM 70% vs retail GM 25%）；② SHOP B2B GMV +96% 是被低估的 e-commerce 新章（vs SMB-only 共識；FY25 GMV $378B +29% / Shopify Payments $84B +38%）；③ Temu / 中國跨境攻擊已從 dumping pivot 到 T01/B2B2C 結構（2025-08-29 de minimis 終止後）— 不是消失而是 redirect 到不同層級；④ 四層分化（規模 / OS / 飛輪 / 跨境）取代「AMZN vs Temu」單一框架。
💡 §0 4 條核心 insight（預告）
1. AMZN ad revenue +20% YoY ($52.3B → $56.2B 2025E)：是 retail GM 25% 的 ~3x 賺錢業務；廣告增速 是 retail GMV 增速的 1.5-2x。被市場視為 retail story 但實為 ad story。
2. SHOP Q4 25 GMV $124B 首破 $100B + B2B +96% / Shopify Payments $84B +38%：SHOP 從 SMB 升級為「OS for commerce」；B2B GMV +96% 是「Shopify B2B 不是傳說是現實」直接證據。
3. Temu de minimis 2025-08-29 終止 + tariff 120%→54% pivot 完成：Temu 沒有消失，是 pivot 到 T01 customs + B2B2C 模式；MELI / Shopee 在地物流仍是護城河。
4. SHOP fwd PE 70x vs AMZN 35x = 2x premium：市場 priced SHOP 為 best-in-class growth，但被忽略的是 SHOP 的 commerce OS 角色（不只 SMB）+ B2B 新引擎；fwd PE 結構性合理。

★ 灰色地帶

  • SHOP 是 SMB 還是 OS for commerce？：傳統認知 SHOP = SMB e-comm SaaS，但 B2B GMV +96% / Shopify Payments $84B / merchant solutions revenue 重塑為 enterprise commerce OS — fwd PE 70x prem 合理。
  • Temu 的真實本質：dumping 還是 wholesaler？：de minimis 終止後 Temu 不是消失，是 pivot — T01 customs clearance + B2B2C + 在地 fulfillment 試水。是「中國 wholesaler with last-mile reach」而非「直接零售商」。
  • WMT Marketplace 是規模龍頭還是傳統 retail？：WMT 北美 e-comm +20%+ YoY 主要靠 Marketplace（3P sellers），是 AMZN-like 模式但規模更小；高速擴張中。
  • MELI 是否歸 LATAM 還是 Global？：MELI 是 LATAM pure-play 但同時是 Global e-comm 的「區域 leapfrog 案例」。本 ID 引用 ID_LATAMEcommerce 但仍 cover MELI 為 cross-region thesis 的一部分。
  • SE Sea Limited 跨 SEA + LATAM：Shopee 主場 SEA，但 Brazil 是第 5 大；混合曝險讓估值彈性高。
💡 §1 Insight
1. 「全球 e-commerce」實際是 7 個結構性差異 segments：規模 marketplace / OS / 跨境 / super-app / 區域 leapfrog / 傳統 retail pivot / niche — 不能用單一 framework。
2. SHOP B2B 是被低估的 OS evolution：從 SMB → OS for commerce 是估值 prem 結構性合理。
3. Temu pivot 是 e-comm sector 重要結構轉變：dumping 不可持續，但跨境玩家結構性留存（不同層級）。

§12 Non-Consensus View（共識差異）

🟡 #1 [structural] 廣告利潤池才是 e-commerce 真結構性 alpha [信心: 高]
共識（Wall Street）：AMZN 是 retail / Marketplace company，估值看 retail GMV growth。本 ID 看法：AMZN ad rev 占 retail 利潤 50%+ / ad GM 70% vs retail 25% / 廣告 +20% YoY = AMZN 真 alpha 是 ad business；fwd PE 應 priced as ad company 而非 retail。
事實 1: AMZN 2024 ad rev $52.3B → 2025E ~$56.2B (+20%)
事實 2: ad GM 70% vs retail GM 25%
事實 3: 廣告占 retail segment 利潤 50%+
→ 推論：AMZN fwd PE 應 32x → 35-38x re-rating；alpha source 是 ad business。
⚠ 證偽：若 AMZN ad revenue 增速降至 <15% YoY 連 2 季。
🟡 #2 [structural] SHOP B2B GMV +96% 是被低估的 e-commerce 新章 [信心: 高]
共識：SHOP 是 SMB e-commerce SaaS company，主要靠 SMB merchant growth。本 ID 看法：SHOP B2B GMV +96% / Shopify Payments $84B / Q4 25 GMV $124B (+31%) = SHOP 升級為「OS for commerce」+ B2B 新引擎；fwd PE 70x prem 結構合理。
事實 1: SHOP B2B GMV FY25 +96%（Q4 +84%）
事實 2: Q4 25 GMV $124B 首破 $100B 季
事實 3: Shopify Payments $84B (+38%) / merchant solutions ramp
→ 推論：SHOP fwd PE 70x 結構性合理；alpha 主來自 OS + B2B 雙引擎。
⚠ 證偽：若 SHOP B2B 增速 <30% YoY 連 2 季 OR GMV 增速 <20%。
🟡 #3 [event-triggered] Temu / 中國跨境攻擊已過峰，但結構性留存 [信心: 中]
共識：Temu 2025-08-29 de minimis 終止後將大幅萎縮；中國跨境 dumping 結束。本 ID 看法：Temu 沒有消失，是 pivot 到 T01 customs + B2B2C + 在地 fulfillment 試水；不再是 dumping 但仍 structural player（不同層級）。
事實 1: 2025-08-29 美國 de minimis 終止；US tariff 120% → 54% pivot
事實 2: Temu T01 customs clearance + B2B2C 模式啟動
事實 3: Temu / Shein 美國 fulfillment 試水
→ 推論：Temu 結構性留存但 alpha source 改變；不再是 dumping pricing。
⚠ 證偽（QC-I22）：若 Temu 2027 美國 GMV 恢復至 2024 peak +20% 連 2 季 → dumping pivot 失敗 thesis。
💡 §12 Insight

2. Thesis #2（SHOP B2B）是 next 12-month catalyst：每季財報直接驗證。
3. Thesis #3（Temu pivot）是 14 天 monitor 條件：post-de-minimis 結構演進需持續追蹤。

結論

全球 e-commerce 進入 Phase IV（2025+ 廣告 + AI + 區域 leapfrog 整合期）。核心非共識三條：（1）廣告利潤池是 e-commerce 真結構性 alpha；（2）SHOP B2B GMV +96% 是被低估的 e-commerce 新章；（3）Temu 攻擊已過峰但結構性留存。

結構性受益層次：（A）規模龍頭 + 廣告（AMZN）；（B）OS for commerce + B2B（SHOP）；（C）區域 leapfrog（MELI / CPNG / SE）；（D）二次受益（V/MA cross-border）。結構性壓力層次：跨境 dumping（Temu / Shein 舊模式）+ niche e-comm（ETSY 被擠）。
