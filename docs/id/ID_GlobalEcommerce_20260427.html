關於「全球電商」，市場最常見的偷懶框架是「AMZN（在地規模龍頭）vs Temu（中國跨境傾銷）」這場二元戰爭。這個框架在 2020–2024 還對，但 2025 起已經失效——de minimis 在 2025-05-02 對中國取消後，Temu 美國日活一度 -48%、傾銷模式不可持續；而真正的 alpha 早已不在「誰賣更便宜」，而在三條被忽略的結構性利潤層。

本報告把全球電商拆成四層分化：① 廣告利潤池——AMZN Q1 26 廣告 $17.2B（+24% YoY、trailing 過 $70B），ad 毛利 ~70% 是 retail 毛利 ~25% 的近 3 倍，AMZN 真身是 ad company；② OS for commerce——SHOP Q4 25 GMV $123.8B（+31%）、B2B GMV +96%、Payments $84B（+38%、占 GMV 68%），從 SMB SaaS 升級為企業級電商作業系統；③ 區域物流 leapfrog——MELI（Q1 26 rev +49%）、SE（Shopee GMV $37.3B +30%）、CPNG 用在地履約築起跨境打不穿的護城河；④ 跨境洗牌——Temu/Shein 從傾銷 pivot 到半托管 + 在地履約 + 廣告變現。

所以本報告不押「電商滲透率上升 = 電商股齊漲」，而是畫出價值集中在哪三層、誰被商品化。結論：押有「廣告閉環 / OS lock-in / 在地物流」的 AMZN / SHOP / MELI，最明確的結構性壓力方是無差異跨境傾銷（Temu 舊模式）與被 Marketplace 擠的 niche（ETSY）。最大的長期裂縫不是 Temu，而是 AI agentic shopping（2027+）可能改寫流量入口、稀釋 AMZN 的廣告閉環。

PART I · DECISION SUMMARY

決策摘要層

三句話看完全球電商整合的現狀、走向與行動，後接 PM 級監測點。這一層是報告的全部重點——下面各節只是把這一層的每個判斷補上證據。

現在 / NOW

全球 e-comm 進 Phase IV——AMZN Q1 26 廣告 $17.2B（+24%、trailing 過 $70B）/ 國際 +11% / 單量 +15%；SHOP Q4 25 GMV $123.8B（+31%）/ B2B +96% / Payments $84B；MELI Q1 26 rev +49% / GMV +42%；Temu de minimis（2025-05-02）後美國日活 -48%、pivot 半托管。廣告利潤池 + OS + 區域 leapfrog 是真 alpha。

未來 / NEXT

3–5 年 e-comm alpha 從「retail GMV」移到「廣告 + OS + 在地物流」三層；AMZN 廣告往 ad-company 估值靠攏、SHOP B2B/Payments 續證 OS 角色、MELI/SE/CPNG 在地物流擋跨境。最大裂縫 = AI agentic shopping（2027+）改寫流量入口、稀釋 AMZN 廣告閉環。

行動 / ACTION

押有廣告閉環 / OS lock-in / 在地物流的 AMZN（廣告 alpha）/ SHOP（OS+B2B）/ MELI（LATAM 飛輪）；WMT/CPNG/SE 為區域受益；避無差異跨境傾銷（Temu 舊模式）與被擠 niche（ETSY）；盯 AMZN 廣告增速與 agentic shopping 採用。買賣由個股 DD + 系統決定。

The Key Call · 本報告最重要的一個判斷

市場用「AMZN vs Temu」單一框架看電商——以為跨境傾銷對在地龍頭是主旋律。真正的非共識是四層分化：AMZN 的真身是 ad company（廣告 $17.2B +24%、毛利 ~70% vs retail ~25%）；SHOP 從 SMB SaaS 升級為企業級電商 OS（B2B GMV +96%、Payments 占 GMV 68%）；MELI/SE/CPNG 用在地物流築起跨境打不穿的護城河；而 Temu de minimis 後不是消失，是 pivot 到半托管 + 廣告變現。電商的 alpha 從「誰賣更便宜」移到「誰握廣告閉環 / OS lock-in / 在地履約」——這才是全球電商的核心賭注。

四層分化具體是：①廣告利潤池——AMZN 廣告 +24% 遠快於國際 retail +11%、trailing 過 $70B，是 ad 毛利 ~70% 的高利潤層，市場仍把 AMZN 當 retail 公司估；②OS for commerce——SHOP B2B GMV +96%、Payments $84B 占 GMV 68%，是「Shopify 不只 SMB」的直接證據；③區域物流 leapfrog——MELI rev +49%、SE Shopee GMV +30%、CPNG 三位數成長，在地履約是跨境玩家 5+ 年才追得上的護城河；④跨境洗牌——Temu 美國日活 -48% 但 pivot 半托管 + 歐洲 +30% GMV，不是退出而是 redirect。盯這四條，比盯「電商滲透率」更能看清錢的流向。

PART II · HISTORICAL ANALOGS

規模整合與跨境退潮，框住多空

歷史不必重演，但兩個對照各給一個教訓：AMZN 2010–2018 規模整合給出「利潤池會悄悄從 retail 移到高毛利層」，2020–2024 跨境傾銷浪潮給出「補貼換市佔不可持續」。

類比一：2010–2018 AMZN 規模整合——利潤池悄悄遷移。上一輪美國電商整合，AMZN 靠規模 + Prime + 物流取得龍頭，eBay 與傳統零售落後。量化錨：看空 AMZN「retail 不賺錢」的空頭錯了十年——AMZN 2010–2025 漲約 30 倍，因為真正的利潤引擎不是 retail（毛利 ~25% 的 commodity），而是悄悄長出來的 AWS 與廣告。對本次 bull 含義：當前 AMZN 廣告 +24%、trailing 過 $70B，正是同一齣戲的下一幕——市場又一次把 AMZN 當 retail 估，而 alpha 在高毛利層。

怎麼讀：當前 Phase IV 同時是「規模整合的下一幕」（利潤往廣告/OS/物流遷移）與「跨境傾銷退潮」（de minimis 取消後補貼模式崩）。bull 的底是握廣告閉環/OS/在地物流者（AMZN/SHOP/MELI），bear（對跨境傾銷）是 Temu 舊模式被 2025-05-02 結構性 reset。

類比二：2020–2024 跨境傾銷浪潮——補貼換市佔的宿命。Temu/Shein/AliExpress 靠 de minimis 免稅 + 直營跨境，2020–2024 在各國電商衝出爆發式 GMV。量化錨：2025-05-02 美國對中國取消 de minimis、US tariff 一度 120% 後談判至約 54%，Temu 美國日活在 2025 年 3→5 月一度 -48%、June 2025 調查顯示 45% 美國消費者減少或停用 Temu。教訓：靠政策套利 + 補貼換來的市佔，政策一變就退潮。但 Temu 沒消失——它 pivot 到半托管（2026 商業化 + 廣告變現、目標年度損益平衡）、把美國 GMV 占比目標從 60% 降到 30%、歐洲靠在地庫存 +30% GMV。判別訊號：跨境玩家賣的是「補貼價差」還是「在地履約 + 廣告變現」。

PART III · SUPPLY SIDE

供給側：電商價值鏈，利潤往三層集中

反直覺但可操作的結論：全球電商的利潤往「廣告閉環 + OS lock-in + 在地物流」三層集中——這三層贏，無差異跨境傾銷與被擠 niche 輸。

供給端可拆五層。廣告利潤池（AMZN/MELI/WMT）：平台自帶購買意圖訊號 + 閉環衡量，ad 毛利 ~70% 是 retail ~25% 的近 3 倍，AMZN 廣告 +24% 是真 alpha。OS for commerce（SHOP）：商家 lock-in + Payments + B2B，從 SMB SaaS 升級企業級電商作業系統。在地物流 leapfrog（MELI/SE/CPNG）：自建履約網絡是跨境玩家 5+ 年才追得上的護城河。1P/3P Marketplace（AMZN/MELI/Shopee）：規模 + Prime 訂閱鎖客。跨境傾銷（Temu/Shein 舊模式）：補貼換 GMV、毛利薄、政策敏感、正被商品化。利潤往前三層集中，跨境傾銷層失血。

Inference · 供給
利潤往廣告閉環 + OS lock-in + 在地物流三層集中
前提：廣告 ad 毛利 ~70%（AMZN +24%）+ SHOP OS lock-in（B2B +96%、Payments 占 68%）+ 在地物流護城河（MELI/SE/CPNG）+ 跨境傾銷被 de minimis 取消打擊 → 利潤往三層集中、無差異跨境傾銷被商品化

當電商滲透率進入 late phase（全球 ~17%→2030 ~25%），單純「賣貨」的 retail 段毛利 ~25% 是 commodity；賺得到的是握廣告閉環（看得到「曝光→購買」的 AMZN/MELI）、握 OS lock-in（商家轉換成本高的 SHOP）、握在地履約（跨境打不穿的 MELI/SE/CPNG）的人。AMZN 廣告 +24% 遠快於國際 retail +11%、SHOP Payments 占 GMV 升到 68%，是「利潤往三層集中」的硬證據；Temu 美國日活 -48% 是「跨境傾銷被商品化」的另一面。

可證偽條件：若 AI agentic shopping 改寫流量入口、讓 AMZN 廣告閉環失效，或跨境玩家在地履約規模化反擊（Temu/Shein 美國 GMV 超 2024 peak），則「三層集中」論部分失效。

PART IV · DEMAND SIDE

需求側：廣告閉環、OS lock-in與區域 leapfrog

需求最剛性的是「廣告主要可歸因 ROI + 商家要降摩擦的全套工具 + 新興市場消費者要可靠履約」——這把錢推向廣告閉環、OS、在地物流三層。

第一引擎：廣告閉環（現需）。電商平台自帶購買意圖訊號、看得到「曝光→購買」，這是廣告主最想要的可歸因 ROI。AMZN Q1 26 廣告 $17.2B（+24% YoY、超 Wall Street 預期 +21.2%）、trailing 過 $70B；MELI 廣告為三引擎之一、WMT Connect 為 #2 RMN。第二引擎：OS lock-in（現需）。商家要的是「一套工具搞定建站 + 支付 + B2B + 履約」——SHOP B2B GMV +96%、Payments $84B 占 GMV 68%（比去年 +4pp），商家越深用、轉換成本越高。第三引擎：區域 leapfrog（現需）。新興市場消費者跳過 PC 直接行動電商 + 要可靠履約——MELI rev +49%（巴西降免運門檻成長引擎）、Shopee GMV $37.3B +30% / 訂單 +29%、CPNG 台灣三位數成長。三引擎都把錢推向「握閉環/lock-in/履約」者。

多空為何都對

多頭測電商滲透率 + 三引擎需求（廣告 + OS + 區域 leapfrog），這幾乎沒爭議；空頭測誰被商品化（無差異跨境傾銷、被 Marketplace 擠的 niche、AI agentic shopping 對廣告閉環的長期威脅）。兩者可同時為真：電商需求是真增長（多頭對），但價值集中於握廣告/OS/物流者、傾銷與 niche 被擠（空頭對）。本報告把重心放在「誰捕獲價值、誰被商品化」。

PART V · VERDICT & SCENARIOS

裁決與三情境（24–36M）

「電商滲透率 + 三引擎需求是真增長」判 INTACT，「價值往廣告/OS/物流集中」判 INTACT，但「AI agentic shopping 是否改寫流量入口、稀釋 AMZN 廣告閉環」判 AT_RISK——故結構看「三層贏家 vs 跨境傾銷/niche 輸」、conviction 卡 mid-high。如硬要對應傳統供需框架：對「可歸因 ROI 的廣告版位 + 在地履約能力」的需求遠大於合格供給（要握第一方意圖資料 + 自建物流），等價於結構性短缺型，稀缺來自閉環 + 履約門檻而非貨架。

Base Case主觀機率 ~50%

PART VI · TAXONOMY

個股特徵與 conviction tier

本框架只輸出「特徵」與「分類」，買哪一檔由個股 DD 系統與 Pure MA 訊號決定。以下 tier 為產業推論的特徵歸類，提及公司僅為「該特徵的代表案例」，不附帶買賣建議。

CORE 🔴

AMZN（規模 + 廣告 alpha）

握規模 + 廣告閉環 + AWS——Q1 26 廣告 $17.2B（+24%、trailing 過 $70B）、國際 +11%、單量 +15%（pandemic 後最快）、AWS +28%（15 季最快）。特徵是「retail 帳面 + 高毛利廣告/雲三引擎」最高純度贏家；市場仍把它當 retail 估、廣告 alpha 未充分 priced。conviction：高（廣告利潤池移轉）
CORE 🔴

SHOP（OS for commerce）

握商家 lock-in + Payments + B2B——Q4 25 GMV $123.8B（+31%）、B2B GMV +96%（Q4 +84%）、Payments $84B（+38%、占 GMV 68%）、FY25 GMV $378.4B（+29.5%）。特徵是「從 SMB SaaS 升級企業級電商 OS」，fwd PE 高溢價反映雙引擎結構。conviction：高（OS + B2B 雙引擎）
CORE 🔴

MELI（區域 leapfrog 飛輪）

握 LATAM 在地物流 + Mercado Pago fintech——Q1 26 rev +49%（近 4 年最快）、GMV +42%、items sold +56%、信用組合 $14.6B（+87%）。特徵是「電商 + 物流 + fintech 三引擎飛輪」，在地履約是跨境打不穿的護城河（深度見姊妹 ID_LATAMEcommerce）。conviction：高 · 投資期壓利潤但份額擴張
EDGE 🟡 / LOSER 🟢

SE/WMT/CPNG/BABA/PDD ｜ ETSY（被擠 niche）

🟡 區域/挑戰者受益：SE（Shopee GMV +30% / Monee 飛輪 / EBITDA 首破 $1B）、WMT（Connect #2 RMN + Marketplace）、CPNG（Korea 龍頭 + 台灣三位數，台灣投資壓利潤、高 beta）、BABA（中國電商 +10% / 國際 +19% / 雲 +26%）、PDD/Temu（de minimis 後 pivot 半托管 + 廣告變現）。🟢 結構性壓力：ETSY（handmade niche 被 Marketplace 擠）。conviction：中（區域受益）/ 偏空（無差異跨境傾銷、被擠 niche）

估值面總結：本主題的核心張力在「電商需求真增長 vs 價值集中（傾銷/niche 被商品化、agentic shopping 長期威脅）」。三層贏家（AMZN 廣告 / SHOP OS / MELI 飛輪）是最高純度 play；SE/WMT/CPNG 是區域受益；結構性輸家是無差異跨境傾銷（Temu 舊模式）與被擠 niche（ETSY）。現處 Phase IV（三層加速），轉空條件 = AI agentic shopping 達 5%+ GMV 稀釋廣告閉環 或 跨境在地履約反擊讓 Temu 美國 GMV 回 2024 peak；de-rating 風險主要在 SHOP（高 PE 高 beta）與 AMZN（廣告增速若塌）。買賣由個股 DD + Pure MA 系統決定。

The One Line

全球電商不是「AMZN vs Temu」、是四層分化——價值流向握廣告閉環 / OS lock-in / 在地物流的 AMZN/SHOP/MELI，無差異跨境傾銷與被擠 niche 被商品化。

這份報告刻意不做的三件事

  1. 不用「AMZN vs Temu」單一框架——明確畫出四層分化：握廣告/OS/物流者贏、無差異傾銷與 niche 輸。
  2. 不把 AMZN 當無腦贏家——標出 AI agentic shopping（2027+）對廣告閉環的長期稀釋裂縫。
  3. 不告訴你哪一檔該買——CORE/EDGE/LOSER 是「特徵的代表案例」，買賣由你的個股 DD 與 Pure MA 系統決定。
