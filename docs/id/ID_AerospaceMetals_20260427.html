Unit Glossary：rev / EPS 採 fiscal year（多數 12 月）；superalloy = 高溫合金（鎳 / 鈷基），用於引擎熱端；premium titanium = 航太級鈦（Ti-6Al-4V 等）；Section 232 = 美國對鋁進口的國安關稅（Trump 1.0 + 2.0 強化）；Midwest Premium = 美國中西部鋁交付溢價；CBAM = 歐盟碳邊境調整機制；VSMPO-AVISMA = 俄羅斯主要鈦廠商（西方供應商分流主因）。

💡 3 條核心 insight（預告）
1. HWM + ATI + Carpenter 三角是航太結構件「toll booth」：90%+ 引擎結構件（HWM）+ 70% A&D mix（ATI）+ 33.1% op margin（Carpenter SAO）— 三家都是 2026 EBITDA / op margin 業界最高，因「航太特殊合金 + 鍛造」高進入門檻 + 多 OEM 認證 lock-in。
2. 鈦供應重組是 5-7 年結構性 catalyst：俄羅斯 VSMPO-AVISMA 受限後，Boeing / Airbus / DoD 被迫分流；ATI 是最大 pure-play 受益者（鈦 + Ni / Co 整合）；2026 ASP 已上修 + 2030 仍持續。
3. 美國 Section 232 鋁關稅 + Midwest Premium $560/tonne 是「再工業化」最直接 alpha：Constellium 2025 rev $8.4B（+15%）+ Ravenswood WV 廠擴產；Kaiser / Alcoa 美國本土廠享 25-30% 政策溢價；對抗中國 / 加拿大鋁的結構性優勢。

§1 產業定義與邊界

本 ID 涵蓋三大子市場：① 結構件 / 鍛造件（HWM 主導）；② 特殊合金（鈦 / 鎳 / 鈷 / 超合金）；③ 初級金屬（鋁原料 / 板材）。三者在航太 / 國防 / EV / 能源轉型四個下游需求共振下進入結構性成長週期。

💡 Insight §1
1. 「三層 toll booth」結構是本 ID 核心差異化：結構件（HWM）+ 特殊合金（ATI / Carpenter）+ 鋁板材（Constellium / Kaiser）三層共同受益航太 ramp + 國防擴產，但毛利結構不同（HWM 30% EBITDA > 特殊合金 25%+ > 鋁板材 12-15%）。

3. 稀土 / 銅是另類別：本 ID 焦點是傳統金屬 + 特殊合金；稀土 / 銅在其他 ID 涵蓋。

💡 Insight §2
1. 本次商品週期不像 2003-2008 — 焦點是「特殊合金」而非大宗：鈦 / 鎳 / 超合金需求 driven by 航太 ramp + 國防 + EV，不是中國基建。
2. 美國本土再工業化是「reverse 1990-2010」：Section 232 + Midwest Premium 重建被荒廢的鋁業；Constellium / Kaiser / Alcoa 是直接受益。

Carpenter Technology（CRS）— specialty alloy 黃金期

CRS 是premium specialty alloy玩家，過去 5 年從「傳統 steelmaker」轉型為「高毛利 specialty alloy」。核心 metrics：

  • 市值 ~$20B（已大幅 re-rate）
  • SAO（Specialty Alloys Operations）adjusted op margin 33.1%（16 連季增加） — 業界最高
  • 產品：premium 鈦 / 鎳 / 鈷 / 增材製造（AM）合金
  • 核心客戶：Boeing / Airbus / GE Aerospace / Pratt & Whitney 引擎熱端零件
  • 2026 焦點：「Margin Maximization」+「Backlog Quality」

解讀：CRS 是「引擎熱端 toll booth」 — Ni / Co 超合金用於 turbine blade（最熱、最關鍵的零件）；33.1% op margin 反映「特殊合金 + 多 OEM 認證 + 高進入門檻」三重護城河。增材製造（AM）合金是下一波 niche。
