§1 產業白話定義與歷史脈絡

玻璃核心載板，白話講就是「把晶片底下那塊承載 + 連線的基板，從有機塑料（ABF）換成玻璃」。誰付錢？做大晶片（AI GPU / 高階 server CPU）的人——因為晶片越大越熱（破 1,000W），塑料底板會翹，玻璃不會。為什麼現在重要？因為 AI 把封裝越做越大，撞上了塑料的物理上限。

先把邊界界清楚。本 ID 聚焦的是「玻璃作為結構層（core）或插入層（interposer）」這個 next-gen substrate 路線，跟單純「把玻璃當 EUV 光罩」（屬 foundry 設備）或「顯示面板玻璃」（LCD/OLED）完全是兩回事。灰色地帶有兩處：① glass-core CPU vs glass interposer——Intel Clearwater Forest 是「玻璃當 core」（取代矽底板 + 有機載板），AMD MI400 是「玻璃當 interposer」（取代矽 CoWoS-S 中介層）；兩者市場獨立，本 ID 同時涵蓋。② 玻璃 vs ABF vs 矽中介層——ABF（有機載板）是玻璃在「載板層」的競爭者（屬母題 AP 範疇），矽中介層（CoWoS-S）是玻璃在「interposer 層」的競爭者。本 ID 視玻璃為「在這兩層都來搶份額、但目前都還是少數派」的挑戰者。

三個歷史轉折點（每個帶日期 + 量化錨點）

轉折一：2017 LPKF 發表 LIDE。LPKF 在 2017 年商業化 LIDE（Laser-Induced Deep Etching，雷射誘導深蝕刻）——這是用超快雷射在玻璃裡「點」出改質區、再用濕蝕刻把改質區溶掉，做出零微裂、光滑、高深寬比的 through-glass via（TGV，玻璃貫通孔）。沒有 defect-free TGV，玻璃就只是一片玻璃、不能當電氣連接的載板。LIDE 是把玻璃從材料變成「可封裝基板」的鑰匙，核心 LIDE 專利族自 2017 商業化起算、保護期估延伸至 2030 年代中後段（約 2035+，依各國 filing 不同），LPKF 並已對中國明顯侵權案採法律行動；目前 >80% 主要玩家選用 LPKF 設備。

轉折二：2020-2023 商業化路線開啟。2020 SKC 成立 Absolics 子公司專攻 glass substrate，2023 Intel Labs 展示原型、Samsung 宣布 R&D。2024-08 Intel 發佈「Build the Future with Intel Foundry and Glass-Core Substrates」白皮書，把玻璃寫進 foundry 產品路線；2024-11 Absolics 拿到美國 CHIPS Act $100M R&D 補助，確立 Covington（喬治亞州）建廠。這階段玻璃從「實驗室 demo」走向「有資本承諾的量產路線」。

轉折三：2026-01 Intel Clearwater Forest 量產 + TSMC SoW-X 揭示。CES 2026 上 Intel 推出 Xeon 6+「Clearwater Forest」——全球第一款量產的 glass-core CPU，整合 glass core + Foveros Direct 3D + EMIB，採 18A 製程、288 core / 12 compute chiplets。但同期 TSMC 在 2026 北美技術論壇揭示 SoW-X（40-reticle、可容 64 顆 HBM、2029 量產）——這條訊息對玻璃陣營是雙面刃：一面確認「大封裝是趨勢」（利好玻璃方向），一面證明「substrate / wafer-scale 路線扛得到 64xHBM」（利空玻璃的「2028 唯一解」急迫性）。這是本次 v2.3 相對 legacy v1.0 最關鍵的事實更新。

怎麼讀：兩次基板換代（HDI、CoWoS-S）滲透到 40% 都花了 5-7 年；玻璃 2026 才 pilot、佔先進封裝 <5%，按同樣節奏推到 30-35% 落在 2031-2033，恰好對上 TSMC 把玻璃 mainstream 推到 2030+。市場現在把玻璃當「2026-2028 引爆」，歷史上基板換代從不引爆、只慢牛。

這段歷史對今天的含義：玻璃方向（翹曲牆是物理硬約束）不可逆，但滲透節奏是「以年計的慢牛」，不是引爆。legacy v1.0 把玻璃講成「2026-2028 陡斜率 + SAM CAGR 57%」，犯的正是「把慢牛當引爆」的錯——本次按 HDI/CoWoS 滲透時滯與市場研究機構務實 CAGR（~8.5-13.5%）下修，方向不變但時點後移。

§2 技術成熟度 + S 曲線

為什麼是現在、不是三年前？因為三件事剛好同時就位：① AI 大晶片功耗破 1,000W 撞上「翹曲牆」（需求拉力）、② LIDE 把 defect-free TGV 做到可量產（技術解鎖）、③ Intel 第一個敢把它放進量產 CPU（商業驗證）。為什麼不是三年後？因為翹曲牆已經是「現在進行式」的物理約束，不是未來威脅。

技術棧 kingmaker 判斷

玻璃載板有 7 個關鍵子技術（材料配方 / TGV 鑽孔 / TGV 金屬化 / RDL 線路 / chip-to-glass bonding / warpage 控制 / panel handling），但 kingmaker 只有兩個：TGV 鑽孔（LIDE）與低 CTE 玻璃配方。前者沒有同等商業化替代（自研落後 5+ 年）、後者由 Corning/AGC/SCHOTT >90% 寡占（熔爐建置 12-18 個月＝高壁壘）。其餘子技術（金屬化、bonding、handling）都是「多家可做、可切換」的非鎖喉環節。

玻璃載板的供給故事不是「誰擴產快」，而是「哪條下游路線勝出」——而無論哪條贏，利潤都往兩個路線無關的鎖喉點集中：TGV 雷射（LPKF）與低 CTE 玻璃（Corning/AGC/SCHOTT）。

下游供給端是四條並行路線，沒有單一贏家：

  • Intel（垂直整合 glass-core）：自家 glass-core CPU，Clearwater Forest 2026 H1 量產，整合 Foveros + EMIB + glass core；glass 佔其 Xeon DC 業務小比例且上升中。優勢是「自有 foundry + 整合 know-how」，劣勢是 glass-core ramp 受 18A yield 牽制（18A yield inflection 在 2026 H1、mature 2027）。
  • Absolics / SKC（merchant interposer）：第一家 merchant glass substrate 廠，Covington 12K m²/yr、第二廠 72K m²/yr 規劃，SKC 注 ₩1.17T 增資；AMD/微軟鎖 60% Phase-1 產能（2027 Venice ramp），Applied Materials + CHIPS Act 支持。優勢是「first-mover + 第三方供應」，劣勢是賭單一路線、執行 + 時程雙風險、毛利 2027 才打平。
  • TSMC CoPoS（玻璃面板封裝）：Chip-on-Panel-on-Substrate，玻璃面板 310×310mm → 515×510mm → 750×620mm，與 Ibiden / Innolux 共驗（2026-06）；pilot 2026、mass prod 2028末-2029。優勢是「TSMC 客戶體系 + panel 成本降 30% / wafer 利用率破 90%」，劣勢是 C.C. Wei 明說「沒有捷徑、至少 2-3 年」、時程最保守。
  • Samsung（Foundry 自代 + SEMCO 載板）：Foundry 自代 glass + Samsung Electro-Mechanics 載板雙線；SEMCO 2027 ramp（與 Sumitomo 合資玻璃 core 材料）、LG Innotek 2028 量產 / 2030 需求。swing player。

供給瓶頸是雙重節流：① 材料端——玻璃熔融爐建置要 12-18 個月，若 2026 下游需求爆發，上游低 CTE 玻璃產能會出現結構缺口；② 製造端——大面積 panel-level 玻璃的 warpage / 破損率仍是良率卡點（薄玻璃 <0.5mm 產線破損 3-5%、panel-level R&D 階段良率 <40%）。這兩道節流讓玻璃「蓋不快」，是玻璃慢牛的物理基礎。

§4 需求側：大晶片驅動是真的，必走時程被推遲也是真的

ASP 動態 + 估值傳導

玻璃載板單片 ASP 走「學習曲線下滑、但每片內嵌價值（TGV 數量）上升」：~$80/片（2023 R&D）→ ~$35/片（2026 量產）→ ~$25/片（2028 規模化）→ ~$18/片（2030 成熟）。ASP 5 年跌 ~55%，但每片 TGV 數量增 3-4x、單封裝面積擴大，total content per package 上升。估值傳導：產業變數（CoPoS 時程、玻璃滲透率）→ 公司 Fwd PE 怎麼動——對 pure-play（LPKF / SKC）是「時程提前 = re-rate、推遲 = de-rate」的高 beta 傳導；對鎖喉點材料商（Corning / AGC）因 glass 佔比小，傳導被稀釋（+1pp 玻璃滲透 ⇒ 倍數擴張有限）；對 Intel 因 glass 是 incremental，傳導幾乎不動估值主軸（18A 才是）。

估值現在 price 了什麼、哪個變數動會 re-rate / de-rating：玻璃 pure-play 的估值錨在「2028 大爆發」，而 TSMC SoW-X 把時程推到 2030+ 後，這個前提需下修 → pure-play 面臨 de-rating 壓力。最會觸發 re-rate 的單一變數是「CoPoS 時程提前」（2027 Q1 Symposium 揭曉）；最會觸發 de-rating 的是「CoPoS 再推遲 + hybrid bonding 繞道」。鎖喉點材料商因 glass 佔比小，估值彈性有限（防守但無爆發）。

§7 Non-Consensus + Priced-in 檢驗 + Kill Scenario

市場主流怎麼看？「玻璃會取代 ABF、Intel 是首發贏家、玻璃 2028 大爆發」。我們哪裡不同？我們認為玻璃方向對但時程被 TSMC 自己推遲、賺錢點不在賭路線的純 play 而在鎖喉點。

🟡 非共識 1：賺錢點不在載板 pure-play，在路線無關的鎖喉點 [F + I] 信心：中-高
  • 共識：賣方 / 媒體多把玻璃故事押在「第一家量產的 merchant 載板（Absolics）」與「首發整合者（Intel）」——把它當下一個 CoWoS pure-play 賭注。
  • 本 ID 看法：[I: 四路線並行（Intel / Absolics / TSMC / Samsung）→ 獨佔溢價被稀釋 → 賭單一路線承擔「賭錯路線」風險] 而 TGV 雷射（LPKF >80% 選用）與低 CTE 玻璃（Corning/AGC/SCHOTT >90%）是四路線都繞不過的鎖喉點 → 利潤往這兩點集中。[F: §3 利潤池表]
  • priced-in（含估值錨）：LPKF 為小市值（~€300-500M 量級）、賣方覆蓋少；2025 營收 €115M（-6% YoY）、2026 guide adj EBIT 仍 -3%~+4.5%（近損益兩平），市場估值仍把它當「景氣循環的小型雷射設備商」、尚未 price 進「玻璃世代每顆晶片收 TGV 授權」的選擇權價值——其 glass 段營收占比目前個位數、估值對「玻璃 ramp」的隱含成長假設接近零（這正是「未 priced」的證據）。Corning/AGC 因 glass 佔全公司比例小（Corning ~8% / AGC semi <5%），估值幾乎沒 price glass 增量。[A: LPKF 市值與覆蓋度為公開區間估計，缺逐筆券商 EV/Rev band，故信心由「高」下修為「中-高」]
→ 投資意義：押鎖喉點 convexity（LPKF）+ 防守 proxy（Corning/AGC），而非賭路線的高 beta pure-play；「可操作」前提是「估值尚未 price 玻璃選擇權」，若後續 LPKF 因玻璃題材 re-rate 到充分 price，可操作性下降。⚠ 證偽：① 出現商業化的 LIDE 替代 TGV 或自研低 CTE 玻璃打破寡占；② LPKF 估值已 re-rate 到 price 進玻璃 ramp（此時分歧「對但已 priced」、不可操作）。
🟡 非共識 2：玻璃 mainstream 時程被 TSMC SoW-X 推到 2030+（非 2028） [F + X] 信心：高
  • 共識：legacy / 早期賣方多把玻璃當「2028 大封裝唯一解」、押 2026-2028 引爆。
  • 本 ID 看法：[F: TSMC 2026 NA Symposium 揭 SoW-X 40-reticle / 64xHBM / 2029] 證明 substrate / wafer-scale 路線扛得到 64xHBM，「玻璃 2028 唯一解」前提被打掉；CoPoS 自己 mass prod 滑到 2028末-2029、C.C. Wei「沒有捷徑」。[X: base 很可能 — 玻璃 mainstream 2030+；bull 可能 — 翹曲牆提前逼玻璃 2027 頂上；bear 可能 — 再推遲 2032+]
  • priced-in：玻璃 pure-play（SKC / Absolics）估值仍部分 price「2028 大爆發」前提 → 此分歧「對且尚未完全 priced」（市場還沒充分 price 進 -2 年時程折損）、可操作（de-rating 尚未走完）。
→ 投資意義：pure-play 現價非無腦進場點、等時程確認；這也是 legacy「Intel 1-2 年獨佔 + 玻璃 2028 大爆發」需降級的核心。⚠ 證偽：CoPoS / Absolics 提前 2027 量產。
🟡 非共識 3：玻璃與 ABF 是共存而非取代（2026-2030） [F + I] 信心：中-高
  • 共識：「玻璃殺 ABF」的取代敘事。
  • 本 ID 看法：[F: Ajinomoto 控 >95% ABF 核心料、Q3 2026 漲價 30%、Ibiden 2026-28 投 ¥500B 擴產仍供不應求] → [I: ABF 是瓶頸 + 仍主力供給 → 玻璃吃的是 ABF 扛不住的大封裝「增量」而非「存量」] 在 2026-2030 窗口共存。
  • priced-in：ABF 載板廠（Ibiden / Unimicron）估值部分被「玻璃取代」恐懼壓抑 → 若市場認知到共存（ABF 還會 +30% 漲價、擴產供不應求），ABF 廠的 de-rating 反而是被錯殺、分歧「對且 mis-priced」。
→ 投資意義：把玻璃當「ABF 替代」算 TAM 會高估玻璃、低估 ABF；ABF 廠（Unimicron）是共存路線的被低估受惠者。⚠ 證偽：ABF 供給崩斷（Ajinomoto 斷料）逼玻璃提前全面頂上。

Kill Scenario / Steel-man 反方（3 條完整論證）

反方 1：TSMC SoW-X / substrate scaling 證明玻璃根本「不必走」、時程再推遲到 2032+。具體路徑：TSMC 把 CoWoS interposer 路線圖一路扛（2026 5.5x → 2028 14x → 2029 SoW-X >40-reticle / 64xHBM），substrate scaling 證明可靠度高（5.5-reticle CoWoS 2026 yield 已破 98%），CoPoS 改稱「並行選項」而非「必走」、mass prod 再滑到 2030。當前證據：TSMC 2026 NA Symposium roadmap、C.C. Wei「至少 2-3 年」。成真窗口：2027 Q4 TSMC 公告 SoW-X 客戶名單 + CoPoS 再推遲。這是對玻璃 thesis 殺傷最大的反方——不是「玻璃失敗」而是「玻璃時程被推遲 2 年」，對 LPKF / SKC 是直接 -2 年 IRR 折損。

反方 2：Hybrid bonding / SoIC 直接 die-to-die，跳過玻璃 interposer 層。具體路徑：BESI + TSMC + AMAT 加速 hybrid bonding，Apple / TSMC / Samsung 的 SoIC 產線進入全 3D 堆疊，部分大晶片不需玻璃 interposer。當前證據：BESI hybrid bonder 產能擴張、AMAT 持 BESI 股份、多 hyperscaler qualifying hybrid bonding。成真窗口：2028 代際轉換決定路線；trigger metric：BESI 2027 hybrid bonder 出貨 >1,500 units。

反方 3：Intel 18A 拖累 Clearwater Forest，glass-core 首發優勢縮水。具體路徑：18A yield 2026 H2 仍不足、Clearwater Forest ramp 受製程拖累、外部 foundry 客戶決策延後（H2 2026 → H1 2027），首發 glass-core 的「demo 價值」無法轉成 foundry thesis 翻轉。當前證據：18A yield inflection 排在 2026 H1、mature 要 2027，時程已多次滑動。成真窗口：2026 Q4 財報 → 2027 Q1 yield 更新。

這三條分歧裡哪條最可操作（對 + 未 priced）：非共識 1（賺錢點在鎖喉點）最可操作——它「對且未充分 priced」（LPKF 估值仍當它小型雷射公司、材料商 glass 增量沒被 price），且不依賴賭哪條下游路線贏（路線無關）。非共識 2（時程推遲）方向對但 de-rating 部分已啟動、操作空間在「等走完」；非共識 3（共存）對但 ABF 廠的 re-rate 需要市場認知翻轉、時滯較長。
