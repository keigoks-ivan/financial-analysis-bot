線上旅遊 OTA × AI 分銷革命
真正的威脅在上游，不在 ChatGPT

市場用一個錯誤的恐慌在估 OTA：「ChatGPT 會直接幫你訂房、繞過 Booking」。2026 年的事實把這個恐慌按了暫停鍵——OpenAI 在 3 月悄悄撤掉了 ChatGPT 內的旅遊 instant checkout，理由是「旅遊太複雜」：用戶很樂意問 AI 點子，但要輸入信用卡時，他們離開、去他們信任的地方訂。願意讓 AI agent 代為完成購買的旅客只有 ~2–8%。Skift 的標題直白：對 Expedia 與 Booking 是好消息。

真正的 AI 威脅在上游，不在下游。Google AI Overview / AI Mode 把搜尋結果頁變成「答案頁」——travel 類 informational query 的 zero-click 升到 65–70%、organic session 掉 20–40%，OTA 的免費 SEO 漏斗被結構性 throttle。這逼 OTA 加大付費行銷（壓 margin）、加大品牌直訪比例（要時間）、並把自己整合進 AI 平台當 inventory 層（短期 hedge）。BKNG 三條路都已 commit，EXPE 落後 12–18 個月。

所以本報告不站「OTA 會不會被 AI 取代」的二元隊。OTA 不會被取代、反而是 agent 的水電瓦斯（inventory + payment + 信任）；真正的分歧在雙寡占內部的永久分化——BKNG 規模 + AI-native + 拆股後估值，與 EXPE 結構落後但深度折讓（fwd P/E ~10x），是兩種完全不同的投資命題。判斷重心放在「誰在 AI 過渡期擴大領先、誰只是便宜」。

PART I · DECISION SUMMARY

決策摘要層

三句話看完 OTA × AI 分銷的現狀、走向與行動，後接 PM 級監測點。這一層是報告的全部重點——下面各節只是把這一層的每個判斷補上證據。

現在 / NOW

OTA 雙寡占內部結構性分化——BKNG Q1'26 營收 +16%／room nights +6%／take rate 10.3%，EXPE GBV +13%（B2B bookings +22%）。AI 雙刃中，下游 agentic 直接訂房短期退潮（OpenAI 2026-03 撤掉 ChatGPT 結帳、僅 ~2–8% 旅客願讓 AI 代付）；上游 Google AI Overview 壓流量才是真威脅。

未來 / NEXT

未來 3–5 年 AI 把 OTA 從「搜尋入口生意」推向「inventory + payment + 信任層生意」；上游 SEO 漏斗結構性 throttle（OTA 加大付費行銷、壓 margin），但 OTA 變成 agent 的水電瓦斯而非被取代。BKNG agentic Trip Planner 領先 EXPE 12–18 個月，估值兩極化。

行動 / ACTION

核心續抱 BKNG（規模 + AI 整合 + 拆股後 fwd P/E ~16x）；EXPE 是 deep-value 折讓票（fwd P/E ~10x、B2B +22% 是唯一抗 AI 段、$5B 新買回）、非 quality long；TCOM 中國 outbound 地緣 alpha 但 Q1'26 待 6/24；TRIP 體質轉差（Q1 虧損、Viator 已併回）退出。買賣由個股 DD + 系統決定。

The Key Call · 本報告最重要的一個判斷

市場把「AI 顛覆 OTA」想成「ChatGPT 直接幫你訂房、跳過 Booking」，這是看錯了方向的恐慌。真正的非共識是——下游 agentic 直接訂房在 2026 已被現實打回（OpenAI 撤掉 ChatGPT 結帳、僅 ~2–8% 旅客願讓 AI 代付，因為旅遊牽涉信任、支付、改退、客服等高 friction），而上游 Google AI Overview 的搜尋 throttle 才是結構性、且已在發生（travel zero-click 65–70%、organic -20–40%）。換句話說，OTA 不會被 AI agent 取代、反而是 agent 的 inventory + payment + 信任層；真正被 AI 改寫的是「免費 SEO 流量」這條上游漏斗。分歧不在「OTA 會不會死」（不會），而在「雙寡占內部誰在 AI 過渡期擴大領先」——BKNG 已 AI-native，EXPE 落後但深度折讓。

三道裂縫具體是：①上游 SEO throttle 是結構性且已發生——Google AI Overview 把 travel 類查詢的 zero-click 推到 65–70%、organic session 掉 20–40%，OTA 的免費流量漏斗永久變窄，必須用更貴的付費行銷補（margin 壓力）；②BKNG 與 EXPE 永久分化、非雙寡占共贏——BKNG room nights +6%／take rate 10.3%／agentic Trip Planner 領先，EXPE 靠 B2B（+22% bookings、不依賴 SEO）撐結構性折讓，兩家不是強弱二元而是兩種命題；③agentic 退潮是 timing 而非永久——OpenAI 撤結帳、Sabre+PayPal+MindTrip、RateGain MCP booking 引擎仍在推；2027–2028 若信任 + 支付 friction 被解，下游威脅會回來。盯這三條，比盯任何單季 GBV beat 都更早看到 thesis 轉折。

PART II · HISTORICAL ANALOGS

兩個歷史類比，框住 AI 衝擊的形狀

歷史不必重演，但兩個對照各給一個可操作教訓：2010 年代 mobile 革命框住「大 OTA 能遷移、小 OTA 被淘汰」，2003 年 Travelocity 的隕落框住「dominance 不是 unbeatable」。

mobile 革命（2010–2015）——大 OTA 透過遷移保留份額，落後者被淘汰。2010 年 mobile 起飛時，市場恐慌「OTA 的 desktop 模式會被 mobile-first apps 取代」。實際結果：BKNG / EXPE 成功遷移到 mobile、市占擴大，被淘汰的是沒能轉型的小 OTA（Travelocity、Orbitz 被邊緣化）。量化對照：2010–2015 BKNG GBV CAGR ~+27%、EXPE ~+18%——mobile 不但沒壓制 OTA growth，反而讓有規模、能投入工程的大 OTA 拉開差距。本次 AI 分銷的含義是同構的：上游 throttle + 下游 agent 都是「載體再轉移」，能 AI-native 整合的大 OTA（BKNG ChatGPT app + agentic Trip Planner）保留並擴大份額，落後者（EXPE 慢 12–18 個月）+ 小 OTA / 區域玩家失去份額。差別只在這次的「遷移成本」更高——要重建整個 funnel 為 AI-native，而非只是做個 app。

怎麼讀：四項顯示「本次大 OTA 仍能透過整合保留份額」，差別在衝擊面更廣（上游已發生）、遷移成本更高（要重建 funnel）。空頭若純押「AI 殺死所有 OTA」會像當年純押「mobile 殺死 OTA」一樣過早；要盯的是「誰能 AI-native 遷移」，而非「OTA 會不會死」。

Travelocity / Sabre 的隕落（2000–2003）——dominance 不是 unbeatable，但要看條件。當年 Travelocity（Sabre 子公司）是美國 #1 OTA，市佔極高；隕落的原因不是「被新技術取代」，而是無法擴張國際 + 無法整合多 supplier + 與 GDS 衝突，2003 後被 Expedia 與 Booking.com 永久超越。反類比警示：BKNG 今天看似 unbeatable，但 Travelocity 的三個致命傷對照 BKNG 是——(a) 100% 美國依賴：BKNG 全球分散（歐洲為基本盤）；(b) 無 alt acc：BKNG 已有大規模 alt acc + Connected Trip 跨類別；(c) 無 AI 整合：BKNG 已 commit agentic Trip Planner + ChatGPT app。三個致命傷 BKNG 都已對沖，所以 Travelocity 是 cautionary tale 而非直接適用。真正能讓 BKNG 重演 Travelocity 命運的唯一路徑，是Google 上游 throttle 永久放大 + Google 自己認真做 hotel booking funnel（Google Hotels meta-search 已存在）——這是 BKNG 投資人最該追蹤的長期反方訊號，比 ChatGPT 更危險，因為 Google 仍佔絕大多數 travel research 入口。

PART III · SUPPLY SIDE

供給側：inventory + payment + 信任的準入門檻

反直覺但可操作的結論：OTA 沒有實體產能的成本曲線，競爭不由邊際成本決定，而由「supplier inventory 廣度 + payment 基建 + 消費者信任」三道準入門檻決定——這正是 AI agent 必須跟 OTA 合作、而非取代 OTA 的根本原因。

供給端的結構是「2 家全球全平台（BKNG / EXPE）+ 1 家中國巨頭（TCOM）+ 一票被擠到邊緣的區域 / 單一品類玩家」。一個 2026 年合格的全球 OTA 至少要同時握有三樣東西：(1) 夠廣的 supplier inventory（旅館 + 機票 + alt acc + 體驗，且是直連而非轉賣）；(2) 自有 payment 基建（跨幣別、防詐、改退）；(3) 消費者信任（品牌 + 客服 + 爭議處理）。能三樣都湊齊的，全球只有 BKNG、EXPE，TCOM 在中國 + outbound 那塊最強並往外延伸。其餘玩家各據一角：ABNB 守 STR、TRIP 守 Experiences、區域 OTA（MMYT / DESP / SEERA）守單一市場。利潤與成長因此明顯往雙寡占集中。

Inference · 供給
三道準入門檻把 AI agent 鎖在「合作」而非「取代」
前提：OTA 無實體產能成本曲線（準入由 inventory 廣度 + payment 基建 + 信任決定）+ AI agent 無自有 inventory / payment / 客服 + 旅遊是高 friction 交易（信任、改退、爭議）→ agent 必須接 OTA 的 inventory layer

當 AI agent（ChatGPT / Perplexity / Copilot）想幫用戶訂房，它沒有自己的 supplier 直連、沒有 payment rail、沒有客服處理改退爭議——這三樣全在 OTA 手上。所以 agent 的理性選擇是「接 OTA 的 inventory API」（Booking.com / Expedia 都已是 ChatGPT app；Perplexity 的 instant checkout 由 Selfbook + Tripadvisor 後台處理）。這不是 zero-sum，而是 vertical（OTA）vs horizontal（agent）的新分工。OpenAI 2026-03 撤掉自己做結帳，正是承認「旅遊太複雜、繞不過信任層」——這是供給門檻 binding 的直接證據。

可證偽條件：若出現「AI agent 直連 supplier（繞過 OTA）+ 自建 payment/客服」並贏回大量交易（如某 agent 直接整合 Sabre/Amadeus GDS 的 2M+ 旅館庫存 + 自有支付且爭議處理可信），或主要 agent 重啟 instant checkout 且滲透率破 15%，則「OTA 是不可繞過的 inventory 層」論失效、OTA 估值重定價。

PART IV · DEMAND SIDE

需求側：搜尋 throttle × agentic 退潮的雙引擎

需求最不可替代的是「旅遊是高頻、高金額、高信任的剛性消費」——旅遊支出在景氣下行時有韌性；最改寫分銷的是 AI，但 2026 它對需求是「上游 throttle 已發生、下游 agentic 退潮」的非對稱雙引擎。

分銷的變化由兩個 AI 引擎共同推動，但方向相反。第一是上游搜尋 throttle（已發生）：Google AI Overview / AI Mode 把 travel 類 informational query 變成「答案頁」，zero-click 升到 65–70%、organic session 掉 20–40%——OTA 的免費 SEO 漏斗結構性變窄，被引用進 AI Overview 的品牌反而多拿 35% 點擊（贏家更集中）。第二是下游 agentic 退潮（2026 反轉）：OpenAI 3 月撤掉 ChatGPT 旅遊 instant checkout，因為「旅遊太複雜」、用戶在輸入信用卡時離開；願意讓 AI agent 代為完成購買的旅客只有 ~2–8%。兩個引擎合起來，把分銷推向「OTA 仍是訂房終點 + 信任層，但上游入口被 AI throttle」——OTA 必須付更多錢買流量、或把自己做成 AI 平台的內建選項。

Inference · 需求
旅遊需求結構性剛韌——但分銷的「免費流量」被 AI 永久重訂
前提：旅遊是高頻高金額剛性消費（experience economy 結構成長）+ 上游 Google throttle 已發生（zero-click 65–70%）+ 下游 agentic 退潮（代付意願 ~2–8%）→ 需求的「量」穩、但分銷的「流量成本」上升

旅遊需求本身很穩——疫後 experience > goods 的消費結構移轉 + 中國 outbound 仍在回補（TCOM outbound 回 2019 的 >120%）。真正被 AI 改寫的不是「量」，是「OTA 怎麼拿到這些量」：過去靠免費 SEO，現在 Google AI Overview 把這條漏斗 throttle，OTA 必須加大付費行銷（壓 margin）或加大品牌直訪（要時間）。下游 agentic 退潮給了 OTA 喘息——直接訂房威脅 2026 沒兌現——但上游壓力是持續的。

可證偽條件：① BKNG/EXPE 行銷費 / GBV 比連兩季顯著上升而 GBV 增速未加速（Google throttle 正在吃 margin）；② AI agent 代付意願回升破 15% 或主要 agent 重啟 instant checkout 且滲透率竄升（下游威脅回來，需求遷移開始）。

多空為何都對

多頭測的是需求的量與 OTA 的不可繞過性（旅遊剛韌、OTA 是 agent 的 inventory 層、雙寡占集中），這在 2026 數據上成立、很可能持續；空頭測的是分銷成本與下游 agent 的長期威脅（Google throttle 壓 margin、2027+ agentic 信任 friction 若被解則繞過 OTA）。兩者可同時為真：OTA 確實是 agent 的水電（多頭對），但 Google 把上游流量變貴、且下游威脅只是延後而非消失（空頭對）。所以本報告把判斷重心放在「雙寡占內部誰在 AI 過渡期擴大領先」，而非「OTA 會不會被取代」。

估值面總結：本主題的核心張力在「OTA 不會被 AI 取代、但 AI 把上游流量變貴 + 雙寡占內部永久分化」。BKNG（拆股後 fwd P/E ~16x）是 AI-native 領先 + 規模 moat 的品質代表，若 bull case（OTA 成為 agent 水電）成立有 re-rate 空間；EXPE（fwd P/E ~10x）是深度折讓票，估值便宜有便宜的理由（B2C 結構落後），B2B +22% 是折讓修正的唯一 catalyst。現處 Phase IV→V（AI 分銷過渡），轉空的條件 = Google throttle 失控壓 margin 或 2027+ agentic 直接訂房回歸繞過 OTA；TRIP / 小 OTA 是結構承壓端；GOOGL 對 OTA 是「上游 throttle」雙刃（既是最大威脅、也是 OTA 最大廣告買家），非 OTA play。

The One Line

AI 沒有殺死 OTA——它繞不過OTA 的 inventory、payment 與信任層；真正被 AI 改寫的是「免費搜尋流量」這條上游漏斗，而問題不是「OTA 會不會死」，而是「雙寡占內部誰在過渡期擴大領先」。

這份報告刻意不做的三件事

  1. 不告訴你哪一檔該買——CORE/EDGE 是「特徵的代表案例」，買賣由你的個股 DD 與 Pure MA 系統決定。BKNG「最佳平台」不等於「最佳進場價」，EXPE「最便宜」不等於「該買」。
  2. 不預測股價走勢——只給「進入哪個情境」的觸發訊號（room nights、行銷費率、AI 代付意願、EXPE B2B），不給點位。
  3. 不假裝知道下游威脅永遠不回來——agentic 直接訂房 2026 退潮是 timing 而非永久，2027–2028 信任 friction 若被 MCP/Sabre+PayPal 解開，威脅會回來，已誠實標為長期 bear 觀察。
