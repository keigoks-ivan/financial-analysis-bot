💡 §0 5 條核心 insight（預告）
1. AI Overview 60% 滲透是 OTA 真正威脅，但對 transactional booking 影響有限（1% vs 27% informational）：travel SEO 真正受傷的是「destination research / blog / inspiration」段，BKNG / EXPE 的「booking intent」query 仍受保護；但 inspiration → booking 的漏斗變窄，OTA 必須加大付費廣告投入（margin 壓縮 catalyst）。
2. BKNG 25-for-1 stock split 不是流動性事件，是 retail-friendly + AI 整合 + Connected Trip 完整化的市場訊號：management 對未來 3 年 confidence 的 explicit signal；split + buyback 加大 + AI capex 全面 commit。
3. EXPE 2026 guide rev +6-9% 與 BKNG Q4 +16% 對比顯示「雙寡占共贏」框架失效：BKNG 把 EXPE 視為 legacy distribution，自己已升級為 AI-native platform；EXPE B2B +24% 是唯一抗衝擊段，但結構性無法翻轉劣勢。
4. TRIP Experiences 50% 占比是 Viator spin-off 的 setup：Viator standalone GBV $4.7B + 18% growth = comp 應 $3-4B 估值（10x ARR）；TRIP 整體 $2B 市值 implied Viator < $1B 是顯著低估；2026-2027 spin-off catalyst 機率 40-60%。
5. Skyscanner（BKNG 子公司）是 sector 最被低估的 agent infra 玩家：OpenAI Operator launch partner（Jan 2025）+ MS Copilot Actions + ChatGPT app + 自身 SEO 較弱所以反而 less exposed AI Overview；BKNG 把 Skyscanner 當作 agent-era 的 hidden champion。

§1 產業定義與邊界

線上旅遊 OTA 在本 ID 的範圍是從消費者搜尋 → 行程規劃 → 訂房 / 機票 / 體驗 → 支付 → 後勤服務的全 distribution layer，加上 2024-2026 啟動的 AI 分發革命（Google AI Overview / ChatGPT / Perplexity / Kayak AI Mode 等）對 OTA 上下游的衝擊。本 ID 與 ID_HotelChains 互鎖：連鎖旅館 ID 看的是 supplier 端（chain 怎麼透過 brand + loyalty 對抗 OTA），本 ID 看的是 distribution 端（OTA 怎麼整合 supplier inventory + 怎麼被 AI 分發革命）。

邊界判斷的關鍵是「是否賣 inventory（vs 賣 brand）」與「是否擁有完整訂房漏斗（vs 上游 referral）」兩條軸：傳統 OTA（BKNG / EXPE / TCOM）兩條都符合；TripAdvisor 是 hybrid（review + Viator booking），列入但獨立分析；Airbnb 是 marketplace（既是 platform 也是 brand），列入但屬替代品 / 競品；meta-search（Kayak / Trivago / Skyscanner / Wego）是 OTA 的 funnel partner，列入但定位為 OTA 子集；AI 平台（OpenAI / Perplexity / Google Gemini / MS Copilot）是新的 distribution layer，本 ID 重點分析其對 OTA 的衝擊與整合機制。

💡 §1 Insight
1. 「OTA vs AI」二元框架是 2024 年觀念 — 真實結構是「OTA 與 AI 平台 vertical/horizontal 雙層分工」：OTA（BKNG / EXPE / TCOM）擁有 inventory + payment + 客服；AI 平台（OpenAI / Perplexity / Google）擁有 user attention + intent capture；兩者必須共生。BKNG 的 ChatGPT app 是這個共生的第一個 case；Perplexity + Selfbook + TRIP 是第二個。
2. Experiences (Viator + GetYourGuide + Klook + ABNB Experiences) 是 sector 中成長最快、估值最被低估的子 segment：FY25 Viator GBV +16% / 18% bookings growth / 50% TRIP rev / 30% TRIP profit；類比 ABNB 把 Experiences 重啟（May 2025 launch）= 雙巨頭都認可這是下一個 $50B+ 子市場；TRIP 不獨立分拆 = $2-3B 估值釋放被卡住。

§2 OTA 演進史 + S 曲線 + AI 分發革命

線上旅遊 OTA 過去 30 年歷經 5 個結構性轉折：(1) 1995-2005 是早期 OTA 興起 — Expedia (1996) / Priceline (1997) / Travelocity / Orbitz 多家分散；(2) 2005-2015 是大整合 — Priceline 收 Booking.com (2005) + Agoda (2007) + Kayak (2013)、Expedia 收 Hotwire + Trivago + Hotels.com + Orbitz + HomeAway；(3) 2015-2020 是行動 + 國際擴張 + Airbnb 平台衝擊；(4) 2020-2024 是 COVID 重塑 + Connected Trip 興起；(5) 2024-2026 是 AI 分發革命啟動 — Google AI Overview 60% 滲透 + ChatGPT/Perplexity/Kayak AI Mode 啟動 agentic booking + OTA 用 GenAI 反擊（BKNG AI Trip Planner / Smart Filter / Auto-Reply / Smart Messenger）。

本次革命最特別的是 「上游 throttle + 下游 disrupt」雙重壓力：(a) 上游 — Google AI Overview 直接吸走 inspiration / research query 的 zero-click，OTA SEO 漏斗被 throttle；(b) 下游 — ChatGPT / Perplexity / Kayak AI 等 agentic platforms 試圖直接 capture booking intent，繞過 OTA 主站。但實際結果是 BKNG / EXPE 用「把自己變成 agent 的 inventory layer」反守為攻 — Booking.com / Expedia 都已經是 ChatGPT 內建 app，這意味 OTA 不被 AI agent 取代，而是與 AI agent 共生。本次循環的核心啟示：OTA 必須是 AI-native + agent-friendly，BKNG 已 commit，EXPE 落後 12-18 個月。

★ 歷史類比（2 個正類比 + 1 個反類比）

正類比 1：2010-2015 mobile 革命對 OTA 的衝擊。2010 mobile 起飛時市場恐慌「OTA desktop 模式被 mobile-first apps 取代」；實際結果是 BKNG / EXPE 成功遷移到 mobile，市占擴大；只有沒能轉型的小 OTA（Travelocity / Orbitz 邊緣化）被淘汰。量化對照：2010-2015 BKNG GBV CAGR +27% / EXPE +18%（mobile 並未壓制 OTA growth）；本次 AI 分發類比 — 大 OTA 透過整合（BKNG ChatGPT app）保留份額，落後 OTA（EXPE 12-18 個月落後）+ 小 OTA（區域玩家）將失去份額。

正類比 2：2010-2020 Airbnb 對 OTA 的衝擊。Airbnb 起飛時市場恐慌「STR 取代 hotel + OTA」；實際結果是 BKNG / EXPE 進入 alt acc 領域（BKNG STR listings 8.6M / EXPE Vrbo 9% global STR share），同時 ABNB 自己也吃 OTA 的飯（13.6% take rate ≈ OTA commission）— 三方都贏。量化對照：2014-2024 BKNG alt acc room nights from 0 → 30%+ of total / EXPE Vrbo $20B+ GBV；本次 AI 分發類比 — BKNG / EXPE 既受 AI 衝擊又受惠 AI 整合，但落後者（EXPE）相對受傷更深。

反類比：2000-2003 Travelocity / Sabre dominance 結構性破裂。當年 Travelocity（Sabre 子公司）是美國 #1 OTA，但失敗的原因是無法擴張國際 + 無法整合多 supplier + 與 GDS 衝突；2003 後被 Expedia 與 Booking.com 永久超越。反類比警示：BKNG 目前看似 unbeatable，但若 (a) Connected Trip 整合失敗（保留 Booking.com / Priceline / Agoda 多平台 vs. 統一），(b) 中國 outbound 失守（被 TCOM 或 Trip.com 國際擴張吃掉），(c) AI 分發平台直接整合 supplier（繞過 OTA），可能重演 Travelocity 命運。多條件對照：Travelocity 當年 (a) 100% 美國依賴 — BKNG 全球分散；(b) 無 alt acc — BKNG 已進入；(c) 無 AI 整合 — BKNG 已 commit。多條件不成立，類比警示為 cautionary tale 但不直接適用。

💡 §2 Insight
1. AI 分發革命的時間表比市場預期慢，但分化結構比市場預期快：2026-2027 AI booking 仍 < 5% travel revenue，2030 才到 25-30%；但 BKNG vs EXPE 的 AI integration gap 已在 2025 形成 12-18 個月領先 — 這是分化結構的「鎖定效應」，3 年內無法翻轉。
2. Travelocity 反類比警示：BKNG dominance 並非 unbeatable，唯一真正能威脅 BKNG 的是 Google（80%+ travel search 入口 + Gemini Travel 整合 + Google Hotels meta-search 自身競品）；2027+ Google 若認真做 hotel booking funnel，BKNG 上游 throttle 將永久放大；這是 BKNG 投資人應該追蹤的最大反方訊號。

§9.5 Kill Scenario / Steel-man 反方

反方論證 #1：Google 認真做 hotel booking funnel — Google Hotels 升級為「BKNG killer」

路徑：Google 2026-2027 把 Google Hotels meta-search + Gemini Travel + AI Overview 整合成一個 vertical-first 的 hotel booking platform，繞過 OTA 直接整合 supplier；Google 自己賺取 commission，OTA 從 Google 流量 -30-50%。當前證據：(a) Google AI Overview 60% 滲透是 throttle 訊號；(b) Google Hotels 已 dominant meta-search 位置；(c) Google Pay 完整 payment infrastructure；但 (d) Google 過去從未認真做 OTA — 反方證據。量化影響：BKNG GBV growth 從 +12% → +5-8% / EBITDA margin 從 36.9% → 30-32%（marketing cost +200-400bps）/ 估值倍數從 24x → 16-18x = 股價 -25-35%。falsification 事件：2026-2027 Google I/O 任一場明確聲明 Google Hotels 升級為 booking platform。最壞情境 drawdown：BKNG -30-40%（從 ~$5,200 至 ~$3,000-3,500）。

反方論證 #2：Apple Intelligence + Maps + Wallet 整合 hotel booking — 繞過全部 OTA

路徑：Apple 2026-2028 用 Apple Maps + Apple Intelligence + Apple Wallet 整合 hotel booking；Siri "book hotel near here" → 直接從 supplier API 拿 inventory + Apple Wallet 付款；Apple 收 take rate 5-10% vs OTA 13-15% — 對 supplier 是更便宜的分發渠道。當前證據：(a) Apple Intelligence 仍 nascent，2025 travel 整合 0；(b) Apple 過去 partner-led 戰略（Apple Maps 與 Yelp / Booking 合作非競爭）— 反方證據。量化影響：若 Apple 2028 拿走 5% online travel = OTA -5% GBV；BKNG / EXPE 各受傷 5-10pp growth。falsification 事件：2026-2028 WWDC 任一場 Apple 明確進場 hotel booking。最壞情境 drawdown：BKNG -10-15%（小於 Google 風險）。

反方論證 #3：EXPE B2B mean-reversion + B2C 進一步崩盤 — EXPE 結構性死亡螺旋

路徑：EXPE B2B +24% Q4 2025 是高基期反彈（部分 lapse）；2026-2027 B2B mean-reverts 至 +10-15%（base rate）；同時 B2C +5% 持續 erode 至 +0%；整體 GBV growth 從 +8% → +3-5%；估值 12x → 8x。當前證據：(a) EXPE B2C 5 年持續 erode，無 catalyst；(b) B2B partner 高基期效應確實存在；但 (c) EXPE Q4 2025 B2B +24% 已 4 季 +20%+ 維持，結構性訊號。量化影響：EXPE EPS FY27 -10-15% / fwd PE 8-10x = 股價 -25-30%。falsification 事件：2026 Q1 Q2 EXPE B2B growth < +18%。最壞情境 drawdown：EXPE -25-30%（從 ~$155 至 ~$110-115）。
