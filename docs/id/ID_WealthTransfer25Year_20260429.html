§1 母題定義與邊界

Great Wealth Transfer 25 年期母題的範圍是2025-2048 美國 Boomer + Silent Gen $84.4T 資產轉移至 Gen X / Millennial / Gen Z（其中 Boomer alone $53T）相關的多 sub-vertical 受益玩家。本 ID 是 Master overview — 跨 6 個 sub-vertical：(1) Wealth Management（SCHW + MS + GS + RIA 行業）；(2) 房地產（Boomer 40% US housing wealth + Millennial 入市）；(3) Trust banks（NTRS + BK）；(4) 保險（estate planning + 終身保險 MET / PRU）；(5) Asset managers（BLK + TROW + BEN）；(6) Premium 消費 mirror（demand-side cross-link 至 Boomer Travel + Luxury 既有母題）。本 ID 與 ID_BoomerTravelSupercycle 是「同一 demographic 的兩面」— 前者看 Boomer 自己 spending（旅遊 / 高端 hotel / cruise），本 ID 看子女 inheritance + Boomer estate planning + 加速 spending。

邊界判斷的關鍵是「是否直接受益 financial wealth flow」與「是否時間框架 25 年期」兩條軸：wealth mgmt（SCHW / MS / GS / RIA）+ trust banks（NTRS / BK）+ asset managers（BLK / TROW / BEN）+ 保險（MET / PRU / AIG）+ 房地產（DHI / LEN / NVR）— 全部 in-scope；premium 消費（Boomer 自己 spending）— 部分 in-scope（demand-side mirror，但詳細在 ID_BoomerTravelSupercycle）；wealth tech（Robinhood / SoFi 部分）— 邊緣（Millennial / Gen Z 為主）；private equity / alternative（BX / KKR）— 部分 in-scope（高端 cohort 配置）；blockchain / crypto wealth — out-of-scope（仍 nascent）。

💡 §1 Insight
1. Wealth Transfer + Boomer Travel + Luxury 三大 master ID 是「同一 demographic 的三個受益面」：Boomer 自己消費（Travel）+ 高端配置（Luxury）+ 代際轉移（Wealth Transfer）共振；建議 portfolio 三 master ID basket 整合配置（10-15 檔 multi-vertical）。

§2 Wealth Transfer 演進史 + 25 年曲線 + 歷史類比

美國代際財富轉移過去 50 年歷經 3 個結構性轉折：(1) 1970-2000 是「第一代財富累積」— Boomer 在 25-55 歲，401(k) 推出（1978 ERISA）+ home equity 累積；(2) 2000-2024 是「第一代財富 peak + 退休準備」— Boomer 36-78 歲，401(k) 累積 peak / housing 大幅升值 / wealth gap 擴大；(3) 2025-2050 是「第一代財富轉移 25 年」 — Boomer + Silent Gen $84.4T 至 Gen X / Millennial / Gen Z；2025-2030 transfer 加速期 / 2030-2040 peak transfer / 2040-2050 完成期。

本次循環的關鍵驅動：(a) Boomer 健康壽命延長 — life expectancy 78 → 80 + 健康壽命 60 → 65；意味 transfer 速度受 demographic 但不是「立即」；(b) Boomer wealth concentration 集中 — top 10% 美國家庭佔 51.4% wealth / top 1.5% 佔 42% 預期 transfer = 高端 family office + private wealth mgmt 業務 飛躍；(c) Estate Tax 政策變化 — Trump 2025 永久化 + 2026 升 $15M/人 = 加速 lifetime trust planning；(d) 「Use it or lose it」心理 — Boomer 60%+ 已決定不留全部給子女 = 加速 self-spending（與 Boomer Travel / Luxury 共振）+ 年 $19k 贈與 + lifetime trust。本 Master ID 認為 transfer peak 在 2030-2040，2030+ wealth mgmt + trust banks 飛躍。

★ 歷史類比（2 個正類比 + 1 個反類比）

正類比 1：1990-2010 401(k) 大規模化（從 nascent 至 mass adoption）。當年 ERISA 1978 後 401(k) 從小規模到 1990-2010 大規模化，產生 SCHW / Vanguard / Fidelity 大型 wealth mgmt 業務 起飛；2000-2010 SCHW assets 5x。量化對照：1990-2010 SCHW assets CAGR ~15% / 業務 EBITDA CAGR ~12%；本次 Wealth Transfer 25 年期推估 SCHW / MS / GS / NTRS 等 5 年 CAGR 12-15% 是合理 base case；類比明確。

正類比 2：2008-2024 ETF 大規模取代 mutual fund。當年 BLK iShares + Vanguard 推動 ETF 從 ~$500B → ~$10T (20x in 16 年)；TROW / BEN 等主動 mutual fund 受擠壓。量化對照：本次 Millennial 接收 $46T transfer 預計大部分配置 ETF + RIA + 部分 alternative；BLK iShares + Vanguard 結構性受益；TROW / BEN 結構性壓力（雖然有 niche 服務）。但本 ID 認為 mutual fund 並非完全消失 — Gen X / 早 Millennial 高端 cohort 反而轉向 RIA + alternative（BX / KKR）。

反類比：2008 金融危機後「下一代將更窮」共識。當年市場曾期望 Gen X / Millennial 將 inheritance 不足 / 學貸高 / 房屋買不起；實際 2010-2024 Millennial 高端 cohort 反而透過科技 / 金融成為 self-made millionaires + 部分 inheritance 提早。反類比警示：「下一代收益」假設可能 underestimate 子女自己 wealth creation；inheritance 反而對於 Gen X / 早 Millennial 是「補充」而非「主要」收入。多條件對照：當年 (a) 危機後 stagnation / (b) housing 凍結 / (c) ETF 仍 nascent。當前 (a) 24 年多頭 / (b) housing 仍升 / (c) ETF mature + RIA mature。多條件不成立但仍是 cautionary tale — Gen X / 早 Millennial 高端 cohort 不一定 fully 仰賴 inheritance。

💡 §2 Insight
1. 本 thesis 的時間 bound 是 2030（transfer 啟動期完成）+ 2040（peak）+ 2050（完成）：投資 horizon 2026-2035 是 sweet spot；2035 後仍 buy-and-hold 但需重新評估配置。
2. 反類比警示：Gen X / 早 Millennial 高端 cohort 自己已 wealth creation，inheritance 是「補充」而非「主要」；本 ID 重點在 wealth mgmt 業務的 fee growth 而非「Millennial 突然有錢」消費 thesis。

§9 風險矩陣 + Kill Scenario

Kill Scenario 反方論證 #1：美國 housing -20% 大幅下行

路徑：housing -20%+ → Boomer wealth -15-20% / 加速 transfer 但同時降低 transfer 規模；wealth mgmt + Boomer self-spending 雙傷。最壞情境：6 檔 basket -20-30%。

反方論證 #2：Estate tax 政策 reverse — 民主黨 2028 上台 + tax 加重

路徑：2028 民主黨上台 reverse Trump tax overhaul / estate tax exemption 從 $15M 降回 $5M / 影響高端 family planning。最壞情境：trust banks + family office +20% short-term boost（加速 planning）但 longer-term 業務降溫。

反方論證 #3：ETF fee 結構性壓縮 + RIA fee compression

路徑：BLK iShares fee 從 ~10 bps → 5 bps / RIA advisor fee 從 100 bps → 75 bps；wealth mgmt + asset mgr 整體 EBITDA -20-30%。最壞情境：basket -15-25%。
