§1 產業白話定義 + 歷史脈絡

白話版：電商這門生意過去比的是「誰賣得便宜、誰到貨快」。但滲透率進 late phase 後，單純賣貨的毛利薄成 commodity——真正的錢在三層：賣廣告（看得到曝光變購買）、賣商家工具（鎖住商家）、自建物流（跨境打不穿）。錢往這三層集中。

白話開場：先說四個關鍵詞。GMV 是平台總交易額（SHOP 一季跑 $123.8B），revenue 是平台抽成後的營收（AMZN 一季 $181.5B），ad revenue 是廣告收入（AMZN 一季 $17.2B）——三個數字口徑不同、不可直接比。de minimis 是美國 $800 以下進口免稅、2025-05-02 對中國取消，這是跨境傾銷退潮的引爆點。本報告問的是：當全球電商滲透率進入 late phase（全球 ~17%→2030 ~25%），這塊大餅的利潤被重分配給了誰。

邊界界定：本報告聚焦「全球電商的利潤往哪三層集中 + 誰被商品化」。納入規模龍頭（AMZN）、OS（SHOP）、區域 leapfrog（MELI/SE/CPNG/BABA）、跨境（PDD/Temu）、傳統零售 pivot（WMT）、niche（ETSY）。LATAM 區域深掘（MELI/NU 的 fintech 飛輪、巴西物流網絡）詳見姊妹 ID_LATAMEcommerce——本份只取 MELI 作「區域 leapfrog 護城河」的 global 案例，不重複 fintech 細節。旅遊 e-comm（BKNG）另屬範圍。灰色地帶是「SHOP 是 SMB 還是 OS」——若 B2B + Payments 續證企業級採用，估值溢價結構性合理。

歷史分三段。第一段：1995–2010 PC 電商——AMZN/eBay 起家，比的是品項齊全。第二段：2010–2024 行動 + Marketplace + COVID + 跨境傾銷——智慧型手機普及、Marketplace 飛輪、行動支付；COVID（2020–2022）把全球滲透率從 ~14% 推到峰值；Temu/Shein/AliExpress 靠 de minimis 免稅 + 直營跨境衝出爆發 GMV。第三段：2025–2026 廣告 + AI + 整合洗牌——de minimis（2025-05-02）取消、跨境傾銷退潮；AMZN 廣告 +24% trailing 過 $70B、SHOP B2B +96%、MELI rev +49%。alpha 從 retail GMV 移到三層利潤池。

歷史類比一：2010–2018 AMZN 規模整合——利潤池悄悄遷移。上一輪美國電商整合，AMZN 靠規模 + Prime + 物流取得龍頭，eBay 與傳統零售落後。量化錨：看空 AMZN「retail 不賺錢」的空頭錯了十年——AMZN 2010–2025 漲約 30 倍，因為真正的利潤引擎不是 retail（毛利 ~25% 的 commodity），而是悄悄長出來的 AWS（毛利 60%+）與廣告（毛利 ~70%）。對本次 bull 含義：當前 AMZN 廣告 +24%、trailing 過 $70B，正是同一齣戲的下一幕——市場又一次把 AMZN 當 retail 估，而 alpha 在高毛利層。歷史類比二：2020–2024 跨境傾銷浪潮——補貼換市佔的宿命。Temu/Shein 靠 de minimis + 直營跨境衝爆 GMV。量化錨：2025-05-02 美國對中國取消 de minimis、US tariff 一度 120% 後談判至約 54%，Temu 美國日活在 2025 年 3→5 月一度 -48%、June 2025 調查顯示 45% 美國消費者減少或停用 Temu。教訓：靠政策套利 + 補貼換來的市佔，政策一變就退潮——但 Temu 沒消失，pivot 到半托管（2026 商業化 + 廣告變現）、美國 GMV 占比目標從 60% 降到 30%、歐洲靠在地庫存 +30% GMV。

本主題無單一價格 cycle，用「電商滲透率 + 三層利潤池」當時序錨。

關於 cycle 統計表的省略：電商隨總經有週期（消費信心、利率），但本主題是結構性利潤遷移（廣告/OS/物流），非單一 ASP cycle，故不立 cycle 統計表；以「電商三段」（1995–2010 PC → 2010–2024 行動/跨境 → 2025–2026 廣告/AI 整合）的編年呈現時序。

這段歷史的可操作含義：當前 Phase IV 同時是「規模整合的下一幕」（利潤往廣告/OS/物流遷移）與「跨境傾銷退潮」（de minimis 後補貼模式崩）。bull 的底是握廣告閉環/OS/在地物流者（AMZN/SHOP/MELI），bear（對跨境傾銷）是 Temu 舊模式被 2025-05-02 結構性 reset。判別一家電商站哪邊，看它賣的是「補貼價差」還是「廣告/OS/在地履約」。

§2 技術成熟度 + S 曲線

為什麼是現在？因為三件事同時成熟：① 平台廣告閉環（第一方意圖資料 + 看得到「曝光→購買」）；② OS 把建站/支付/B2B/履約整成一套；③ 在地物流網絡規模化（跨境打不穿）。三者讓「單純賣貨」這層 commodity，利潤被前三層吸走。

技術成熟度有三個拐點。廣告閉環成熟：電商平台自帶購買意圖訊號、看得到全鏈歸因，廣告主給的 ROI 可量化，AMZN 廣告 +24% 是「閉環廣告優於曝光廣告」的證據；AI 推薦再加碼（個人化把轉換率推高）。OS 整合成熟：SHOP 把建站 + Payments（占 GMV 68%）+ B2B（+96%）+ 履約整成一套，商家越深用越難搬家。在地物流規模化：MELI 6 大配送中心、Shopee SPX、CPNG 韓國次日達 + 台灣 70% 覆蓋——自建履約是 5+ 年資本 + 密度才追得上的護城河。三者交會，「單純賣貨 + 跨境傾銷」這層被商品化，利潤往廣告/OS/物流上移。

§3 供給側：現在與未來 + 利潤池

反直覺但可操作的結論：全球電商的利潤往「廣告閉環 + OS lock-in + 在地物流」三層集中——這三層贏，無差異跨境傾銷與被擠 niche 輸。

供給端可拆五層。廣告利潤池（AMZN/MELI/WMT）：平台自帶購買意圖訊號 + 閉環衡量，ad 毛利 ~70% 是 retail ~25% 的近 3 倍，AMZN 廣告 +24% 是真 alpha。OS for commerce（SHOP）：商家 lock-in + Payments + B2B，從 SMB SaaS 升級企業級電商作業系統。在地物流 leapfrog（MELI/SE/CPNG）：自建履約網絡是跨境玩家 5+ 年才追得上的護城河。1P/3P Marketplace（AMZN/MELI/Shopee）：規模 + Prime 訂閱鎖客，但 retail 段毛利薄。跨境傾銷（Temu/Shein 舊模式）：補貼換 GMV、毛利 5–10% 薄、政策敏感、正被商品化。利潤往前三層集中，跨境傾銷層失血。

利潤池遷移

整條鏈的利潤正從「單純賣貨 + 跨境傾銷」往「廣告 + OS + 在地物流」遷移：retail 段毛利 ~25% 是 commodity，廣告 ad 毛利 ~70% 是同一家公司內部的高毛利層、SHOP Payments 占 GMV 升到 68%（lock-in 加深）、在地物流是密度護城河。被搶的是無差異便宜貨與跨境傾銷玩家，搶到的是廣告閉環（AMZN/MELI）、OS（SHOP）、在地物流（MELI/SE/CPNG）。AMZN 廣告 +24%（遠快於國際 retail +11%）與 Temu 美國日活 -48% 是利潤遷移的一體兩面。（依 QC-18，全球電商利潤池逐段精確百分比難拆，以定性 + 增速/份額數呈現。）

§4 需求側：現在與未來 + 三角驗證

需求最剛性的是「廣告主要可歸因 ROI + 商家要降摩擦的全套工具 + 新興市場消費者要可靠履約」——這三股剛需把錢推向廣告閉環、OS、在地物流三層。

第一引擎：廣告閉環（現需）。電商平台自帶購買意圖訊號、看得到「曝光→購買」，這是廣告主最想要的可歸因 ROI。AMZN Q1 26 廣告 $17.2B（+24% YoY、超 Wall Street 預期 +21.2%）、trailing 過 $70B；MELI 廣告為三引擎之一、WMT Connect 為 #2 RMN。第二引擎：OS lock-in（現需）。商家要的是「一套工具搞定建站 + 支付 + B2B + 履約」——SHOP B2B GMV +96%（Q4 +84%）、Payments $84B 占 GMV 68%（比去年 +4pp）、Merchant Solutions rev Q4 +35%，商家越深用、轉換成本越高。第三引擎：區域 leapfrog（現需）。新興市場消費者跳過 PC 直接行動電商 + 要可靠履約——MELI rev +49%（巴西降免運門檻是成長引擎）、Shopee GMV $37.3B +30% / 訂單 +29%、CPNG 台灣三位數成長。三引擎都把錢推向「握閉環/lock-in/履約」者。

  • 我們說 Y：SHOP B2B GMV +96% / Payments 占 GMV 68% / 首破 $100B GMV 季 = 升級為「企業級電商 OS」+ B2B 新引擎；fwd PE 70x 高溢價結構合理。🔵 [X: base 很可能 — B2B 續 +40%+；bear 可能 — B2B 降至 < +30%]。
  • Priced-in：市場 part-price 了 SHOP 的成長（70x 高分位），但「企業級 OS + B2B 第二引擎」的結構性 未充分 price——分歧 2 的 OS 升級是 next 12-month catalyst。
→ 推論邏輯：SHOP 從 SMB 升級企業 OS，B2B 是新 alpha 層。⚠ 證偽：SHOP B2B 增速連 2 季 < +30% YoY 或 GMV 增速 < +20%。
🟡分歧 3：「Temu de minimis 後就萎縮消失」信心：中
  • 共識說 X：Temu 2025-05-02 de minimis 終止後將大幅萎縮、中國跨境傾銷結束。🟢 [F: T1/T2 — Temu 美國日活 -48%、45% 美國消費者減少使用]。
  • 我們說 Y：Temu 沒消失、是 pivot——半托管（2026 商業化 + 廣告變現、年度損益平衡目標）+ 在地履約 + 美國 GMV 占比 60%→30%、歐洲 +30% GMV。不是傾銷但仍 structural player（不同層級）。🔵 [X: base 很可能 — 半托管 ramp、美國 GMV 不回 peak；bear 可能 — 在地履約規模化反擊]。
  • Priced-in：市場 price 了 Temu 傾銷退潮（PDD de-rate），但「半托管 + 廣告變現轉型成敗」未充分 price——分歧 3 的 pivot 結局是 PDD 估值關鍵變數。
→ 推論邏輯：跨境傾銷退潮但結構性留存，alpha source 從補貼價差移到半托管 + 廣告。⚠ 證偽（QC-I22）：Temu 美國 GMV 恢復至 2024 peak +20% 連 2 季 → 在地履約反擊成功、跨境結構重設。

Steel-man 反方（最有力的 3 條）

  • 反方 1（AI agentic shopping 改寫入口）：ChatGPT/Gemini 整合 buying agents、用戶不再進 AMZN 直接 LLM 購物 → AMZN 廣告閉環稀釋、ad 增速放緩。當前證據：OpenAI/Anthropic agent capability 提升、AMZN 整合 Alexa+ AI shopping。證偽窗口：agentic shopping 占 e-comm GMV、AMZN 廣告增速。成真時間窗：2027–2030。
  • 反方 2（跨境玩家在地履約反擊）：Temu/Shein 大規模設美/歐在地履約 → 跨境玩家 +10pp share 反擊、MELI/Shopee 在地護城河被打穿。當前證據：Temu 半托管 + 歐洲 +30% GMV、Shein 美國設廠進度。證偽窗口：Temu/Shein 美國 GMV vs 2024 peak。成真時間窗：2027+（在地履約 5+ 年才能匹配 MELI/Shopee）。
  • 反方 3（衰退 + 監管拆分）：美國衰退 → 電商整體 5–10% drag、高 PE 高 beta 的 SHOP de-rate；或 FTC 強制 AMZN 拆分 Marketplace/AWS/廣告。當前證據：FTC 2025 起 AMZN 訴訟、但 Trump 治下 FTC 動作放緩。證偽窗口：GDP/消費信心、FTC 訴訟進度。

市場最低估的：最低估的upside是分歧 1 的「AMZN 廣告 alpha 被重估為 ad company」與分歧 2 的「SHOP B2B 企業級 OS」；最低估的風險是反方 1 的「AI agentic shopping 稀釋廣告閉環」。三者都該獨立監測。

三條分歧裡最可操作的是分歧 1（AMZN 廣告 alpha）+ 分歧 2（SHOP 企業 OS）——前者有明確領先指標（廣告增速 vs retail 增速），後者每季財報直接驗證（B2B GMV + Payments 滲透）。分歧 3（Temu pivot）次之、是 14 天 monitor 條件。最大長期裂縫是反方 1（agentic shopping），但 2027+ 才是壓力期。
