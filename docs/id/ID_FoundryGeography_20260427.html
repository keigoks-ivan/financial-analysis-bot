💡 3 條核心 insight（預告）
1. 「去台化」是 narrative 而非 reality：TSMC Arizona Fab 21 第一座 fab 已量產 4nm（與台灣同等良率），但全球 TSMC leading-edge 產能 ~70% 仍在台灣。即使 Arizona 6 fabs + Kumamoto 2 fabs + Dresden 1 fab 全部完工（2030），TSMC 海外 leading-edge 產能也僅 ~20-25%。
2. Tesla AI5/AI6 在 Samsung Taylor + TSMC Arizona 雙代工是 vertical hyperscaler 新模式：$16.5B 7.5 年合約 + dual-source derisking + 跨美國本土兩家 foundry — 這個模式如果 Apple / NVIDIA 跟進，Samsung Foundry 從「2nm 邊緣玩家」結構性升級為「leading-edge 第二選擇」。
3. Intel Ohio 延到 2030 + Magdeburg 取消是 IDM 2.0 商業模式破產訊號：原 $100B Ohio mega-fab 推到 2030+；德國 Magdeburg + 波蘭組裝廠取消；Intel Foundry 的全球版圖已崩解；只剩美國本土 + 少數 18A/14A 客戶（MSFT、AWS）— Intel Foundry thesis 從「全球第三大 foundry」退到「美國本土 niche」。

§1 產業定義與邊界

「Foundry Geography」是個跨越商業 + 政策 + 地緣三層的主題。本 ID 把焦點放在「leading-edge foundry 的全球地理分配」，特別是 2024-2030 這波擴產 / 重組對未來 5-10 年產業格局的結構性影響。本節定義五個地理 hub + 對應玩家：① 台灣（TSMC 主場）；② 美國（Arizona / Texas / Ohio）；③ 日本（Kumamoto JASM + Hokkaido Rapidus）；④ 歐洲（Dresden ESMC）；⑤ 韓國（Samsung 國內 + 美國 Taylor）。

灰色地帶（★ 邊界爭議）

  • Rapidus 算 leading-edge 還是「政府實驗」？ Rapidus 與 IBM JV 在 Hokkaido 建 2nm 試產線，技術源自 IBM 2nm research。本 ID 把它納入但標註「2030+ 仍是 wildcard」 — 如果成功是日本 leading-edge 突破，不成功則仍 ≈ 0。
  • Tesla AI5/AI6 雙代工算「customer-foundry 變化」還是「dual sourcing」？ 兩者皆是。本 ID 把它視為「customer-foundry 新模式」的 first-mover — 改變 Samsung Foundry 在 leading-edge 的競爭地位。
  • Intel Ohio 算「美國 leading-edge 產能」嗎？ 推到 2030 後實際時程不確定。本 ID 在 §4 TAM 中把 Intel Ohio 列為「2030+ 可能性」而非 base case。
💡 Insight §1
1. 「去台化」是敘事而非現實：即使所有海外擴產順利，TSMC 台灣本土仍掌握全球 leading-edge 70%；地緣分散是「補充」而非「替代」。
2. Rapidus 是 2030+ wildcard：成敗將決定日本是否從 leading-edge 0% 拉到 3-5%；對 Tokyo Electron / Disco / 信越化學等日系設備材料廠是核心 catalyst。
3. Tesla AI5/AI6 雙代工是 customer-foundry 新模式：Apple / NVIDIA 是否跟進是 Samsung Foundry 結構性翻身的關鍵。

💡 Insight §2
1. 1985-2000 美日半導體競爭是反向類比：當年日本 DRAM 從 80% → 30% 的衝擊是因為政府介入；本次台灣 TSMC 是「商業壟斷 + 軟性 know-how」，不容易被政策撼動 — 這是 thesis #1 的歷史基礎。
2. 2000-2010 中國製造業外移是「先預防」這次的反面教材：CHIPS Act 在 2022 動手是「pre-emptive defense」；如果 2030 後中國真有衝擊，美國有 buffer。
3. TSMC 在 leading-edge foundry 的地位類比 ASML 在 EUV 設備：地理分散能改變比重但改變不了核心；TSMC know-how 是「軟性護城河」。

§5 Value Chain

foundry 全球擴產的 value chain：上游（半導體製程設備 AMAT / LRCX / TEL / ASML）→ 中游（foundry 自己 TSMC / Samsung / Intel）→ 下游（fabless 客戶 NVDA / Apple / AMD / MSFT / Tesla）。關鍵觀察：上游設備廠是「最確定」受益者 — 不論 fab 蓋在哪裡，都需要 ASML EUV、AMAT 沉積、LRCX 蝕刻；地理分散反而增加設備總量需求（每個 fab 都需要完整一套設備）。

§7 Unit Economics

關鍵 cost premium：美國 fab vs 亞洲 25-30%（人工 + 規範 + 建設）；日本 vs 亞洲 ~10-15%；歐洲 ~20-25%。CHIPS Act 補貼可填補 50-70% 但不全部；剩餘必須由客戶 / 政府吸收。

Apple / NVIDIA 願意付溢價是商業可持續關鍵 — Apple 2026 在 Arizona 100M+ chips 是「demonstrating willingness to pay」的明確訊號。

§9.5 Kill Scenario

🟡 反方 1：Intel 18A 商業化失敗，全球 foundry 退到 TSMC + Samsung 雙寡占 信心：中（30-40%）
→ Intel 18A 若 2026-2027 yield 不及預期 → 失去 MSFT、AWS 等大客戶 → 退出 leading-edge foundry。Intel Foundry 整體商業模式崩解；對 TSMC + Samsung 是大利好但對全球 foundry 多元化是負面。
⚠ 何時成真：2027 末 Intel 18A yield 仍 < 60% + MSFT/AWS 任一退出
🟡 反方 2：Rapidus 2nm 量產失敗，日本 leading-edge 仍 ≈ 0 信心：中（35%）
→ 歷史上日本 foundry 嘗試多次失敗；Rapidus 是 IBM 技術授權但 know-how 仍需多年 build；2nm 是極其困難的節點。
⚠ 何時成真：2028 末 Rapidus 2nm 仍未進入 risk production
🟡 反方 3：美中科技戰升級使全球 foundry 分裂為「美國體系」+「中國體系」 信心：低（10-15%，但結構性影響極大）
→ 若美中關係惡化到「全面 decoupling」，TSMC / Samsung 必須在「美國 vs 中國」二選一；中國市場（佔全球半導體 30%）失去；產業 fragmenting。
⚠ 何時成真：美中達成「leading-edge 完全脫鉤」協議
