KEY CALL — 本報告最重要的一個判斷

市場用「Boeing lag / Airbus 領先」的市佔框架看商航，真正的非共識是——2026 的瓶頸是引擎不是機身。同一個 Q1，Boeing 七年首次「季度交付」超越 Airbus，靠的不是 Boeing 變強多少，而是 Airbus 被 Pratt GTF 引擎短缺卡住交付（A320neo 家族 -24%、整年指引砍到 ~870 機）。當需求塞滿（~16,683 機在手、約 12 年覆蓋）、而稀缺來自「合格產能 + 引擎熱端零件 + 認證門檻」三道結構卡點，利潤池就會結構性往握有最稀缺產能的環節傾斜。分歧不在「商航會不會復甦」（很清楚），而在「哪一段最稀缺、現價還剩多少 alpha」。

§1 產業白話定義 + 歷史脈絡

商業航太賣的不是「一架更快的飛機」，而是「在需求塞滿、產能交不出來的十年裡，誰握有最稀缺的那一段產能」。付錢的是全球航空公司與租賃商，而他們現在最痛的不是飛機不夠好，是訂了也要等十年、舊機只能延役硬撐。

白話開場：商業航太是個雙寡占疊雙寡占的產業——機身上 Boeing 與 Airbus 兩家分天下，narrowbody 引擎上 CFM（GE+Safran 合資的 LEAP）與 Pratt & Whitney（GTF）兩家分天下，再加一群做售後零件與大修的高利潤玩家（TransDigm、Heico、各引擎廠的售後段）。它的收入模式分三段：賣新機（OEM，低利潤、現金流前重後輕）、賣引擎（中高利潤、靠 installed base）、賣售後（最高利潤、靠機隊飛行時數與大修週期）。理解這個產業的關鍵，是看懂「同一架飛機，OEM 賺一次薄的、引擎廠賺一次中的、售後賺三十年厚的」。

邊界界定：本報告聚焦「商業客運飛機的製造、引擎與售後」這條主軸。納入 OEM（BA / AIR）、narrowbody+widebody 引擎（GE / RTX-Pratt / Safran / Rolls-Royce / MTU）、售後零件與 MRO（TDG / HEI）、以及結構件與商務/區域機（SPR / GD / ERJ）。不展開純國防航太（LMT / NOC，是 ID_DefenseModernization 範圍）、航太特殊金屬（HWM / ATI，是 ID_AerospaceMetals 範圍）、以及下游航空公司（DAL / UAL，是需求方非製造方）。灰色地帶在「RTX」——Pratt + Collins 是 commercial 主場（佔 RTX 營收約 55%）、Raytheon 段是國防，本 ID 與 ID_Defense 雙列，差異化在 Pratt GTF 售後與 Collins 航電。

歷史脈絡分四個轉折。第一次：1970–1990——747／DC-10 widebody 革命，jet age + 全球航線把 Boeing 推上王座、麥道隨後出局。第二次：1990–2010——LCC（低成本航空）崛起把 narrowbody 需求結構化，Airbus 用 A320 突破、與 Boeing 737 形成穩固雙寡占。第三次：2010–2020——燃油效率革命，787／A350 用複材機身、LEAP／GTF 用新一代引擎，把 fuel burn 砍下一個世代；同時 narrowbody 引擎正式進入 CFM vs Pratt 雙寡占。第四次（本次）：2022 起——COVID 後 RPK 超疫前復甦，疊加 Boeing 品質修復與 LEAP installed base 進入售後收割期，但整段被「合格產能交不出來」這道結構卡點定義。

最硬的壓力測試：2018–2020——兩起空難導致 737 MAX 全球停飛約 22 個月，Boeing 股價自高點跌約 75%，Airbus 趁勢拿下市佔。這是商航史上對單一機型最嚴重的信任打擊。其後遺症延續至今：Boeing 的月產率被 FAA cap、品質文化被重塑、且 2025-12 把機身供應商 Spirit AeroSystems 整個收回（$4.7B 股權 / 含債 $8.3B、FTC 2026-02 finalize consent order），把機身品質責任全部內部化。Q1 26 的 143 機交付、114 是 737、七年來首次季度超 Airbus——是「翻轉為真」的硬證據，但 FCF 仍 -$1.5B，代表這場 turnaround 還在「交得出來、但還沒賺到現金」的階段。

歷史類比：2010–2018 後 GFC 復甦。當年 RPK 六年約 +60%、OEM 合計交付從約 990 升到約 1,600 機、BA+AIR 股價漲約 4 倍。本次的量化差異是供給更緊——上一輪在手訂單覆蓋約 4–5 年、總裝產能相對充足；這一輪覆蓋約 12 年（~16,683 機）、卻被引擎熱端零件與認證門檻卡死。所以「這次更像供給受限」不是口號，是「12 年覆蓋 + 引擎短缺反咬 Airbus」對「4–5 年覆蓋 + 產能充足」的對照。

商航有可統計的交付/在手 cycle，但本輪的價格訊號不在新機 ASP（list price 黏住），而在二手機價與租金——看稀缺滲透到哪。

關於 cycle 統計的口徑：商航的需求 cycle 用 RPK + 訂單衡量、供給 cycle 用交付 + 在手衡量，兩者本輪嚴重背離（需求塞滿、交付受限）。最能反映「合格產能稀缺」的價格訊號是二手 narrowbody 機價與租金創歷史新高——新機交不出來，舊機才會被搶。時序錨點：2026-04 單月訂單 164 架（YoY +763%）把合併在手推到 ~16,683 機；而兩家 OEM 合計年交付仍僅約 1,400 機——訂單與交付的剪刀差，就是本輪的核心張力。

這段歷史的可操作含義是：真正該怕的 bear 不是「航太股太貴」（那是市場情緒），而是「合格產能突然正常化（兩家 OEM 同步把總裝爬上來 + 引擎短缺解除）」使供給受限紅利消退、或「Boeing 品質再復發」把 FAA cap 打回去。只要產能仍追不上需求、且 Boeing 不出大事，利潤往引擎/售後集中的結構就成立——這把 bear 的天花板從「估值」重新定位到「產能正常化速度 + Boeing 品質可控性」。

§2 技術成熟度 + S 曲線

商航的技術成熟度可用「全球 RPK 與 OEM 交付」的 S 曲線定位。本輪的特徵是：需求曲線已超疫前並持續上行，但交付曲線被產能壓在較低斜率——這正是「供給受限」在圖上的長相。

技術演進的四代脈絡：第一代 widebody 革命（1970–1990）、第二代 narrowbody 雙寡占成形（1990–2010）、第三代複材 + 新引擎效率革命（2010–2020）、第四代 COVID 後供給受限復甦（2022+）。本輪的技術重點不在「飛機更快」，而在引擎熱端零件的良率與壽命——LEAP 與 GTF 都用了更高的渦輪前溫度換效率，代價是熱端零件（高溫合金葉片、粉末冶金盤）的耐久度與大修頻率成為產能瓶頸。Pratt GTF 的 powder-metal 污染問題導致約 720 機 grounded，正是這道技術成熟度尚未完全收斂的證據。

歷史類比帶量化錨：1995–2001 widebody supercycle。上一輪引擎售後黃金期出現在 1990 年代末——747／777 的 installed base 進入大修週期，GE 的引擎與售後營收三年內倍增，股價在 1995–2000 約上漲 +500%。本輪的相同點是 installed base 進入收割期（LEAP 2024–2030 約三倍）；量化差異是規模——當年是 widebody（基數小、單機售後價值高），這輪是 narrowbody（基數大約 5x、單機售後價值較低但機數遠多）。所以「LEAP 售後黃金期類比 1995–2001」不是口號，而是「installed base 收割 + shop visit 量增」的結構同型；差異在 narrowbody 的規模讓總量更大、但單機溢價較薄，這也是為什麼 GE CES margin 26.4%（高但非極端）而非當年 widebody 的更高水位。

S 曲線的擴散階段定位：本產業的「採用曲線」不是新技術滲透，而是installed base 進入大修週期的滲透。LEAP 在 2016 開始大量交付，第一批引擎正好在 2024–2028 進入首次重大 shop visit——這是售後黃金期的起點，而非終點。換句話說，GE 的售後收入曲線目前在 S 曲線的「加速段」前段，而非已飽和的成熟段——這是 §6 估值傳導判斷「售後黃金期僅在開始」的技術依據。

技術 → 投資的橋
引擎熱端零件的良率，是本輪「合格產能」的最底層瓶頸。誰能把熱端零件良率與 shop visit 產能爬上來，誰就解鎖最稀缺的利潤——GE LEAP 內部 shop visit +50%、CES margin 26.4%，代表 GE 在這道瓶頸上領先；Pratt 的 grounding 則代表 GTF 仍在收斂中。這把「技術成熟度」直接翻譯成「誰拿走售後利潤池」。

§3 供給側：現在與未來 + 利潤池

供給端的結構是「兩家 OEM 雙寡占 + 兩家 narrowbody 引擎雙寡占 + 一群高利潤售後玩家」，但 2026 真正的故事在誰被什麼卡住。三道結構卡點：① OEM 總裝產能（Boeing 被 FAA cap + 自身品質、Airbus 被供應鏈）；② 引擎熱端零件（高溫合金、精密鑄件、shop visit 產能）；③ 認證門檻（新引擎/新變體要 FAA/EASA 認證、無法繞過）。

利潤池量化：以 EBIT margin 排序，售後零件（TDG ~50%+、HEI ~25–28%）> 引擎 OEM（GE CES 26.4%、RR 15%+、Safran 15–18%）> OEM 主機廠（AIR ~10–12%、BA 回升中、仍負 FCF）。sector profit pool 因此高度向引擎/售後傾斜——這也是為什麼「買 OEM beta」遠不如「買引擎/售後現金流」精準。GE Aerospace 作為「引擎 + 售後」雙重曝險的純玩家，是這個利潤池結構的最大受益者。

供給側的可操作結論：利潤往最稀缺的環節集中，而最稀缺的是引擎熱端零件與 shop visit 產能。GE（LEAP 售後黃金期）、TDG（proprietary 售後定價）、RR（widebody 售後復甦）是利潤池傾斜的直接受益者；OEM 主機廠是「彈性大但利潤薄」的 turnaround 賭注。

§4 需求側：現在與未來 + 三角驗證

商航需求由兩個引擎共同推動。第一是 RPK 復甦：全球旅運已超疫前約 15% 並持續成長，新興市場與跨境旅遊把 narrowbody 需求結構化，訂單因此塞滿——2026-04 單月湧入 164 架訂單（YoY +763%），把合併在手推到 ~16,683 機的歷史高。第二是 機隊老化：因為新機交不出來，航空公司被迫延役舊機、加大可靠度與妥善率投資，把售後（MRO、shop visit、PMA 零件）需求結構性墊高——二手 narrowbody 機價與租金漲到歷史新高，就是「新機稀缺」的直接價格訊號。

§5 供需裁決 + 短中長期推估 + 投資時鐘

供需裁決：本產業不是傳統「供需平衡/失衡」框架能套的——它的供需失衡是「需求遠超合格產能且結構化十年」，不會自然均衡。框架逃逸需架橋：這是「產能爬坡受限型」復甦——稀缺不來自需求過熱、也不來自供給去化，而來自三道無法快速移除的供給卡點（OEM 總裝產能 + 引擎熱端零件 + 認證門檻）。所以判斷不問「何時供需平衡」，而問「合格產能爬坡多快、哪段先解鎖」。

§5 框架逃逸架橋
傳統供需曲線在這裡失效，因為供給曲線是「垂直且僵固」的——短期內合格產能近乎無法增加（建一條總裝線、認證一款引擎、訓練 MRO 技師都要數年）。所以本節改用「產能爬坡受限型」框架：把判斷重心從「供需何時均衡」移到「三道卡點各自的解鎖時程」，並以二手機價/租金（最敏感的稀缺價格訊號）替代新機 ASP 當溫度計。具體機制是——稀缺來自三層疊加：① OEM 總裝產能（Boeing 受 FAA 月產率 cap + 自身品質文化重建、Airbus 受最終組裝與上游供應鏈），這層解鎖看「月產率能否穩定爬升」；② 引擎熱端零件（高溫合金鑄件良率、粉末冶金盤供應、shop visit 廠房與技師），這層解鎖看「LEAP/GTF shop visit 產能與良率」；③ 認證門檻（新變體如 737 MAX 7/10、新引擎如 GTF Advantage 都要 FAA/EASA 逐一認證、無法用錢買快），這層解鎖看「認證時程」。三層任一卡住，整條供給鏈就限速——這正是「想增也增不快」的結構本質，也是為什麼利潤會持續往最稀缺那層（目前是引擎熱端零件）集中。

資本週期替代指標：用三個替代訊號檢查資本是否理性而非過熱——① 引擎認證 lead-time（LEAP 2013→2016 EIS、GTF Advantage 2018→2026 EASA cert）顯示供給端擴張受認證節奏鎖死、非錢能買快；② Boeing FY25 capex／折舊 ~0.82x（投資低於折舊＝產能受限而非過度擴張）；③ Pratt 完整 GTF→Advantage 切換要到 ~2028，熱端零件瓶頸延續。三者一致＝「需求確定但合格產能爬坡受限」的理性擴張，非投機過剩，支持裁決 INTACT。

§6 產業經濟學與估值傳導

商航的估值傳導要分三段獨立看，因為三段的估值錨完全不同。OEM 主機廠（BA）：fwd PE 不適用（FCF 仍負），用 EV/EBITDA + FCF 拐點定價——BA 2026–2027 EBITDA 從翻轉 base 大幅成長，2028+ FCF 復甦可達 $8–10B（vs 當前負值），是 sector 最大 leveraged turnaround。引擎 OEM（GE）：fwd PE ~36x 對應 +12–15% 成長 = PEG ~2.5，溢價高但合理（售後收入 visibility 高、installed base 三倍）。售後零件（TDG）：fwd PE ~40x+ 反映 50%+ EBITDA + proprietary 定價權，是 sector 最高合理區。

Priced-in 檢驗：售後高 multiple 已 price 多少順風 信心：高
[F: GE ~36x fwd P/E、TDG ~40x+；兩者皆在歷史估值 band 高位] [I: reverse 推導——GE ~36x 對應市場已假設「售後黃金期延續 5+ 年 + CES margin 維持 26%+ + LEAP installed base 三倍如期」；TDG ~40x+ 對應「proprietary 定價 5–7%/yr + 機隊老化結構順風 + M&A optionality 全兌現」。換言之，多年順風已被 price 進去大半，容錯空間薄——任何售後增速放緩或 margin 回落都會觸發 de-rate]。
→ 推論：售後段「贏家很清楚但很貴」。GE/TDG 是品質溢價合理但非便宜；alpha 在「等回調」而非「追高位」。Boeing 則相反——估值（FCF base）尚未 price 翻轉成功，彈性大但 binary。

reverse 推導 GE 的 ~36x 拆成幾年順風：把 ~36x fwd P/E 對照 sector 中性引擎/工業 multiple（~18–22x），溢價約 +14–18x。要支撐這個溢價，市場必須假設 GE 在 2026–2030 維持 EPS CAGR ~15–20%——而這幾乎完全來自售後（CES 段 margin 26.4%、占利潤池主體），而非 OEM 引擎交付（引擎交付期初甚至壓低 margin，靠後續 shop visit 回收）。換算：~36x 約等於「市場已把 LEAP installed base 三倍 + 至少 5 年 shop visit 黃金期 + margin 不回落」三個假設全部 price 進去。這意味著 GE 的 upside 不在「售後黃金期會發生」（已 price），而在「售後黃金期比市場想的更長/更高」——這是個比舊版「fwd PE 35–40x 是 lower bound」更誠實的判斷：lower bound 只有在「售後超預期」時才成立，否則 ~36x 就是 fair value 上緣。

同理拆 TDG 的 ~40x+：TDG 的估值錨是「proprietary 售後零件 + 5–7%/yr 機械式提價 + 機隊老化結構順風 + M&A 滾動」。reverse 推導——~40x+ 對應市場假設「商用售後雙位數成長（current +16%）+ 提價權不被 OEM 自製 PMA 或反壟斷監管打斷 + 槓桿收購模式持續」。容錯點在第二項：若 OEM（BA/AIR）或航空公司加速採用 PMA 替代件、或歐美對 proprietary 提價啟動監管，TDG 的 multiple 會最先 de-rate。所以 TDG 是「最高品質但最依賴監管/競爭現狀不變」的標的——順風已 price 多年、裂縫在制度而非營運。
