§1 產業定義與邊界

本 ID 聚焦 panel-level packaging 的兩條獨立 S 曲線：（A）FOPLP（Fan-Out Panel-Level Packaging）——無 interposer、純 RDL 直連、由 OSAT 主導、量產時序提前；（B）CoPoS（Chip-on-Panel-on-Substrate）——含 interposer（silicon/glass）、TSMC 主導、量產時序滯後。兩者共用「panel format（rectangular substrate）」這個基礎結構，但服務的 chip tier、量產時序、玩家 profile 完全不同。本 ID 的 value-add 就是把這兩條曲線拆開分析，避免市場混淆。

灰色地帶（boundary 爭議本身就是 insight）

本 ID 識別三個重要灰色地帶，每個都是市場常犯混淆：

1. WMCM vs CoPoS — 都在 TSMC AP7 但完全不同：WMCM（Wafer-level Multichip Module）是 12-inch wafer-level，Apple A20 / M5 用，2026 年 60K wpm ramp；CoPoS 是 panel-level（310mm panel 起步），AI 客戶用，2028 才量產。媒體經常把兩者算成同一條 S 曲線——這是錯的。WMCM 屬於 InFO 升級路徑（Apple-driven），CoPoS 屬於 CoWoS 升級路徑（NVDA/AVGO-driven）。本 ID 嚴格區分。

2. FOPLP（無 interposer）vs CoPoS（含 interposer）— 服務不同 chip tier：FOPLP 適合 PMIC / RFIC / 中低階 ASIC（不需要 HBM 高速 I/O 整合），cost premium 低；CoPoS 必須含 interposer 處理 GPU+HBM 多堆疊高速互連，cost premium 高但 thermal & signal integrity 強。市場常把「panel-level」當作單一概念，但兩者的客戶、毛利結構、量產時序天差地遠。FOPLP 2026-2027 已經在 ramp（PTI / ASE）；CoPoS 2028 才開始——如果分析師沒分清，就會誤判 2027 年 panel-level 的真實受益者。

3. 玻璃 interposer vs 玻璃 core substrate — 兩者都用玻璃但層級不同：玻璃 interposer 是 CoPoS 內部的中間層（替代 silicon interposer，2028+ 才主流）；玻璃 core substrate 是 substrate 本體（替代 ABF 有機 core，Intel EMIB-T 已展示）。Corning / Schott / AGC 同時供應兩者，但對應的 BOM 量、毛利、客戶完全不同。一個玻璃 substrate 可以換多個 ABF 但仍承接 silicon interposer；玻璃 interposer 才是 ABF + silicon interposer 雙取代的終局。

三個灰色地帶都指向同一個結構性 insight：panel-level 不是 single technology，而是一個包含多條時序錯開、玩家不同的 S 曲線族群。

💡 §1 Insight：時序錯開讓 alpha 在不同年份有不同贏家
事實基礎：FOPLP 已量產（PTI 2017、ASE 2019），CoPoS 2028 才量產，玻璃主流要等 2028+，WMCM 2026 ramp（不是 panel）。推論邏輯：四條時序差最大 4 年的曲線被市場壓縮成「panel-level wave 2026-2030」一句話，因此每個年份的真實受益股不同，被「panel-level 概念股」籠統歸類就會錯買。投資意義：2026-2027 配置 PTI + ASE + ONTO + LPKF + Eternal；2028+ 才轉到 TSMC + Corning + Resonac。

§2 技術演進 + S 曲線 + 歷史類比

panel-level packaging 不是 2026 才出現的新技術——PTI 2017 年就已量產 PiFO（Pillar Integration FO），ASE 2019 年在 Kaohsiung 啟動 300x300mm FOPLP。但 2024 年之前 panel-level 一直是 mid-range ASIC / 邊緣應用，TAM 僅 $160M（Yole）。真正的「量產競賽」開始於 2025-Q3，當 TSMC 確認 CoPoS pilot 時程進入 AP7、Apple 拍板 WMCM at AP3、ASE 公布 K28 K28 facility、PTI 啟動 Youda Plant 5x reticle line——四個重大決策幾乎同時發生，把 panel-level 從 niche 推向 mainstream。本章解構這個演進如何發生、為什麼是現在、與哪些前例可比。

§3 核心技術棧拆解

panel-level packaging 把 12-inch round wafer 換成 rectangular panel（300x300mm 起，510x515mm 主流，600x600mm 終局）。看似簡單的「形狀變更」實際上挑戰整個製程鏈：lithography stepper grid 要重新校準、die placement 從 ±10μm 退步到 ±50μm（panel 越大越退）、warpage 從 wafer 級爆漲到 panel 級（CTE-induced thermal stress 與面積成 quadratic）、TGV（through-glass via）需要全新的雷射工藝。本章拆解六大子技術，並 ranking 良率卡點。

💡 §3 Insight：技術 kingmaker 不在 OSAT 也不在 TSMC，而在 LPKF / Corning / Onto
