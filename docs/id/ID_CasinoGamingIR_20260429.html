Casino + Gaming + Integrated Resorts

發布日：2026-04-29 ｜ 涵蓋 ticker：18 檔（5🔴 / 7🟡 / 6🟢）｜ 🟡 占比：18% ｜ 投資時鐘：Phase II 早中段（Macau 復甦 + Singapore 創高 + UAE 開幕 + 線上雙寡占成熟） ｜ industry-analyst v1.7 ｜ 兄弟 ID：嬰兒潮母題 · 連鎖旅館 · 郵輪 / 旅遊

Unit Glossary：GGR = Gross Gaming Revenue（賭場核心 KPI，扣除中獎賠付前的賭注總和）；IR = Integrated Resort（賭場 + 酒店 + 餐飲 + 表演 + 會議的整合度假村，新加坡 / 澳門 / 拉斯維加斯）；Mass / Premium Mass / VIP = 賭場 3 段客群：mass 大眾 / premium mass 高端中段 / VIP（高端 high-roller，受監管限制）；EBITDAR = EBITDA + Rent（賭場業常用，因部分 properties 是 lease）；Hold rate（線上）= 賭場保留 % of handle（DKNG 8.1% Q4 25 → target 10% by 2027）；Handle = 線上下注總額；MUPs = Monthly Unique Payers（DKNG 4.8M Q4 25）；ARPMUP = Avg Revenue per MUP（DKNG $139 Q4 25 +43%）；Sports betting = 體育博彩（合法 38 州 + DC，2025 美國 industry $16B+）；iGaming / Online casino = 線上賭場（合法 7 州 + DC：NJ / MI / PA / WV / CT / DE / RI）；Concession（澳門）= 6 大博彩執照（GEG / SJM / Sands China / Wynn Macau / MGM China / Melco）；RMB stimulus = 中國刺激政策推動 outbound 旅遊和澳門 GGR；Vision 2030（沙烏地）= MBS 國王經濟轉型計劃 / Qiddiya 賭場合法化 long-tail；UAE Wynn Al Marjan = 阿拉伯聯合酋長國首家 regulated 賭場（Q1 2027 開幕）。所有 financials = Q4 FY25 actual（截至 2026-04 已公告）。

§1 產業定義與邊界

Casino + Gaming + Integrated Resorts（IR）在本 ID 的範圍是實體賭場（含整合度假村 IR）+ 線上 sports betting + 線上 iGaming（線上賭場）+ 區域博彩四 sub-vertical 整合，跨美國 / 澳門 / 新加坡 / 中東 / 歐洲多地理。本 ID 與兄弟 ID 互鎖：嬰兒潮母題 ID 是 demographic 視角（高端 leisure），連鎖旅館 ID 是 hospitality 視角（hotel chain），本 ID 是 entertainment + gaming 視角。

邊界判斷的關鍵是「是否含 gaming + 大規模 hotel 整合」與「是否監管獨佔 / 開放」兩條軸：實體 IR（MGM / WYNN / LVS / CZR）+ 線上博彩（DKNG / FLUT）+ 中型 / regional（PENN / BYD / BALY）+ 國際 IR（Genting / 港股 6 大澳門）— 全部 in-scope；純 hotel chain（MAR / HLT 已在 ID_HotelChains）— out-of-scope；純 sports media / fantasy（pre-2018 era）— out-of-scope；vacation ownership / 純 timeshare — out-of-scope；blockchain / crypto gambling — out-of-scope（regulation 不確定）。

§2 Casino + Gaming 演進史 + S 曲線 + 歷史類比

美國 + 全球 Casino industry 過去 50 年歷經 4 個結構性轉折：(1) 1976-2000 是 LV 黃金期 + Atlantic City 興起 — 美國本土 dominance；(2) 2000-2014 是 Macau 開放（2002 第一批 concession）+ Singapore IR 興起（2010 開幕）+ 線上 fantasy；(3) 2014-2020 是 Macau VIP crackdown（2014）+ COVID 衝擊 + 線上 sports betting 開放（2018 PASPA repeal）；(4) 2020-2030 是 Macau 復甦 + Singapore 創高 + UAE 開幕 + 線上雙寡占成熟。本次循環最特別的是 實體 + 線上 + 新興市場（UAE / 沙烏地）三引擎同時加速。

本次循環的關鍵驅動：(a) Macau 復甦 — 2025 GGR ~85% pre-COVID + 2026 +8.3% forecast；中國中產 outbound + RMB stimulus + 高端 mass shift；(b) 線上博彩成熟 — DKNG/FLUT 雙寡占 78% market；hold rate 8.1% → 10% structural margin 升級；(c) UAE / MENA gaming 開放 — Wynn Al Marjan 2027 開幕 + 沙烏地 Vision 2030 long-tail；(d) Singapore 創高 — MBS Q4 EBITDA $806M「歷史最強」+ 監管獨家保護。本 ID 認為 sector 進入「Phase II 早中段」— Macau + Singapore + 線上同步加速 + 新興市場開幕，是 2026-2030 sweet spot。

★ 歷史類比（2 個正類比 + 1 個反類比）

正類比 1：2002-2010 Macau 開放（從 SJM monopoly 到 6 concession）。當年 Macau 從中國 outbound nascent → IR mega-resort 大興起，2002-2010 GGR CAGR +25%；中國經濟成長 + outbound 旅遊政策放鬆共同驅動。量化對照：2002-2010 Macau GGR 從 ~$3B → ~$23B (8x)；本次 UAE Wynn Al Marjan 2027 開幕 + 沙烏地 Vision 2030 long-tail 是「第二個 Macau 開放」— 預估 2030 MENA gaming TAM $30-50B（Wynn Al Marjan + Qiddiya + 其他）。

正類比 2：2010 Singapore IR 開幕（2 個 monopoly: MBS + RWS）。當年 Singapore 一夕成為亞洲第二大 IR 市場，MBS 開幕 5 年 EBITDA 從 0 → $1B+；新加坡監管獨家 + 印尼 / 印度 / 中國 outbound 三引擎。量化對照：MBS 2010-2015 EBITDA CAGR +50%+ / 2025 Q4 EBITDA $806M（年化 $3.2B）= 15 年 30x；本次 Wynn Al Marjan 預估 2028-2030 EBITDA $1.5-2B（vs MBS 5 年 $1B）— 更快爬升因 Wynn 品牌 + 高端定位 + UAE 經濟更大。

反類比：2018-2020 線上博彩「將取代實體」共識。當年 PASPA repeal 後市場曾期望線上博彩將快速取代實體賭場，DKNG IPO 2020 估值 P/S 達 15-20x。實際結果是 (a) 線上 + 實體共生而非 zero-sum；(b) 2025 Macau / Singapore 創歷史新高同時 DKNG / FLUT 也創高；(c) 整合 brand（BetMGM）反而是 hybrid moat。反類比警示：「新技術會取代舊」是錯誤觀念；應該認可 line-based + digital 不同 TAM。多條件對照：當年共識假設 (a) 客戶 100% 轉向線上，(b) 實體賭場 cycle 性衰退，(c) demographic 趨勢支持。實際 (a) 線上 + 實體分屬 leisure pyramid 不同層 — 線上是日常 entertainment / 實體是 destination experience；(b) Boomer + 嬰兒潮高端 demographic 在實體高端段強化；(c) 中國 + 中東 outbound demographic 強化實體 IR。

2. 反類比警示：「線上取代實體」共識是教科書錯誤；實際 line-based + digital 是兩個獨立 TAM，互補而非取代；BetMGM hybrid model 是 sector 中最完整 K 形 hedge 範例。

§9.5 Kill Scenario

反方論證 #1：中國 outbound 政策反向 — Macau 結構性受傷

路徑：2027-2028 中國新領導人或重大事件導致 outbound visa 收緊 / 反腐 2.0；Macau 2026 +8.3% 增長 reverse 至 -10-15%；MGM China / Sands China / Wynn Macau 等估值 -30-40%。當前證據：(a) 2014 反腐曾使 VIP -34%；(b) 中國目前刺激消費反向；但 (c) 中美緊張 + 不確定。量化影響：Macau operator -30-40%；MGM / WYNN / LVS 整體 -15-20%。falsification 事件：中國政策變化 / 2027 領導人換屆。最壞情境 drawdown：sector basket -20-30%。

反方論證 #2：UAE Wynn Al Marjan 開幕延遲 / 規模低於預期

路徑：建設延遲 6+ 個月 / 監管不確定 / 中東地緣動盪。當前證據：(a) 2025-12 hotel topped out；但 (b) 大型 IR 開幕通常延遲；(c) 中東地緣風險。量化影響：WYNN 2028-2029 EBITDA -50% vs base case；估值倍數 -2-3x。falsification 事件：建設進度 / 開幕時間。最壞情境 drawdown：WYNN -15-20%。

反方論證 #3：線上 sports betting hold rate 不升 / 受監管壓制

路徑：DKNG / FLUT hold rate 卡在 8% 不升至 10%；新州 立法 hold rate cap / 廣告限制。當前證據：(a) 部分州（IL）已加稅；但 (b) DKNG / FLUT 都 commit 10% target。量化影響：DKNG / FLUT FY27 EBITDA -25-35% vs base case。falsification 事件：每季 hold rate 數字。最壞情境 drawdown：DKNG / FLUT -25-30%。
