§1 產業定義與邊界

Discount Retail 在本 ID 的範圍是美國折扣零售兩大 sub-sector 整合分析 — Off-Price（TJX / ROST / BURL / OLLI / Five Below niche）+ Dollar Store（DLTR / DG）。本 ID 的核心切入點是不應把兩個 sub-sector lump together — 兩者對應 K-shape 兩端不同消費者、結構模型截然不同、關稅曝險完全相反、AI 時代的護城河來源也分歧。

邊界判斷的關鍵是「是否核心消費者是 trade-in / trade-down / 結構性低收入」與「商業模式是 closeout treasure hunt 還是 dollar-point essentials」兩條軸：Off-Price（TJX 旗下 TJ Maxx + Marshalls + HomeGoods + Sierra + HomeSense / ROST + dd's Discounts / Burlington / Ollie's / Five Below）— 全部 in-scope；Dollar Store（DLTR 旗下 Dollar Tree + 已售 Family Dollar / DG / 99 Cents Only 已破產）— 全部 in-scope；Mass merchant（Walmart / Target / Costco）— 部分 in-scope 作為比較 reference；Department store（Macy's / Kohl's）— in-scope 作為「market share donor」對手；純 e-commerce（Amazon / SHEIN / Temu）— out-of-scope（不同模式）；Off-price 海外（TJ Maxx Europe）— 部分 in-scope 含於 TJX。

💡 §1 Insight
1. Off-Price 與 Dollar Store 是兩個結構截然不同 sub-sector — sector ETF 是錯誤工具：客群（中高收入 vs 低收入）/ 商業模式（treasure hunt vs essentials）/ 關稅曝險（90% 第三方 vs 直接進口）/ AI 護城河（物理 immune vs digital 部分受擠壓）。最佳投資工具是 K-shape barbell（TJX + DLTR）而非 sector basket。
2. Department store 是 sector 中最 confirmed 結構性 loser：Macy's 27% dept 客戶支出 → TJX (vs 25% prior) 是 1pp YoY 加速；2030 預估 35-40%；Kohl's mid-tier stuck；M / KSS short thesis 持續有效。
3. Big Lots Chapter 11 2024 + 99 Cents Only 退場 = 給 OLLI / FIVE / DLTR 留下 share gain 空間：weak operator 退場是 dollar / extreme value sub-sector 第二受益點。

§2 技術演進 + 商業模式 + 歷史類比

Discount retail 從 1976 TJ Maxx 開店至今 50 年經 4 個明確階段。Phase 0（1976-1990）是創立期 — TJX 1976 開第一家 TJ Maxx / Ross 1982 / Burlington 1972 / DG 1939 / Dollar Tree 1986；都是地區性玩家。Phase 1（1990-2010）是規模化期 — TJX 全國擴張 + 國際進軍（Marshalls 收購 1995 / HomeGoods 1992 / TK Maxx UK 1994）；DG / DLTR 經 IPO 後 1990s-2000s 規模化；2008 金融危機是第一個重大 share migration（中收入 trade-down + Macy's 第一波結構壓力）。Phase 2（2010-2020）是百貨死亡 + e-commerce 共存期 — TJX/ROST 對 e-commerce「免疫」（treasure hunt 線上難複製）；2010-2020 Macy's / JCPenney / Sears 連年關店；DG 規模建造（Vasos 2008-2018 任 CEO 爆發期）；DLTR 2015 收購 Family Dollar（後證實是錯誤決定 → 2025 售出）。Phase 3（2020-2030）是 K-shape + 關稅 + AI 時代 — 通膨 + SNAP 收緊 + 中收入壓力 + 關稅「2025-2026 trade war 2.0」+ Apple Intelligence Sherlock 母題。

當前位置（2026-04-30）= Phase 3 早中段。三明確訊號：(1) Macy's 27% dept 客戶支出已流向 TJX（vs 25% prior 是 1pp YoY 加速）[T2: Retail Dive Macy's 27%]；(2) DLTR Family Dollar 售出（July 2025）+ multi-price 5,300 stores 16% 銷售 = pure-play 規模化新階段[T1: DLTR Q4 2025 PR]；(3) DG turnaround 進度 — op margin 5.2% → 6.1% Q4 + DG Media Network FY25 $170M[T1: DG Q4 2025 Transcript]。

歷史類比（QC-I14 + QC-I28 含正反例）

正類比 #1（百貨死亡 ≈ 2003-2008 大型百貨第一輪 collapse）：1990s-2000s Sears / JCPenney 已開始衰落但 2008 金融危機才真正崩塌；2010-2020 Macy's / Kohl's 持續關店；2020-2030 是「第二輪 collapse」最後階段 — 零售業 mall anchor 模式結構終結。Off-price 從中受益 + dollar store 受 lower-income trade-down 受益。差異：當前已不是 cycle 而是結構終局。

正類比 #2（K-shape 強化 ≈ 1970s-1980s stagflation 期 discount 興起）：1970s 通膨 + 失業率高 = TJX / Ross / Walmart 創立期 + 規模化機會；同樣 2025-2030 通膨 stickiness + SNAP 收緊 + 關稅 = K-shape 強化；DLTR / DG 受益（低收入剛性需求）+ TJX / ROST 受益（中高收入 trade-in 找價值）。

反類比 #1（DLTR ≠ 必然成功的 Walmart 1990）：Walmart 1990 是規模 + EDLP + 供應鏈三同時 step-function；DLTR 2026 是 multi-price + Family Dollar 售出 + 通膨 tailwind 三 partial step；本 ID 認為 DLTR 估值倍數 step-up 30-40% 是合理但不會像 WMT 1990-2000 10x。

反類比 #2（DG turnaround ≠ 必然成功的 1995 Walmart Sam Walton 回歸）：Vasos 2008-2018 第一任建造規模 + 2024 回任是 Sam Walton 回歸 narrative；但 DG 2018-2024 期間 Vasos 缺席時遭遇 SNAP 收緊 + Walmart Q1 2026 加碼 rural + Amazon Rural Prime 三同時 — 結構天花板比 Walton 1995 嚴峻。本 ID 認為 DG turnaround real but capped — Phase II recovery 不是 breakout。

反類比 #3（Off-price ≠ 完全 AI immune）：物理 treasure hunt 不可被 agentic AI 直接替代；但 Apple Intelligence + ChatGPT 有可能 sherlock TJX 的「app inventory tracking」 / 客戶 loyalty 部分；2027-2030 SoFi-style AI 推薦 + 行動 app 整合是 off-price 必須投入的 capex；不是 0 risk。

💡 §2 Insight
1. Phase 3（2020-2030）是 sector K-shape 強化 + 百貨死亡終局 10 年最強 share migration 階段：對應 TJX / ROST / DLTR 估值倍數 step-up 30-50%；M / KSS short thesis 持續有效。
2. 1970s discount 興起是當前最佳歷史類比：通膨 + 失業 + 中產壓力 = discount 黃金期；但 2025-2030 強度比 1970s 更高（K-shape 比 stagflation 結構更深）。
3. DLTR multi-price 規模化 vs DG turnaround 是兩個結構不同 trade：DLTR 是「pure-play 升級」/ DG 是「Phase II recovery capped」；DLTR 估值倍數 step-up 比 DG 大。

§3 商業模式深度拆解

Discount retail 兩大 sub-sector 商業模式拆解 — 同樣是「便宜」但 unit economics 完全不同。

Off-Price 模型（TJX / ROST / BURL）

採購模型：closeout-driven — 製造商 / 品牌商過剩、退貨、季末 inventory；TJX 90% 採購來自第三方 vendors（vendors 處理關稅 + 海運）[T2: Retail Dive 90% third-party]；不固定 SKU + 隨機到貨 = treasure hunt experience。定價模型：MSRP -20% 至 -60%；不打折 (no markdown) — 因為 inventory 本身已是低價買進。毛利率：TJX FY26 guide pretax 11.3-11.4% / Q4 FY25 11.6%；ROST FY26 op margin 12.0-12.3%；Burlington 略低。客流模型：高頻次 visit（avg 14-20x/year vs Macy's 3-5x）；低 basket size（$30-40 vs $80-120 Macy's）；treasure hunt 創造 dopamine + 重複造訪 moat。商店規模：~25,000-30,000 sq ft 標準店；TJX 5,000+ stores、ROST 1,800+、Burlington 1,000+。FY26 新店：TJX 146 net new、ROST 110、Burlington 110[T2: Retail Dive 2026 store growth]。

Dollar Store 模型（DLTR / DG）

採購模型：直接進口 + 私有品牌 + 部分 closeout；DLTR 約 40% 來自進口（直接受關稅影響）；DG 約 30% 進口；vs off-price 90% 第三方 vendors。定價模型：固定價格點 — DLTR 多年 $1.00 / 2021 起 $1.25 / 2024 multi-price $1.50/$3/$5；DG 從 $1-$10 不等多價 essentials。毛利率：DLTR Q4 2025 39.1% (+150bps)；DG Q4 op margin 6.1%（vs 5.2% prior, recovery）；明顯比 off-price 厚但 OPEX % 較高。客流模型：低 frequency / 中 basket / 急用模式 — DG 平均客每月 2-3 次 / DLTR 1-2 次；rural footprint（DG 75% 店在 20K 以下人口城鎮）；地理覆蓋 + 「小急購」moat。商店規模：~7,500-10,000 sq ft 小型店；DLTR 8,000+ Dollar Tree（已售 Family Dollar）/ DG 20,000+。FY26 新店：DLTR 適度 + multi-price 轉換；DG 適度 + 國際首推墨西哥。
