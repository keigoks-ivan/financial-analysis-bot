氫能 / 燃料電池（Hydrogen / Fuel Cell）

發布日：2026-05-05 ｜ 涵蓋股票：14 ｜ Phase I（CAPEX 爆發）｜ 🟡 占比：18% ｜ mega=energy / sub_group=hydrogen

單位 Glossary（QC-I20）：LCOH = $/kg H2（life-cycle 全成本，含 OPEX+CAPEX 攤提）｜ Electrolyzer CAPEX = $/kW（DC input power 計）｜ Fuel cell stack ASP = $/kW（system level，非 stack-only）｜ TAM 口徑 = total H2 production value $/yr（含 grey/blue/green）｜ Revenue 口徑 = annual run-rate（FY26E）｜ 重卡單位 = FCEV Class 8 units／年；§4 全球氫產量 = Mt/yr（百萬公噸）。

§0 TL;DR — 一句非共識 thesis

⚠ QC-I7 T1 來源警語：本報告 T1 來源占比約 53%（≤ 60% 強制門檻），主因綠氫 LCOH / electrolyzer ASP / 全球綠氫產量等核心數字大量依賴 IEA / IRENA / BloombergNEF / Hydrogen Council 等 T2 機構研究而非公司一手 IR；fuel cell 段（BE / PLUG / Cummins）T1 比例高，但氫經濟段（LIN / APD / NEL / ITM）T2 主導。讀者使用本 ID 結論時應視結構性 thesis（§12）為「機構 + 政策觀點綜合」而非單一公司可驗證事實，並對 §4 TAM / §7 LCOH 數字保留 ±20-30% 不確定區間。
本 ID 一句 thesis：氫能 2026 是「補貼已成立但 LCOH 仍高、玩家結構未定」的 Phase I 早期；市場誤把「綠氫不可避免」與「PLUG 將起死回生」捆綁，但這兩件事相互矛盾 — 綠氫真的擴產會拉爆 electrolyzer ASP 戰，純 play 商（PLUG / FCEL）反而最受傷；真正受益是工業氣體寡占（LIN / APD）+ SOFC 24/7 power 的 BE，前者收 capex 過路費、後者卡進 AI 數據中心 prime power 縫隙。BE-AEP 1GW deal 是 fuel cell 從「被拋棄類股」轉「AI power 二供」的轉折點；PLUG / FCEL 在 45V $3/kg 落實後 12-18 月內若 LCOH 仍 $4-5/kg，going concern 風險將在 FY27 H1 引爆。

💡 §0 核心 Insight
市場把「氫能」當成單一賭注（boom-or-bust），但氫能其實是三個獨立子賭注：(a) 綠氫擴產（賭 45V + LCOH 跌）→ 受益是 LIN / APD / Linde-style 工業氣體；(b) Fuel cell stationary power（賭 AI DC prime power 縫隙）→ 受益是 BE 一家獨秀（SOFC 95% 效率對手少）；(c) Fuel cell mobility（賭重卡 / 物料搬運）→ 大概率 lose 給 BEV，PLUG / FCEL / Hyzon 是 zombie；分開買才能避免 thesis 互相對沖。

§1 產業定義與邊界

「氫能 / 燃料電池」涵蓋從製氫 → 儲運 → 終端應用三段完整鏈條，但範圍邊界長期混淆。本 ID 嚴格區分氫的顏色（製程碳排量 proxy）與氫的用途（工業 vs 能源 vs 載具），兩者交叉是 thesis 真正分水嶺。

顏色分類最重要：灰氫（grey，SMR 天然氣重組無 CCUS）佔 2026 全球 95Mt 產量的 ~62%，主要供化肥與煉油，碳排 9-10 kgCO2/kgH2，雖大但不在「能源轉型」敘事內[T2: IEA Global Hydrogen Review 2025]。藍氫（blue，SMR + CCUS >90% 捕碳）2026 商用量 ~2 Mt，代表案例為 Air Products NEOM / Louisiana clean H2 facility（11 億美元 capex）[T1: APD Investor Update Q1 FY26]。綠氫（green，再生能源 electrolysis）2026 全球 <1 Mt，但裝機 electrolyzer capacity 已達 5+ GW，呈陡峭 ramp[T2: IEA 2025 Hydrogen Review p.42]。其他顏色（粉氫核能 electrolysis、青綠氫天然氣熱裂解）量極小，本 ID 歸入「其他 <1%」。

用途上，本 ID 聚焦能源用途（fuel cell stationary power / mobility / 工業熱解 / SAF 合成燃料），同時將傳統化工用途（化肥合成 ammonia 80%、煉油 hydrocracking 18%）列為 out-of-scope；此切分原因是後者已有 100 年市場、需求曲線跟 GDP 走，跟「能源轉型」thesis 無關。但化工用途與綠氫共用基礎設施（electrolyzer / 儲存 / pipeline），所以工業氣體公司 LIN / APD 同時觸及兩端，本 ID 仍納入。

💡 §1 邊界 Insight
「綠氫」與「fuel cell」雖同屬氫能 ID 但是近乎獨立的兩個 TAM — 綠氫主要驗證能源轉型 thesis（取代化石燃料），fuel cell 主要驗證電化學 thesis（取代內燃機 / 取代電池）。Bloom Energy（BE）的核心 thesis 是 SOFC 直接燒天然氣（不需綠氫），跟綠氫 TAM 無關；PLUG 跨兩端但綠氫端做不大、fuel cell 端 PEMFC 沒競爭力，是 thesis 兩面挨打的代表。投資意義：買 BE 不應該看 IRA 45V 進度（只看 AI DC PPA 進度）；買 LIN / APD 不應該看 fuel cell 重卡銷量（只看 electrolyzer GW build-out）— 把混雜邏輯拆開才不會被亂敘事帶偏。

§2 技術演進史 + S 曲線 + 歷史類比

氫能不是新技術，是三度起飛三度失敗的「老兵」。第一波 1970s 石油危機後（卡特政府氫經濟構想），第二波 2000-2005（Iceland 氫經濟、布希氫高速公路、Honda FCX），第三波 2015-2018（豐田 Mirai 量產、加州氫補貼）；前三波皆在「LCOH 高、加氫站不足、加上石油價格回穩」三因素下夭折。本次第四波（2021-）有三個結構性差異讓本 ID 視為「不一樣」：(a) 政策力度 — IRA 45V $3/kg 是史上最大氫補貼，等同把美國綠氫 LCOH 從 $5-7/kg 砍到 $1-2/kg；(b) 應用場景轉移 — 從乘用車（前三波焦點）轉向 stationary power（AI DC）+ 重卡 + SAF；(c) 工業需求拉動 — EU 鋼鐵 / 化肥 ammonia 強制脫碳，需求側 mandate 比補貼更硬。

歷史類比 1（正類比）：太陽能 LCOE 2010s 跌 90%

2010 太陽能 LCOE 約 $359/MWh，2020 降至 $40-50/MWh，跌幅 88%；驅動因子是中國產能擴張 + Wright's Law（每倍出貨成本降 ~22%）+ 補貼托底。今日 electrolyzer 也呈類似結構 — 中國 Longi / Sungrow / 大連化物所大規模產能（2025 中國 alkaline electrolyzer ASP $300-400/kW vs 西方 $800-1,200/kW）[T2: IEA Global Hydrogen Review 2025 p.78]。本 ID 預估 electrolyzer ASP 2030 跌至 $400/kW（西方）/ $200/kW（中國），LCOH 降至 $1.5-2/kg[T2: Hydrogen Council 2024 Insights p.32]。類比限制：太陽能光子是免費「燃料」（陽光），氫能 feedstock（電力 + 水）有 OPEX，所以 LCOH 永遠不可能跌到太陽能那麼接近零；本 ID 修正類比為「fuel cell stack 走太陽能 panel 路徑（CAPEX 跌 90%）但 LCOH 仍受電價綁架（不會像 LCOE 跌到 $20/MWh）」。

歷史類比 2（正類比）：LNG 2010s 從利基到主流

2010 全球 LNG 貿易量 290 Bcm，2020 達 488 Bcm（+68%），由 shale 革命 + 日本 Fukushima 後核電減少 + 歐洲依賴俄氣切換驅動；LNG 不是 1995 才出現的新技術，但因供需錯配 + 政策需求產業鏈在 10 年內完整重組（liquefaction + LNG 船 + regasification 三段）。氫能 2026-2036 結構極類似：政策需求（脫碳）+ 供需錯配（中東 / 澳洲低成本 vs 歐日韓需求）+ 產業鏈重組（electrolyzer + ammonia carrier + 氫到氨重組設施）[T2: Bloomberg NEF LNG Outlook 2024]。類比限制：LNG 終端 burner / 渦輪本來就在用，氫的終端（fuel cell / 氫渦輪）還在驗證；故氫的「LNG 路徑」會晚 LNG 約 5 年。

歷史類比 3（反類比）：Iceland 2002-2005 氫經濟失敗

2002 冰島宣布「全球第一個氫經濟國」計畫，由 Reykjavik 公車隊（3 台 fuel cell bus）+ 1 座加氫站試點；DaimlerChrysler、Norsk Hydro、Shell 投資。2005 計畫實質中止：原因為 (a) 公車運轉成本是柴油 4-5 倍，(b) 加氫站維護成本高且故障頻繁，(c) 基礎設施擴展沒人付費。本 ID 與 Iceland 失敗的差異：(a) 本次有 IRA 45V $3/kg 補貼托底，Iceland 無類似力度補貼；(b) 本次主場景是 stationary power 與重卡（不是公車 / 乘用車），fuel cell + reformer 不需公共加氫站；(c) 本次中國 + 歐盟 + 美國三地同步，不是冰島單一小國孤注。但風險仍類似：若 LCOH 在 2028 年仍 > $4/kg、且 BEV 重卡同時量產，氫重卡賽道會重演 Iceland — PLUG / FCEL / Hyzon 都可能變成現代 Reykjavik bus depot。[T2: Wikipedia Hydrogen economy（hist）]

歷史類比 4（反類比）：3D TV 標準失敗（QC-I28 必備反類比）

2010-2014 TV 廠（Samsung、LG、Sony）力推 3D TV，HDMI 1.4 + Active/Passive shutter 標準齊備、影院 3D 普及度高，但 2014 後消費者拒買、Netflix 等內容商棄做、2017 全產業認賠殺出。失敗原因：供應端推力（廠商）大但需求端拉力（消費者）小。本 ID 警惕：氫能重卡（heavy truck）目前同樣是OEM 推力（Cummins / Daimler / Volvo / Hyundai 投產）大但 fleet operator 拉力小（沃爾瑪 / 亞馬遜實際採購極少）。fuel cell 物料搬運（Walmart 已採購 38K+ PLUG GenDrive 堆高機）是唯一已 cross-chasm 的應用，但金額小（year ~$200M）[T1: PLUG 10-K FY25 p.18]。反類比警示：若 2027 重卡 fleet 採購量仍 < 5,000 units/yr（FY26E 已 ~4K），氫重卡賽道應視為失敗，全產業重新定價。

💡 §2 S 曲線 Insight × 3
① 氫能 2026 的 S 曲線真實位置不是「Early-Mid」是「重新嘗試 Phase I」 — 前三波都是「Phase I 失敗倒退」，本次有 IRA 45V 與 EU REPowerEU 政策托底所以不會立刻倒退，但 LCOH 仍未跌破 cost-competitive 門檻，故是「Phase I 中前段」而非市場部分敘事的「Phase II 起飛期」。投資意義：CAPEX 受益者（LIN / APD / electrolyzer 製造商）先吃到，downstream 應用受益者（fuel cell OEM / FCEV 車廠）至少要等到 2028+ 才能對股價兌現。

② 真正的 cost down 路徑是「中國 + Wright's Law」不是「美歐補貼」 — 太陽能 LCOE 跌 90% 主要來自中國 panel 量產，補貼只是托底；氫能也會走相同路徑。中國 Longi / Sungrow electrolyzer ASP 已是西方 1/3，2030 全球 ASP 下緣很可能由中國設定。投資意義：純美國本土製造 thesis（PLUG）會被中國產能價格戰擠壓；Linde / APD 因為「賣氫氣 / 賣 service」不直接受 electrolyzer ASP 戰影響，反而更安全。

③ Iceland 2005 + 3D TV 2014 兩個反類比警告同一件事 — 「供應端推力大、需求端拉力弱」。氫重卡 / 氫乘用車目前都呈此特徵；BE SOFC stationary power 是唯一已驗證需求端有拉力的賽道（AEP 1GW + Equinix 100MW + AWS PPA 為證）。投資意義：把氫能 thesis 拆成「supply-push」與「demand-pull」兩籃，重倉後者（BE / LIN / APD）+ 輕倉甚至 short 前者（PLUG / FCEL）。

§3 核心技術棧拆解

氫能技術棧分三大層 × 多子技術：上游製氫（PEM / Alkaline / SOEC electrolyzer）、中游儲運（高壓氣態 / 液氫 / 氨 / pipeline）、下游應用（PEMFC / SOFC / 氫渦輪 / 氫 ICE）；每層都有 kingmaker 子技術，且不同子技術的良率瓶頸與專利持有者高度集中。

上游製氫 — Electrolyzer 三路線競賽

Electrolyzer 是綠氫成本決定者。PEM（Proton Exchange Membrane）：Nel ASA / ITM Power / Cummins Accelera / Plug Power 主導，特點是響應快（適合波動再生能源）、CAPEX 高（$1,200-1,500/kW 西方）、貴金屬 Pt/Ir 用量大；2030 ASP 預估降至 $700/kW[T1: ITM Power Investor Presentation FY24]。Alkaline：Nel / John Cockerill / 中國 Longi / Sungrow / 大連化物所主導，CAPEX 最低（$300-500/kW 西方，$200-300/kW 中國），但響應慢、適合 baseload 電力配對；是中國市場 ~75% 主流路線[T2: BloombergNEF Hydrogen Outlook 2024 p.45]。SOEC（Solid Oxide Electrolyzer）：Bloom Energy（BE）獨霸（全球唯一商用 SOEC + SOFC 雙向設備）、Topsoe / Sunfire 二線；高溫運行（700-850°C）效率最高（85-90% LHV vs PEM 65-70%）但啟停慢、堆疊壽命短[T1: BE Q4 FY25 10-K p.14]。BE 在 SOEC 領先地位是 §11 把 BE 列 🔴 核心的核心理由。

中游儲運 — 物理路徑分歧

氫儲運是純物理問題：氫密度極低（標準狀態 0.09 kg/m³），需高壓 / 液化 / 化學鍵合三選一。高壓氣態（350 bar 重卡 / 700 bar 乘用車）：Toyota / Honda / Hyundai 採用，碳纖維罐成本高（$400-800/kg H2 capacity）；專利持有者 Hexagon Composites（Norway）、Faurecia（法）、ILJIN（韓）[T2: Hexagon Purus IR Q4 FY25]。液氫 LH2（-253°C）：Linde / Air Products / Iwatani 三家寡占（liquefier 技術門檻極高），Air Products NEOM 巨型 cryogenic facility 是當前最大商用案例。氨 ammonia carrier：FY26 突起的暗黑路徑 — 氨儲運成本是氫的 1/4-1/3，但需在終端裂解回氫；日韓 Yara / Mitsubishi 大力推進，2030+ 主流跨洋運輸路徑。Pipeline：歐洲計畫 2030 完成 23,000 km 氫專用 backbone（Hydrogen Backbone），但鋼材氫脆問題（hydrogen embrittlement）使老天然氣管線不能直接用，需新建或內襯 polymer，本地儲運唯一規模化路徑[T2: European Hydrogen Backbone 2024]。

下游應用 — Fuel Cell 雙路線

PEMFC（質子交換膜）：低溫（80°C）、響應快、適合 mobility（重卡 / 乘用車 / 物料搬運）；Toyota / Hyundai / Cummins / Plug 主導；瓶頸是 Pt 貴金屬 + 膜電極組（MEA）耐久性 + bipolar plate 量產；BLDP（加拿大 Ballard）是 stack 技術領先者但商業化慢。SOFC（固態氧化物）：高溫（700-850°C）、效率最高（55-65% 電效率，搭配熱回收 80-90% 總效率）；BE 全球獨家商用、Doosan FC 韓股二線、三菱重工技術儲備中；瓶頸是材料（YSZ 電解質、LSCF 陰極）+ 啟停壽命；最契合 24/7 stationary power 場景，這是 BE-AEP 1GW deal 的物理基礎[T1: BE Q1 FY26 IR]。

💡 §3 技術棧 Insight × 2
① BE 的 SOEC + SOFC 雙向技術是「fuel cell 界的 ASML」雛形 — SOEC 製氫 + SOFC 用氫，同一電池 stack 可以正反運行，全球商用唯一一家；技術門檻 + IP 集中度極高（10+ 年積累 + 200+ 專利）；2026 BE 是 fuel cell 唯一非 zombie 的純 play，剩下 PLUG / FCEL / BLDP 都在 burn cash；投資意義：BE 估值倍數應比 fuel cell 同業高 3-4x（FY27E EV/Revenue 6-8x vs PLUG 1-2x）。

② 真正的 kingmaker 是「氨 carrier 路徑」不是 H2 pipeline — H2 pipeline 因氫脆問題（hydrogen embrittlement）+ 沿線土地紛爭，2030 前歐洲 23,000 km backbone 大概率 delay 50%+；反觀氨 NH3 是已成熟海運商品（化肥用），跨洋運輸成本是氫的 1/4，FY27 起日韓 Yara / Mitsubishi 將大規模採購中東 / 澳洲綠氨，再在終端裂解。投資意義：氨碳基礎設施（liquefier、cracker、port）受益者反而是 Yara、Mitsubishi、APD 而非氫 pipeline 純 play。

💡 §4 TAM Insight × 3
① TAM 預估歷史 100% 高估綠氫，本 ID 對齊保守口徑 — 2017 / 2020 / 2021 三次預估都被打臉 7-40x；本 ID 取 2030 綠氫 25Mt（vs Hydrogen Council 50Mt）並把對應 fwd multiple discount 50%。投資意義：對 PLUG / FCEL 估值不應採 sell-side 50Mt 假設；對 BE / LIN / APD 因業務多元，TAM discount 影響較小。

② 中國拿走 50%+ TAM，西方上市公司 SAM 只有 35-45% TAM — alkaline electrolyzer 中國本地玩家（Longi / Sungrow / 大連化物所 / 四川天威）2025 全球出貨佔 60%+；西方純 electrolyzer 廠（Nel / ITM）已連年虧損。投資意義：避開純 electrolyzer 製造 thesis（沒護城河）；轉向「賣 service / 賣氫氣」端（LIN / APD）或「SOFC 獨家」（BE）。

③ SAF $80-120B TAM 是被低估的氫能下游 — IATA 強制 SAF 滲透率 2030 達 5%（EU ReFuelEU 6% / 美 SAF Grand Challenge 10% by 2030）；SAF 是 power-to-liquid（綠氫 + CO2 → e-jet fuel）的第一個強制需求 mandate。投資意義：間接受益是 LIN / APD（提供綠氫 + 工業氣體）；非常 non-obvious 但 LCOH 高位反而是 SAF 的友軍（航空業願意付綠氫溢價，化工 / 化肥不願意）。

§5 供應鏈 Value Chain

議價權近 5 年轉移史（QC-I19 定性表述）

① 利潤池正在從「fuel cell pure play」流向「工業氣體寡占」 — LIN / APD 的 $/kg H2 賣斷模式 + 5-10 年長約鎖價，享受 45V 補貼直接落入毛利；fuel cell 玩家賺 stack ASP 但同時受 ASP 跌價戰；2026-2030 LIN / APD 產業 EV / EBITDA multiple 應從 18-22x 升至 24-28x（重估為 decarbonization platform）。

③ SAF mandate 是 2030 前最被低估的下游 puller — IATA 強制 SAF 滲透率 + EU ReFuelEU 6%、美 SAF Grand Challenge 10% 兩個 mandate 不可逆，e-jet fuel 是綠氫的高溢價終端。投資意義：LIN / APD 接 SAF 廠（如 LanzaJet）的綠氫供應合約，2027-2030 H2 收入有結構性 +20-30% upside；Honeywell UOP（HON）作為 SAF reactor 催化劑供應商是 non-obvious 受益者。
