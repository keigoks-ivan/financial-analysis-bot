★ 本 ID 核心非共識 Thesis 市場用「中國消費循環」框架看 Anta / Li-Ning vs Nike / Adidas — 認為 China 復甦就會帶動 Nike China 反彈。本 ID 認為：① Anta Group 2025 中國市占 23.0% > Nike 20.7% 是 6 年永久結構性反超非循環（Anta Group 2018 12.5% → 2025 22.9% +10pp；Nike Greater China FY19 $6.2B = FY25 $6.6B inflation-adjusted -25%）；② Anta 真正 alpha 不是主品牌（+3.7%）也不是 FILA（+6.9%）而是 "Other Brands" segment +59.2%（Descente/KOLON/Maia/Jack Wolfskin 合計 RMB 17B 已超 Li-Ning 全集團 RMB 29.6B 的 60%；Op margin 28% 高於 Anta brand 21% & FILA 26%）；③ Amer Sports（Anta 持股 44.5%）是 Anta 的全球化載體，Salomon $2B + Arc'teryx $2B 雙過 USD 2B 是 Anta 全球版 FILA 案例（FILA 2009 從 USD 0.06B → 2024 RMB 26.6B 44x，市場仍視 AS 為獨立 luxury technical apparel）；④ Topsports 12.99% 股息殖利率是「Nike China 衰退結構化」最直接交易工具（FY25 RMB 27B / FY26 H1 profit warning，但 dividend cushion 已 price in 衰退至少 2 年）。
💡 §0 4 條核心 insight（預告）
1. Anta Group 23% > Nike 20.7% 已是 confirmed 數據（Euromonitor 2025），不是預測 — 過去 12 個月主流券商從「Nike will recover」轉為「Nike China zero growth = baseline」共識成形。
2. Anta 「Other Brands」三段成長路徑：FILA 2009-2024 15 年走完從 RMB 0.6B → 26.6B 44x；Descente / KOLON 2024-2026 進入 fast scale 階段（合計 RMB 10B+）；Maia Active / Jack Wolfskin 2026-2030 是 next leg。Anta 模式 = 私有化 → 中國本地化 → 全球反向擴張。
3. Adidas 是「中國 turnaround 個股」但不是 Anta 路徑：Gulden 2023 接手後 GC 連 7 季 +ve（FY24 +10% / FY25 H1 +13/+8/+6 decel），與 Anta 結構性 share gain 邏輯不同 — Adidas 是「重拾流失 share」，Anta 是「永久切走 Nike share」。
4. Topsports 12.99% 殖利率是 Nike China 衰退最直接的交易工具：股價 HK$2.78 / Mkt cap HK$17.36B / FY26 H1 profit warning + dividend yield ~13%；對應 Nike Greater China 連 6 季 -ve，Topsports 失去通路議價權的結構性訊號。

★ 灰色地帶

  • FILA China：是 Anta 的還是國際品牌？：FILA 商標權屬韓國 FILA Holdings；Anta 2009 買的是「永久經營權 + 商品開發權 in Greater China」。FILA China 收入計入 Anta 集團 segment，但 FILA 全球品牌資源（米蘭 design / 義大利 heritage）仍是國際 asset — 本 ID 將 FILA 列「Anta 反向收編高端」段。
  • Amer Sports：是 Anta 的還是獨立公司？：Anta 持股 44.5%（最大股東）+ 創辦人 Chip Wilson + 騰訊 + 方源資本 = consortium 結構。法律上 AS 是 NYSE 獨立公司，但 Anta 享受 Salomon / Arc'teryx 中國 +42% 高速成長的 partial economics — 本 ID 將兩者並列。
  • Lululemon China +28% 算「高端段」還是「外國品牌」？：商業上是外國品牌，但定位是高端 athleisure，與 Anta + FILA + Arc'teryx 高端段同質。本 ID 將 LULU 列「對照」段（高端段唯一倖存的非中國品牌）。
  • Topsports：是「品牌」還是「通路」？：法律是品牌零售商，商業是 NKE/ADS 主要中國代理通路。本 ID 將其列「對照」段，作為 Nike China 衰退的 read-through 工具。
  • Adidas China：算 turnaround 還是循環反彈？：本 ID 視為 turnaround（Gulden 2023 接手 + 道歉 + 本地化），但其 GC FY25 已從 +13% → +8% → +6% decel，意味 turnaround 黃金期可能已過。
💡 §1 Insight
1. 「中國本土運動」不是單一概念：至少 4 個結構性差異子象限（國潮 / 細分 / 反向收編 / 通路）— 用單一「Anta vs Nike」框架掩蓋了內部 K-shape 真實差距（Anta Other +59% vs Anta 主品牌 +3.7% 差 55pp）。
2. Anta 跨 3 個 sub-segments（國潮主品牌 + FILA 中端 + Other 高端）+ 透過 Amer 跨第 4 段（全球技術服飾）— 是其 fwd PE ~13x 仍被視為「中國運動股」而非「全球運動集團」的根本原因。
3. Topsports 與本土國潮品牌反向相關：Anta 23% 上升 = Nike 20.7% 下降 = Topsports 流水衰退。三者形成 zero-sum trade — 同時 long Anta + short Topsports 是 sector 內 alpha pair。

💡 §2 Insight
1. Anta 是「東亞 Unilever-LVMH 混合體」：用 HUL 模式收 FILA / Descente / KOLON 中國經營權，用 LVMH 模式收購 Amer Sports 全球品牌組合 — 兩種 playbook 同時運作。
2. Anta Group 中國 share 已過 inflection（22% → 30% 是 next leg）：類比三星 1990-2000 韓國 share 35% → 65% 路徑，預期 Anta 2030 中國 share 28-30%。
3. Other Brands segment（2024 RMB 17B）是「Anta 的下一個 FILA」：FILA 從 RMB 0.6B → 26.6B 用了 15 年；Other Brands 在 2024 起點 RMB 17B，意味 2030 達 RMB 60-80B 是 base case，這是 Anta 永久 alpha 來源。
