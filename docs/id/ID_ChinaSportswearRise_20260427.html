關於「中國運動服飾」，市場最常見的偷懶看法是「中國消費復甦 → Nike China 就會反彈」。這把結構當成了循環。真相是：Anta Group 2025 中國市占 ~21-23% 已永久反超 Nike 20.7%（Euromonitor；Li-Ning 9.4% / Adidas 8.7%），而 Nike 大中華連 7 季 -ve（Q3 FY26 -7%、cc -10%、Q4 guide -20%），管理層自己說清庫存要拖到 FY27。這不是一個會反彈的循環低點，是六年結構性份額轉移。

但本報告要揭的第二層更關鍵：Anta 真正的 alpha 不是主品牌（+3.7%）也不是FILA（+6.9%），而是「Other Brands」segment（FY25 RMB 17B、+59.2%、OPM 27.9%、DESCENTE 零售破百億）+ Amer 大中華（FY25 $1.86B、+43.4%）。必須誠實標明：這兩條曲線是無機（M&A 驅動）——DESCENTE / KOLON / Maia / Jack Wolfskin / Puma 29% 都是買來的，Amer 是控股 44.5% 的併表。把它與主品牌有機 +3.7% 放同一格估值是錯的口徑。

所以本報告不押「中國運動 = 普漲」，而是畫出誰永久切走份額、誰被結構化吃掉。結論：結構性贏家是多品牌結構的 Anta（2020.HK）+ 全球化載體 Amer（AS）；國潮 niche 是 Li-Ning / 361° / Xtep；最明確的鏡像輸家是通路代理 Topsports（6110.HK）——1H FY26 銷售 -5.8%、淨利 -9.7%、中期股息已從高基期砍到 RMB 0.13，正是 Nike 中國衰退結構化的 cash 端證據。

PART I · DECISION SUMMARY

決策摘要層

三句話看完中國本土運動服飾的現狀、走向與行動，後接 PM 級監測點。這一層是報告的全部重點——下面各節只是把這一層的每個判斷補上證據。

現在 / NOW

Anta 集團 FY25 RMB 80.22B（+13.3%）、中國市占 23% 已永久反超 Nike 20.7%。三引擎：Anta brand +3.7% / FILA +6.9% / Other Brands +59.2%（RMB 17B、OPM 27.9%、DESCENTE 破百億）。Amer 大中華 FY25 $1.86B（+43.4%）。Nike 大中華連 7 季 -ve（Q3 FY26 -7% / cc -10% / Q4 guide -20%）。

未來 / NEXT

3–5 年 Anta 用 FILA 反向收編 playbook（15 年 0.6B→28.5B）把 Other Brands + Amer 滾成第二、第三條曲線；國潮 share 進 28–30% 高原；全球化（SE Asia 1,000 店 + Puma 29% + Jack Wolfskin）是 next leg。最大裂縫 = 無機成長口徑與頂級競技段仍由 Nike/Adidas 守。

行動 / ACTION

押多品牌結構性贏家 Anta（2020.HK） + 全球化載體 Amer（AS）；Li-Ning / 361 / Xtep 為國潮 niche；對照組 Nike / Adidas / LULU 看 share 消長。Topsports（6110.HK）是 Nike 中國衰退鏡像、FY26 股息已砍非 cushion。買賣由個股 DD + 系統決定。

The Key Call · 本報告最重要的一個判斷

市場看「中國消費循環 → Nike China 反彈」，真正的非共識是結構非循環——Anta Group 2025 中國 23% > Nike 20.7% 是六年永久份額轉移（Anta 2018 12.5% → 2025 ~23%，Nike 同期 22.9% → ~16%）。而 Anta 真正 alpha 不在主品牌（+3.7%）或 FILA（+6.9%），是無機驅動的 Other Brands（+59.2%、OPM 27.9%）+ Amer 大中華（+43.4%）。誠實揭露：這兩條是 M&A 滾出來的（DESCENTE / KOLON / Maia / Jack Wolfskin / Puma 29% / Amer 控股 44.5%），不能與有機 +3.7% 同口徑估值。最明確的鏡像輸家是 Topsports——Nike 中國衰退把它的股息逼到從高基期砍至 RMB 0.13。

三道裂縫具體是：①永久反超——Anta 23% > Nike 20.7% 是 Euromonitor confirmed 數據（2025）非預測，新疆棉（2021-03）是不可逆的 inflection；②無機 alpha 的口徑問題——Other Brands +59.2% / Amer +43.4% 光鮮但是買來的，organic 主品牌只有 +3.7%，估值若把無機段當有機高成長定價會錯；③通路鏡像——Nike 中國連 7 季衰退把 Topsports 1H FY26 推到銷售 -5.8% / 淨利 -9.7% / 股息砍，這是衰退結構化的 cash 端讀數。盯這三條，比盯「中國消費復甦」更能看清份額的流向。

PART II · HISTORICAL ANALOGS

三星 / HUL / LVMH，框住多空

歷史不必重演，但三個對照各給一個帶量化錨的教訓：三星給「本土→全球的份額路徑」，HUL 給「外商品牌的本地化高毛利模型」，LVMH 給「收購後 OPM 翻倍的操盤」。

類比一：三星 / 現代——本土份額過半後的全球擴張。韓國電子 / 汽車 1990–2010 從 OEM → 自主品牌 → 全球品牌，本土份額從 ~35% → 65% 後再向海外擴張至全球前三。量化錨：Anta 集團中國 share 從 2018 的 12.5% 走到 2025 的 ~23%（+10.5pp），對應 Nike 同期 22.9% → ~16% 的鏡像流失——這是「本土過半前」的軌跡，類比三星預期 Anta 2030 中國 share 28–30%。差異：Anta 全球擴張靠收購 + 投資（Amer 持股 / Puma 29%）而非直接 brand build，capital-efficient 但聚合性較弱（多品牌 vs 三星單一品牌）。

怎麼讀：三個 playbook 同時運作——Anta 是「東亞 HUL-LVMH 混合體」。bull 的底是三星式份額路徑 + FILA 式反向收編可複製到 Other Brands；bear（對估值）是 LVMH 式無機成長若被當有機高成長定價，倍數會錯。

類比二：HUL 印度本地化——外商品牌的高毛利模型。Unilever 透過 HUL 收本土品牌 + 本地化經營，印度 OPM 25%+。量化錨：FILA China 同結構——外國商標 + Anta 本地化 + Op margin ~26%；FILA 從 2009 RMB 0.6B 到 2025 RMB 28.5B（44x/15 年）就是「反向收編可複製」的範本。這正是 Other Brands +59.2% 的 thesis 基礎——DESCENTE 2025 零售破百億、成為集團第三個破百億品牌（前兩個是 Anta、FILA），是「下一個 FILA 曲線」的第一個實證。類比三：LVMH 收購後操盤。LVMH 收 Bulgari $5.2B / Tiffany $15.8B 後 OPM 翻倍。量化錨：Anta 收 Amer $5.2B（2019）→ IPO → ~$20B（~4x/5y），Salomon 破 $2B（+35%）、Arc'teryx 領軍 technical apparel——但這是無機成長，估值要把它與有機 +3.7% 分開算。教訓：三個類比都成立，但第三個提醒——別把買來的成長當內生高成長付倍數。

PART III · SUPPLY SIDE

供給側：利潤池往多品牌 + 反向收編高端集中

反直覺但可操作的結論：利潤正從「國際品牌（Nike/Adidas）+ 通路代理（Topsports）」往「國潮多品牌（Anta/Li-Ning）+ Anta 反向收編高端（FILA/Other/Amer）」雙引擎遷移。

供給端可拆四層。上游 OEM（裕元/寶成）：毛利 5–12%、議價權低，本土訂單上升部分對沖 Nike 訂單下滑。中游國潮品牌（Anta/Li-Ning/361/Xtep）：2025 國產合計中國 share ~32%，GM 41–49% / OPM 11–24%；2018–2025 +20pp share gain。中游反向收編高端（FILA/Other Brands/Amer）：GM 56–72% / OPM 25–30%，是利潤池增量的核心——但口徑提醒：這一層多為無機（買來的品牌）。下游通路（Topsports/Pou Sheng）：GM 40–42% / OPM 5–8%，議價權隨 Nike/Adidas 中國 share 從 42% → ~30% 崩跌。利潤往「國潮 + Anta 高端 segment」集中，國際品牌與其通路代理失血。

Inference · 供給
多品牌 + 反向收編把利潤池切走，但無機段口徑要分
前提：Anta 23% > Nike 20.7%（永久）+ Other Brands OPM 27.9% > FILA 26% > Anta brand 21% + Amer 大中華 +43.4% + Topsports 議價權崩 → 利潤往國潮多品牌 + Anta 高端 segment 集中

當 Anta 用「私有化 → 中國本地化 → 全球反向擴張」把 FILA / DESCENTE / Amer 一個接一個滾起來，利潤池就從國際品牌往 Anta 多品牌矩陣集中；Nike 大中華連 7 季 -ve、Topsports 1H FY26 淨利 -9.7% 是利潤流失的一體兩面。但必須切口徑：主品牌 +3.7% 是有機，Other Brands +59.2% / Amer +43.4% 是 M&A 驅動的無機段——前者反映 brand health，後者反映收購整合能力，估值不可混為一談。

可證偽條件：若 Other Brands 在無新併購下連 2 期 <+25%、且 Amer 大中華 <+20%，則「反向收編可無限複製」論失效；或若 Nike 大中華連 2 季 >+5% 反彈，則永久反超論受挫。

PART IV · DEMAND SIDE

需求側：国潮結構棧與高端 K-shape

需求最剛性的是「Z 世代國產偏好 + 高端段的可炫耀性」——國潮不只是愛國主義，是 5 層 stack 的結構現象；而高端段在中國呈 K-shape（FILA/Arc'teryx/LULU 共享、mass 段價格戰）。

第一引擎：国潮結構棧（現需）。國產品牌崛起是 5 層 stack：L1 產品性能（中端已追平、頂級競技仍弱）、L2 文化敘事（新疆棉「站隊」效應）、L3 通路（線上滲透 35%+、Tmall/Douyin 雙 11 本土 top 3）、L4 KOL 明星（王一博/谷愛凌）、L5 反向收編高端（FILA/Amer/Jack Wolfskin）。Z 世代國產偏好從 2018 ~41% 升到 2024 ~78%（+37pp）是結構訊號。第二引擎：高端段 K-shape（現需）。高端 athleisure（>RMB 800/件）由 FILA + Arc'teryx + LULU 共享——LULU 中國大陸 FY25 +28%（從 +35% decel）是高端段唯一外國倖存者；Amer 大中華 +43.4% / Salomon XT-6、Arc'teryx Beta 用 wait list + 稀缺供應複製 LVMH 路徑。需求把預算推向「國潮性價比 + 高端可炫耀性」兩端，mass 段中間被 Pinduoduo/白牌價格戰侵蝕。

多空為何都對

多頭測國產份額轉移（Anta 反超 + Z 世代偏好），這幾乎沒爭議；空頭測無機成長口徑 + 頂級盲點（Other Brands/Amer 是買來的、頂級競技段仍由 Nike/Adidas 守）。兩者可同時為真：國產永久反超是真（多頭對），但 alpha 集中於無機段、有機主品牌只有 +3.7%、頂級 brand premium 仍弱（空頭對口徑與品質）。本報告把重心放在「永久 vs 循環」與「有機 vs 無機」兩個切口。

PART V · VERDICT & SCENARIOS

裁決與三情境（24–36M）

「國產永久反超 Nike」判 INTACT，「Anta 多品牌 + 反向收編是結構性 moat」判 INTACT，但「Other Brands/Amer 無機 alpha 能否可持續複製」判 AT_RISK——故結構看「Anta/Amer 贏、Nike/Topsports 輸」、conviction 卡 mid-high。如硬要對應供需框架：對「可信國產高端 + 性價比運動供給」需求 > 合格供給（要品牌力 + 收購整合），等價於結構性短缺型，稀缺來自品牌建構棧而非產能。

Base Case主觀機率：很可能
Anta/Amer 吃份額、Nike 續衰退、Topsports 萎縮

Anta 集團續 +10–13%、國潮 share 往 28% 走；Other Brands 維持 +30–40%（含新併購墊高）；Amer 大中華 +30%+；Nike 大中華續 -ve（清庫存拖到 FY27）；Topsports 流水續降、股息續砍。國產永久反超延伸、無機 alpha 持續但被誠實標記。

個股特徵與 conviction tier

CORE 🔴

Anta（2020.HK）

對照組 🟡 / 鏡像輸家

NKE / ADDYY / LULU ｜ Topsports（6110.HK）

🟡 對照：NKE（大中華連 7 季 -ve、Q4 FY26 guide -20%、清庫存拖到 FY27——受害已 partial priced）、ADDYY（Gulden turnaround 但 decel +13→+8→+6%）、LULU（高端段唯一外國倖存者、+28% decel）。鏡像輸家：Topsports（Nike/Adidas 中國最大代理、1H FY26 -5.8%/-9.7%、中期股息已砍至 RMB 0.13——dividend trap 確認中、非 yield cushion）。conviction：偏空（Nike/Topsports）/ 中（LULU 高端例外）

估值面總結：本主題的核心張力在「國產永久反超是真 vs alpha 集中於無機段（多數有機主品牌只 +3.7%）」。Anta（多品牌 + 反向收編）+ Amer（全球化載體）是最高純度贏家；Li-Ning/361/Xtep 是國潮 niche；結構性 / 鏡像輸家是 Nike（份額永久流失）與 Topsports（議價權崩 + 股息已砍）。現處 Phase III 後段（永久反超 + 全球化啟動），轉空條件 = Other Brands/Amer 無機 alpha 失速 或 Nike 大中華 inflection；de-rating 風險主要在 Anta（無機段被當有機定價的修正）與 Topsports（dividend trap）。買賣由個股 DD + Pure MA 系統決定。

The One Line

這不是中國消費循環、是 Anta 用多品牌 + 反向收編把 Nike 的中國份額永久切走——但真正的 alpha 是買來的（Other Brands/Amer），口徑別跟有機 +3.7% 混算。

  1. 不把「中國消費復甦」當成 Nike China 反彈的理由——明確區分永久結構轉移 vs 循環低點。
  2. 不把 Other Brands +59.2% / Amer +43.4% 當有機高成長——誠實標記為 M&A 驅動的無機段，估值口徑要分。
  3. 不告訴你哪一檔該買——CORE/EDGE/對照組是「特徵的代表案例」，買賣由你的個股 DD 與 Pure MA 系統決定。
