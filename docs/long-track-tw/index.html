訊號流程說明 — 純 E3、沒有「半倉買回」問題(對照美股 SMH 頁)

台股版只有一套訊號：E3,沒有美股 SMH/QQQ 頁那層 ST(10,3) 半倉出場閘門，所以不存在「出場的半倉怎麼買回來」這個問題 —— E3 跌破一條線就減 ⅓、收復一條線就加 ⅓,買回訊號就是均線/動能自己收復的那一週收盤，不需要第二層閘門。
每週五收盤只算一個數字：每標的目標倉位 = E3 =(W40 + W52 + TSMOM)÷ 3,四檔 0 / 33% / 67% / 100%; 與上週目標的差額就是下一個交易日要下的單。2330 與 0050 各自獨立跑訊號與資金、日報酬 50/50 合成，本頁頂部的倉位數字即當前目標。
TSMOM 為月頻(月底判定),在週線盤整裡不翻面，所以那 ⅓ 多數時候不動 —— 真正把大盤回撤砍半的，是 W40 / W52 兩條週均線在趨勢轉折時各讓出 ⅓。

為什麼台股不掛美股那層 ST 閘門：美股 SMH/QQQ 的 ST 半倉閘門搬到台股過不了同一把尺 —— 它的 Calmar 0.83 反而輸給「同 CAGR 少持一點 E3」的靜態稀釋線(0.85)、MDD 也沒更淺。 原因是台股急跌後常 V 型反彈(2020、2022 底、2024-08),週線閘門砍在低點、追在高點，傷害集中在上半期(H1 Calmar 0.73 vs E3 0.82)。 跨市場結論：趨勢本體(E3)會搬家、出場閘門這層不會。 E3 本身對大盤 B&H 已是經典趨勢交換 —— 讓出約 4pp CAGR、把 -39.5% 回撤壓到 -23.75%(Calmar 0.62 → 0.85),詳見 回測頁。保留意見：2008 GFC 不在樣本、2330 集中度高。
