§4 序章：第一性原理 × 逆向思考

存在意義

CDNS 解決的是「現代 SoC 設計（每代多 10x 電晶體 / 多 5x 互連複雜度）變得不可能用人腦完成」這個物理 / 心理硬約束。台積電 N2 / N1.4 / N1 製程一個 die 動輒 1,000 億電晶體，沒有 EDA 工具則 chip design 無法收斂。CDNS + SNPS 雙寡占建立工具 + IP + Methodology 三層壟斷，全球無 Apple / NVIDIA / 高通 / Google / 三星不依賴它們。若 CDNS 明天消失，SNPS 一家獨大會立刻 +30% 工具價，全球 chip design TTM (time to market) 會延長 6 個月，AI 算力部署速度會放緩。

逆向推演（Munger Inversion 三維）

① 技術維度

  • (1) Open-source EDA（OpenROAD, Magic）成熟到取代商業：學界 + Google 推動，目前距離商用還有 10+ 年差距。機率：低。
  • (2) AI 自動化 chip design 跳過 EDA：Anthropic / OpenAI 直接從 spec → silicon。機率：低（需要 verification 工具仍走 EDA）。
  • (3) 量子計算取代傳統矽：但仍需要設計工具，且 CDNS 已切入。機率：低。
  • (4) Synopsys 大舉領先（Ansys 整合成功）：SNPS 收 Ansys 整合 multi-physics 領先一代。歷史對標：Cadence 2010 收 Tensilica。機率：中。

② 管理維度

  • CEO Anirudh Devgan（2021 上任）穩健，研發背景；接班風險低。
  • R&D 占 Rev ~40%（業界最高），維持技術領先；Capex 配置合理。

③ 結構維度

  • 美中科技戰中國 EDA 全面禁運：中國市場 ~12%，若全部消失 = 一次性 -12% revenue 衝擊。歷史對標：Huawei 2019 制裁。機率：中。
  • 新運算範式（neuromorphic / quantum）規模化：未來 10 年才會見到，CDNS 可順勢轉型。

§9 護城河分析

評分：9 / 10（A 級）

護城河來源

  • 網路效應：中-強。CDNS 工具用得越廣，第三方 IP / Methodology 越多。
  • 無形資產：強。25,000+ 專利 + AI/ML 演算法 + Tensilica IP。
  • 轉換成本：極強。客戶換 EDA 工具需重訓工程師（6-12 個月）+ flow 重建 + IP 移轉，TCO 數千萬美元。
  • 規模優勢：強。R&D 占 Rev ~40%，僅次於 SNPS。

競爭鴻溝

  • (1) Synopsys 為唯一可比對手（合計 ~70% 市佔）。
  • (2) 第三方 EDA（Siemens Mentor、Ansys、ANSI 等）市佔合計 < 15%，且專注 niche。
  • (3) Open-source EDA（OpenROAD、Magic）距商用 10+ 年差距。
