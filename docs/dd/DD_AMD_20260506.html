§4 序章：第一性原理 × 逆向思考

存在意義

AMD 的核心存在意義是確保 x86 CPU 與 GPU 計算市場存在「可信第二來源」，避免單一供應商（Intel 在 x86 / NVDA 在 GPU）對全球計算基礎設施的定價與供應形成壟斷。若 AMD 明天消失：(1) hyperscaler（AWS / Azure / GCP / Oracle）瞬間失去對 INTC server CPU 議價籌碼，CPU 採購單價立即 +30-50%；(2) AI infrastructure 完全依賴 NVDA 一家，hyperscaler 自研 ASIC（TPU / Trainium / Maia）的 fallback 路徑被砍，等於把整個 AI 產業押在 TSMC 4nm/3nm 的 NVDA 配額；(3) gaming console（PS5 / Xbox Series X 都用 AMD APU）失去單一 SoC 供應商，需要 2-3 年重新設計。

逆向推演（Munger Inversion）三維

① 技術維度 — 什麼技術替代會讓 AMD 3 年內失去 50% 價值？

  • NVDA Rubin / VR200 power efficiency 領先 2x（歷史對標：Intel Itanium 2002 vs Power4 — 同代差距 2x 即可讓客戶不換軌）。AMD MI400 若 Perf/W 落後 Rubin >30%，hyperscaler 維持 90/10 分配 NVDA / AMD。
  • ASIC 取代 GPU（歷史對標：bitcoin mining 從 GPU 轉 ASIC 18 個月內完成）。若 hyperscaler 的 TPU / Trainium / Maia 在 inference 上 cost-per-token 低於 GPU 60%+，AMD GPU 業務直接被擠至 training-only niche。
  • CPU 端 Arm 取代 x86（歷史對標：Apple Silicon 2020-2024 從 0 至所有 Mac）。AWS Graviton 4 / 5 + 自研 Arm SoC 若在 server 達到 30% 市佔，AMD EPYC 反成 INTC 同病相憐。
  • Optical I/O / silicon photonics 跳級（NVDA 已投資 Ayar Labs / Lightmatter）— 若 AMD 在光通訊整合落後，rack-scale 性能差距會放大。

② 管理維度 — 什麼決策路徑會自我毀滅？

  • Capex 慣性：AMD 是 fabless，本身 Capex 不大（FY25 $0.97B vs Rev $34.6B = 2.8%），主要透過 TSMC wafer purchase 反映。風險在於 2026 過度承諾 wafer 但 GPU 銷售不及預期 → inventory write-down（歷史對標：INTC 2024 多次 write-down）。
  • Lisa Su 退場：Lisa Su 自 2014 接任至今，個人聲望已 = AMD 品牌的一部分。若她在 MI400 量產關鍵期（2026-2027）退休或健康出問題，過渡期執行力風險高。歷史對標：Apple Steve Jobs 2011 → Tim Cook 平穩過渡，但 Sun Microsystems Bill Joy 2003 退場後業務崩潰。
  • 整合失敗：Xilinx（370 億）+ Pensando（49 億）+ ZT Systems（49 億，2024）三筆併購尚未完全消化，若 2026 再啟動大型併購（rumored 收購 Astera Labs / Tenstorrent），可能觸發整合疲勞。

③ 結構維度 — 什麼產業變化會讓商業模式過時？

  • 計算範式轉換：若量子計算 / 神經型態計算（neuromorphic）在 2030 前對特定 AI 工作負載達商用化，傳統 GPU/CPU 計算市場 TAM 衰退。歷史對標：mainframe → x86 server 1985-2000 期間 IBM mainframe 收入腰斬（雖然 mainframe 仍存活但已成 niche）。
  • 監管型分拆：FTC / EU 若認定 AMD + Xilinx 合併造成 FPGA 壟斷，要求剝離 — 雖然機率不高（Xilinx 整合已 4 年），但若 NVDA 反壟斷案敗訴會拖累整個 AI 半導體類股估值（歷史對標：AT&T 1984 分拆讓整個產業重新洗牌）。
  • 地緣政治：中國市場結構性失去：MI308（中國閹割版）若被禁止出口或中國全面轉用 SMIC + Huawei Ascend，AMD 失去 ~10-15% TAM。但結構性影響有限（vs NVDA 同命運）。

逆向推演結論：最大威脅是 NVDA Rubin 在 networking + power efficiency 拉開到 2 代差距，導致 AMD 從「第二來源 + 議價籌碼」退化為「便宜的 NVDA」（定價權崩潰）。但 OpenAI / Oracle 多年合約已部分 hedge 此風險（產能已分配）。價值陷阱模式判斷：非陷阱，但有「成長股估值陷阱」反向風險（高估值對成長兌現失望的單邊風險）。
