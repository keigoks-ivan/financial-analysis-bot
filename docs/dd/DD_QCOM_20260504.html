§4｜序章：第一性原理 × 逆向思考（v12.2 三維 Munger）
A｜存在意義

Qualcomm 解決的是「無線通訊基本協議的設計與授權」這個高度集中的 IP 壟斷工藝。CDMA → 3G → 4G → 5G → 6G 的標準制定、晶片實作、和裝置整合，QCOM 在 1985-2010 之間累積了核心專利，使「任何賣手機的公司，無論 chip 是否買 QCOM，都需付授權金」這個結構成立。這不是製造也不是品牌，是「無形資產 IP 收稅」的商業模式。

若 QCOM 明天消失，全球 5G 部署不會立即停擺（其他廠商可短期填補應用處理器空缺：聯發科、Apple in-house、Samsung Exynos），但 IP 授權的全球收費結構會崩解 18-24 個月，OEM 之間 royalty 重新洗牌，無線標準制定也會減速 1-2 個世代。

② 管理維度

  • Cristiano Amon 接班執行力：2021 接 Steve Mollenkopf，5 年 record 平淡；Auto/IoT 多元化方向正確但執行速度需驗證。若 Auto 不能維持 +30% YoY，CEO 信用受打擊。

  • Arm vs QCOM Nuvia 訴訟：2026 仲裁中，若敗訴需賠償 $5-10B 額外金額且 PC SoC 戰略受阻，是 tail risk。

③ 結構維度

  • QTL 授權商業模式被結構性挑戰：歐盟、印度、日本反壟斷監管壓力持續；OEM 集體拒付（中國品牌試圖）會結構性削減 QTL Margin。
  • Edge AI 主導權外移：若 OpenAI/Anthropic 與 Apple/Samsung 建立直接 OS 整合，Snapdragon NPU 變成「dumb compute」，差異化縮減。
  • Auto SoC 中國本土化：地平線 / 黑芝麻 / 華為 MDC 已在中國 OEM（比亞迪 / 蔚來 / 小鵬）取得 design win，QCOM Auto 進入中國市場受阻。

§9｜護城河分析（v12.3 二維拆解）

二維評分（v12.3 強制）：執行護城河 7.0/10 + 定價力護城河 7.0/10 → 合併 7.0/10

Non-single-axis：QCOM 非 SaaS / 銀行 / 保險 / 寡占公用事業，不適用 single-axis escape rule；二維拆解為強制。

執行護城河（Execution Moat）：7.0/10

定義：組織執行既定 roadmap 的能力 — 工程 / 供應鏈 / 產品上市 / 大客戶交付。

② 對下游（OEM 客戶：手機 / Auto / IoT）

下游客戶結構分三層，議價力各異：第一層 Apple（~10% Rev FY26，FY27 降至 0%） — 歷史上 Apple 以 sole-source 採購換取特別工程支持，議價力最強，QCOM 讓步 royalty cap 上限，Apple 完全退出後此 pressure 消失但 revenue 也跟著消失。第二層中國 Android OEM（~40% Rev）— QCOM 佔高階機型 SoC 近乎 sole-source，但聯發科 Dimensity 9500 突破後，OEM 擁有雙供應商談判籌碼，QCOM 每季議價 margin 受壓 1-2pp；若自研化加速（小米玄戒 / 三星 Exynos），議價空間將進一步收窄。第三層 Auto OEM（~12% Rev，快速增長）— 3-5 年設計定點 certification 週期使議價力最低（OEM 無法臨時換供應商），QCOM 在 Auto 端擁有最強議價地位，且 Snapdragon Ride Elite ASP 持續上修。

③ 地緣政治 / 去中化風險

中國市場佔 QCOM 約 45% 營收（含出口至中國 OEM 及 QTL 授權）；中美晶片管制 export control 對 QCOM 的直接影響相對有限（5G modem / handset SoC 非列管先進 AI 晶片），但間接影響存在：① 中國 OEM 在去美化壓力下加快自研化（政府補貼支持），MTK + 自研 SoC 雙路並進；② 美國 export control 升級（若 HuaWei 等被禁客戶名單擴大）使中國地區 OEM 訂單存在 tail risk；③ 台灣地緣風險（TSMC 主代工基地）使 QCOM 與大多數 fabless 同業共擔供應鏈集中度風險。QCOM 的 QTL 授權業務（中國 OEM 皆為 licensee）在美國制裁下處於更複雜位置 — 禁止特定中國實體付費可能使 QTL Rev 受損，需持續監測 OFAC / BIS 最新禁令清單。
