§4｜序章：第一性原理 × 逆向思考

存在意義

KLA 解決的痛點是「半導體製造的不可逆良率損失」。每片 leading-edge 晶圓造價 $15-25K，從 photomask 到 wafer 處理可能跨 1,000+ 步驟，任何單一缺陷會放大成整片報廢。KLA 的 inspection + metrology 在每個關鍵步驟攔截缺陷，把潛在報廢轉化為早期修正信號 — 對 foundry / IDM 而言這不是「成本」，而是「可量化的 yield insurance」（典型 fab 每提升 1%p yield 可帶來數億美元年收入）。

若 KLA 明天消失，世界會出現：(a) 7nm 以下 leading-edge wafer 良率立刻崩跌 10-30%p，AI accelerator / HBM 量產延後 2-3 年；(b) HBM 與 advanced packaging 的 die-to-die / chip-to-chip 量測中斷，TSMC CoWoS / Samsung Pyeongtaek HBM 量產癱瘓；(c) 沒有對手能在 2 年內補上 — ASML HMI 集中 e-beam、AMAT SEMVision 是 review 工具，optical + electron beam + reticle + wafer 全棧只有 KLA 一家。

逆向推演（Munger Inversion）三維展開

① 技術維度：什麼技術替代會讓 KLAC 3 年內失去 50% 價值？

  • AI-based wafer review 取代光學/電子束：若 ML model 能從 wafer-edge 信號重構 defect map，不需 KLA 的物理 inspection 工具 — 機率低（物理量測仍是 gold standard），歷史對標 Intel 2016 內部 AI defect classification 失敗
  • ASML HMI 整合 EUV + e-beam metrology 一站式方案：若 ASML 把 metrology 內嵌 EUV scanner 減少獨立量測，KLA 失去切入點 — 部分發生（ASML HMI YoY 高個位數），但局限 e-beam，optical + reticle 仍 KLA
  • AMAT SEMVision 全棧 e-beam 攻入 inline metrology：2025/2 H20 推出 — QC-23 🟡 點對點（不到生態攻擊等級），3 年內難撼動 KLA 50%+ share
  • Chiplet / heterogeneous integration 改變量測模型：若 die-to-die 量測需求降低（透過設計優化），AP 增速放緩 — 但反向：chiplet 越普及，需要的量測越複雜

歷史對標：Intel 在 2015-2018 嘗試自建 wafer inspection 失敗（成本與認證壁壘），轉回採購 KLA。類似 ARM 之於 RISC-V — 替代者持續存在但無法在認證密集 + 高 stake 場域取代主導

② 管理維度：什麼決策路徑會自我毀滅？

  • Capex 慣性 + 收購整合失敗：若 KLA 重複 2019 Orbotech 模式（買 specialty 但 ROIC 稀釋），但金額放大 5 倍 — 例：嘗試 $20B+ 收購 ASML 或 AMAT — 機率極低（管理層保守，2010 後最大併購 Orbotech ~$3.4B）
  • 創辦人退場：Rick Wallace CEO 自 2006 在位（20 年），即將過渡。歷史對標 Intel 從 Andy Grove → Craig Barrett → Paul Otellini 的轉折期
  • 資本配置失衡：若回購 + 股息攀至 ≥ 100% FCF，犧牲 R&D 投入（目前 R&D/Rev 13%，產業頂尖） — 機率低，2030 目標已 commit R&D 持續投入

歷史對標：Intel 2010s 創辦人退場後 R&D allocation 失衡，10nm 延後 4 年導致市佔流失。KLA 需要避免類似 leadership transition 期間的 capital allocation 失誤

③ 結構維度：什麼產業變化會讓商業模式過時？

  • WFE 結構性下行：若 Moore's Law 完全 stagnate（節點微縮速率掉到 2-3 年/代以上），WFE 萎縮 → KLA 收入不可逆下行。機率低（gate-all-around / backside power / chiplet 仍在推進）
  • 監管型分拆：若 US 強制把 process control 視為 critical infrastructure 強制分拆 — 機率極低（KLA 是國家戰略資產）
  • 計算範式轉換：若量子計算或光子計算取代 CMOS — 但即使如此，新範式也需要量測。KLA 的 know-how 是平台性

歷史對標：AT&T 1984 被反托拉斯強制拆解前持有 90%+ 美國電話市佔。但 KLA 是「全球」寡占，不存在單一監管機構強制拆解的 jurisdiction（不像 Standard Oil 1911）

護城河來源（v12.3 強化 sourced evidence）

  • 網路效應：不完全是傳統 network effect，但具備「客戶安裝基台 → Service 多年期合約 → 數據積累 → 下一代 tools 認證快速通過」的 data flywheel。規模閾值：客戶已採用 KLA tools 多代後，roadmap dependency 已實質鎖定（典型 5-10 年）
  • 無形資產：10,000+ 專利（含 EUV reticle inspection、HBM die-to-die 量測核心專利），R&D 投入 $1.6B/年（13% of Rev）— 對手需 5-10 年 + $5-10B 投入才能追上單一細分
  • 轉換成本：客戶換掉 KLA tool 需 (a) $50-100M tool 重新採購 + (b) 6-12 個月 qualification + (c) process recipe 重建（中斷 yield 風險不可承受）— 實質不可逆
  • 規模優勢：單一客戶 install base 越大，KLA Service margin 越高（Service GM > 65%）；對手需達到 1/2 KLA 安裝基台才能 break-even — 認證 + 時間 + 資本三層壁壘

競爭鴻溝（v12.3 具體 mechanism）

為什麼資本充足的對手（AMAT / ASML / LRCX）仍無法跨越？

  • 時間壁壘：每代 leading-edge node 共同開發週期 5-7 年；KLA 已完成 N+1, N+2 多代共同設計（A2/14A node），對手從零起步需 5-7 年才能切入下一代
  • 資本壁壘：單一細分（如 reticle inspection）對手需 ~$3-5B R&D + $1-2B 認證投入才能達到 KLA 一半水準；多細分需 $20B+ 累積投入（已超過任一對手 R&D 預算）
  • 認證壁壘：客戶 process recipe qualification 6-12 個月 / tool；多代 node 已累積 process-specific tuning，對手即使技術追上也需「客戶願意承擔切換 yield 風險」— 在 leading-edge 不可能
