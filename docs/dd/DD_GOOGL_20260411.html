最終裁決:🟡 觀望(Watch,偏正面 — 等短期 R:R 改善或 Cloud 現金流變現)

Alphabet 是 v8 序列中最接近 NVDA 的「品質 + 估值雙達標」案例,但短期 R:R 稍嫌不足。關鍵數字:Google Cloud +48% YoY、Cloud backlog $240B(+55% QoQ)、Gemini MAU 750M、Search Q4 +16.7%(加速)、ROIC 27.97%、20Y 均 ROIC 32.76%。估值面:Forward P/E 23.31 略低於 5Y 均 24.83,PEG 合理,與分析師中位目標 $375(+18%)存在健康 gap。唯一主要風險是 2026 Capex 指引 $175–185B(+80% vs 2025),這將使 FCF 在 2026 年承壓,意味 EV/FCF 51.77 可能進一步惡化。技術面:現價位於 MA50 $293.83 與 MA200 $302.90 之上,但 MA50 < MA200 殘餘「死亡交叉」狀態,中期結構比短期結構更強。短期 R:R 1.82x 未達 3.0 門檻,中期 2.88x 達 2.0 門檻,長期 2.01x 🟡 臨界。與 NVDA 的完美 ✅/✅/🟢 不同,GOOGL 目前是 ❌/✅/🟡 的「中等進場」情境。

財報信號與假設校準

  • H1(Search 未被 AI 顛覆,仍為核心金流引擎):強化Search 全年 YoY 加速(Q1 +9.8% → Q4 +16.7%),打臉所有「AI 將毀滅 Search」的看空論。
  • H2(Google Cloud 成為第三個 Hyperscaler 盈利引擎):強化+48% YoY、backlog $240B、ARR $70B。
  • H3(Capex 週期會在 2027 後回落至合理水位):削弱2026 指引 $175–185B 超出預期,Capex/Rev 比率將攀升至 ~40%。

這份財報的整體基調是「Cloud 與 Search 雙引擎上修 + Capex 嚴重上修」。隱含預期在「營收成長」維度變得更容易兌現,但在「FCF 轉換率」維度變得更難兌現——這就是當前短期 R:R 稍嫌不足的核心原因。

Q4 2025 Beat(Rev $113.8B + EPS $2.82)+ Cloud +48% YoY + backlog $240B——短期 newsflow 非常正面。但市場對 Capex $185B 有疑慮,股價從 $349 回檔至 $319(−8.6%),反映 Capex 擔憂已部分定價。引用 §1。

③ 長期護城河邏輯(3 年後)

Google 擁有 AI 時代唯一的「全棧自主」——從 TPU 晶片(Ironwood 第 7 代) → DeepMind 模型(Gemini 3)→ YouTube / Search / Android / Chrome / Workspace 通路。這是 v8 序列中唯一在四個層級都自主的公司(NVDA 有晶片但無通路、MSFT 有通路但仰賴 NVDA、AVGO 有晶片但無模型)。ROIC 28% 穩定,10Y ROIC 均 30%+ 證明長期複利能力。引用 §2.5 A/C/D。

§0 序章:第一性原理 × 逆向思考

存在意義

Alphabet 解決的是「全人類資訊的即時檢索、組織與分發」這個 AI 時代前半段最重要的問題。Google Search 每日處理 85 億次查詢,是全球 70%+ 網路資訊流的起點;YouTube 每日播放 50+ 億小時影片,是全球最大視訊平台;Chrome + Android 覆蓋 70%+ 裝置作業系統;DeepMind 是 AI 技術史上最重要的研究機構之一(AlphaGo、AlphaFold、Gemini)。若 Alphabet 明天消失,全球資訊檢索會立即停擺 3–6 個月(Bing + Baidu 產能無法快速填補),YouTube 的替代(TikTok、Instagram)無法承接長影片生態,70% 的 Android 手機會失去系統更新,Gemini 模型的科研價值會使 AI 研究倒退 12 個月。這個存在意義僅次於 NVDA + TSMC,但高於 AVGO / MSFT / MRVL / STX。

逆向推演(Munger Inversion)

  • 地緣政治曝險:總部美國,全球分散。硬體依賴 TSMC 2nm(TPU 代工),間接曝險台海。中國營收佔比極低(~3%)。
  • 技術替代風險:① OpenAI / ChatGPT 是 Search 最大威脅,但 Q4 Search +16.7% 顯示 Google 正在用 AI Overview 反擊成功;② Meta AI 在社交推薦威脅 YouTube(次要);③ Apple Intelligence 威脅 Android 端 AI 入口(小影響)。
  • 管理層失職:Sundar Pichai 被批評「反應慢」但實際執行優秀——Gemini 3 成功追趕、Cloud 成為第三家 Hyperscaler、TPU 自主供應都是結構性勝利。Capex $185B 決策是重大資本配置,需持續觀察兌現。
  • 價值陷阱路徑:非傳統陷阱。若出現問題,最可能是「隱性資本密集型」——Capex 持續高位但 Cloud 增速減退,使 FCF 結構性下修。時間軸 18–24 個月內可見端倪。

§3 護城河分析

評分:9 / 10(與 MSFT 並列,次於 NVDA、TSMC 的 10/10)

護城河來源

  • 網路效應:✅✅✅ 極強。Search(85 億日查詢)、YouTube(50 億日小時)、Android(70% 裝置)、Chrome(65% 瀏覽器)皆為雙邊或多邊網路。
  • 無形資產:✅✅ DeepMind 是全球最強 AI 研究機構之一,Google Brain 過去累積的 Transformer、TPU、Gemini 等創新形成長期 IP 壁壘。Google 品牌為全球前 5 大。
  • 轉換成本:✅ 中等。消費者層面低(使用者可用 Bing),但企業層面(Workspace、Cloud)的資料遷移成本極高。
  • 規模優勢:✅✅ 明顯。Google TPU 自主供應形成單位算力成本優勢,與 NVDA GPU 形成替代競爭。

競爭鴻溝

為什麼對手無法跨越?
① Microsoft Bing 投入 15 年仍只有 ~5% 市佔,證明 Search 的數據飛輪與品牌認知是最強的護城河之一;② OpenAI / Anthropic 有模型但無通路、無自主算力、無廣告變現生態;③ Meta 有社交通路但無 Search 生態;④ Apple Intelligence 僅限 Apple 生態。每個競爭對手只能威脅一個維度,而 GOOGL 有四個維度。

§5 產業格局、議價權與 Unit Economics

市場量化

  • 數位廣告 TAM ~$650B(2025),GOOGL 主導,但 Meta、Amazon、TikTok 持續壓力
  • Cloud TAM ~$700B(2025)→ $1.5T(2030E),CAGR 16%
  • AI 模型 TAM ~$100B(2025),GOOGL 約 15% 份額

議價權

  • 對上游:對 TSMC(TPU 代工)、Samsung / SK Hynix(HBM)為重要但非最大客戶,議價中等。
  • 對下游:對廣告主、Cloud 客戶、使用者議價力強。使用者為「免費使用」,廣告主為「拍賣」,Cloud 為「大額合約」。
  • 地緣:總部美國,相對分散。中國營收 ~3%,最低。

營收組成

  • Google Services:~80%(Search 57% + YouTube Ads 11% + Other 12%)
  • Google Cloud:~15%(FY25),快速擴張中
  • Other Bets:~1%(Waymo 等)

單位經濟

Search CPC 因 AI Overview 不降反升,證明 AI 不是 Search 的顛覆者而是優化器;Cloud ARR 每新增 $1,LTV 約 $8(假設 5 年留存 + margin 30%);YouTube Premium 與 Ads 混合變現,ARPU 穩定成長。

§6 公司治理與資本配置

股東結構

創辦人 Larry Page + Sergey Brin 共持有 Class B(雙重投票權)~51%,仍為實質控股。機構持股集中於 Vanguard、BlackRock、State Street。Sundar Pichai 持股極少但薪酬深度綁定股價。

資本配置軌跡

過去 10 年主要為「R&D + 回購 + 少量戰略併購」。2026 年轉向「R&D + Capex 大幅擴張」,是歷史性決策。回購紀律極好:均價 $170 vs 當前 $319,回購效率優於大部分同業。併購策略保守(YouTube 2006、Android、DeepMind 皆為歷史最佳併購),少有災難性案例。

SBC 評估

SBC/Revenue 從 6% 降至 4.5%,呈改善趨勢。v8 序列中屬中等水準(NVDA 1.5% 最低,MRVL 6–8% 最高)。
