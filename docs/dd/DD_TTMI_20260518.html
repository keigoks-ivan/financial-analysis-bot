§4｜序章：第一性原理 × 逆向思考（v12.2 擴充）

存在意義

剝離財務表象，TTM Technologies 解決的長期且不可逆痛點是：「在地緣政治分裂時代，美國國防 + AI 算力都需要『非中國』的高層數 PCB 供應」。PCB 是電子產品的「神經系統載板」，沒有 PCB，所有計算 / 通訊 / 雷達 / 飛彈導引都是死的。但 PCB 製造 2010-2020 年因為環保 + 勞動成本問題大量遷至中國（中國目前佔全球 PCB 產能 ~55%），同時美國國防部與超大規模雲端業者（hyperscaler）在 2018-2024 年發現一個共同問題：「敏感應用沒辦法用中國造的 PCB」。國防部直接立法 IPC-1791 + Trusted Foundry 認證，hyperscaler 則因擔心後門 / 知識產權 / 供應鏈韌性而開始要求「友岸製造」。

TTMI 的存在意義 = 把這個「非中國高層數 PCB」缺口填補起來。若 TTMI 明天消失，美國國防部要重新認證 Sanmina / Jabil（兩者 PCB 業務規模都比 TTMI 小一個量級，且皆未持有 IPC-1791 Trusted Supplier 同等認證），AI 超大規模業者要選擇 Compeq（台灣）/ Unimicron（台灣）作為 fallback —— 但兩地都面臨地緣政治曝險，沒有「美國本土 + Trusted + 高層數 + 規模」的替代者。

逆向推演（Munger Inversion）三維展開（v12.2 強化）

① 技術維度：什麼技術替代會讓 TTMI 3 年內失去 50% 價值？

  • 光通訊取代電氣 PCB：Co-Packaged Optics (CPO) / Linear Pluggable Optics (LPO) 把光纖直接 co-package 進 ASIC，跳過 PCB 走 trace 的需求；但目前 TAM 預估 2030 才達 PCB 的 20% — 路徑 risk 較遠。歷史對標：銅纜 vs 光纖的取代花了 25 年。
  • 玻璃基板（Glass Substrate）取代 ABF/有機 PCB：Intel 18A + 台積 A14 開始導入玻璃基板（2027 量產），可能讓高層數 PCB 需求被「substrate 上移」吃掉。歷史對標：FC-BGA 取代 wire bond 用了 15 年。
  • Chiplet + 3D 封裝：把更多功能塞進封裝內，減少 PCB 上 IC 數量 → 板級 PCB 簡化 → 高層數 PCB 需求降低。但 chiplet 仍需要更複雜的 substrate，TTMI 有 IC substrate 業務可受惠。
  • 軟性電子（Printed Electronics）+ Direct Write：噴墨打印電路，跳過傳統蝕刻製程；但目前僅適用於低層數低密度應用，對 TTMI 主力 (HDI / Rigid-Flex / 國防) 無威脅。

② 管理維度：什麼決策路徑會自我毀滅？

  • Capex 慣性：FY26 Capex 已從 $240M 拉到 $320M（+33%）；若 FY27 再拉到 $400M 但需求增速放緩到 +15%，會直接觸發 FCF 連續 3 年負 + 評等被降至投資級下限。歷史對標：Micron 在 2018-2019 過度投入 NAND fab，導致 2019 −60% 修正。
  • 整合失敗：Telephonics 整合花了 2 年（2022-24）才達 $12M synergies，若新一輪併購（CEO Roks 出身 Teledyne 慣於 M&A）出現整合滑點，會讓 SG&A / R&D 持續膨脹。歷史對標：HP/EDS 整合 (2008) 6 年都沒順利。
  • 創辦人退場後文化漂移：Tom Edman（前 CEO，1999 加入）2025 Sep 卸任，連續四季賣股；新 CEO Roks 才 8 個月，國防 program 高度依賴客戶關係 + 機構記憶，過渡期 risk。

③ 結構維度：什麼產業變化會讓商業模式過時？

  • 美中關係解凍 / China PCB 重新獲認證：機率低（< 5%）但若發生，IPC-1791 / Trusted Supplier 護城河消失，TTMI 失去定價 premium 5-10 pp。歷史對標：南韓 PCB 廠在 2010s 重新獲日本汽車客戶認證後，定價 premium 消失。
  • 監管型分拆：若 DoD 認定 TTMI 「太大不能倒」，可能要求其分拆軍工部門獨立上市，從而失去現有 A&D 段 cross-subsidize 商用段的能力。機率極低（< 2%）。
  • 計算範式轉換（量子 / 神經形態）：若 quantum computing / neuromorphic computing 2030 前商用化，可能徹底改變算力載板需求；但時間軸 > 10 年，影響在 thesis 範圍外。

§9.C｜競爭鴻溝

為什麼資本充足對手（Jabil $20B mcap / Sanmina $7B / Celestica $25B）仍無法跨越 TTMI 在 A&D 高階 PCB 的位置？

  • 認證 vs 投資的時間差：IPC-1791 Trusted Supplier 認證需 3-5 年累計合規記錄 + DoD 審計 + 客戶長期合作證明；對手即使現在開始建廠，2030 前都拿不到同等級認證
