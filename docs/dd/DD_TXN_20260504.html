問題回答
最可能的價值陷阱模式？非陷阱。TXN 不便宜（FPE 30x = 5Y 67% 分位 + EV/EBITDA 30x），不存在「便宜有理」前提。風險是「Cycle peak 估值倍數壓縮」+「Capex 沉重期 ROIC 暫被壓抑」雙重議題。

「可能是陷阱」的最強反駁FPE 30.74x 在 5Y 分位 67% 接近上限 + 4 週漂移 +44.17% + Bollinger 突破上軌 113.5% = 動能末段；過去 10Y peak FPE 38x（2024/9）後 18 個月股價跌至 $152 W52 低（−42%）；TXN 是強週期股，當前 EPS $5.45 為 trough，市場給 cycle peak 的 28-30x PE × cycle trough EPS 是「錯配定價」。若 FY27 Auto/Industrial 雙端轉冷（中國 EV 補貼退坡 + Industrial Capex 收斂），FY28E EPS 可能回落至 $7-8，FPE 25x 對應 $175-200（−30~−38%）。
持有期間如何判斷陷阱發生① Auto YoY 連 2 季 < +5%（中國 EV 退坡或全球 Auto 庫存調整啟動）→ H1 嚴重削弱；② Industrial YoY 連 2 季 < +10%（Capex 週期反轉前奏）→ H2 警報；③ FY+1 EPS 共識停止上修連 90 天 → cycle peak 訊號；④ FCF Margin 連 2 季 < 12%（Capex 反彈或營收轉弱）→ ROIC 兌現受阻。

最終定性：🟡 觀察期 — Analog 龍頭基本面強且兌現中，但 cycle peak FPE 30x + 4w +44% 漂移後上行空間已大幅壓縮，等回測 BB 中軌 $208 或 W52 $194 後再考慮進場。

§4｜序章：第一性原理 × 逆向思考（v12.2 三維 Munger）
A｜存在意義

Texas Instruments 解決的是「電子裝置中的類比訊號處理」這個極度分散且長尾的工業需求。Analog chip 不像數位 SoC 有「摩爾定律」，產品壽命長（5-15 年）、每顆 chip ASP 低（$0.50-$50）、但每個產品 BOM 內需要數十至數百顆，TXN 約 80,000 個 SKU 跨 100,000+ 客戶服務。這是「廣度勝深度 + 客戶鎖定 + 成本曲線」的綜合護城河。

若 TXN 明天消失，全球工業 / 汽車 / 數據中心電子產品的 BOM 需要 6-18 個月重新驗證 ADI / NXPI / MCHP 等替代品，且替代過程需要客戶端重新設計。但替代是可行的（不像 TSMC N2 沒有第二供應），TXN 的不可替代性弱於 TSMC、強於一般 chip 廠。

  • SiC / GaN 替代矽基 Analog：Wolfspeed / Onsemi 在電動車功率半導體 SiC 領先，TXN 2024 才推 SiC 產品線。若 EV 功率半導體完全 SiC 化，TXN Auto Power Analog 部分受影響。

  • 中國 Analog 國產替代：聖邦微 / 矽力傑 / 思瑞浦在中國市場低端 Analog 蠶食 TXN，但高端工業 / Auto 仍以 TXN/ADI 主導。

② 管理維度

  • Capex 過度擴張：FY22-25 累計 Capex ~$17B 是 5 年前 3-4x；若 cycle 反轉時 Sherman / Lehi 利用率不足，固定成本壓抑 margin 與 ROIC。已部分兌現於 FY24-25 GM 從 68.8% → 57%。
  • Haviv Ilan CEO 接班：2023 接 Rich Templeton，2 年 record 平淡（Capex 高峰期）；FY26 cycle 復甦是首個真正考驗。
  • Silicon Labs 收購整合：2026 5 月宣布的 Silicon Labs 收購（embedded wireless）需要 18-24 個月整合，文化 + 銷售 channel 整合風險。

③ 結構維度

  • Auto 結構性轉型：EV 普及 → BOM 變革 → Analog content per car 變動（部分增加 Power Management，部分被 SoC 整合）。淨影響中性偏正，但需 5-7 年觀察。
  • Industrial Capex super cycle 反轉：2025-27 全球 Reshoring / 美國基建 / Industrial Automation 推動 Capex；若 2028 後反轉，TXN Industrial 業務將直接承壓。
  • 地緣政治 / Export Control：中國市場約 ~25% Rev，中美科技戰升級可能影響。

價值陷阱路徑判定：四選一 — 「隱性資本密集型 + 產業結構衰退」混合型。Capex / Rev 25%+ 為結構性高，若 cycle 反轉而 Capex 已 sunk，ROIC 將被壓抑 3-4 年。但時間軸 5-7 年，非當前 18 個月議題。
