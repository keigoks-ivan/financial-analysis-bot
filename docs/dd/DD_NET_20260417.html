§2 序章：第一性原理 × 逆向思考

存在意義

Cloudflare 解決的是「網路不應該由少數壟斷者決定效能、安全、審查的底層」這個結構性問題。具體而言：

  • CDN + DDoS 防護：當 2016 年 Mirai 殭屍網路能癱瘓 Dyn，整個美東互聯網停擺；Cloudflare 的 anycast + always-on 邏輯讓攻擊流量被自動吸收到離來源最近的節點——這是「分散式抗打擊」的網路基礎建設。
  • Zero Trust / SASE：企業 IT 不再信任 VPN，需要「身分驗證 at every request」；Cloudflare 把這層挪到自家全球網路邊緣，等於在 AWS/Azure 之上建立一張「安全網」。
  • Workers / R2 / D1：開發者不想跟 Lambda cold start 打架、不想付 AWS egress 費用——Workers 0ms cold start + R2 zero egress 挑戰了 hyperscaler 的壟斷收租模式。

若 Cloudflare 明天消失：(1) 約 20% 互聯網流量需重新路由至 AWS CloudFront / Akamai / Fastly，短期必然頁面載入時間劇增、DDoS 攻擊復甦；(2) 10+ 萬家企業的 Zero Trust 政策需立即切換至 Zscaler / Palo Alto，中小企業無力負擔等價產品；(3) 免費 SSL 憑證簽發（Cloudflare 是最大的 Let's Encrypt 使用者之一）會讓數百萬小網站回到 HTTP。這是「公共基礎建設」等級的存在意義。

逆向推演（Munger Inversion）：要讓這筆投資 3 年虧 50%+ 的路徑

  • 地緣政治曝險：Cloudflare 330+ cities 分布全球，包含中國（合資 JD Cloud）、俄羅斯（2022 退出）、中東。若美中脫鉤升級至要求「全球網路基礎建設不得同時服務對手陣營客戶」，Cloudflare 的「全球網路」敘事將瓦解。機率中等但尾部風險大。
  • 技術替代風險：(1) Hyperscaler（AWS CloudFront/GCP/Azure Front Door）整合自家 AI 推論 + edge 功能，價格戰吞噬 Cloudflare 毛利；(2) 量子網路 / Starlink 級別的物理層重構使 anycast 網路優勢被新架構淘汰（2030+ 尾部風險）；(3) WebAssembly runtime 標準化後，Workers 的「獨家 runtime」優勢消失。技術替代時程 5-10 年，3 年內主要威脅來自 (1)。
  • 管理層失職：Matthew Prince + Michelle Zatlyn 雙創辦人持股 + dual-class 10x voting → 若 Prince 個人健康/退出、或過度追逐敘事（如大買併購整合失敗）、或資本配置失當（2025-06 新發 $2B 0% convert 已屬激進但合理），敘事破滅可導致一季 -30%。
  • 價值陷阱路徑：四種模式中最可能的是 護城河侵蝕型——hyperscaler 直接整合 edge + AI 推論，將 Cloudflare 定位擠回「CDN 廠商」而非「邊緣平台」。若發生，Forward PE 會從 176x 壓縮至 Akamai 等級的 14x，意味著 $15.68 目標價（-92%）。但機率 <20%，當前數字還看不見這個路徑發生中。
