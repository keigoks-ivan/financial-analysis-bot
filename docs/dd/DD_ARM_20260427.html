§2 序章：第一性原理 × 逆向思考

存在意義

ARM 解決「全球運算晶片 ISA 標準的協同性 + 長期能效競爭」這個半導體產業 30 年的根本架構性問題。從 1991 年第一顆 ARM CPU 開始，ARM 把 CPU 設計的「指令集架構」（ISA）標準化，使得任何想設計低功耗高效能 CPU 的公司都能採用 ARM IP，而不必從零開發。這個 ISA 標準在 30 年內累積了：(1) 1500萬+ 開發者；(2) 數十億顆 ARM CPU 出貨（智慧手機 99% / IoT 50%+ / Apple Silicon / AWS Graviton / NVIDIA Grace）；(3) 完整 software / OS ecosystem（Linux / Android / Apple iOS / 嵌入式系統）。若 ARM 明天消失，全球半導體產業將立即陷入「ISA 真空」——所有依賴 ARM 設計新晶片的公司（Apple, Nvidia, Qualcomm, MediaTek, Samsung, Microsoft Azure, Google, AWS）都要從零重建 ISA 標準（或集體切換到 RISC-V，但 RISC-V 缺乏 ARM 30 年累積的軟體生態）。產業重啟成本估計 $500B+，需 10-15 年才能恢復。

在 AI 浪潮中，ARM 的角色是「AI 計算的 standard ISA layer」：當 hyperscaler（Microsoft Azure Cobalt 100、Google Axion、AWS Graviton）和 NVIDIA（Grace）都選擇 ARM-based 自研 CPU（而非 x86），ARM royalty 自動跟隨 AI DC capex 擴張。每顆 hyperscaler 自研 CPU 出貨，ARM 收 ~$3-15 royalty。FY26 royalty +27% 為 AI DC ARM 採用加速的早期數據驗證。

逆向推演（Munger Inversion）

若要讓這筆投資在 3 年內虧損 50% 以上，最可能的路徑：

  • 地緣政治曝險：ARM 為英國上市（NASDAQ ADR），核心 IP 在英國 Cambridge，受英國政府保護。但 ARM China 為獨立合資（複雜歷史治理問題），佔 ARM 營收 ~20-25%。中美科技戰升級可能影響 ARM China 業務。短期地緣政治風險：中。
  • 技術替代風險：(1) RISC-V 開源 ISA：長期 5-10 年可能蠶食 ARM 部分 IoT 和 niche 市場。Meta、Tesla、Western Digital 已試 RISC-V，但目前市佔 < 3%；(2) x86 市佔回升：Intel/AMD 在 server CPU 仍主導，但 AI DC 趨勢明顯反轉至 ARM；(3) NVIDIA + 大型客戶自研非 ARM ISA：機率極低，因為生態系成本。
  • 管理層失職：CEO Rene Haas 自 2022 加入後執行紀律強，royalty rate 結構性提升 + 推出 CSS 新產品線。風險點：軟銀（持股 ~89%）為母公司，IPO 後 lock-up 結束可能持續減持影響流動性。SoftBank 已減持 ~5% 至 ~84%（FY26），趨勢需觀察。
  • 價值陷阱路徑：四類陷阱中，最可能的是「盈餘品質型」+「產業結構衰退」雙重風險——若 hyperscaler 大客戶（佔 ARM 營收 ~30%）AI capex 在 FY28-29 進入 plateau，ARM 高 PE 將難以維持，估值可能從 109x → 70x（-36% 修正）。但 royalty 模式長尾收益使 ARM 有「再來一次」機會。

§7 護城河分析

評分：9 / 10（S 級護城河，全球半導體 ISA 標準 30 年壟斷者）

護城河來源

  • 網路效應：✅ 極強。1500萬+ 開發者基於 ARM ISA 開發軟體，使任何想設計新 CPU 的公司都選擇 ARM（避免重新建立軟體生態）。網路效應強度為半導體業最強。
  • 無形資產：✅ 極強。ARM 持有數萬+ 專利（涵蓋 CPU 設計、ISA 擴展、power management），ARM v9 architecture 是行業標準。Apple、Qualcomm、MediaTek、Samsung 等大型 chip 設計公司若不使用 ARM 必須完全重建軟體 stack。
  • 轉換成本：✅ 極強。Hyperscaler 客戶設計 ARM-based CPU 需 3-5 年驗證 + 大量軟體 porting。一旦投資完成，切換到 RISC-V 或 x86 成本 $500M+ 且需 5+ 年。NRR 估 ~110%+（同客戶 royalty 持續成長）。
  • 規模優勢：✅ 強。$4.5B revenue 提供 R&D ~$1.2B / 年（vs RISC-V Foundation 整體 ~$50M / 年），R&D 規模差距 ~24x，使 ARM 持續創新領先。

競爭鴻溝

為什麼資本充足的對手仍無法跨越？

  • RISC-V 開源 ISA：理論上「免費」，但缺乏 ARM 30 年累積的軟體生態、開發者工具、performance 優化。當前市佔 < 3%，主要在 niche（IoT、嵌入式、Western Digital storage controller）。短期 5 年無法挑戰 ARM 主流市場。
  • x86 (Intel/AMD)：在 Windows desktop / server 仍主導，但 AI DC 趨勢明顯反轉至 ARM（Microsoft Cobalt、Google Axion）。能效劣勢使 x86 在 AI DC 退守。
  • NVIDIA / 大型客戶自研 ISA：NVIDIA 已選擇 ARM（Grace），AMD/Intel 反對 ARM 但被 hyperscaler 客戶推著走。Apple / Qualcomm 都是 ARM 大客戶，無動力自研。
  • 中國自研 ISA（LoongArch / Sunway）：中國以外無生態，僅地緣政治考量下使用。
