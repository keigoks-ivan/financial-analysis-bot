③ 結構維度（v12.2 新增）：什麼產業變化會讓商業模式過時？

  • 中國平台輸出（TikTok + Temu + SHEIN）規模優勢：若中國 e-commerce 在 SEA 取得 50%+ 市佔，Shopee 從 #1 變 #2 → 估值倍數系統性下移。歷史對標：日本電商 Rakuten 在亞馬遜進入後 30 年從 #1 → #4。

SE 下游分五類，整體議價力結構性 mixed — Shopee 消費端中等 / VIP 強 / Garena whale 高 / Monee 借款人強。(a) Shopee 一般消費者：50% SEA 市佔 + 5 億+ MAU 形成 default e-commerce app，但 TikTok Shop 25-30% / Lazada 15% / Tokopedia（印尼）alternative 存在，多 platform shopping 已成 reality → 議價力中等（用戶可比價 + 換 app）；mitigated by SPX 物流速度差異化 + VIP 訂閱黏著。(b) Shopee VIP 訂閱者（10M+ users, retention 80%+，QoQ +40%）：差異化服務（免運費門檻、退貨優惠、early access deals）已建立 brand loyalty + sunk cost，議價力強 — Amazon Prime SEA 翻版。(c) Garena 玩家：Free Fire 14 億累計安裝 = sunk cost + in-game social network friction，轉換成本高；但 whale 經濟 ~5M paying users 集中 ~50% bookings 為 dual-edged — single IP age 是 binary risk（§5.F），議價力高但 fragile。(d) Monee 貸款 / 銀行客戶：SEA underbanked 1.5-2 億人口的 entry point，信用記錄 + 自動扣款 = high friction switch；vs 傳統銀行的 less convenient experience → 議價力強。(e) Shopee 廣告主：Q1 26 廣告 Rev +80% YoY 證實廣告主 buying，但 ROI vs TikTok / Meta 廣告需持續證明，議價力中等。
