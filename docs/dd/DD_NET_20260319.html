§0 ｜ 序章：第一性原理 × 逆向思考

存在意義（第一性原理）

Cloudflare 解決的核心問題可以用一個問題來提問：「當每一個 HTTP 請求都需要經過全球網路傳輸時，誰來確保它安全、快速、且不被中間人攔截？」

Cloudflare 擁有 330+ 個全球 PoP（Points of Presence），覆蓋全球 95% 以上的網際網路用戶（所有人都距離 Cloudflare 節點在 50ms 以內）。這個網路已成為全球約 21.3% 的網站的「無形守門人」——每一次用戶造訪這些網站，流量都會先經過 Cloudflare 的邊緣節點進行安全過濾、CDN 加速和負載均衡，再傳回到源伺服器。

2026 年的戰略轉折點：Cloudflare CEO Matthew Prince 在 Q4 2025 財報中宣布了一個改變投資邏輯的核心論點——「如果 AI Agent 是新一代的網路用戶，那 Cloudflare 就是他們在上面跑的平台，以及他們必須通過的網路。」這不是比喻，而是可量化的事實：2026 年 1 月，AI Agent 在 Cloudflare 網路上生成的週請求量相比六個月前翻倍，且相比人類驅動的請求已達「數量級」差距。

若 Cloudflare 明天消失：全球 21.3% 的網站將立刻失去 DDoS 保護、CDN 加速和 SSL/TLS 終結功能；數百萬使用 Workers 和 Workers AI 的開發者應用程式將斷線；超過 80% 的頂尖 AI 公司（依賴 Cloudflare 的 AI Gateway 和推論邊緣）的 AI 流量管理將中斷。替代方案（Akamai、AWS CloudFront、Azure CDN）雖存在，但「安全+性能+開發平台三合一的邊緣作業系統」無法立即替換。

逆向推演（Munger Inversion）——讓投資 3 年虧損 50%+ 的路徑

① AI Hyperscaler 直接搶佔邊緣推論市場 最大威脅

AWS Lambda@Edge、Azure CDN with AI、Google Cloud Edge Network 都在向「邊緣推論」推進。若 Amazon 或 Microsoft 開放其邊緣節點給外部 AI 推論，並以服務捆綁（已有 AWS、Azure 使用者的低摩擦遷移）削價競爭，Cloudflare Workers AI 的成長可能大幅受限。具體路徑：AWS 宣布 Lambda@Edge 支援原生 LLM 推論（無需遷移至 Cloudflare Workers），使 4.5M 開發者社群的「邊緣 AI」遷移動力消失。
反制因素：Cloudflare 目前的開發者體驗（DX）領先 AWS 至少 2-3 年。AWS 的邊緣服務以複雜著稱，而 Workers 以簡潔聞名。4.5M 活躍開發者形成了強力的網路效應護城河。但若 AWS 將 AI 工具鏈（Bedrock）深度整合進邊緣，切換成本可能被軟化。

② AI 導致網路安全「自動化化」——減少人為採購決策 中期結構風險

部分市場恐慌認為 AI 編碼工具（如 Anthropic Claude Code Security、GitHub Copilot Autofix）可自動修復程式碼漏洞，使傳統的「安全即服務」需求減少。這個邏輯如果成立，Zero Trust 安全市場（Cloudflare One 的核心）會從「必須買」變成「可選補充」，壓縮 ACV 成長。
我們的反駁：Cloudflare 的安全護城河在於「網路層安全」（DDoS、DNS 過濾、Bot Management），這是 AI 工具無法取代的——因為這些是在數據抵達任何伺服器之前就發生的過濾，需要全球分散式節點，不是程式碼修復可以解決的問題。但 SASE/Zero Trust 的銷售週期確實可能受 AI 恐慌情緒影響而延長。

③ 成長降速至 <20% YoY，高估值多重壓縮 估值最大尾部風險

NET 以 EV/Sales ~24× 定價隱含「長期保持 28-30%+ 的收入成長」。若宏觀 IT 預算緊縮（企業普遍凍結非必要支出）或 AI 帶來的網路效率提升使客戶「以更少的 Cloudflare 服務做更多事」，成長降速至 20% 以下，估值可能從 24× 壓縮至 12-15×，導致股價從 $220 跌至 $110-140。這個情境在 2022 年曾成真（NET 從 $182 跌至 $38）。

④ CEO 刑事調查（西班牙足球轉播版權案）目前低概率，需監控

馬德里法院已傳喚 Matthew Prince（CEO）於 2026/04/07 到庭，以「嫌疑人」身份說明 Cloudflare 是否協助非法足球轉播（西班牙聯賽版權爭議）。目前定性為「協助調查」，非直接犯罪指控，且 Cloudflare 作為中立基礎設施提供商的法律地位有保護。但若情況升級，可能影響管理層專注度並引發機構投資者的風險規避反應。

  • AI Agent 流量爆炸：「2026 年 1 月，AI Agent 在 Cloudflare 網路上生成的週請求量超過傳統人類驅動請求的一個數量級，且比六個月前翻倍。」——這是 Cloudflare 宣稱自己是 Agentic Internet OS 的硬數據依據

  • 「Pay-per-Crawl」新模式：2025 年底推出的新收入流——幫助內容擁有者向 AI 爬蟲收費，Cloudflare 抽取交易費用，類似 AI 時代的「通行費收取者」

刻意迴避的問題

  • Workers AI 具體收入佔比：分析師多次追問，管理層只說「成長強勁但佔整體比例不大，不單獨披露」——這讓市場無法量化 AI 收入的實際規模
  • CEO 西班牙法律案：在 Call 中未提及。分析師也未追問（可能受限於法律顧問建議）
  • R2 和 Workers AI 的 Unit Economics：管理層說「毛利率的邊際差異來自產品組合」但拒絕細分各產品毛利率

③ 產業 Imply 萃取

需求端：AI Agent 正在重塑網路流量結構，且 Cloudflare 是最直接受益者

Cloudflare 所披露的「AI Agent 流量翻倍」是整個 AI 基礎設施行業最重要的前瞻信號之一。這意味著企業不只是在建更多 AI 應用，而是這些 AI 應用已在實際執行大量自動化任務（網路請求）。對同業的推論：Fastly（FSLY）的邊緣 CDN 應也受益，但規模和開發者平台深度遠不及 Cloudflare；Akamai（AKAM）雖體量更大，但在開發者生態系和 Workers AI 等 AI-native 功能上落後。

競爭格局：Zero Trust 市場的鏈接創造了「超級護城河」

Cloudflare 的 DBNRR 120% 意味著現有客戶平均每年多付 20% 的費用，且連續上升（Q3 119% → Q4 120%）。在企業 IT 支出趨於謹慎的環境下，DBNRR 上升是護城河擴大的明確信號——客戶不只是黏著，而是在持續升級到更多 Cloudflare 的服務。對 Zscaler（ZS）是中性信號（兩者均受益於 Zero Trust 採用）；對 Palo Alto Networks（PANW）的 Platformization 策略也是正面信號（企業願意在單一平台增加支出）。

對總體經濟的敏感度

Cloudflare 是中高度宏觀敏感股——企業 IT 預算收縮會放緩大客戶的合約擴張和新客戶的採購決策。但其核心基礎設施屬性（許多客戶「刪掉 Cloudflare 就斷網」）使 churn 率維持在較低水準。關稅/貿易戰衝擊對 Cloudflare 的直接影響很小（純軟體服務，無實體供應鏈），但對客戶 IT 預算的間接抑制效應是潛在逆風。

護城河來源詳析

① 全球邊緣網路規模：最難複製的物理護城河

Cloudflare 在全球 330+ 城市的存在點（PoP）讓它的邊緣節點密度超越 AWS、Azure、GCP 各自的 CDN 網路——但關鍵不只是數量，而是全球一致性：開發者寫一份 Workers 代碼，Deploy 後自動在全球 330+ 節點同步運行，95% 的全球網路用戶都在 50ms 之內。這是 AWS Lambda 無法複製的——AWS 讓開發者手動選擇 Region，而 Cloudflare 的全球化是「零配置」的。複製 Cloudflare 的全球網路需要數十億美元和 5-10 年的佈建時間，且需要與各國電信商建立直接互連（IX）協議，這是一個結構性進入障礙。

② 開發者生態與網路效應：4.5M 開發者的飛輪

Cloudflare Workers 擁有 4.5M 活躍人類開發者，是 AWS Lambda 的直接競品（AWS Lambda 的開發者估計超千萬，但大多綁定 AWS 生態）。Workers 的獨特優勢是「去中心化部署體驗」——一個指令，全球。這個開發者社群形成了強烈的網路效應：更多開發者 → 更多第三方整合 → 更多應用生態 → 吸引更多開發者。每個在 Workers 上建應用的開發者都把他們的應用鎖定在 Cloudflare 的基礎設施上（Workers API 不相容於 AWS Lambda 或 Azure Functions），形成高轉換成本。

③ 數據位置護城河（位於一切流量的前端）：獨一無二的情報優勢

Cloudflare 每天處理全球 45M+ HTTP 請求/秒，看到全球網路上幾乎所有類型的攻擊流量。這個「全球威脅情報」讓 Cloudflare 的安全產品（Bot Management、DDoS、WAF）持續學習並超越競爭對手——每個新型攻擊在 Cloudflare 看到後，立刻更新所有客戶的保護規則。Zscaler 或 Palo Alto 沒有這個規模的「主動情報飛輪」。Morningstar 分析師明確指出：Cloudflare 的 Moat 是 Wide（寬護城河）。

④ Agentic Internet 的戰略卡位：2026-2030 年最重要的新護城河

AI Agent 在網路上的請求量按指數成長，每個 AI Agent 每次執行任務都需要：安全的 HTTPS 連接（Cloudflare SSL/TLS）、DDoS 保護（避免被攻擊）、快速的 DNS 解析（Cloudflare DNS，全球最快的公共 DNS）、邊緣推論（Workers AI，讓 Agent 在距離用戶最近的地方執行 AI）、以及 Agent 之間的安全通訊（Cloudflare Workers 的 Service Binding）。換言之，如果 AI Agent 是「新用戶」，那 Cloudflare 是他們「上網」必須通過的每一個路口。「Pay-per-Crawl」這個新功能更讓 Cloudflare 成為 AI Agent 的「收費站」，從流量中獲取交易費用。

競爭鴻溝——為什麼 AWS、Azure、GCP 無法輕易複製？

超大規模雲端的核心商業模式是「把更多計算引入自己的數據中心」，而 Cloudflare 的模式是「把計算推到離用戶最近的地方」——兩者在商業邏輯上衝突。若 AWS 把邊緣推論做得更便宜更簡單，它實際上在 cannibalize 自己的 EC2 和 Lambda 業務。這是大型企業的創新兩難困境（Innovator's Dilemma），是 Cloudflare 的時間護城河。

獲利品質警訊

① 淨利 vs. OCF：Non-GAAP 健康，GAAP 持續虧損但改善中

FY2025 GAAP 淨虧損 $102.3M，但主因是 SBC（非現金費用）。OCF 趨勢持續改善：Q4 2025 FCF $99.4M（16%），已連續數季大幅超越 GAAP 淨利，顯示現金業務質量高。GAAP 虧損收窄（FY2023: -$183.9M → FY2024: -$78.8M → FY2025: -$102.3M，FY2025 小幅惡化因加速投資）。

② RPO 與 Current RPO 加速：最強的前瞻性財務信號

Q4 2025 RPO $2.496B（+16% QoQ，+48% YoY）；Current RPO（未來 12 個月）佔總 RPO 63% 且 +34% YoY。RPO 加速是 Cloudflare 未來 12-18 個月收入能見度最高的信號——這筆已籤約的金額代表即使一分錢新合約不簽，2026 上半年的收入也基本有保障。

③ Gross Margin 微幅下滑：⚠️ 需要監控

Q4 2025 Gross Margin 74.9%（低於長期目標 75-77%），相較 FY2024 全年約 76.2% 小幅下降。主因是 Workers AI 等計算密集型產品佔比上升（需要 GPU 成本）。管理層稱「各業務線的 Unit Economic Margin 一致」，但混合毛利率承壓將是未來幾季的觀察重點。若 AI 推論產品持續快速成長但毛利率不及傳統 CDN/安全產品，Gross Margin 可能進一步下滑至 72-74%，需對 GAAP/Non-GAAP 利潤率預期重新校準。

④ FCF 轉換率快速提升：核心財務改善信號

Q4 FCF Margin 16%（vs 一年前 10%），顯示 Operating Leverage 正在實現。FY2026 管理層「對 FCF 共識感到滿意」——暗示 FCF 成長路徑比 Non-GAAP EPS 更健康（管理層對 EPS 指引刻意保守，但對 FCF 不保守）。FCF CAGR（FY2024→FY2026E）預計超過 50%，是所有財務指標中最亮眼的。

議價權

對上游（AWS、Azure 等 IaaS）：Cloudflare「不擁有數據中心，只租空間」的模式使其對 IaaS 供應商有一定依賴，但通過多元化供應商關係和自建網路互連（IX）降低了單一依賴。R2 Storage 以「零 Egress 費用」直接對抗 AWS S3 的「數據引力」護城河，顯示 Cloudflare 有意識地切斷 Hyperscaler 對客戶數據的鎖定。

對下游（企業客戶）：議價能力持續提升。DBNRR 120% 表明客戶不只不離開，而是主動增加支出。最大年度合約 $42.5M（FY2025 Q4）說明 Fortune 500 級別的大客戶願意簽署高價值長期合約。「Pool of Funds」合約創新更使客戶預付大額服務包，提前「投票」給 Cloudflare。

Unit Economics

Free → Paid 轉化漏斗：Cloudflare 擁有數百萬免費用戶（包括個人網站主和開發者），轉化為付費的漏斗模式是其低 CAC 的秘密。4.5M Workers 活躍開發者中，大多數目前使用免費額度——未來付費化是巨大的未開發收入潛力。

大客戶 ($100K+/年) 的 Unit Economics：4,298 大客戶貢獻 73% 收入（$1.58B），平均每客戶 ACV 約 $367K/年。每新增一個大客戶：CAC（銷售週期長，企業級銷售成本高）vs. LTV（平均客戶 DBNRR 120%，意味著每年增加 20%，LTV 在留存率 90%+ 假設下極高）——LTV/CAC 比率估計超過 5-8×，非常健康。

Workers AI 的 Unit Economics（新興）：按 Token 計費，邊際成本主要是 GPU 使用費（Cloudflare 使用 NVIDIA GPU 在全球 PoP 部署）。由於 GPU 成本尚高，Workers AI 目前是低毛利率甚至負毛利率的吸引開發者的策略產品，預計 FY2027+ 才能實現正貢獻。

§8 ｜ 投資結論與情境預測

財報信號與估值銜接

Q4 2025 財報（2026/02/10）是 Cloudflare 歷史上最全面的超越季度：Revenue +4% vs 指引、RPO +48% YoY、ACV +50%（2021 年後最快）、FCF Margin 翻倍至 16%、DBNRR 回升至 120%。這份財報確認了三件事：（1）企業大客戶採購正在加速；（2）Agentic Internet 已從敘事變成可量化的流量數據；（3）Operating Leverage 正在實現。

財報後股票飆漲 16%，估值從 ~20× EV/Sales 擴張至 ~24×。目前股價 $220 隱含 FY2026 EV/Sales 26.9×——已計入了很多樂觀預期。中性 DCF 合理價 $210 顯示僅有微幅上行空間，主要的投資機會來自：（a）Agentic Internet 超預期（樂觀情境 +50%）；（b）回調後的入場機會（目標 $170-190）。

最終結論

🟡 持有 / 等待理想建倉點——目標回調至 $170-190 建立初始倉位（倉位建議：5-8%）

Cloudflare 是 AI 時代互聯網基礎設施演進的核心受益者，其「Agentic Internet 作業系統」的戰略定位有充分的技術和市場基礎——物理層面的全球 330+ PoP 邊緣網路、4.5M 開發者生態、20%+ 網路覆蓋、以及 AI Agent 流量爆炸的真實數據，全部指向「NET 正在從 CDN 公司轉型為 AI 時代的互聯網基礎設施核心」。但當前股價 $220（EV/Sales ~24×）已充分定價了這個樂觀情境，3 年概率加權回報僅 3.5%，沒有足夠的安全邊際。

最佳策略：未持倉者在 $170-190（EV/Sales 約 19-21×，接近 CrowdStrike 估值）區間建立 5-8% 初始倉位，下跌至 $150-160 可二次加碼；已持倉者維持持有，設定 Q1 2026 財報（2026/04/30）為再評估節點。

最關鍵監測指標

核心指標：RPO（剩餘履約義務）季度年增率 + Current RPO 佔比趨勢（2026/04/30 財報）

RPO 是 Cloudflare 最領先的前瞻指標——它衡量客戶的「未來 Cloudflare 支出承諾」，是 DBNRR 改善或惡化的先行指標。Q4 2025 RPO +48% YoY 是強勢信號，Q1 2026 需要維持 +35%+ 才能支持當前估值。

觸發重新評估的閾值：

  • 📈 正向信號（考慮提前建倉/加碼）：Q1 2026 RPO +40%+ YoY（加速）+ Revenue 上超指引超過 2%（即 >$632M）+ Non-GAAP FCF Margin 維持 16%+。這將確認 Agentic Internet 敘事正在轉化為實際收入，估值溢價有進一步擴大的基礎。目標股價修正至 $260-280（6 個月）。
  • 📉 負向信號（考慮等待或減倉）：Q1 2026 Revenue 低於指引（$620M）+ RPO 成長降速至 <30% YoY + Gross Margin 降至 72% 以下。這將確認 Workers AI 成本拉高毛利率的結構性壓力，股價可能回調至 $160-180，是理想的初始建倉機會。

次要指標：CEO 西班牙法律案（2026/04/07 聽訊）——若升級為系統性刑事追訴（低概率），可能影響 Matthew Prince 的管理專注度。目前定性為個人調查，對業務無直接影響，但需每季度追蹤。
