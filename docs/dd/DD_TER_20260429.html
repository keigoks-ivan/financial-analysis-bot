§2｜序章：第一性原理 × 逆向思考

存在意義

Teradyne 解決的是「如何驗證每一顆數位、類比、混合訊號 IC 在出廠前都能在規格內運作」這個半導體業必經的物理閘道。從 Apple A19 / NVIDIA GB200 / AMD MI300 / Broadcom Tomahawk 5 / Marvell ASIC，到 HBM 記憶體、Mobile RF 前端、Auto MCU — 全部要過 ATE 測試，否則無法量產出貨。Teradyne 若明天消失，全球 leading-edge AI chip 量產會在 6-9 個月內中斷（Advantest 雖能部分 take over，但 capacity ramp 需 12-18 個月，且 SoC 測試 IP 切換需重新驗證）。這不是「測試效率」 — 是 integrated circuit-to-end-customer 的 last-mile gatekeeper。其護城河源於：① ATE 設備平台 IP 累積（J750、UltraFLEX、IG-XL）30 年；② 客戶 toolchain lock-in（每個 chip product line 都需 fixture + test program 開發，切換成本 $5M+/chip）；③ ATE 設備 ASP $2-5M / 機台，duopoly 結構 + 客戶忠誠度極高。

逆向推演（Munger Inversion）

若要讓這筆投資在 3 年內虧損 50% 以上，最可能的路徑：

  • 地緣政治曝險：TER HQ Boston，但客戶 70%+ 在亞洲（TSMC / Samsung / 中國 fabless）。最大風險：① 中美 ATE 出口管制升級（如 Advantest 已被部分管制），使 TER 失去中國 ~25% revenue；② 台海衝突中斷 TSMC 客戶採購週期；③ 中國本土 ATE 公司（華峰測控、長川科技）+ 政府推進「半導體設備國產化」政策，搶 TER mid-end SoC 測試業務（已開始 +5-10% 中國市佔）。時間軸 2026-2028 是觀察窗。
  • 技術替代風險：① Built-in Self Test (BIST) 取代外部 ATE：高階 chip 內建測試 IP（如 ARM Memory BIST）可能減少 ATE 需求；② On-die test （Aurora）+ Wafer-level test：TSMC、Samsung 推動 wafer-level test 取代 package-level，降低 ATE 設備需求頻率；③ AI-driven test acceleration：若 chiplet design + advanced packaging 大量普及，每片 wafer 測試時間縮短 30-50%，ATE 設備需求 / chip output ratio 下降。時間軸 2027+。
  • 管理層失職：① CEO Greg Smith（接任 Mark Jagiela 約 2 年）尚未通過完整 cycle 測試；② 資本配置：FY 2025 retornar 174% FCF（$785M / $450M FCF）= 過度回購（在 $200-300 區間買回），若 2026 H2 cycle pullback，回購均價 $250 vs 跌至 $150 場景，capital allocation 質疑；③ Robotics 整合（Universal Robots + MiR）5+ 年後仍 operating loss，可能成為 capital trap，類似許多半導體公司「multi-segment 拼湊」失敗案例。
  • 價值陷阱路徑：四選一中最可能的是「產業結構衰退型變體」+ 「估值體系下修」雙重打擊。具體情境：FY 2026 是 AI ATE peak earnings，FY 2027 EPS $8.70 兌現後，FY 2028+ growth 進入 single-digit + cyclical pattern；市場意識到後，PE 從 cycle peak 30-40x 壓回 mid-cycle 18-22x；股價可能從 $380 → $200（-47%）。觸發條件：H2 26 沒有第二波 AI catalyst + Advantest 在 memory test 領先。
