  • HBM 標準化 / commoditization：若 HBM4 / HBM5 spec 高度標準化使 metrology 需求降低（從「每片 wafer 100% 檢測」降至「sample-based」），會壓縮 ONTO ASP 與量。短期可能性低（HBM5 仍在 spec 階段）。

ONTO 上游供應鏈為 fabless 模型，主要採購項目分四類：(a) 專屬零組件（lasers / optics / precision motion stages），主要供應商為 Zeiss / Hamamatsu / Newport / Aerotech，ONTO 採用多元化採購策略（每類組件至少 2-3 家供應商），議價力中等偏強，年採購額 ~$300M+，FY25 採購成本通膨壓力 < 5%；(b) EDA 工具（Synopsys / Cadence）為產業標準，議價力低，年費 ~$15M，所有 IC 設計公司同樣承受；(c) TSMC 先進製程代工（Atlas G6 自身用 N5/N3 chip），ONTO 年代工費 < $50M wafer 規模相對小於 NVIDIA / Apple，議價力低，但 TSMC 對中小客戶仍以服務為主，無重大議價損失；(d) Rigaku 27% 戰略股權後，透過股權合作可獲得 X-ray metrology IP 優先採購權，降低未來 hybrid metrology cross-sell 採購成本，這是 2026-05-05 $710M deal 的戰略邊際效益[businesswire 2026-04-20]。整體上游議價力中等偏強，AI 順風期間供應商需求充裕使 ONTO 議價力進一步強化。

② 對下游客戶（議價力：中等，dual-track 結構稀釋單一客戶風險）

ONTO 下游客戶高度集中於全球 top-10 半導體 foundry + memory 大廠：SK Hynix（HBM4 量產主驅動，估佔 12-15% Rev） + TSMC（CoWoS + N3/N2 logic foundry，估佔 10-15%） + Samsung Foundry / Memory（5-10%） + Micron（HBM4 H2 26 量產，5-8%） + Intel Foundry / GlobalFoundries（合計 5-8%） + 中國 fab（SMIC / YMTC 等合計 ~10%，受 export control 限制）。客戶集中度 top-5 ~50-55%（v12.3 §8.H 揭示），議價力中等：fab 對 metrology bug 容忍度極低（一個 bug = $50M+ wafer 報廢），更看重 reference platform 信任 vs 純價格 → ONTO 在 ASP 議價力強（連續代際 +20-50% ASP uplift）；但 SK Hynix HBM4 集中度上升使 ONTO 對 SK Hynix 議價力下降，且 fab 維持 2 家以上 vendor pool（dual-track）使 ONTO 不能單方面強加 pricing。Recent case：2026 Q1 NG GM 76% 維持但 GAAP GM 49.7%（受整合費用拖累），顯示產品 IP 議價力仍領先（76% NG）但成本壓力傳導使 GAAP 下行。

③ 地緣政治曝險（議價 / 監管：中度，美中 export control 為主要結構性風險）

ONTO 地理收入分布：北美 ~45%（美國 fab + 客戶 — Intel Foundry / Micron / Applied）、亞太 ~40%（台灣 TSMC + 韓國 SK Hynix/Samsung + 日本 Micron Hiroshima/Rigaku 合作 — HBM4 + CoWoS 量產基地）、中國 ~10%（SMIC / YMTC / 長江存儲，受美國 export control 部分限制）、歐洲 ~3%（GlobalFoundries Dresden）。Recent case：2025-12 美國商務部更新 EAR（Export Administration Regulations）擴大 advanced node metrology 出口限制清單，ONTO 主動避免敏感技術（Atlas G6 不對中國 fab 出口 N5 以下節點配方）但 mature node metrology 仍可出口，FY25 中國 Rev 持平未顯著下滑。台海地緣風險（TSMC + 整個亞太供應鏈集中）為間接風險，若衝突發生 ONTO 短期 Rev 可能下行 30-40%；但 ONTO 已啟動 US/Israel/Japan 多元化生產佈局（Wilmington MA + Israel + Japan via Rigaku）部分緩解。整體地緣議價結構性壓力為中度，AI 順風使政策風險暫時被 demand 蓋過，但 2027-2028 美國總統選舉後 export control 可能再緊縮為主要不確定性。
