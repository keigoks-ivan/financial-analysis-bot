§4｜序章：第一性原理 × 逆向思考

存在意義

ONTO 解決的核心痛點是「AI advanced packaging（HBM、CoWoS、2.5D/3D logic）的訊號完整性、缺陷檢測與訊號分析」。當 GPU + HBM + CoWoS interposer + organic substrate 堆疊在 5-8 層 die 上時，每一層的對位精度（overlay）、bonding 缺陷、through-silicon vias (TSVs) 缺陷必須在量產前 100% 檢測——否則 1% 缺陷率會導致 $30K+ 的 GB200 GPU 整顆報廢。

逆向推演（Munger Inversion）三維展開

① 技術維度：什麼技術替代會讓 ONTO 3 年內失去 50% 價值？

  • Camtek 在 HBM4 metrology 全面 reference 化：Camtek 已是「tool of reference for 3D metrology at all major HBM players」+ FY25 HPC 占 45-50% Rev + Hawk/Eagle G5 platforms 從 2025 30% Rev → 2026 50%+ Rev 路徑。歷史對標：KLA vs Lasertec 在 EUV mask inspection 之爭 — Lasertec 從 2018 起 ramp 後 5 年取得壟斷地位。若 Camtek 同樣以 HBM4 reference 切入下一代 HBM5 邏輯複製，ONTO 在 3D metrology 子市場可能流失 30-40% 份額。
  • KLA / Applied Materials 跨界 advanced packaging：KLA 持有 17.5% metrology + inspection 市佔（產業最大），若全力進入 advanced packaging，ONTO 在 high-end 應用會被擠壓。歷史對標：KLA 進入 NAND inspection 後 5 年取代 Tokyo Seimitsu 領先地位。
  • Silicon photonics packaging 取代部分電子封裝：若 CPO（Co-Packaged Optics）2028 H2 量產，部分 GPU + optical interconnect 場景會繞過 ONTO 的 PCIe Gen 6/7 retimer/AEC 量測 use case。歷史對標：Intel 光通訊量產嘗試 2015-2020，量產延後 5+ 年。對 ONTO 影響：5+ 年外，目前不影響近期。
  • Dragonfly G5 ramp 失速：若 G5 在 Q3-Q4 26 ramp 卡關（每季 nearly doubling 路徑停滯），會直接抹去 H2 thesis。

② 管理維度：什麼決策路徑會自我毀滅？

  • 連續 2 起大型 M&A 整合失敗：Semilab USA（2025-11，$432M closed）+ Rigaku 27% 戰略股權（2026-05，$710M 公告）。短時間連續執行 ~$1.14B M&A，在過去 ONTO（中型公司，FY25 Rev $1B）很罕見。歷史對標：Marvell-Inphi（2021，$10B）整合 3 年後仍未完全發揮綜效；KLA-Orbotech（2019，$3.4B）整合期內 ROIC 從 25% → 18%。ONTO 同期整合 2 個標的，2026-2027 是執行高風險期。
  • R&D / Capex 慣性：FY25 R&D ~$215M（21% Rev）vs FY24 ~$160M（16% Rev），上升幅度大但 Rev 成長僅 +1.8%。若 FY26-27 維持 R&D +25%/年但 Rev 成長 < 30%，會侵蝕 OPM。歷史對標：Lam Research 2018-2019 R&D 慣性使 OPM 從 28% → 22%。
  • 創辦人 / 主管離職：CEO Michael Plisinski（2019 至今）+ CFO Mark Slicer（2021 至今）為合併後核心管理層。Rigaku 整合期間若任一離職將觸發市場信心問題。

③ 結構維度：什麼產業變化會讓商業模式過時？

  • AI capex 進入 plateau（FY28-29）：若 hyperscaler AI capex 在 FY28 開始 plateau（從 +30% YoY 降至 +5%），ONTO 高 PE 將難維持。歷史對標：WFE 2018-2019 cycle 結束 ONTO 股價 -50% in 12 months。HBM 供需在 2027-2028 會從緊缺轉平衡，是 ONTO 估值倍數壓縮的最大結構性風險。
  • HBM 標準化 / commoditization：若 HBM4 / HBM5 spec 高度標準化使 metrology 需求降低（從「每片 wafer 100% 檢測」降至「sample-based」），會壓縮 ONTO ASP 與量。短期可能性低（HBM5 仍在 spec 階段）。
  • 計算範式轉換（量子 / 神經型計算）：5+ 年外，若部分 AI 訓練移轉到量子 / 神經型晶片，會改變封裝量測需求。歷史對標：2010s GPU 取代 CPU 訓練 → NVIDIA 取代 Intel 領導地位。對 ONTO 影響：5+ 年外。

價值陷阱路徑判定：四類陷阱中，最可能的是「盈餘品質型」+「產業結構衰退型」雙重風險（Camtek HBM4 reference 持續 + AI capex plateau + Semilab/Rigaku 整合費用侵蝕 EPS）。短期最大風險為「估值 mean reversion + 整合執行不順」雙重壓力。

§9｜護城河分析（評分 8/10）

護城河來源

  • 網路效應：弱（IC 設計 / 設備公司無傳統網路效應），但有「fab 採用網路」效應 — 一旦 TSMC N2 reference platform 採用 Atlas G6，所有 logic 客戶（NVIDIA / AMD / Apple / Qualcomm）都跟進。Q1 26 客戶名單包含 TSMC + SK Hynix + Samsung + Micron + 4 家 logic foundry。
  • 無形資產：強（Dragonfly G5 + Atlas G6 + Iris combined IP；Rudolph + Nanometrics 30+ 年 metrology IP 累積）。
  • 轉換成本：強（fab 一旦 design-in 一個產品代別，會延續 2-3 代；driver / firmware / process-of-record 整合成本 + 時間 12-18 月；qualification 過程 6-9 月）。
  • 規模優勢：中（fabless 模型對 TSMC / Samsung 議價力非首位，但晶圓需求快速增長使供應商議價弱化；R&D 規模 $215M+/年使後進者難追趕，但仍小於 KLA $1.5B+）。

競爭鴻溝

為什麼 KLA（市值 $130B+）、Camtek（市值 $5B）、Lasertec 等仍無法跨越？

  • 多代產品 IP 連續性：Atlas / Dragonfly / Iris 不是單代設計，需要從 G3 → G4 → G5 → G6 連續迭代。Camtek 在 advanced packaging 2D inspection 領先但 3D metrology / advanced nodes 不全；KLA 主攻 front-end，advanced packaging metrology 後進。
  • Fab 信任壁壘：fab 對 metrology bug 容忍度極低（一個 bug = $50M+ wafer 報廢成本），需要 2-3 個成功代別才建立信任。ONTO 在 Atlas G3-G5 累積信任後 G6 自然延續。
  • 多平台整合（Rudolph + Nanometrics + K&S Lithography + Lumina + Semilab + Rigaku）：跨產品線 cross-sell 是後進者難以複製的工程整合。
