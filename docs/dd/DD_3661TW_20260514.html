§4 序章：第一性原理 × 逆向思考

存在意義

Alchip 解決的問題不是「設計 AI 晶片」，而是「當 hyperscaler 想自研矽，卻沒有完整的 backend physical design + advanced packaging integration 能力時，誰能幫他們把架構規格從 RTL 走完到 wafer-out」。在 2026 的世界，AWS 有 Annapurna（front-end SoC architecture）、Google 有 TPU 內部團隊、Meta 有 MTIA 團隊，但 backend 階段（physical implementation、CoWoS interposer 整合、CoWoS / SoIC 配額分配、yield optimization、IP 採購整合）需要的是「10+ 年 ASIC tape-out 經驗 + TSMC 配額議價權 + Synopsys/Cadence IP 採購規模優勢」的設計服務 partner。Alchip 在 ASIC 設計服務鏈中扮演的角色是「AWS 自研矽從 spec 到 chip 的 backend executor」 — 若它消失，AWS Trainium 3 的 RDL interposer 工程能力（Marvell 已驗證做不好）將找不到第二個能在 6 個月內接手的設計服務商，AWS 2026 H2 整個 Project Rainier 將延後 1-2 季。這個價值是不可被替代的「capacity-allocated execution capability」。

逆向推演（Munger Inversion）— 3 年內虧損 50% 路徑

  • 地緣政治曝險：總部新竹 + 設計團隊 80%+ 在台灣，極端兩岸情境下 design tape-out 中斷直接擊穿 AWS H2 Trainium 3 量產。但這也是 ASE / TSMC / AVGO（透過 TSMC）的共同風險，非個股獨有。
  • 技術替代風險：Annapurna 自製化加速 — 2025 Q1 Trainium 3 Mariana 變體已是 Annapurna 直接 own 的 tape-out（vs Anita 由 Alchip lead），代表 AWS 已具備複製 Alchip backend 工作的能力。若 2027 Trainium 4.5 / 5 完全內製化、Alchip 退到 secondary supplier，NRE 單價可能跌 50-70%。
  • 管理層失職：Alchip 創辦人 Kenneth Tai（戴尚義）為核心 SoC 設計專家、現任董事長兼 CEO；過去資本配置紀錄優（無重大併購失誤、capex 控制在 Rev 5-8%）。風險點：dual-class 股權集中（內部人 3.6% + 機構 40%），少數股東保護不強。
  • 價值陷阱路徑（4 選 1）：盈餘品質型 — 2024 NI 6.45B vs FCF 0.74B（FCF/NI 0.115）顯示「盈餘已實現但現金未到」結構；2025 反向（NI 5.60 vs FCF 12.62）只是 working capital timing。若 H2 26 wafer-out 大量 ramp 後 AR 暴增、FCF 與 NI 持續背離 > 2 季 → 盈餘品質型陷阱信號真實。

§9 護城河分析

評分：8 / 10（execution dimension 7-8 / pricing power dimension constrained by AWS dual-track）

護城河來源

  • 網路效應：中等 — 越多 hyperscaler 選擇 Alchip backend，TSMC + Synopsys 等 ecosystem partner 越願意給 Alchip 優先配額與 IP 折扣；但 hyperscaler 是 high-stakes single buyer，網路效應強度有限
  • 無形資產：強 — (a) 5 代 Trainium tape-out 經驗（從 Trainium 1 2020 → 4 2027）= 不可複製的「execution track record」；(b) Synopsys 224G SerDes 戰略 partnership = IP 採購規模優勢；(c) CoWoS 60K wpm 配額 lock = 28% 整體 CoWoS 配額
  • 轉換成本：高 — 一旦選定 design house，從 RTL spec → tape-out → mass production 切換 backend partner 需 6-12 個月重來 + 風險巨大（Marvell Trainium 2 RDL interposer 失誤就是最好的反向案例）
  • 規模優勢：強 — 配額型 scale：Alchip 拿到 70K wpm CoWoS 是 Marvell + MediaTek + GUC 的總和。在「capacity-rationed」市場中，配額本身就是護城河

競爭鴻溝

為什麼資本充足的對手（如 Marvell、GUC、SiTime）無法跨越？(1) Marvell Trainium 2 RDL interposer 執行失誤代表 backend physical design 不是有錢就能做好的；(2) GUC 是 TSMC affiliate 但偏向 Google ecosystem（Axion CPU + TPU 周邊），無 AWS hyperscale 訓練 ASIC 經驗；(3) MediaTek 強在 IP（SerDes + WiFi）但 backend physical 是新進入者，2026 才剛拿下 TPU 8i；(4) CoWoS 配額已被 NVIDIA + AVGO + Alchip 三家鎖死 70%+，新進入者要等 2027 H2 第二家先進封裝廠商上線才可能搶到配額。

Execution Moat vs Pricing Power Moat

Alchip 的護城河必須區分為兩個 dimension，否則容易導致 IRR overestimation：

  • Execution moat（強，7-8/10）：5 代 Trainium tape-out 經驗 + RDL interposer know-how（Marvell Trainium 2 失誤就是反向證明）+ Synopsys 224G SerDes partnership + CoWoS 28% 配額 lock。這個 dimension 是 thesis-period alpha 的 driver — 12-18 個月內 AWS 找不到替代者。
  • Pricing power moat（弱，constrained）：AWS Annapurna 對 Trainium 3 採 Anita（Alchip lead）+ Mariana（Annapurna own）雙軌結構，per SemiAnalysis Mariana ~800-900K units 為 majority volume，Anita ~500-600K 為 minority。Mariana economics 對 Alchip 不利（Annapurna 直接買 wafer + Alchip 只收 backend 設計費 + 沒 wafer markup）。Dual-track 本質就是 AWS 保留對 Alchip 議價權的機制。這個 dimension 限制 mature state 的 margin / multiple ceiling。

合併評分 8/10 反映 execution moat 強 + pricing power moat 弱的綜合 — 12-18 個月 inflection alpha 真實，但 5+ 年 long-term IRR ceiling 受 pricing power constraint cap。
