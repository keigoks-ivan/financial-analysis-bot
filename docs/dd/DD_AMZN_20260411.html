最終裁決:🟡 觀望(Watch,偏正面 — Jassy Letter 已定價,等下一個確認點)

Amazon 是 v8 序列中最特殊的「零售 + AWS + AI 晶片」三引擎混合體。關鍵數字:AWS +24% YoY(13 季最快)、AWS backlog $244B(+40% YoY)、AI 營收年化 $15B、Trainium2 部署 1.4M+ 顆、自研晶片年化 $20B(三位數成長)、AWS 貢獻 57% 營業利益。但質量層面的挑戰明顯:ROIC 僅 12.92%(v8 序列唯一低於 15% 門檻)、FCF 10Y CAGR 僅 1.47%(基本持平)、2026 Capex 指引 $200B(+56% vs $131B)、Q1 Operating Income 指引區間中位低於共識 $22.2B。估值面:Forward P/E 26.65–30.17 略低於 5Y 均 67.33 但遠高於 GOOGL 23、MSFT 19.5。技術面:股價剛因 Jassy 4/9 Shareholder Letter 上漲 5.6%,已充分反映 AI/AWS 利多。三層 R:R 為 ❌ / ✅ / 🟡,與 GOOGL 相似但品質略遜。

財報信號與假設校準

  • H1(AWS 在 AI 時代份額保持 30%+):強化AWS +24% YoY(13 季最快),backlog +40%。
  • H2(Trainium 成為 NVIDIA GPU 替代方案,拿下 Anthropic):強化Trainium2 1.4M+、Trainium3 已發布、Anthropic Project Rainier 持續擴大。
  • H3(零售業務進入結構性盈利期):不變零售貢獻 43% OI 但增速不如 AWS。
  • H4(FCF 在 Capex 週期後回升):削弱2026 Capex +56% 至 $200B,FCF 將被擠壓至 2027+。

這份財報 + Jassy Letter 的組合讓「成長故事」強化,但「現金流品質」進一步削弱。隱含預期在營收/AI 維度變得更容易兌現,在 FCF/ROIC 維度變得更難兌現——這是典型「好公司 + 資本密集升級期」的雙向張力。

§0 序章:第一性原理 × 逆向思考

存在意義

Amazon 解決的是「地球上任何商品從 A 點送到 B 點的最低成本」+「企業計算資源的按需租用」兩個結構性問題。Amazon.com 每年配送 50+ 億個包裹,是美國 40%+ 電商 + 歐洲 30% 電商的基礎設施;AWS 提供全球 31% 的公有雲計算,是 Netflix、Airbnb、Meta 廣告、NASA、Capital One 等數千家頂級客戶的底層。若 Amazon 明天消失,① 全球電商供應鏈會倒退 10 年至 eBay + Shopify 碎片化狀態;② 60%+ 的網際網路服務會瞬間斷線(因為許多 SaaS 依賴 AWS);③ Prime 會員的影片、音樂、Kindle 生態會失去唯一連貫的訂閱入口。這個存在意義的關鍵特徵是「廣度極大但深度略遜」——每個業務都有可替代方案(AWS 有 Azure/GCP、Retail 有 Shopify、Prime 有 Netflix),但沒有任何公司能同時替代所有業務。

逆向推演(Munger Inversion)

  • 地緣政治曝險:製造業依賴中國(2024 年 ~30% 供應商),貨運依賴全球海運與鐵路。對中國製造業的依賴使得美中貿易緊張會直接影響毛利。
  • 技術替代風險:① Shopify + Meta 廣告組合在 D2C 領域威脅 Amazon 廣告與賣家生態;② Azure + GCP 在 AI 時代 AWS 份額被蠶食(當前 AWS 市佔 ~31% 仍 #1 但增速已不如 Azure +24% vs Azure +39% vs GCP +48%)。
  • 管理層失職:Andy Jassy 2021 接任以來執行優秀——AWS 加速、廣告業務突破 $50B、零售盈利化。但 2026 Capex $200B 的決策是重大賭注,若 Trainium 未成功,將是史上最大資本錯配。
  • 價值陷阱路徑:非傳統陷阱。若出現問題,最可能是「隱性資本密集型」——Capex 持續高位但 FCF 不恢復,使估值尺從 P/E 切換至 P/FCF 時股價承壓。時間軸 18–24 個月內可見端倪。

§3 護城河分析

評分:8 / 10(低於 NVDA/TSMC 10、GOOGL/MSFT 9,但高於 AVGO 9 等級之下)

護城河來源

  • 網路效應:✅ 強。Prime 會員 1.8 億+ 形成雙邊網路(消費者 + 賣家),AWS 生態工具(Lambda、S3、DynamoDB 等)形成多邊開發者網路。
  • 無形資產:✅ 中等。Amazon 品牌 + Prime 訂閱綁定 + AWS 工程品牌。技術 IP 相對 NVDA/GOOGL 次要。
  • 轉換成本:✅✅ 極高。企業遷出 AWS 需要重寫應用 + 重新訓練團隊,Prime 會員遷出等於失去 Netflix / 音樂 / Kindle 綁定。
  • 規模優勢:✅✅ 明顯。Fulfillment 網路的規模成本優勢是 Shopify / Walmart 難以複製的,Amazon 單位配送成本低於競爭對手 30%+。

競爭鴻溝

為什麼對手無法跨越?
① Walmart 雖有實體店但缺少 AWS 資金來源,無法持續投資 Fulfillment;② Shopify 是 D2C 賣家平台但無法直接服務消費者;③ Azure / GCP 技術相當但缺少 Amazon 的零售業務數據互補;④ 阿里巴巴在美國市場不具營運能力。Amazon 的護城河來自「三業務互補」而非任一單獨業務的絕對優勢。

§5 產業格局、議價權與 Unit Economics

市場量化

  • 全球電商 TAM ~$6T,Amazon 佔 12%(美國佔 40%)
  • 全球 Cloud TAM ~$700B,Amazon AWS 佔 31%
  • 全球數位廣告 TAM ~$650B,Amazon 佔 ~10%

議價權

  • 對上游:對賣家(電商)、ISV(AWS)、廣告主議價力極強——賣家依賴 Amazon 通路,企業 SaaS 依賴 AWS 基礎設施。
  • 對下游:對消費者議價中等——消費者對價格敏感但對便利依賴。
  • 地緣:供應鏈高度依賴中國,AWS 全球 30+ region 分散。

營收組成

  • Retail(Online stores + Physical):~55%
  • Third-party Seller Services:~25%
  • AWS:~18%(但貢獻 57% 營業利益!)
  • Advertising:~8%,高毛利,快速成長
  • Subscription(Prime):~7%

單位經濟

Prime 會員 LTV 極高(年費 $139 × 5+ 年留存 × margin ~40% = LTV $278+);AWS 客戶 LTV 取決於消費水平;廣告業務單位經濟最優(Marginal cost 接近零)。

§6 公司治理與資本配置

股東結構

Jeff Bezos(前 CEO,現執行董事長)持有 ~9%,是美國科技巨頭中創辦人持股最高者之一。Andy Jassy(現 CEO)自 AWS 一路晉升,深度懂技術與業務。機構持股 ~60%,Vanguard、BlackRock 為前兩大。

資本配置軌跡

Bezos 時代(1997–2021)奉行「長期犧牲短期」的資本配置——持續虧損投入 AWS 最終成為 Amazon 的核心利潤來源。Jassy 時代(2021–)延續這個哲學,當前 $200B Capex 是歷史最大單一賭注,將 Amazon 從「輕資本軟體公司」進一步轉型為「重資本基礎設施公司」。無回購、無股息,100% 再投資。

SBC 評估

SBC/Revenue ~3%,低於 MSFT 4%、GOOGL 4.5%、AVGO 4%,為 v8 序列中較低水準。這是 Amazon 資本配置紀律的表現之一。
