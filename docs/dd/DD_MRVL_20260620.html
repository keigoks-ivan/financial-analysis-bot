§4 序章：第一性原理 × 逆向思考

存在意義

Marvell 解決的長期不可逆痛點：當運算從「通用 GPU」走向「客製化 + 異質運算」，資料中心需要把矽（custom XPU）、光（optical DSP / PAM4 / 1.6T）、與網（DCI、scale-up/scale-out interconnect、AEC）整合成「可量產、可演進、可互連」的系統。Marvell 是極少數同時握有「客製矽設計服務 + 高速連接 IP（SerDes、PAM4 DSP）+ 光通訊」全套積木的廠商。若它明天消失：(1) AWS Trainium、Microsoft Maia 等 hyperscaler 自研晶片少了一個（與 Broadcom 並列的）僅有的兩家 co-design 夥伴之一，custom ASIC 市場將更集中於 Broadcom（議價權惡化）；(2) 800G→1.6T 光模組與 DCI 的 DSP 供給會出現空缺（Marvell 是 PAM4 DSP 的市場領導）。它的不可替代性在於「連接 + 客製矽」的交集，而非單點。

護城河如何累積而成，見 §9.D 時間軸（Cavium 2018 / Inphi 2021 / Innovium 等併購構築的 IP 組合）。

逆向推演（Munger Inversion）

若要讓這筆投資在 3 年內虧損 50%+，最可能路徑：

Alchip — 已實現的新進威脅（單獨成段）：台灣 ASIC 設計服務商，以「成本優勢 + 在地工程 + 台積電緊密協作」切入 hyperscaler custom 競標。已實際奪下 AWS Trainium3 socket（Marvell 因 Trn2 執行不佳 + 成本而落敗），並把 CoWoS 預訂量年增 200% 至 60K wafers。Alchip 的切入點正是「成本敏感、設計相對標準化」的 custom 程序，直接威脅 Marvell 的最大 custom 客戶（AWS）。雖其毛利率（設計服務模式）低於 Marvell，但在「客戶要的是 cost-plus 代工」的賽局裡，低成本反而是武器而非弱點。這是本份 DD 中 Marvell 護城河最具體、已發生的裂縫，直接壓低 §9 pricing power 維度至 7。

Astera Labs (ALAB) — 鄰接蠶食：connectivity 純玩家（PCIe/CXL retimer、smart cable、fabric switch），GAAP GM 76.3%、營收 $308M/季（+93% YoY）。在 AI 伺服器 scale-up / rack 內連接的利基快速擴張，與 Marvell 的 connectivity 業務（AEC、retimer）正面重疊。其高毛利 + 高成長使其在資本市場享有更高倍數，並持續吸走 connectivity 鄰接份額。

NVIDIA — 既是投資者也是 toll booth（特殊關係）：2026-03-31 投資 Marvell $2B 並開放 NVLink Fusion，讓 Marvell 的 custom XPU + scale-up optics 接入 NVIDIA 生態（Vera CPU / ConnectX / Spectrum-X）。表面是背書與 TAM 擴張；實質是 NVIDIA 把「半客製 AI 基礎設施」也納入自己的生態護城河（soft lock-in）——Marvell 成為 NVIDIA 生態的供應商而非獨立替代者。對 Marvell 是「量的機會 + 戰略地位的折讓」並存。
