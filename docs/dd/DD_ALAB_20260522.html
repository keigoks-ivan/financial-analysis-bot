§4｜序章：第一性原理 × 逆向思考

存在意義

剝離財務表象，ALAB 解決的不可逆人類痛點是「AI 計算單元（GPU/CPU/Accelerator）之間信號完整性的物理瓶頸」。PCIe Gen 6（64 GT/s NRZ）的物理信號 reach 僅 ~12-16 英寸，跨 chassis 即衰減失敗 — retimer 是物理層上的 SerDes 信號重建工程，無 retimer 則 GPU rack-level 多卡互連根本無法建構。若 ALAB 明天消失，世界會出現的不可替代混亂：

  • (1) AI training cluster 從 NVIDIA HGX H100/B200/B300 至 next-gen GB200/GB300/Vera Rubin 平台的 PCIe scale-up 路徑全部受阻；hyperscaler 必須回到 Broadcom / Marvell（領先 6-12 月）+ Synopsys IP 組合應急，整體 AI infrastructure 部署時程倒退 12-18 個月
  • (2) 三家 hyperscaler（NVIDIA / Microsoft / Google）NVLink Fusion 商用化路徑斷裂 — NVLink Fusion 是 NVIDIA 將 NVLink 對外 license 的計畫，ALAB 是 ecosystem partner，斷裂後 NVLink Fusion 仍可推進但量產時程延後 ~9 個月
  • (3) CXL Memory expander（Microsoft Azure 已 production）斷供，部分 in-memory database / vector store workload 必須回退至傳統 DDR + RDMA 路徑（latency 上升 3-5x）

核心痛點本質：PCIe Gen 6 的物理層 signal integrity 是「不可避免的 engineering tax」 — 每代 PCIe spec 速度翻倍（PCIe 4 = 16 GT/s → PCIe 5 = 32 → PCIe 6 = 64 → PCIe 7 = 128），但 reach 與 signal eye 比例 inverse — retimer 不是可選，是 mandatory chip — ALAB 是該細分市場的 NVIDIA。

逆向推演（Munger Inversion）三維展開

① 技術維度：什麼技術替代會讓 ALAB 3 年內失去 50% 價值？

  • Silicon photonics 取代電子 retimer：CPO（Co-Packaged Optics）若在 PCIe Gen 7（2027-2028）成熟，將部分取代 retimer use case。歷史對標：Intel optical interconnect 嘗試 2015-2020，量產延後 5+ 年，目前 Intel / NVIDIA / TSMC 仍以 packaging 內 photonics 為主，rack-level 仍需 retimer。機率：30-40% 至 2028。
  • NVIDIA / Hyperscaler 自研 retimer ASIC：CRDO 2025 教訓 — AWS Trainium 3 將 SerDes 從 Credo 換到 Synopsys IP。若 ALAB top 2-3 hyperscaler（NVIDIA、Google、AWS）任一自研 PCIe retimer ASIC，可能蠶食 30-40% 市佔。歷史對標：AWS Annapurna 收購（2015）後 Nitro chip 取代部分 Mellanox 功能。機率：20-30% 至 2028。
  • UALink 取代 PCIe-based 互連：UALink 1.0/2.0 雖然 ALAB 是參與方，但若 UALink 主導廠（AMD、Broadcom）跳過 retimer 直接 native UALink interconnect，將部分削弱 ALAB Aries 6/7 需求。機率：15-25%。
  • NVIDIA NVLink-only ecosystem（封閉化）：若 NVIDIA 切斷 NVLink Fusion partner 計畫，ALAB 失去 NVLink ecosystem 機會。機率：15%（NVIDIA 商業策略上需要 open partner 對抗 UALink）。

② 管理維度：什麼決策路徑會自我毀滅？

  • Capex 慣性：ALAB 是 fabless（TSMC 代工），Capex 主要是 R&D 與 IP 投資。風險在於 R&D 資金過度集中單一產品線（如過度押注 Aries 6 而忽略 Scorpio / Leo 多元化）— 目前管理層已警惕，FY25 R&D 占 Rev 33% 分散至 4 條產品線，但 Aries 系列仍佔 R&D >40%。歷史對標：Inphi 2017-2019 過度押注 Optical DSP 單條產品線，後被 Marvell 收購。風險低（管理層多元化執行中）。
  • 創辦人退場：CEO Jitendra Mohan + CFO Mike Tate 共同 25+ 年 IC 經驗，IPO 2024-03 後 18 個月內 founder lockup 過期；2026 Q1 / Q2 若任一創辦人或核心高管離職 → 信心打擊。歷史對標：Maxim 創辦人 Jack Gifford 2007 離世後團隊重組陣痛 18 月。風險低（無離職跡象）。
  • 整合失敗（M&A）：ALAB 過去無重大 M&A，但若 2026-2027 為加速 Photonics / Optical CPO 路徑進行大型併購，整合風險可能拖累 18-24 個月 OPM。歷史對標：Marvell 2021 acquisition of Inphi 整合 18 月後成功，但 2022 acquisition of Cavium 整合拖累 36 月。風險中（若進行大 M&A）。

③ 結構維度：什麼產業變化會讓商業模式過時？

  • AI 計算範式轉換：若 AI computing 從 GPU/CPU dense PCIe interconnect 轉向 photonic mesh / fully integrated SoC（如 NVIDIA Grace Hopper Superchip / AMD MI300X 路徑）— 不需要 retimer 的 monolithic system 占比上升，retimer TAM 萎縮。歷史對標：Sun Microsystems UltraSPARC 1990s 部分 Co-processor 不需 PCIe，Sun 被取代後該 socket 消失。機率：20-30% 至 2030。
  • Web/AI 結構性消失：AI / cloud 工作負載若被某種更高效運算範式取代（如量子 / neuromorphic / photonic compute），整個 hyperscaler ecosystem 萎縮。歷史對標：AT&T 監管型分拆 1984 後 ecosystem 重組。機率：< 5% 至 2030。
  • 計算範式轉換（量子 / neuromorphic）：若量子 / neuromorphic compute 在 2028-2030 達到 production scale，部分 AI workload 從 GPU 遷移至新範式 — retimer TAM 結構性受壓。歷史對標：x86 取代 mainframe 1990s 過渡 10+ 年。機率：< 5% 至 2030（量子 compute 仍在 noisy intermediate scale）。

逆向推演綜合：3 年內失 50% 價值機率約 25%（主要來自 Marvell 競爭實質化 + Hyperscaler ASIC 化 2 個 path 重疊），5-10 年機率約 35-40%。投資 thesis 的核心防線是「Aries / Scorpio / Leo / Taurus 4 條產品線 + NVLink Fusion 開放標準參與權」雙重 hedging。
