§4 序章：第一性原理 × 逆向思考

存在意義

國巨的核心痛點：「電子裝置從消費電子 → 汽車 → AI server 演進，被動元件（capacitor / resistor）的單裝置用量呈指數成長，但 capacity 擴張受限於投資週期 6-8 年。」每個 AI server 使用 MLCC 達 1 萬-1.5 萬顆（一般 server 1,000 顆，10-15x）；hyperscaler 2026 capex 級 capex（Microsoft .5T multi-year + Amazon .25T + Google .8T+），帶動 server MLCC 全球需求結構性翻倍。國巨的存在意義是「在 MLCC 結構性短缺週期中提供 specialty 高容量／高壓被動元件，讓 AI / 汽車 / 工業裝置能夠繼續電子化升級」。若 Yageo（與 Murata / Samsung 三巨頭）明天消失：AI server 出貨延遲 9-18 個月、AI training 基礎設施投資延後、消費電子高階產品（手機、PC）成本上升 10-15%。

逆向推演（Munger Inversion）三維展開

① 技術維度

  • 固態電池 / 新材料替代 MLCC：若超級電容、薄膜電容、固態電容大規模商用 → MLCC 需求下滑。但這類替代需 7-10 年滲透，3 年內機率 < 5%。歷史對標：tantalum 電容 vs MLCC 競爭 30 年，MLCC 持續勝出。
  • 系統整合化壓縮 MLCC 用量：若 AI server 改採 integrated power module（如 PMIC + 電容整合）→ 單台用量減少 30-50%。歷史對標：Intel 整合 voltage regulator 過去 10 年壓縮主機板電容用量 ~20%。3 年內機率 15-25%。
  • 大陸廠商技術突破：三環、風華等中國廠商若突破高容量 / 高壓 MLCC（目前仍落後 2-3 代），可能進入 high-end 市場分食。3 年內機率 10-20%。

② 管理維度

  • 併購過度 / 整合失敗：Yageo 過去 7 年完成 4 大併購（Pulse, KEMET, Schaffner, Shibaura），若 Shibaura 整合困難或產生減損 → 短期 EPS 拖累。歷史對標：Schaffner 2024 整合需 2-3 年才能產生 synergy。
  • 創辦人退場：陳泰銘董事長 70 歲（2024 年），接班議題逐漸浮現；若管理層更迭期間策略不連貫 → 競爭力短期下降。
  • Capex 慣性：若 cycle peak 後仍持續 over-build capacity → 下個 cycle 谷底 ROIC 受壓。歷史對標：Yageo 自身 2018 cycle peak 後 over-build，2019-2020 ROIC 從 25% 降至 8%。

③ 結構維度

  • AI hyperscaler capex 退潮：若 hyperscalers 2028-2030 capex 結構性下修（投資回報率不達預期），AI server MLCC 需求下滑。歷史對標：2000 dot-com 泡沫 cisco 級 capex correction。3 年內機率 15-25%。
  • 消費電子整體萎縮：智慧手機 / PC 整體需求 5 年衰退（佔 Yageo ~25-30% 營收）→ 量受限。歷史對標：2018-2023 智慧手機出貨從 14 億台降至 11 億台。慢變數，10 年機率高。
  • 地緣政治分割供應鏈：若美中供應鏈分割加劇，中國 OEM 強制本土化 MLCC → Yageo 中國市佔下降。歷史對標：2018 中美貿易戰開始的本土化 trend。中期風險。

價值陷阱路徑判定：2327 失敗劇本為「① 技術 + ② 管理 + ③ 結構」三重風險疊加。最現實的 6-12 個月風險是「MLCC cycle peak + 4W +111% 動能 mean reversion」雙觸發 → 股價回檔 -30 至 -50%；但 thesis 結構（AI MLCC）仍會在 1-2 年後重新確認。短期 buyer 應觀望，長期 buyer 可在 NT$300-450 區分批進場。
