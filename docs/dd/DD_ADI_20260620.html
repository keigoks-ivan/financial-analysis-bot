§4　序章：第一性原理 × 逆向思考

存在意義

ADI 解決一個不可逆的物理痛點：真實世界是類比的，運算世界是數位的，兩者之間必須有一座高精度、低雜訊、高可靠度的橋。每一個感測（溫度、壓力、電流、影像、聲音）、每一次電源轉換、每一條高速資料鏈，都需要把連續訊號轉成位元、再把位元轉回連續世界。ADI 是這座橋上最頂尖的造橋者之一——data converter（ADC/DAC）、放大器、電源管理、RF/微波、MEMS、隔離技術。

若 ADI 明天消失：工廠自動化儀控、汽車 BMS（電池管理）與 ADAS 感測、5G/資料中心的時脈與電源、醫療影像與生命徵象監測、航太國防的精密訊號鏈，會立即出現無法用軟體替代的硬體斷層。這不是「提升效率」，而是物理層的不可替代——類比 IP 是數十年類比工程師 know-how 的累積，無法靠演算法蒸餾，也無法靠資本快速複製（這正是 AI 取代風險 🟢 的根因：ADI 是 AI 基礎設施的「賣鏟人」而非被取代者）。

護城河如何累積而成（Hittite→Linear→Maxim 的併購 + 內生工藝），見 §9.D 時間軸。

逆向推演（Munger Inversion）

若要讓這筆投資在 3 年內虧損 50%+，最可能路徑：

  • 地緣政治曝險：ADI 採混合製造（內部 fab + 台積電等代工），台灣/亞洲供應鏈集中；中國營收佔比中等。關稅與出口管制是 newsflow 風險，但類比成熟製程多不在先進節點管制清單，結構性衝擊有限。
  • 技術替代風險（最關鍵）：類比 IP 替代極慢——但客戶 in-house（自製）是真實長期風險（如大型工業/汽車 OEM 自建電源/感測），時間軸 5-10 年、機率低。更現實的是 TXN 的 300mm 成本碾壓：TXN 以 300mm 內製把每顆成本壓低 ~40%，在價格敏感的 catalog 類比段對 ADI 形成持續份額壓力（QC-23 🔴 生態攻擊，見 §9）。
  • 管理層失職：ADI 是連續併購型（Linear $14.8B、Maxim $21B），下一筆大型併購若再次高價、且整合不如預期，會稀釋增量 ROIC（自我毀滅路徑詳見 §12 資本配置計分卡）。
