§4 序章：第一性原理 × 逆向思考

存在意義

Celestica 解決的不是「電子代工」，而是 hyperscaler 能否在 18 個月窗口內把下一代 AI 算力從晶片邏輯轉化為可運轉的機架級系統。Meta、Google、Amazon、Microsoft 的 AI 軍備競賽中，晶片不是瓶頸，「把晶片變成可量產可冷卻可組網的整機架」才是瓶頸。Celestica 的 1,100+ 設計工程師（含近 400 軟體）能在數天內為客戶 stand up 1.6T 液冷網路 prototype；其 850+ 專利集中在高速訊號與熱解，是 hyperscaler 自建設計團隊或切換到 Flex/Jabil 都需要 12–24 個月才能複製的能力。若 Celestica 明天消失，hyperscaler 的下一代 AI 機架（liquid-cooled 1.6T、custom silicon adjacency）會延後上線 6–18 個月，足以讓 AI 模型訓練成本曲線錯失關鍵節點，影響面從 NVDA、AMD GPU 出貨延遲到 OpenAI / Anthropic 模型迭代節奏。

競爭鴻溝

「為什麼資本充足的對手仍無法跨越？」答：因為時間窗短（hyperscaler 1.6T → 3.2T 升級節奏 18 個月），需要在「設計能力 + 製造彈性 + IP 庫」三項同時就位。具體 mechanism：

  • 多代 hyperscaler 信任壁壘：hyperscaler 對 HPS Switch 的 firmware bug 容忍度極低（一個 bug = $50M+ data center 訓練重啟成本），需要 2-3 個成功代別才建立信任。CLS 在 800G / 1.6T 累積信任後 3.2T 自然延續，Flex/JBL 從 0 起步進度慢 12-18 月。
  • 液冷整機架設計 know-how：CLS 850 專利 70% 集中在熱解（液冷 / 風冷 hybrid）。Flex 2025 引入 JetCool 液冷整機架是直接威脅，但仍需 12-18 月認證；JBL 規模更大但設計團隊集中在汽車與醫療段，AI 設計能力落後 12-18 個月。
  • 製造 + 設計服務雙能力整合：純 ODM（廣達 / 緯創）只賣製造，設計服務不足；純設計（fabless）無製造能力。CLS 整合兩端為 hyperscaler 提供「one-stop」，後進者需構建整套 ecosystem。

議價權三段獨立（v12.3 強制 ≥ 60 字獨立段落）

對上游供應商

CLS 對上游供應商議價力整體偏弱。具體拆解：(1) ASIC / GPU 供應商（NVDA / Broadcom / Marvell / AMD）：CLS 是「指定採購」（hyperscaler 客戶指定使用 NVDA Blackwell / AVGO Tomahawk / MRVL custom silicon），無採購談判力，year-on-year ASP 漲幅完全 pass through。NVDA Blackwell GPU 售價 2024-2025 漲幅 30%+，CLS 採購成本相應上升，但 HPS 設計溢價同步上升使毛利率仍能擴張。(2) 被動元件 + 連接器（Murata / TE Connectivity / Amphenol）：年訂單 $300-500M 規模，CLS 議價力中等，可獲 5-10% volume discount。(3) 封裝測試（OSAT，主要 ASE / Amkor）：CLS 議價力低（封裝測試容量緊縮 2024-2026），需提前 12 個月鎖定。(4) 製造設備（ASML EUV、AMAT、KLA）：對 CLS 影響間接（CLS 不買晶圓設備，但代客戶間接受惠）。整體上游議價力偏弱，但 AI 順風使供應商議價更弱（chip shortage 紓緩中）。

對下游客戶

CLS 對下游客戶議價力結構性弱（且持續惡化），這是 thesis 最大 fragility。具體拆解：(1) Top 1 客戶（推測 Meta，Q4 25 = 36%）：sole-source 設計 lock-in（1.6T HPS Meta 客製 rack），議價力中等，但客戶集中度上升（FY24 28% → FY25 36% = +8pp）使長期議價力惡化；Meta MTIA + Reality Labs reorg 顯示 in-house ASIC 化路徑啟動，CLS 角色從「設計+製造合夥」可能退化為「製造代工」。(2) Top 2-3 客戶（推測 AWS + Google，合計 27%）：dual-track / second-source 結構（廣達 / 緯創 / Foxconn 並行 ODM），議價力低，AWS Custom silicon（Trainium 系列）+ Google TPU 內部設計能力強，CLS 議價權薄弱。(3) Top 4-10 客戶（MSFT + ATS 客戶，合計 16%）：multi-source 採購，議價力中等，但 ATS 段毛利較低（aerospace / 醫療客戶談判力強）。(4) 合約結構：典型 2-4 年 master agreement + quarterly PO；無 take-or-pay 鎖定；訂單能見度 12-24 月（mgmt 強調 36 個月為樂觀框架）。整體下游議價力弱（GM 12% 看似領先 EMS 同業，但 FY25 Top 10 從 73% → 79% 集中度上升使議價權邊際惡化）。
