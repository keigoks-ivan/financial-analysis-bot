§4｜序章：第一性原理 × 逆向思考

存在意義

Vicor 解決的是 「power delivery 物理極限」—— 隨單一 chip 的 thermal envelope 從 500W（A100 時代）走向 1,000W（H100）→ 1,500W（Blackwell）→ 預估 2,500W+（Rubin），傳統的「48V → 12V → 1V」三級電源轉換無法在小空間內完成 1MW+ rack 的電流密度需求（簡單算術：1MW rack ÷ 12V backplane = 83,000A 銅線截面，物理不可行）。Vicor 的 Factorized Power Architecture（FPA）原始 IP 把問題重構為「800V → 48V backplane → 48V→1V load-proximate」三步走，把 chip 附近 1V 高電流區壓縮到 BGA package 內，是 1MW+ rack 唯一可量產的物理解法。

若 Vicor 明天消失：(1) 已下訂的 $300M backlog 砸鍋（短期 hyperscaler GB300 出貨延遲 6-12 個月），(2) MPWR/Infineon/Delta 在「lateral integration」的 silicon-based 方案就是當前唯一可選擇路徑（成本較低但 power density 上限明顯，1MW+ rack 仍需重做架構），(3) ITC LEO 立即失效 — 中國 / 越南電源廠繞道進口禁令解除，全 AI 加速器電源市場進入 commodity 價格戰。Vicor 不是「可被替代」，是「替代品強迫整個 GPU rack 重新設計」。

逆向推演（Munger Inversion）三維展開

若要讓這筆投資在 3 年內虧損 50% 以上，最可能的路徑：

① 技術維度

  • SiC / GaN-on-Si 突破 → 取代 FPA 三步走架構：若 Infineon / Wolfspeed 在 2027 前推出 single-stage 48V → 1V GaN module（直接做 8:1 conversion 而非 Vicor 的 4:1 + 4:1 分階），物理層面繞過 FPA 基礎專利 — 歷史對標：Intel Pentium 4 NetBurst 被 ARM 架構從根本繞過。機率：中（GaN scaling 仍受 driver IC + magnetic 限制）。
  • NVDA 走 OCP 開源 power 規範 → commodity 化：若 Open Compute Project 出新 power spec 強制 hyperscaler 統一介面，IP 護城河被 spec 標準化稀釋 — 歷史對標：Cisco 路由器 in-house ASIC 被白牌 + open-source NOS 蠶食。
  • 1.6T optical interconnect 提早 → rack 規模降低：若光互連讓 1MW rack 變成多個 200kW pod，集中式高功率 module 需求下降 — 歷史對標：Intel Itanium 64-bit 被 x86-64 從不同方向繞過。
  • HBM-on-package 集成電源 → in-substrate VRM 由 chip 廠自製：若 TSMC InFO 或 CoWoS 內建 micro-VRM，Vicor 就失去 load-proximate 優勢 — 歷史對標：Sun Microsystems 工作站被 Intel + Linux x86 上下夾擊。

② 管理維度（NEW）

  • Vinciarelli 退場 / 健康問題 → key person risk 兌現：CEO + Chairman + 27% 持股，創辦人 44 年掌舵；無明確繼任者公開揭露；近 60 天加快賣股（雖 10b5-1）令市場對 succession 隱憂升溫 — 歷史對標：Apple 1985-1996 Jobs 離開期 / Tesla CEO succession 焦慮週期。機率：中（年齡與健康未公開警訊但統計上不可忽略）。
  • Capex 慣性 + Fab 2 預算膨脹 → operating leverage 反向：管理層從 greenfield 改「收購現有建物」是好事，但若 Fab 2 規模設定 $500M+、ramp 期間 OPEX 升至 GP 60%+，2027-2028 EPS dilute 反而下修 — 歷史對標：Tesla Model 3 ramp 期 2017-2018 cash burn。
  • 授權策略過度激進 → 客戶轉自製或聯合反訴：licensing 喊得太兇可能逼大客戶（NVIDIA / hyperscaler）聯合扶持替代陣營（MPWR + Infineon），重演 2010 Qualcomm 過度索取授權費 → Apple/Samsung 內製 + 集體反訴的歷史 — 機率：低-中（VPD IP 仍是 sole-source 且 ITC LEO 在手）。

③ 結構維度（NEW）

  • AI 訓練 capex 觸頂 → 整體 HPC 電源 TAM 不及預期：若 2027-2028 GPT-X 後訓練回報遞減、hyperscaler capex 從 +50% YoY 跌至 +5%，整個 AI 電源 SAM 縮水至 Vicor 預期一半 — 歷史對標：dot-com 2001 思科 / 朗訊光纖路由器需求斷崖。機率：中（生成式 AI 變現節奏存疑，但 inference capex 接棒可能性高）。
  • 監管型分拆 — 美國對「critical AI infrastructure」實施反壟斷：若 Vicor IP 被裁定為「essential facility」（電網 / 鐵路 / 電信前例），強制 FRAND 授權上限 — 護城河 royalty rate 從現在估計 ~3-5% 砍至 1% — 歷史對標：AT&T 1982 baby bell 分拆 / Standard Oil。機率：低（科技公司很少觸發 essential facility doctrine）。
  • 計算範式轉換 — Quantum / neuromorphic / optical compute 加速：若 5-7 年內 mainstream 訓練從 GPU 轉向新範式，high-density 電源需求 shape 完全改變 — 歷史對標：Sun Microsystems SPARC 被 x86 取代 / DEC Alpha 消失。機率：極低（5 年內主流地位不變）。

價值陷阱路徑判定

若 VICR 變成價值陷阱，最可能是「盈餘品質型的次型 — 成長兌現延遲」：current EPS 漂亮但靠 IP licensing 一次性高峰拉高，產品段成長若 Fab 1 ramp 不順 + Fab 2 推遲，FY28-29 EPS 增速比 FY26-27 共識緩，市場 de-rate 50% 至「中段成長股」估值。本 DD 在 §5 R1-R3 + §10 FCF lumpiness 章節有對應監測指標。

① 對上游供應商（獨立段）

Vicor 上游集中度為中等：核心依賴 (1) wafer foundry — Vicor ChiP 製程的 power IC 由 TSMC / GlobalFoundries 提供 wafer，但 wafer 不是 Vicor 的 cost dominant；(2) magnetics 材料（ferrite cores、bobbins）— 多家供應商（TDK、Murata、Hitachi Metals），無單一依賴；(3) 自有 fab packaging substrate — Andover MA 自製為主。近 3 年議價案例：FY24 因 wafer 緊缺 GM 略受壓（51.2% vs FY23 50.6% 持平），FY25 wafer 環境緩解 + Vicor mix shift 至高 ASP 產品 → GM 跳至 57.3%。無 single-supplier 風險；替代難度 Medium-Low。

② 對下游客戶（獨立段）

客戶集中度高（引用 §8.H：Top 1 ~20% Cerebras / Top 5 ~55%）但議價地位強：(1) 為 sole-source 物理解決方案的客戶（Cerebras + 部分 hyperscaler GB300 design）價格幾乎無對手 benchmark；(2) 合約結構為「product purchase orders + royalty agreements」混合 — royalty 為 multi-year 結構（一般 5-10 年）；(3) 近 3 年無客戶流失公開 case；(4) 定價能力直接證據：FY25 GM +6pp YoY，遠超產業中位數 ~2-3pp 改善 → 定價權確認。主要客戶留存率：100%（Cerebras 從 WSE-2 → WSE-3 持續使用 Vicor）。

③ 地緣曝險（獨立段）

Vicor 核心生產地美國 Andover MA（Fab 1）— 100% US-based manufacturing；Fab 2 規劃亦在美國（management 4/21/26 法說證實）。供應鏈最高集中點：(1) wafer 多元採購（TSMC 台灣 / GlobalFoundries 紐約），(2) magnetics 全球供應，(3) ChiP packaging US 自製。政策風險具體路徑：(1) 對中關稅：Vicor 為美製產品 + ITC LEO 對中國繞道進口擁有 enforcement 優勢 → 政策變化對 Vicor 反向利好；(2) 出口管制：對中國 AI chip 出口管制可能影響 hyperscaler 中國市場部署，但 Vicor 美國 + 西歐 + 日韓市場為主，影響小；(3) Trump 2.0 trade protectionism → Vicor 為「美國再工業化」直接受益標的。已有 / 規劃中的分散化措施：Fab 2 為 capacity redundancy，但仍 US 境內。
