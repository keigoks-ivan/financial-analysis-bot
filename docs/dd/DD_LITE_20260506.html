§4 序章：第一性原理 × 逆向思考

存在意義

LITE 的核心存在意義是把電訊號轉換為光訊號的「物理層必經點」 — 沒有 LITE（或同業 Coherent / Sumitomo / Mitsubishi）製造的 EML laser，所有 100G/200G/400G/800G/1.6T 光通訊 transceiver 都無法存在。AI scale-out 架構中，每個 GPU 對外通訊 (NVLink + InfiniBand + Ethernet) 都需要 transceiver；rack 內部的 scale-up 也越來越多採用光通訊（不再純 copper）。若 LITE 明天消失：(1) 全球 hyperscaler 1.6T transceiver 年產能立即減少 50-60% → AI cluster build 全面延後 12-18 個月；(2) Coherent 短期內無法填補空缺（產能 + 良率 catch-up 需 18 個月）；(3) 中國 EML 廠商加速佔領 200G niche，但對美 hyperscaler 不可賣（出口管制反向版）。

逆向推演（Munger Inversion）三維

① 技術維度 — 什麼技術替代會讓 LITE 3 年內失去 50% 價值？

  • Co-Packaged Optics (CPO) 顛覆 pluggable transceiver（歷史對標：Mainframe → Server 1985-2000 期間 IBM 失去 80% 市值）。若 NVDA / Marvell 將 EML laser 直接整合至 GPU/switch chip 的 silicon，pluggable transceiver 市場萎縮 60%+。LITE 在 CPO 也有產品，但價值鏈位置從「卡式模組整合者」退化為「laser die 供應商」，毛利率壓縮 30-40%。
  • Silicon Photonics 取代 InP-based EML（歷史對標：硬碟 → SSD 2010-2018 完全替代）。Silicon photonics 整合更高、成本更低，但目前 1.6T 仍需 InP EML。若 silicon photonics 在 3.2T (2028) 突破到不需 InP，LITE 80% 業務消失。
  • Custom ASIC 取代 standard transceiver：hyperscaler 自研 optical engine（Google + Meta 已啟動），切走 transceiver 廠 ARPU。
  • Quantum communication 跳過光通訊：長線（10+ 年）量子網路若商用化，傳統光通訊產業萎縮 — 機率低但時間軸內仍可能。

② 管理維度 — 什麼決策路徑會自我毀滅？

  • Capex 慣性過度：LITE FY25 Capex $231M（vs Rev $1.65B = 14%），FY26 Capex 預期 $400M+ for 新 fab + 1.6T 量產線。若 1.6T TAM 不及預期 → 巨額 inventory write-down + 折舊壓力（歷史對標：Cisco 2001 dot-com 過度 capex 後 5 年股價反彈不利）。
  • CEO Alan Lowe 退場：Lowe 2018 接任 CEO 整合 Oclaro 後管理穩定，但 LITE 已連續經歷 2 次 turnaround。若 1.6T 量產關鍵期 CEO 交接，過渡期執行力風險高。
  • 過度收購整合風險：FY25-26 多次 M&A rumour（部分 Coherent 業務、Cloud Light 等），若再啟動大型併購可能觸發整合疲勞。

③ 結構維度 — 什麼產業變化會讓商業模式過時？

  • 計算範式轉換：若 AI 工作負載大量轉至 edge / on-device 推論（取代 hyperscaler-only），DC 光通訊 TAM 結構性收縮（歷史對標：mainframe → PC 1985-1995 IBM 危機）。但目前看不到此 trend，LLM 仍在向更大規模發展。
  • 監管型分拆 / 反壟斷：若 LITE 200G EML 真達到 60%+ 市佔，可能觸發美國 / 歐盟反壟斷調查，要求授權 IP 給其他廠（歷史對標：Microsoft 2001 反壟斷案）。但此風險僅在 2028+ 才可能浮現。
  • 地緣政治：中國市場結構性失去：LITE 不對中國銷售高端光通訊（出口管制），且中國 EML 廠商（華工 / 三安）持續進步。中國 1.6T 自給率 5 年內可能達 50%+，LITE 失去中國子市場（已 hedge）。

逆向推演結論：最大威脅是 CPO + silicon photonics 在 3.2T (2028) 顛覆 pluggable transceiver。LITE 在 CPO 有產品 hedge 但價值鏈位置會退化。價值陷阱模式判斷：非陷阱，但有「成長股估值陷阱」反向風險（單季 mild miss 觸發 -30-40% 倍數壓縮）。
