§4 序章：第一性原理 × 逆向思考（v12.2 擴充：5.1K）

存在意義

MACOM 解決一個高度具體的技術問題：製造高頻 / 高功率 / 寬頻 specialty 半導體元件，覆蓋從 RF (射頻、毫米波) 到 optical (光電轉換) 到 microwave (微波) 的「物理層 enabling components」。MACOM 不做 mainstream CPU/GPU，做的是 100GHz+ / 1.6T+ / 高功率 RF 等「需要 III-V semiconductor + 高度 customized IC 設計」的應用 — 這是 silicon 工藝（5nm/3nm/2nm）無法替代的物理層產品。

解決的痛點：

  • silicon photonics 整合不完備：800G/1.6T optical transceiver 仍需獨立 laser driver / TIA / photodetector 才能達到 commercial 性能（power / latency / yield）— MACOM 在 200G/lane photodetector 是 sole-source 設計
  • defense / aerospace 不可替代：US trusted foundry status + ITAR / DoD 認證 → 國防雷達 + 5G/6G 軍用 mmWave + EW(電子戰) 系統的核心元件
  • 5G/6G mmWave 基站：GaN-on-SiC PA (power amplifier) 提供 silicon 無法達到的 power density

若 MACOM 明天消失：

  • 不可替代性：Coherent / Lumentum / Sumitomo 可填補 80% optical 需求但 200G/lane photodetector 領先暫無 — 2026 800G/1.6T ramp 延後 6-12 個月
  • 產業連帶：NVIDIA Photonics SuperPOD / Broadcom Jericho-4 / Marvell Celestial AI 等 AI 基建 timeline 受影響
  • defense impact：F-35 雷達、AEGIS、Patriot 等武器系統的 GaN-on-SiC 替代受影響（無 Wolfspeed RF 後 Tier 1 trusted source 集中度進一步上升）
