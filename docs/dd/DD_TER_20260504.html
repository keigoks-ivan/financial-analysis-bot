§4 序章：第一性原理 × 逆向思考

存在意義

Teradyne 解決的是「每顆半導體晶片在出廠前必須驗證功能正常」這個半導體產業誕生以來的核心問題。每片 GPU、每顆 CPU、每個 HBM stack、每個 AI ASIC 在 wafer probe 與 final test 階段都需要 ATE。若 TER 明天消失，全球 SoC 測試產能短期損失 30-35%，但 Advantest（Japan）會在 12-18 個月內擴產填補（價格短期 +30-50%、長期回穩）。TER 並非「不可替代」，而是「ATE 雙寡占」之一 — 與 quality compounder 的差異：TER 是 cyclical commodity-test 業務，cycle peak 時 ROIC 39%、cycle trough 時 17%。商業模式介於商品（cyclical）與必需品（不可替代）之間。

逆向推演（Munger Inversion）三維展開

① 技術維度

  • 內建 BIST（Built-in Self-Test）取代外部 ATE（時間軸：2028-2032）：若 NVDA / Apple silicon 內建測試模組成熟，外部 ATE 需求結構性壓縮。歷史對標：外部 chipset → 整合 SoC（2010-2020），IDT / IDTI 等市值蒸發。TER 反制：cooperate 方向，整合至 ATE 平台
  • AI 自動化測試取代 ATE（時間軸：2030+）：若機器學習主導晶片測試，ATE 變成軟體 + ML 平台。TER 反制：已併購 Universal Robots + AI test platform R&D
  • 3D / 異質整合測試需求：HBM stack test 已是高需求，但若 chiplet test 標準化（CXL、UCIe），TER 需持續投入

② 管理維度

  • 過度回購 cycle peak 股票：CEO Greg Smith Q1 24 接任，目前 capital 紀律 OK（Q1 buyback de minimis）；若 cycle peak 大量回購會是 Boeing 模式
  • Robotics 段拖累：UR / MiR 仍未 ramp 至有意義規模（$91M / 7% mix）；若繼續虧損可能拖累整體 EPS
  • 缺乏 founder 持續紀律：TER 已是職業 CEO 多代，無 dual-class founder lock；治理分散

③ 結構維度

  • ATE cycle 本質：每 5-7 季 cycle 切換一次；cycle peak 時 GM 60%+、cycle trough 時 GM 55%；EPS swing 50-60%
  • Apple-style customer 集中：歷史 Apple 占 SoC test 30%+；2022-2023 切換至 Advantest 後 TER 受傷大；目前 AI ASIC 多客戶分散風險
  • 地緣政治：對中國半導體業曝險中等（< 10% 直接，但間接影響 mobile / IDM）
