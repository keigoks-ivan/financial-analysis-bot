§4 序章：第一性原理 × 逆向思考

存在意義

Amazon 解決的長期不可逆痛點：「全球零售與計算基礎設施的整合層」。具體拆解四層：

  • 零售索引層（Retail + Marketplace + Logistics）：全球最大電商，Prime 200M+ 會員，物流網絡 200+ fulfillment centers + 1000+ delivery stations。若 Amazon 明天消失，全球 e-commerce 退回 2010 水準（Walmart 替代速度極慢，物流基礎設施需 10+ 年重建）。
  • 計算基礎設施層（AWS）：全球第一公有雲 32% 市佔（vs Azure 22%、GOOGL 12%），15 年積累。AWS 不僅是 commodity cloud，更是 enterprise AI 平台（Bedrock、Trainium、SageMaker）。沒有 AWS，全球至少 1/3 of internet workload 中斷。
  • 廣告與商業情報層（Ads + Audience）：第三大 ad platform（vs GOOGL、META），$69B+ TTM 廣告收入。Retail intent data 為廣告主提供 conversion data 黃金，是 GOOGL/META 無法複製的差異化。
  • AI 晶片基礎設施層（Trainium / Inferentia）：與 GOOGL TPU 雙占自研 AI 晶片市場。Trainium $225B 訂單承諾代表 enterprise AI 在 NVIDIA 之外的「第二供應源」，戰略意義超越商業意義。

四層交織形成的飛輪：Retail 用戶 → Prime 會員 → 廣告 audience → Audience data → AWS 客戶 → Trainium 訓練 → AI 平台 → 增強 Retail 推薦 → 更多用戶。明天若 Amazon 消失，全球零售 + 1/3 雲運算 + AI 晶片雙占之一同時真空 = 比 GOOGL 影響更廣（因 Amazon 觸及 physical commerce）。

逆向推演（Munger Inversion）

若要讓 AMZN 在 3 年內虧損 50% 以上，最可能路徑：

  • 地緣政治曝險：(1) 中國反制裁針對 AWS（在中國運營有限但仍存在）；(2) Trainium 製造依賴 TSMC（台海風險與其他 hyperscaler 同陣線）；(3) 歐盟 DMA / DSA 罰金（已被罰多次）；(4) 美國反壟斷對 Marketplace 第三方賣家政策 — Lina Khan 任 FTC 主席期間訴訟仍未結案。實質曝險：中-高。
  • 技術替代風險：(1) Walmart + Shopify 雙占電商（Walmart 已是亞馬遜後第二大 e-com，Shopify 為 SMB 賣家平台）；(2) NVIDIA 在 inference 端鞏固護城河，使 Trainium 無法擴大；(3) 新一代「自主購物 agent」由其他公司主導（如 OpenAI 推出購物 agent 直接接管購買決策，繞過 Amazon UI）。
  • 管理層失職：(a) Capex 過度擴張至 ROIC 跌破 WACC — 目前 ROIC ~13%，WACC ~9%，超額僅 4pp（vs GOOGL 19pp、MSFT 19pp）= 安全邊際薄；(b) Andy Jassy 任 CEO 後 Bezos 退居 Executive Chairman，戰略方向集中度降；(c) 多次大型併購（MGM $8.5B、One Medical、iRobot）整合效果有限。
  • 價值陷阱路徑：四選一最可能 → 「隱性資本密集型」。FY25 FCF $7.7B vs Capex $131.8B 是 5.6% 比率（FCF 僅 Capex 的 5.8%），意味著 90%+ 的 OCF 都被 Capex 吃光。若 2026-27 Capex 維持 $200B+ 而 AWS 增量收入未能等比例擴張，FCF 結構性壓抑。
