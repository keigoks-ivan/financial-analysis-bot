§4 序章：第一性原理 × 逆向思考

存在意義

Magna 解決的不可逆痛點是：「OEM 不可能自製所有汽車零件而仍維持資本效率」。一輛現代汽車有 ~30,000 個零件、~1,500 個次系統，沒有任何 OEM（包括 Toyota / VW）能完整垂直整合所有 systems engineering 能力，特別是涉及 ① 跨平台共享元件（如座椅、車身結構件、外飾）、② 需要規模才能達成單位成本下行（如鏡子總成、燈具、減震器）、③ 安全認證週期長（6-8 年）的零組件。Magna 在這三類別中提供「one-stop, multi-region, multi-program」服務，OEM 將其視為「半 in-house」團隊。Magna Steyr（整車組裝）更進一步解決「OEM 不想為小量產品建專用工廠」的痛點——BMW 5 Series、Mercedes G-Class、Toyota Supra（GR Supra）都在 Graz 廠生產。

若 Magna 明天消失：① OEM 將被迫接管 343 個工廠（不可能）；② Magna Steyr 客戶（BMW / Mercedes）將失去年產 10+ 萬台彈性產能；③ 全球 ~17 萬名員工需被 Tier 2 / OEM 吸收，造成至少 18 月供應鏈中斷。**世界會出現的不可替代混亂：OEM 的「外部 R&D + 製造彈性」消失，每家 OEM 都要回頭做 0 億美元的內部 systems engineering 投資。

逆向推演 三維展開（v12.2 強化）

① 技術維度

什麼技術替代會讓 MGA 3 年內失去 50% 價值？

  • OEM 大規模內製化（in-source）：若 Tesla / 中國新勢力（BYD / Geely）成功示範「整車 in-house 製造 + 9% margin」，傳統 OEM 跟進 in-source 50% 零件 → Magna 營收 -30%、PE 壓縮至 5x → 股價 -50%。歷史對標：Sun Microsystems（被 OEM 整合）。
  • EV 平台化大幅減少零件數：Tesla Model Y 全車架構零件數比同級 ICE 車少 30%（壓鑄一體化、ADAS 整合），若 ICE OEM 全面跟進，Body Exteriors / Power & Vision 訂單需求 -25%。歷史對標：智慧手機 BOM 數從 1,500 → 800（功能整合對零件商不利）。
  • SDV（Software-Defined Vehicle）取代 OEM-Tier 1 分工：軟體 OTA + 中央運算 ECU 取代分散 ECU → ADAS / Vision 業務被 NVIDIA / Qualcomm 等矽片商吃下。歷史對標：Mobile baseband chip（高通崛起取代多家小供應商）。
  • Lidar / 視覺感測商品化：MGA 的 active safety 業務被中國低成本廠（如禾賽、速騰）擠出。歷史對標：LED 照明從 Cree 轉至中國 LED 廠。

② 管理維度（v12.2 新增）

什麼決策路徑會自我毀滅？

  • Capex 慣性復發：若管理層為追隨 EV 趨勢再次推 .5-3B/年 capex 至 FY28（重演 FY23 .55B 教訓），ROIC 將再降至 3-4%，市場對 capital allocation 信心崩盤。歷史對標：Intel 2018-2022 fab capex 超支導致 PE 從 16x 降至 9x。
  • Swamy Kotagiri 退場：Kotagiri 是 2021 後「Margin 優先」戰略的設計者，若 2027-28 退休（Stronach 家族影響仍在 board），新 CEO 可能回到「規模驅動」舊路徑。歷史對標：GE 在 Welch 退場後 strategic drift。
  • Fisker 級別 EV 客戶再失敗：若管理層為追求營收簽下另一個高風險新創（如 Rivian / Lucid），認列大規模沖銷 → 投資人信任歸零。Fisker 已是 first strike。

③ 結構維度（v12.2 新增）

什麼產業變化會讓商業模式過時？

  • 全球 LVP 結構性萎縮至 7,500 萬輛/年（從目前 9,200 萬下行 18%）：年輕世代汽車擁有率下降 + Uber/共享經濟 + 中國市場飽和 → 全產業 capacity 過剩 30%。歷史對標：北美鋼鐵業 1980-2000。
  • OEM 結構性分拆 / 重組：北美「Big 3」被中國 / 韓國 OEM 大幅侵蝕，Stellantis 或 Ford 進入結構性虧損 → MGA 客戶基礎 -20%。歷史對標：Nokia / Motorola 失去手機市場。
  • 監管型強制本地化：USMCA / Inflation Reduction Act 進階版要求 90% 在地化生產 → MGA 在墨西哥 / 加拿大產能需重整，短期 capex 高、長期 margin 不確定。
  • EV 電池 / 馬達一體化：若電池廠（如寧德時代）+ 馬達整合成「powertrain in a box」直接賣 OEM，Power & Vision 業務空間縮小 40%。歷史對標：Intel "Inside" 整合對主機板廠的衝擊。

客戶集中度極高（Top 6 OEM 占 75%，引用 §8.H），議價權結構性弱。合約結構為「Program-based」（一個 program 5-7 年合約 + RFP rebid 每代車型）。近 3 年客戶流失 case：① 2024 Fisker 破產 → 0M 認列（contract manufacturing 風險顯化）；② Stellantis 部分 Jeep 平台轉中國 Tier 1（< 1% 影響）。定價能力的直接證據：FY25 ASP 成長 ~+1.5%（mix shift to ADAS / 鋁件為主，非真正 price increase），無法主動提價。

③ 地緣曝險

核心生產地：北美 48.6%（含墨西哥）/ 歐洲 ~38% / 亞洲 ~13%。供應鏈最高集中點：墨西哥 Tier 1 base（USMCA 範圍內），如果美墨關稅再升 → 部分 OEM 可能要求 MGA 重新對北美 in-source。政策風險具體路徑：① USMCA renegotiation 2026 H2 → 北美在地化比例可能再上修；② EU CBAM 對鋁件影響；③ 中國對外資 Tier 1 限制（MGA 中國業務僅 ~5%，影響小）。已有分散化措施：MGA 已在墨西哥擴產（規避美國工會風險），地緣曝險中等可控。
