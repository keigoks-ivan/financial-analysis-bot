§0｜序章:第一性原理 × 逆向思考

存在意義

Teradyne 解決的是「晶圓良率驗證」這個半導體製造最後的品質關卡。每一顆 NVDA GB300 GPU、每一顆 HBM3E/HBM4 堆疊、每一顆 AMD MI400、每一顆 Apple A19 ASIC、每一顆汽車雷達 MCU,在封裝完成後必須在 Teradyne 或 Advantest 的 ATE(Automated Test Equipment)上跑完功能測試、速度分級、良率驗證。單台高階 Semi Test handler 價格 $3–$8M,一條先進 HBM 測試線可達 $50M+,客戶群極度集中於 Top 10 半導體製造商與 OSAT。若 Teradyne 明天消失,Advantest(日系)在 SoC 領域仍可接手約 60% 份額,但 Teradyne 在 Memory HBM 已佔近 50% 份額、在 Compute SoC 佔 ~50% VIP 份額,短期取代成本極高——測試程式、handler 校準、probe card 介面全部需要重新驗證,行業級遷移至少 12–18 個月。

逆向推演(Munger Inversion):3 年內虧 50% 以上的路徑

  • 地緣政治曝險:中國為 Teradyne 歷史上重要市場(約 15–20% 營收),美國出口管制已經限制向中國先進製程客戶出貨。若進一步擴大至成熟製程或反向遭遇中國 Pre-Buy 逆轉,會壓縮 5–10pp 成長。
  • 技術替代風險:Advantest 在 SoC 領域長期領先(V93000 平台),若 Teradyne 的 UltraFLEX 在下一代 800G+ SERDES、HBM4/HBM5 測試上失去技術領先,市佔可能在 24 個月內倒流。
  • 管理層失職:Universal Robots + MiR 兩個 Robotics 業務自 2015 年併購以來仍在虧損或微利狀態,消耗資本卻沒有貢獻 EPS。管理層堅持「Physical AI」敘事正在把資本配置合理化,但若 3 年內 Robotics 仍無盈利,將被迫減值。
  • 價值陷阱路徑:最可能惡化劇本為「半導體 capex 週期反轉 + 估值極端重估」純週期型。TER 歷史 P/E 均值 27.28x,當前 Forward P/E ~60x(5Y 均 33.55x 的 1.8 倍),P/E 溢價在 v8 系列中最極端。Q4 2025 EPS beat +32.4% 為歷史級,但這種級別的 beat 幾乎等同「週期中段至高峰」訊號。一旦 Semi capex 從加速擴張切換至穩態,AT&E 是反應最敏感的子產業之一。

§3｜護城河分析

評分:8/10(全球嚴格基準)

護城河來源

  • 網路效應:強。客戶(TSMC/Samsung/Micron/SK Hynix)使用 TER UltraFLEX + IG-XL test programming language,形成工程師生態鎖定。切換成本極高。
  • 無形資產:強。高速 SERDES、DDR5/6、HBM 測試 IP 與多年 calibration 資料為 Advantest 難以複製的關鍵。
  • 轉換成本:極強。一條量產測試線投入通常 $30–50M,更換平台需重寫 test program + 重新驗證 + ramp-up 損失,客戶高度黏著。
  • 規模優勢:中等偏強。SoC ~50% 市佔支撐 R&D 攤提與 field service network,但與 Advantest 相比並無絕對規模差。

競爭鴻溝

Advantest 是最強對手,在 SoC 長期與 Teradyne 平分市場,兩者之間每 3–4 年輪流領先。當前 Teradyne 領先來自 HBM + VIP Compute socket。Cohu、Aehr Test 在 burn-in 等特殊領域具利基。中國本土廠商(華峰、長川)在低階測試擴張但未觸及高階。資本充足對手進入需 3–5 年 R&D 累積。

擴大中(HBM + VIP Compute),但 SoC 與 Advantest 的競爭是長期拉鋸戰。下一代 HBM4/HBM5 測試標準將決定 24 個月後的市佔格局。
