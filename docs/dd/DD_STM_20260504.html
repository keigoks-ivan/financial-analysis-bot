§4 序章：第一性原理 × 逆向思考

存在意義

STMicroelectronics 解決的是「歐洲擁有自主先進類比 + 功率半導體 + MEMS 製造能力」這個地緣政治硬約束。歐洲在數位邏輯（CPU/GPU）已輸給美國/亞洲，但在類比、功率（SiC IGBT）、MEMS（汽車 + 工業感測器）、ST33 安全晶片、Auto MCU 仍掌握 second-source 地位。STM + Infineon + NXP 三家形成歐洲半導體「鐵三角」，提供 BMW/VW/Stellantis/Bosch 不依賴美國的類比 IC 與 SiC 動力電子供應鏈。AI DC 業務（AWS multi-year billion + NVIDIA 800V DC + Silicon photonics）切入第二成長曲線。若 STM 明天消失，全球 MEMS 感測（智慧手機、汽車陀螺儀）將剩下博世 + InvenSense 雙寡占，歐洲汽車 SiC 供應鏈將過度依賴 Wolfspeed/Onsemi（美國）+ 三菱（日本）。
