§2｜序章：第一性原理 × 逆向思考

存在意義

Netflix 解決的是人類對「故事」的普世需求與全球娛樂供給的破碎化兩個長期不可逆問題。在 Netflix 之前，觀眾被線性電視節目表、地理疆界、發行窗口、語言壁壘切割；Netflix 建構了一條直連「世界任何創作者 → 任何地區觀眾」的管道，並用演算法把全球內容 TAM 從「美國人看美國劇」擴展到「南韓《魷魚遊戲》能在巴西、德國、日本同時爆紅」。

若 Netflix 明天消失：① 全球非英語內容（K-drama、Anime、西班牙語劇）失去最大的跨國分銷管道，Squid Game 這類全球現象級 IP 幾乎無法再出現；② 全球 2-3 億家庭在同一訂閱週期內需同時重建娛樂消費習慣，各地區本土 OTT 在短期內無法整合全球內容；③ 全球內容產業每年失去 ~$18B 的採購預算與 ~19,000 小時原創內容出口管道。這不是「效率提升」層級的問題，而是全球文化流通基礎設施的消失。

逆向推演（Munger Inversion）——讓 NFLX 3 年虧損 50% 的路徑

維度具體情境
地緣政治曝險內容製作多元分散（美、韓、墨、印、歐），地緣風險相對低；主要曝險為：① 美中貿易緊張波及國際串流（中國市場 Netflix 本來就沒有，影響有限）；② 俄烏戰爭延續使 EMEA ARM 壓制；③ 印度內容審查加劇限制地方擴張
技術替代風險① 短影音（TikTok、YouTube Shorts）持續吃掉「沒有特定內容想看」的被動娛樂時間，Netflix 8-9% TV 時間份額可能被 YouTube 12.6% 繼續拉開；② AI 生成短片讓內容供給爆炸，Netflix 的「策展型長片」優勢被稀釋；③ VR/AR 內容形態演進若由 Meta 或 Apple 主導分發，Netflix 錯失下一代客廳
管理層失職① 過度付費買下非核心體育（如當初若花 $76B 搶 NBA，現金流會崩）；② Games 部門擴張失控，每年 $2B+ 虧損無回報；③ Hastings 退出後董事會動能喪失，Sarandos/Peters 雙 CEO 模式出現路線分歧；④ 買回在高位擴大（2025 $9.13B 均價 ~$80 post-split OK，但若 2026 在 $130+ 繼續大量買回則資本配置失誤）
價值陷阱路徑最可能：隱性資本密集型——Cash content spend $20B/年為維持競爭力所需，若競爭對手（Paramount+WBD 合併後 ~200M subs + $40B+ 內容預算）迫使 Netflix 內容成本再翻倍，FCF Margin 可能從 20.9% 壓回 15% 以下，同時 ad 變現不及預期則雙殺。這是最需要警惕的陷阱模式（雖然目前信號偏弱）

§7｜護城河分析

評分：8 / 10（全球上市公司基準、嚴格）

護城河來源

  • 網路效應：中強——用戶越多 → 觀看數據越豐富 → 推薦越精準 + 內容投資 ROI 越高 → 留存/新簽更強。規模閾值：~100M paid subs（2019 達成後 ROIC 跳升）
  • 無形資產：中強——Netflix 品牌 = 全球串流代名詞；原創 IP（Stranger Things、Squid Game、Wednesday）帶品牌溢價
  • 轉換成本：弱-中——單一串流訂閱轉換摩擦低；但多訂閱家庭的「Netflix as default」習慣強（平均 5.1 hrs/day per active account）
  • 規模優勢：強——$18.6B content spend（vs DIS $23.4B 含線性、PARA+WBD 合併後 ~$40B 含線性）；單位內容成本攤到 300M+ households 為產業最低；global CDN + ISP 合作布建 10+ 年無可複製

競爭鴻溝

資本充足對手（Amazon、Apple、Disney）仍無法跨越的原因：① 內容策展演算法依賴 15+ 年觀看數據；② 全球分發協議 + 190 國家/36 語言 localization 基礎設施；③ 大量原創 IP 庫形成 content moat（Amazon/Apple 仍須購買版權補缺口）。
