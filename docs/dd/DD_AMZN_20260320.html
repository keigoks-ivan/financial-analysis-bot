§0 ｜ 序章：第一性原理 × 逆向思考
存在意義（深度版）

Amazon 解決的是商業社會最普遍的兩個摩擦：「我想要某樣東西，但不確定去哪裡找 / 買最划算」以及「我的公司需要算力，但不想自建 IT 基礎設施」。這兩個痛點都是長期存在且不可逆的需求。

零售端：Amazon 消除了全球消費者在「選擇、價格比較、配送便利性」上的摩擦。Prime 的次日達 / 同日達改寫了消費者對「時間成本」的期待，這個行為模式一旦建立便無法逆轉。若 Amazon 明天消失，全球 Prime 2.2 億會員失去核心便利，2 百萬第三方賣家的主要銷售渠道消失，全球電商生態鏈局部崩潰。

雲端端：AWS 解決的是「不應該讓業務創新被 IT 基礎設施採購週期拖累」的問題。AWS 讓一個兩人新創用信用卡就能部署媲美財星 500 企業的算力，這是工業革命以來最大的企業「槓桿」之一。若 AWS 消失，全球 30%+ 的互聯網流量（Netflix、Airbnb、Stripe、NASA 等數十萬客戶）同時中斷，後果是全球規模的數字基礎設施危機。

2026 年更深層的意義：Amazon 正從「世界最大商店 + 最大雲端」演變為「AI 基礎設施的中立平台」——透過 AWS Bedrock（多模型）、Trainium / Inferentia 自研晶片、Anthropic 深度合作、OpenAI 新夥伴關係，Amazon 試圖成為企業 AI 採購的「中立基礎設施提供商」，類似電力公司的定位：不押注哪個 AI 模型最終勝出，而是為所有贏家提供算力。

💸 $200B Capex ROI 失敗
當前最大財務壓力源		 2026 Capex $200B（vs 2025 $132B，+52%），而 FY2025 FCF 已壓縮至 $11.2B（-71% YoY）。若企業 AI 工作負載商業化速度低於預期，$200B Capex 無法在 2027-2028 年轉化為足夠的 OCF 反彈，FCF 持續低迷。市場需要看到 AWS 利潤率擴張和 FCF 反彈，否則「投資回報黑洞」敘事持續壓制估值。 中等 若 FY2026 AWS 成長降至 15-18%（vs 當前 24%）：FCF 惡化敘事強化，估值壓縮 -30~45%
⚖️ FTC 反壟斷試審（2026-10）
結構性監管風險		 FTC 起訴 Amazon 在 Marketplace 使用反競爭手段（「Project Nessie」價格操控算法、強制賣家使用 FBA）。試審定於 2026 年 10 月。若法院裁定 Amazon 需拆分 Marketplace 或限制物流整合，影響廣告 + 第三方賣家收入模型（年收入 ~$150B 的利潤池）。 低至中 強制拆分 Marketplace：EPS 影響 -20~30%，估值長期折扣
🌐 AWS 市佔被蠶食
Cloud 增速放緩風險		 Azure（+39%）和 Google Cloud（+48%）的增速均高於 AWS（+24%）。AWS 在 AI 工作負載上面臨「Azure 有 OpenAI / Google Cloud 有 DeepMind」的差異化競爭。若 AWS 的 AI 工作負載份額無法抵禦競爭，成長率持續下行至 15-18%，高估值倍數失去支撐。 中等 AWS 市佔每下降 1 個百分點 = 年損失 ~$2B 高毛利收入；EPS 下調 + 倍數壓縮
📦 零售/廣告景氣下行
宏觀敏感性		 消費端：美國消費衰退 / 關稅上升侵蝕北美電商利潤。廣告端：宏觀廣告支出萎縮。零售 + 廣告是 Amazon 最大的 OCF 來源（~$100B/年），若同時受壓，AWS Capex 的資金池收窄，財務壓力倍增。 低（短期）/ 中（若衰退） 嚴重消費衰退：整體 EPS -20~30%，拖累估值
最大系統性風險：$200B Capex + FTC 試審 + AWS 增速放緩三者同時發生——FCF 黑洞、EPS 下調、監管折扣三重壓縮，理論上可造成 -50%+ 股價損失。但這也是當前 $209 vs 52 週高點 $259（-19%）的主要折扣來源——市場已部分定價，問題是定價是否充分。

（最核心護城河）		 極強（建設期超過 20 年） AWS 自 2006 年起建立的全球數據中心網絡（99 個可用區，33 個地理區域）是 20 年資本積累的結果。企業遷移至 AWS 的「搬家費」包括：重構應用架構、重訓工程師、測試可靠性——平均遷移成本超過 $500K/企業，且完成後很少反向遷移。$244B 積壓訂單說明客戶已鎖定多年，轉換成本極高。
電商飛輪（Prime + 物流） 強（難以複製的物流網絡） Prime 2.2 億付費會員（年費 $139）創造了全球最大的預付費購買行為承諾。Amazon 在美國建立了 1,500+ 配送設施，實現全美 90% 地區次日達或同日達——這個物流密度是沃爾瑪或 Shopify 等競爭者無法在 10 年內複製的基礎設施護城河。8 億+ 同日達訂單（FY2025）是這個護城河的商業化驗證。
廣告數據護城河 強（購買意圖數據 = 廣告最高價值） Amazon 廣告的獨特性在於：基於實際購買行為的定向（vs Google 的搜尋意圖、Meta 的社交行為）。買家在搜尋「跑鞋」時投放廣告 vs 在準備購買「跑鞋」時投放廣告——後者的轉換率高 30-40%，使廣告主的 ROI 更高，形成壓倒性的廣告定價能力。$85B 年化廣告收入的利潤率估計超過 50%。
自研晶片（Trainium / Graviton） 中強（建立中，$10B+ 年化） Amazon 是全球少數有能力設計並大規模部署自研 AI 晶片的公司（其他是 Google TPU、Apple A-series、微軟 Maia）。Trainium2 為 AI 訓練提供的性價比據稱比 Nvidia H100 高 30-40%（特定工作負載）。這讓 AWS 可以用更低成本提供 AI 服務，而利潤率比使用 Nvidia GPU 更高。$10B+ 年化且三位數成長，是護城河正在強化的信號。
AI 中立平台（AWS Bedrock） 快速建立中（差異化策略） AWS Bedrock 提供 Anthropic（Claude）、Meta（Llama）、Mistral、Cohere 以及現在 OpenAI 的多模型存取——定位為「AI 模型不可知論」平台。企業不需要押注哪個 AI 模型勝出，只需要在 AWS 上按量付費調用任何模型。這個策略解決了企業「選模型選錯了怎麼辦」的顧慮，比 Azure（OpenAI 強綁定）和 GCP（Gemini 優先）更中立，是 AWS 在企業 AI 採購上的獨特競爭優勢。
