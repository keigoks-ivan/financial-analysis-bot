§2 序章：第一性原理 × 逆向思考

存在意義

Teradyne 解決的長期痛點是：「半導體晶片在交付前必須被驗證無誤」——這是比晶片製造本身更沒彈性的剛性需求。AI 加速器、HBM 堆疊、自動駕駛 SoC 每一顆晶片出貨前都必須經過 ATE 測試，測試設備是晶片「出生前的產房」。若 TER 明天消失，全球只剩 Advantest 能做高階 SoC 與 HBM 測試，AI GPU 產能將出現 bottleneck，因為測試產能本身有 12 個月的 lead time，短期無法替代。

延伸的存在意義：Universal Robots 作為全球最大協作機器人平台，讓中小企業（無大型 ABB/FANUC 預算的）也能部署機器人——這個「民主化自動化」的使命在人口老化與勞動力短缺背景下，是另一個不可逆的人類需求。

逆向推演（Munger Inversion）

若 TER 要在 3 年內虧損 50%+，最可能的路徑：

  • 地緣政治曝險：TER 客戶集中於台積電、三星、SK 海力士、美光、NVIDIA。若台海衝突或美中科技戰升級導致先進製程產能被切斷或轉移，TER 的高階 SoC 測試需求可能暴跌 30-40%。美國本土製造回流雖為 TER 友善，但轉移期可能有 18-24 個月的需求空窗。
  • 技術替代風險：短期無強力替代。中長期（5+ 年）若產業轉向 chiplet + co-package + self-test（on-die BIST）普及，外部 ATE 設備需求強度下降——但 TER 已在做 SLT（System-Level Test）佈局以對沖此風險。
  • 管理層失職：CEO Greg Smith 2024 總薪酬 $12M（90%+ 為股權），接任者——非激勵錯位。但若於此估值高點進行大型 M&A（例如為了延伸 Robotics 生態系以高價收購），可能觸發資本配置錯誤警訊。
  • 價值陷阱路徑：當前非陷阱——但若 AI capex 於 2026 下半年或 2027 年見頂，TER 可能複製 2022-23 的 EPS 腰斬情境（兩年 EPS 從 $5.47 → $2.73，跌 50%），此時 PE 從 58x 壓縮至 20x，股價最大回檔可達 70%。這不是「價值陷阱」（品質沒有變差），而是「成長股週期見頂」的典型路徑。

§7 護城河分析（報告核心）

評分：8 / 10（ATE 雙寡頭結構堅實，但非 ASML/TSMC 等級的絕對壟斷）

護城河來源

  • 網路效應（弱）：Test program 在晶片開發社群與 IC 設計工具有些許網路外部性，但非核心。
  • 無形資產（強）：UltraFLEXplus test 平台與客戶晶片 tape-out 流程深度整合；test pattern library 是每次 tape-out 幾十萬美元的沉沒成本。
  • 轉換成本（非常強）：從 TER 換到 Advantest 需重寫 test program、重新驗證數千個 test vectors、重新認證客戶 qualification——一個大客戶可能花 9-18 個月才能完成切換，且風險極高。這是 TER 最大的護城河。
  • 規模優勢（中）：R&D 規模（$400-500M/年）對中小對手（Cohu, Chroma）構成障礙，但 Advantest 規模近似，雙方互卡。

競爭鴻溝

資本充足的對手仍無法跨越，因為：① 客戶 qualification 流程每個新平台需 2-3 年驗證；② test engineer 人才池小而專，挖角困難；③ IP library 累積需要 10+ 年的 tape-out 經驗——這不是砸錢能買到的。
