§4 序章：第一性原理 × 逆向思考

存在意義

CoreWeave 解決一個 GPU 時代的「資本配置 + 速度」問題：當 hyperscaler（Microsoft、Google、Amazon）的 capex 已年化 $100B+ 仍無法滿足內部 AI workload demand，誰能扮演 dedicated AI GPU 「outsourced capacity」 partner？CRWV 從 ETH mining startup（2017）轉型為 AI GPU cloud（2022）的歷史定位，使其成為 Microsoft 在 2022-2025 ChatGPT 浪潮中最關鍵的 outsourced GPU partner — Microsoft 2024 / 2025 兩年從 CRWV buy 67% revenue 集中度反映了這個共生關係。如果 CRWV 明天消失，Microsoft Azure AI 短期內無法填補 $30-40B 級的 GPU 容量缺口，OpenAI 的 ChatGPT 服務可能在 6-12 個月內出現大規模容量限制。

① 技術維度：什麼技術替代會讓 CRWV 3 年內失去 50% 價值？

② 管理維度：什麼決策路徑會自我毀滅？

  • founder 退場：三位 founder（Intrator CEO、Venturo CSO、McBee CDO）任一退場可能影響戰略連續性

③ 結構維度：什麼產業變化會讓商業模式過時？

  • 監管型分拆 / Microsoft 強制 spin-off Azure 的 AI workload → 客戶 single-point lock 結構解除
  • 計算範式轉換（Transformer → state-space / neuromorphic） → 對應全新硬體需求
  • 能源短缺 + 電價暴漲：CRWV 無自建 power source（vs NBIS Pennsylvania 1.2GW），對 grid 電價更敏感

價值陷阱路徑：CRWV 最有可能變成「隱性資本密集型 + 客戶集中型」雙重陷阱 — Debt $25-29B 對單一 Microsoft 客戶 67% 依賴的脆弱組合是 NBIS thesis 沒有的特殊風險。
