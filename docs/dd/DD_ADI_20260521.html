§4｜序章：第一性原理 × 逆向思考

存在意義

ADI 解決一個結構性、不可逆的長期問題：類比訊號（電壓、溫度、壓力、光、聲、磁）必須被精準轉換為數位訊號，才能進入 AI / 自動化系統處理，再轉回類比驅動實體世界。這個轉換邊界（analog-digital boundary）在物理層面不可省略：

  • 工業 4.0 / Industrial AI：智慧工廠每個感測點（震動、溫度、馬達電流）需要 ADI 的 precision ADC + signal conditioning 才能讓 AI 邊緣判斷；典型 design 是 ADI ADC + power management + isolation 全棧
  • AI infrastructure：data center 每張 GPU 卡 / 每個 power delivery network 需要高效率 DC-DC、PMIC、power monitoring；ADI 的 LTC（Linear Technology, 整合於 2017）+ Maxim power 在這層佔關鍵地位
  • 電動車 / ADAS：BMS（battery management）、GMSL（Gigabit Multimedia Serial Link，車用高速影像連接）、ADAS sensor fusion 都依賴 ADI 全棧
  • 醫療電子：ECG、超音波、CT scanner 等 high-precision signal chain

若 ADI 明天消失：① 工業 IoT 升級停擺 6-12 個月（替代 TI 廣度不足、英飛凌偏 power 缺 precision analog）；② 中高階汽車 BMS / ADAS 設計流程被迫重新驗證；③ AI data center power 設計 BOM 增加 10-15% 成本。這不是「提升效率」式的存在意義，而是「物理層斷點」式的結構必須。

逆向推演（Munger Inversion 三維展開）

① 技術維度：什麼技術替代會讓 ADI 3 年內失去 50% 價值？

  • 光學 sensing 替代電氣 sensing（部分業務）：fiber-optic sensor 在工業 / aerospace 部分場景已開始取代電子 sensor；但 mainstream Industrial 4.0 仍依賴電氣（成本、可重編程性）。歷史對標：1990s 數位相機侵蝕 Kodak 底片，速度需 ~10 年（ADI 緩衝期足夠）
  • 整合式 SoC（系統單晶片）內建 ADC/DAC：NVIDIA / Intel / AMD 高階晶片開始整合 power management 與部分 sensor interface（縮小 ADI 的 attach 機會）。但 high-precision 訊號鏈仍需獨立晶片（物理限制：noise、isolation、temperature drift）
  • 純量子感測（高階研究）：未來 10-15 年若量子 sensor 商業化 → 全行業重排，但 3 年內機率極低
  • 歷史對標：Sun Microsystems 被 x86 + Linux 取代（10 年），Intel 被 ARM + RISC-V 緩慢侵蝕（15+ 年）— analog 行業變動週期長於 digital，ADI 3 年內被技術替代機率 < 5%

② 管理維度（NEW）：什麼決策路徑會自我毀滅？

  • Capex 慣性 / 自建晶圓廠失利：ADI 目前 hybrid 製造（部分自建 + 大部分 foundry）；若強推 fab 投資追 IDM 模式失敗，可能 GM 跌 5-10pp。歷史對標：Cypress 2010s 試圖擴 fab 失敗、Maxim 2010s 一度 fab 過剩
  • 整合疲勞 / 創辦人 / 領導層流失：CEO Vincent Roche（2013 起）+ CFO Richard Puccio 是 Maxim 整合主要操盤手；若 Roche 退休後接班人不力 + Maxim 整合留任高管離職 → execution moat 8.5 可能下滑 1-2 點。歷史對標：Intel CEO 變動引發 10nm 延誤；AT&T 多次 CEO 失敗整合 Time Warner
  • 過度多角化 M&A（追逐 AI narrative）：若 ADI 用 $20-30B 收購一家 AI 軟體 / 系統公司 → ROIC 立即下行（從 ROIC trough 6.4% 雪上加霜）。歷史對標：HP Autonomy（$11B 收購後 $8.8B 減損）、Microsoft Nokia（$7B 減損）

③ 結構維度（NEW）：什麼產業變化會讓商業模式過時？

  • 晶片 IP 開放 / RISC-V 風潮擴至 analog：若 analog IP 從專利保護 → 開源化（如 Google 牽頭 SkyWater 130nm PDK 開放）→ ADI 的 IP moat 受侵蝕。但 analog IP 高度依賴 process 與 manufacturing know-how，開源化速度比 digital RISC-V 慢 10 年
  • monetary regime 大轉換 / 美國半導體政策反轉：若 CHIPS Act 重新洗牌、補貼從 IDM 轉到 Foundry → ADI hybrid 模式失利。歷史對標：1970s 日本半導體政策變動讓美國公司洗牌；2024 中國半導體政策劇變讓國際公司應對
  • computing paradigm 轉換（neuromorphic / analog computing）：若 analog computing chip 從 digital ADC/DAC 介面消失 → ADI 部分 attach 業務萎縮。歷史對標：Intel x86 vs Mobile ARM 的 25 年轉換

逆向推演結論：ADI 3 年內失去 50% 價值的最可能路徑是 ②管理維度（Capex 失誤 / 過度 M&A 追逐 AI），機率 ~10%；技術 ①與結構 ③維度 3 年機率 < 5%。主要 hedge：管理層 capital allocation 紀律 + Maxim 整合留任管理。

價值陷阱路徑判斷：本次 DD 主要 trap 風險是「盈餘品質短期警訊（GM 見頂 + tariff pull-forward）」型，非結構性「護城河侵蝕」型。Q2 FY26 的 GM 73% 為 cycle / non-recurring 重新定價紅利疊加，下個 6-12 月需 normalize；tariff pull-forward 的 auto 動能下半年「還回去」。Thesis 結構穩固，估值短期透支。

高效能類比 + 電源管理 IC 全球 TAM ~$80B (2025)，CAGR ~6-8% 至 2030（達 ~$110-120B）；其中 Industrial AI smart factory 細分 TAM 從 $15B → $35B (5Y)，AI infra analog/power 細分 TAM 從 $5B → $15B (5Y) 結構性新增。當前供需狀態：cycle bottoming → recovery 中段（庫存去化完成，需求補庫 + 結構升級雙加成）。

議價權（三段獨立）

① 對上游供應商

ADI 對 foundry（TSMC 65/40nm BCD 為主、自建 hybrid Wilmington 8 吋 fab 補充） + EDA tool（Synopsys / Cadence） + 設備（KLA / Lam / AMAT）有合理議價權。Foundry 端因 ADI process node 較舊（> 28nm 為主），不與 leading-edge TSMC capacity 競爭，議價空間穩定。近 3 年無明顯供應端漲價傳導；EDA 採訂閱授權 ~$50-100M/年，談判分散。供應商集中度：TSMC 占 wafer outsourcing ~50%，但因 process node 非 cutting-edge，切換 fab 風險低。

② 對下游客戶

ADI 客戶高度分散（Top 10 ~30%），議價權強。Industrial 客戶（萬家分散）lock-in 7-15 年，提價能力近 3 年明顯（ASP +5-8% YoY 含 mix）。Auto Tier 1（Bosch、Continental、Denso、ZF）採 dual-source 設計，ADI 與 NXPI/Infineon 直接競爭，提價能力中等。Hyperscaler（Comms data center 客戶 NVIDIA/AVGO/AWS）議價權較高但 ADI 因全棧整合（signal+power）議價地位仍穩。近 3 年無重大客戶流失公告，Auto FY25 cycle bottom 期間訂單延遲但無 design win 流失。

③ 地緣曝險

ADI 主要生產地：美國（Wilmington MA 8 吋 fab、總部）、愛爾蘭（Limerick 6/8 吋 fab）、新加坡 + 馬來西亞（assembly/test）。美國本土製造比例 ~30-40%，受益 CHIPS Act 補貼。Foundry 外包 50% 在 TSMC（台灣），exposure 至中美科技競爭。出口管制風險：Auto / Industrial 為主要市場，受 export control 影響 ADI 約 5-8% 營收（中國敏感終端應用）。已有的分散化措施：① Maxim 整合後 fab 網絡擴展；② 越南 / 馬來西亞 assembly capacity 持續擴大；③ 與 GlobalFoundries 等次要 foundry 多元化。
