§0｜序章：第一性原理 × 逆向思考

存在意義

AMD 解決的核心痛點是高效能運算的壟斷替代問題。在 x86 CPU 領域，若 AMD 明天消失，Intel 將成為唯一 x86 供應商——所有雲端超大規模廠商、企業伺服器與 PC OEM 將失去議價能力，CPU 創新節奏將由單一供應商決定。在 AI GPU 領域，AMD 是唯一具備完整 datacenter GPU 生態系的 NVIDIA 替代者（Instinct GPU + ROCm 軟體棧 + 封裝整合能力）。若 AMD 消失，AI 基礎建設的供應鏈集中度將進一步極端化，NVIDIA 的定價權將毫無牽制。

更具體地說：AMD 的 EPYC 處理器在五年內從接近 0% 的伺服器市佔率攀升至約 39%，這不僅是技術執行力的證明，更代表了整個產業對「不讓單一供應商主宰運算基礎設施」的結構性需求。

逆向推演（Munger Inversion）

若要讓 AMD 投資在 3 年內虧損 50%+，最可能的路徑：

① 地緣政治曝險

AMD 是 fabless 設計公司，100% 依賴台積電先進製程（5nm/3nm）。台海衝突或台積電產能中斷將直接癱瘓 AMD 的交貨能力。2025 年的 MI308 出口管制事件已造成約 $440M 存貨相關損失——這僅是中國限制，尚未觸及台灣風險。此外，Helios 系統使用的 HBM（高頻寬記憶體）依賴 SK 海力士與三星，兩者均位於韓國半島。

② 技術替代風險

最大威脅來自hyperscaler 自研晶片的加速滲透：Google TPU、Amazon Trainium/Inferentia、Microsoft Maia 均已進入量產階段，專門針對自家推理工作負載優化。若 AI ASIC 到 2030 年達到 $85B 市場規模（Yole 預估），AMD 的 Instinct GPU 在推理端的 TAM 將被侵蝕。時間軸：2～3 年內可見顯著影響。NVIDIA 的 CUDA 生態系黏性仍遠強於 AMD 的 ROCm，如果 ROCm 未能在軟體成熟度上縮小差距，大型客戶可能直接跳過 AMD 轉向自研。

③ 管理層失職情境

Lisa Su 的資本配置紀律至今良好，但風險在於：(a) 為追趕 NVIDIA 而在 GPU 軟體與系統整合上過度投資（2025 年 OpEx 已 YoY +24%），若 MI450/Helios 的客戶採用率低於預期，R&D ROI 將急劇下降；(b) 2022 年 Xilinx 收購產生的 ~$50B 商譽若因 Embedded 業務持續低迷而面臨減損測試壓力。

② Earnings Call 語氣解析

管理層基調：樂觀，較上季進一步升溫

Lisa Su 在 call 中將 2025 定義為 AMD 的「defining year」，並使用「multi-year demand supercycle」描述 AI 運算需求。相較上季（Q3 call）聚焦產品路線圖，本季語氣更具系統性野心——首次強調「full AI factories」和 rack-scale 系統戰略。

Forward Guidance

Q1 2026 Revenue guidance ~$9.8B ±$300M（+32% YoY，QoQ -5% 為正常季節性）。Non-GAAP GM ~55%（回歸「乾淨」毛利率，無存貨回沖）。Data Center revenue 全年成長目標 >60%。市場對 guidance 的反應偏負面（盤後跌 ~8%），原因是部分分析師期待更激進的全年指引。

刻意強調的主題

反覆出現的關鍵字：「MI450」（H2 2026 量產）、「Helios rack-scale」（Q3 2026 部署）、「Venice」（下一代 EPYC CPU，H2 2026）、「6 gigawatt-scale deployments」。管理層希望市場相信：AMD 正從晶片供應商轉型為 AI 基礎設施完整方案提供者。

刻意迴避的問題

① 中國 MI308 銷售：Su 僅確認 Q1 $100M，明確表示「不預測額外中國營收」——暗示出口管制風險仍在演變中。② Semi-custom（遊戲主機）2026 全年展望：僅說「significant double-digit decline」，未給出精確數字——下一代 Xbox 延遲至 2027 的影響較預期更深。③ MI450 的具體客戶訂單量：雖強調「multiple gigawatt-scale opportunities」，但未點名任何已確認的大型客戶合同。

③ 產業 Imply 萃取

需求端

AI 基礎建設支出持續加速。Su 提到的「multi-year supercycle」與 hyperscaler 2026 capex 指引一致（Meta/MSFT/Google 均上修）。EPYC CPU 的 ASP 提升（+31% YoY in Client）暗示企業願意為更多核心數和更高效能付溢價。

供給端

AMD 庫存 QoQ 增加 $607M 至 $7.9B，管理層稱是「為滿足強勁 data center 需求而主動備貨」。這對台積電先進製程產能的需求訊號是正面的，但也意味著 AMD 正承擔更高的存貨風險。HBM 供應鏈（SK 海力士）的配置似乎已得到保障，但 MI450 的 HBM4 需求可能在 H2 2026 產生瓶頸。

競爭格局

Intel 在伺服器 CPU 持續失血——AMD EPYC 市佔已達 ~39%。在 AI GPU 端，AMD 仍是 NVIDIA 的唯一有意義替代者，但 hyperscaler 自研晶片（ASIC）的崛起正在侵蝕整個 GPU TAM 的增量部分。

對同業連帶推論

AMD 的強勁 EPYC 數據暗示 Intel 的 Xeon 市佔流失仍在加速。台積電的先進製程利用率將維持高檔。SK 海力士與三星的 HBM 訂單能見度良好。NVIDIA 的定價權短期內仍穩固，但 AMD 的 Helios 系統方案可能在 H2 2026 開始爭取到邊際客戶。

§3｜護城河分析（報告核心）

評分：7 / 10

以全球上市公司為基準。AMD 擁有真實但有條件的護城河——核心在於 x86 雙寡佔地位與產品執行力，但在 AI GPU 端仍面臨 NVIDIA 的巨大軟體生態差距。

網路效應：弱至中等

x86 生態本身有弱網路效應（軟體相容性）。ROCm GPU 軟體平台正建構開源社群效應（downloads 10x YoY），但距 CUDA 的黏性仍有顯著差距。ROCm 尚未跨越「開發者主動選擇」的臨界點。

無形資產：中等偏強

x86 ISA 架構授權是極高壁壘——全球僅 Intel 與 AMD 擁有。這是最核心的護城河來源。半導體設計 IP（Zen 架構、CDNA GPU 架構、Infinity Fabric 互連技術）需要數十年累積與數千名工程師，非資本可快速複製。但 AMD 沒有顯著的品牌溢價定價能力——與 NVIDIA 相比，AMD 往往需要提供更好的性價比才能贏得客戶。

轉換成本：中等且上升

Server CPU 的轉換成本正在增加：企業客戶一旦在 EPYC 上部署工作負載、建立軟體認證、訓練 IT 團隊，更換 CPU 平台的真實摩擦成本以年計。AI GPU 端，模型訓練基礎設施一旦建立在 ROCm 上（如 OpenAI 的 6GW 部署承諾），短期內不會遷移。但 CUDA 到 ROCm 的初始轉換仍是 AMD 的最大銷售障礙。

規模優勢：中等

AMD 的 fabless 模式意味著它不擁有晶圓廠規模優勢（那屬於台積電）。但其設計規模正接近臨界點：Revenue $34.6B 支撐了 ~$13B R&D+OpEx 的再投資水平，若持續成長，R&D 攤銷在每美元營收中的占比將下降，形成規模報酬遞增。

競爭鴻溝

資本充足的對手（如 Intel）為何無法跨越：Intel 在製程工藝上已落後台積電 2-3 年（Intel 18A 仍未量產）；而 AMD 搶先使用台積電最先進製程，在效能功耗比上維持領先。Broadcom、Marvell 等具備晶片設計能力但缺乏 x86 授權。ARM 架構在伺服器端（Ampere、Graviton）正在上升但目前僅在特定工作負載上有競爭力，x86 生態的龐大軟體基底短期內不可替代。

護城河趨勢：擴大（但有條件）

x86 Server CPU 護城河持續擴大（EPYC 市佔加速上升）。AI GPU 端護城河正在「從無到有」建構中，但是否能穩固取決於 ROCm 軟體生態成熟度與 MI450/Helios 的客戶驗證結果。

24 個月內最可能瓦解護城河的變化（按威脅排序）：

1. NVIDIA Blackwell + Rubin 架構大幅拉開效能差距——若 MI450 的 training 效能 benchmark 不及 Rubin 的 70%，大型客戶將無動力遷移。
2. Hyperscaler 自研 ASIC 加速取代推理端 GPU——直接壓縮 AMD AI GPU 的可獲取市場。
3. ARM 架構在 Server CPU 的滲透加速——AWS Graviton 4 / Google Axion 若在價效比上超越 EPYC，將侵蝕 AMD 的 CPU 增長故事。

§5｜產業格局、議價權與 Unit Economics

市場量化

Data Center Semiconductor TAM（2024）：$209B（Yole Group），至 2030 年預計接近 $500B（不含 AMD 自家 $1T 估計）。GPU 細分市場 2024 $100B → 2030E $215B。AI ASIC 2030E $85B。
CAGR：Data Center 整體 ~15-20%；AI GPU 子市場 ~25-30%；Server CPU ~18%。
供需狀態：AI 加速器供不應求（HBM 為瓶頸），Server CPU 供需平衡偏緊。

議價權

對上游（台積電）：中等偏弱

AMD 是台積電前五大客戶之一，但 Apple 與 NVIDIA 的優先級更高。先進製程（3nm/2nm）的晶圓價格由台積電主導。AMD 無法自行轉單——全球僅台積電能提供所需的製程節點。

對下游（客戶）：中等且改善中

Server CPU 端：對 hyperscaler（Microsoft、Meta、Google、Oracle）有合理議價力——客戶需要 AMD 作為 Intel 的制衡。但大客戶的訂單規模使其具備反向議價力。
AI GPU 端：作為 NVIDIA 替代者，定價權有限——客戶選擇 AMD 主因是性價比而非無替代性。客戶集中度高：前 5 大 hyperscaler 可能占 Data Center 營收 60%+。

地理分布曝險

設計：美國（Santa Clara）。製造：100% 台積電（台灣）。封裝測試：台灣 + 馬來西亞。HBM 供應：韓國。地緣風險高度集中於台灣與東亞。

§8｜投資結論

財報信號與估值銜接

Q4 FY2025 財報大幅 Beat 市場預期（Revenue +6.4%、EPS +16%），但 Q1 guidance 未能進一步超越最樂觀預期。這份財報讓 Reverse DCF 隱含預期（22-25% FCF CAGR）變得更容易兌現——FCF 已從 $2.4B (FY24) 躍升至 ~$6.7B (FY25)，年增近 3 倍，證明 AMD 的現金生成能力正處於非線性改善階段。H2 2026 的 MI450 ramp 將是關鍵催化劑。

投資判斷：進場（有條件）

① 短期信號（0-12 個月）

Q4 EPS 大幅 Beat 且 FCF 創新高，Q1 guidance +32% YoY 在季節性調整後仍屬強勁。但 semi-custom 2026 全年將大幅下滑，可能使整體營收增速在 H1 2026 看起來放緩。短期催化劑在 H2：MI450 量產出貨 + Venice EPYC 發布，Q1-Q2 為相對平靜期。當前價位的短期風險：若 Q1 財報（5 月 5 日）的 Data Center 增速低於 60% YoY 目標節奏，可能觸發下修。

② 中期成長確信度（1-3 年）：高

分析師共識 EPS CAGR 47.9% 的驅動來源：(1) EPYC 伺服器 CPU 持續搶佔 Intel 市佔（量 + 價），占成長的 ~40%；(2) Instinct AI GPU 從 $5B (FY24) 向 $20B+ (FY27E) 的規模化（量驅動為主），占成長的 ~50%；(3) Client CPU 市佔擴張提供底層支撐。能見度：高——Server CPU 有三年產品路線圖（Turin → Venice → 下一代），AI GPU 有 OpenAI 等公開大客戶承諾。

③ 長期護城河邏輯（3 年後）

持有 AMD 超過 3 年的邏輯基礎：

• Runway（§2.5A）：8-10 年。Data Center TAM 從 $209B → $500B+（2030E），AMD 在 CPU 端持續擴張市佔，AI GPU 端從接近 0 向 10%+ 滲透。

• ROIC 趨勢（§2.5D）：方向正確但絕對值仍偏低。隨著 Xilinx 商譽攤銷完畢（~2032 年後）及營收規模翻倍，GAAP ROIC 將自然提升——但這需要 5+ 年的耐心。

• 護城河連動（§2.5C）：強化型——每多一個 EPYC/Instinct 客戶都增加生態系黏性，ROCm 社群效應正在累積。

3 年後的持有理由成立，但前提是 AI GPU 業務必須在 H2 2026 證明規模化能力。
