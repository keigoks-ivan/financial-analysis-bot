§4｜序章：第一性原理 × 逆向思考

存在意義

Lam Research（LRCX）是 WFE（Wafer Fab Equipment）「Big 3」之一（與 Applied Materials、KLA 並列），解決「把矽晶圓上的奈米級結構刻蝕 + 沉積出來」這個半導體製造最關鍵的物理問題。每一個 NAND flash bit、每一個 DRAM cell、每一個 logic transistor 都需要 LRCX 的 etch（刻蝕）+ deposition（沉積）設備才能製造。當 3D NAND 從 200 層升至 300+ 層（2027 完成大部分轉換），當 DRAM 進入 1c node + DDR5/HBM4，當 2nm/A16/A14 logic 全面量產，LRCX 的設備需求與「層數 × 結構複雜度 × 晶圓量」三因子成正比。

具體不可替代性：① etch 全球 #1（TEL 是 #2 但落後 5 年技術深度，特別是 ALE atomic layer etch）；② deposition 與 AMAT 雙寡占（ALD/CVD/PVD）；③ HBM TSV 近乎獨家（Syndion etch + SABRE 3D copper plating + VECTOR TEOS 3D dielectric exclusive 給 SK Hynix/Samsung）；④ R&D 累積 40+ 年 etch know-how 不可短期複製；⑤ 全球 service network 250+ sites 對 wafer fab 24/7 支援。若 LRCX 明天消失：① 全球 30%+ etch 設備失去原廠服務；② NAND / DRAM 高良率 process 中斷；③ HBM TSV 製程須 18-24 個月才能轉至 TEL（次選）。

逆向推演（Munger Inversion）

若要讓這筆投資在 3 年內虧損 50%+，最可能的路徑是什麼？

  • 地緣政治曝險：China 34% revenue（中），US export controls 2025-2026 持續加嚴。但 LRCX 已主動 reduce China dependence（從 ~45% peak 降至 34%），且設備出口仍合規通道存在。地緣政治 risk: 中。
  • 技術替代風險：① TEL etch 加速追趕 LRCX；② 新進入者中國本土（中微 + 北方華創）在低階 etch 突破，但高階 ALE / HBM TSV 仍 5-10 年差距；③ EUV-only 製程替代部分 multi-pattern etch（但 EUV + etch 仍互補）。技術替代 risk: 低-中。
  • 管理層失職：CEO Tim Archer（2018 上任）執行力強，FY24-26 連續 beat-and-raise；CFO Doug Bettinger（2014 上任）保守。資本配置：FY25-26 5Y 85% FCF return 框架 + $2B buyback。激勵錯位風險低。低。
  • 價值陷阱路徑：四選一最可能 = 「產業結構衰退」變體──但屬「cycle 反轉」非「結構性」：WFE 是高 cyclical（2022 -25%、2024 -10% trough），下一個 down cycle 可能 2028-2029。當前 2026-2027 是 supercycle 高位，後段須提前 risk off。

50% 虧損的最高機率組合：① 2027 WFE 共識從 $145B 反轉至 $115B（cycle peak 過後）；② 同期 HBM 投資 plateau；③ China export controls 突然 strict（< 25% revenue）；④ FY+1 PE 從 35x reset 至 20x。三者 / 四者同時發生機率約 15-20%。
