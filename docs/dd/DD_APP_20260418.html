§4 序章：第一性原理 × 逆向思考

存在意義

AppLovin 解決「廣告預算的邊際 ROAS 無法即時歸因」這個自數位廣告誕生以來的基本痛點。傳統 DSP（The Trade Desk 類）把決策權交給買方——買方須自選 audience、自配預算、自測 creative；AXON 2.0 把決策權交給 AI：每秒 200 萬筆 auction、10 億裝置的 cross-app behavioural graph、即時 ROAS 反饋。對中小型遊戲廠商，AXON 的「黑箱但可預測」比 Meta Advantage+（封閉生態）或自建 DSP（缺資料）更有效。若 APP 明天消失：中小 dev 被迫選擇（a）把錢丟進 Meta/Google 封閉生態、失去跨平台歸因；（b）自建 DSP、承受 60% 的資料劣勢。它取代的不是一家公司，而是「中小廣告主獲得可預測 ROAS 的能力」。

逆向推演（Munger Inversion）

  • 地緣政治曝險：研發在美國/EMEA，資料處理雲端 AWS/GCP 美國為主；手遊發行業務已於 2025/5 以 $400M + 20% 股份賣給 Tripledot，地緣風險大幅降低。主要曝險是「AI/隱私監管」而非生產地。
  • 技術替代：① Apple iOS ATT+ / Privacy Sandbox 若封鎖 cross-app device ID，AXON 資料飛輪被截（時間 12-18 個月）；② Meta 若開放 Advantage+ API 給第三方 dev（可能性中）；③ 新進 AI-agent 廣告 orchestrator（OpenAI Operator 類）整合 GPT-5 + API（2026-2027）。
  • 管理層失職：CEO Foroughi 近 5 年 25 筆內部人交易全為賣出（2026/3/12 再賣 $19M，2025 套現 $80M）；資本配置激進回購（2025 買 $2.58B = 65% FCF）。潛在失職：併購分心（Wurl CTV 整合失敗）、或在短報告壓力下過度回購耗損投資彈性。
  • 價值陷阱路徑（四選一）：護城河侵蝕型。若 SEC 認定 AXON 跨 app 資料提取違規且強制修改歸因架構，AUC 回退 → ROAS 下滑 → 中小 dev 流失 → 規模飛輪反轉。此為主要下行劇本。
