存在意義

ASML 解決的是「人類能否延續摩爾定律，繼續在更小尺度上製造更密集的運算」這個不可逆的問題。EUV 微影是 3nm 以下節點唯一能量產的技術——沒有 ASML，就沒有 TSMC N3/N2/A14/A10、Samsung 2nm GAA、Intel 18A/14A、Samsung/SK Hynix HBM4/HBM5。若 ASML 明天消失，2026 全球 AI 訓練晶片、iPhone 19 晶片、車用 SiC 驅動 IC、所有先進 GPU/TPU 全部停擺 2-3 年（對手無替代技術時間線）。從第一性原理看，ASML 等於「人類算力增長的物理 bottleneck 上游 gatekeeper」，這類位置整個資本主義史上僅有少數幾家（Standard Oil、IBM SystemZ、de Beers 鑽石）。

逆向推演（Munger Inversion）

  • 地緣政治曝險：製造集中於荷蘭 Veldhoven，Zeiss 德國獨供光學。真正風險不是生產中斷（政治不確定但荷蘭 + 德國皆為北約），而是 MATCH Act 通過後美國對 ASML 施壓限制 DUV 對中。影響區間：2026 中國占比從 20% 可能再降至 5–10%，但非 0（ASML 仍可對 non-SMIC 客戶出貨）。
  • 技術替代風險：Canon nanoimprint（NIL）已在 Kioxia 5nm 量產，成本 −40%；但 NIL 在 logic 高效能節點的 overlay / defect 仍落後 EUV 兩代，TSMC 已明確排除其在 A14 以下節點使用。時間軸：5–8 年後可能分食 < 5% TAM（主要是成熟 node memory / low-end logic）。
  • 管理層失職：CEO Fouquet（2024 上任）、資本配置紀錄優——2023–2025 累計回購 €15B，均價約 €600–700（遠低於現價 €1,244），證明價格敏感。主要風險是 High-NA 量產期間因 yield 問題衝擊 gross margin，或激進併購破壞 core lithography 聚焦。目前無相關訊號。
  • 價值陷阱路徑：四選一 → 無陷阱特徵。產業結構週期是唯一需警惕面向——若 AI capex 2027–2028 進入消化期，成長放緩（但非結構性衰退，是週期性）。

§9 護城河分析

評分：10 / 10（以全球上市公司為基準，採最嚴格標準）

護城河來源

  • 無形資產（專利 + Know-how）：ASML 在 EUV 光源、10nm 以下光學、overlay 控制、光罩 defect 修正的累計專利超過 16,000 件；Zeiss / TRUMPF 獨供關鍵零組件，供應鏈僅此一家。即使中國政府 2023–2026 累計投入超 US$200B 想複製，仍未量產 28nm DUV，更遑論 EUV。
  • 轉換成本：單台 EUV 光刻機 €170M–€380M（High-NA），客戶 install 後需 6–12 個月 yield 優化，光罩、製程、設備配合形成「系統性綁定」。TSMC / Samsung / Intel 無法中途更換供應商，只能擴大或縮減採購量。
  • 規模優勢：ASML 單機售價 €170M+，若無全球 150+ 客戶攤提 R&D €4.7B（14% of Revenue）根本無法負擔。後進者（包含 Canon NIL）因客戶不敢押注，研發陷入 chicken-and-egg 死循環。
  • 網路效應（弱）：服務生態（install base 6000+ machines × service contracts）形成 switching cost，但非典型網路效應。

競爭鴻溝

「為什麼資本充足的對手仍無法跨越？」——EUV 光源（Cymer 收購）+ 光學（Zeiss 獨供）+ 20 年光刻 know-how 的三重綁定。Canon NIL 走 nanoimprint 路線，物理層完全不同，但 overlay 精度仍落後 EUV 兩代。Intel 自己（PUS photonics）、Rapidus、SMIC 等全部嘗試過，均以放棄或延期收場。行業經驗：2012–2018 年 Nikon 從 60% 市佔萎縮至 < 5%，就是因為無力跟進 EUV 研發。
