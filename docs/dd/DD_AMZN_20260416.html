§2｜序章：第一性原理 × 逆向思考

存在意義

Amazon 解決的是「摩擦成本」這個根本問題——零售摩擦（找商品、取得商品的時間與價格）、企業計算摩擦（自建資料中心的資本與時間）、以及廣告投放摩擦（找到有意願購買者）。這三個摩擦本質上都是「等待時間的金錢化」：消費者等不及、企業等不起、廣告主等不起精準投放的轉化。

若 AMZN 明天消失：全球 3 千萬家 SaaS / AI 初創公司（使用 AWS 的佔比 > 50%）會失去運算基礎，大量服務須重啟於 Azure/GCP——但兩者合計產能不足以立即吸收；美國 37.6% 的 e-commerce 訂單需要在 2 週內被 Walmart + Target + Shopify 消化，實體物流量將癱瘓；Prime Video / Music / Pharmacy / Whole Foods / Kuiper / Zoox 等橫向佈局的服務同時中斷。世界會出現至少 6-12 個月的「基礎設施脫臼」，這不是效率退步，而是不可替代的基礎層停擺。

逆向推演（Munger Inversion）：要讓這筆投資 3 年內虧 50%，路徑為何？

  • 地緣政治曝險：AMZN 本身供應鏈相對分散（印度、東南亞、墨西哥製造），但 AWS 資料中心高度集中於美國、愛爾蘭、東京、新加坡；若中美科技戰升級到禁止美系雲端出口到中國區（目前仍允許部分），AWS 中國客戶流失（<2% 營收但成長動能來源）；台海衝突會中斷 AWS 晶片供應（Trainium2 由台積電代工）。
  • 技術替代風險：（1）分散式推理——若 NVIDIA 推出企業自建 AI cluster 的 turnkey 方案，中型客戶不再需要 AWS；（2）Edge Computing——若推理能夠大幅下放到手機/AR 裝置，雲端推理需求減半；（3）DeepSeek 風暴重現——若 2027 再有開源模型顛覆 OpenAI 閉源模式，AWS Bedrock 的捆綁優勢瓦解。時間軸：24-36 個月內有可能。
  • 管理層失職：Jassy 若繼 $200B capex 後 2027 再追加 $250B 但 FCF 持續負值，會動搖市場信心；LEO 衛星（Kuiper）若無法與 Starlink 競爭，每年 $8-10B 沉沒成本；若併購再失誤（如 MGM 花 $8.5B 未見回報），資本配置評價下降。
  • 價值陷阱路徑：四選一最可能的惡化劇本是隱性資本密集型——若 AWS 成長放緩但 capex 已鎖定（長週期折舊資產），ROIC 會從 14% 掉回 2022 年的 2%，FCF 長期 $10B 以下，市場會重新定價為「資本密集的電信式公司」，FPE 從 26x 壓到 15x，導致 40% 跌幅。
