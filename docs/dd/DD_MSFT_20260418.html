§4｜序章：第一性原理 × 逆向思考

存在意義

Microsoft 解決「企業知識工作的底層作業系統」這個問題——Windows + Office + Azure + Teams 三十年累積，全球 3.5 億企業客戶的生產力流程、身份驗證、資料儲存、協作、AI 助手「一站式」完成。若 MSFT 明天消失，全球 90% 以上的大型企業生產力會立即癱瘓，從 Word / Excel 文件、Teams 視訊、Azure AD 身份驗證到 Office 365 mailbox，無一可快速替代。這是「深度系統耦合的 infrastructure lock-in」，不是可替換的 app。AI 時代 MSFT 的角色進一步擴大——Copilot 成為「企業 AI 的 default 入口」，與 Windows 一樣成為新一代 OS 層。

逆向推演（Munger Inversion）：讓這筆投資 3 年內虧損 50% 的路徑

維度情境
地緣政治MSFT 供應鏈分散（印度 / 美國 / 歐洲），台海衝突間接影響（GPU 供應），地緣風險中等
技術替代若 OpenAI 自建 AI infra 完全脫離 Azure（但 $250B 合約 + Stargate 已鎖定），或 Google Gemini + DeepMind AGI 達到無法追趕的代差（目前 Gemini 3 已與 GPT-5 同級）
管理層失職Nadella 2014 CEO 至今，資本配置紀錄優秀（Activision $69B、LinkedIn $26B、Nuance $19B 皆 accretive）。退休或繼任問題是中期風險
價值陷阱路徑隱性資本密集型——當前每年 Capex $100B+（FY26E），若 AI 工作負載未按預期貨幣化，資本支出被系統性低估維持競爭力所需成本。但 2026 Q2 財報顯示 OI margin 逆向擴張至 47%，反證 Capex 投入正在產生回報。
