§0｜序章：第一性原理 × 逆向思考

存在意義

Microsoft 解決的是「企業組織的數位基礎設施」— Windows + Office + Azure + AD 構成全球企業的作業系統、通訊、身份驗證、與雲端運算底層。若 Microsoft 明天消失：(1) 全球 3 億 M365 commercial seats 立即失去 email/Teams/SharePoint，企業溝通中斷；(2) Azure 上托管的數百萬企業應用與 AI 工作負載同時離線；(3) Active Directory / Entra ID 認證失效，絕大多數企業員工無法登入任何內部系統。這不是「效率降低」而是「組織癱瘓」。

逆向推演（Munger Inversion）

  • 地緣政治：美國本土公司，直接風險低。間接風險：中國市場收入（~3%）被切斷；歐盟反壟斷要求拆分 bundling（M365 + Teams 已被 EU 調查）。
  • 技術替代：Google Workspace 在 SMB 持續蠶食 M365 share；AWS 在 IaaS 仍領先；開源 AI（Llama/Mistral）削弱 OpenAI 壟斷 → MSFT 的 AI 溢價被稀釋。時間軸 3–5 年。
  • 管理層失職：Satya 目前無可挑剔，但 capex $97.7B 的 allocation 若錯誤（AI 數據中心過度建設），3–5 年後 write-down 風險顯著。
  • 價值陷阱路徑：最可能為「隱性資本密集型」— capex 從 $44B 翻倍至 $98B，FCF 從 $74B 降至 ~$35B，若 AI 回收期 > 5 年，MSFT 從「輕資產印鈔機」變成「資本密集型公用事業」。
