§4 序章：第一性原理 × 逆向思考

存在意義

BWXT 解決的是「核心國防能源主權」痛點。美國海軍核動力艦隊（72 艘核潛 + 11 艘核航母）100% 仰賴 BWXT 製造的反應爐 — 這個事實在過去 65 年（1960s 起）未曾改變。製造一具海軍級壓水反應爐需要：(1) 重型 forging 能力（< 5 家美國公司），(2) ASLB / DoE 機密分發授權（沒有路徑可加速），(3) 數十年累積的 specialized welding / metallurgy know-how，(4) 持續穩定 supply chain（HEU 燃料 / 受控原料）。若 BWXT 明天消失：美國海軍核艦隊 mid-life 維護中斷，新建 Virginia / Columbia / Ford 級停工 5-10 年，國家安全直接受損 — 這不是「不可替代的混亂」，是「結構性危機」。同樣 logic 延伸：Mo-99 醫用同位素北美主要供應、SMR 商用反應爐壓力容器（BWRX-300 / Rolls-Royce SMR 已下單 BWXT）、Project Pele 軍用微反應爐 — 都是高門檻 + 政府信任 + 多代累積能力的疊加。

  • SMR 商用部署延遲：若 NuScale / X-energy / Kairos 商用部署時程從 2027-2030 推延至 2032+，BWXT SMR optionality value 大幅縮水。

  • 3D 列印 / 增材製造對手追上：GE Hitachi / Rolls-Royce 自有 forging + welding 能力提升，在商業 SMR 端可繞過 BWXT。

歷史對標：Westinghouse 2000s（傳統 PWR 主導 → 韓國 / 中國本土化追上 → 2017 破產）。

② 管理維度

  • CEO Geveden 接班：Geveden 自 2017 帶領 BWXT 從 $40 至 $217，個人關係主導 DoE / Navy 高層；若 5 年內退場，繼任者重新建立信任 transition cost 大。
  • 過度收購整合風險：Kinectrics + Aerojet + PCG 連續收購 → Net Debt 從 $1.0B 升至 $2.0B（FY24 → FY25）+ Net Debt/EBITDA 從 1.6x 至 2.9x，若 PCG 整合不順 + 再追加 → balance sheet 壓力。
  • Cost-plus margin 結構天花板：海軍合約 cost-plus 限制 OM 永遠 cap'd 在 12-15%；過度依賴海軍會限制 long-term margin upside。

歷史對標：L3Harris 2020-2022（M&A 整合慢 + Defense 預算 plateau → -30%）。

③ 結構維度

  • 美國國防預算 plateau：若 FY28+ 國防預算成長放緩至 2-3%（疫後 super-cycle 結束），海軍 expansion 速度減半。
  • 監管型分拆：若 BWXT 規模 $50B+ 後遭反壟斷審查（壟斷海軍 nuclear），強制分拆 commercial vs government。
  • Trump 政策反轉風險：若 2028 政府轉向 + 核能 EO 撤銷 → 商業 SMR optionality value bust。

歷史對標：AT&T 1984 強制分拆；General Dynamics 1990s 國防 cycle 谷底 -50%。
