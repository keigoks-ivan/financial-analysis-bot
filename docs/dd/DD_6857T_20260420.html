§4 序章

Advantest（創立 1954）是全球第一大 SoC & memory ATE（Automated Test Equipment）製造商。2011 年收購 Verigy 後確立 SoC tester 雙寡占格局（vs Teradyne）。核心產品：T9000（next-gen SoC）、V93K（memory + SoC）、HBM test 平台。2026 年 AI semiconductor 測試需求爆發為主要成長驅動。
