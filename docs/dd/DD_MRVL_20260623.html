§1｜投資結論詳述

本結論基於以下完整分析得出，詳見各章節；人面對裁決見 §14。

Marvell 是 AI 資料中心兩條最硬骨幹——「連接層」（光/銅互連 DSP、112G/224G SerDes、1.6T PAM4、矽光子）與「客製運算」（hyperscaler custom XPU / ASIC）——的少數全棧供應商之一。FY26 營收 $8.195B（+42% YoY 創高），資料中心段 Q1 FY27 達 $1.833B（占 76%、+27% YoY），非 GAAP 營業利益率回到 35.0%，OCF 創高 $638.8M。生意品質一流、AI 受益純粹（🟢）、護城河 A 級。問題不在公司，在價格：股價自 4 月低點 $139.69 兩個月 +122% 至 $307.86，FY+2 Fwd PE 49.9x（FY+1 76x）居 5 年 ~90 分位、PEG 1.06，且現價已高於 44 位分析師中位目標 $240。在此價買進＝賭未來 2-3 年成長全數兌現且倍數不收斂；風險調整後 5Y IRR 僅 ~4.2%/yr（低於大盤）。對既有持倉者：thesis 完整、續抱合理，但這不是加碼價。

§2｜序章：第一性原理 × 逆向思考

存在意義

AI 算力的瓶頸早已不只是 GPU 本身，而是「把成千上萬顆加速器連成一台巨型電腦」的連接層，以及「為特定工作負載量身訂作矽晶片」的客製運算。Marvell 同時站在這兩條骨幹：它的光/銅互連 DSP 與高速 SerDes 決定了 AI cluster 內 scale-up / scale-out 的頻寬與功耗；它的 custom ASIC 設計平台讓 hyperscaler 能把自家模型的算法刻進矽。若 Marvell 明天消失，AWS Trainium、Microsoft Maia、Meta MTIA 等自研晶片計畫，以及資料中心內 800G/1.6T 光模組的 DSP 供應，會出現真實的供給缺口與設計斷層——AVGO 雖能補位，但雙供應商格局崩成單寡占，會讓 hyperscaler 的議價地位與供應安全同時惡化。這是「不可替代的混亂」，不是空泛的「提升效率」。

逆向推演（Munger Inversion）：如何 3 年內虧 50%+

  • 技術替代風險（最高）：custom ASIC 是「逐代 bakeoff」生意——每一代每一個 socket 都要重新競標。Trainium3 已示警（丟給 Alchip）。若 Marvell 連續再丟 2-3 個 top-4 socket，custom 成長能見度崩，疊加互連端被 co-packaged optics（CPO）/ 矽光子 結構性替代可插拔光模組 DSP，雙引擎同時失速。歷史對標：3661 GUC/Alchip 在先進製程 ASIC 的份額擺盪顯示「代設計」護城河的脆弱性。
  • 地緣政治曝險：晶片於 TSMC（台灣）製造，先進封裝集中；中國終端曝險相對低，但出口管制升級若波及高速互連/AI 加速器，會壓縮 TAM。曝險中等。
  • 管理層失職：壓成 1 行——出價紀律惡化（再做一筆昂貴 M&A 重演 Inphi 商譽問題）或在高位過度回購。自我毀滅決策路徑詳見 §10 資本配置計分卡。
  • 價值陷阱路徑（四選一）：盈餘品質型——GAAP ROIC 6.2% < WACC、GAAP EPS 被 $1.8B 一次性出售利益灌水。但因底層 FCF 正、TAM 擴張，這比較像「過度定價的優質股」而非典型盈餘陷阱（架構性替代分級見 QC-23：custom 端 Trn3 失手＝🔴 生態攻擊級，互連端 CPO＝🟡-🔴 待觀察）。

護城河如何累積而成，見 §7.D 護城河變遷時間軸。

介於「grow with 客戶」（互連端，隨客戶 AI cluster 擴張）與「客戶可 second-source / 自研」（custom 端，Trn3 已驗證）之間。互連端良性、custom 端高風險。

④ 留存經濟

半導體口徑：以「最大平台 design win 留存 + attach rate」衡量。互連 DSP 的 design win 留存高（速率世代切換成本高）；custom 的「平台留存」被 Trn3 打破一次。留存 3 年趨勢：互連穩、custom 出現一次反轉 → 客戶結構風險 🔴（高集中 + dual-track + 已現 second-source 動作）。

§9｜產業格局

產業態勢三軸裁決（QC-39，雙向掃描）

本標的所處產業態勢 = 雙向拉鋸（競爭惡化中 × 結構需求轉好中），疊加一個未定價的法規尾部。

  • 軸 A 競爭（惡化）：custom 端在最大客戶 AWS 連兩代失單（Trn3/4 → Alchip），compute ASIC 份額估 12%→8%（2027）；互連端 Broadcom Taurus BCM83640（3nm 1.6T DSP，2026-03-11）正面進逼 MRVL ~70% 份額。sourced：SemiAnalysis / Tom's Hardware / Futurum。
  • 軸 B 供需 durability（轉好/強）：AI capex 2026 $660-690B、2027 ~$820B、~75% AI，RPO ~$700B backing → 需求結構性強至 2027；唯 2027+ 受電力/ROI 約束、雲收入 +15% << capex +80%，digestion 風險真實。
  • 軸 C 其他結構變數（負，QC-41 補入）：① 中國 ~38-44% 營收的出口管制尾部（高速 networking silicon 若入管制 → 互連直接受衝擊）；② NVLink Fusion 鎖進稀釋 custom ASIC 獨立性 + 削弱對開放 UALink 陣營客戶競爭力；③ CPO/矽光子 2028+ 對可插拔 DSP 的結構替代。

淨裁決：需求面結構順風（利多 H3/互連量）被「競爭惡化（custom 份額）＋ 法規尾部（中國）＋ 生態鎖進」抵銷 → moat_trend → holding（非 ↑）；§11.5 Bear 機率維持 30%（不因「產業在缺」壓低，因競爭/法規逆風同向）。

市場量化 + 供需 durability 裁決

AI 資料中心 TAM（custom 運算 + 高速互連 + switching）：CY2025 ~$55-60B → CY2028 ~$94B（公司框架）→ CAGR ~20%+；當前供需＝結構性緊缺（AI 算力需求 > 供給）。
