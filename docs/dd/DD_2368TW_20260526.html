§4 序章：第一性原理 × 逆向思考

存在意義

2368 解決的痛點：AI 伺服器（NVIDIA HGX / GB200 / Rubin、CSP 自研 ASIC 板）需要極端規格的高層數高銅量 PCB（typically 20+ layers + 高頻訊號完整性 + 高功率分配能力）。當 NVIDIA Blackwell B200 / Rubin R200 等 GPU 從 1.5kW 升至 2.5-3kW，PCB 需要支援更高的電流密度與更精密的訊號 routing — 這是 commodity PCB 廠商做不到的技術升級段。

若 2368 明天消失：(1) NVIDIA GB200 / B100 / B200 量產的 PCB 供應鏈缺一個重要 OEM partner；(2) Amazon ASIC UBB 大客戶失去高市占 supplier，需要 12-18 個月 qualify 替代源；(3) AI server OEM 廠（鴻海、廣達、緯創）對高階 PCB 採購需轉至 Ibiden / Unimicron / 中國廠，但這些短期 capacity 已滿。

不可替代性層級：「AI server 高層 PCB capacity + NVIDIA 認證 + Thailand 海外擴產規避地緣風險」三位一體是 2368 真正的存在意義。

逆向推演（Munger Inversion）— 三維展開

① 技術維度

  • NVIDIA 採購策略改變：若 NVIDIA 把更多 PCB 訂單轉至 Ibiden / Unimicron 多元化，2368 share 下滑 → 護城河 -1 分。歷史對標：2015-2017 Apple 把 PCB 訂單從 Foxconn 轉至 Pegatron + Wistron 多元化 → Foxconn AAPL share 下滑 -20%
  • 陶瓷基板取代有機 PCB：若 advanced packaging 改用陶瓷 substrate 取代 PCB（如 CoWoS 後段）→ 2368 業務段萎縮。但這條路徑機率 < 10%（5Y 內），陶瓷 substrate 成本仍高於 PCB 5-10x
  • EDA + AI 自動化 PCB 設計：Cadence Allegro AI 設計輔助讓更多廠商能設計高層 PCB → 競爭加劇。但 PCB 製造能力（yield、銅量處理）為實體工藝瓶頸，不易被 AI 自動化
  • 中國 PCB 技術追趕：滬電股份 WUS、深南電路 SCC 在 advanced PCB 段技術追趕 → 取代 2368 在中階段位置。機率 30%（5Y 內），但 NVIDIA / Amazon CSP 客戶因地緣對中國 vendor 依賴度降低 → 利好台灣 PCB 廠

② 管理維度

  • Capex 失控：FY25 Capex 6.8B + FY26 計畫 7.2B + Thailand 1.3B = 累積 > 15B over 24 個月。若 NVIDIA Rubin 量產推遲，Capacity 用不滿 → fixed cost 沈重。歷史對標：2018 SanDisk over-investment in 3D NAND fab → 後續 5 年 ROIC 從 25% 跌至 10%。Watchlist：FY27 H1 Thailand 廠 yield ramp
  • 單一管理層風險：金像電由創辦家族經營，CEO 經驗豐富但 succession 不明確（公開資料缺）。若創辦人退場且交棒不順 → 戰略連續性風險
  • 客戶集中：雖然多客戶（NVIDIA + Amazon + ASIC），但 NVIDIA + Amazon 合計可能 > 50%，dual-track 風險存在

③ 結構維度

  • AI capex 結構性反轉：若 2027-2028 hyperscaler 因投資回報壓力降速 AI capex → 整個 AI server PCB segment 萎縮。但這條 5Y 機率 < 25%
  • 美中貿易戰升級 → 雙重影響：(i) 利好 — 美國 hyperscaler 加速去中國 PCB 供應；(ii) 利空 — 若中國對台製品反制，台灣 PCB 廠出口受限。Thailand 廠為地緣分散化措施，部分對沖
  • 計算範式轉換：量子 / neuromorphic / photonics 2030+ 進入商用 → 傳統 PCB 段需求結構性下移。機率 < 15%（10 年內）
