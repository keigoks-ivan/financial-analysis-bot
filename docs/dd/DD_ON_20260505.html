若 onsemi 明天消失：① 全球 EV 產線 ~25% SiC 模組需求斷供，需 STM / Infineon / 中國 BYD 半導體吃下，量產良率與汽車級認證需 18-24 個月補上；② AI 資料中心 PSU + 電源樹數位化（48V → CPU rail）關鍵元件斷供，hyperscalers 設計需重做；③ 自動車 image sensor 全球 #1（CIS for ADAS）斷層，BMW / Tesla / Hyundai L2-L3 系統需 partner shift。混亂期 12-18 個月。

逆向推演（Munger Inversion 三維）

① 技術維度：什麼技術替代讓 ON 3 年內失去 50% 價值？

  1. GaN（氮化鎵）取代 SiC 在 800V 以下 EV 電源：Navitas、Power Integrations 已在小功率 GaN 商業化。若 800V EV charger 從 SiC 切到 GaN（成本 -30%），ON 失去 SiC 50% 應用空間。歷史對標：MOSFET 替代 BJT（1980s）、IGBT 替代 MOSFET（1990s）— 都是 10-15 年漸進取代，但 EV 加速度可能 5-7 年。
  2. STMicro 與 Tesla 簽 LTSA 鎖定全球最大 EV 廠商：Tesla 4680 + Cybertruck SiC 完全鎖 STM，ON 在最大 EV OEM 失去 design win，市佔從 25% 跌至 18%。
  3. Wolfspeed 200mm 量產 + 中國本土廠（如時代電氣、士蘭微）追趕：產能溢出導致 SiC ASP -20-30%，毛利率從 SiC 預期 50%+ 跌至 35%。
  4. AI DC 電源樹商品化：MPS、TI、Infineon 同步衝刺 server power，ON 沒能 take share，AI DC ARR 停在 $400-500M（vs 預期 $1B+）。

② 管理維度：什麼決策路徑會自我毀滅？

  1. Capex 慣性錯置：CEO Hassane 退出 13 條低毛利產品線後，Capex 仍在 9-19% Rev 高位（vs 同業 5-8%）。若 SiC LTSA 持續取消但 Capex 已 sunk → 折舊壓力把 GM 鎖在 35-38%。歷史對標：Sun Microsystems 2001-2008 在 SPARC 投入過深，最終被 Oracle 收購（-60%）。
  2. 創辦人退場 / 戰略 reset：Hassane 是 2021 戰略聚焦的關鍵推手，若 2027-2028 退休，繼任者重新追求營收（接 low-margin 業務）會稀釋 GM 結構。

③ 結構維度：什麼產業變化會讓商業模式過時？

  1. Auto OEM 直接收購 SiC fab：Tesla 已自建 4680 電池廠先例。若 BYD / VW / GM 直接收購 STM / ON 的 SiC 部門（vertical integration），merchant SiC 商業模式消失。歷史對標：Apple 2020 自研 M1 silicon（-70% Intel iPad/Mac 收入）。
  2. 監管型分拆：DOJ / EU 反壟斷把 SiC 五大廠（STM、Infineon、ON、Wolfspeed、ROHM）強制 portfolio 分割，商業模式重組。可能性低但非零。
  3. EV 成長 cap 在 30% 滲透率：若全球 EV 滲透率不過 30%（vs IEA 樂觀 50%+），SiC TAM 砍半，所有 SiC 玩家 over-capacity。

價值陷阱路徑判定

四選一中最可能：盈餘品質型 + 動能末段過熱（複合）。理由：FY25 GAAP EPS $0.29 vs Non-GAAP $2.35（差距 8x，含重組費 $700M+） — 市場用 Non-GAAP forward EPS 在現價 25x 定價已 price-in 完整復甦；同時 90% YTD rally + BB 上軌外，動能訊號明顯。非結構性產業衰退（cycle 週期，TAM 仍擴張）、非隱性資本密集（Capex 已 ramping down）。
