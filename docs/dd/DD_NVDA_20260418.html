§4 ｜序章：第一性原理 × 逆向思考

存在意義

剝離財務表象，NVIDIA 解決的痛點是：讓「大規模平行運算」從科學實驗室走向所有行業的實用工具。CPU 為「序列運算」設計，但人類最有價值的智慧問題——蛋白質折疊、自駕決策、語言理解、影像生成——本質都是平行。CUDA + NVLink + 自研 Spectrum-X + NVSwitch + 軟體生態讓 GPU 成為「所有平行計算的預設基礎設施」。若 NVIDIA 明天消失，全球 AI 模型訓練進度會被拉開 3-5 年——這不是速度問題，是整個研究週期被打斷。替代者（AMD + TPU + Trainium）能接手部分推論市場，但在前沿訓練 cluster（≥ 10,000 GPU 規模）實際已無等效替代，Rubin NVL72 推出後差距進一步拉開。

逆向推演（Munger Inversion）

  • 地緣政治曝險：所有先進 GPU 由台積電（TSM）3nm / 2nm 代工，CoWoS 封裝也是 TSM 產能瓶頸。若台海衝突，NVDA 產能將歸零 6-18 個月。次要曝險：中國市場（H20 禁令後 China 營收 < 5%，曝險已大幅降低）。
  • 技術替代風險：（1）Google TPU v6/v7、AWS Trainium 3、Meta MTIA 等 hyperscaler 自研 ASIC；（2）AMD MI400 / MI450 系列（2nm、432GB HBM4e）技術差距縮小；（3）若大語言模型架構出現「不適合 GPU 的」重大變形（純 MoE、neuromorphic），CUDA 護城河的相關性下滑。時間軸：ASIC 短期 1-2 年取代部分 inference，長期 3-5 年威脅訓練；架構典範轉移難以預估。
  • 管理層失職：近乎不可能。Jensen Huang 持股 ~3.5%、資本配置紀律極佳（FY26 回購 ~$30B，基本都在股價相對低位執行）。最大風險是繼任問題（Jensen 62 歲）。
  • 價值陷阱路徑：最相關是「產業結構衰退」——不是 AI 退潮，而是「通用 GPU 相對 ASIC」的地位結構性下滑。惡化劇本：hyperscaler 自研 ASIC 替換率從 15%（2025）升至 40%（2028），同時 PyTorch XLA / JAX 降低 CUDA 鎖定，導致 NVDA 錢包份額從 80%+ 降至 55%。
