§4 序章：第一性原理 × 逆向思考

存在意義

Sandisk 解決的是「數位資料的低成本、高密度、低延遲非揮發儲存」這個自半導體誕生以來的核心問題。NAND flash 儲存了人類每天產生的數位資料 — 從 iPhone 照片、Netflix 串流、企業資料庫、到 AI 訓練資料集和 inference 模型權重。若 SNDK 明天消失，全球 NAND 供給瞬間損失 ~12.4%，但會立即被 Samsung（32%）+ Kioxia（15%）+ SK Hynix（19%）+ Micron（10%）填補（價格短期上行 30-50%、6-12 個月修復）。SNDK 並非「不可替代」，而是「商品供應商之一」。這是與 GOOGL / NVDA 等 quality compounder 的本質差異。

逆向推演（Munger Inversion）三維展開

① 技術維度：什麼技術替代會讓 SNDK 3 年內失去 50% 價值？

  • 新一代記憶體技術徹底取代 NAND（時間軸：2028-2032）：若 PCM (3D XPoint successor)、ReRAM、MRAM 等 SCM 技術突破成本曲線，NAND 可能被部分蠶食。歷史對標：HDD → SSD（2010-2025），Seagate / WD 的 HDD 業務市值在 15 年間 −50% 而 SSD vendor 上升。SNDK 反制：HBF 為 NAND 適應 AI inference 的延伸，但若 SCM 突破 cost-per-bit barrier，HBF 也未必能守。
  • HBM4/5 將 inference workload 完全鎖入 DRAM（時間軸：2027-2030）：若 HBM 容量翻倍至 1TB/stack 且成本下降 50%，NAND 在 inference 角色被擠壓。歷史對標：tape → HDD（1990s-2000s）。SNDK 反制：HBF 標準制定中，但市場接受度未定。
  • YMTC（中國長江存儲）解除設備限制（時間軸：2027-2030）：若美國放寬出口管制，YMTC 月產能從目前 ~30K wafer 擴至 100K+，全球 NAND 供給 +15-20%，ASP 結構性下行。歷史對標：太陽能板 2010-2015 中國產能擴張致 ASP -80%。SNDK 無法反制（地緣政治外生）。
  • Cloud-native 儲存抽象化使 NAND 商品化加深（時間軸：2026+）：AWS / GCP / Azure 進一步抽象化儲存層，hyperscaler 議價力進一步上升。SNDK 反制：5 年 take-or-pay 合約已對抗一輪，但 contract renewal 後價格戰可能加劇。

② 管理維度：什麼決策路徑會自我毀滅？

  • Capex 慣性 cycle 末期擴產：典型商品週期錯誤 — cycle peak 時加碼擴產，2027H2 cycle 反轉後 utilization 跌至 70%、固定成本被攤薄 → 大幅虧損。歷史對標：Samsung 2017-2019 NAND 過度擴產，導致 2019 NAND 部門虧損。SNDK 防禦：當前 Capex/Rev 4%（極低），管理層紀律強，但 FY27 計劃 Capex 仍是觀察重點。
  • HBF 開發失敗 / 標準被 SK Hynix 主導：若 SNDK 在 HBF 標準制定中失去發言權，HBF 變成 SK Hynix-led 而 SNDK 為 follower。歷史對標：HBM 早期 SK Hynix 主導 vs Samsung 後進。SNDK 防禦：與 SK Hynix 共同創立 OCP workstream，但長期主導權不確定。
  • 過度回購 cycle peak 股票：$6B 回購授權，若管理層在 $1,000+ 大量執行，cycle 反轉後股價 $400 區間時相對股東是價值毀滅。歷史對標：Boeing 2018-2019 cycle peak 回購 $43B，2020 後價值崩塌。SNDK 防禦：管理層需公開承諾「cycle-aware buyback」，但目前未明示。

③ 結構維度：什麼產業變化會讓商業模式過時？

  • 儲存從硬體商品轉為 cloud service 訂閱：如果 hyperscaler 進一步把 storage 作為 utility 對外賣，NAND vendor 變成 component supplier 而非 solution provider。歷史對標：x86 commodity → AWS Cloud（2010-2020），Intel 失去 server 定價權 60%。SNDK 反制：Stargate platform 部分模糊邊界，但長期商業模式風險仍在。
  • 監管與供應鏈分裂：US-China 半導體出口管制持續升級，若進一步限制 SNDK 在中國銷售（中國占 ~25% NAND TAM），收入結構性受傷。歷史對標：Huawei 在美國出口管制下手機業務蒸發。SNDK 部分曝險。
  • 計算 paradigm 轉換降低資料儲存需求（極端，2030+）：若邊緣 AI / 量子計算降低資料中心儲存需求增長率，NAND TAM 可能進入結構性低成長。但機率低（資料量持續爆炸是長期 trend）。
