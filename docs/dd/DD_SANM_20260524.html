§4 序章：第一性原理 × 逆向思考

存在意義

Sanmina 解決的是「複雜電子產品（伺服器、通訊設備、醫療器材、國防系統）從設計到量產的供應鏈整合」問題。EMS（Electronics Manufacturing Services）這個產業之所以不可替代，是因為： （1）品牌商（OEM）若自建工廠需鎖定數十億美元 capex 且承擔週期風險，反而失去靈活度； （2）大規模量產所需的供應鏈關係、製程 know-how、地理分散 footprint 需要 30+ 年累積； （3）某些產業（醫療、國防、通訊）需要嚴格認證（FDA、ISO、ITAR），不是任何工廠都能接單。

SANM 自 1980 年（Jure Sola 創立）起在 PCB 與 backplane 製造站穩，2000 年代併購 SCI Systems（變成 Sanmina-SCI），擁有少數 EMS 廠中同時 own PCB（最上游）+ enclosures + final assembly 的垂直整合。2025 年 10 月以 .62B 從 AMD 取得 ZT Systems，加入 hyperscaler 級 AI rack-scale 整機能力，將事業組合從「commodity EMS」躍升至「AI infrastructure provider」。若 SANM 明天消失，全球至少 10-20 家中型 OEM（如 Cisco、Juniper、Ciena）需要 12-18 個月重新分配製造產能，期間造成嚴重供應鏈混亂。

逆向推演（Munger Inversion）三維展開

① 技術維度：什麼技術替代會讓 SANM 3 年內失去 50% 價值？

  • Hyperscaler 自建 rack assembly（最高機率）：AWS、Google、Meta 過去 5 年已建立內部 server design 團隊；若 Microsoft / Oracle 跟進建立 ZT-like 內部能力，ZT 業務基礎瓦解。歷史對標：Apple 2020 年自研 M-series 切斷 Intel 訂單，Intel 一年內失去 4B 業務。
  • Liquid cooling / immersion cooling 規範化（中機率）：若 cooling 從各廠商 customize 變成「open standard」（如 OCP 規範），SANM 在熱管理上的 IP 失去定價權。
  • ARM 或 RISC-V 改變 server 架構（中低機率）：若 ARM-based / RISC-V server 滲透率 5 年內超 30%，目前 x86-based ZT 業務需大規模 retool。
  • 3D advanced packaging 改變 PCB 需求（低機率）：若 chiplet + 2.5D/3D 封裝大幅取代 PCB 上的 high-speed signal routing，SANM 上游 PCB 業務需求結構性下滑。歷史對標：Sun Microsystems 1990s 押注 SPARC server，x86 標準化後 5 年消失。

② 管理維度：什麼決策路徑會自我毀滅？

  • Capex 慣性：管理層為追求 FY27 .7B run rate，可能在 2026-27 H2 投入 0B+ capex 擴產；若 AI capex 在 2028 進入消化期，SANM 將承擔產能閒置（FCF margin 從 5%壓至 2%）。歷史對標：Cisco 2000 年代過度擴張光通訊產能，2002 年認列 .5B 庫存減損。
  • 創辦人退場：Jure Sola 1949 年生，2026 年 77 歲；若 5 年內退場，無強勢繼任者可能導致戰略漂移。歷史對標：Solectron 2000 年代失去創辦人後逐步被 Flextronics 併購消失。
  • 整合失敗：ZT 文化（hyperscaler hyper-iteration）vs SANM 文化（傳統 EMS process discipline）若衝突，可能在 18 個月內喪失 ZT 核心 talent + 客戶信任。歷史對標：HP 收購 Compaq（2002）後 3 年才完成文化整合，期間錯失幾個關鍵 server 世代。

③ 結構維度：什麼產業變化會讓商業模式過時？

  • AI server 商品化加速：若 NVIDIA 推出 reference rack design + open-source firmware，hyperscaler 可直接購 GPU + 規格化 chassis，跳過 ZT 這層 customization。歷史對標：iPhone 出現後 Nokia / BlackBerry custom firmware 失去意義，5 年內消失。
  • 監管型分拆：若 US 政府要求關鍵基礎設施製造完全 onshore（包括 PCB 上游），SANM 在中國（蘇州）、馬來西亞、墨西哥的產能需重新配置；轉移成本可吞噬 3-5 年 FCF。歷史對標：AT&T 1984 年強制分拆案。
  • 計算範式轉換：若量子計算或 neuromorphic chip 在 5-7 年內商用，整個 GPU rack 範式被替代。歷史對標：Intel 2010 年代押注 mobile chip 失敗、被 ARM 主導。

價值陷阱路徑判斷：若這是一個價值陷阱，最可能是「盈餘品質型 + 隱性資本密集型」雙混合 — ZT 整合期 GAAP 與 Non-GAAP 差距大（一次性整合費用），若 Non-GAAP OpMargin 兩季 < 6.0%，市場會質疑 organic 是否撐得起當前 18.9x PE。

護城河來源（v12.3：強化 sourced evidence）

  • 網路效應：基本不具備（EMS 是 B2B 1:1 關係，無平台效應）。
  • 無形資產：弱。SANM 持有約 ~1,500 件專利（多為 PCB / process / thermal management），但非結構性壟斷；醫療、國防認證（FDA / ISO 13485 / ITAR / AS9100）具備一定壁壘但非 SANM 獨有。

競爭鴻溝（v12.3：具體 mechanism）

為什麼資本充足的對手仍無法跨越？三個維度：

  1. 時間壁壘：建立 SANM-equivalent vertical integration（PCB → modules → rack）需要 10-15 年；單純收購無法快速 replicate 因為 PCB 業務在 EMS 圈是 capital-intensive 且 talent-scarce。
  2. 資本壁壘：複製 SANM 80 個 site 的 global footprint 需要 -15B 累積 capex；近年無人有此意願（甚至 Flex / JBL 也在剝離低 ROI footprint）。

§11 產業格局

市場量化

  • EMS TAM：當前 ~$650B（2026），5Y CAGR ~8%（到 2030 達 ~$950B）。當前滲透率：占全球電子製造 ~30%，剩 70% 仍由 OEM 自建工廠（仍有外包空間）。

  • 供需狀態：AI infra 段緊缺（HBM、CoWoS、power module 供應限制）；EMS 一般段平衡偏寬鬆（疫情後 inventory 已正常化）。

議價權（v12.3：三段獨立）

①對上游供應商

供應商集中度高且替代難度 high。SANM 採購集中於 PCB raw material（銅箔積層板，主要從台灣建滔、南亞、Iteq）、connectors（Amphenol、TE Connectivity 寡占）、半導體（Intel、AMD、NVIDIA — 客戶兼供應商雙重身份）。近 3 年議價案例：2022-2023 supply chain shortage 期間被迫接受成本上漲（gross margin 暫壓 50-100bps），2024 後逐步恢復；對 connector 供應商持續無顯著 leverage。Top 3 供應商占採購估 > 60%，切換成本高（認證 + qualification 6-12 個月）。

②對下游客戶
