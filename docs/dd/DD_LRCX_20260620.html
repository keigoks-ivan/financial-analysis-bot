§4｜序章：第一性原理 × 逆向思考

存在意義

半導體製造是「在矽晶圓上反覆沉積薄膜、再精確蝕刻出三維結構」的數百道循環。隨著電晶體從平面走向 3D（FinFET → GAA、3D NAND 堆疊數百層、backside power、HBM 堆疊），蝕刻（把材料精確刻掉）與沉積（把材料精確鋪上）的步驟數隨結構複雜度指數上升。LRCX 是這兩個領域的全球領導者之一——尤其在「高深寬比蝕刻」（3D NAND 要在數百層中刻出又深又直的孔）與原子層沉積上，技術門檻極高。

若 LRCX 明天消失，世界會出現的不可替代混亂：3D NAND 與先進 DRAM（HBM）的製造會嚴重受阻——LRCX 在記憶體蝕刻/沉積的市佔極高，記憶體廠（Micron/Samsung/SK海力士）的產能擴張會卡關。先進邏輯的 3D 結構良率爬坡也會延後。這是「先進記憶體與 3D 邏輯能否量產」的關鍵設備供應。護城河如何累積而成，見 §9.D 時間軸。

逆向推演（Munger Inversion）

若要讓這筆投資在 3 年內虧損 50% 以上，最可能的路徑：

  • 記憶體週期反轉 + 估值倍數回歸（最高機率路徑）：記憶體（~1/3 營收）是半導體最週期的子段。當前 49x 是建立在 HBM/DRAM 週期高點的 peak earnings 上。若 2027-28 記憶體供過於求（廠商擴產後），EPS 下修 + 倍數從 49x 殺回 23x（5Y 中位）→ 雙殺 −50%+。FY24 LRCX 營收即 −14%（上一輪 memory 下行），證明這條路徑真實存在。
  • 技術替代風險：蝕刻/沉積的替代風險低（製程必需），但份額被 AMAT（沉積）/ TEL（蝕刻）侵蝕的競爭風險中等（非壟斷）。⛔ 架構替代分級見 QC-23。
  • 管理層失職：機率低。LRCX 資本配置紀律佳（§12 計分卡 A）；風險是高位回購（FY26 在 49x 高位回購需觀察）。
  • 價值陷阱路徑（四選一）：最可能是 產業結構衰退型的「記憶體週期」變體——不是 TAM 永久萎縮，而是週期高點誤判為結構成長。49x 把週期峰值當常態，這是週期股最危險的買法。
