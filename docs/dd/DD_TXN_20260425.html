§4｜序章：第一性原理 × 逆向思考

存在意義

Texas Instruments（TXN）是全球 analog / embedded processing 半導體 Big 3 之一（與 ADI、Infineon 並列），解決「把實體世界（電壓、電流、溫度、運動）數位化」這個結構性、不可逆的問題。每一台車、每一台工業機械、每一台 AI server、每一個 IoT device 都需要 TXN 的 power management IC + signal chain + microcontroller 才能運作。當 AI server 用電從 30kW 升至 100kW+，當電動車取代燃油車（每車 $750+ analog content vs $300 燃油車），當工業 4.0 推動 sensor 密度倍增，TXN 是核心 enabler。

具體不可替代性：① 80,000+ products SKU（Big 3 中最廣泛）+ 100,000+ 客戶分散（最大客戶 < 5% revenue）= 任一 product 流失影響微小但替換 TXN 的客戶要 redesign 數百個 SKU 成本極高；② Sherman TX SM1/SM2 + Lehi UT LFAB2 三座 300mm 自有 fab = 規模 + 內製整合度；③ 22 年連續調漲股息 + ROIC 22% 證明資本配置紀律；④ Silicon Labs $7.5B 收購（2026-02 公告，4/30 股東投票）擴大 wireless microcontroller 至 $11B SAM。

逆向推演（Munger Inversion）

若要讓這筆投資在 3 年內虧損 50%+，最可能的路徑是什麼？

  • 地緣政治曝險：中國 ~25% revenue + 部分 fab 依賴日本 / 歐洲後段。但 TXN 已 in-source 大部分後段且 Sherman/Lehi fab 是美國本土，地緣政治 risk: 低-中。
  • 技術替代風險：① RISC-V open source MCU取代 TI MSP430 series（中國 + 新創進攻 microcontroller 低端）；② SiC / GaN 取代 Si power IC（TXN 已布局但落後 ON / Infineon）；③ analog 內製化（NVDA / 蘋果自設計 power management）。技術替代 risk: 中，2028+ 須監測。
  • 管理層失職：CEO Haviv Ilan（2023 上任）執行力強，FY24-25 連續 beat-and-raise；CFO Rafael Lizardi（2017 上任）保守。資本配置：FY24-25 Capex 重投資期過後啟動 Silicon Labs 收購整合。激勵錯位風險低。低。
  • 價值陷阱路徑：四選一最可能 = 「隱性資本密集」變體──但屬「暫時性」非結構：FY23-25 Capex 累計 $14B 是史上最大 fab 投資週期，FCF Margin 從 30% 降至 14.7%。但這是 Sherman / Lehi 二期建置 + LFAB2 啟用，2026 後 Capex 從 $5B → $2-3B = FCF inflection。風險僅在「Capex 投資延長至 2028+ 而非 2026 完成」。

50% 虧損的最高機率組合：① 2026 H2 Industrial 增速從 +30% 退至 +10%（cycle 假復甦）；② 同期 Auto / 中國連續弱；③ FY+1 PE 從 36x reset 至 22x（cycle peak 後合理）；④ Silicon Labs 整合超預期延後 1 年。三者 / 四者同時發生的機率約 15-20%。
