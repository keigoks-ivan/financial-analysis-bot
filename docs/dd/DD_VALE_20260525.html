§4 序章：第一性原理 × 逆向思考

4.1 存在意義

Vale 是全球第二大鐵礦商（FY25 328 Mt vs Rio 305 Mt vs BHP 290 Mt），巴西第一大企業，解決「Brazil 提供全球最高品位的低碳鐵礦」這個剛性需求。Carajas 65% Fe 高品位是歐洲 + 日本 + 韓國 + 中國高端鋼鐵的不可替代來源 — 因為 hydropower 煉鐵的世界趨勢需要高品位 fines 才能達到 ESG 指標（特別是歐洲 CBAM 與 H-DRI 煉鋼路徑）。

Vale 的獨家性在於：(1) Carajas Complex 65%+ Fe 全球最高品位（vs BHP WAIO 主流 62% + Rio Pilbara 主流 62%；唯一可比者是 Rio Simandou 65% 但 ramp 慢）；(2) Vale Base Metals 是巴西唯一垂直整合銅 + 鎳業務（Salobo + Sossego + Onça Puma + Sudbury + Voisey's Bay）；(3) 巴西 mining 國家代表 — 過去 5 年 Mariana + Brumadinho 兩大 dam 災難後管理層 + 安全體系大幅重組。

若 Vale 明天消失：(1) 全球高品位 fines 供給少 ~50%（Carajas + S11D），歐洲 / 日韓 ESG 鋼鐵轉型直接停擺 2-3 年；(2) 全球鎳供給少 ~10%（Sudbury + Voisey + Onça Puma + PTVI 33.9%），EV 三元電池供應鏈受衝擊；(3) 全球銅供給少 ~2-3%（Salobo + Sossego + Sudbury）；(4) 巴西 GDP 直接 -1.5pp + Pará / Minas Gerais 州經濟崩潰。

4.3 逆向推演（Munger Inversion）— 三維

① 技術維度

  • H-DRI 氫直接還原煉鋼：Carajas 65% Fe 高品位 fines 接近 DR-grade（67%+）要求 — 反而受益於 H-DRI 趨勢，成為歐洲 / 日韓 ESG 鋼鐵的 specialty 供應商。歷史對標：1990s ESG 標準推出後 high-quality fines 享有 premium 持續至今。10Y 內機率：高（70%+），是 Vale 唯一結構性利好之一。
  • 銅 / 鎳 EV 需求變化：EV 鎳需求若被 LFP 電池進一步替代 → Vale 鎳業務減值風險。CATL 2024 sodium-ion + LFP 占比已從 2020 的 35% 升至 2025 的 65%。Vale 鎳業務面臨結構性逆風。歷史對標：1990s 鎘鋅電池被鋰電池替代後 — 部分礦山關閉。10Y 機率：中-高（40-50%）。
  • 巴西 H-DRI 內部需求：巴西自身鋼鐵業 ArcelorMittal Brasil + Usiminas 可能加碼 H-DRI → Vale 國內鐵礦需求結構性下滑 - 機率低 (15%)。
  • 銅替代：Salobo 銅業務 ~10% 占比，銅替代風險與 BHP/Rio 同等量級但占比較小。

② 管理維度

  • Brumadinho 法律餘波持續：2019-01 dam 災難 270 人死亡 → 巴西法律追訴持續至今 + 安全升級壓力。歷史對標：BP Deepwater Horizon (2010) 持續 10+ 年法律餘波 + $60B 累計賠償。Vale 已累計 ~$23B 賠償 + 安全升級，可能再 $10B 未來 10 年。
  • CEO 異動歷史：Murilo Ferreira 2011-2017 → Fabio Schvartsman 2017-2019 (Brumadinho 後辭職) → Eduardo Bartolomeo 2019-2024 (turnaround period) → Gustavo Pimenta 2024-08 (現任，財務背景)。新 CEO 風險：可能策略轉變（如加速 VBM 分立 / 賣鎳業務）。
  • Vale Base Metals 分立計畫：2023 重組 + Engine No.1 + Saudi PIF $3.4B 入股 13% → 2027-2028 預期 IPO unlock $25-30B value。風險：IPO 推遲 / 估值不達預期 / 母公司 derate。
  • 巴西工會 + ESG 壓力：巴西 mining 工會強，安全 + 工會壓力結構性高於 BHP 澳洲。

③ 結構維度

  • 中國鋼鐵需求結構衰退：Vale 鐵礦 70% 銷往中國（vs BHP 70%, Rio 50%）— 集中度與 BHP 並列最高。若中國 2026-2030 -15-25% → Vale 鐵礦 EBITDA -30%+。歷史對標：日本 1970s 鋼鐵需求達峰後 30 年下行 -50%；韓國 2010s -20%。中國規模 4x 日本但下行軌跡可能類似。
  • Lula 政府 royalty / tax / 工會強化：Lula 2023-01 上任後巴西左翼政府持續討論 mining royalty 加碼 + 工會權利強化 + 環保監管加碼。2026-10 巴西總統大選是 binary event — 右翼勝選 = unlock value，Lula 繼承人勝選 = 持續壓制。歷史對標：1973 智利 Allende 政府將 Anaconda 國有化（極端 case，但 Vale 已被類似動機壓制 -27% PE discount）。
  • 地緣集中：Vale 基本上 100% 巴西運營（除加拿大鎳 ~10% + 印尼 PTVI 33.9%）— 政治集中度最高 (vs BHP / Rio 多地)。對沖少。
  • 美中商品戰：若 2028 後 commodity 戰 escalate，Vale 因主要客戶 70% 中國，可能被迫在「西方制裁 vs 中國客戶」之間選邊。Saudi PIF 持 VBM 13% 給西方戰略 hedging 但不夠。
