§4 序章：第一性原理 × 逆向思考

4.1 存在意義

Vale 是全球第二大鐵礦商，巴西第一大企業，解決「Brazil 提供全球最高品位的低碳鐵礦」這個剛性需求。Carajas 65% Fe 高品位是歐洲 + 日本 + 韓國 + 中國高端鋼鐵的不可替代來源 — 因為 hydropower 煉鐵的世界趨勢需要高品位 fines 才能達到 ESG 指標。若 Vale 明天消失，全球高品位 fines 供給少 ~50%，歐洲 / 日韓鋼廠的 ESG 轉型計畫直接破產。

4.2 護城河形成歷史

4.3 逆向推演 三維

① 技術維度

  • H-DRI 煉鋼：Carajas 高品位 fines 是 DR-grade 候選 → 反而受益於 H-DRI 趨勢
  • 銅 / 鎳 EV 需求變化：EV 鎳需求若被 LFP 電池進一步替代 → Vale 鎳業務減值風險

② 管理維度

  • Brumadinho 法律餘波：管理層持續面對訴訟、賠償、安全升級壓力
  • CEO 異動歷史：Eduardo Bartolomeo 2019-2024 任，2024-08 Gustavo Pimenta 接任 — 新人接手 turnaround
  • Vale Base Metals 分立：可能未來 IPO 或進一步分立 → unlock value

③ 結構維度

  • 中國鋼鐵需求結構衰退：Vale 鐵礦 70% 銷往中國，集中度最高，受影響最大
  • Lula 政府 royalty / tax：巴西左翼政府可能加碼 mining royalty + corporate tax
  • 地緣分裂：Vale 基本上 100% 巴西運營 — 政治集中度最高 (vs BHP / Rio 多地)
