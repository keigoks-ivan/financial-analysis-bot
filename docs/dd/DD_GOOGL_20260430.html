§4 序章：第一性原理 × 逆向思考

存在意義

Alphabet 解決的長期不可逆痛點：「人類認知與計算的全球索引層」。具體拆解三層：

  • 知識索引層（Search + YouTube）：將全球資訊（網頁、影片、商業實體）轉化為可檢索、可貨幣化的數據資產。Search 每秒處理 99,000 query，全球單日 8.5B query 量。若這個層次明天消失，全人類在「找答案」的效率會倒退 20 年（不只是搜尋慢，更包含廣告、商業推薦、內容發現整套生態）。
  • 計算基礎設施層（Cloud + TPU）：全球第三大公有雲，且是唯一「硬體 + 軟體 + 模型」全棧自研的玩家（TPU 自研晶片不依賴 NVIDIA + Gemini 自研模型 + GCP 平台）。AI 時代計算需求每 2 年翻倍，GOOGL 是少數能規模供應的玩家。
  • 智能介面層（Gemini App + AI Overviews + Workspace AI）：將 LLM 能力整合進 Search、YouTube、Gmail、Docs 數十億用戶級產品。MAU 從 2025/7 450M → 2025/10 650M（3 個月 +44%），是少數能與 ChatGPT 正面交鋒的 AI 助手。

三層交織形成的飛輪：Search query → 訓練 Gemini → Cloud 提供 inference → 廣告變現 → 投入 Capex → Cloud 規模擴張 → 更便宜的 inference → 更好的 Search → 更多 query。明天若 Alphabet 消失，全球 web 廣告（年市場 $400B）瞬間真空、近 1/3 的雲計算 workload 中斷、AI 模型訓練資料集失去最大來源（YouTube 視訊、Search 反饋）。

逆向推演（Munger Inversion）

若要讓 GOOGL 在 3 年內虧損 50% 以上，最可能路徑：

  • 地緣政治曝險：GOOGL 在中國本土無業務（Search 自 2010 退出），但 TPU 製造主要在台積電（台海緊張 = 全 hyperscaler 共同風險）；歐盟 DSA / DMA 持續加碼罰金（2024-25 已罰 €5B+）；美國對外國 ad tech 法案進展。實質曝險：中。
  • 技術替代風險：(1) 「對話式搜尋」永久取代「網頁搜尋」— 但 GOOGL 自己也在做（Gemini Overviews），更可能是內部蠶食而非外部取代；(2) 蘋果 / Anthropic / OpenAI 在 OS 層整合 — Apple Intelligence 預設搜尋夥伴若從 Google 切換給 OpenAI（年付 $20B+ 流量保護費）= 立刻 -10% 估值殺；(3) Web 結構性消失（agent-to-agent 直接互通取代瀏覽器）— 仍 5-10 年外。
  • 管理層失職：(a) 過度擴張 Capex 至 ROIC 跌破 WACC — 目前 ROIC 28% vs WACC ~9%，超額 19pp，要崩到 WACC 以下需 Capex 多砸 3 年無回報；(b) 收購整合失敗 — Wiz $32B 是史上最大併購，整合風險真實但 Wiz 本身已是雲安全領導者，整合難度低於 Mandiant；(c) 創辦人退場 — Pichai 任期穩定，無接班危機。
  • 價值陷阱路徑：四選一最可能 → 「隱性資本密集型」。FCF 表面尚可，但維持 AI 競爭力的 Capex 系統性低估（每年加碼）。若 2026 Capex $185B、2027 $230B、2028 $280B 而 Cloud margin 沒守住 30% → FCF 從 $73B 壓縮至 $50B = P/FCF 從 56x 升至 80x = 估值再評估。但 Q1 2026 Cloud margin 32.9% 已大幅反駁此情境，需後續持續觀察。
