§4 序章（公司與產品概覽）

BE Semiconductor Industries（成立 1995，Duiven 荷蘭）是全球半導體後段封裝（back-end assembly）設備領導商，產品線：Die Bonding（wire bond / flip-chip / hybrid bonding）、Packaging（molding、trim/form、singulation）、Plating（電鍍）。旗艦產品為 Datacon 8800 Ultra Plus 系列 hybrid bonder + 次世代 Chameleon。

§5 論點錨定（核心投資論點 + 反論）

核心論點（3 層結構）

  1. 結構：BESI 是 hybrid bonding 全球 60% 市占獨占玩家，AMAT 2025-04 買 9% 股份並 co-develop 整合設備，形成「ASML-Zeiss」型寡占結構（詳見 Hybrid Bonding ID 非共識 #2）。
  2. 週期：HBM4 2026-Q3 量產 → HBM5 2028 量產 → HBM6 2030+，每代都強制使用 hybrid bonding。同時 TSMC SoIC 產能爆發（6μm → 3μm pitch），AMD MI400 / Apple M5 / Intel Clearwater Forest 全採用。
  3. 公司：FY25 Q4 orders €250.4M（+105% YoY），AI 訂單占 50%；管理層紅利連續 10+ 年；GM 63.9% 歷史最高。

反論（最強 Bear Case）

若 hybrid bonding 在 2028-2030 被 glass substrate + 傳統 flip-chip 組合取代（Intel Clearwater Forest 2026-Q1 已量產 glass-core CPU），BESI 面臨存量市場收縮。但此風險發生在 2028 後，且 glass 與 hybrid bonding 部分互補（glass interposer 仍需 flip-chip 或 hybrid）。
