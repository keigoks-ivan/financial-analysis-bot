市場偵探

聚合全站偵測層的事實警報網 · 描述器非擇時——把「已經在跑、卻沒人盯著被觸發」的偵測層（市場監測異常引擎、分位翻折、跨資產擁擠交易、資產輪動象限翻轉、regime 位移、總經時鐘、財測落差）收斂成單一按嚴重度排序的訊號清單；每條訊號帶跨日持續狀態（新增／持續／升級／冷卻）。

定位：本頁是描述器（describer），不是擇時訊號（timing signal）。每條訊號只陳述機械層已經算出的事實（z-score／分位／COT／象限翻轉／快照位移），不判斷、不擇時、不給買賣指令。底層偵測層各自獨立維護、更新頻率不一（監測／輪動日更，擁擠／regime 週更，總經時鐘月頻，財測落差事件觸發），本頁只做彙整、狀態追蹤與排序；「冷卻」表示條件已消失、處於寬限觀察期。
