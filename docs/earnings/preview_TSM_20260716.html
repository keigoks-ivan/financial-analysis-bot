四、錯價判讀（implied move vs 三情境）

隱含波幅 9.6% 是上限讀法——straddle 到期日（7/17）距凍結日尚有 11 個交易日，內含非事件時間價值；對照 Q1'26 財報前市場預期的單日事件幅度約 4.4%（TipRanks 當時報導），本次純事件定價估落 5–6% 區間。近四季實際隔日反應：Q2'25 上漲、Q3'25 平盤（record 仍不漲）、Q4'25 盤前 +7%、Q1'26 −3.1%（beat 仍跌）——四季中兩季 beat 無反應或下跌，市場對「beat 本身」已鈍化。

合成判讀：市場與底稿無重大分歧，本次 preview 無方向性 edge。承保漂移 +1.0%（共識與 DD Base 幾乎重合）、共識 EPS 即 guide 中點映射——市場 price 的是 Base 至 Base 偏 Bull。可操作的不是方向而是結構：（一）事件選擇權以純事件口徑估仍偏貴，不宜買波動；（二）真正的資訊量在三個 guide 訊號（checklist #1/#3/#4），它們決定 H1/H2 假設的驗證進度，而非本季 EPS；（三）現行裁決＝進場（條件式核心），本頁不構成加減碼訊號——部位動作回 DD §14 與 PM 規則。
