訊號流程說明 — 出場的半倉怎麼買回來?(以 2026 春季 QQQ 為實例)

兩套訊號完全獨立、各自演:E3 的三個訊號自己進出(每跌破/收復一條線動 1/3), ST 閘門自己開關(受閘半倉跟著 E3 或歸零)。買回訊號就是 Supertrend 自己翻多的那一週收盤 (週收上穿被棘輪壓低的上軌)— 不需要等均線重新給進場訊號,閘門是「狀態」不是一次性停損。
實務上每週五收盤只算一個數字:目標倉位 = 0.5×E3 + 0.5×(ST開 ? E3 : 0), 與上週目標的差額就是下一個交易日要下的單。本頁頂部的倉位數字即當前目標。
決策日(週收盤)部位發生了什麼
2026-03-20100% → 67%E3 自己先出了 ⅓(W40 跌破),ST 還在多方。兩個半邊都縮:0.5×⅔ + 0.5×⅔
2026-03-2767% → 17%ST 翻空(對應 TradingView 圖上的 Sell 標籤,週收 562.58)+ E3 再掉到 ⅓。受閘半倉歸零:0.5×⅓ + 0
2026-04-0217% → 33%E3 回到 ⅔(W52 收復),ST 仍空 → 受閘半倉不跟
2026-04-1033% → 50%E3 回滿 3/3,ST 仍空。未受閘半倉滿、受閘半倉仍是 0
2026-04-1750% → 100%ST 翻多(對應 TradingView 圖上的 Buy 標籤,週收 648.85)→ 受閘半倉一次買回
誠實帳單:這次受閘半倉 562 附近出、648 買回 — V 型反轉裡閘門是付錢的(高 15% 追回)。 ST 閘門賺錢的場景是 2022 型持續陰跌(出場後不必更高買回);全期統計淨賺(MDD -26.9% → -21.9%、Calmar 0.57 → 0.64), 但個別 episode 會像這次一樣難看。OOS 重審追蹤的就是這個比例有沒有走樣 — 對照線為 E3,見 回測頁。
