問 1: 是 Base Breakout 還是 Extended?
  ├─ Base Breakout (剛突破 < 4 週)
  │    → 直接建立 1/2 倉位,剩 1/2 等首次回踩 21DMA
  │    (不要等 50DMA,base breakout 通常不回那麼深)
  │
  ├─ Continuation (突破 1–6 個月內)
  │    → 等回踩 50DMA,分批建倉
  │
  └─ Extended (距 50DMA > 20%)
       → 跳過,放觀察清單,等下次 base 重建
