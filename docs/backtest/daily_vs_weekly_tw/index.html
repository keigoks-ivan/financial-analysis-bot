最接近的兩個嘗試 — 提升日線、且不拿 DD 換

目標：提高 Calmar 但靠降變異/改善每筆品質，不靠加槓桿換 DD。這兩個是整個搜尋裡離週線最近的——但仍指向同一個結論。

綜合：能穩健提升日線的是 ① ensemble(四檔 DD 全降、Calmar 全升),但逼近不超越；③ ATR 帶能在 SPY 真過尺但不可移植。 兩者本質都是「用分散或放慢出場去重建週線」——做得好就逼近、做過頭就扛爆。週線那個免參數的點，就是效率前緣。

三條鐵律

1. 慢打敗快。週線把日線雜訊平滑掉、不被盤整洗，這是 edge 本身，不是可調選項。日線怎麼修都只能逼近、不能超越。
2. 做空股票指數結構性失敗。上漂逆勢 + 熊市暴力反彈軋空，方向濾網、盤整濾網、出場確認、時間架構、MA 組合，任何組合都救不了。
3. whipsaw 是保險費，消不掉。它買到的是「吃得到 V 反彈」；任何想濾掉它的閘門(加在週線上)濾掉的反彈比省的盤整更多，淨虧。

方法與限制

  1. 判生死的尺(matched-CAGR):過尺 = Calmar > 週線 且 同 CAGR 下 MDD 比「把週線稀釋到同 CAGR」更淺。raw Calmar 會在不同 CAGR 間誤導(軸6 的海市蜃樓),這把尺才公平。
  2. 固定軸：日線斜率 N=5、出場確認 N=2(掃過 1/2/3)、ER window 20、ADX/CHOP period 14。週線 = W52/104/250、4 週斜率、破 W52 即出(N=1)。成本單邊 7bps,閒置 SHY/BIL(美股)或現金(台股)。
  3. 方法論收穫：24 格 × 多個調過的參數裡冒一個 raw 高 0.05 的本是噪音；matched-CAGR 尺 + 跨標的複製檢驗正確擋下它。別信單一 in-sample 最佳格。
  4. 真實資料：yfinance,週線基準對齊站上長軌頁發佈值。
