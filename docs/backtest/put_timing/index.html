誠實負結果 — 沒有穩健贏家

事前擇時買 put 無法穩定地「有利」。put 是保險支出，不是能改善長期風險調整報酬的 alpha。

問題：長期持有 SPY／QQQ 之下，什麼條件買 put 有利？三輪研究、100+ 配置，答案分兩層，都不樂觀。 絕對面：56 個 Stage-1 配置的 put 本身平均損益全部為負（付 $1 權利金拿回 $0.20–$0.80），這是波動率風險溢價 VRP 的另一面，符合事前預期。 組合面：以「MDD 改善 ≥2pp 且 CAGR 代價 <1pp」的事前門檻篩，只有 2 個 QQQ 配置通過——一個是路徑運氣（最需要的三次大跌全漏接），一個只跟 B&H 打平。 Round 3 用趨勢閘門把陰跌用正 carry 避掉、只讓 put 保急崩盲區，機制驗證完全成立（9/9 急崩補上），但 18 個配置無一把 Calmar 買回 trend-only 的水準。

方法與合成定價偏差揭露（誠實框架是這頁的賣點，務必先讀）： 無免費歷史選擇權鏈，全部 put 以 Black-Scholes 歐式合成定價。隱波代理 σ_ATM = ^VIX（SPY）／^VXN（QQQ）， VIX／VXN 本身已含 skew 溢價、略高估 ATM put 成本（方向上對買方保守）。Skew 用線性 bump（每低於 S 1% 加 0.5 vol pt）， 另跑 {0.0, 1.0} 敏感度——flat-vol（skew=0）會系統性低估 OTM put 成本、讓避險看起來比實際吸引人（tw_options 稽核教訓）。 期限結構 flat（30 天 IV 當 60 天用）為已知偏差。成本：權利金 1%／邊、股票挪倉 7 bps／邊。 執行 t 日算訊號、t+1 收盤成交（無 look-ahead）；股息 DRIP、無交易的 run 可精確重現 total-return B&H。 兩 put 規格固定不調參：P1 = 5% OTM 60 天、P2 = ATM 30 天。

這在測什麼

問題與兩層判準

底倉 SPY／QQQ 長期持有（B&H、股息再投入）。從隱波水位、布林、乖離率、VRP、期限結構、價格狀態六個面向切入， 每個面向獨立測試。絕對面看訊號觸發後 put 本身的平均損益（putEV = pnl_per_prem）； 組合面看疊加後 vs 純 B&H 的 CAGR／MDD／Calmar。headline 用組合面。

已知前提：VRP 的另一面

無條件買 put 長期是負期望值——這是波動率風險溢價的另一面，事前就知道。真正的問題不是「買不買」， 而是「有沒有一個條件訊號能只在該買時買，把負 EV 翻正、或至少換到划算的 MDD 改善」。 所有條件訊號要打敗的基準是：純 B&H、與無條件 put ladder（永遠避險）。

為什麼是誠實負結果頁

這頁的價值不在 headline 贏，而在過程嚴謹：每個亮眼格都做逐筆解剖（樂透分解）、 標注多重比較警告（100+ 配置下小樣本「正 EV」是雜訊的預期行為）、跑 IS／OOS 一致性與 skew 敏感度。 結論的一致性本身就是資訊：put 擇時的失敗不是訊號選得不好，是負 carry 在結構上壓過任何免費資訊能給的擇時精度。

Round 2：週線訊號與複合條件——所有「正 EV」格都塌縮成 2020-02 樂透

加測週乖離、週布林、更高日線門檻，及三個「保險便宜 AND 市場過熱」複合條件。兩輪合計 80+ 配置——多重比較之下，小樣本正 EV 是雜訊的預期行為。
展開 Round 2 逐項解剖

週布林（20 週，z ≥ 2.0／2.5／3.0）：死路。z ≥ 3.0 兩標的 22 年一次都沒觸發；z ≥ 2.5 只觸發 3–8 筆、putEV ≈ −1.00；週線平滑掉了日線布林賴以觸發的短期過熱。比日線版更沒有東西。

週乖離（週收盤 vs 52 週均線 ≥20%）：一格亮、解剖後是樂透。唯一亮格 QQQ ATM｜wbias52_20（CAGR 16.31%、putEV +0.28、IS／OOS 兩窗都贏），但 23 筆中 +$1.75M 幾乎全來自兩張樂透（2020-02-18 進場 +$1.74M、2024-07-08 +$2.01M）；扣掉這兩筆其餘 21 筆合計 −$2.00M。COVID 復甦漲勢中連虧 7 筆 −$2.7M。SPY 版 22 年零觸發、無法跨標的驗證。倖存者偏誤，不是結構優勢。

複合條件（便宜 AND 過熱）：理論最順、實證仍倒。「只在保險便宜且市場過熱時買」讓 putEV 從深負收斂到 0 附近——但收斂到 0 之後就停了。SPY cheap_stretch（VRP 低 AND 乖離 ≥10%）putEV +0.14，但最大一筆（2020-02-13 進場）佔正報酬 95%；扣掉它其餘 10 筆 −$0.98M。QQQ lowiv_hi 的 EV ≈ 0 但 Calmar 仍輸 B&H：「不虧錢的保險」依然拖累組合。

Round 2 總結：週線化、更極端門檻、複合條件三條路都試過，沒有推翻結論。能複製、能跨標的、能跨規格的穩健贏家仍然是零。

Round 3：Calmar 比較——趨勢閘門是免費午餐，再疊 put 反而倒退

結論：put 是保險，不是 alpha

核心結論：在長期持有 SPY／QQQ 之下，沒有任何條件訊號讓事前擇時買 put 變成穩定有利。 絕對面全部負 EV；組合面唯二通過門檻的配置一個是路徑運氣、一個只跟 B&H 打平；SPY 唯一贏 Calmar 的 iv_high80 是危機後反應式避險、對 skew 敏感、對陰跌倒貼。 不推薦把 put overlay 當作提升風險調整報酬的手段。

若目的純粹是保險（願意付確定成本換取尾部緩衝），資料支持的唯一誠實選項是無條件 ladder ＋ X1（3× 了結），因為只有它 100% 到場、能被普通回檔接住。已知成本收益如下（列出，非推薦）：
