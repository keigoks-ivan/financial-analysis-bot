路徑解剖

1999 年系統以 95% 曝險吃滿泡沫(+94%),S0 從未觸發。2000-03 頂點後，狀態層在 S1↔S1.5 之間徘徊， 2000-10-06 才進 S2(NDX 已自高點跌約四成)。隨後 Grid 接刀： G12(2000-11)→ G4(2000-12,曝險 80%)→ G3(2001-02-20,曝險回滿 95%)。 滿倉後 NDX 再跌 -61.6% 至 2002-10-07 低點，帳戶同段 -58.2%。 全程在 S2 蹲到 2003-03(狀態層判斷正確、零 whipsaw),毀滅全部來自 Grid 加碼後的續跌。

W_E 承受度(G4 輸入):W_E=50% 下，此路徑對總資產貢獻約 -38.5pp(Grid OFF 對照：-57.8% → 約 -28.9pp)。 IC-4「無斷路器、無更低防守層」當初在未檢視 G2 下承擔 — 此即當時繞過的東西。 註：NDX 為價格指數(無股息，當年 <0.5%/年)、現金以 ^IRX 13 週國庫券利率合成(2000-02 約 5-6%,有利防守端); 從泡沫頂點起算是刻意的最壞路徑設定。
