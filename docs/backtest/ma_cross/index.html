0/20 過尺。交叉系統 CAGR 常不輸(甚至贏)週線，但 MDD 一律更深——交叉出場落後，崩盤時多扛一段(SMH 10/30 MDD −44%)。即使 QQQ 50/200 raw Calmar 衝到 0.48>週線 0.43，同 CAGR 下它 MDD −28.6% 仍比週線滿倉 −25.4% 深 → 被尺擋下。慢趨勢跟隨的通病：用回撤換報酬，過不了「等報酬比 MDD」這關。

三條鐵律

1. 交叉是落後訊號。等快慢均線交叉，趨勢早走一段；進場買在追高、出場賣在落後，回吐大。事件前瞻邊際接近零。
2. 死亡交叉不是賣訊。死叉常出在反彈前(落後 + 均值回歸)，賣了反而後悔；做空更是踩進熊市暴彈區。
3. 慢趨勢跟隨用 MDD 換報酬，過不了尺。CAGR 可以不輸週線，但回撤更深；matched-CAGR 尺(等報酬比 MDD)正確判它輸。週線 score-5 仍是最優。

方法與限制

  1. 交叉定義：金叉 = 快均線(SMA)上穿慢均線；死叉 = 下穿。系統 = 快>慢做多、否則持有現金，交叉日進出。配對 10/30、20/50、50/100、20/100、50/200(經典金叉)。
  2. 成效定義：事件前瞻超額 = 訊號後持有 N 日報酬 − 同檔隨機日同期報酬。系統側用 matched-CAGR 尺(同 /backtest/daily_vs_weekly/)。
  3. 固定軸：成本單邊 7bps，閒置 SHY/BIL(美股)/現金(台股)。週線基準 = W52/104/250 score-5、破 W52 即出，對齊長軌頁發佈值。
  4. 與 MA 敏感度頁的分工：ma_sensitivity 測「交叉當連續濾網、哪個長度穩健」；本頁測「交叉事件本身的擇時邊際 + 交叉系統過不過尺」。
