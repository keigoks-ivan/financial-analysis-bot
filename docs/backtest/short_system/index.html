做空系統回測 — 失敗實驗紀錄

⚠ TL;DR — 為什麼這是失敗實驗

這個系統在 20 年裡累積虧損 $545K (-54.5%)，CAGR -3.97%，獲利因子 0.45。 我測試了 7 個變體 (包括 MA20 出場、N=2 確認、ATR 停損) — 沒有任何一個能轉正。 把它加入 80/20 組合 (LTO 80% + 做空 20%) — 沒有任何變體能讓 Calmar 提升 ≥ 0.05，全部反而拖累純 LTO 的表現。

失敗的根因: 不是參數問題,是市場結構問題。QQQ 過去 20 年從 $40 漲到 $540 (13.5x), 在這個史上最強的科技股多頭裡,任何單純技術做空系統都會輸給「均值回歸的反彈頻率 > 有意義下跌的訊號清晰度」。

1. 策略規則

第一層: 200DMA Gate (空頭過濾)

每日判斷: close < MA200 → short_mode = ON; 否則 short_mode = OFF。
Mode OFF 時不允許新做空進場;若已有空單,立刻按收盤價平倉。

第二層: 10/20/60 DMA Score 進場 (做空)

只在 short_mode = ON 時生效。同時滿足:

  1. close < MA10 AND close < MA20 AND close < MA60
  2. MA20 斜率 < 0 (lookback N=5)
  3. MA60 斜率 < 0 (lookback N=5)

出場 (敏感): close > MA10 立即平倉,N=1 (碰到就出)。
強制出場: 200DMA gate 翻 OFF 時強制平倉。

6. 失敗模式分析 — 為什麼這是市場結構問題

數學上為什麼不可能

做空系統要正期望值,需要滿足:
勝率 × 平均盈利 > 虧率 × 平均虧損

本系統實際:
    28% × $21,597 = $6,047 (期望盈利)
    72% × $18,505 = $13,324 (期望虧損)
    每筆期望值 = -$7,277

要打平,需要:勝率 ≥ 46% 或 盈虧比 ≥ 2.57x。 7 個變體裡最高勝率 28%、最高盈虧比 1.5x,沒有一個接近這些門檻。

結構性原因 (學術文獻早就證明)

  • QQQ 過去 20 年 13.5x 漲幅 — 史上最強的科技股多頭。
  • 不對稱性: 上漲沒上限、下跌有下限 (-100% 是最差)。多頭可以漲 1000%,空頭最多賺 100%。
  • 均值回歸的反彈頻率 > 趨勢延續的清晰度: 在空頭區間裡,2-5 天反彈是常態,真正持續下跌的窗口非常稀有。
  • 實務上還有借券成本: 本回測未計入。實際做空 QQQ 還要付 0.5-2% 年化借券利息和股息補償,會讓結果更差。

7. 教訓與替代方案

💡 學到了什麼

1. 做空美股大盤指數是結構性虧損業務 — 不是參數能解決的。
2. 「對的方向不等於對的執行」 — 2022 通膨熊是經典反例,QQQ 跌 30%、做空模式 ON 92% 的時間, baseline 系統還是虧 $63K,被 V 轉打爆 11 次。
3. MA10 出場太敏感是真實問題,但只是表象 — 修正後仍輸,因為更深層的問題是市場結構。
4. 「做空保險」這個概念對 LTO 純多策略沒有幫助 — LTO 本身的 -20% MDD 已經夠好, 加做空只會拖累報酬,MDD 改善有限 (~2pp)。

如果你還是想要「黑天鵝保險」,正確的做法

  1. OTM Put options on SPY/QQQ — 定期買 5% OTM put,用權利金當保險費。下跌時 P&L 槓桿放大,盤整時只虧定額權利金。
  2. VIX call options — 在 VIX < 15 (低恐慌) 時買遠月 call,危機時 VIX 飆到 30-80,期權翻數倍。
  3. 保留 10-20% 永久現金 — 不投入,等大跌時加碼。報酬機會成本明確、無爆倉風險。
  4. 加入 GEM 雙動能 — 前面證明 GEM 在 2008 完美避開 (1.30% vs SPY -36.81%), 提供與 LTO 不同的「危機機制」。組合配置 70% LTO + 30% GEM 是更實用的解。

這些都是「定義性止損 / 分散」型保險,不是「主動賺空頭錢」型保險。後者在歷史上唯一商業成功的案例是 long/short equity hedge funds, 但它們做的是個股 alpha 而非市場 beta,本質與本回測的單純技術系統完全不同。

8. 最終評等

⚠ 不建議使用 — 結構性失效

合格門檻 (CAGR ≥ 8%)未通過 (-3.97%)
合格門檻 (Sharpe ≥ 0.5)未通過 (-0.30)
合格門檻 (Calmar ≥ 0.3)未通過 (-0.07)
合格門檻 (MDD ≥ -35%)未通過 (-60.44%)
能否提升 LTO 組合 Calmar ≥ +0.057 變體全部失敗
是否有任何單一事件「值得做空」只有 2008 GFC 一次

這個系統的存在價值是「對照組 / 教訓」: 它證明「在正確時機 (200DMA gate) 用技術訊號做空美股大盤」這條路在過去 20 年是死路。 保留這份紀錄是為了避免未來重複相同的實驗,以及在有人提議「加做空 alpha 來源」時有數據可以拒絕。
