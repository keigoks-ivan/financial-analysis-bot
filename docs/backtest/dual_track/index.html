做空腿是致命傷;但移除它之後,任何權重下仍不及 Long Track Only

v7 Ch12 原始雙軌設計在修正方法基準(warmup 自上市起算、閒置收 SHY/BIL、收盤決策次日生效) 重跑後,結論比舊頁更強:樣本補回 2008 後 CAGR +4.35%、MDD -38.72%(舊頁 +4.60% / -26.83%, 漏掉 GFC 是 warmup bug 的假象)。本次補做兩個結構假設測試: H-A 移除做空 — CAGR +3.31pp、MDD 砍半(-18.48%),確認做空腿(勝率 19.2%、 獲利因子 0.41)是第一致命傷; H-B 軌道權重掃描 — 長多化之後 CAGR 隨長線軌權重單調上升,終點就是已採用的 Long Track Only(+9.72% / -20.08%)。 日線軌唯一的貢獻是回撤略淺與後十年 Calmar 較高,代價是 1~2pp CAGR — 不足以推翻現行 E3 採用。 「雙軌結構能不能救」的問題就此關閉。

已知限制與決策狀態

  1. 本頁是驗屍報告,不是活系統 — 雙軌多空 2026-06 已否決;本次重檢的目的是在修正基準上 關閉「結構能不能救」的問題,結論是不能。現行採用:核心 SPY/QQQ E3、進攻位 SMH/QQQ E3。
  2. 假設測試均為樣本內 — H-A/H-B 是同一段歷史上的重新切分;它們用於解釋失敗原因, 不構成新配置的採用依據。
  3. Layer-2 70/30 規格未變更 — 權重掃描是證據,不是決策;v7 Ch12 文件規格的變更需擁有者明示。
  4. 日線軌 H2 Calmar 優勢未追蹤 — A70 後十年 Calmar 0.99 的紀錄留檔;若未來市場進入 高頻 V 型反轉常態,可重開這條證據線,重開條件:E3 滾動 3 年 Calmar 連續落後 A70 對照線。
  5. 期貨槓桿未模擬 — 原始 Ch12 以期貨為標的;本頁用 ETF 全額保證金近似, 槓桿與保證金壓力下做空腿的傷害只會更大。

策略規則(v7 Ch12 原文,本頁未變更)

每標的資金分兩條獨立軌道,可同時持有相反方向;50/50 配置 SPY/QQQ。

短線軌(70% 資金,日線): 進場 收盤 > D60、D120、D200 且 D120、D200 斜率(5 日)> 0(score 5)且收盤 > W52(週線安全網); 出場 連續 2 日收盤 < D60。做空為鏡像(score 1、收盤 < W52)。
長線軌(30% 資金,週線): 進場 週收盤 > W52、W104、W250 且 W104、W250 斜率(4 週)> 0(score 5); 出場 週收盤 < W52(N=1)。做空為鏡像。

修正基準:warmup 自上市起算、模擬自 2006-01、收盤決策次日生效、 單邊 7bps 計於倉位變動、閒置資金收 SHY/BIL。 軌道與標的各自獨立計算,日報酬加權合成(不直接相加淨值)。
