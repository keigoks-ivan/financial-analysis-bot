1. 策略規則

原始 v7 Ch12 系統設計。每個標的內部分為兩條獨立軌道:
短線軌 (70% 資金): D60/D120/D200 日線均線,N=2 出場確認,可做多可做空。
長線軌 (30% 資金): W52/W104/W250 週線均線,N=1 出場,可做多可做空。
兩軌獨立運作,可同時持有相反方向。50/50 配置 SPY 和 QQQ 兩個標的。
