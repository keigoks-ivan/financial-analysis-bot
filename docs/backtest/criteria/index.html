📐 框架 v3 — 2026-06 修訂

「績效輸 Buy & Hold,可以說這個策略不好嗎？」— v2 答不了這題，所以有 v3

LT SMH/QQQ STX50 的 CAGR(+13.84%)輸給 SMH/QQQ B&H(+18.09%)。這算不算「策略不好」？ v2 的絕對門檻只能回答「能不能投」，從來沒有正式回答過「比最懶的替代方案好嗎」。 v3 把評估重組為五層漏斗:數字有效性 → 絕對門檻 → 相對基準雙軸支配性(新) → 組合邊際貢獻(新成文)→ 採用決策。核心新規則只有一條： 「策略不好」有且只有一個正式定義 — 對宣告的自然基準，在報酬軸與風險調整軸上同時落敗(被支配)。 其餘一切都是交換，值不值取決於風險預算，用第 2 層的四個工具判讀。 門檻帶維持 v1/v2 不變(它們剛通過 2026-06 的自我壓力測試);改變的是門檻在整個評估裡的位置。

0. 評估漏斗總覽

每一層回答一個不同的問題；任何一層出局就終止。層與層互相獨立 — 支配性過關不豁免門檻，門檻過關不豁免組合檢驗。
層回答的問題工具出局案例(站內)

1. 第 0 層 — 數字有效性(方法論紀律)

下游所有分析的前提。任何一條不過，該系統的數字視為無效，先修數再談評估。前五條來自 2026-06 修正週期；第 6~8 條為 v3 新增(可證偽性與預註冊承襲日內研究線，移植性來自 W52 多標的實驗)。
  1. 分母完整性 — CAGR 必須從模擬起點起算；MA warmup 用上市起的資料補齊，不准讓指標 NaN 吃掉前幾年。 教訓：舊 Long Track 頁的「20 年 +10.95%」分母實際只有 15.5 年(首筆交易起算),修正後 +9.59%; LTO QQQ 同病，+12.33% → +10.63%。凡是從首筆交易起算的歷史數字一律視為無效。
  2. 樣本內紀律 — 「優化」只做結構性消融(整個條件移除/加回),不做參數網格； 消融發現只記錄為證據，不在樣本內改規則；headline 永遠留給忠實基準。 案例：進場斜率冗餘(n=2 事件，未採用)、出場參數微調(實測更糟，否決)。
  3. 子期間雙視窗(H1 2006-2015 / H2 2016-至今) — 取代單一 walk-forward。 任一子期間 Calmar 不到全期一半 → 標記 regime 集中。 案例：LT SMH/QQQ 全期 Calmar 隨 SMH 權重單調上升，但 H1 在 50% 見頂、100% SMH 時只剩 0.16 — 「更好的系統」其實是「更大的 regime 賭注」。
  4. 現實假設 — 閒置資金計 SHY/BIL 利息(空倉時間集中在高利率熊市，影響不可忽略); 單邊成本 7bps;訊號在所屬 bar 收盤確認並執行。
  5. 預註冊與全揭露 — 門檻、網格、子期間在跑數之前申報；敏感度掃描全數揭露，不准沉默擇優。 日內線(TXF / SSF)已全面實施；權益線回測自 v3 起比照辦理。
  6. 統計可證偽性(v3 新增) — 樣本太少的優勢與運氣不可區分。交易筆數 < 20 筆/20 年的子軌， 其 standalone 指標降級為「描述性」，不得作為配置依據(長軌單獨約 8 筆/20 年即屬此類); 訊號層假設比照日內線標準：雙半期 t ≥ 2 才算存在。
  7. 合格 ≠ 採用，採用 = 決策 + 條件 — 數字全過只代表「值得進入 L4 決策」。 採用紀錄必須包含日期、保留意見、OOS 重審與退場條件(詳見第 6 節)。
  8. 規則可移植性不可假設(v3 新增，2026-06 W52 移植實驗的教訓) — 規則優勢是「規則結構 × 標的回撤-修復幾何」的交互作用，換標的 = 新系統，必須重走整個漏斗。 實證：W52 同規則零調整移植 — SPY 有效交換(風險調整勝出，非近支配)→ QQQ 失敗(無效交換)→ SMH 輸 Faber。 慢 U 型熊市(寬基)獎勵不對稱嚴進場；深 V 修復(高 beta 成長)讓嚴格進場變成結構性遲到。 「在 A 標的驗證過」永遠不是 B 標的的證據。

2. 第 1 層 — 絕對門檻(維持 v1 帶)

業界基準 + 個人判斷框架。「可疑」帶與「不及格」帶同樣重要 — 太好的數字是過擬合警報，不是慶祝理由。

及格8 ~ 12%接近大盤、回撤須顯著更淺
良好12 ~ 15%有明確 alpha
優秀15 ~ 20%頂級主動水準
可疑> 20%⚠ 過擬合 / regime 集中警報

不及格< 0.3報酬不抵回撤痛苦
及格0.3 ~ 0.5接近 Buy & Hold
良好0.5 ~ 1.0風險效率有實質優勢

輔助指標

Sortino:通常為 Sharpe 的 1.2~1.5 倍；顯著高於 Sharpe 代表獲利分布偏正(賺多虧少)。及格 ≥ 0.8、優秀 ≥ 1.2。
期末資產:CAGR + 時間的衍生品，不是獨立指標。$1M 起始 20 年：10% → $6.7M(B&H 水準)、12% → $9.6M、15% → $16.4M、20% → $38.3M(進入可疑區)。
交易次數:過少(<20 筆/20 年)→ 觸發 L0 可證偽性降級；過多 → 成本與執行負擔。趨勢系統合理區間約 25~100 筆/20 年。

最低門檻(合格)

  • CAGR ≥ 8%
  • MDD ≥ -35%
  • Sharpe ≥ 0.5
  • Calmar ≥ 0.3
  • 樣本期 ≥ 15 年(含至少 2 個熊市)
  • 子期間穩健：H1 / H2 Calmar 皆 > 0
  • 分母完整：CAGR 自模擬起點起算、MA warmup 補齊

優秀標準(有商業價值)

  • CAGR ≥ 12%
  • MDD ≥ -25%
  • Sharpe ≥ 0.8
  • Calmar ≥ 0.5
  • Sortino ≥ 1.0
  • L2 判定 ≥ 有效交換(v3:取代舊「Sharpe > B&H」單條)
  • 子期間平衡：H1 / H2 Calmar 差距 < 3 倍
門檻的角色與極限：絕對門檻回答「能不能投」— 它過濾不可投資的(回撤深到必然棄船、報酬低到不值得執行), 並用可疑帶攔截好到不像話的。它回答不了「好不好」:一個 CAGR 14% 的系統可能輸給躺著不動(基準 18%), 一個 CAGR 9% 的系統可能是站內最接近支配基準的交換(W52)。「好不好」是第 2 層的工作。

3. 第 2 層 — 相對基準：雙軸支配性分析(v3 新增核心)

每個系統必須宣告自然基準 = 同一資產池裡「最懶但可執行」的替代方案(通常是 B&H),再加同家族最強站內系統作第二基準。然後在兩條軸上比：報酬軸(CAGR)與風險調整軸(Sharpe = 波動效率、Calmar = 尾部效率 — 兩者回答不同問題，可能互相矛盾)。

五種判定

判定定義處置
支配候選 ⚠報酬軸與風險調整軸全贏 不是慶祝 — 先觸發過擬合 / regime 集中審查(全贏太稀有，通常是樣本或方法問題)
近支配報酬軸幾乎打平(讓 < 1pp),風險調整軸明確贏 最理想的真實狀態；確認 L0 有效性後優先配置
有效交換報酬軸讓分，Sharpe 與 Calmar 皆明確贏 值不值取決於風險預算 — 用下方四工具判讀，不能因 CAGR 較低就說「不好」
混合Calmar 贏、Sharpe 輸或打平 砍掉的是尾部、沒提升波動效率 — 對「以 MDD 定義風險」的人有價值，對「以波動定義風險」的人沒有；讓分越多越貴
無效交換報酬軸讓分，風險調整軸沒有換到(≈ 基準) 付了主動管理的代價、沒拿到保護 — 接近劣勢，除非 L3 有強互補證據，否則否決
被支配報酬軸與風險調整軸全輸 這是「策略不好」唯一的正式定義 — 直接否決，無需 L3/L4

判讀交換的四個工具

  1. 槓桿等價(等風險比較) — Sharpe / Calmar 的本義是「每單位風險的報酬」。在可槓桿工具(期貨)上， 風險效率高的系統可以加曝險到與基準相同的風險預算，屆時報酬軸的「落後」會消失甚至反轉 — 所以在固定風險預算下，Calmar 高的那個才是更好的系統。 三個必揭露的 caveat:波動拖累使報酬非線性、融資成本吃掉一部分、槓桿下 MDD 放大也非線性。 本站實際不使用槓桿；此論證的用途是確立比較的正確座標系,不是建議開槓桿。
  2. 可持有性 — B&H 的 CAGR 隱含「完美持有紀律」：那 18% 的前提是抱住 -57% 不砍倉、不被贖回、不手滑。 回測裡的 B&H 沒有情緒，真實資金有。一個 MDD -22% 的系統和一個 -57% 的基準， 在「真的能執行 20 年」這個維度上不是同一種東西。
  3. 終值誠實 — 反過來也要誠實：複利下報酬軸的讓分極其昂貴，不准只報 ratio 不報終值。 讓 4pp、20 年 = 終值少一半。如果效用函數就是終值極大化、且風險預算真的吃得下基準的回撤 (部位小、無槓桿、無贖回壓力),那 B&H 對那個人就是更好的選擇 — 這不是策略不好，是目標函數不同。
  4. 終點敏感性 — 結束在牛市頂點的樣本偏袒 B&H,結束在熊市谷底的樣本偏袒趨勢系統。 判定前在 2~3 個替代終點(上一次熊市谷底、上一次盤整尾)重看結論是否翻轉； 會翻轉的判定要標註，不會翻轉的才算穩。統計上同理：20 年樣本中 Sharpe 差距 < ~0.15 通常無法與雜訊區分， 但 MDD 砍半是結構性的(來自空頭期離場),不受小樣本懷疑論影響。

判讀範例：LT SMH/QQQ STX50 vs SMH/QQQ 50/50 B&H — 「CAGR 輸 4pp,這策略不好嗎？」

1
兩軸事實。報酬軸：13.84% vs 18.09%,讓 4.25pp — B&H 贏。 風險調整軸：Sharpe 0.90 vs 0.79、Calmar 0.63 vs 0.32(2 倍)、MDD -21.9% vs -56.9% — 策略全贏。 → 不是被支配，是有效交換。「不好」這個詞在定義上就套不上。
2
終值誠實。$1M、20 年：策略 14.2 倍 vs B&H 約 29.8 倍 — 終值少了近一半，這是交換的真實代價，必須明說。
3
槓桿等價。Calmar 0.63 vs 0.32:每承受 1 單位回撤，策略賺 2 倍。線性粗估 1.5× 曝險(期貨可行): CAGR ≈ 18~19%(已扣融資與波動拖累的量級)、MDD ≈ -31% — 報酬追平 B&H,回撤仍只有它的一半。 B&H 做不出反向操作(無法把 -57% 降下來而不犧牲更多報酬)→ 等風險座標系下，策略弱支配基準。
4
可持有性。B&H 的 18% 要求抱住 -57%(樣本內 SMH 段最深 -64%)。半導體單一產業 ETF 腰斬再加碼的路徑， 對多數資金配置不可執行 — 基準的報酬軸優勢有一部分是紙上的。
5
終點敏感性。本樣本終點(2026-06)落在 AI/半導體大多頭高點附近，B&H 的 CAGR 被終點加持； 終點若在 2022-10 或 2009-03,報酬軸差距大幅收斂甚至反轉。同一系統在 10 年窗(2016 起)對 QQQ B&H 是兩軸全贏(25.7%/-20.0% vs 21.7%/-35.1%)— 窗口與基準的選擇就能改寫故事， 所以判定必須：固定自然基準 + 兩個視窗取交集 + 標註終點位置。
6
結論。有效交換 — 用 4pp CAGR 買回撤從 -57% 到 -22% 的壓縮。值不值取決於風險預算； 此外它在組合裡是進攻位 sleeve(L3),最終考題是組合層的貢獻，不是 standalone 對 B&H 的擂台。 Sharpe 差 0.11 在 20 年樣本接近雜訊邊緣，但 MDD 砍半是結構性的 — 交換的價值主要在尾部，這也與「混合」型系統劃清界線。

4. 第 3 層 — 組合邊際貢獻(v3 成文)

sleeve 的最終考題不是 standalone 對 B&H,而是：加進組合後，組合的 Calmar / MDD 有沒有變好。一個 standalone 平庸的系統可能是好 sleeve(低相關),一個 standalone 合格的系統可能是壞 sleeve(與既有部位同漲同跌)。
  1. 危機期相關性優先於全期相關性 — 全期低相關但 2008/2020/2022 同步 -30% 的 sleeve 沒有分散價值； 相關性必須分段報：全期 / 各危機窗（2008/2011/2018/2020/2022）/ 高利率盤整段。
  2. with / without 是唯一裁決 — 把候選 sleeve 以擬議權重加入現行組合， 報組合層 CAGR / MDD / Calmar 的前後對照；邊際 Calmar 沒有改善 → 不論 standalone 多漂亮都不加。 案例：RSI2 MR standalone 已被支配，組合檢驗再確認「同權重換成現金更好」— 互補性被現金支配，雙重否決。
  3. 第二基準：同家族最強系統 — 新候選除了贏 B&H,還要跟站內同家族最強者比； 輸了就只剩「與它低相關」一條入場路。案例：週線 Supertrend standalone 輸 W52, 但與 E3 成員的出場時點互補 → 不作 standalone 採用，保留為 E4 成員候選 / 出場閘門(STX50 即此用法)。
  4. 角色決定評分權重 — 防守位(核心)看 Calmar 與危機行為；進攻位看 CAGR 與 regime 集中度(可疑帶); 互補位看相關性與危機月份報酬。用同一張表評所有角色 = 類別錯誤。 站內現行角色：防守主力 = E3(SPY/QQQ)、進攻位 = STX50(SMH/QQQ)、 互補缺口 = 跨資產趨勢(Turtle 商品腿 / Clenow)— 與美股趨勢家族低相關，是把組合 MDD 再往下壓唯一沒走過的路。
  5. 曝險調整視角 — 平均曝險 13% 的系統(RSI2)與滿倉系統的 headline CAGR 不可直比； 報「每單位曝險報酬」作輔助欄。但注意：曝險調整能解釋數字，不能豁免 with/without 裁決。
