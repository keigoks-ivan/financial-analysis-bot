1. 各指標的業界門檻

沒有單一指標能定義「合格」,但業界對每一項都有共識的等級劃分。

Sortino Ratio · 下行風險調整後報酬

通常是 Sharpe 的 1.2 ~ 1.5 倍。標準同 Sharpe 但更精準衡量「下行風險」。

若 Sortino 顯著高於 Sharpe，代表獲利分布偏正(賺多虧少),這是好事。 一般及格門檻 ≥ 0.8,優秀 ≥ 1.2,卓越 ≥ 1.8。

2. 組合標準

單一指標好不夠,要同時通過多項才算合格。

最低門檻 (合格)

  • CAGR ≥ 8%
  • MDD ≥ -35%
  • Sharpe ≥ 0.5
  • Calmar ≥ 0.3
  • 樣本期 ≥ 15 年
  • Walk-forward 通過 (Sharpe 差距 < 0.3)
  • 參數敏感度通過 (CAGR 極差 < 3pp)

優秀標準 (有商業價值)

  • CAGR ≥ 12%
  • MDD ≥ -25%
  • Sharpe ≥ 0.8
  • Calmar ≥ 0.5
  • Sortino ≥ 1.0
  • Sharpe > 同期 B&H Sharpe (證明有 alpha)
規則: 任何一條未達標 → 直接出局,無論其他指標多漂亮。例如 CAGR 20% 但 MDD -55% 的系統,還是不合格。

4. 個別系統評等

Long Track Only 合格 + 優秀 + 穩健

在 20Y 和 10Y 兩個視窗都通過所有最低門檻、Calmar 都 ≥ 0.55 (勝過 B&H)、 樣本期 16 年、Walk-forward + 參數敏感度都通過。有資格上實盤的核心系統。

LTO QQQ only Long Track 進攻變體

把 Long Track Only 從 50/50 SPY/QQQ 改成 100% QQQ 集中持有,放棄 SPY 的分散保護換取更高絕對報酬。 20Y CAGR 12.33% (vs 50/50 baseline 10.95%, +1.38pp),10Y CAGR 衝到 15.34%。 代價: MDD 從 -19.99% 加深到 -25.18%,Calmar 從 0.55 降到 0.49。 適合作為組合中的「進攻位」搭配防守型 (W52 / GEM)。 若單獨持有,50/50 baseline 仍是 Calmar 最佳選擇。

W52 斜率濾網 5/5 全部過關

20Y CAGR 9.59%、Sharpe 0.89、Sortino 1.14、MDD -20.05% 全部通過合格門檻, 且 Sharpe 0.89 超過所有其他系統。10Y 維持 -20.05% MDD,證明非對稱進出場設計穩定。 Beta vs SPY 僅 0.51 (相關性 0.72)，提供與 Long Track Only 不同的 alpha 來源。 適合與 Long Track Only 組合,降低集中度風險。

六狀態機 樣本內勝者,未驗證

20Y CAGR 15.14% 漂亮,但 MDD -40% 不合格。10Y 的 21.24% CAGR + 1.03 Sharpe 看似優秀, 但 21% CAGR 已進入「可疑」區,加上沒做 walk-forward 驗證, 應懷疑是過擬合。需要外推樣本驗證才能採信。

GEM 雙動能 老掉牙的合格系統

各項指標都剛好及格,沒一項突出。優點是邏輯極簡 (只有 2 個規則)， 20Y 和 10Y 表現幾乎一致 (CAGR 8.15% vs 8.93%),這個穩定性反而證明它沒過擬合。 信心度最高,但天花板也最低。

雙軌多空 不該存在的系統

兩個視窗都不合格。它存在的價值是當對照組 — 證明短線軌拖累了整個系統。 短線軌獲利因子僅 1.15,佔 70% 資金,把長線軌的優勢稀釋掉了。

5. 最終結論

推薦配置

單一系統: Long Track Only (50/50 SPY/QQQ, W52/W104/W250)
雙系統組合: 50% Long Track Only + 50% W52 斜率濾網
三系統組合: 40% Long Track Only + 40% W52 斜率 + 20% GEM

Long Track Only 衝報酬 (CAGR 11%, Sharpe 0.86)。 W52 斜率濾網 Sharpe 最高 (0.89)，月度決策、相對 SPY beta 僅 0.51,提供與 LTO 不同的 alpha 來源 (LTO 是雙週均線交叉、W52 是月度非對稱進出)。 GEM 降低 vol 並對 GFC 型危機提供額外保險 (2008 -1.30% vs SPY -36.81%)。

不推薦

六狀態機看似漂亮但 MDD -40% 不可接受,且未做穩健性驗證,有過擬合疑慮。
雙軌多空已被證明是錯誤設計,短線軌應被移除。
