今日觀點

TWVIX-RV（yang_zhang,20d）價差 +12.3 點，regime=RICH：賣方權利金相對『這波實際波動有多大』偏貴，對收租方有利，年化收租率因此顯著墊高。但誠實地說，RICH IV 常常和盤面波動放大（含急漲／melt-up）同時出現 —— covered call 在這種情境下會提早封頂、把最肥的那段漲幅讓給買方，『租金好收』和『放棄上檔的代價變大』是同一件事的兩面，不是只挑好的講。

今日 regime＝RICH（IV 36.61% 對 RV 24.32%，利差 +12.29pt）：IV 偏貴對賣方有利， 但 RICH 常常和盤面波動放大（含急漲／melt-up）同時出現——這正是 covered call 最容易提早封頂、把最肥漲幅讓給買方的情境。 「今天適合賣」和「賣了之後最痛的情境更容易發生」，是同一組數字的兩面。

研究

TWVIX 有歷史的 70 個交易日窗口（2026-04-01 ～ 2026-07-14）切出 3 段完整的 21 交易日滾動週期（2026-04-01 ～ 2026-07-03）。這段剛好是一段急漲行情：純 buy-hold 累計 +41.0%，covered call（每段重新賣一口 Δ0.30 call）累計只有 +19.3%，單段平均報酬 +12.5%（buy-hold）對 +6.1%（covered call）。落差主要來自第一段：指數單月 +22.9%、遠遠超過所賣 call 的履約價，封頂把大半漲幅讓給了買方 —— 這正是策略設計的取捨（收租換上檔），在這種急漲樣本裡會系統性跑輸，不是策略『失靈』。

誠實結尾

  1. 這頁證明了什麼：今天這一刻的結構是可信的——履約價、權利金、覆蓋率、壓力測試全部串起來， 且權利金已用 TAIFEX 即時結算價（real_settle）校正過，不是純模型 mark。
  2. 這頁沒有證明什麼：研究區段只有 3 個高度相關的完整月度週期（同一段急漲行情的不同切片）， 有效獨立樣本接近 1，不足以支撐「這個策略長期優於／劣於 buy-hold」的任何統計結論。
  3. 這段樣本剛好是策略最不利的情境：急漲行情下 covered call 累計 +19.29% 遠遠跑輸 buy-hold 的 +41.01%——這是設計上「收租換上檔」的必然代價，不是策略失靈，但也提醒不要只挑好的月份講故事。
  4. 下檔風險沒有乾淨的止損：最大虧損只是「整個部位歸零」的名目量級，實際下檔跟裸持股票幾乎一樣， 權利金只提供 2.36% 的緩衝，遇到修正或崩盤式下跌時杯水車薪。
  5. 覆蓋率不是 100%，封頂也不是絕對的頂：口數用資金整除取整數，殘餘未覆蓋名目在履約價之上仍全額參與漲幅， 這個殘值揭露在頁面上，不要假裝這個部位是乾淨的滿倉封頂。
  6. 要讓這件事變真，下一步是：累積更長的 TWVIX 歷史與真實 TXO 歷史成交鏈， 才能把「模型 mark 的 3 個週期」換成足夠樣本的可信回測；在那之前，這頁只是誠實的桌邊筆記，不是已驗證的 edge。
