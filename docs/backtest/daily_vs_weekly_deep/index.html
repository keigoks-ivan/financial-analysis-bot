✓ 結構性可預測(r=+0.88) · 但動態輪動不可吃

市場趨勢度以 r=+0.88 預測長軌 Calmar(事前可算)—— 但這是慢的結構性事實，不是可交易的動態訊號：跨市場輪動全輸「只用美國」

長軌的 edge 集中在強趨勢市場(全球頁)。本頁問四個更深的問題： ① 能不能事前量化？ 能——市場的週線效率比(ER,衡量趨勢乾淨度)以 r=+0.88 預測週線 Calmar。 ② 弱 edge 是市場還是匯率？ 是市場——換成當地貨幣一樣弱(改善 -0.01、與 FX 波動相關 +0.07≈0)。 ③ 為什麼弱市場弱？ 趨勢不持久→系統大半時間在現金(在場%低)+ 進場被洗(勝率低、持有短)。 ④ 能不能動態輪動吃？ 不能——用趨勢度或動量排名、只部署在「當下最會趨勢」的市場，全部輸給「只用美國」(Calmar 0.46 vs 0.09–0.16),而且追高反而加深回撤。 靜態關係轉不成動態訊號：這個 edge 只能用靜態選市場(部署在結構上會趨勢的美國)吃，不能用輪動吃。

所有輪動方案全輸「只美國長軌」(0.46),也沒贏「全市場平攤」(0.16),而且追高把 MDD 弄到 −29 ~ −40%。 原因：(1) 跨市場 ER 差距僅 0.17–0.26、滾動排名不能預測「下一期誰會趨勢」；(2) 集中在「當下最熱」的市場 = 買在已噴上去之後，這些多是 EM、然後一起 risk-off 崩(相關性高)→ MDD 更深。典型的動量崩盤/追高陷阱。

結論 — 靜態選市場，不要動態輪動

  1. edge 結構性、可事前預測(r=+0.88)。市場的週線趨勢度(ER)就能預判長軌在那裡行不行——這是真關係。
  2. 但靜態關係 ≠ 動態訊號。「有些市場數十年來持續比較會趨勢(美國)」是慢的結構事實；它不能轉成「這個月該押哪國」的擇時訊號——滾動趨勢度/動量排名跨市場無預測力，追高還反咬。
  3. 正確吃法 = 靜態選市場。把長軌部署在結構上會趨勢的市場(美國),就對了——這正是站上 US / 0050 長軌的現役配置。沒有任何全球輪動打得贏。
  4. 方法論：橫斷面 r=0.88 很誘人，但把它當動態訊號用就是過度外推。結構關係要用結構手段(選市場)吃，不是用時序輪動吃——這也呼應全站「真關係存在、但花俏 overlay 失敗、簡單結構勝出」的一貫主題。
