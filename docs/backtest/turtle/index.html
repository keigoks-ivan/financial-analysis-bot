評等與定位

在投資組合中的定位

Turtle 系統不是本站其他系統(LTO / W52 / 六狀態機 / GEM)的競爭對手或替代品 — 它是不同資產類別的補充。

適合的角色

  • 商品/外匯/利率部位的趨勢捕捉器 — 不要碰 SPY(修正版 3.1% 勝率已證明不適用)
  • 低相關性 alpha 來源 — 與 LTO / E3 等股市趨勢系統相關性低,組合風險分散好
  • 「衝報酬器」 — 原始 System 2 的 CAGR +22.53% 是本站最高,但 -38.33% MDD 不能單獨持有

不適合的場景

  • 單獨用於股票策略 — SPY 子集在修正版勝率 3.1%,完全失效
  • 「2026 修正版」 — 經回測證實是劣化版,不要採用
  • 不能承受 -38% 級回撤的投資人 — 趨勢系統的本質就是少數大年的暴利 + 多數年的小虧

建議使用方式(2026-06 拆解研究後更新)

用原始 System 2(55/20,不要 2026 修正),聚焦 CMDTY 三腿(USO / GLD / DBA) — 拆解研究證實 FXE 與 TLT 是淨負貢獻腿,舊版「含 FXE」的建議撤銷。 作為股票趨勢核心(E3 + STX50)之外的互補 sleeve,+20% 權重附近是甜蜜點 (組合 Calmar 0.64 → 1.03,月再平衡下成立;+30% 劑量遞減)。 實作需期貨或管理期貨工具(腿內部 ~3.3× 曝險);採用待擁有者 L4 決策。

已知限制

  1. ETF 是期貨的不完美代理 — 全額保證金、無槓桿;USO 內含展期損耗、FXE 不含利差收益。 真實期貨帳戶下槓桿與保證金壓力會放大回撤,本頁數字偏樂觀的一面是無保證金追繳,偏保守的一面是無法用名目槓桿放大 Sharpe。
  2. 隱含槓桿(2026-06 量測) — sizing 公式使總名目曝險平均 3.31×、p90 6.0×、最大 10.7×; headline CAGR 是「3.3× 曝險的報酬」,與本站無槓桿系統的 CAGR 不在同一基底上,比較請用 Sharpe/Calmar。 現金 ETF 帳戶無法執行本系統,需期貨保證金。
  3. 有效樣本自 2007-01-05 起 — 引擎取 6 市場交集,DBA 上市最晚; 「2005 起」的名目窗實際少了兩年,且錯過 2005-06 的商品行情前段。
  4. SPY 腿結論的邊界 — 「3.1% 勝率」出自修正版;原始版 SPY 腿 P&L 為正(勝率 22.0%)。 「指數不適用突破」的證據強度來自多個獨立來源(本頁 + 做空系統 + 雙軌做空腿),不依賴單一數字。
  5. 交易紀錄只含已平倉交易 — 期末仍持倉的部位計入淨值但不入交易表; 引擎不追蹤分單位進場日期,交易表以出場日排序、進場價為各單位均價。
  6. 成本模型簡化 — 2bps/邊滑價近似,未含佣金與市場衝擊; 高頻加碼(最多 4 單位)下實際成本略高。
  7. 敏感度網格是樣本內掃描 — 「最佳區間 Entry 40-45」是描述性證據,不構成參數變更的採用依據; 本頁維持 55/20 原始規格。
