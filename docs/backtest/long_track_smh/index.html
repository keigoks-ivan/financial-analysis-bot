🔬 盤整傷害假設測試(2026-06-12)

已知限制

  1. Regime 集中 — 全部超額報酬來自 2016 起的半導體超級週期;H1(2006-2015)它輸給現行配置。 現在採用 = 押注 AI/半導體資本週期延續,這是標的判斷,不是系統判斷。
  2. dot-com 不在樣本 — SMH 在 2000-02 跌約 -85%,資料自 2000-06 起、模擬自 2006 起; 「半導體寒冬下的 whipsaw + gap 成本」從未被實測。SMH B&H 樣本內 MDD 已達 -64.2%。
  3. 集中不是分散 — SMH/QQQ 標的相關 0.86,兩者都是科技;這個配置是「全倉科技 beta + 趨勢濾網」, 失去 SPY 腿的跨產業緩衝(2020 單週多跌 8pp 就是帳單)。
  4. 勝率更低 — 33.3% vs 現行 40.4%,SMH 波動大、whipsaw 更頻繁;心理負擔高於數字表面。
  5. 與六狀態機的對照是樣本內推論 — 「超額報酬是 beta 不是機制」的結論基於同一段歷史。 2026-06-11 已採用為進攻位;與六狀態機的並行對照持續記錄,半年重審一次。
