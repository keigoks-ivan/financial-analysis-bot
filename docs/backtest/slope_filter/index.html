結構性結論:W52 非對稱設計的優勢是 SPY 特定的。 「出場敏感 + 進場嚴格」賺的是慢牛慢熊的節奏財(SPY 的熊市夠長、修復夠慢,嚴格進場錯過的不多); 高 beta 成長標的(QQQ/SMH)的深 V 修復讓五條件變成結構性遲到成本,標的波動越烈、嚴格進場越虧。 家族角色不變:SPY 防守對照;進攻標的的趨勢曝險由 E3 / STX50 家族負責(週頻、進出對稱性更好)。

已知限制

  1. V 轉代價 — 每次急跌急彈(2009、2020)都晚 2~12 個月回場,錯過反彈初期最肥的一段; 這是五條件設計的結構性代價,不可參數化消除。
  2. 避險腿不是保險 — 2022 股債雙殺,AGG 同跌;空手段的「防守」依賴債券與股票負相關, 該假設在高通膨環境失效。
  3. 月頻決策 — 月底才動作,月內崩跌(2020-03)會完整吃到;對照日頻/週頻系統時注意頻率差異。
  4. 單標的 — headline 只有 SPY 一腿,沒有跨標的分散;2026-06 的 QQQ/SMH 擴展測試(上節) 證實規則不可直接移植到高 beta 標的,與 Long Track(SPY/QQQ)比較時注意。
  5. 換倉次數少(19 次) — 統計樣本小,參數穩健性主要靠規則簡單而非樣本量。

策略規則

出場(敏感,單一條件):月收盤 < W52 → 100% 轉入 AGG(2003 前以 SHY 拼接)。
進場(嚴格,五條件全過):月收盤 > W52 且 > W104 且 > W250, 且 W104 斜率 > 0 且 W250 斜率 > 0 → 100% 轉入 SPY。
週線均線(W52/W104/W250)以 W-FRI 週收盤計算,月底決策時取最近一根已完成週線的均線值(引擎 v2,2026-06 審計); 斜率 = 均線當值 − 前一週值。交易成本單邊 0.1%。月度決策,初始資金 $100,000。
