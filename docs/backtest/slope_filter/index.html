已知限制

  1. V 轉代價 — 每次急跌急彈(2009、2020)都晚 2~12 個月回場,錯過反彈初期最肥的一段; 這是五條件設計的結構性代價,不可參數化消除。
  2. 避險腿不是保險 — 2022 股債雙殺,AGG 同跌;空手段的「防守」依賴債券與股票負相關, 該假設在高通膨環境失效。
  3. 月頻決策 — 月底才動作,月內崩跌(2020-03)會完整吃到;對照日頻/週頻系統時注意頻率差異。
  4. 單標的 — 只有 SPY 一腿,沒有跨標的分散;與 Long Track(SPY/QQQ)比較時注意。
  5. 換倉次數少(17 次) — 統計樣本小,參數穩健性主要靠規則簡單而非樣本量。

策略規則
