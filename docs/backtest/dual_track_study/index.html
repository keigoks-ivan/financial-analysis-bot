方法論與限制

引擎：長軌 = ensemble_experiment.run_fractional(E3,真實 live 引擎);短軌 = rsi2_mr_backtest.run_strategy(BASE_CONFIG)。兩軌各自在 SMH/QQQ 上跑，轉日報酬後 50/50 跨標的、再依長/短權重合成(報酬加權，非淨值相加)。成本 7bps/邊。

限制：① 權重事後挑(in-sample);② RSI(2) 短軌依賴收盤/MOC 撮合，可執行性未定；③ 探索軸表為簡化 sleeve(示意);④ SMH 自 2010 起(液性);⑤ 研究頁，非規則變更。生產化要滾動權重 + 實單撮合驗證。
