機械版 ≠ 超級績效

即使偏誤在幫它，機械版仍跑輸大盤 — 它買到的是「回撤控制」，不是 alpha

把站上 RS+VCP screener 的最嚴條件(rs≥80、vcp≥75、accelerating、距高<5%、量比<0.8)規則化， 8% 硬停損 + 跌破 50DMA 出場，8 槽等權。結果：CAGR +12.2%、MDD -20.6%、Calmar 0.59。 對照同期 SPY 買持 13.6%、QQQ 買持 19.3% —— 即便存活者偏誤在灌水，CAGR 仍輸大盤(QQQ 海放)。 它真正交出的價值是 MDD 只有 -20.6%(大盤 −34%)。勝率僅 41%、353/464 筆出場是跌破 50DMA —— 機械進場抓不到 Minervini 裁量的 pivot 精度，緊停損被反覆洗。讓系統「超級」的那層，正是無法機械化的那層。

槓桿實驗：三根槓桿能不能把「報酬↔回撤」前緣往外推？

想逼近 Minervini「精準買在 pivot、停損幾乎不觸發、贏家一路抱」的 alpha,試了三個方向。每個都一次只改一個變數對照基線。結論：三根都推不開前緣 —— 因為那層 alpha 本質是裁量，不可機械化。

槓桿 A — 出場改聰明(分階段、隨獲利放寬)

只改出場一個變數(進場 cond_a、8 槽等權不變)。staged = 未證實前緊(8% 停損 + 跌破 50DMA);獲利 +20% 賣出半倉、停損拉到成本價；剩下的 runner 改用較寬的均線跌破才出 → 讓贏家一路抱。

方法論 — 怎麼跑的，哪裡忠實、哪裡不忠實

進場 / 出場 / 部位規則
  • 進場(cond_a,站上 screener 最嚴條件):rs_score≥80 且 vcp_score≥75 且 rs_trend=accelerating 且 距 90 日高<5% 且 量比(10/60 日均量)<0.8。歷史上平均 1.2 檔/日、53% 的交易日至少 1 檔。
  • 出場：進場價 −8% 硬停損(盤中，跳空穿越則以開盤價成交)或收盤跌破 50DMA(隔日開盤出),擇一先到。
  • 部位：8 槽等權，每筆 = 總資產/8,贏家續抱不再平衡；空槽持現金 → 自動漸進式曝險(平均 73%)。
  • 執行：訊號用當日收盤算、隔日開盤成交(無前視);成本 7bps/邊(5 佣金 + 2 滑價)。

為什麼這不是「超級績效」
Minervini 的 edge 主要在裁量的進場擇時(VCP 樞紐點的精準買進，讓 8% 停損幾乎不會被觸發)與基本面選股(SEPA)。 本回測只能機械化「趨勢樣板 + VCP 分數」這層可量化的濾網，抓不到那個 pivot 精度 → 緊停損被反覆洗(勝率 41%)。 加上存活者偏誤已在灌水，CAGR 仍輸大盤，證明：可機械化的部分不是讓系統超級的部分。
已知限制與下一步
  • 存活者 + 成分股前視偏誤(本頁最大限制)→ 真值看 Path A 前進測。
  • 未納入 Minervini 的漸進式加碼、賣在強勢(sell into strength)、基本面 SEPA 篩選。
  • 單週 gap 穿越停損無解(2020 COVID 即一例)。
  • 2026 半年 +49% 為小樣本，勿外推。
