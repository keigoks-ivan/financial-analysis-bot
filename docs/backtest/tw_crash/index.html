✓ 結論：含崩盤樣本一進來，趨勢就反過來風險調整支配 B&H

「台股 B&H 難敗」確實是 2009+ 無崩盤的樣本假象

所有 0050 波段結論都建立在 2009 起、整段缺 2008 的樣本上(總覽頁的 matched-window 已指出這是窗口效應)。 這頁把同一套零調參系統，套到歷史含 2000(−66%)與 2008(−58%)的台股標的：結果每個趨勢系統都把 MDD 從 −60〜−66% 砍到 −28〜−44%, Sharpe 持平略高、Calmar 全勝 —— 正是你跨系統的結構性教訓「趨勢溢酬在尾部，不在波動效率」在台股、用真實崩盤的再驗證。 保留：這是風險調整(尾部)支配,不是 CAGR 支配 —— 像 2330 這種一路噴的單名，B&H 原始報酬仍勝，趨勢用上漲讓步換崩盤砍半。

三個誠實的 nuance

  1. 這是「風險調整 / 尾部」支配，不是 CAGR 支配。Sharpe 只小勝、贏的是 Calmar(MDD 砍半)。 在像 2330 這種又強又久的單名上，B&H 的原始 CAGR / 期末資產仍勝($85M vs $30M)—— 趨勢用上漲讓步換崩盤砍半，值不值看你的回撤容忍度。 這正是你框架裡的「有效交換 / 風險調整支配」，不是「全面支配」。
  2. STX50 / E3 > Long Track 在高 beta / 深 V 標的(EWT、2330)。嚴格進場的 Long Track 墊底， 再次印證「進攻標的要用較對稱進出的家族」的可移植性教訓 —— 慢 U 型熊獎勵嚴格進場，深 V 修復懲罰它。
  3. Long Track 在 ^TWII 的 2000「+2%」是 warmup 假象，不是訊號。W250 要 ~5 年暖身，1999 起跑時它整段 2000 都還空在現金。 2000 真正有在交易的是六狀態 / STX50 / E3 / Faber,它們的 −17〜−40% 才是真保護。
方法與保留：五個系統全部零調參沿用(規則來自美股 / 0050 已固定 → 整段對台股天然 OOS)。 ^TWII 為加權價格指數(無股息),但 EWT / 2330 含息已佐證結論非無息造成。 Long Track 的 W250 暖身吃掉 EWT / 2330 前 ~3 年(在現金，壓低其 CAGR,已標註)。EWT 現金腿暫用 NT$ 1%(應為 USD 利率，對趨勢系統影響極小)。 危機窗口為峰→谷近似；交易成本沿用各 port 設定。原始數據 results/tw_crash_sample.json。
