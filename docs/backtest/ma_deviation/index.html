乖離 = 好買訊(極端) · 爛賣訊(全部)

把所有交易日按乖離率分桶，前瞻超額報酬是一條「微笑曲線」——兩端(深超賣 / 強噴出)有 edge，中段(貼著均線)最沒用；而當賣訊時兩個方向都是反指標

最強的買點是長多格局裡的深度回檔(價跌到上升 200 日線下 15%+)：60 日前瞻超額 +22.0%、勝率 96%——但極稀有(n=26)、跨標的不穩，幾乎注定上不了過尺尺。 反過來當賣訊更糟：高乖離之後續漲(賣=賣飛)、深負乖離之後反彈(停損=賣在最低)。把乖離出場疊到採用中的週線長軌上，2/24 過尺且皆為幻象——停損主動傷害、停利不是無感就是換個 k 就垮的單格雜訊。

① 買 — 乖離微笑曲線

每個交易日按「價格相對均線偏離 %」分桶，量該狀態下持有 N 日的報酬減同檔隨機日同期報酬(扣掉資產漂移)。>0 = 在這個偏離程度買有擇時邊際。往左=價在均線下方(超賣)，往右=上方(噴出)。此處為「均線上升中」版本(回踩/脫離在多頭格局)。

③ 把乖離出場疊到週線長軌 — matched-CAGR 過尺

用採用中的週線長軌(score-5 / 破 W52 出場)逐日部位，疊一道乖離出場(高乖離停利 dev≥+k% / 低乖離停損 dev≤−k%，乖離對 50 日線)。過尺 = Calmar > 週線 且 同 CAGR 下 MDD 比「現金稀釋的週線」更淺。綠=達/超週線。

怎麼讀這張表 ＋ 那把尺的白話

  • 底層不動：週線 score-5 進場 / 破 W52 出場 → 每天一個 0/1 部位(持有 or 現金)。
  • 疊乖離閘門(只減不加的 AND 遮罩)：停利 = 週線多單 且 dev<+k%；停損 = 週線多單 且 dev>−k%。它只能把週線本來要抱的部位砍掉一段,永遠不會加碼。
  • 欄位：CAGR/MDD/Calmar = 加閘門後的成績;「稀釋週線@同CAGR MDD」= 基準線(把純週線用現金稀釋到同 CAGR 後的 MDD,要被它打贏);曝險 = 在場內的時間比例(看閘門到底有沒有觸發)。
  • 過尺 = Calmar > 週線 且 MDD 比稀釋週線更淺(兩條都要)。
白話:想要較低的報酬,你大可不加任何閘門、直接少買純週線、其餘放現金。乖離閘門若不能在「同報酬下回撤更小」這件事上贏過這個最廉價的替代方案,就是沒有加值。

停損全數未過尺、且多數主動傷害(把你洗在最低，MDD 反而加深)；停利大多無感(SPY/QQQ 對 50 日線很少脹到 +10%，overlay 幾乎不觸發)，偶有 1～2 格過尺也是換個 k 就垮、跨標的不複製的單格幻象。乖離改善不了週線的出場。

三條鐵律

1. edge 在極端、不在中間。絕大多數交易日落在「貼著均線」的中段桶，那裡前瞻超額是負的。乖離只有走到尾端(深超賣 / 強噴出)才有訊號——但尾端事件稀有、跨標的不穩。
2. 乖離是好買訊、爛賣訊。高乖離 = 動能(別賣)，低乖離 = 超賣(別停損)。把乖離拿來當出場，兩個方向都站在報酬的對立面。
3. 出場改善不了週線。停利停損疊上去一律無效或有害——再次撞上鐵律：週線長軌的回吐/洗盤是保費，閘門只是搬石頭砸腳。核心倉位：在多頭排列裡待著。

方法與限制

  1. 成效定義：超額前瞻報酬 = 訊號後持有 N 日報酬 − 同檔「隨機任一日」同期報酬，隔離擇時邊際、不被資產長期上漂灌水。買側為「狀態條件期望」(每日一觀測、序列相關)，故只讀均值/勝率，不讀 t 值；賣側為跨越事件(較不重疊)。
  2. 過尺(matched-CAGR)：把週線稀釋到候選同 CAGR 後比 MDD，並要求 Calmar > 週線。raw Calmar 在不同 CAGR 間會誤導，這把尺才公平——與 /backtest/daily_vs_weekly/ 同一把。
  3. 固定軸：乖離對 MA20/50/200，上升過濾 = MA > MA(20日前)。前瞻 20/60 日。成本單邊 7bps，閒置 SHY/BIL(美股)/現金(台股)。週線基準 = W52/104/250 score-5、破 W52 即出。
  4. 真實資料：yfinance，週線基準對齊長軌頁發佈值(QQQ 10.80%/−25.37%/Calmar 0.43 等)。
