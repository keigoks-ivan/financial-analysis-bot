🎰 風險調整勝 B&H,但輸給更簡單的純長軌

曝險縮放狀態機：贏 B&H 的 Sharpe/Calmar/MDD,但被樸素的 Long Track 壓過

美股六狀態機(QQQ 核心 + SMH 攻擊 + BIL 安全的三狀態曝險縮放機)套到單一 0050:狀態由 0050 週線收盤 vs MA52 驅動(0.97 遲滯 + 連兩週規則),S1 滿倉 95%、跌破退 S2 防守 47.5%、再以 S5 每週 +15% 漸進回補。結果：在 0050 上是個體面的風險調整改善 — CAGR +14.14%(vs B&H +16.36%)、MDD -23.73%(vs -33.83%)、Sharpe 0.97(vs 0.94)、Calmar 0.60(vs 0.48)。但它輸給更簡單的純週線長軌(Sharpe 1.10/Calmar 0.68)— MA52 單閘 + 漸進回補的縮放，不如 W52/104/250 三線 score-5 的擇時乾淨。關鍵限制：美股版靠 SMH 攻擊腿的超額報酬，在單一 0050 下無法重現(攻擊腿被併入本體),所以這裡看到的只是縮放機制本身的價值。屬探索性測試，未採用。

已知限制

  1. SMH 攻擊腿缺席 — 美股版超額報酬主要來自 SMH/QQQ regime 賭注，單一 0050 下無法重現；本頁只反映縮放機制本身。
  2. 輸給更簡單的純長軌 — 風險調整被 Long Track(Calmar 0.68)壓過；多一層狀態機在 0050 沒換到更好的結果。
  3. S2 殘餘曝險 — 防守態仍留 47.5%,回撤保護不如完全出場的趨勢軌徹底。
  4. 樣本無 2008 — 0050 自 2009,系統性崩盤缺席；^TWII 對照補上。
  5. 資料品質 — yfinance 0050.TW 幻影分割已修復(2014-01-02(×0.24936));正式採用前應以 TWSE/券商還原股價重驗。

策略規則

六狀態機(v1.1 三狀態):狀態由核心資產週線收盤 vs 52 週均線(MA52)驅動。
S1 進攻 = 95% 權益 / 5% 現金；跌破條件(收盤 < MA52×0.97,或連兩個週五 < MA52)→ S2。
S2 防守 = 47.5% 權益 / 52.5% 現金；週五收盤 > MA52 → S5。
S5 爬坡 = 每週 +15% 權益，自 47.5% 漸進回補至 95% 後 → S1;爬坡中跌破 MA52 → 退回 S2。
單一 0050 移植：權益 = 100% 0050(美股版 QQQ 75%+SMH 20% 併入),現金 = NT$ 1.0/年；成本單邊 7bps、日頻 mark-to-market、週五決策。模擬自 2010-01。
