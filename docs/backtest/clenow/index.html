1. 系統規格

Andreas Clenow 在《Following the Trend》(2012) 提出的長期趨勢跟蹤系統,是 CTA 業界的標竿之一。 與海龜系統最大差異: 進場前必須先有趨勢過濾 (EMA cross),不是純突破。

進場規則 (3 條件全部滿足)

  1. 趨勢過濾: EMA(50) > EMA(100) (多頭) 或 EMA(50) < EMA(100) (空頭)
  2. 突破訊號: 收盤價 = 最近 50 日最高價 (多頭) 或最低價 (空頭)
  3. 未持倉: 該市場沒有現有部位 (不加碼)

波動率加權倉位 (Clenow 核心)

shares = (equity × 0.002) / ATR(100)
每個持倉貢獻約 0.2% 的每日帳戶波動率,組合層級目標年化波動率 15-20%。 與海龜的「每筆 1% 風險」不同 — Clenow 控制組合波動率,不是單筆損失。

出場規則 (任一觸發)

  1. EMA 反轉: 多頭時 EMA(50) 下穿 EMA(100); 空頭反之
  2. 止損觸發: 進場價 ± 3 × ATR(100), 不移動

9. 評等與定位

在投資組合中的定位

Clenow 系統不是本站股票系統 (LTO/W52/六狀態機/GEM) 的競爭對手 — 它是不同資產類別 + 不同策略風格的補充。

適合的角色

  • 多資產 trend-following 的基準 — Sharpe 0.71 是業界 CTA 的中位水準
  • 危機年份 alpha 來源 — 2008 (+32%)、2022 (+32%) 兩次危機年都大賺
  • 低相關性配置 — 與股票系統 (LTO/W52) 在不同年份表現,組合風險分散好
  • 絕對報酬最高 ($14.73M, 21 年) — 比所有股票系統 + 60/40 都多

不適合的場景

  • 不能承受 -44% 回撤 + 2 年套牢的人
  • 追求 Sharpe / Calmar 最佳: 60/40 或 Long Track Only 更好
  • 需要每年穩定報酬: 「失落十年」是真實的 (2012、2018、2019、2023 連續平庸/負)

建議的修正

  1. 剔除金屬類別 (拖累 -$1.17M) — PPLT/SLV/CPER 過去 20 年都是 whipsaw 熱區
  2. 剔除 PPLT 和 FXA — 個別大虧元兇
  3. 只用美股 ETF 子集 — 留 SPY/QQQ/IWM/XLK 等強趨勢標的,可能 CAGR 更高 + MDD 更小
