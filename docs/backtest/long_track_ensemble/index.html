訊號規則

W40 / W52:週線(W-FRI)收盤 > 40 週 / 52 週均線 → 該訊號看多。stateless,無進出場遲滯。
TSMOM:月底收盤的 12 個月總報酬 > 同期 SHY/BIL 報酬 → 看多。月頻,Moskowitz/Ooi/Pedersen (2012)。
倉位:看多訊號數 ÷ 3,即 {0, ⅓, ⅔, 1};訊號在所屬 bar 的最後交易日收盤確認並執行,次日起生效。
其餘假設與 Long Track 修正版完全相同:閒置部位 SHY/BIL、|Δ倉位| 單邊 7bps、 兩標的獨立跑訊號後日報酬 50/50 加權、warmup 自 1999、模擬 2006-01 起。

對照組:E0 = Ch12 修正版基準(score-5 進場 / W52 出場,二元);F1 / T1 = 單訊號二元版;E2 = 去掉 W40 的二訊號集成。
