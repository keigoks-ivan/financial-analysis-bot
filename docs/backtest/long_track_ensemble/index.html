E0 的回撤錄是「盤整、gap、緩熊」各一;E3 把盤整壓下去之後,gap(2020)浮上來變成 #1 — 風險不會消失,只會換形狀。要把 -21% 再往下壓,訊號層已經到頂,只剩部位大小與跨資產兩條路。

判決與後續

  1. 證據方向:E3 在 Sharpe(0.88)、Calmar(0.54)、子期間平衡上全面優於基準,CAGR +1.74pp, 換手率相當 — 是目前股票趨勢核心所有變體中綜合最好的。
  2. 為什麼不採用:(a) 全部為樣本內證據,Calmar +0.06 的幅度尚不足以排除運氣; (b) W40 引入新 lookback,是輕度參數選擇;(c) TSMOM 腿加深 gap risk(2020 -21.15% vs 基準 -18.75%), 2025-04 關稅崩跌型的快市對它不利;(d) stateless 訊號放棄了 Ch12 score-5 的進場紀律,屬規則變更, 需走 design/validation 流程。
  3. OOS 追蹤計畫:自 2026-06 起記錄 E3 與基準的實時分歧;觀察重點是下一次盤整叢集 (假設預測 E3 較淺)與下一次快速崩跌(假設預測 E3 較深)。兩個事件後重審。
  4. 最小可採納版本:若未來只取一步,E2(W52 + TSMOM,不含 W40)是不引入任何新參數的版本, 但其 COVID 回撤 -23.18% 更深 — 取捨記錄在案。

訊號規則

W40 / W52:週線(W-FRI)收盤 > 40 週 / 52 週均線 → 該訊號看多。stateless,無進出場遲滯。
TSMOM:月底收盤的 12 個月總報酬 > 同期 SHY/BIL 報酬 → 看多。月頻,Moskowitz/Ooi/Pedersen (2012)。
倉位:看多訊號數 ÷ 3,即 {0, ⅓, ⅔, 1};訊號在所屬 bar 的最後交易日收盤確認並執行,次日起生效。
其餘假設與 Long Track 修正版完全相同:閒置部位 SHY/BIL、|Δ倉位| 單邊 7bps、 兩標的獨立跑訊號後日報酬 50/50 加權、warmup 自 1999、模擬 2006-01 起。

對照組:E0 = Ch12 修正版基準(score-5 進場 / W52 出場,二元);F1 / T1 = 單訊號二元版;E2 = 去掉 W40 的二訊號集成。
