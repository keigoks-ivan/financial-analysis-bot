誠實結尾

  1. 賣的是有限風報比，不是印鈔機。今天這張單的損益兩平勝率是 91.7%—— 代表大多數情境下小賺，一次尾部情境就可能吃掉多筆順風單的收益，這是結構本身的性格，不是運氣問題。
  2. 1000 點護盤腳把虧損封頂，但封頂的數字本身仍然真實存在。最大虧損 NT$596,050， 「到期崩盤」情境下就是這個數字全額實現——買長腳的目的是讓虧損有底，不是讓虧損消失。
  3. 樣本極短，regime 判斷不具長期代表性。TAIEX/TWVIX 為共用資料層每日快照；TWVIX 歷史僅 70 個交易日（3.5 個月），樣本極短，regime 分類與研究統計不具長期代表性。真實 TXO 結算價快照日：2026-07-13（TAIFEX openapi 只給最近一個交易日的快照，非逐日歷史；若快照日早於 as_of，兩者間的現貨落差以模型價交叉核對）。 結構定價：優先採用 TAIFEX 真實結算價（模型價僅作交叉核對）。壓力測試一律用 Black-76 重定價（沒有歷史選擇權鏈可回測，模型 mark 不是真實成交價）。 今天判定 RICH 只代表跟過去 3.5 個月比起來權利金偏肥，不是跟過去多年台股歷史比較後的結論。
  4. 歷史校準用的是現貨報酬，不是選擇權真實損益。沒有 TXO 歷史選擇權鏈可以回測這個結構的真實成交價， 壓力測試情境的量級是照 TAIEX 現貨史上最差單日／五日報酬校準，用 Black-76 重定價模擬出來的，不是實際歷史損益序列。
  5. 這頁證明了什麼：今天這一刻 IV 對 RV 確實偏貴、賣方結構上有利，管線（今日觀點、建議結構、壓力測試）邏輯一致； 這頁沒有證明的是這個 edge 能撐過一次真正的崩盤——因為有 TWVIX 資料以來，這個策略還沒被崩盤考過。
