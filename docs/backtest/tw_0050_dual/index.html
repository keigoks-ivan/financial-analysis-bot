雙軌多空在 0050 被支配:做空全程虧、日線軌加劇 whipsaw,純週線長軌每一軸都更好

把 v7 Ch12 的原始雙軌系統(日線短軌 D60/120/200 + 週線長軌 W52/104/250,兩軌獨立、皆可多可空,ST 70%/LT 30%) 規則完全不變套到 0050。誠實結論與美股 dual_track 頁一致:被支配。 拆解四個變體就看得出價值從哪裡漏掉 — D0 全雙軌多空(Sharpe 0.92、Calmar 0.39、MDD -30.83%)輸給它自己的純週線長軌 A0 (Sharpe 1.10、Calmar 0.68、MDD -21.30%)的每一個軸。 兩個元兇:① 做空是純拖累 — 日線短軌做空 3 筆、勝率 0%(台股長多和美股一樣懲罰空單), 去掉做空的 A70 立刻把 Sharpe 從 0.92 拉回 1.02; ② 日線短軌加劇 whipsaw — 純日線軌 MDD -38.5%,比純週線長軌 -21.3% 深得多。 D0 對 B&H 也是輸:CAGR +12.17% vs +16.36%、Calmar 0.39 vs 0.48。 它確實買到真實的緩跌保護(2011 +0.7% vs B&H -19.9%、2020 COVID -5.2% vs -28.2%、 ^TWII 2008 GFC -37.97% vs -54.01%),但保費(盤整來回 + 讓出報酬 + 單週 gap 無解)在 0050 太貴。 本頁為否決記錄:雙軌多空在 0050 不該採用,純週線長軌(見 0050 STX50 頁的 Ch12 binary)嚴格更優。

已知限制

  1. 做空可行性與成本 — 台股做空需融券(有券源/利息/強制回補限制)、反向 ETF(00632R,有每日複利耗損)或期貨; 本回測以部位模型呈現,未計這些摩擦,真實做空成本只會讓 D0 更差。
  2. 長軌暖身偏誤 — W250 需約 5 年,0050 資料自 2009,30% 的 LT 子軌 2010–2013 被迫持現金; A0(純長軌)的優異部分來自這個結構性偏誤,公平比較見 0050 STX50 頁的相同註記。
  3. 樣本無 2008 — 0050 自 2009 起,雙軌最該發光的系統性崩盤缺席;^TWII 對照補上 2008,結論仍是被支配。
  4. 資料品質 — yfinance 0050.TW 有幻影分割壞 bar(2014-01-02(×0.24936))已修復;正式採用前應以 TWSE/券商還原股價重驗。
  5. 單一標的高集中 — 0050 約 5-6 成半導體;無跨產業緩衝,單週 gap 直接全額承受。

策略規則(v7 Ch12 雙軌)

日線短軌(70%):指標 D60/D120/D200(日線),斜率 D120/D200 回看 5 日。 score 5(多)= 收盤 > D60、D120、D200 且 D120、D200 斜率 > 0;score 1(空)= 全部相反。 進場多 = score 5 且 收盤 > W52(週線安全網);進場空 = score 1 且 收盤 < W52。 出場 = 連續 2 日收盤跌破/突破 D60(N=2)。
週線長軌(30%):W52/W104/W250,斜率 W104/W250 回看 4 週。 score 5/1 同義;出場 = 1 根週線跌破/突破 W52(N=1)。
獨立性:兩軌完全獨立、各自管理資金、可同時持有反向部位(regime 轉換時短軌多 + 長軌空)。 Long only 變體(A70/A100/A0)= 關掉做空。
成本單邊 7bps 計於 |部位變動|;閒置與做空抵押品以 1.0/年 NT$ 短率計息; 四個子軌各自獨立跑、日報酬加權合成(CLAUDE.md Rule 1)。模擬自 2010-01。
