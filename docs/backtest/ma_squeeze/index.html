✗ 糾結不加值 — 點火 edge 為負，過尺是底層趨勢在做工

把多條均線糾結(收斂)後的向上突破當「趨勢點火」進場——但糾結這道濾網的前瞻超額是負的，而且比沒有糾結的純突破還差；系統看似過尺，拆開看是底層的「突破持有 + 跌破緞帶底(≈200日)出場」在做工，不是糾結

點火事件 60 日前瞻超額 -2.1%(勝率僅 41%)，比「無糾結純突破」對照的 -0.8% 更差——緊縮後的突破更容易假突破。系統側 3/8 過尺，但無糾結對照在同標的過得一樣好或更好(QQQ 純突破 0.63 ≥ 糾結 0.57/0.61)→ 過尺來自趨勢持有 + 緞帶底出場，糾結沒貢獻。 且過尺全擠在 QQQ/SPY(週線基準相對弱)、在 SMH/0050(週線強)上全失敗——正是弱基準假象，非真 alpha。

關鍵對照：QQQ 上「純突破」(0.63) ≥「糾結」(0.57/0.61)——糾結濾網沒加值，過尺全是底層的「突破趨勢持有 + 緞帶底出場」在做工(緞帶底≈200日線，是個不錯的出場)。SPY 只在 q=0.33 單格過(脆弱、q=0.2 就掉)。SMH/0050 兩者皆敗。過尺集中在週線最弱的 QQQ/SPY、缺席於週線最強的 SMH/0050 = 弱基準假象。

三條鐵律

1. 糾結後的突破更容易假突破。低波動緞帶收斂後的第一個突破，前瞻超額是負的、且比一般突破更差——「蓄勢待發」是事後敘事，事前沒有可交易的方向 edge。
2. 對照組是照妖鏡。系統過尺要先問「拿掉這個點子，剩下的部分(突破持有 + 200日出場)是不是自己就過了」。這裡答案是 yes ⇒ 糾結沒貢獻。
3. 過尺集中在弱基準市 = 假象。只在 QQQ/SPY(週線相對弱)過、在 SMH/0050(週線強)敗，是擊敗弱基準的低標假象，不是 alpha——與全球 14 國的結論一致。

方法與限制

  1. 糾結定義：緞帶 = SMA 20/50/100/200 日；寬度 =(最大 MA − 最小 MA)/收盤；糾結 = 寬度落在過去 252 日的最低 q 分位(q=0.20/0.33,越小越緊，非循環/僅用過去)。發散點火 = 糾結後 10 日內收盤突破緞帶上緣。系統出場 = 收盤跌破緞帶下緣。
  2. 對照組(關鍵):同樣的緞帶突破進、緞帶底出，但不要求事前糾結——用來隔離「糾結」本身的貢獻。
  3. 成效定義：事件前瞻超額 vs 同檔隨機日；系統用 matched-CAGR 尺(同 /backtest/daily_vs_weekly/)。成本單邊 7bps,閒置 SHY/BIL(美股)/現金(台股)。週線基準對齊長軌頁發佈值。
