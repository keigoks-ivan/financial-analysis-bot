⚠ 長軌的 edge 集中在強趨勢市場 · 「日線過尺」是弱基準假象

多數國家 ETF 的週線 Calmar 都很弱(0.05–0.30,中位 0.13)—— 長軌不是全球普世；而日線「過尺」全擠在週線最弱的市場，是低門檻假象，非日線 alpha

美股(SPY/QQQ/SMH 週線 Calmar 0.43–0.57)和台灣 0050(0.90)是例外:那裡有乾淨持久的趨勢，長軌週線發揮得好、日線怎麼修都贏不了(0/72、0/24)。 但在其他 14 國，週線 Calmar 普遍只有 0.05–0.30——長軌的風險調整 edge 並非全球普世，而是集中在強趨勢市場。 日線在 9/14 國「有格子過尺」，但全部擠在週線最弱的市場(日本 0.05→12 過、印度 0.12→9 過、港 0.13→6 過、巴西 0.05→5 過；而韓 0.30/加 0.25 幾乎不過)—— 這是打敗一個趨近零的基準的低門檻假象，那些「最佳日線」本身也只有 0.2–0.33 的爛值。過尺數 ∝ 週線之弱，不是日線之強。

看「週線 Calmar」與「過尺/24」兩欄的反向關係：週線越強(韓 0.30、加 0.25)→ 日線越過不了；週線越弱(日 0.05、巴西 0.05、印 0.12、港 0.13)→ 過尺越多。 這把「日線過尺」釘成弱基準假象——不是日線在那些市場找到 alpha,而是長軌的週線在那些市場本來就不太管用(兩個時間架構都平庸)。ATR 帶在 3 國「過尺」(日/港/巴西)也全是同一回事：都是週線最弱的市場。

結論

  1. 長軌的風險調整 edge 不是全球普世，而是集中在強趨勢市場。US(0.43–0.57)、0050(0.90)是少數真正發揮的；14 國 ETF 週線 Calmar 中位僅 0.13,0/14 達到美股水準。
  2. 「日線打不贏週線」這條結論在強趨勢市場穩固成立(美股 0/72、0050 0/24,以及這裡週線較強的韓/加);在週線本就弱的市場，日線能「過尺」純粹是門檻在地板上，best daily 仍只有 0.2–0.33。
  3. 方法論：matched-CAGR 尺是相對基準。跨市場比較時，「過尺率」要連同「基準有多強」一起看；否則弱市場的低門檻會被誤讀成日線的優勢。
  4. 實務：這套長軌要部署，標的選擇(有沒有乾淨持久趨勢)比時間架構/濾網的微調重要得多——這也呼應 全球斜率穩健性 的市場相關結論。
