1. 策略規則

與 Long Track Only (50/50 SPY/QQQ) 完全相同的規則,但只持有 QQQ:
週線均線 W52 / W104 / W250,斜率 lookback N=4 週,出場確認 N=1 (週線收盤跌破 W52 即出場)。
進場條件: 週線收盤 > W52, W104, W250 且 W104, W250 斜率 > 0 (score == 5)。
放棄 SPY 的分散保護,集中曝險於 NASDAQ 100 (科技股),追求更高絕對報酬。
