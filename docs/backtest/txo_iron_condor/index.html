誠實結尾

  1. 賣的是有限風報比，不是印鈔機。今天這張單的獲利區間寬度（佔現價%）是 30.46%（38,737 ~ 52,363）—— 代表大多數情境下兩翼都小賺，一次尾部情境（不論方向）就可能吃掉多筆順風單的收益，這是結構本身的性格，不是運氣問題。
  2. 兩翼護盤腳把虧損封頂，但封頂的數字本身仍然真實存在，而且是雙向的。最大虧損 NT$125,488， 崩跌或噴出到期情境下就是這個數字全額實現——買兩翼更遠履約價的目的是讓虧損有底，不是讓虧損消失， 而且這個底是「兩個底」，不是單邊裸賣的「一個底」。
  3. 樣本極短，regime 判斷不具長期代表性。taiex.csv / twvix.csv 僅約 3.5 個月連續每日歷史，無完整逐履約價歷史選擇權鏈，VRP regime 分類是用這段短樣本自己的滾動 zscore 算出，不是完整選擇權史校準的統計分布，研究段落的結論僅供參考、非可信 edge。 今天判定 RICH 只代表跟過去約 3.5 個月比起來權利金偏肥或偏薄，不是跟過去多年台股歷史比較後的結論。
  4. 歷史校準用的是現貨報酬與模型定價，不是選擇權真實成交損益。沒有 TXO 逐履約價歷史選擇權鏈可以回測這個結構的真實成交價， 壓力測試情境是照 TAIEX 現貨史上情境量級、用 Black-76 重定價模擬出來的；真實結算價快照也顯示模型未建模 put skew， call 側可能被本模型高估、put 側被低估（詳見上方研究段落）。
  5. 這頁證明了什麼：今天這一刻兩翼權利金確實處於 RICH 水位、雙賣結構在權利金厚度上有優勢， 管線（今日觀點、建議結構、壓力測試）邏輯一致；這頁沒有證明的是這個結構能撐過一次真正的崩盤或軋空—— 因為有 TWVIX 資料以來，這個策略還沒被任一個尾部情境真正考過，而且它同時暴露在兩個尾部之下，不是一個。
