# DS Schema — 11 章節欄位必填 / 選填

每個章節列出：**最低必填內容**（critic 會檢查）與 **選填強化**。

---

## §0 TL;DR + ds-meta

| 欄位 | 必填 / 選填 | 內容規格 | 字數 |
|:---|:---:|:---|:---|
| `<meta name="ds-skill-version">` | 必填 | content="v1.0" | — |
| `<meta name="ds-theme">` | 必填 | CamelCase theme | — |
| `<meta name="ds-publish-date">` | 必填 | YYYY-MM-DD | — |
| `<script id="ds-meta">` JSON | 必填 | 見下方 schema | — |
| Cross-link callout | 條件必填 | 若 `related_ids` 非空，必須出現於 §0 內 | 1 段 |
| `.ds-thesis` thesis box | 必填 | 一句話 thesis ≤ 200 字 + 1-2 段展開 | 100-200 字 |
| 段落引文 | 必填 | ≥ 1 段敘述介紹 DS 主軸 | 100-200 字 |

### ds-meta JSON 欄位

| key | 型別 | 必填 | 規格 |
|:---|:---|:---:|:---|
| `theme` | str | 必 | 一句敘述題目（中文 / 英文皆可） |
| `skill_version` | str | 必 | `vN.N` |
| `ds_version` | str | 必 | `vN.N` |
| `publish_date` | str | 必 | `YYYY-MM-DD` |
| `thesis_type` | enum | 必 | `structural` / `event-triggered` / `mixed` |
| `ai_exposure` | enum | 必 | 🟢 / 🟡 / 🔴 |
| `mega` | enum | 必 | 15 mega 白名單之一（`semi` / `bio` / ...） |
| `sub_group` | enum | 必 | mega 對應 sub_group 白名單之一 |
| `quality_tier` | enum | 必 | `Q0` / `Q1` / `Q2` |
| `oneliner` | str | 必 | ≤ 200 字一句話 thesis |
| `history_window_years` | int | 必 | 1-200，標示 §1 回顧多遠（年） |
| `forecast_horizon_years` | int | 必 | 1-30，標示 §6 推估到幾年後 |
| `related_tickers` | list | 必 | 至少 1 筆，每筆 `{ticker, role, depth, beneficiary}` |
| `related_ids` | list | 選 | 同 theme ID 檔名 stem（不含 `.html`） |
| `sections_refreshed` | dict | 選 | keys: `history` / `supply_demand` / `forecast`，values 為 YYYY-MM-DD |

## 全文配額

| 配額項 | 上限 / 下限 |
|:---|:---|
| 總字數 | 4,000-6,500 字 |
| 表格數 | ≤ 4 張（§2、§5、§6、§11 各 1） |
| 每張表行數 | ≤ 8 行（不含表頭） |
| 文字字元比例 | ≥ 80%（總可見字元中） |
| `.ds-implication` 段落數 | 每章 1 個（§0、§11 除外，共 9 個） |
